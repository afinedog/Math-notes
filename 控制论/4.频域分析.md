# 频域分析

- **基本思想**：
- **频域**：$w\in [0,\infty)$

## 频率特性 

- **频率响应**：线性控制系统中，正弦信号的频率为 $w$，稳态输出随频率从 $0$ 到 $\infty$ 变化的规律
- **频率特性**：稳态正弦响应与输入正弦信号之比
  - **极坐标形式**：$G(jw) = \cfrac{X|G(jw)|e^{j\angle G(jw)}}{Xe^{j0}} = A(w)e^{j\p(w)}$
    - **幅频特性**：输出和输入的正弦信号的幅值之比 $A(w) = |G(jw)|$
    - **相频特性**：输出正弦信号的相角之差 $\p(w) = \angle G(jw)$
  - **直角坐标形式**：$G(w) = X(w) + jY(w)$
    - **实频特性**
    - **虚频特性**
- **图示方法**：
- **传函定理**：$G(jw) = \cfrac{Y(jw)}{R(jw)}$

### 开环频率特性

- **开环传递函数的频率形式**：由传函定理，$G(jw)H(jw)$
- **步骤**：
  - 开环传函写成时间常数型，得到：
    - 开环增益 $K$
    - 系统型别 $\nu$
    - 典型环节的串联方式
  - 确定各环节转折频率，由低到高排列，并绘制低频段、高频段渐近线
  - 求取误差曲线
  - 绘制曲线
    - **开环截止频率**：$L(w)$ 通过 $0dB$ 线的频率 $w_c$，用 $L_a(w)$ 近似代替

## 幅相频率特性

- **幅相频率特性曲线（Nyquist图）**：
  - **正方向**：逆时针

### 典型环节

- **求解**：
  - 首先求出 $G(s)$，替换成 $G(jw)$
  - 添项裂项化为直角坐标形式
  - 任意角公式化为极坐标形式

### 开环Naquist曲线

- **凹凸性**：
  - 若无一阶微分环节，则是纯凸曲线
  - 若含有一阶微分环节，则曲线会出现凹凸
- **步骤**：
  - 首先化为标准形式，求出 $A(w),\p(w)$
  - 求起始点、终止点、与实轴交点的（点）或（渐近线）
  - 根据凹凸性描线即可

## 对数频率特性

- **对数幅频特性**：$L(w) = 20\lg |G(jw)|$
- **对数频率特性**：$\p(w)$ 不变
- **对数频率特性曲线（Bode图）**：
  - 横轴是 $\lg w$，指数刻度，单位是旬距（$dec$）
    - 每走一个单位刻度，$\lg w$ 增加1，$w$ 增加十倍，单位长度中越往后越密
  - 纵轴
    - $L(w)$，线性刻度，单位是分贝（$dB$）
    - $\p(w)$，线性刻度，单位是度
- **优点**：
  - $w$ 较小时，横轴间距大，便于研究低频段
  - $w$ 较大时，横轴间距小，将中高频段划分到一起
  - 串联时，将幅值计算也化为了加减法
  - 典型环节的（频率特性曲线的渐近线）均可用分段的直线画出，容易求出从而传函

### 典型环节

- **截距**：一般是 $w = 0.1$，若取为 $w = 0$，则需要绘画低频渐近线（没必要，取 $0.1$ 就行）
- **比例环节**：斜率 $0$，截距 $20\lg K$（$w = 0$）
- **积分环节**：斜率 $-20dB/dec$，截距 $20$（$w= 0.1$）
- **惯性环节**：
  - 低频段斜率 $0$，截距 $0$<font color='chartreuse'>（比例环节）</font>
  - 高频段斜率 $-20dB/dec$<font color='chartreuse'>（积分环节）</font>
  - **转折频率**：$w = \dfrac{1}{T}$
    - （低频段渐近线）和（高频段渐进线）的交点的频率
    - 左边是低频段，右边是高频段
  - **折线对数幅频特性**：渐近线 $L_a(w)$
  - **误差修正曲线**：横轴为 $\dfrac{k}{T}$，纵轴为 $\D L(w) = L(w) - L_a(w)$
    - **应用**：在转折频率的某邻域内，对渐近线加以修正
- **振荡环节**：曲线，一般不会画
- **微分环节**：
  - 理想微分：积分环节的倒数，斜率 $20dB/dec$，截距 $-20db/dec$
  - 一阶微分：一阶微分环节的倒数
  - 二阶微分：曲线
- **延迟环节**：斜率 $0$，截距 $0$，相角 $\p(w) = -57.3\degree w\tau$

### 开环Bode图

- **串联定理**：多个典型环节串联起来的系统，其Bode图是多个典型环节曲线的拼接（串联的环节斜率相互叠加）
  - **理解**：
    - **闭环与开环的相对性**：多个闭环系统串联时，每个闭环传函在总系统中就相当于开环传函。<font color='chartreuse'>（只有总系统的首或尾处的反馈环节才会引出闭环传函，其它的都可以被简化为开环）</font>
- **近似定理**：振荡环节视为斜率 $-40dB/dec$，转折频率为 $s^2$ 的系数的倒数平方根（形式为 $a^2s^2 + bs + 1$ 时）
  - 相当于舍去第二项，并视为两个惯性环节
- **步骤**：
  - 化为标准形式，分解为典型环节乘积
  - 确定转折频率，由低到高标注
  - 求取截距
    - 若第一个是积分环节，则易求
    - 若第一个是惯性环节，则
  - 误差修正
  - 绘制曲线
    - **开环截止频率**：$L(w)$ 通过 $0dB$ 线的频率 $w_c$，用 $L_a(w)$ 近似代替

- **最小相位系统**：虚轴右侧无极点、零点
- **最小相位性**：相同幅频特性的系统中，最小相位系统的（相角变化量绝对值）最小
- **幅相对应性**：
- **非最小相位情况**：
  - 包含非最小相位环节
  - 含有延迟环节

### 反求开环传函

- **步骤**：已知实验曲线 $L(w)$
  - **近似**：用斜率为 $20k$ 的渐近线 $L_0(w)$ 对其近似
  - **一一对应**：由最小相位系统的幅相对应关系，写出 $L_0(w)$ 的最小相位传递函数 $G_0(s)$、最小相位对数相频曲线 $\p_0(w)$
  - **检验相位**：$w\to \infty$ 时，若 $L(w)$ 的斜率为 $-(n-m)20$，相角为 $-(n-m)\frac{\pi}{4}$，则是最小相位
  - **检验子环节**：若是非最小相位
    - **检验延迟**：若 $w$ 增加时，$\p(w)$ 相位滞后增加，说明有延迟环节
    - **检验延迟**：若 $w$ 增加时，$\D\p(w)$ 趋于常值，则超前环节含有非最小相位

## 频率稳定判据

- **时域分析判稳**：劳斯判据
- **频域分析判稳**：奈奎斯特判据

### 再论闭环特征函数

- $F(s) = 1+GH =  \cfrac{N(s)+M(s)}{N(s)}$ 
- **性质**：
  - 零极点：
    - 零点：闭环极点
    - 极点：开环极点
    - 零极点均为 $n$ 个<font color='red'>（实际系统中，总有 $n\geq m$）</font>
  - 像平面：
    - $F$ 像平面包围原点 $\LR$ $GH$ 像平面包围 $(-1,0)$


### 奈奎斯特判据

- **辐角原理**：$R = P-Z$

#### 开环传函无积分环节（原点不是闭环极点）

- **奈奎斯特围线 $\G_s$**：原像的右半平面无穷大半圆，周线方向为顺时针
  - 设
    - $F$ 在右半平面的零点数为 $Z$（闭环极点），极点数为 $P$（开环极点）
    - $\G_s$ 在 $F$ 下的像 $\G_F$ 包围原点的次数为 $R$
  - **性质**：用 $\G_s$ 和 $GH$ 函数，只需变动 $w$ 即可得到 $F$ 的极点位置
    - **证明**：
      - 由 $n\geq m$ 易得
        - 原像沿虚轴移动时，$F$ 的像是曲线的平移 $(1,0)+\G_{GH}$
        - 原像沿半圆移动时，$F$ 的像是单点 $(1,0)$
      - 从而只需要 $w$ 从 $-\infty\to\infty$，即可得到 $R$
      - 再由开环传函易得，故开环极点 $P$ 易得
  - **总结**：
    - 系统稳定 $\LR w$ 变化时 $GH$ 的像包围 $(-1,0)$ 的次数 $R = P$
    - 系统临界稳定 $\LR w$ 变化时 $GH$ 的像穿过 $(-1,0)$

#### 开环传函有积分环节（原点是闭环极点）

- **奈奎斯特围线**：挖去一个原点小圆

### 开环频率极坐标图法

- **对称性**：极坐标图下，设 $w$ 从 $0\to\infty$ 时包围 $(-1,0)$ 的次数为 $N$
  - 系统稳定 $\LR P = 2N$
  - 若系统不稳定，则极点个数为 $Z = P-2N$

### 穿刺法

- **正穿越**：$w$ 增加时，从上半平面通过 $(-\infty,-1)$ 穿到下半平面
- **半次穿越**：起始点在 $(-\infty,-1)$ 上，但没有穿越
- 系统稳定 $\LR N = N_+ - N_- = \cfrac{P}{2}$
- 若系统不稳定，则极点个数为 $Z = P-2N = P-2\big[ N_+ - N_- \big]$

### 对数法

- $(-1,0)$ 在Bode图上对应 $L(w) = 0dB$，$\p(w) = -\pi$
- **正穿越**：相频曲线从下往上穿过 $\p = -\pi$
- 系统稳定 $\LR N = N_+ - N_- = \cfrac{P}{2}$
- 若系统不稳定，则极点个数为 $Z = P-2N = P-2\big[ N_+ - N_- \big] $

## 稳定裕度

- **相位裕度**：幅值为 $1$ 时向量与负实轴的夹角
  - $\g = \p(w_c) +\pi$
    - **开环截止频率（幅值穿越频率）$w_c$**：幅值为 $1$ 时的频率
  - **意义**：将系统相角滞后 $\g$，则临界稳定。滞后 $>\g$，则不稳定
- **幅值裕度（增益裕量）**：相角为 $-\pi$ 时幅值的倒数
  - $h = \cfrac{1}{A(w_g)}$
    - **相角交界频率 $w_g$**：相角为 $-\pi$ 时的频率
  - **意义**：将系统幅值增大 $h$ 倍，则临界稳定。增大 $>h$，则不稳定

### 相对稳定性分析

- 相位裕度和幅值裕度大则响应速度满，小则振荡加剧
- **直接计算法**：设最小相位系统开环传递函数 $G(s)H(s) = \cfrac{K\prod\limits^m_{j=1} T_js + 1}{s^v\prod\limits^n_{i=1} (T_is + 1)}$
  - 则相位裕度为 $\gamma = \pi + \p(w_c)  = \pi - v\cdot \frac{\pi}{2} - \sum\limits^{n-v}_{i=1} \arctan T_iw_c + \sum\limits^m_{i=1} \arctan T_jw_c$
- **相对分析**：高阶系统无法准确计算 $w_c$ 的值，通过开环对数幅频渐进特性曲线穿过 $0dB$ 线的坐标 $L(w_c) = 0$ 的解来代替 $w_c$
- 若 $w_c$ 处 $L(w)$ 的斜率
  - 为 $-40dB/dec$，则可能稳定也可能不稳定，若稳定也相位裕度较小
  - 为 $-60dB/dec$ 或更大，则很可能不稳定
  - **中频段**：设计系统时，$w_c$ 附近应当在 $-20dB/dec$ 

## 频域分析

- **低频段**：第一个转折频率左侧
  - 由积分环节 + 开环增益决定
- **中频段**：$w_c$ 附近的频段
  - 最小相位系统中，$L(w)$ 和 $\p(w)$ 有唯一对应关系
  - $w_c$ 处斜率对 $\g$ 的影响最大
- **高频段**：中频段以后 $(w>10w_c)$ 的频段
  - $20\lg M(w) \approx L(w)$

### 设计理念

- 在阶跃和斜坡作用下无稳态误差
  - 低频段斜率 $-20$ 或 $-40$
  - 低频段具有较高的分贝值
- 具有足够的稳定裕度
  - $L(w)$ 以 $-20dB/dec$ 的斜率穿过 $0dB$ 线，具有一定的中频段宽度
- 快速响应能力
  - $L(w)$ 有较高的截止频率 $w_c$
- 抗干扰能力
  - $L(w)$ 的高频段尽可能低

## 习题

- 给出开环传函、输入信号，写出频率特性
  - 用开环传函算出 $A(jw)$、$\p(jw)$、闭环传函 $\Phi(jw) = Ae^{i\p}$
  - 由 $r(t)$ 得到 $w$，代入即可
- 画出Bode图并判稳
  - 由开环传函，写出转折频率
  - 由典型环节公式（背下来）写出 $\p$、$A$
  - 求相角裕度 $\g$
    - 利用 $A(w_c) = 1$ 求出开环截止频率 $w_c$
    - **$A(jw)$ 的估计法**：估计 $w_c$ 的大致位置，然后其左边的转折频率取系数，右边的转折频率取1
  - 若 $\g$ 为正，幅值裕度 $K_g>1$
- Naguist曲线判稳
  - 画出Naguist曲线
    - 求解 $w = 0^-,0^+$ 时 $G(jw)$，得到起始点
      - 若起始点是无穷远点，求解 $\lim\limits_{w\to 0} Re\ G(jw)$，得到渐近线与虚轴距离
    - 求解 $Im\ G(jw) = 0$，得到实轴交点
    - 求解 $w = -\infty,+\infty$ 时 $G(jw)$，得到终止点
      - 若终止点是无穷远点，求解 $\lim\limits_{w\to\infty} Re\ G(jw)$，得到渐近线与虚轴距离
  - 判稳
    - 求出开环传函的右端极点个数 $P$
    - 查看 $N$ 个数
      - 镜像包围法
        - 首先求出Naguist围线的 $0\to\infty$ 部分
        - 然后由于其关于实轴对称，对称曲线即为 $-\infty\to 0$ 部分
        - 然后观察围线即可
      - 穿刺法
    - 求出闭环右端极点个数 $Z = P-2N$
