# 根轨迹分析

- **基本思想**：系统的控制性能均由闭环传函的参数决定，而闭环传函由开环传函完全表出。写成零极点形式后，参数变为零极点与开环根轨迹增益。故只要分析零极点和根轨迹增益的动态变化即可得到控制系统的性能变化

## 根轨迹

- **根轨迹**：系统的开环传递函数的某个参数从0变为无穷时，闭环极点（特征方程的根）在复平面上变化的轨迹
- **根轨迹法**：在 $s$ 平面标注系统闭环极点
- **根轨迹方程**：$1+G(s)H(s) = 0$<font color='chartreuse'>（解即为闭环极点）</font>
  - **开环根轨迹增益**：$K^* = K^*_GK^*_H$
    - 零极点形式下，前向传递系数 $K^*_G$，反馈传递系数 $K^*_H$
  - **闭环根轨迹增益**：不能提公因式，从而不能写成系数形式，只能用开环系数+传函表示

### 闭环零极点传函

- **闭环零点**：前向零点、反馈极点
- **闭环极点**：难以求取
- **闭环根轨迹增益**：
- **形式转换**：$K^* = K^*_GK^*_H = \cfrac{K_GK_H\prod \tau_k\prod \tau_l}{\prod T_k\prod T_l} = K\cfrac{\prod \prod}{\prod \prod}$

### 求解根轨迹方程

- **根轨迹方程（开环零极点形式）**：$\cfrac{K^*\prod\limits^m_{j=1} (s-z_j)}{\prod\limits^n_{i=1} (s-p_i)} = -1$
  - 开环零点为 $z_j$，开环极点为 $p_i$，闭环极点为 $s_i$
  - **开环偶极子**：开环零点和极点的有序对 $(z_i,p_i)$
- **解的条件**：将复平面写成指数形式，即可得到解 $s$ 需要满足的两个条件：
  - **相角条件**：$\sum\limits^m_{j=1} \angle (s-z_j) - \sum\limits^n_{i=1} \angle(s-p_i) = (2k+1)\pi$
    - **应用**：绘制根轨迹
  - **幅值条件**：$K^* = \cfrac{\prod\limits^n_{i=1} |s-p_i| }{\prod\limits^m_{j=1} |s-z_j| }$
    - **应用**：求取根轨迹增益、闭环极点具体位置

### 绘制根轨迹

- **根轨迹的性质**
  - **连续性**：根轨迹是连续曲线
  - **对称性**：特征方程的根是实数，故根轨迹关于实轴对称
  - **分支性**：每个闭环极点都对应一条根轨迹，称为分支
    - 分支数等于闭环传函的阶数
  - **起始点**：
    - 起点是 $K^* = 0$，开环极点
    - 终点是 $K^* = \infty$，开环零点
      - $m$ 条根轨迹终止于有限点（有限零点）
      - $n-m$ 条根轨迹终止于无穷远点（无限零点）
  - **实轴特殊性**：实轴上区域 $r$，若右边开环实数零极点个数之和为奇数，则 $r$ 是根轨迹
    - **证明**：实轴左边的点相角始终为0，与共轭极点的相角和为 $2\pi$，与实轴右边的点相角和始终为 $-\pi$
      - 所以只有偶数个时，相角条件才不被满足
  - **渐近线**：
    - **渐近线与实轴交点**：$\sigma_a = \cfrac{\sum\limits^n_{i=1} p_i - \sum\limits^m_{j=1} z_j}{n-m}$
    - **渐近线与实轴夹角**：$\varphi_a = \cfrac{(2k+1)\pi}{n-m}$
  - **起始角和终止角**：
    - **起始角**：起始点的切线与实轴的夹角 $\t_{p_i}$
    - **终止角**：终止点的切线与正实轴的夹角 $\t_{z_i}$
  - **分离点**：
    - **几何意义**：根轨迹在复平面中相交后分离
    - **代数意义**：闭环特征方程出现重根
    - **求解**：设 $G(s)H(s) = K^*W(s)$，则重根满足 $\begin{cases} \dfrac{dW}{ds} = 0 \\ K^* = -\dfrac{1}{W(s)} \end{cases}$，求解即可
  - **会合角和分离角**：
    - **会合角**：进入分离点时与实轴正方向的夹角
    - **分离角**：离开分离点时与实轴正方向的夹角
  - **虚轴的交点**：临界稳定状态
    - **求解特征方程**
      - 劳斯判据法
      - 待定系数法，代入 $s = jw$，求出 $w，K^*$
  - **根之和与根之积**
    - $n-m \geq 2$ 时，$\sum\limits^n_{i=1} s_i = \sum\limits^n_{i=1} p_i$
      - **平衡性**：由于开环极点和不变，若部分闭环极点向右，则必有另一部分向左
    - $-\sum\limits^n_{i=1} s_i = a_1$
    - $(-1)^n\sum\limits^n_{i=1} s_i = a_n$
  
## 广义根轨迹

- **常规根轨迹**：$K^*$ 变化的根轨迹
- **广义根轨迹**：
  - **参数根轨迹**：负反馈系统中，非 $K^*$ 参数 $T_a$ 变化的根轨迹
    - **转化**：$Q(s) + T_aP(s) = 0$
      - 其中 $T_a$ 是参数，$Q(s),P(s)$ 是不含参首一多项式
      - 则 $G_a(s)H_a(s) = \cfrac{T_aP(s)}{Q(s)}$
  - **零度根轨迹**：存在开环右半零极点，或具有正反馈回路的根轨迹
    - **最小相位系统**：不存在开环右半零极点
    - **转化**：幅值条件相同，相角条件改为 $2k\pi$

### 零极点位置

- **稳定性能（闭环极点）**：
  - 闭环极点是否处于左半平面
  - 闭环极点到虚轴的距离
- **控制精度（稳态误差）**：
- **动态性能**：
  - **运动形式（实复数）**：实数极点是单调响应，复数极点是振荡响应
  - **动态平衡性（超调量）**：
    - 闭环主导极点的衰减率
    - 闭环零点的位置
  - **响应速度（闭环零极点）**：
    - 闭环零点的位置
    - 闭环极点到虚轴的距离

### 稳定条件

- **条件稳定系统**：$K^*$ 在一定的范围内才会稳定的系统

### 增减极点

- **增加开环零点 $z_1$**：使根轨迹发生趋向附加开环零点 $z_1$ 的方向变化，且渐近线倾角增大，系统变得稳定
  - 附加的开环零点离虚轴越近，吸引作用越强
- **增加开环极点 $p_1$**：增加根轨迹的分支数，使根轨迹发生背向附加开环极点 $p_1$ 的方向变化
  - 附加的开环极点离虚轴作用越近，排斥作用越强
- **增加开环偶极子 $(z_1,p_1)$**：对根轨迹的形状影响很小，但会增加开环放大系数
