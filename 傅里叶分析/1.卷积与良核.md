## 复数域的傅立叶级数

- **复周期函数的 $n$ 阶傅里叶系数**：$$\hat f(n) = a_n = \frac{1}{T}\int^T_0 f(y)e^{-\dfrac{2n\pi}{T}iy}dy$$
  - <font color='chartreuse'>周期函数的傅里叶变换是可数级数，故傅立叶系数的自变量为整数</font>
- **复周期函数的傅里叶级数**：$$f(x) \sim S(f) = \sum^\infty_{n = -\infty}a_n\cdot e^{\dfrac{2n\pi}{T}ix}$$
- **傅立叶系数求导性质**：$\hat f'(n) = in\hat f(n)$
- **$2\pi$ 周期函数的复单位圆表示法**：对于实轴上的 $2\pi$ 周期函数 $f$，设 $f(x) = F(e^{ix})$，则变为复单位圆上的函数 $F(e^{i\t})$
  - <font color='chartreuse'>这样一来，傅立叶公式就不必再考虑数分中的 $\sin$ 和 $\cos$，统一写为复指数形式
  - 傅立叶分析本质是单位圆上的调和分析，这样写更加直观一些</font>
  - **傅里叶级数的洛朗展式意义**：
    - 若设 $e^{ix} = z$，则 $f(x) = \sum c_n e^{inx} \LR F(z) = \sum c_nz^n$
      - <font color='chartreuse'>傅立叶级数实际上就是复的幂级数</font>
    - 由洛朗系数公式易得 $c_n = \dis\frac{1}{2\pi i}\int_T \frac{F(z)}{z^{n+1}}dz$

### 唯一性

- **（定理2.1）可积相同定理**：若 $f$ 的傅立叶系数均为 $0$，则 $f$ 在所有连续点上为 $0$
  - **证明（实值函数）**：
    - 反设 $f$ 是 $[-\pi,\pi]$ 上的周期实值函数，$f(0)>0$ 且为连续点
    - 如果能够构造三角多项式族 $p_k$，使得 $p_k(0)\to\infty$，那么显然就有 $\dis\lim_{k\to\infty} \int^\pi_{-\pi} p_k(\t)f(\t)d\t = \infty$，与傅立叶系数均为零矛盾
      - <font color='chartreuse'>下面是具体构造过程</font>
      - 由原点连续性，可取 $|\d|\leq \dfrac{\pi}{2}$ 使得 $[0,\d]$ 上总有 $f(\t) > \cfrac{f(0)}{2}$
        - <font color='chartreuse'>保证非负性，使得这一部分不干扰结果</font>
      - 取 $\e$ 使得 $p(\t) = \e + \cos\t$ 在 $[\d,\pi]$ 上恒有 $|p(\t)| < 1-\dfrac{\e}{2}$
        - <font color='chartreuse'>保证 $p$ 在大部分区域有界，从而是良定义的</font>
      - 再取 $\eta<\d$ 使得 $[0,\eta]$ 上总有 $|p(\t)|\geq 1+\dfrac{\e}{2}$
        - <font color='chartreuse'>保证存在一个无界区域</font>
      - 最后取 $p_k(\t) = \big[ p(\t) \big]^k$
    - 由题设得傅立叶系数均为 $0$，故 $\forall k$ 都有 $\dis\int^\pi_{-\pi}f(\t)p_k(\t)d\t = 0$
      - <font color='chartreuse'>下面还需要对极限情况进行证明</font>
      - 在 $|\t| < \eta$ 时，$p \geq 1+\frac{\e}{2}$
        - 由积分下界不等式，该区域积分 $\geq 2\eta\frac{f(0)}{2}(1+\frac{\e}{2})^k\to \infty$
      - $\d < |\t| < \pi$ 时，$p < 1 + \frac{\e}{2}$
        - 由积分上界不等式，该区域积分 $ \leq 2\pi B(1-\frac{\e}{2})^k \to 0$
      - 当 $\eta< |\t| < \d$ 时
        - 由函数非负性，该区域积分非负
      - 综上，$[-\pi,\pi]$ 上极限积分无穷大
  - **证明（复值函数）**：
    - 设 $f(\t)=  u(\t) + iv(\t)$，由于 $\wh{\overline{f}}(n) = \overline{\hat f(-n)}$，故 $u,v$ 的傅里叶系数均为零。由上面结论即得连续点均为零
- **（推论2.2）连续相同定理**：若 $f$ 连续且傅里叶系数均为 $0$，则 $f=0$

- **唯一性定理**：
  - 若仅改变可积函数的一个点，不足以改变其傅里叶级数
  - 改变连续函数的任意点都会使傅里叶级数不同

### 收敛性

- **收敛速度定理**：函数越光滑，傅立叶级数收敛速度越快
  - **证明**：只需要证明余项的傅立叶系数 $\to 0$ 的速度变快 
- **（推论2.3）一致收敛定理**：若 $f$ 连续且傅立叶部分和绝对收敛，则傅立叶级数一致收敛于 $f$
  <!-- - $\lim\limits_{N\to\infty} S_N(f)(\t) = f(\t)$（关于 $\t$ 一致） -->
  - **证明**：
    - 系数级数绝对收敛，则与其同阶的Fourier级数也绝对收敛
    - 再由连续性，得Fourier级数的极限函数 $g$ 连续
    - 由交换求积性，得 $f$ 与 $g$ 的Fourier系数相等
    - 将上个推论应用到 $f-g$ 上，则 $f=g$
- **（推论2.4）绝对一致收敛定理**：设 $f$ 二阶连续可微，若 $\hat f(n) = O\dkh{\cfrac{1}{|n|^2}}，|n|\to\infty$，则傅立叶级数绝对一致收敛于 $f$
  - **证明**：
    - 由傅里叶系数求导公式 + 积分上界不等式 + 绝对值不等式，$2\pi |n|^2\hat f(n) \leq C$
  - **（推论）绝对收敛定理**：
    - 连续可导的函数，傅立叶系数绝对收敛
    - 满足Holder条件的函数，傅立叶系数绝对收敛

## 傅里叶级数的卷积意义

### 核函数

- **$N$ 阶狄利克雷核**：$D_N(x) = \sum\limits^N_{n=-N} e^{inx}$
  - **Fourier系数**：$a_n = \begin{cases} 1,\quad |n|\leq N \\ 0,\quad 其它 \end{cases}$
    - <font color='chartreuse'>E-F公式 + 三角函数正交性易得</font>
  - **收敛形式**：$D_N(x) = \cfrac{\sin\Big[ (N+\frac{1}{2})x \Big]}{\sin(\frac{x}{2})}$
    - <font color='chartreuse'>显然定义式是等比数列，直接求和即可</font>
- **泊松核**：$P_r(\t) = \sum\limits^\infty_{n=-\infty} r^{|n|}e^{in\t}  \quad \t\in[-\pi,\pi]，r\in[0,1)$
  - **收敛形式**：$P_r(\t) = \cfrac{1-r^2}{1-2r\cos x + r^2}$
    - <font color='chartreuse'>显然可以分解为两个共轭的等比数列，分别求和后合并即可</font>
- **$N$ 阶费耶核**：$F_N(x) = \cfrac{\sum\limits^{N-1}_{i=0} D_i}{N}$
  - **收敛形式**：$F_N(x) = \cfrac{1}{N}\cfrac{\sin^2(\frac{Nx}{2})}{\sin^2(\frac{x}{2})}$
    - <font color='chartreuse'>复变函数三角级数</font>

### 卷积

- **卷积**：设 $f,g$ 是单位圆上复值可积函数，则 $(f*g)(x) = \dis\frac{1}{2\pi} \int^\pi_{-\pi} f(y)g(x-y)dy$ 称为它们的卷积
  - <font color='chartreuse'>卷积可以看作是一种加权平均，一个是权函数，一个是被平均函数
  - 卷是指翻转，积是指L1空间的内积</font>
- **傅立叶级数的卷积表示**：$S_N(f)(x) = (f*D_N)(x)$
  <!-- - 系数是积分（y），后面跟着三角函数的指数形式（x），线性合并得N阶Dirichlet核 -->
- **卷积的性质**
  - **对称性（交换律）**
    - **证明**：易得固定 $x$ 时，对任意函数都有 $\dis\int^T_{-T} F(x-y)dy = \int^T_{-T} F(y)dy$<font color='chartreuse'>（实际上，它就是$F$ 和 $1$ 的卷积）</font>，再取 $F(y) = f(y)g(x-y)$ 即可
  - **双线性**
    - **证明**：由定积分的线性即可
  - **结合律**
    - **证明**：累次积分 + 换元即可
  - **连续性**：
    - **传递引理**：$f$ 和 $g$ 均连续时，$f*g$ 连续
      - **证明**：由连续定义作差，绝对值不等式 + $g$ 连续 + 积分有界，导出 $\e$ 不等式
    - **（引理3.2）可积逼近定理**：设 $f$ 存在上界 $B$，则存在圆上的连续函数列 $\{f_k\}$ 满足
      - **L1范数逼近性**：$\dis\lim_{k\to\infty} \int^\pi_{-\pi} |f(x) - f_k(x)|dx =0$
        - <font color='chartreuse'>数值分析中称为一致逼近</font>
      - **一致有界性**：$\sup\limits_{x\in[-\pi,\pi]} |f_k(x)| \leq B，(\forall k\in N^+)$
      - **证明**：由实变函数的可测逼近定理即可
    - **证明**：
      - 由 $f,g$ 可积性，构造引理的逼近列 $f_k,g_k$
      - 三角不等式得 $|f*g - f_k*g_k| \leq |(f-f_k)*g| + |(g-g_k)*f_k|$
        - 第一项由积分绝对值不等式 + $g$ 一致有界性 + $f_k$ 逼近性，得收敛到0
        - 第二项由积分绝对值不等式 + $f_k$ 一致有界性 + $g_k$ 逼近性，得收敛到0
      - 综上 $f_k*g_k$ 一致收敛到 $f*g$。再由传递引理即得 $f*g$ 连续
  - **可积性**：
    - **证明**：由定积分的乘积可积性即可
  - **傅立叶卷积公式**：$\widehat{(f*g)}(n) = \hat f(n)\hat g(n)$
    - **证明**：
      - 左式展开，积分换序，卷积对称性传递到积分区间上，从而 $g(x-y)$ 换元为 $x$，从而分离x和y的积分，得到独立乘法，即右式
- **卷积的其它应用**：
  - 傅立叶：良核函数的卷积是一种加权平均
  - 概率论：分布函数求和
  - 信号处理：一串信号持续输入并不断衰减。其中f是（输入值）关于（输入时间点）的函数，g是（衰减后的值）关于（输入后经过时间）的函数，则T时刻的总信号存在值为 $(f*g)(T)$
  - 机器学习：图像的平滑处理，即将其与边缘值关联起来，那么应该进行某种加权平均。离当前点越远的矩阵元影响越小，即权重越小

## 良核

- **良核**：若函数族 $\{K_N(x)\}^\infty_{n=1}$ 满足下述条件，则称为良核
  - **归一性**：$\dis\frac{1}{2\pi}\int^\pi_{-\pi} K_N(x)dx = 1\quad (\forall N\in \N^+)$
    - <font color='chartreuse'>良核的总质量为 $1$，卷积过程中不会缩放原函数</font>
  - **一致有界性**：$\dis\int^\pi_{-\pi} |K_N(x)|dx \leq M$
    - <font color='chartreuse'>良核的绝对质量有限，不会出现反常爆炸情况</font>
  - **紧集中性**：$\dis\lim_{N\to\infty}\ \int_{\d\leq |x| \leq \pi} |K_N(x)| dx = 0 \quad (\forall \d>0)$
    - <font color='chartreuse'>良核的质量集中在原点处，极限为狄拉克函数</font>
- **（定理4.1）恒等逼近定理**：若 $\{K_N\}$ 是良核，则任意圆上函数 $f$ 在连续点处均满足 $\lim\limits_{N\to\infty} (f*K_N)(x) = f(x)$
  - <font color='chartreuse'>良核将 $y$ 的权重全部赋予到原点上，从而 $x-y=x$（详细解释见stein实变第三章）</font>
  - **证明**：由收敛定义作差 + 积分绝对值不等式 + 区间分离，发现良核极大处区间极小（良核积分有界），函数有界处良核为0，同时由函数连续性得到另一个 $\e$。
  - **推论**：处处连续时可导出一致的结果
  - **实例**：费耶核是良核
  - **反例**：狄利克雷核只满足归一性，不满足一致有界性和紧集中性
    - 若狄利克雷核是良核，则傅立叶级数总收敛。但可惜并不是，故傅立叶级数的点态收敛性是复杂的


## 级数的弱收敛性

### Cesaro求和

- **N阶切萨罗均值**：设 $s_n = \sum\limits^n_{k=0}c_k$ ，则 $\sigma_N = \cfrac{s_0+...+s_{N-1}}{N}$ 即为 $\{c_k\}$ 的 $N$ 阶切萨罗均值
- **切萨罗可加**：若切萨罗均值收敛，则称级数切萨罗可加
  - **推论**：切萨罗可加时，切萨罗均值收敛到原级数和 $s$
    - <font color='chartreuse'>数分知识即可</font>
- **费耶定理**：（费耶核卷积）是（狄利克雷核卷积）的切萨罗均值 $$\sigma_N(f)(x) = \frac{S_0(f)(x) + ... + S_{N-1}(f)(x)}{N} = (f*F_N)(x)$$
  - **（引理5.1）**：费耶核是良核
    - **证明**：
      - 已知 $F_N(x) = \cfrac{1}{N}\cfrac{\sin^2(\frac{Nx}{2})}{\sin^2(\frac{x}{2})}$
      - 由数分定积分章节知识易得归一性
      - 由非负性即得一致有界性
      - 再由于 $\d < |x| < \pi$ 时 $\sin^2(\frac{x}{2}) \geq c_\d$，得紧集中性
  - **（定理5.2）**：圆上可积函数，其任意连续点上的傅立叶部分和切萨罗可加
    - **证明**：恒等逼近定理即可
  - **（推论5.3）可积相等定理**：圆上可积函数，若傅立叶系数均为0，则连续点上均为0
    - **证明**：此时傅立叶级数为0，从而其切萨罗均值为0。再由切萨洛可加性即可
  - **（推论5.4）维尔斯特拉斯逼近定理**：圆上连续函数可被三角多项式一致逼近
    - **证明**：傅立叶级数的切萨罗部分和即为三角逼近

### Abel求和

- **Abel均值**：$A(r) = \sum\limits^\infty_{k=0} c_kr^{k}$
- **Abel可加**：若对任意 $r\in [0,1)$，阿贝尔均值收敛，且 $\lim\limits_{r\to 1} A(r) = s$，则称级数阿贝尔可加
  - <font color='chartreuse'>当 $r\to 1$ 时，收敛到原级数和</font>
- **强弱关系**：级数收敛 $\subseteq$ Cesaro可加 $\subseteq$ Abel可加
  - **反例**：
    - $\sum\limits^\infty_{k=0} (-1)^k$ 阿贝尔可加，切萨罗可加，但级数不收敛
    - $\sum\limits^\infty_{k=0} (-1)^k(k+1)$ 阿贝尔可加，但切萨罗不可加

### 泊松定理


- **函数的Abel均值**：设 $f$ 的傅立叶系数为 $a_n$，则 $A(r)(f)(\t) = \sum\limits^\infty_{n=-\infty} r^{|n|} a_ne^{in\t}$
- **泊松定理**：函数的阿贝尔均值就是泊松核卷积 $$A_r(f)(\t) = (f*P_r)(\t)$$
  - **（引理5.5）**：泊松核在 $r\to 1$ 时为良核
    - **证明**：
      - 已知 $P_r(\t) = \sum\limits^\infty_{n= -\infty} r^{|n|}e^{in\t} =  \cfrac{1-r^2}{1-2r\cos\t + r^2}$
      - 易得其一致收敛，从而可交换求积分
      - 分母化为 $(1-r)^2+2r(1-cos\t) \geq c_\d > 0（\d < |\t < \pi|）$，得到紧集中性
  - **（定理5.6）傅立叶级数阿贝尔可加定理**：连续点上的傅立叶级数阿贝尔可加
    - **证明**：Abel弱化性 + 可积相等定理
  - **（定理5.7）泊松积分的性质**：$u(r,\t) = (f*P_r)(\t)$ 满足
    - $u$ 存在两个连续导数，且全微分恒为0
      - **证明**：
    - 在连续点上 $\lim\limits_{r\to 1} u(r,\t) = f(\t)$
      - **证明**：
    - 若 $f$ 连续，则 $u$ 为满足上面两个条件的稳态热方程 $\frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial \t^2}$ 的唯一解
      - **证明**：
        - 设 $v(r,\t)$ 是满足稳态热方程，且收敛到 $f$ 的数
        - 代入稳态热方程，由指数导数的不变性，约去指数部分，得到Fourier系数的方程 $ a''_n(r) + \frac{1}{r}a'_n(r) - \frac{n^2}{r^2} a_n(r) = 0$
        - 这个方程的解为 $a_n(r) = A_nr^n + B_nr^{-n}$
          - 由Fourier系数与 $r$ 的有界性，$B_n = 0$
          - 由Fourier级数收敛性，$A_n$ 为 $f(\t)$ 的Fourier系数
          - 由连续函数Fourier级数唯一性，$u=v$
