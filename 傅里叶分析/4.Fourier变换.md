# Fourier变换（stein版本）

- Fourier变换是连续形式（积分形式）下的Fourier级数
- Fourier变换：$\hat{f}(\xi) = \int^\infty_{-\infty} f(x)e^{-2\pi ix\xi}dx$
  - 结果是Fourier系数
  - 通过取特值将Fourier系数解出来
  - $\xi$ 的数学意义是变换后的新变量（取代原来x的位置，成为新的函数）
  - $\xi$ 的数学本质是原先级数中的序号 $n$
- Fourier逆变换：$f(x) = \int^\infty_{-\infty} \hat{f}(\xi)e^{2\pi ix\xi}d\xi$
  - 结果是Fourier级数（原函数）
  - 把Fouier系数乘上三角函数，再累加成级数（这里将级数近似为积分形式）
  - $\xi$ 的数学意义是变换前的旧变量（即将被x取代位置，成为原来的函数）
  - $\xi$ 的数学本质是原先级数中的序号 $n$
- **实际意义**
  - **周期变换**：我们可以把非 $2\pi$ 周期的函数变为 $2\pi$ 周期，但这终究还是狭隘了。实际上，可以把任何函数（看作周期无穷大）都变换为 $2\pi$ 周期
  - **函数化简**：将任意函数化简成三角级数，具有统一形式和良好性质
  - **衡量平滑性**：Fourier系数衰减越快，原函数越平滑
  - **工科应用**
    - 微分积分线性化（一般用Laplace变换）

## Fourier变换

### 定义域

- **慢衰减性moderate decrease**：$f$ 是连续函数，且 $|f(x)| \leq \large\frac{A}{1+x^{1+\varepsilon}}$（本质是p级数比较，但log需要用到极限）
  - **慢衰减空间 $\mathcal{M}(R)$**：其可以成为向量空间
    - 线性、平移不变性、膨胀性、连续性
    - Fourier变换的定义域
- **快衰减性rapidly decrease**：$\forall k,\ell$，有 $\sup\limits_{x\in\mathbb{R}} |x|^k|f^{(\mathscr{\ell})}(x)| < \infty$
  - Schwarz空间成员的必要条件
- **Schwarz空间 $\mathcal{S}(R)$**：R上的所有无穷可微函数
  - **封闭性**：其对四则运算、微分运算、多项式因子、Fourier变换、卷积运算均封闭
  - 经典成员
    - 傅里叶高斯函数：$e^{-x^2}$
    - 有限支撑函数
- **Fourier变换性质（陈纪修）**
  - 平移性
  - 膨胀性（尺度性）
  - 微分积分线性化
- **（定理1.3）Fourier变换封闭性**：$f\in \mathcal{S}(R) \Rightarrow \hat{f}\in \mathcal{S}(R)$
  - **证明**：微分线性即可

### 高斯函数核

- **（定理1.4）特征向量**：$e^{-\pi x^2}$
  - **单位性**：$\int_R e^{-\pi x^2}dx = 1$
  - **特征性**：其Fourier变换和自身相等
  - **证明**：
    - 线性：老办法，二重积分 + 极坐标换元
    - 特征性：微分线性，变形 + 比较即可
- **（推论1.5）变换良核**：设 $K_\delta = \large\frac{1}{\sqrt{\delta}}e^{\frac{-\pi x^2}{\delta}}\normalsize(\delta>0)$，则 $\widehat{K_\delta}(\xi) = e^{-\pi\delta\xi^2}$
  - **证明**：换元 $\frac{x}{\sqrt{\delta}}$ 即可
  - 测不准原理：该函数和变换，在原点不能同时达到峰值或低谷
  - **（定理1.6）**：该集族为良核
    - **证明**：直接算
- **（推论1.7）良核特征性**：S空间内，$(f*K_\delta)(x)$ 一致收敛到 $f(x)$（$\delta\to 0$）
  - **证明**：S空间内，$f$ 一致连续，从而有界
    - $(f*K_\delta) - f(x) = \int_R K_\delta(t)|f(x-t)-f(x)|dt$
    - 应用三角不等式，再分离区间
      - 极小区间上，由 $f$ 连续性得 $\varepsilon$
      - 无界区间上，由良核极小性得 $\varepsilon$
    - （√）

## Fourier逆变换

- **（命题1.8）乘积公式**：S空间内，$\int_R f(x)\hat{g}(x)dx = \int_R \hat{f}(x)g(x)dx$
  - **证明**：$F(x,y) = f(x)g(y)e^{-2\pi ixy}$，其对于x和y均为慢衰减性，从而可累次积分。且 $\int F_1(x)dx = \int F_2(y)dy$
  - **本质**：就是积分而已……
- **（定理1.9）Fourier逆变换**：
  - **变换积分引理**：$f(0) = \int_R \hat f(\xi)d\xi$
    - 由乘积公式，$\int_R f(x)K_\delta(x)dx = \int_R \hat f(\xi)\widehat{K_\delta}(\xi)d\xi$
    - 再令 $\delta\to 0$，由良核性质即可
  - 令 $F(y) = f(y+x)$，则 $f(x) = F(0)$，即为Fourier逆变换
- **（推论1.10）**：Fourier变换是S空间上的双射
