# 集合代数

- 参考教材：程士宏（仅第一章）、folland（主要）

## 集合部分

### 集合代数（仅对单次集合运算封闭）

- **集合系**：集合的集合
  - $\pi$ **系**：交封闭的非空集合系
    - **真 $\pi$ 系**：$\{A，B，A\cap B\}$
  - **半环**：交封闭，（元素差可表为元素有限并）的非空集合系
    - **真半环**：
      - R上左开右闭区间 $\{(a,b]\}$
        - 差不封闭：$(a,b] - (c,d] = (a,c]\cup (d,b]$
      - $\{\{1\},...,\{n\},\{S_n\}\}$
      - 有限拓扑基：$X=\{1,2,3,4\}，\mathcal{B}=\{\{2\},\{3\},\{1,2,3\},\{2,3,4\}\}$
  - **环**：并和差封闭的非空集合系
    - **真环**：
      - R上左开右闭区间的有限并 $\mathop{\bigcup}\limits^\infty_{n=1}\{\mathop{\bigcup}\limits^n_{k=1} (a_k,b_k]\}$
        - 补不封闭
      - $0$ ~ $n$ 的所有数集构成的系 $\mathcal{P}(S_n)$，全集为 $N$，则其补不封闭
      - 整数上加减乘封闭的集合 $\{\{a\},\{b\}，\{a+b\}，\{a\cdot b\}\mid a,b\in Z\}$（实际上是 $Z$）
        - 全集为加减乘除封闭的集合 $\{\{a\},\{b\}，\{a+b\}，\{a\cdot b\}，\{a/b\}\mid a,b\in Z\}$（实际上是 $Q$）
        - 则其补不封闭
  - **半域**：交封闭，（元素补可表为元素有限并）的非空集合系
    - **真半域**：
  - **域**：交和补封闭的非空集合系（**域也称为代数**）
    - **真域（不是 $\lambda$ 系）**
      - 不含X和空集的域
      - R上的有理数域，单增极限为无理数，不封闭
      - R上所有区间
- **封闭定理**：域 $\subset$ 环 $\subset$ 半环 $\subset \pi$ 系
  - **元素分析证明**
  - **运算证明**：
    1. 交的补=并，补的交=差
    2. 有限个元素的并（半环）= 1个元素的并（环）
    3. 定义中半环已是 $\pi$ 系


### 可列代数（可列次运算封闭）

- **单调系**：（单增集列）都具有（可列并封闭性）
- **$\lambda$ 系**：具有总空间元素 $X$，包含差封闭性，单增列可列并封闭性
  - **等价命题**：含有X，补封闭性，不相交集列可列并封闭性
    - **证明**：X+差封闭=空集+补封闭，差封闭+单增可列并封闭 = 不相交可列并封闭
  - **实例**：拓扑空间中的开集和闭集
- **$\sigma$ 域**：具有总空间元素 $X$、补封闭性、可列并封闭性
  - **等价命题**：可测集所有运算性质
  - **实例**：Borel集、Lebesgue可测集、幂集 $\mc P(X)$、平凡域 $\{\varnothing,X\}$
    - **可数集系**：若 $X$ 不可数，则 $A = \set{E\subset X\mid E或E^c是可数集}$ 是 $\sigma$ 域
    - **半环并系**：半环上所有元素的有限并族是代数
      - **证明(folland)**：
  - **积子集传递性**：设 $X$ 是高维集合，$M_\a$ 是每个因子集合 $X_\a$ 上的 $\sigma$ 域，则
    - **张量 $\sigma$ 域**：$\otimes_\a M_\a = \set{\pi_\a^{-1}(E_\a) \mid E_\a\in M_\a}$ 是 $X$ 上的 $\sigma$ 域
    - 若维度可数，则 $\otimes_\a M_\a$ 是 $\set{\prod E_\a\mid E_\a\in M_\a}$ 生成的 $\sigma$ 域
  - **积生成传递性**：设 $M_\a$ 由 $\mc E_\a$ 生成，则
    - （$\ms E_\a$ 子集的逆投影）可生成 $\otimes_\a M_\a$
    - 若维度可数，则（$\ms E_\a$ 子集的直积）可生成 $\otimes_\a M_\a$
  - **张量 $\sigma$ 域是积集合的稠密子集**
- **单调定理**：$\sigma$ 域 $\subset \lambda$ 系 $\subset$ 单调系
  - **运算证明**：
    - 法1：由封闭定理，结论易得
    - 法2：极限封闭性而已，结论易得

#### 可列代数 $\leftrightarrow$ 集合代数

- **推论（进化到 $\lambda$ 系的方法）**：
  - **引理**：$\lambda$ 系和 $\pi$ 系不相包含
    - **证明**：易得 $\l$ 系不满足交封闭性
    - **实例**：$R$ 上所有（未完）
  - **添加可列性**：$\lambda$ 系 = 单调系 $\land$ 全集
- **推论（进化到 $\sigma$ 域的方法）**：
  - **添加可列性**：$\sigma$ 域 = 域 $\land$ 单调系
    - **证明**：
      - $\sigma$ 域是域：（可列并封闭）蕴含（有限并封闭）
      - 单调系 + 域 = 可列可加性
        - 取非单调集列 $A_n$ 的单增子列 $A_{n_k}$，其具有极限 $A$
        - 由于若 $A_{n1} \subset A_{n2}$，则 $A_{n1}\cup A_{n2} = A_{n2}$
        - 从而 $\mathop{\bigcup}\limits^\infty_{n=1} A_n = \mathop{\bigcup}\limits^\infty_{k=1} A_{n_k} = A$
    - **实例**：交并差补封闭，但单增极限不封闭
    - **本质**：单调极限 = 并/交集极限
  - **添加交封闭性**：$\sigma$ 域 = $\lambda$ 系 $\land\ \pi$ 系
    - **证明**：
      - 具有总空间 + 包含差封闭性 = 补封闭性
      - 交封闭性 + 补封闭性 = 并封闭性
      - 从而其形成域。其余证明同上
    - **实例**：
      - 拓扑空间中开集和闭集
        - $[a,b) = \mathop{\bigcup}\limits^\infty_{n=1} [a,b-\frac{1}{n}]$，可列并不封闭
        - 其为 $\lambda$ 系，不是 $\sigma$ 域。<font color='chartreuse'>添加交封闭性后，进化为Borel集</font>
  - **添加补封闭性**：$\sigma$ 域 = 具有总空间元素 $X$ 的 $\sigma$ 环
    - **$\sigma$ 环**：对可列并封闭的环
    - **证明同上**

### 集合系的生成

- **交集生成法**
  - 生成环 $r(\mathscr{S})$，生成单调系 $m(\mathscr{S})$，生成 $\lambda$ 系 $l(\mathscr{S})$，生成 $\sigma$ 域 $\sigma(\mathscr{S})$
- **半环生成定理**：半环 $\mathscr{Q}$ 生成的环 $r(\mathscr{Q}) = \mathop{\bigcup}\limits^\infty_{n=1}(\mathop{\bigcup}\limits^n_{k=1} A_k)$，其中 $A_k$ 两两不交
  - **互包证明**：
    - 环的并封闭性得 左>右
    - 环定义+生成环最小性得 右>左
  - **实例**：半环 $\{\{1\},\{2\},...,\{n\},\{S_n\}\}$ 生成的环 $\mathcal{P}(S_n)$
- **域生成定理**：$\mathscr{A}$ 是域，则 $\sigma(\mathscr{A}) = m(\mathscr{A})$
  - **互包证明**：
    - 域是单调系，生成单调系最小性得 左>右
    - 由域含全集 + 环进化条件 + 环定义得 右是域。再由生成域最小性得 右>左
  - **本质**：$\sigma$ 域进化定理
  - **实例**：
- **$\pi$ 系生成定理**：$\mathscr{P}$ 是 $\pi$ 系，则 $\sigma(\mathscr{P}) = l(\mathscr{P})$
  - **互包证明**：
    - $\sigma$ 域是 $\lambda$ 系，生成 $\lambda$ 系最小性得 左>右
    - 由 $\pi$ 系定义得 右是 $\pi$ 系，由 $\pi\to\sigma$ 进化条件得 右是 $\sigma$ 域，再由生成 $\sigma$ 域最小性得 右>左
