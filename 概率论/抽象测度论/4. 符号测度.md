# 符号测度

## 符号测度

- **符号测度**：$\nu: \mathcal{M}\to \overline{R}$，满足空集值为0、可列可加性<font color='chartreuse'>（因为无正性，故也无单调性）</font>
  - **单向无限性**：对确定的符号测度 $\nu$，正负无穷最多只能有一个出现在值域中
    - **证明**：反设存在 $\nu(A) = \infty，\nu(B) = -\infty$
      - 则由可列可加性，$\nu(A\cup B) = \nu(A) + \nu(B\j A) = \nu(B) + \nu(A\j B)$
      - 左项要求并集测度为正无穷，右项要求为负无穷，矛盾
  - **子集有限传递性**：设 $\nu$ 是 $(X,\mc M)$ 上符号测度
    - 若 $A,B\in \mc M，B\subset A$，且 $|\nu(A)|$ 有界
    - 则 $|\nu(B)|$ 也有界
  - **并集有限传递性**：若 $A_n$ 不交，且并集测度有界，则 $\sum\limits^\infty_{n=1} |\nu(A_n)| < \infty$
  - **极限封闭性**：单增集列满足 $\nu(\mathop{\bigcup}\limits^\infty_{j=1} E_j) = \lim\limits_{j\to\infty} \nu(E_j)$
  - **表示定理**：
    - **差定义**：两个测度的差是符号测度
    - **积分定义**：设 $f$ 是可测函数，且正负反常积分至少有一个收敛
      - 则 $\varphi(A) = \int_A fd\mu\quad (\forall A\in \mathcal{M})$ 是符号测度
      - **本质**：符号测度用于表示（可测函数在集合上值的变化量）
- **正集合**：符号测度非负的集合
- **负集合**：符号测度非正的集合
- **空集合**：符号测度为0的集合

### 正负分解定理

- **$\nu$ 的Hahn分解**：构成可测空间分割的一对正负集合 $P,N$
- **$\nu$ 的Jordan分解**：满足 $\nu = \nu^+ - \nu^-$ 的两个变差
  - **上/正变差**：$\nu^+(A) = \int_A f^+ d\mu$<font color='chartreuse'>（正变化量）</font>
  - **下/负变差**：$\nu^-(A) = \int_A f^- d\mu$<font color='chartreuse'>（负变化量）</font>
  - **全/绝对变差**：$|\nu| = \nu^+ + \nu^-$<font color='chartreuse'>（绝对变化量）</font>
  - **推论**：三者都是符号测度
- **内核测度（半有限测度）**：$\nu^*(A) = \sup\set{\nu(A)\mid B\subset A，B\in \mc M}$
  - **本质**：子集的最大测度值
  - **非限制性**：$\nu^*(A) \geq \nu(A)$
    - 当 $A$ 是负集合时，其子集可能有正集合，此时 $\nu^*(A) > 0 > \nu(A)$）
    - **本质**：用来分析集合中测度是否为单号变差（变差均为正或负）
  - **非负性 + 单调性**
  - **空集为0性**
- **绝对等测核引理**：设 $A$ 是有界符号可测集
  - 则 $\forall \e>0$，存在符号可测集 $A_\e \subset A$ 满足
    - $\nu(A_\e)\geq 0$
    - $\nu^*(A\j A_\e) \leq \e$
  - **证明**：反设所有 $\nu(A_0)\geq 0$，都是 $\nu^*(A\j A_0) > \e_0$
    - 由可列可加性，可直接设最极端的情况
    - 不妨取 $A_0 = \varnothing$，则变为 $\nu^*(A)>\e_0$，由定义得存在可测子集 $\nu^*(B_1)>\e_0$
    - 再对 $A\j B_1$ 应用，得 $B_2$。不断进行下去，得到不相交列 $\{B_n\}$
    - 设 $B = \mathop{\bigcup}\limits^\infty_{n=1} B_n$，则 $B\subset A$ 且 $\nu(B) = \sum\limits^\infty_{n=1} \nu(B_n) > \infty\cdot\e_0$，与 $A$ 有界性矛盾
  - **理解**：就是不断切掉极小集合（为了方便取为不相交），但是由假设，每个极小集合测度有界，故取无穷个极小集合后，原集合有界，但测度被削为0，矛盾
  - **本质**：
    - 当 $A$ 是正集合时，具有绝对等测性（差集中不存在正负变差抵消的情况）
    - 当 $A$ 是负集合时，内部一定存在非负集合（没用，因为肯定存在空集）
  - **推论**：当 $A$ 是负集合时，绝对等测核应为
    - $\nu(A_\e) \leq 0$
    - $\nu^*(A\j A_\e) \geq -\e$
- **纯负核引理**：任何真负集合 $A$，都存在符号可测集 $A_0\subset A$ 满足
  - $\nu(A_0) < 0$
  - $\nu^*(A_0) = 0$
  - **证明**：由正等测核引理，存在正等测核 $C_1$
    - 此时 $\nu(A\j C_1) = \nu^*(A\j C_1) \leq \e_1$
      - 在 $A\j C_1$ 中选 $C_2$，不断进行下去
    - 设 $C = \mathop{\bigcup}\limits^\infty_{n=1} C_n$，则 $\nu(C) = \sum\limits^\infty_{n=1} \nu(C_n) \geq 0$，且 $0\leq \nu^*(A\j C) \leq \e_n\to 0$
    - 再由可列可加性，此时 $A\j C$ 就是纯负核
  - **理解**：正等测核的补定理，不断删去内部的正集，就只剩下纯负集合
  - **本质（纯负集合）**：$\nu^*(A) < 0$（内部只有负子集）
- **Hahn分解定理**：任何符号测度都存在唯一的Hahn分割
  - 且 $\nu(A\cap P) \geq 0 \geq \nu(A\cap N)$
  - **证明（程）**：
    - $\mc M^- = \set{A\in\mc M\mid \nu^*(A) = 0}$ 是 $\sigma$ 环
      - <font color='chartreuse'>内核零测集（纯负集合族）是完备的</font>
    - **存在性**
      - 设 $\a= \inf\{\nu(A)\mid A\in \mc M^-\}$，由于 $\varnothing\in \mc M^-$，由下确界性得 $\a \leq 0$
      - 取纯负集族中的下确界逼近列 $\{A_n\}\in\mc M^-$，设 $N = \mathop{\bigcup}\limits^\infty_{n=1} A_n$
        - 由 $\sigma$ 环封闭性得可测性
        - 非限制性 + 内核单调性得 $\nu(A\cap N) \leq \nu^*(A\cap N) \leq \nu^*(N) = 0$
      - 设 $P = X\j N$
        - 由 $\sigma$ 环封闭性得可测性
      - 反设存在 $P$ 的子集 $\nu(A) < 0$
        - 由纯负核引理，存在真负集合 $A_0\subset A$ 满足 $\nu(A_0\cup N) = \nu(A_0) + \nu(N) < \nu(N)$
        - 由非限制性 + $\{A_n\}$ 内核零测性得 $\nu(N) \leq \nu(A_n) + \nu^*(N\j A_n) = \nu(A_n)$
          - 两边取极限即得 $\nu(N) \leq \a$
        - 综上得 $\nu(A_0\cup N) < \a$，与下确界性矛盾
      - **理解**：很简单，就是若 $P$ 存在负子集，则其内部存在纯负核，将核并入 $N$ 不会改变 $N$ 的内核零测性（纯负集合全集性），但会降低 $\mc M^-$ 原先的测度下确界。由于下确界是一个确定的值，从而类似数论的无穷递降法，所有 $P$ 中纯负核会自动向 $N$ 中跌落，直到 $P$ 中不存在任何负子集
    - **唯一性**
  - **证明(folland)**：
    - 设 $m$ 是正集合测度上确界，则存在集列 $\nu(P_j)\to m$，由可列可加性，$\nu(\mathop{\bigcup}\limits^\infty_{j=1} P_j) = m$
    - 只需 $N = X\j P$ 是负集合即可
      - 易得 $N$ 不包含任何非空正集合
      - 设 $\{A_j\}$ 是 $N$ 中集列
        - 正数列 $\{n_j\}$ 表示最小的存在 $B\subset A_{j-1}$ 满足 $\nu(B)>\nu(A_{j-1}) + \dfrac{1}{n_j}$ 的序号
        - 由 $N$ 负集合性，$n_1$ 是存在的
      - 设 $A = \mathop{\bigcap}\limits^\infty_{j=1} A_j$，则由极限封闭性，$\infty > \nu(A) > \sum\limits^\infty_{j=1} \dfrac{1}{n_j}$，故 $n_j$ 是无穷大量
        - 再由于必定 $\exists B,n$ 使得 $B\subset A，\nu(B) > \nu(A)+\dfrac{1}{n}$
        - 而当 $n_j\to\infty$ 时，必定有 $n<n_j$，$B\subset A_{j-1}$，与最小性矛盾
    - **唯一性**：反设不唯一，则 $(P\j P')\subset P$ 且 $(P\j P')\subset N$，只能是空集合
    - **理解**：
- **Jordan分解定理**：任何符号测度都存在唯一的Jordan分解
  - 且 $\nu^+ = \nu^*，\nu^- = (-\nu)^*$
  - **证明**：
    - **存在性**：取 $\nu^+(A) = \nu(A\cap P)$，$\nu^-(A) = \nu(A\cap N)$ 即可
    - **正负变差性**：
      - 单调性得 $\forall B\subset A，\nu(B)\leq \nu^+(B)\leq \nu^+(A)$，从而 $\nu^+(A) \geq \nu^*(A)$
      - 再由 $\nu^+(A) = \nu(A\cap P) \leq \sup\{\nu(B)\mid B\subset A，B\in \mc M\} = \nu^*(A)$
      - 故第一式成立，同理可证第二式
  - **理解**：Jordan分解就是将符号测度（可测函数在集合上的总变化）分离为正负变差（可测函数在集合上的正增长和负减少）
- **性质**：
  - $\nu = \int_E (\chi_P-\chi_N)d|\nu|$
  - **理解**：
    - 总变差 = 正变差 - 负变差
    - 绝对变差 = 正变差 + 负变差
  - $L^1(\nu) = L^1(\nu^+) \cap L^1(\nu^-)$
  - **理解**：总可测 = 正可测且负可测
  - $\dis\int fd\nu = \int fd\nu^+ - \int fd\nu^- \quad (f\in L^1(\nu))$
  - **理解**：第一式的积分写法

#### 小总结

- 给出标准的正负集合分解方法（Hahn分解），才能更好地分离正负变差（Jordan分解）来研究有界变差，从而研究可微性

## 微分理论

### 绝对连续性（任意变差均有界）

- **符号测度 $\nu$ 关于测度 $\mu$ 绝对连续**：若 $\forall\mu(E) = 0$，都有 $\nu(E) = 0$，则 $\nu \ll \mu$
  - **本质**：
    - 类似泛函的更强范数
    - 实分析中，只要区间划分足够小，则总变差有界（从而可导）
- **等价命题**：设 $(X,\mc M)$ 上 $\nu$ 是有限符号测度，$\mu$ 是正测度
  - 则 $\nu \ll \mu \LR \forall \e > 0，\exists \d > 0$，$\forall \mu(E)<\d$ 都有 $|\nu(E)| < \e$
  - **证明**：
    - **引理**：易得 $\nu \ll \mu \LR |\nu|\ll \mu$ 且 $|\nu(E)|\leq |\nu|(E)$，不妨设 $\nu$ 是正测度
      - **理解**：总变差=0时，可能是符号测度为0，或是正负抵消。对应上面三个条件
    - **充分性**：同引理，都易得
    - **必要性**：反设 $\forall n，\exists E_n\in\mc M$，满足 $\mu(E_n) < 2^{-n}，\nu(E_n)\geq\e$
      - 设 $F_k = \mathop{\bigcup}\limits^\infty_{n=k} E_n$，$F = \mathop{\bigcap}\limits^\infty_{k=1} F_k$，<font color='chartreuse'>即 $F$ 是 $E_n$ 的上极限</font>
        - 则 $\mu(F_k) < \sum\limits^\infty_{n=k}\dfrac{1}{2^n} = 2^{1-k}$，取极限得 $\mu(F) = 0$
      - 但是由假设，$\nu(F_k)\geq \e$，则 $\nu(F) = \lim\limits_{k\to\infty} F_k \geq \e$，与绝对连续矛盾
    - **理解**：取上极限，$E_n$ 极小（$\mu$ 测度），但其上值的变化（$\nu$ 测度）有界，与绝对连续矛盾
  - **本质**：将符号测度看作函数，即变为绝对连续的 $\e-\d$ 语言
  - **推论**：若 $f\in L^1(\mu)$，则 $\forall \e > 0$，都存在 $\d>0$ 使得 $\forall \mu(E)<\d$ 都有 $|\int_E fd\mu| < \e$
    - **理解**：就是符号测度的积分形式

### R-N导数

- **导数性质**：设 $(X,\mc M)$ 中，$\nu$ 是 $\sigma$ 有限符号测度，$\mu,\l$ 是 $\sigma$ 有限测度，且 $\nu \ll \mu \ll \l$
  - 则
    - **绝对连续传递**：$\nu \ll \l$
    - **链式法则**：且 $\dfrac{d\nu}{d\l} = \dfrac{d\nu}{d\mu}\dfrac{d\mu}{d\l}\ae \l$
    - **可积传递**：若 $g\in L^1(\nu)$，则 $g(\dfrac{d\nu}{d\mu})\in L^1(\mu)$，且 $\int gd\mu = \int g\dfrac{d\nu}{d\mu}d\mu$
  - **证明**：
    - 由于可Jordan分解，不妨设 $\nu \geq 0$，则 $g = \chi_E$ 时符合式子，从而由逼近定理，简单函数符合式子。由单调收敛定理，非负函数符合式子，最后由线性得一般函数符合式子
    - 设 $g = \chi_E(\dfrac{d\nu}{d\mu})$，则 $\nu(E) = \int_E \dfrac{d\nu}{d\mu}d\mu = \int_E \dfrac{d\nu}{d\mu}\dfrac{d\mu}{d\l}d\l$ 总成立，从而由a.e.的定义，得链式法则成立
  - **推论**：
    - 若 $\mu\ll\l\ll\mu$，则 $\dfrac{d\l}{d\mu}\dfrac{d\mu}{d\l} = 1 \ae \l或\mu$
  - **反例**：设 $\mu$ 是勒贝格测度，$\nu$ 是 $(\R,\mc B_\R)$ 的原点，易得 $\nu\perp\mu$，此时 $\dfrac{d\nu}{d\mu}$ 不是R-N导数，称为狄拉克 $\d$ 函数
- 若 $\mu_1,...,\mu_n$ 是 $(X,\mc M)$ 中测度，则存在测度 $\mu$ 满足 $\forall \mu_j\ll\mu$（也即 $\mu = \sum\limits^n_{i=1}\mu_j$）
