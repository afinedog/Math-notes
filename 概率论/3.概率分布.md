# 概率分布

## 赋值样本点

- **随机变量**：无法确定取值的变量（相当于赋值（编号）的样本点）
  - 随机变量是样本空间上的实值函数，$X:\Omega \to R，\omega \to X(\omega)$
  - 随机变量的取值（包含于一个数集）是一个事件，可以定义概率
- **概率范围作差性**
  - $\{a<X\leq b\} = \{X\leq b\} - \{X\leq a\}$
- **概率分布函数 $F(x)$**：$F(x)\equiv P(X\leq x)$
  - **性质（基本上是上限函数的性质）**
    - 单调不减性：包含关系 + 概率的非负性
    - 右连续性：用概率的连续性 + 单调递减 $\delta$ 子列 $\frac{1}{n}$，化为无限交
      - 因为 $F(x)$ 在x点是有意义的。x左边是闭集，x右边是开集，此时左边逼近x有意义（闭-开=点），右边逼近x无意义（开-开=无意义）
      - 右极限一定跨越点，左极限可以是逼近点（如果定义用开集，则相反）
    - 左极限开集性：用概率的连续性 + 单调递增 $\delta$ 子列 $\frac{1}{n}$，化为无限并
      - 端点的跃度性：$P(X=x) = F(x)-F(x-)$
        - 连续 $\Leftrightarrow$ 端点为0
        - 离散型随机变量的函数为阶梯函数，不连续
    - 正极限为1，负极限为0
  - **应用**：证明是否存在概率密度
- **概率密度 $f(x)$**：$F(x) = \int^x_{-\infty}f(s)ds$
  - 非连续分布函数无概率密度
  - **导数性质**：分布函数连续且有限区间内只有有限点无连续导数，则 $f(x) = \begin{cases} F'(x) ，F'(x)存在 \\ 0\qquad ，其他 \end{cases}$
    - **证明**：分区间后加和
  - **积分性质**：$P(x\in A) = \int_A f(s)ds$
    - **证明**：N-L公式 + 积分区间选择

## 离散型随机变量

- **离散型随机变量**：只取有限个值或可列个值的随机变量
- **概率分布**：随机变量在 $X_k$ 中取值，则 $p_k = P(X=x_k)，(k\geq 1)$ 为X的概率分布
- **概率分布列**：$\{p_k\}$ 为概率分布列
  - 非零性、互不相容性、和1性
- **伯努利分布（Bernoulli）**：$X\sim\mathcal{B}(1,p)：\begin{cases} P(X=1)=p \\ P(X=0)=1-p \end{cases}$
  - **示性函数（特征函数）**：$I[A] = \begin{cases} 1，A发生 \\ 0，A不发生 \end{cases}$
  - 任何事件的示性函数均服从伯努利分布
- **二项分布（Binomial）**：$X\sim\mathcal{B}(n,p)：P(X=k) = C^k_np^k(1-p)^{n-k}$
  - 二项独立重复试验，成功k次的概率
- **泊松分布**：$X\sim \mathcal{P}(\lambda)：P(X=k) = \dfrac{\lambda^k}{k!}e^{-\lambda}（\lambda = \mu t）$
  - **极限性**：泊松分布是二项分布的极限，其中 $n\to \infty，p\to 0$
    - **证明**：
      - 把时间 $t$ 分为 $n$ 段，每段发生的概率为 $p_n = \mu\cfrac{t}{n}$，用二项分布求极限得 $$P(X=k) = \lim\limits_{n\to\infty}C^k_n p^k_n(1-p_n)^{n-k} = \cfrac{(\mu t)^k}{k!}e^{-\mu t}\lim\limits_{n\to\infty} \cfrac{n\cdots (n-k+1)}{n^k(1-\frac{\mu t}{n})^k}$$
      - 易得右侧极限值为 $1$，故 $P(x=k) = \cfrac{(\mu t)^k}{k!}e^{-\mu t}$，即 $X\sim\mathcal{P}(\mu t)  = \mathcal{P}(np_n)$
  - **应用**：
    - 二项分布在 $n\to\infty$ 次尝试中成功 $k$ 次的概率
      - 同一时间内 $n$ 个地区的试验
      - 同时位置上 $n$ 个时间段的试验
    - $n$ 个独立时间段中第一次成功的时间为 $k$ 的概率
      - 时间间隔 $T$ 内成功概率为 $F$，时间段数量 $N(t)$，每个时间段相互独立
    - **应用性质**：
      - 平稳性：$[t_0,t_0+t)$ 中的呼叫数量只与t有关，与 $t_0$ 无关
      - 无记忆性
      - 普通性：时间间隔充分小时，同时成功两次的概率为0
      - 再生性
- **超几何分布**：$X\sim H(N,M,n)：P(X=m) = \cfrac{C^m_MC^{n-m}_{N-M}}{C^n_N}$
  - 和1性：$(1+x)^N = (1+x)^M(1+x)^{N-M}$，两边同项因式相等
  - 极限情况：
    - $N\to\infty$，每次选到的概率 $p_N = \frac{M}{N}$ 不能为0或1，则 $M\to\infty、N-M\to\infty$
      - $n、m$ 有界
    - $\frac{C^m_M}{M^m} \to \frac{1}{m!}$，……
    - $P(X=m) = \frac{n!}{m!(n-m)!}p^m(1-p)^{n-m}$
  - **近似公式**：$P(X=m) \approx C^m_np^m_N(1-p_N)^{n-m}$（二项分布）
  - **应用**：N个中有M个次品，无放回抽取n个，找到m个的概率
- **几何分布**：$X\sim G(p)：P(X=k) = (1-p)^kp$
  - **应用**：
    - 独立重复试验中，在第 $k$ 个事件首次成功的概率
    - $P(X>k) = (1-p)^k$
      - **证明**：等比数列求和公式
  - **无记忆性**：$P(X=k+1|X>k) = P(X=1)$
  - **无记忆性定理**：服从几何分布 $\LR$ X存在无记忆性
    - **证明**：
      - 必要性直接计算即可
      - 充分性用归纳法
    - **理解**：无记忆性的分布函数，为了大于k和等于k都具有相似形式，必须要用等比数列。而 $(1-p)^k$ 在等比数列中很有用
- **帕斯卡分布**：$X\sim \mathcal{PA}(r,p)：P(X=k) = C^{r-1}_{k-1}(1-p)^{k-r}p^r，k=r,r+1,...$
  - **应用**：独立重复试验中，成功r次时，总共进行k次的概率
    - 固定最后一次成功不动，前面全排列
- **负二项分布**：$X\sim \mathcal{NB}(r,p)：P(X=k) = C^{r-1}_{k+r-1}(1-p)^kp^r，k=1,2,...$
  - **应用**：
    - 独立重复试验中，成功r次时，失败k次的概率
      - 固定最后一次成功不动，前面全排列（换了个序号而已）
    - 赌资分配问题：先胜m局则有全部赌资，若甲胜概率为p，实际胜k，乙胜概率为1-p，实际胜l，求赌资分配
      - 假设继续进行，甲赢的概率：乙赢的概率进行分配
      - $P(甲) = \sum\limits^{m-l-1}_{i=0}C^{m-k-1}_{m-k-1+i} p^{m-k}(1-p)^i$
        - 从甲获胜时乙赢 $0、1、2...、m-l-1$ 局这样一直累加

### 随机游动

## 连续型随机变量

- **连续型随机变量**：存在概率密度 $f(x)$ 的随机变量
  - 和1性：$\int^{+\infty}_{-\infty} f(x)dx = 1$
  - 端点为0性：$P(x=a) = 0$
    - 概率密度只有导数意义，没有点点意义，求概率需要积分
  - 概率的积分性质
  - 概率分布函数的连续性
- **均匀分布**：$X\sim\mathcal{U}(a,b)：f(x) = \begin{cases} \frac{1}{b-a}，x\in(a,b) \\ 0，\quad x\not\in(a,b) \end{cases}$
  - 示性函数表达：$f(x) = \frac{1}{b-a}I_{(a,b)}$
  - **应用**：Poisson分布中，总共只有一个成功，则其均匀分布（直接计算即可）
- **指数分布**：$X\sim \mathcal{E}(\lambda)：f(x) = \begin{cases} \lambda e^{-\lambda x}，x\geq 0 \\ 0\qquad， x<0 \end{cases}（\lambda > 0）$
  - 非负随机变量
  - **无记忆性定理**：服从指数分布 $\LR X$ 具有无记忆性
    - **证明**：
    - 必要性易得
    - 充分性：条件概率的乘除性质 + 指数函数的乘除线性
      - $P(X>t) = P(X>t+s\mid X>s) = \frac{P(X>t+s)}{P(X>s)}$，满足乘除线性。它是指数函数
        - **线性**：若 $f(x+y)=f(x)+f(y)$，则 $f'(x) = \frac{f(\triangle x)}{\triangle x} = a$，积分即可
        - **指数**：若 $f(x+y)=f(x)f(y)$，则 $f'(x) = f(x)·\frac{f(\triangle x)-1}{\triangle x} = \lambda f(x)$，解常微分方程即可
        - **理解**：类似利用线素求函数的思想
      - 再因为 $\lambda$ 是自由参数，所以可以选取底为 $e$
    - **理解**：无记忆性，即限制起不到作用，即限制域和限制域内的概率是独立的，因此满足乘法公式
  - **应用**：
    - Poisson分布第一次成功的时间为 $k$ 的概率
      - $X\sim \mathcal{P}$ 中，$X=1$ 时，概率是指数形式
      - Poisson分布的处处独立性（无记忆性）
- **正态分布**：$X\sim N(\mu,\sigma^2)：f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}exp(-\frac{(x-\mu)^2}{2\sigma^2})$
  - **标准正态分布密度**：$\mu=0，\sigma=1，\varphi(x)=\frac{1}{\sqrt{2\pi}}e^{-\frac{x^2}{2}}$
  - **标准正态分布函数**：$\varPhi(x)$
  - **正态分布函数的标准化**：$P(X\leq a) = \varPhi(\frac{a-\mu}{\sigma})$（积分换元）
  - 对称性（平方性）、中心极值性
- **Gamma分布**：$X\sim\Gamma(\alpha,\beta)：f(x) = \begin{cases} \frac{\beta^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x}，x\geq 0 \\ 0\qquad\qquad\quad ， x<0 \end{cases}$
  - **应用**：
    - Poisson分布中，第m次成功的时间为k的概率
    - **证明**：
      - $F(t) = P(T\leq t|N(T) = k) \\ = P(N(t) \geq k) = 1-\sum\limits^{k-1}_{j=0}P(N(t)=j)$
      - 求导，得到错位相减，最终结果为 $\frac{(\mu t)^{k-1}}{(k-1)!}\mu e^{-\mu t}$

## 随机变量映射后的分布

- **微分法定理**：$P(X\in D) = 1$（D是开集），$g(x)$ 在D内连续
  - 若 $P(X=x) = g(x)dx$，则 $g(x)$ 是概率密度
  - **证明**：本质是证明存在概率密度。
    - $P(x)$ 在D外为0，则 $F(x)$ 在D外为常函数，从而可导
    - D内有连续导数
    - 因此存在概率密度。
  - **推论（链式法则）**：$P(X = h(y)) = f(h(y))\cdot |h'(y)|dy$
  - **应用**：微分法求随机变量映射后的概率密度
    - 正态分布标准化映射：$Y = \frac{X-\mu}{\sigma}$
  - **拓展**：分段单调可微情况
- **威布尔分布**：
  - $Y = (\frac{X}{a})^\frac{1}{b}，X\sim\mathcal{E}(1)$
  - $Y\sim W(a,b)：P(Y=y) = f(y) = aby^{b-1}exp(-ay^b)，y>0$
  - **应用**：
    - **可靠性**：使用寿命大于某个固定值的概率
    - 产品寿命超过y的概率
- **对数正态分布**：$Y=e^{N(\mu,\sigma^2)}\sim Ln(\mu,\sigma^2)$
- **离散型的映射**：$P(Y=k) = P(X=h(k))$（不需要导数，直接代入）

### 小总结

- 为什么要用 $f(h(y))$ 的形式
  - 映射语言：因为每一点的概率密度仅仅依赖于该点。而我们已知的是x的分布。当x被映射到y后，要总结的是y的分布。但y的分布还得从x推出。所以数学意义应当是x，而数学本质应当是y。所以采用x的随机变量，但是自变量是y，通过映射 $x = h(y)$ 打通桥梁
  - 确实，它像是当初的 $f(x)$ 一样，解释只会显得啰嗦
- 或者用方程法解释：
  - $Y = (\frac{X}{a})^\frac{1}{b}，X\sim\mathcal{E}(1)$，左右两边求导相等，变换得结果

## 习题

- 求概率密度
  - 定义法（分布函数法）：首先利用条件得到概率公式，从而求出分布函数，然后求导
  - 微分法（映射法）：已知映射关系，和一个函数的概率密度
    - 先取定开区域，证明可以使用微分法
- 模型的多种视角
  - 同时空问题（超几何分布）
    - 排列视角
    - 选择视角
  - 卫星分布问题
    - 定卫星动军事：取24小时为间隔，军事行动起始时间从0到24
    - 定军事动卫星：卫星是均匀分布 $(0,24)$，军事时间 $(a,a+6)$，则 $P(\xi\notin (a,a+6)) = 1-\int^{a+6}_a \frac{1}{24}dx$
- 最大最小函数：
  - $max(X,Y) = \frac{(X+Y)+|X-Y|}{2}$
  - $min(X,Y) = \frac{(X+Y)-|X-Y|}{2}$
  - $P(a<min(X,Y)\leqslant b) = P^2(X>a) - P^2(X>b)$
- 线性单元函数：**离散卷积公式**
  - 离散 + 连续：$Z=X+Y$，二项分布X和指数分布Y（独立）
    - $P(Z = z) = \sum\limits^n_{k=1} P(X=k,Y=z-k)（k\in [0,n]\cap N）$
    - 独立性得 $f(z) = \sum\limits^n_{k=1} C^k_n p^k(1-p)^{n-k} · \lambda e^{\lambda(z-k)}$
  - 连续 + 连续：$Z=X+Y$
    - $P(Z=z) = \int^z_0 dx\int^{z-x}_0 f(x,y)dy$
- 非线性单元函数：
  - 二元函数单重积分的变量代换公式：$I(y) = \int f(x,y)dx$，若 $z = g(y)$，则 $I(z) = \int f(x,z)[g^{-1}(z)]'dx$（焯，这不就是含参变量积分的换元公式？看来是我学艺不精）
