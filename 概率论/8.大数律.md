# 大数律和中心极限定理

## 弱大数律（不一定成立）

- **弱大数律**：$X_n$ 为随机变量序列， $\large\frac{S_n - a_n}{b_n}\normalsize \overset{p}{\to} 0$
  - <font color='chartreuse'>本质：收敛</font>
  - **中心化数列**：实数列 $a_n$
  - **正则化数列**：正数列 $b_n$
  - **标准形式**：$\forall X_i\in L_1$ ，$a_n = ES_n，b_n = n$
    - **本质**：一阶中心矩收敛
- **Markov弱大数律**：若 $\lim\limits_{n\to\infty} \large\frac{Var(S_n)}{n^2} \normalsize = 0$，则成立标准形式的弱大数律
  - **证明**：广义M不等式中，$g(x) = x^2$
  - **本质**：只要方差总和小于 $n^2$，就
- **切比雪夫弱大数律**：$\forall X_i$ 不相关且方差有界，则成立标准形式的弱大数律
  - **证明**：方差独立线性公式 + M弱大数律
- **伯努利弱大数律**：n重Bernoulli试验的成功次数设为 $Z_n$，则 $\frac{Z_n}{n}\xrightarrow{p} p$
  - **证明**：Markov不等式的直接推论

### 特征函数

- **连续性推论**：弱大数律成立 $\Leftrightarrow \lim\limits_{n\to\infty} Ee^{it\frac{S_n-a_n}{b_n}} = 1$（特征函数列极限为1）
  - **证明**： p到d，再由连续性定理得结论
- **Khinchin弱大数律**：独立同分布的 $X、\{X_n\}$，有 $\frac{S_n}{n}\xrightarrow{p} EX$
  - **证明（特征函数法）**：
    - 独立同分布 + 指数线性 + 特征函数Taylor展开式，得
    - $\phi_n(t) = (1+\omicron(\frac{t}{n}))^n\to 1$，从而收敛
  - **推论（何书元弱大数律）**：$\dfrac{1}{n}\sum\limits^n_{i=1}(X_i-\mu_i)\xrightarrow{p} 0$
    - **证明**：由独立同分布，$\sum\limits^n_{i=1} (EX-EX_i)\xrightarrow{p} 0$

### 应用

- **Monte Carlo（蒙特卡洛法）**：可测绝对可积函数 $f(x)，x\in [0,1]$
  - $U_n\in U[0,1]$ 是独立随机变量列
  - $I_n = \frac{1}{n}\sum\limits^n_{k=1} f(U_k)$
  - 则 $I_n\xrightarrow{p} \int^1_0 f(x)dx$
- **Slutsky引理**：若 $X_n\xrightarrow{d} X，Y_n\xrightarrow{p} 0，W_n\xrightarrow{p} 1$，则 $W_nX_n + Y_n\xrightarrow{d} X$
  - **证明**：夹逼
    - 设 $x\in C(P(X\leqslant x))$，再取 $\varepsilon_1$ 使得 $(1\pm\varepsilon_1)x \in C(P(X\leqslant x))$
    - 证明两个不等式：
    - $P(W_nX_n \leqslant x) \leq P(X_n\leqslant 1+\varepsilon_1 x) + P(|W_n-1|\leqslant \varepsilon_1)$
      - 全部取补集，则右端化为 $P(X_n > 1+\varepsilon_1 x，W_n\in (1-\varepsilon_1,1+\varepsilon_1))$
      - 画图证明比较直观一些
    - $P(X_n\leqslant 1-\varepsilon_1 x) \leq P(W_nX_n\leqslant x) + P(|W_n-1|\geqslant \varepsilon_1)$
      - 证明同上
    - $n\to\infty$ 得
      - $P(X\leqslant (1-\varepsilon_1)x) \leq \liminf\limits_{n\to\infty} P(W_nX_n\leqslant x) \leq \limsup\limits_{n\to\infty} P(W_nX_n\leqslant x) \leq P(X\leqslant (1+\varepsilon_1)x)$
    - $\varepsilon\to 0$ 即得 $W_nX_n\xrightarrow{d} X$
  - **理解**：
  - **本质**：收敛的线性组合封闭性
  - **推论**：$W_n\xrightarrow{p} 0$ 时，线行组合收敛到0

## 中心极限定理（不一定服从）

- **$X_n$ 服从中心极限定理**：$\frac{S_n-a_n}{b_n}\xrightarrow{d} \mathcal{N}(0,1)$
  - **$S_n$ 具有渐近正态性**
  - **推论**：特征函数充要条件
- **Levy中心极限定理**：独立同分布，具有二阶矩，$EX_i = \mu$，则 $\xi_n = \frac{S_n-n\mu}{\sqrt{n\sigma^2}} \overset{d}{\to} N(0,1)$
  - **证明（特征函数法）**：
    - $\phi(t) = Ee^{itX}$，则 $\phi'(0) = iEX_i = 0$，$\phi''(0) = i^2EX_i^2 = -1$
    - Taylor展开得到 $\phi(t) = 1-\frac{t^2}{2} + \omicron(t^2)$
    - 则 $\frac{1}{\sqrt{n}}\sum X_j$ 的特征函数为 $\phi_n(t) = \big[ \phi(\frac{t}{\sqrt{n}}) \big]^n \to e^{-\frac{t^2}{2}}$
    - 由标准化得标准情况
  - **推论**：$S_n\sim N(n\mu，n\sigma^2)$ 近似成立
    - $\overline{X_n} \sim N(\mu,\frac{\sigma^2}{n})$ 近似成立

### Levy形式的应用

- 直接计算：
  - 当n较大时，转换为标准正态分布，$P(S_n > k ) = P(\frac{S_n-n\mu}{\sqrt{n\sigma^2}} > \frac{k-n\mu}{\sqrt{n\sigma^2}}) = P(Z>\frac{k-n\mu}{\sqrt{n\sigma^2}})$
- 二项分布的样本收敛：
  - $\tilde{p} = \frac{S_n}{n}$
  - $\frac{S_n-np}{\sqrt{n\tilde{p}\tilde{q}}}\overset{d}{\to} N(0,1)$
  - **证明**：
    - $\tilde{\sigma}^2 = \frac{1}{n}\sum\limits^n_{i=1} (X_i-\overline{X_n})^2 = \overline{X_n}(1-\overline{X_n})$，由中心极限定理，$\frac{S_n-np}{\sqrt{n\tilde{\sigma}^2}}\overset{d}{\to} N(0,1)$
    - 则 $n\tilde{p}\tilde{q} = n\tilde{\sigma}^2$
  - **推论**：$P(S_n\leq b) = \varPhi(\frac{b+0.5-np}{\sqrt{npq}})$
- 再生性的样本收敛
  - $\mathcal{P}(n) = \sum \mathcal{P}(1) \sim N(\mu,\sigma)$
- **依概率增长速度**：独立同分布，$\lim\limits_{n\to\infty} P(n(\mu-\varepsilon) \leq S_n \leq n(\mu+\varepsilon)) = 1$
  - **证明**：直接不等式变形 + 中心极限定理
- **无界收敛条件**：独立同分布，$EX_i = \infty$，依然有 $\lim\limits_{n\to\infty} \frac{1}{n}\sum\limits^n_{j=1} X_j = \infty \quad a.s.$
  - **证明**：构造下界数列 $X_i^M = \min(X_i,M)$，其均值收敛，从而可应用大数律得到a.s.收敛
    - 然后取 $M\to\infty$ 即可，由单增得收敛到无穷
- **低阶忽略性**：$\xi\overset{d}{\to} N(0,1)，\eta_n\overset{p}{\to} 0，则 \xi_n + \eta_n \overset{d}{\to} N(0,1)$
  - **证明**：分离 + 极限定义即可，$P(\xi_n+\eta_n\leq x) \leq P(\xi_n \leq x+\varepsilon，|\eta_n| \leq \varepsilon)$
- **强化中心极限定理**：用 $\tilde{\sigma} = \frac{1}{n}\sum\limits^n_{j=1} (X_j-\overline{X_n})^2$ 或 $\frac{1}{n-1}\sum\limits^n_{j=1} (X_j-\overline{X_n})^2$

### 不同分布的独立列（L-F定理）

- 独立随机变量列，$EX_n = a_n$，$VarX_n = \sigma_n^2$ 均存在
  - 设 $S_n = \sum\limits^n_{k=1}X_k$，$B_n^2 = VarS_n$
- **Lindeburg条件**：
  - $\forall \tau >0，\lim\limits_{n\to\infty} \frac{1}{B_n^2}\sum\limits^n_{k=1} E\{(X_k-a_k)^2\cdot I(|X_k-a_k|\geqslant \tau B_n)\} = 0$
- **Feller条件**：若满足Lin条件，则成立：
  - **单差值归零**：$\max\limits_{1\leq k\leq n} |\frac{X_k-a_k}{B_n}| \xrightarrow{p} 0$
  - **单方差归零**：$\lim\limits_{n\to\infty}\max\limits_{1\leq k\leq n} \cfrac{\sigma_k^2}{B_n^2} = 0$
- **证明**：
  - *Cauchy收敛*：放缩，$\max\to \cup\to \sum\to \sum EI$，最后得到L条件的形式，从而依概率收敛
  - 方差 $\to$ 期望，然后用 $\tau B_n$ 分割区间，取极限 $n\to\infty，\tau\to 0$ 即得L条件的形式
- **理解**：
  - 单个随机变量的收敛无法影响总体的方差
  - 单个方差的变化无法影响总体的方差
- **推论**：$\lim\limits_{n\to\infty} B_n = \infty$
  - **证明**：$\max\limits_{1\leq k\leq n}|X_k-a_k|$ 是非降的，所以为了满足Cauchy收敛，必须有上述条件

#### L中心极限定理

- **Linderburg中心极限定理**：$\frac{S_n-ES_n}{B_n} \xrightarrow{d} \mathcal{N}(0,1)$
  - **证明**；
- **推论（弱化条件）**：存在正常数列 $\{L_n\}$，使 $\max\limits_{1\leq k\leq n}|X_k| \leqslant L_n，\lim\limits_{n\to\infty} \cfrac{L_n}{B_n} = 0$
  - **证明**：
    - 期望绝对值不等式 + 放缩得 $\max\limits_{1\leq k\leq n} |EX_k| \leq L_n$
    - 从而 $\max\limits_{1\leq k\leq n} |X_k-a_k| \leq 2L_n$
    - 再由条件2，即得Lin条件
- **Lyapunov条件（进一步弱化）**：$\exist \delta>0$，使 $\lim\limits_{n\to\infty} \cfrac{1}{B_n^{2+\delta}}\sum\limits^n_{k=1} E|X_k-a_k|^{2+\delta} = 0$
  - **证明**：
    - $|X_k-a_k|\geq \tau B_n$，两边化为 $\delta$ 次幂，得到不等式
    - 用此不等式将Lin条件放缩即可得到Ly条件
  - **本质**：Lin条件的放缩

### 中心极限定理的充要条件

- **正态吸引场**：分布函数 $F(x)$，满足 $\lim\limits_{x\to\infty} \cfrac{x^2\int^\infty_x f(u)du}{\int^x_{-x}u^2f(u)du} = 0$
- **充要条件**：若 $X_1$ 的分布函数属于正态吸引场 $\Leftrightarrow$ 存在 $a_n,b_n$ 使得中心极限定理成立
- **Lin充要条件**：Lin中心极限定理 + Feller条件成立 $\Leftrightarrow$ Lin条件成立

### 多维中心极限定理

- $X_n$ 为m维，若 $\forall \|s\| = 1$，有 $s^TX_n\xrightarrow{d} \mathcal{N}(0,1)$，则 $X_n\xrightarrow{d} \mathcal{N}(0,I)$

## 分布函数一致连续性

- **2-积分范数**：R上的函数 $f(t)$ 可变换为Riemann积分空间中的范数 $\lim\limits_{T\to\infty} \frac{1}{2T}\int^{T}_{-T} |f(t)|^2dt$，则称存在2-积分范数
- **引理**：若 $\nu$ 为有限实测度，则 $\|\int e^{itx}d\nu(x)\|^2$ 存在，值为 $\sum |\nu(x)|^2$
- **等价定理**：$F_n$ 与 $F$ 的下列性质等价：
  - 一致收敛
  - 依分布收敛，且 $\sum |\mu_n(\set{x}) - \mu(\set{x})|^2\to 0$
  - 特征函数一致收敛，且 $\|\phi_n(t) - \phi(t)\|\to 0$

### a.s.收敛

- **级数a.s.收敛**：存在 $P(\Omega_0)=1$，$\forall \omega\in \Omega_0$，$\sum\limits^\infty_{n=1}X_n(\omega)$ 均收敛
- **矩判定引理**：若 $\exist r\in (0,1]$，有 $\sum\limits^\infty_{n=1} E|X_n|^r<\infty$，则级数a.s.收敛
  - **证明**：C-R不等式
- **Kolmogorov判定引理**：若独立列满足 $EX_n=0，\sum\limits^\infty_{n=1} EX_n^2<\infty$，则级数a.s.收敛
  - **证明**：
  - **推论**：若 $r\in (1,2]$，独立列满足 $EX_n=0，\sum\limits^\infty_{n=1} E|X_n|^r<\infty$，则级数a.s.收敛
    - **证明**：
- **传递引理**：有界独立列 $\{X_n\}$
  - 若级数a.s.收敛，则期望级数、方差级数均收敛
  - 若 $EX_n=0$，且 $\sum\limits^\infty_{n=1}VarX_n = \infty$，则级数a.s.发散
  - **证明**：
- **Kolmogorov三级数定理**：独立列的级数a.s.收敛
- **充分条件**：$\exist c>0$，同时满足
  - $\sum\limits^\infty_{n=1}P(|X_n|>c) < \infty$
  - $\sum\limits^\infty_{n=1}E(X_nI(|X_n|\leqslant c))$ 收敛
  - $\sum\limits^\infty_{n=1}Var(X_nI|X_n|\leqslant c) < \infty$
- **必要条件**：$\forall c>0$，上面三级数均收敛

### 强大数律

- **强大数律**：存在 $a_n，b_n$，使得 $\frac{S_n-a_n}{b_n}\to 0\space a.s.$
  - <font color='chartreuse'>区别：</font>
- **标准形式**：$\{X_n\}$ 独立同分布，则 $\lim\limits_{n\to\infty} \overline{X_n} = \mu\quad a.s.$
    - **证明**：由不等式2 + B-C引理即得
    - **推论**：$\lim\limits_{n\to\infty} \frac{1}{n}\sum\limits^n_{j=1} x_j = \mu$，实际中必定出现
      - 数学期望的强相合估计：$\mu$
      - 样本方差的强相合估计：$\sigma^2$
- **Kolmogorov强大数律**：$\{X_n\}$ 独立同分布
  - $\begin{cases} a_n = n\cdot a \\ b_n= n \end{cases}$ 成立  $\Leftrightarrow E|X_1|<\infty，EX_1=a$
- **Marcinkiewicz强大数律**：$\{X_n\} $独立同分布
  - $\begin{cases} a_n = n\cdot a \\ b_n=\sqrt[r]{n} \end{cases}$ 成立 $\Leftrightarrow E|X_1|^r<\infty，a = \begin{cases} EX_1，\qquad 1\leqslant r < 2 \\ 任意实数，0<r<1 \end{cases}$
