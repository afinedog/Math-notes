# 概率公式

## 加法公式（容斥原理）（事件的并）

- $P(A\cup B) = P(A) + P(B) - P(AB)$
  - **证明**：化为互斥的集合（直接可加性）即可
- **Jordan公式(定义式)（容斥原理的概率形式）**：$$\begin{matrix} P(\mathop{\cup}\limits^n_{j=1}A_j)  & = & \sum\limits^n_{j=1}A_j + (-1)\sum\limits_{1\leq j_1 < j_2\leq n}P(A_{j_1}A_{j_2}) + ... \\\ \\ && + (-1)^{k-1}\sum\limits_{1\leq j_1<j_2<...<j_k\leq n} P(A_{j_1}A_{j_2}...A_{j_k}) + ...  \\\ \\ && + (-1)^{n-1}P(A_1A_2...A_n) \end{matrix}$$
  - **本质**：交并分配律
- **Jordan公式(一般式)**：如果所有排列的概率相等，则
  - $P(\mathop{\cup}\limits^n_{i=1}A_i) = \sum\limits^n_{k=1} (-1)^{k-1} C^k_n P(A_1A_2...A_k)$
- 优点是把复杂的n选m问题分解成多个n选1问题
- **应用**
  - 复杂排列问题：n人选m球，至少有一个选中/选不中
  - 配对问题：n个人乱序排列编号，求至少有一个得到自己顺序编号的概率

## 乘法公式（事件的交）

### 独立事件

- $P(AB) = P(A)P(B)$
- **独立性的运算传递**：集合运算公式证明，或概率互斥可加性证明
  - 互斥独立性：$A\not\sim B \Leftrightarrow \bar{A}\not\sim B$
  - 子集独立性
  - 并集独立性
  - 交集独立性
- **伪独立**：只要满足公式，即使看起来是相关事件，它也是独立的
  - **染色骰问题**：三个面纯色 + 一个面三色，则投骰子出现三个颜色的事件是独立的
    - $P(A)=P(B)=P(C)=\frac{1}{2}，P(AB)=P(AC)=P(BC)=\frac{1}{4}=P(ABC)$
    - 三个事件两两相互独立，但不相互独立
  - 乘法公式的数量：$\sum\limits^\infty_{i=2} C^i_n = 2^n-n-1$
- **相互独立**：任意个事件组合都独立 > **两两相互独立**
  - 相互独立 $\Leftrightarrow P(D_1...D_n) = P(D_1)...P(D_n)$ 均成立（$D_i = A_i或\overline{A_i}$）
    - **证明**：
      - 充分性：等式共有 $2^n$ 个，而且互斥本身就具有组合事件的功能，所以相当于所有乘法公式成立，从而相互独立
      - 必要性：归纳法（课件里的证明贼麻烦，为啥？）
- **独立事件列**：相互独立的一列事件
- 独立进行的试验，其事件相互独立
  - 看成一个试验，则其事件是各个独立试验的事件积
  - $^\#(A_1...A_n) = k_1×k_2×...×k_n$（分数乘法结合律得到结果）
- **小概率原则**（墨菲定律）：如果概率存在，则无限次数试验中，该事件一定会发生
  - **证明**：概率的连续性
- 独立事件满足加法公式
  - 事件独立（乘法公式）（事件的交） $\neq$ 事件不相容（可列可加）（事件的并）

### 条件概率

- $P(B| A)$：已知A发生的条件下，求B发生的概率
- **统计学意义下的概率**：生男生女问题中，$50\%$ 是统计学定义下的概率，而并非独立事件的概率
- **条件概率公式**：$P(B| A) = \frac{P(AB)}{P(A)}$
  - **证明**：古典概型的样本点证明法
- **乘法公式**：$P(A_1A_2...A_n) = P(A_1)P(A_2| A_1)...P(A_n| A_1...A_{n-1})$
  - **证明**：条件概率定义式 + 累乘法
- **$\sigma$ 域的限制**：
  - 已知概率空间 $(\Omega,\mathcal{F},P)$，若 $C \in \mathcal{F}$ ，则条件概率 $P_C$ 也是 $\mathcal{F}$ 上的概率
    - 已有非负性
    - $\Omega$ 只需取子集 $C \to$ 和1性
    - 独立性的交集传递 $\to$ 可列可加性
  - **叠加公式**：$P_C(A| B) = P(A| BC)$（定义证明）
    - 本质是限制的叠加
- **应用**：
  - 无法挽回问题（摔坏、造假）
- 独立事件满足条件概率公式

## 限制公式（线性）

### 全概率公式（Lebesgue可测）（限制的相加）

- $P(B) = P(B| A)P(A) + P(B|\bar{A})P(\bar{A})$
- **应用**
  - 分类概率：坐自行车和汽车晚点概率
  - 抽签问题：n个签里m个中，无放回抽取，第j次抽中的概率还是 $\frac{m}{n}$
    - 拿出和不拿出的签之间仍可以看作一个整体（因为结果是一维的，不存在结果之间的影响，也就是前面和后面的签构成独立试验）
    - 超几何分布，不是条件概率，可以看作同一时空内，也就是排列，$P=\frac{m·(n-1)!}{n!}$
  - 不记名问题（二重分类问题，隐瞒信息的同时不影响概率）
  - 赌徒破产模型（递推公式）：本金a元，赢b元结束，输赢概率相等，求输光概率
    - 设k元输光为 $B_k$，本局赢为 $A$，则 $P(B_k|A) = P(B_{k+1})$，$P(B_k|\bar{A}) = P(B_{k-1})$
    - 由全概率公式得到递推公式，从而 $P(B_a) = 1-\frac{a}{a+b}$
  - 零和博弈模型：n个赌徒想要赢得其它所有人的本金
    - 第i个赌徒输光概率：$P(a_i) = 1-\frac{a_i}{\sum a_i}\to 1$
  - 波利亚坛子模型：b个黑球，r个红球，取完球后放回相同的球a个，求第n次取出黑球的概率 
    - 1、数学归纳法 + 二阶全概率公式证明 $P = \frac{b}{b+r}$
    - 2、全概率公式证明
- **完备事件组**：互不相容的一组事件
- **全概率公式（完备事件组）**：$P(B) = \sum\limits^n_{j=1}P(A_j)P(B|A_j)$

### Bayes公式

- **条件概率的全概率公式形式**：$\large P(A|B) = \frac{P(B|A)P(A)}{P(B|A)P(A) + P(B|\bar{A})P(\bar{A})}$
- **应用**
  - 条件反转问题：二重分类问题下，已知二级结果，反推一级结果
  - B和A是无包含关系的两个事件，则可以定义限制。且利用补集，限制可互推
  - **贝叶斯决策**：决定哪种二重分类更适用
    - 定义先验概率和后验概率
    - 抽取我们期望的特征，求该后验概率确定时不同抽取方法的先验概率，哪个大选哪个
- **Bayes公式（完备事件组）**：$\large P(A_j|B) = \frac{P(A_j)P(B|A_j)}{\sum\limits^n_{i=1}P(A_i)P(B|A_i)}，1\leq j\leq n$

## 概率极限

- **上极限**：$A_n\ i.o.$
- 包含关系、对偶关系、单调收敛关系
- **Borel-Cantelli引理**：
  - 若概率级数收敛，则 $P(A_n\ i.o.) = 0$
  - **证明**：
    - $P(\mathop{\bigcap}\limits^\infty_{n=1}\mathop{\bigcup}\limits^\infty_{j=n} A_j) = P(\mathop{\cap}\limits^\infty_{n=1}C_n\downarrow)$
    - 概率连续性，$ = \lim\limits_{n\to\infty}P(\mathop{\cup}\limits^\infty_{j=n}A_j)$
    - 三角不等式，$\leq \lim\limits_{n\to\infty}\sum\limits^\infty_{j=n}p_j$
    - 由Cauchy收敛原理，结果为0
  - 若事件独立，概率级数发散，则 $P(A_n\ i.o.) = 1$
  - **证明**：
    - $P(\mathop{\cap}\limits^\infty_{n=1}\bar{A}_j)$，独立乘法公式 + 补集的概率
    - 由不等式 $1-x \leq e^x$，指数性质化乘法为加法，极限为0，则补集极限为1
  - **推论**：独立事件列的概率上极限只能是1或0
  - **应用**：墨菲定律（P=1）、低维几何概型（P=0）

## 总结

### 时空问题总结

- 挑选法：空间不变，时间变化
- 排列组合法：空间变化，时间不变
- 比如抽签问题，空间上显然与抽签序号k无关，但时间上不易看出。

### 习题

- **三门问题（时空换序法）（互斥但不独立）**
  - 主持人不知道车在哪里：此时相当于在选择前就给你打开了一扇门，车在门里的概率变为 $\frac{1}{2}$
  - 主持人知道车在哪里：此时无论你怎么选，打开的都是羊。相当于没打开，变为抽签问题。 $\frac{2}{3}$
  - 主持人知道车在哪里的概率为P：
    - 不能简单应用上述两种情况的全概率公式。因为主持人不知道车在哪里，但依然打开了羊的门，此时构成条件概率。所以考虑问题时，应该考虑三个事件
    - 也就是说简单全概率项 $P(B|A)P(A)$ 中，A在（1）中是必然事件，但在（3）中是条件概率事件，$P(B|A)和P(A)$ 都需要再次拆分（因为你不知道主持人是知道还是不知道，相当于（2）情况的存在影响了（1）情况自身的概率，很奇妙，它们互斥，但不独立），所以必须考虑三个事件
    - 实际上，下面这个方法才是一般的方法
  - **完备法**：一共有三个事件：$\begin{cases} A：选手选中羊 \\ B：主持人打开羊 \\ C：主持人知道车在哪里 \end{cases}$
- **二胎问题（统计学约束了实际情况）（时空一致法）**
- **棋局问题（事件列法 + 方程法）**：
  - $A_n$：到第n局时甲获胜，$B_n$：到第n局乙获胜，$C_n$：到第n局平局
  - 若没有平局，$P(A_n) = P(D_{n-2})p^2$，$P(B_n) = P(D_{n-2})q^2$，再由和1性，得到 $P(A_n) = \frac{p^2}{p^2+q^2}$
  - 若有平局，因为前面的求法和平局的概率无关，而且有平局的获胜条件和没有平局一样，所以概率相等（相当于平局是穿插在胜负局中的无意义事件）
  - **本质**：因为没有用古典概型法，用的方程法
- **棋局问题（古典概型法 + 数列法）**：若没有平局情况
  - 不断列举甲获胜的情况，发现其概率构成等比数列，加和得到概率 $P = \frac{p^2(1+pq)}{1-pq}(q=1-p)$

### 多项式系数

- $r+(n-r) = n$，即把n分为2个部分的分法，共有 $C^r_n = \frac{n!}{r!(n-r)!}$
- $r_1+...+r_k = n$，即把n分为k个部分的分法，共有 $\frac{n!}{r_1!...r_k!}$
- n个数里有重复地取r个，共有 $C^r_{n+r-1}$ 种取法
- 二项式系数性质
  - $C^n_k = 0（n>k>0）$
  - $\sum\limits^n_{k=1}C^k_n = 2^n$
    - **证明**：数论
  - $\sum\limits^n_{k=1} C^k_nC^{n-k}_n = C^n_{2n}$
