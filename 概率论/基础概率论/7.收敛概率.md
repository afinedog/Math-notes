# 极限定理

## 概率的收敛性

### 概率不等式

- **Markov不等式**：$P(|X|\geqslant \varepsilon) \leq \frac{1}{\varepsilon} E|X|$
  - **本质**：部分区域积分小于整体区域积分
  - **理解**：
    - 证明方法为：
      - 将P化为均值（示性函数法）
      - 将均值化为积分（积分不等式法）
  - **证明（示性函数期望）**：
    - 示性函数 $I[|X|\geqslant \varepsilon] \leq \frac{|X|}{\varepsilon}$
      - 若X大于 $\varepsilon$，则左边为1，右边大于1
      - 若X小于 $\varepsilon$，则左边为0，右边为正数
      - （伯努利分布的0-1性，或边界性，或极值性）
    - 两边取期望，即得到结果
      - 左侧：伯努利分布的均值即为其成功概率
      - 右侧：期望的内含性
  - **证明（积分不等式）**：
    - 变形得 $\color{blue}\int^\infty_\varepsilon \varepsilon P(|X|=x)dx \leqslant \int^\infty_0 xP(|X| = x)dx$
      - <font color='blue'>左边积分区域小，系数大。右边积分区域大，系数小
      - 右减左即可得证（其实挺类似上面的过程的，毕竟是一道题，思想是不会变的）</font>
    - <font color='chartreuse'>若将左边设为 $g(\varepsilon)$，则 $g'(\varepsilon) = P(|X|\geqslant\varepsilon) - \varepsilon P(|X|=\varepsilon)$（但没用）
    - $(0,+\infty)$ 上 $e^{-x}$ 积分为1，故还可添项化为积分形式作差（但没用）</font>
- **Chebyshev不等式**：$P(|X-EX| \geqslant \varepsilon) \leq \dfrac{1}{\varepsilon^2} Var(X)$
  - **证明**：两边的E取方差算式，再应用M不等式即可证明，$P(|X-\mu|^2 \geq \varepsilon^2) \leq \frac{E(X-\mu)^2}{\varepsilon^2}$
  - **本质**：
    - 上面是期望形式，下面是方差形式
    - 换元 $Y = |X-EX|$ 后，下面是上面的平方形式，过程相同

### 弱收敛

- $X_n\overset{d}{\to} X$：在 $F(x)$ 连续点上 $\lim\limits_{n\to\infty} F_n(x) = F(x)$
  - **理解**：分布函数列在连续点处收敛
- **依分布收敛**：弱极限是分布函数
  - **反例**：弱收敛但不依分布收敛
    - $F_n(x) = \begin{cases} 0，\quad x<-n \\ \frac{x+n}{2n}，-n\leqslant x < n \\ 1，\quad x\geqslant n \end{cases}$，是分布函数列
    - $F(x) = \frac{1}{2}$，不是分布函数
- **实分析关系**：$F_n\ w.$（依分布收敛）弱于 $F_n\ p.w.$（点态收敛）
  - **反例**：点态收敛但不依分布收敛
    - $F_n(x) = 1（x\geqslant \frac{1}{n}）$，其满足右连续性，是分布函数列
    - 点态极限函数 $F(x) = 1（x>0）$ 不满足右连续性，不是分布函数
  - **进化方法**：若要满足右连续性，只需保证在连续点上收敛
- **Levy连续定理**：依分布收敛 $\Leftrightarrow$ 特征函数点态收敛 $\lim\limits_{n\to\infty}\phi_n(t)=\phi(t)$
  - **证明**：唯一决定性

### 依概率收敛（依测度收敛）

- $X_n\overset{r}{\to} X：\lim\limits_{n\to\infty} P(|X_n-X|\geqslant \varepsilon) = 0$
  - **理解**：密度函数列依测度收敛
- **极限归一性**：独立同分布的随机变量，极限下算术均值等于数学期望
  - $\lim\limits_{n\to\infty} P(|\overline{X_n}-\mu| \geq \varepsilon) = 0$
  - **证明**：均值方差 + Ch不等式
- **应用**：
  - **求线性概率的极限**：级数转换为均值
  - 配合内积不等式，求概率大小关系
  - **常用不等式**：独立同分布列，当 $EX_i = 0$ 时
    - $E(\overline{X_n})^4 < \frac{3\mu_4}{n^2}$
      - **证明**：独立性得仅幂变量期望不为0，转化为级数 $ES_n^4$，逐二次展开，利用 $X^2$ 的内积不等式即可
    - $P(|\overline{X_n}| \geq \sqrt[8]{\frac{1}{n}}) < \frac{3\mu_4}{\sqrt{n^3}}$
      - **证明**：取四次方后，上述不等式 + M不等式

### 平均收敛（依范数收敛）

- **平均收敛**（$X_n\overset{L_r}{\to}X$）：$E|X_n-X|^r\to 0$
  - **完备性**：$\forall X\in L_r$，均存在平均收敛到 $X$ 的序列
    - **证明**：构造
      - $X_n = \sum\limits^\infty_{m=-\infty} \frac{m-1}{2^n} I(\frac{m-1}{2^n}\leqslant X < \frac{m}{2^n})$
      - 则 $|X_n-X| \leqslant \large\frac{1}{2^{rn}}$
    - **理解**：指示函数积分性 + Kronecker选择性
    - **本质**：类比测度论基本方法 + 可测逼近定理，可测函数总存在简单函数列可逼近

### 一致可积

- **一致可积**：$\lim\limits_{a\to\infty}\sup\limits_{n\in N}E(|X_n|I(|X_n|\geqslant a)) = 0$
  - **理解**：无穷大处的序列最大值，积分（期望）为0
- **p积分比较**：一致可积则 $\cfrac{\max\limits_{1\leqslant k\leqslant n} |X_k|}{n}\xrightarrow{p} 0$
  - **证明**：分割为 $|X_n|<a$（上界不等式）和 $|X_n|\geqslant a$（一致可积性），分别得到0
- **充要条件**：类比实分析可积性
  - *奇点处积分极小*：$\forall \varepsilon > 0，\exist \delta(\varepsilon) > 0，\forall P(A)<\delta$，有 $\sup\limits_{n\in N} E(|X_n|I(A)) < \varepsilon$
  - *积分有界性*：$\sup\limits_{n\in N}E|X_n| < C$
  - **充分性**：
    - 若 $a\to\infty$，自然有 $a>\frac{C}{\delta}$
    - 由M不等式 + 期望有界性，$P(|X_n|\geqslant a) \leq \frac{E|X_n|}{a} < \frac{C}{a} < \delta$
    - 再由极小区域积分归零性，$\sup\limits_{n\in N}E(|X_n| I(|X_n|\geqslant a)) < \varepsilon（a\to\infty）$
    - 定义得一致可积性
  - **必要性**：$A:|X_n|\geqslant a$，在 $a\to\infty$ 时，必有 $P(A)<\delta$，否则概率不归一
    - 由一致可积性，$\sup E(|X_n|I(A)) = 0（a\to\infty）$，（奇点处积分极小）
    - 若期望无界，由极大处期望极小 + 有界处期望有界 + 概率的无穷处归零性，得到矛盾（积分有界性）
  - **理解**：期望可类比实分析中的积分，只不过多了概率空间的限制，定义也有所不同
  - **推论（充分条件）**：
    - **p积分比较**：若 $\exist p>0$，使得 $\sup\limits_{n\in N}E|X_n|^{1+p}$ 存在，则一致可积
      - **证明**：
    - **比较定理**：$\exist Y\in L_1，\forall x>0，\sup\limits_{n\in N}P(|X_n|>x) \leqslant P(|Y|> x)$，则一致可积
      - **证明**：
- **平均收敛判别准则**：给定 $r>0$
  - *一致可积传递*：若 $\{|X_n|^r\}$ 一致可积，且 $X_n\xrightarrow{p}X$，则 $X\in L_r$，且 $X_n\xrightarrow{L_r} X$
  - **证明**：
  - **平均收敛性**
    - p收敛 + Riesz定理，存在子列a.s.收敛
    - 再用Fatou引理，得 $E|X|^r\leqslant E(\liminf\limits_{k\to\infty} E|X_{n_k}|) \leqslant \liminf\limits_{k\to\infty} E|X_{n_k}|^r \leqslant\sup\limits_{n\in N} E|X_n|^r < \infty$
  - **极限**
    - 再由 $X_n$ 一致可积，比较得 $X$ 也一致可积。
    - 将 $E|X_n-X|^r$ 用 $\varepsilon$ 拆分区间为收敛处 + 非收敛处（$C_r$ 不等式拆分期望，并分别讨论 $X、X_n$）
  - *平均收敛传递*：若 $X_n\in L_r$，且 $X_n\xrightarrow{L_r} X$，则 $X\in L_r$，且 $X_n\xrightarrow{p} X$
  - **证明**：
    - **期望有界**：添项 + $C_r$ 不等式，得 $\sup E|X_n|^r$ 有界
    - **奇点处极小**：添项 + $C_r$ 不等式，得 $E(|X_n^r|I(A))$ 极小
  - **本质**：$C_r$ 不等式的拆分
- **等价定理**：随机变量 $X$ 和其p逼近列 $X_n$ 均存在一阶矩，则下列条件等价
  - $X_n$ 一致可积
  - 一阶平均收敛 $E|X_n-X| \to 0$
  - 期望收敛 $EX_n\to EX$
  - **证明**：
    - $1\leftrightarrow 2$：平均收敛判别准则
    - $2\to 3$：期望三角不等式 + 比较定理
    - $3\to 2$：$(X-X_n)^+ \leq X$，且 $(X_n-X)^+\xrightarrow{p} 0$，从而由控制收敛定理，$E(X_n-X)^+\xrightarrow{p} 0$

### 习题

- **强弱关系**：
  - $d < p.w.\quad d<p$
    - p.w.不收敛，但弱收敛于 $P(X=0) = 1$
      - 退化分布 $P(X_n=\frac{1}{n}) = 1$，分布函数 $F_n(x) = \begin{cases} 0,x<\frac{1}{n} \\ 1, x\geq \frac{1}{n} \end{cases}$，极限函数为 $g(x) = \begin{cases} 0,x\leq 0 \\ 1,x > 0 \end{cases}$，在 $x=0$ 上不满足右连续，不是分布函数
      - **本质**：
    - p不收敛，但弱收敛于 $-X$：
      - $X\quad -1\quad 1 \choose \qquad \frac{1}{2} \quad \frac{1}{2}$，令 $X_n = -X$，它们同分布，但不依概率收敛
      - **理解**：它们同分布，从而满足弱收敛。但它们完全对称，从而不依概率收敛
      - 独立同分布的 $X_n\sim \mathcal{B}(0,1)$，均不依概率收敛
      - **本质**：d只关心分布情况，r要关心概率情况，而a.s.还关心迭代中的情况，$L_r$ 是等度收敛，即差值的积分（期望）也要归零
  - $p < a.s.\quad p<L_r$
    - a.s.不收敛，但p收敛：
      - $X\equiv 0，Y\sim U(0,1)，X_{2^{n-1}+k} = I(\frac{k-1}{2}< Y < \frac{k}{2^n})$
      - **证明**：
        - 依概率收敛：$P(|X_m|>\varepsilon) = P(X_m = 1) = \frac{1}{2^n}$
        - a.s.不收敛：$\forall \omega\in \Omega，\exist m，X_m(\omega) = 1$
      - 在实分析中，这是一个收敛区间随迭代而变化的例子
    - $L_r$ 不收敛，但p收敛：
      - $X(\omega) = 0，X_n = n（\omega\in (0,\frac{1}{n})）$
      - **证明**：
      - 在实分析中，这是一个序列和积分无法交换的例子（期望的本质也是积分）
  - a.s. 和 $L_r$ 无关
    - $L_r$ 收敛，a.s.不收敛
      - $X_n = I(\frac{n-2m}{2^m} < \omega < \frac{n+1-2m}{2^m})，2^m\leqslant n < 2^{m+1}$
      - a.s. 不收敛：omega取无理数时均为非收敛区间
      - $L_r$ 收敛
    - a.s.收敛，$L_r$ 不收敛
      - 上上例中也 a.s. 收敛
    
- **互推关系**：
  - $p \to d$：
    - **证明（变形法）**：$P(|X|\geq \varepsilon)$ 内部不等式的变形 + 转化
    - **证明（放缩法）**：左右连续，即 $F(x-) \leq \underline{\lim\limits_{n\to \infty}} F_n(x) \leq \overline{\lim\limits_{n\to \infty}} F_n(x) \leq F(x+)$
    - **本质**：同变量必定同分布
  - $d \to p$：（**收敛条件**：弱收敛到常数随机变量时，也依概率收敛）

    - **证明（变形法）**：将 $P$ 转化为 $F_n$，定义证明
    - **本质**：防止同分布但非同变量的情况
  - $p.w.\to d$
    - **证明**：定义即可
  - $d\to p.w.$（**收敛条件**：分布函数一致连续）
  - $a.s. \to p$
    - **Cauchy收敛原理**：若几乎必然收敛 $X_n\to X \quad a.s.$，则 $\max\limits_{k\geq n} |X_k - X|\overset{p}{\to} 0$
    - **证明**：$A_n = \mathop{\bigcup}\limits^\infty_{k=n} \{|X_k-X| > \varepsilon\}$ 是单减集列

  - $p\to a.s.$ （**收敛条件**：函数列的收敛区间在迭代中不移动）

  - $L_r\to p$
