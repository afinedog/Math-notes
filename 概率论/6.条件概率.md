# 向量的相关性

- 视角：
  - 向量视角：条件事件有两个变量，X和Y，本质上是具有相关性的向量
  - 空间视角：$X$ 在 $Y_i$ 空间上的限制

## 条件分布

- **条件事件**：$X|Y$
  - 条件事件的取值：$A$ 发生后，$X\geq 0$，则为非负随机变量
  - 条件事件独立性：$\{X\leq x\}$ 和 A独立，则 $X和A独立$
  - 和1性 $\Leftrightarrow$ 全概率公式
- **条件随机变量**：$X|\{Y=y_j\}$，定的是Y，变的是X
  - 离散型：条件概率公式：$P(X=x_i|Y=y_i) = \frac{p_{ij}}{q_i}$，分子是离散分布，分母是边缘分布
  - 连续型：$f(x|y_i) = \frac{f(x,y_i)}{f_Y(y_i)}$
- **条件概率空间**：$(\Omega,\mathcal{F},P_A)$

### 条件数学期望

- **条件数学期望**：$E_{y_i}X = E(X|Y=y_i) = \sum\limits^\infty_{j=1} x_jP(X=x_j|Y=y_i)$
  - **示性函数形式**：
    - $E(X|A) = \frac{E(X|I[A])}{P(A)}$
      - **定义证明**：$\{X>x,A\} = \{XI[A]>x\}$，然后用条件概率公式即可
      - **绝对值分解证明（本质上是一个东西）**：
        - $E(X|A) = E(X^+|A) - E(X^-|A) = \frac{E(X^+I[A])}{P(A)} + \frac{E(X^-I[A])}{P(A)}\\$ 重新组合即可
  - **理解**：
- **数学期望的全概率公式**
  - $EX = \sum\limits^\infty_{i=1} P(A_i)E(X|A_i)$（完备事件组）
- **数学期望的条件形式**：
  - $EX = \sum\limits^\infty_{i=1} \big[E(X|Y=y_i)·P(Y=y_i)\big]$
  - $EX = \int^{+\infty}_{-\infty} \big[E(X|Y=y)·f(y)\big]dy$
  - 数学期望具有内含性，无论先算还是后算，都是一个结果
  - $EX = E\big[ E(X|Y=y_i) \big]$
- **条件数学期望的性质**
  - **线性**：$E[(aX+bY)|A] = aE(X|A) + bE(Y|A)$（把 $E$ 换成 $E_A$ 即可）
  - **独立乘法公式**：$E(X|A) = EX$
    - **定义证明**
    - **绝对值分解证明**：$E(X|A) = E(X^+|A) - E(X^-|A) = EX$
      - **绝对值筛选变量**：$X^+ = \frac{|X|+X}{2}，X^- = \frac{|X|-X}{2}$
  - **本质**：换了个概率空间，但是数学期望是依赖于变量的，所以所有概率空间上性质不变
- **应用**：
  - 分解问题，先算条件期望，再用条件形式算整体期望

#### 习题

- $P(X^2=k)$ 和 $EX^2$ 的区别：前者是 $X^2$ 的取值的概率，后者是 $X^2$ 的取值的均值。这么理解就可以。而我以前习惯把 $X^2$ 先逆投影到X上，这是微分法的思想。但微分法的思想不能用到数学期望中来。
- 独立同分布，$S_n = \sum\limits^n_{i=1}X_i$，则 $E(X_i|S_n) = \frac{1}{n}\sum\limits^n_{j=1}X_j$（同分布下，数学期望相等，且独立加和公式）
- 答题格式
  - **函数形式的期望**：$E(X|Y=y) = y$（y为参数）
  - **含参期望**：$E(X|Y) = Y$
  - **含参变量形式的密度**：$f(X|Y)$（y为参数）

### 条件密度

- **条件分布函数**：$F_{X|Y}(x|y) = P(X\leq x|Y=y) = \int^x_{-\infty} \frac{f(s,y)}{f_Y(y)}ds$ (含参变量上限积分)
- **条件密度**：$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}$
- **独立限制公式**：$X、Y独立 \Leftrightarrow F_{X|Y}(x|y) = F_X(x)\ / \ f_{X|Y}(x|y) = f_X(x)$
- **条件正态分布**：$(X,Y)\sim N(\mu_1,\mu_2;\sigma_1^2,\sigma_2^2;\rho)$
  - $\mu_y = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$、$\sigma_y^2 = (1-\rho^2)\sigma_1^2$
  - 计算得到 $f_{X|Y}(x|y)\to N(\mu_y,\sigma_y^2)$
  - 然后根据正态分布的形式，确定为条件正态分布的期望和方差
  - $X|\{Y=y\} \sim N(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)，(1-\rho^2)\sigma_1^2)$

## 极值分布

- **顺序统计量**：取一组样本 $X_1,...,X_n$，在某个统计量下的相应观测值为 $x_1,...,x_n$
  - 将观测值从小到大排列为 $x_{(1)},...,x_{(n)}$，则对应的 $X_{(1)},...,X_{(n)}$ 就是顺序统计量
  - **独立同分布**：样本和总体的分布相同，且相互独立

### 最值的分布

- 最值其实就是 $max(X_i)、min(X_i)$，它的分布求法前面已经有了，用定义法即可
- **最大值的分布**：$F(x) = P(\mathop{\bigcap}\limits^n_{i=1} \{X_{(i)} \leq x\})$
  - 由独立同分布性，**分布函数** $= \prod\limits^n_{i=1} P(X\leq x) = \big[ F(x) \big]^n$
  - **密度函数**：$f(x) = nf(x)\big[ F(x) \big]^{n-1}$
- **最小值的分布**：$F(x)$
  - **Jordan法**：$F(x) = P(\mathop{\bigcup}\limits^n_{i=1} \{X_{(i)} \leq x\})$
    - <font color='chartreuse'>（这里要用Jordan加法公式，太麻烦，换一种方法）</font>
  - **反向法**：$F(x) = 1-P(X_{(1)}>x) = 1-\prod\limits^n_{i=1}P(X_i>x)$
  - 由独立同分布性，**分布函数** $ = 1-\big[ 1-F(x) \big]^n$
  - **密度函数**：$f(x) = nf(x)\big[ 1-F(x) \big]^{n-1}$
- **最大最小值向量 $(X_{(1)},X_{(n)})$ 的联合分布**：
  - $F_{1,n}(x_1,x_n) = P(\purple{X_{(1)}\leq x_1},\red{X_{(n)} \leq x_n})$
  - 添项作差得到 $P(X_{(n)}\leq x_n) - P(\purple{X_{(1)}>x_1},\red{X_{(n)}\leq x_n})$
  - 后项化乘法为交集，然后再利用独立性乘法公式，得到 
    - $P(\mathop{\bigcap}\limits^n_{j=1}(x_1 < X_j\leq x_n)) = \prod\limits^n_{j=1} P(X_1<X_j \leq X_n)$
  - **分布函数** $[F(x_n)]^n - [F(x_n)-F(x_1)]^n$

### 顺序统计量的分布

- **n维随机向量微分法**：$g(\vec{y}) = \sum\limits^m_{i=1}f(\vec{x_i})|J_i|d\vec{y}$（分段函数形式）
- **顺序统计量的总联合密度（独立同分布）**：$(X_{(1)},...,X_{(n)})$
  - 大小排序依赖于 $x_i$ 的值，和随机变量序号没有关系，所以取 $X$ 全排列即可
  - $P(X_{(1)} = x_1,...,X_{(n)} = x_n) = P(\mathop{\bigcup}\limits_{i\in I} \{X_{i_1}=x_1,...,X_{i_n}=x_n\}) = n!P(X_1 = x_1,...,X_n = x_n)$
  - 独立乘法公式，得联合密度为 $n!f(x_1)...f(x_n)dx_1...dx_n$
- **某个统计量的概率密度**：$X_{(k)}$
  - $P(X_{(k)} = x) = C^{k-1}_nC^1_{n-k+1} P(\mathop{\bigcap}\limits^{k-1}_{i=1}\{X_i<x\}, X_k=x, \mathop{\bigcap}\limits^\infty_{j=k+1}\{X_j>x\})$
  - 再使用独立乘法公式
  - $g_k(x) = nC^{k-1}_{n-1} [F(x)]^{k-1} [1-F(x)]^{n-k}f(x)dx$
- **任意二个统计量的联合密度**：$(X_{(i)},X_{(j)})(i<j)$
  <br>
  - $P(X_{(i)} = x, X_{(j)} = y)$
- $C^{i-1}_nC^{j-i-1}_{n-i+1}C^{n-j}_{n-j+2}C^1_2 · [F(x)]^{i-1}·[F(y)-F(x)]^{j-i-1}·[1-F(y)]^{n-j}f(x)f(y)\\$ 

#### 习题

- 均匀分布的联合密度：$g(x_1,...,x_n) = \begin{cases} \frac{n!}{a^n}，0<x_1<...<x_n<a \\ 0，其他情况 \end{cases}$

## 母函数

- **前提**：非负整数随机变量，定义 $0^0 = 1$
- **概率母函数**：$g(s) = E(s^X) = \sum\limits^\infty_{j=0} s^jP(X=j)，s\in [-1,1]$
  - $P(X=k) = \frac{g^{(k)}(0)}{k!}$
    - 类Lagrange差值多项式的求导降次——线性筛选公式
  - $EX = g'(1)$
    - 求导得常数
  - 唯一性
    - 期望唯一性 + 指数函数单射传递
  - 独立线性公式：$Y=\sum\limits^n_{i=1}X_i，g_Y(s) = \prod\limits^n_{i=1}g_i(s)$
    - 幂的乘法线性
- **应用**：
  - 等概率事件，化为等比数列求和，简化计算
  - 母函数 $\Leftrightarrow$ 分布/期望（**无简化意义**）
  - 求解线性分布
    - **再生性**：二项分布、泊松分布
    - **Taylor级数**：$\sum\limits^\infty_{k=1} C^k_n s^k = (1+s)^k$
    - **减法**：$X-Y$ 的母函数：$E(s^{X-Y}) = E(s^X)E((s^{-1})^Y) = g(s)g(s^{-1})$

### 母函数变形

- **数量随机的独立随机变量之和**：随机变量为 $X$，数量为 $N$，$Y = \sum\limits^N_{j=1} X_j$
  - X的母函数 $g(s) = \sum\limits^\infty_{j=0} s^jP(X=j)$
  - N的母函数 $f(s) = \sum\limits^\infty_{i=1} s^iP(N=i)$
  - Y的母函数 $h(s) = \sum\limits^\infty_{h=0} s^hP(Y=h)$
    - $P(Y=h) = \sum\limits^\infty_{i=1}P(N = i\mid X_1+...+X_i = h)$
    - $h(s) = \sum\limits^\infty_{h=0} P(N=i)\sum\limits^\infty_{i=1}P(X_1+... = h) s^h = \sum\limits^\infty_{h=0} P(N=i) \big[ g(s) \big]^i = f(g(s))$
- **复合定理**
- **复合泊松分布**：$h(s) = e^{\lambda(g(s)-1)}$

### 常用母函数

- $X\sim \mathcal{B}(n,p)$：$g(s) = (1-p + sp)^n$
- $X\sim \mathcal{P}(\lambda)$：$g(s) = e^{\lambda(s-1)}$
- $X\sim G$：$g(s) = \frac{sp}{1-s(1-p)}$


## 特征函数

- **复值随机变量**：$Z=\xi+i\eta$，由两个随机变量组成
  - $EZ = E\xi + iE\eta$
- **特征函数**：$\phi(t) = Ee^{itX} = Ecos(tX) + iEsin(tX)$
  - **Fourier逆变换性**：$\phi(t) = \int^\infty_{-\infty} f(x)e^{itx}dx$
  - **本质**：密度函数视为变换后，则特征函数是原函数，x是Fourier级数序号，t是级数变量
- **逆转公式**：
  -  <font color='chartreuse'>分布函数</font> $\qquad\Leftrightarrow\qquad\quad$ <font color='chartreuse'>特征函数</font>
  - $F(b)-F(a) = \frac{1}{2\pi} \lim\limits_{T\to \infty} \int^T_{-T} \frac{e^{-ita}-e^{-itb}}{it}\phi(t)dt$
  - **证明**：右式做Fourier逆变换，特征函数化为密度函数积分。易得变换后依然可积
  <br>
    - 交换积分得右式 = $\int_R g(T,x,a,b)f(x)dx$
  <br>
    - $g(T,x,a,b) = \large\frac{1}{\pi}\int^T_0 \big[ \frac{\sin [t(x-a)]}{t} - \frac{\sin [t(x-b)]}{t} \big]dt$
    - 由Dirichlet积分，$\lim\limits_{T\to\infty} g = \begin{cases} 0, \quad 其它区间 \\\frac{1}{2}, \quad x=a或b \\1,\quad x\in(a,b) \end{cases}$
    - 用N-L公式计算积分即可
  - **本质**：Fourier变换 + N-L公式
  - **推论**：
    - **唯一性**：由特征函数可确定分布函数（$a\to-\infty,b=x$ 即可）
    - **控制性**：若特征函数绝对可积，则分布函数导数（密度函数）存在且连续（微分不等式得可控 + 控制收敛定理）
- **性质**
  - $|\phi(t)| \leq 1$
    - **证明**
      - 内积不等式：$(Ecos(tX))^2 \leq Ecos^2(tX)$，sin同理
      - 欧拉公式：$|\phi(t)| = |Ecos(tX) + iEsin(tx)|$，两项分别运用内积不等式，化一即可
  - **共轭反号性**：$\phi(-t) = \overline{\phi(t)}$
    - **证明**：欧拉公式
  - **独立线性公式**：$Y=\sum\limits^n_{i=1}X_i，\phi_Y(t) = \prod\limits^n_{i=1}\phi_i(t)$
    - **证明**
      - 归纳法
      - 分解为 $E[e^{itY}e^{itX_n}]$，线性+独立性拆解，得到 $\prod * \phi(n)$
    - **推论（幂封闭性）**：特征函数的正整数幂还是特征函数
  - **一致连续性**：特征函数在R上一致连续
    - **证明**：一致连续定义 + 欧拉公式，分区间讨论 + 积分上界不等式
  - **非负定性**：实数列 $\{t_n\}$ 和复数列 $\{\lambda_n\}$，有 $\sum\limits^n_{k=1}\sum\limits^n_{j=1} \phi(t_k-t_j)\lambda_k\overline{\lambda_j}\geqslant 0$
    - **证明**：乘积分解 + 共轭化模 $E|\sum\limits^n_{k=1} e^{it_kX}\lambda_k|^2$，其非负定
  - **微分性**：若 $X\in L_n$，则特征函数可 $n$ 阶微分，$\phi^{(k)}(0) = i^kEX^k$
- **应用**
  - 比较项数：
    - $\phi(t) = \sum\limits^\infty_{k=0} a_kcos(kt) = \sum\limits^\infty_{k=0} a_k·\frac{e^{ikt} + e^{-ikt}}{2} = \sum\limits^\infty_{k=0} e^{-itX}· P(X=k)$
    - $P(X=k) = \frac{a_k}{2}$
    - 降幂公式、换底公式等
  - 独立线性：
    - 再生性：二项分布、泊松分布、正态分布

### Taylor展开式

- 若
  - $X$ 的任意阶矩存在
  - 存在收敛半径 $t_0$，使得余项 $\lim\limits_{n\to\infty} \dfrac{|t|^nE|X|^n}{n!} = 0，|t|\leqslant t_0$
- 则特征函数可展开为 $\phi(t) = \sum\limits^\infty_{k=0}\dfrac{(it)^k}{k!}EX^k$
  - **证明**：余项极限为0即可
    - 作差 + 逆用期望线性 + 期望绝对值不等式，得到余项

### 判定法则

- **幂封闭性**
- **绝对值封闭性**：$|\phi(t)|^2$ 也是特征函数
  - **证明**：设 $Y\sim X$，$Z = X-Y$，则 $\phi_Z(t) = \phi(t)\overline{\phi(t)} = |\phi(t)|^2$
- **凸组合**：$\{\phi_n(t)\}$ 列和分布律 $\{p_n\}$ 的组合 $f(t) = \sum\limits^\infty_{n=1} p_n\phi_n(t)$
- **凸组合封闭性**
  - **证明**：
    - 特征函数的凸组合与分布函数的凸组合一一对应
      - 由特征函数一致收敛性，交换积分即可
    - 分布函数的凸组合还是分布函数
  - **例子**：随机取下标的随机变量和，其特征函数也是凸组合

### 向量特征函数

- **联合特征函数**：$\phi(t_1,...,t_n) = \large\int^\infty_{-\infty}...\int^\infty_{-\infty} e^{i\sum\limits^n_{k=1} t_kx_k}f(x_1,...,x_n) dx_1...dx_n$

- **独立判别定理**：X与Y独立 $\Leftrightarrow$ 特征函数期望线性 $E^{i(t_1X+t_2Y)} = Ee^{it_1X}Ee^{it_2Y}$

## 依分布收敛

- **随机变量依分布收敛** $X_n\overset{d}{\to} X$：在 $F(x)$ 连续点上 $\lim\limits_{n\to\infty} F_n(x) = F(x)$
  - **函数弱收敛（等价命题）**：$F_n\overset{w}{\to} F$
- **Levy连续定理**：依分布收敛 $\Leftrightarrow \forall t，\lim\limits_{n\to\infty} \phi_n(t) = \phi(t)$
  - **应用**：
    - **正态分布级数的收敛性**：$\lim\limits_{n\to\infty} P(S_n\leq x) = \frac{1}{\sqrt{2\pi}\sigma} \int^x_{-\infty} exp(-\frac{(s-\mu)^2}{2\sigma^2})ds$
      - **证明**：
        - 特征函数具有级数形式
        - 特征函数收敛，故级数收敛
    - 常用极限：$\lim\limits_{n\to\infty} (1-\frac{t^2}{2n}+\omicron(\frac{1}{n}))^n = e^{-\frac{t^2}{2}}$
    - **二项分布收敛到标准正态分布**：
      - 非标准化：$S_n \overset{d}{\to} N(np,np(1-p))$
      - 标准化：$Y_n = \frac{S_n - np}{\sqrt{np(1-p)}} \overset{d}{\to} N(0,1)$
      - **证明**：
        - $Ee^{itY_n} = \big[ (q+pe^{is})e^{-isp} \big]^n$
        - 归并e后Taylor展开，得到下式
        - $ (1-\frac{t^2}{2n}+\omicron(\frac{1}{n}))^n \to e^{-\frac{t^2}{2}}$
        - 特征函数趋近，从而分布也趋近
