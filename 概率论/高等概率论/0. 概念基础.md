## 可测函数

- **概率空间**：满足 $P(X) = 1$ 的测度空间 $(X,\ms F,P)$
  - **事件**：可测集
  - **事件的概率**：可测集的测度
- **随机变量**：a.s.有限的可测函数
- **古典概型**：$(X,\mc P(X), P)$，其中 $P(A) = \dfrac{\#(A)}{\#(X)}$
  - **准分布函数**：$\R$ 上非降右连续实值函数
  - **分布函数**：正负无穷极限为0和1的准分布函数
  - **服从分布 $f\sim F$**：$F(x) = P(f\leq x) \pad (\forall x\in\R)$
- **随机元**：$f:(X,\ms F,P)\to (Y,\ms S)$
  - **随机元的概率分布**：$Pf^{-1}$
  - **本质**：随机变量中，分布 = 概率分布在Borel族中 $\pi$ 系上的值
- **左连续逆**：$F^\lto(t) = \inf\set{x\in\R \mid F(x)\geq t}$
  - **本质**：右侧分位数
  - 左连续性
  - 逆性：$F^\lto(t) \leq x \LR F(x)\geq t$
- **对应定理**：任何分布函数，均存在对应的概率空间和随机变量
