# 统计意义

## 数学期望（加权平均值）

- **总体**：所有随机变量构成的集合
- **总体分布**：总体组合成的随机向量的分布
- **总体平均**：总体的算术平均值 $\mu = \frac{1}{n}\sum\limits^n_{i=1} x_i$
- **随机变量的数学期望（均值）$E(X)$**：
  - **存在条件**：收敛级数 $\sum\limits^\infty_{j=0} |x_j|p_j$（积分同理）
    - 级数收敛是必要条件，所以要加绝对值。它并不是数学期望（积分同理）
    - **数学期望是无穷**：级数不收敛
  - **离散型**：收敛级数 $\sum\limits^\infty_{j=0} x_jp_j$
  - **连续型**：收敛积分 $\int^{+\infty}_{-\infty} xf(x)dx$
  - **唯一性**：随机变量X的概率分布一旦确定，它的数学期望也就可以确定（可以计算）
    - 同分布的随机变量数学期望相同。可以对概率分布定义数学期望
  - **平移意义**：数学期望又称为概率分布的重心
  - **关于点c的k阶矩**：$E(X-c)^k$
    - 原点矩
    - 中心矩
    - **偏度系数**：$\beta = \frac{}{}$
    - **矩空间 $L_r$**：存在r阶矩的所有随机变量
- 数学期望和平均值的关系
  - 数学期望其实就是加权平均值，$x_i$ 是样本数值，$p_i$ 是权重（所占比例/所占数量）
  - 算数平均值是特殊的加权平均值，每个样本的权重相等（概率相等）
- **中位数**：$P(X\leq m) = F(m) = \frac{1}{2}$

### 频率极限

- **数学期望的统计意义**：无限次独立重复试验的收敛平均值 $\overline{X_n} = \frac{\sum X_i}{n}$
  - **证明**：（引论）频率比例的极限 = 概率
    - 则 $\overline{X_n}(n\to\infty) = \sum\limits^{k最大值}_{k=0} k× 频率比例N(概率p_k) = E(X)$
  - **应用**：
    - 柯西分布的数学期望不收敛，平均值发散
    - 柯西分布的绝对值，数学期望无穷，平均值随试验次数增加而递增

### 概率分布的数学期望

- $X\sim \mathcal{B}(1,p)$：$EX = p$
  - **应用**：示性函数服从伯努利分布，所以 $EI[Y\leqslant y] = F_Y(y)$
- $X\sim \mathcal{B}(n,p)$：$EX = np$
  - **证明**：利用 $jC^j_n = nC^{j-1}_{n-1}$ 配凑二项式幂
- $X\sim \mathcal{P}(\lambda)$：$EX=\lambda$
  - **证明**：分离 + 退化 = 配凑“概率和1式”
- $X\sim G(p)$：$EX = \frac{1}{p}$
  - **证明**：逐项求导定理，配凑为级数的导数
- $X\sim \mathcal{E}(\lambda)$：$EX = \frac{1}{\lambda}$
  - 直接计算
- $X\sim H(N,M,n)$：$EX = \frac{nM}{N}$
  - **证明**：抽签问题中，有放回和无放回都是 $EX_i = \frac{M}{N}$，分解成抽签问题，利用期望线性即可
- **对称分布**：$f(\mu+x)=f(\mu-x)$，则 $EX=\mu$
  - **证明**：换元为 $t+\mu$，线性分离为 $\int_R \mu f(x)dx + \int_R tf(t+\mu)dt$
  - **应用**：
    - 正态分布 $EX=\mu$
    - 均匀分布 $U(a,b)\ EX=\frac{a+b}{2}$

### 计算方法

- 非负随机变量的期望要么发散到无穷，要么有界，总之可以计算
- **映射定理**：
  - $E|g(X)| = \int^{+\infty}_{-\infty} |g(x)|f(x)dx$
  - $Eg(X) = \int^{+\infty}_{-\infty} g(x)f(x)dx$（绝对值期望有界时才有意义。若绝对值期望无界，可能发散）
  - **理解**：因为数学期望只和分布整体有关，所以并不需要微分法
- **二维情况以及映射定理**
- **换序公式**：非负随机变量，$EX = \int^{+\infty}_0 P(X>x)dx$
  - **证明**：
    - $x =\int^x_0 dy = \int^{+\infty}_0 I[y<x]dy$
    - 期望公式化为二重积分，累次、换序、组合即可
    - $EX = \int^{+\infty}_{-\infty} xf(x)dx = \int^{+\infty}_0 (\int^{+\infty}_{-\infty} I[x>y]f(x)dx)dy = \int^{+\infty}_0 P(X>y)dy$
  - **离散型证明**：$EX = \sum\limits^\infty_{j=1}\sum\limits^j_{k=1}p_j = \sum\limits^\infty_{k=1}\sum\limits^\infty_{j=k}p_j$ 
  - **理解**：画一个二维下三角表格。前面是横着加，后面是竖着加
    - 横着相加只是数量的累积，也就是给权重以样本值（而不是通常意义的给样本值以权重）
    - 竖着加贯彻了这一特征，还采用了筛选制，一层一层淘汰那些样本值不够的权重
  - **本质**：把x积分化，把单个随机变量二维化

### 性质

- **绝对收敛性（前提）**：$EX存在 \Leftrightarrow E|X|<\infty$
  - （讨论期望性质前，应当首先证明期望的存在性）
  - **线性**：
    - 绝对收敛性：三角不等式
    - 线性公式：积分线性
  - **独立乘法公式**：独立性使得积分区域独立（都为R），体现在公式上就是 $f(x_1,x_2) = f(x_1)f(x_2)$（实际上只要不相关即可）
    - 绝对收敛性：累次积分公式
    - 独立乘法公式：累次积分公式
  - **单调性**：$P(X_1\leqslant X_2) = 1 \longrightarrow EX_1\leqslant EX_2$
    - 令 $Y=X_2-X_1$，利用线性即可
- **应用**：
    - **二维推一维**：$X\sim N(0,1)，EX^2 = \frac{E(X^2+Y^2)}{2} = 1$
    - **分解**：
      - 二项分布分解为n个伯努利分布，超几何分布分解为n个抽签问题
      - 配对问题，分解为 $X_i=第i个元素正确配对$，概率为 $P(X_i=1) = \frac{A^{n-1}_{n-1}}{A^n_n}$，从而利用线性即可
  - $E|X|=0 \longleftrightarrow P(X=0)=1$
    - 以概率1发生：$X=0\ wp1.$(with probability 1)
    - 几乎处处必然：$X = 0\ a.s.$

#### 推论

- **C-R不等式**：若 $X_1,X_2\in L_r$，则 $E|X_1+X_2|^r \leq C_r(E|X_1|^r + E|X_2|^r)$
  - 其中 $C_r = \begin{cases} 1，\quad 0<r<1 \\ 2^{r-1}，\quad r\geqslant 1 \end{cases}$
  - **证明**：
    - $r\geqslant 1$ 时，$x^r$ 是凸函数，从而三角不等式 + 凸函数，得 $|\frac{X_1+X_2}{2}|^r \leqslant \frac{|X_1|^r + |X_2|^r}{2}$，两边取期望即可
    - $0<r<1$ 时，三角不等式 + 取通分 + 放缩，得 $|X_1+X_2|^r \leqslant \frac{|X_1|+|X_2|}{(|X_1|+|X_2|)^{1-r}} \leqslant |X_1|^r+|X_2|^r$
  - **理解**：
- **Jensus不等式**：Borel可测实值凸函数 $g(x)$，若 $EX$ 存在，则 $g(EX) \leqslant Eg(x)$
  - **理解**：期望的本质是均值，和凸函数的定义有联系

## 方差（差的平方的平均值）

- **总体方差（算术方差）**：$\sigma^2 = \frac{1}{m}\sum\limits^m_{i=1} (x_i-\mu)^2$
- **方差（加权方差） $\sigma_{XX}$**：$Var(X) = E(X-\mu)^2$（数学期望收敛）
  - **标准差 $\sigma_X$**：$\sqrt{Var(x)}$
- **唯一性**：由数学期望的唯一性传递过来
- **计算公式**：
  - $Var(X) = E(X^2-2X\mu+\mu^2) = EX^2-(EX)^2$
    - **理解**：方差的展开式，可以看作先算平方期望，再算普通期望，再作差。反正数学期望是唯一的，其具有所有代数性质（结合律、分配律、消去律等），不同计算顺序结果相同
    - **应用**：基础方法，应当首先尝试
  - $Var(X) = E[X(X-1)] + EX - (EX)^2$
    - **理解**：同上，根据唯一性，单纯的配凑，
    - **应用**：配凑幂函数的导数、配凑阶乘时都可以逐级使用
      - $E[X(X-1)] = \sum\limits^\infty_{k=1} k(k-1)·p_k$（映射定理）

### 概率分布的方差


- $X\sim \mathcal{B}(1,p)$：$Var(X) = p(1-p)$
  - **证明**：$X^2=X$，计算公式即可
- $X\sim \mathcal{B}(n,p)$：$Var(X) = np(1-p)$
  - **证明**：$EX^2 = E[X(X-1)] + EX$
    - 映射定理 + 组合数配凑
- $X\sim \mathcal{P}(\lambda)$：$Var(X)=\lambda$
  - **证明**：分离 + 退化即可
- $X\sim G(p)$：$Var(X) = \frac{1-p}{p^2}$
  - **证明**：配凑二阶导数
- $X\sim H(N,M,n)$：$Var(X) = \frac{nM}{N}(1-\frac{M}{N})·\frac{N-n}{N-1}$
  - **证明**：
    - $EX = n\frac{M}{N}$
    - 把每次抽取的事件进行分解，变为伯努利分布的n个加和。
    - $EX_iX_j = P(X_i=1,X_j=1)$，两事件不独立，使用条件概率，得 $\frac{M-1}{N-1}·\frac{M}{N}$
    - 利用非独立线性公式
- $X\sim \mathcal{U}(a,b):Var(X) = \frac{(b-a)^2}{12}$
  - **证明**：直接计算
- $X\sim \mathcal{E}(\lambda)$：$Var(X) = \frac{1}{\lambda^2}$
  - **证明**：直接计算
- $X\sim N(\mu,\sigma^2):Var(X) = \sigma^2$
  - **证明**：$EY^2 = 1$ 已知，利用映射即可

### 性质

- **阶性**：$Var(a+bX) = b^2Var(X)$
  - 数学期望计算即可得到
- **最小性**：$Var(X) = E(X-\mu)^2 < E(X-c)^2（c\neq \mu）$
  - 数学期望添项分离 + 平方非负性
  - **理解**：平均数的平移意义 + 对称性 = 奇函数积分归零
- **稳定意义**：$Var(X)=0 \longleftrightarrow P(X=\mu) = 1$
  - 平方非负性 + 非负=0则必然
- **非独立加法公式**：$Var(\sum\limits^n_{j=1} X_j) = \sum\limits^n_{i=1}\sum\limits^n_{j=1} \big[ E(X_iX_j) - \mu_i\mu_j \big] = \sum\limits^n_{j=1}Var(X_j) - 2\sum\limits_{i\neq j} E(X_iX_j-\mu_i\mu_j)$
  - **证明**：利用 方差普通计算公式 + 数学期望线性
  - **理解**：前项是方差线性公式，后项是协方差。（协方差是方差线性公式的余项，即非独立部分）
  - **本质**：概率的加法公式，本质就是集合的加法公式，任何集合都有类似的式子
- **独立线性**：$Var(\sum\limits^n_{j=1} X_j) = \sum\limits^n_{j=1} Var(X_j)$
  - **证明**：若独立，则只有 $EX_j^2 - (EX_j)^2$ 不为0，而该项结果为 $Var(EX_j)$
  - **理解**：定义域互不干扰，证明同数学期望
    - 实际上只需要不相关即可。这里是充分条件
- **标准化**：$Y = \frac{X-EX}{\sigma_X}$，此时 期望为0，方差为1
- **独立方差平均值**：相互独立，且共同方差的随机变量，$Var(\frac{1}{n}\sum\limits^n_{j=1}X_j) = \frac{n}{n^2}\sigma^2 = \frac{1}{n}\sigma^2$
- **无偏估计**：有放回和无放回的抽样，数学期望都相同 $EX=np$，$E\tilde{p} = p$，则 $\tilde{p}$ 是 $p$ 的无偏估计
  - 超几何分布和二项分布的数学期望满足无偏估计

## 二维相关性

- **随机变量的内积**：$\lang X,Y\rang = E(XY)$
- **内积不等式**：$|E(XY)| \leqslant \sqrt{EX^2EY^2}$
  - **证明**：类似向量的构造平方并使用非负性
    - 构造 $E(aX+bY)^2 = \begin{pmatrix} a,b \end{pmatrix}\begin{pmatrix} EX^2 & E(XY) \\ E(XY) & EY^2 \end{pmatrix}\begin{pmatrix} a\\b \end{pmatrix}$，是二次型
    - 平方的数学期望非负，因此 $\Sigma$ 满足非负定性，从而行列式 $\geqslant 0$，移项得到结果
  - **推论**：
    - 等号成立则 $\exist a,b\neq 0，E(aX+bY)^2=0 \longleftrightarrow P(aX+bY=0) = 1$
    - $EX^2< \infty\Rightarrow E|X|< \infty$（添项E1）（或利用内积不等式）
- 独立同分布的随机变量
  - **样本均值（算术平均值）**：$\tilde{\mu} = \frac{1}{n}\sum\limits^n_{j=1} X_j$
    - 样本均值期望 $E\tilde{\mu} = \mu$
  - **样本方差**：$\tilde{\sigma}^2 = \frac{1}{n-1}\sum\limits^n_{j=1} (X_j-\tilde{\mu})^2$
    - 样本方差期望 $E\tilde{\sigma}^2 = \sigma^2$
    - $E\tilde{\sigma} < \sigma$（添项E1）

### 协方差

- 本节中使用的是标准差
- **协方差 $\sigma_{XY}$**：$Cov(X,Y) = E[(X-\mu_X)(Y-\mu_Y)]$
  - 还是二次，但是由两个一次混合的
- **计算公式**：$\sigma_{XY} = E(XY) - (EX)(EY)$
  - **理解**：
    - 数学期望独立乘法公式的余式
    - 方差线性的余式
  - **应用**：普通公式

### 线性相关性

- **相关系数 $\rho_{XY}$**：$ \rho(X,Y)= \frac{\sigma_{XY}}{\sigma_X\sigma_Y} =（变换矩阵的余弦） \frac{\overrightarrow{X}·\overrightarrow{Y}}{|\overrightarrow{X}||\overrightarrow{Y}|}$
  - **性质**
    - XY不相关：$\sigma_{XY}=0$
    - **标准化性 + 有界性**：$|\rho| = E(\frac{X-\mu_X}{\sigma_X})(\frac{Y-\mu_Y}{\sigma_Y})\leqslant 1$
      - **证明**：$E(\frac{X-\mu_X}{Var(X)})^2 = 1$，利用内积不等式
    - **融合性**：$|\rho| = 1 \longleftrightarrow P(Y=a+bX)=1 \longleftrightarrow E(XY) = \sqrt{EX^2EY^2}$
    - $\rho = 1 \overset{标准化后同分布}{\Longleftarrow}$ 线性相关 $\overset{融合性}{\Longrightarrow}$ 内积不等式 $\overset{取等条件}{\Longrightarrow}$ 标准协方差
      - 融合为单变量，内积不等式自然就取等号
      - <font color='red'>相关系数 $\rho$ </font>是两个随机变量标准化后的<font color='red'>协方差</font>
        - $\rho$ 不是变换矩阵的角度余弦，因为前面的二维正态分布中，标准化后 $Y_1,Y_2$ 同分布，但 $(X,Y)$ 标准化后也不同分布
  - **理解**：
    - 随机变量角度（融合性），$\rho$ 为 $\frac{EXY-(EXEY)}{(EX^2-EX)(EY^2-EY)}$，若Y是X线性，则可融合，变为1，否则不是1
    - <font color="chartreuse">随机向量角度（夹角性），$\rho$ 为标准向量 $S = (\frac{X-\mu_X}{\sigma_X},\frac{Y-\mu_Y}{\sigma_Y})$ 的变换阵 $\\ (X,Y) = SA + (\mu_X,\mu_Y)$ 的余弦 </font>
- **线性相关性**：$\exist a_i\neq 0，P(\sum\limits^n_{i=1}a_iX_i = 0) = 1$
  - $\rho$ 只表示线性相关性
- 独立情况 $\subset$ 不相关情况
  - 独立条件：可分解（单点性质）
  - 不相关条件：协方差公式（总体性质）
    - 定义域不独立：圆内均匀分布，对称性，$EX=EY=0，E(XY)=0$

### 小总结

- 内积不等式的意义是协方差，取等条件是线性相关。因此出现了由协方差定义的相关系数
- 协方差是一次变量乘积XY，因此也只能表示线性的相关性
  - 实际上，还有二次相关性和其它类型的相关性。它们需要更复杂的乘积定义

### 习题

- 不相关的随机变量列 $X_i$，若数学期望和方差均相同，则
  - $E(S_n) = n\mu，Var(S_n) = n\sigma^2$
  - $E(\frac{S_n}{n}) = \mu，Var(S_n) = \frac{\sigma^2}{n}$
  - $T_n = \sum (-1)^{i+1}X_i，E(T_n) = \begin{cases} 0，n=2m \\ \mu，n=2m+1 \end{cases}，Var(T_n) = n\mu^2$
- 二值随机变量不相关则独立
  - **证明**：设 $\xi:a,b，\eta:c,d$
    - **计算法**：$E(AB) = E(A)E(B)$，即 $acp_1p_2+... = [ap_1+b(1-p_1)][cp_2+d(1-p_2)]$
    - **变换法**：示性函数 $\delta_A = \frac{\xi-b}{a-b}，\delta_B = \frac{\eta-d}{c-d}$，而示性函数的协方差为 $P(AB)-P(A)P(B)$，从而相关=独立

## 随机矩阵的期望和方差

- 为了区分，规定：小写是向量，大写是矩阵
  - 维度在MATLAB中，指矩阵的xyz轴数量
  - 在线代中，指矩阵的行列数
- **期望的内含性**：$EY = \begin{pmatrix} EY_{11} & EY_{12} \\ EY_{21} & EY_{22} \end{pmatrix}$
- **期望的线性**： $\begin{cases} E(aX^T) = aEX^T \\ E(AYB) = AE(Y)B \\ (EY)^T = E(Y^T) \end{cases}$（内含性即可）
- **二维协方差矩阵**：$\Sigma = E\big[ (X-\overrightarrow{\mu})^T(X-\overrightarrow{\mu}) \big]$
  - 内含性：无论先算还是后算，都是一个结果（结合律、唯一性等）（线性变换对于基也有这个性质）
  - 矩阵对称性
  - 矩阵半正定性：构造二次型，$a\Sigma a^T = E\big[ \sum\limits^2_{i=1}a_i(X_i-\overrightarrow{\mu}_i)^2 \big] \geqslant 0$
  - 矩阵退化 $\Leftrightarrow$ $P(\sum\limits^2_{i=1}a_i(X-\overrightarrow{\mu}_i)=0)=1$ $\Leftrightarrow$ $X_1,X_2$ 线性相关

### 二维正态分布

- 协方差矩阵 $\Sigma = \begin{pmatrix} \sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2 \end{pmatrix} = \begin{pmatrix} Var(X) & \sigma_{XY} \\ \sigma_{XY} & Var(Y)\end{pmatrix}$
  - $\sigma^2$ 是方差，$\rho$ 是线性相关系数
    - **证明**：
      - 标准正态分布 $Y\sim N(\bold{0},I)$，一般正态分布 $X=YA+\overrightarrow{\mu}$
      - $\begin{cases} EX = \overrightarrow{\mu} ，（X的期望）\\ EY = 0 ，（Y的期望）\\ E(Y^TY) = 1 ，（Y的协方差）\end{cases}$
      - $\Sigma = E[(X-\overrightarrow{\mu})^T(X-\overrightarrow{\mu})] = A^TE(Y^TY)A = \Sigma$
        - 协方差矩阵 $\Leftrightarrow$ 逆标准化矩阵
        - 对应前面的标准化和协方差的关系
        - **推论**：$X_1、X_2$ 独立 $\Leftrightarrow$ 线性不相关
          - **粗俗理解**：正态分布定义域独立
          - **证明**：变换阵A是对角阵 $\Leftrightarrow \Sigma$ 是对角阵
