# Newton型迭代法

## 线性化方法

- **方程组的线性化方法**：
  - 设已知方程组 $F(\bs x) = \bs 0$
  - 定义仿射映像 $L_k:\R^n\to \R^n，\bs x\mapsto A_k(\bs x-\bs x^k) + F(\bs x^k)$
  - 将 $L_k(\bs x) = \bs 0$ 的解作为原方程组的新近似 $\bs x^{k+1}$，就是线性化迭代法
- **平行弦方法**：将所有 $A_k$ 都取为非奇异常数方阵 $A$
  - <font color='chartreuse'>维数 $n=1$ 时，它以过曲线上某点 $\Big( x_k,f(x_k) \Big)$ 的斜率固定为 $A$ 的直线与 $x$ 轴的交点为新迭代点 $x_{k+1}$
  - 维数 $n=2$ 时，它以过曲面上某点 $\Big( \bs x_k,f(\bs x_k) \Big)$ 的（由 $A$ 的值决定的两平面）（在 $xy$ 平面上的交线的交点）为新迭代点 $\bs x_{k+1}$</font>
- **牛顿法**：将 $A_k$ 取为 $\Big[ F'(\bs x^k) \Big]^{-1}$
- **牛顿型迭代法**：由于求逆比较困难，故采用各种方法去近似 $\Big[ F'(\bs x^k) \Big]^{-1}$


### 牛顿法的收敛性

- **牛顿迭代函数**：$G(\bs x) = \bs x-\Big[ A(\bs x) \Big]^{-1}F(\bs x)$
- ****：
  - 设 $F:D\subset \R^n\to \R^n$ 在 $\bs x^*$ 处F可导，$\bs x^*$ 是方程的解，$S_0$ 是其邻域
  - 若 $A:S_0\to L(\R^n)$ 在 $\bs x^*$ 处连续，$A(\bs x^*)$ 非奇异
  - 则存在闭球 $S = \ol S(\bs x^*,\d)\subset S_0$ 使得 $G$ 在 $S$ 上有定义，且在 $\bs x^*$ 处有F导数 $$G'(\bs x^*) = I-\Big[ A(\bs x^*) \Big]^{-1} F'(\bs x^*)$$
  - **推论**：若还有 $\rho(I-\Big[ A(\bs x^*) \Big]^{-1} - F'(\bs x^*)) < 1$，则 $\bs x^*$ 是迭代序列的吸引点
- **局部收敛定理**：
  - 设 $F:D\subset \R^n\to \R^n$ 在 $\bs x^*$ 的开邻域处F可导，$\bs x^*$ 是方程的解
  - 若 $F'(\bs x^*)$ 非奇异
  - 则存在闭球 $S = \ol S(\bs x^*,\d)\subset S_0$ 使得 $G$ 在 $S$ 上有定义，且牛顿迭代法超线性收敛到 $\bs x^*$
  - **推论**：
    - 若 $F'$ 还满足压缩性条件，则牛顿迭代法至少二阶收敛 $$\forall \bs x\in S，\|F'(\bs x) - F'(\bs x^*)\| \leq \a\|\bs x-\bs x^*\|$$
      - **证明**：
    - 若 $F$ 在 $\bs x^*$ 处二阶F可导，$\forall \|\bs h\|\neq 0$ 都有 $F''(\bs x^*)\bs h\bs h \neq 0$，则牛顿迭代法二阶收敛
      - **证明**：
- **简化迭代公式**：<font color='chartreuse'>如果完全沿用一维情况的牛顿法迭代公式，那么需要每一次迭代都需要求Jacobi矩阵的逆，计算量过大。因此我们直接用线性方程组对其替代</font> $$ \begin{cases} \bs x^{k+1} = \bs x^k + \D \bs x^k \\\\ F(\bs x^k) + F'(\bs x^k) \D \bs x^k = 0\end{cases} $$

### 习题

- **超线性收敛的必要条件**：
  - 设序列超线性收敛到 $\bs x^*$，且总有 $\bs x^k\neq \bs x^*$，则 $\lim\limits_{k\to\infty} \cfrac{\|\bs x^{k+1} - \bs x^k\|}{\|\bs x^k - \bs x^*\|} = 1$
    - **证明**：
  - **非充分反例**：下面的序列收敛到 $\bs x^* = 0$，且成立定理条件。但并不超线性收敛 $$\begin{cases} \bs x^{2i-1} = \cfrac{1}{i!} \\\\ \bs x^{2i} = \bs x^{2i-1} \end{cases}$$

## 牛顿法的变型

### 修正牛顿法

- **思想**：希望综合简化牛顿法的计算简便与牛顿法的快速收敛两个优点
- **迭代序列**：每一步迭代中，做 $m$ 次简化牛顿迭代 $$\begin{cases} \bs x^{k,0} = \bs x^k \\\\ \bs x^{k,i} = \bs x^{k,i-1} - \Big[ F'(\bs x^k) \Big]^{-1}F(\bs x^{k,i-1}) & 1\leq i\leq m\\\\ \bs x^{k+1} = \bs x^{k,m} \end{cases}$$
  - <font color='chartreuse'>每步的内循环共用一个逆矩阵 $F'(\bs x^k)^{-1}$，但用不同的函数值矩阵 $F(\bs x^{k,i-1})$</font>
- **计算量比较**：
  - 牛顿法中，每一步求 $F$ 需要计算 $n$ 次 $f_i$，求 $F'$ 需要计算 $n^2$ 次 $f_i'$，还要再求一次逆
  - 简化牛顿法中，只需求一次即可应用到所有迭代步中
  - 修正牛顿法中，虽然计算量变大了，但实际的计算效率是提高的
- **收敛速度定理（3阶）**：
  - 设 $F:D\subset \R^n\to \R^n$ 在 $\bs x^*$ 的邻域 $S$ 中F可导，$F'(\bs x^*)$ 非奇异
  - 当取 $m=2$ 时，若 $\forall \bs x\in S$，都存在 $\g > 0$ 满足下式，则 $\bs x^*$ 是迭代序列的吸引点 $$\|F'(\bs x^*) - F'(\bs x^*)\| \leq \g \|\bs x-\bs x^*\|$$
  - **估计式**：$\|\bs x^{k+1} - \bs x^*\| \leq c\|\bs x^k - \bs x^*\|^3$<font color='chartreuse'>（具有三阶收敛速度）</font>
  - **推论**：修正牛顿法的收敛速度为 $m+1$ 阶
    - **证明**：
- **效率比较**：
  - 设计算 $f_i(\bs x^k)$ 和 $\pa_i f_i(\bs x^k)$ 的工作量均为 $\dfrac{1}{n}$，求逆工作量为 $v$
  - 牛顿法每步的工作量为 $W = n+1+v$，收敛阶为 $p = 2$，从而效率为 $e = \cfrac{\ln 2}{n+1+v}$
  - 修正牛顿法每步的工作量为 $W = n+m+v$，收敛阶为 $p = m+1$，从而效率为 $e = \cfrac{\ln (m+1)}{n+m+v}$
  - <font color='chartreuse'>若不考虑 $v$，则易得 $m\geq 2$ 时修正牛顿法效率更高</font>


### 带参数的牛顿法

#### 阻尼因子

- **思想**：若出现 $F'(\bs x^*)$ 奇异或病态时，引入阻尼因子 $\l_k$ 将其变为非奇异阵
- **迭代序列**：$$\bs x^{k+1} = \bs x^k - \Big[ F'(\bs x^k) + \l_k I \Big]^{-1} F(\bs x^k)$$
- **局部收敛定理**：
  - 设 $F:D\subset \R^n\to \R^n$ 在 $\bs x^*$ 的邻域F连续可微，$F'(\bs x^k)$ 非奇异
  - 设 $\mu_1,...,\mu_n$ 是矩阵 $F'(\bs x^*)$ 的特征值，$$\b = \min\limits_i \hkh{\cfrac{|\mu_i|^2}{2\text{Re}\mu_i} \mid \text{Re}\mu_i > 0} \\\ \\ \eta = \min\limits_i \hkh{\cfrac{|\mu_i|^2}{-2\text{Re}\mu_i} \mid \text{Re}\mu_i < 0} $$（若定义不存在，则设为正无穷）
  - 则对任意 $\l \in (-\b,\eta)$，参数牛顿法的迭代序列均在 $S_0$ 中，且收敛到 $\bs x^*$
  - **证明**：

#### 下降程序

- **思想**：已知当 $|f(\bs x^k)|$ 单调递减时，牛顿法收敛。故引入下降参数 $w_k$ 使其满足单调性，即可将收敛范围扩大
- **迭代公式**：$$\bs x^{k+1} = \bs x^k - w_k\Big[ F'(\bs x^k) \Big]^{-1} F(\bs x^k)$$
- **大范围收敛定理**：
  - 设 $F:D\subset \R^n\to \R^n$ 满足
    - **有界性**：$\|F(\bs x^0)\| \leq \eta$
    - **连续可微性**：在区域 $D_0 = \{\bs x: \|\bs x - \bs x^0\| \leq \rho\b\eta\}$ 上F可导，且导数可逆
    - **导数有界性**：在区域 $D_0$ 上满足 $$\|\Big[ F'(\bs x) \Big]^{-1}\| \leq \b \\ \ \\   \forall \bs x,\bs y\in D_0，\|F'(\bs x) - F'(\bs y)\| \leq \g\|\bs x-\bs y\|$$
  - 则
    - 当 $\rho \geq \cfrac{2\a}{\e^2} > 1$ 时，方程在 $D$ 中有解
    - 当 $\begin{cases} w_k \in (0,1] \\\\ 0<\e \leq w_k\a \leq 2-\e \\\\ \a = \g\b^2\eta \end{cases}$ 时，序列收敛到解
    - **证明**：

## 牛顿松弛型迭代法

- **思想**：将牛顿法迭代公式写成差分形式，再对差分引入松弛因子 $w$，起到加速作用
- **牛顿方程组**：$F'(\bs x^k)\D \bs x^k = -F(\bs x^k)$
  - <font color='chartreuse'>将牛顿迭代公式变形为线性差分方程组
  - 之前的方法都是直接求逆解方程组，但实际上还可用LU分解或QR分解等方法
  - 如果用迭代法解方程，就会出现双层迭代。外层是非线性的，内层是线性的</font>
- **非线性-线性迭代**：
  - 设 $A_k = B_k-C_k$，将 $C_k$ 移到右边，设 $H_k = B_k^{-1}C_k$
  - 内层迭代公式为 $$\bs x^{k,m+1} = H_k\bs x^{k,m} + B_k^{-1}b_k$$
  - 外层迭代公式为 $$\bs x^{k,m+1} = H^{m_k}_k\bs x^{k,m} + B_k^{-1}b_k$$
  - 总迭代公式为 $$\bs x^{k+1} = \bs x^k - \Big( H^{m_k-1}_k + \cdots + H_k + I \Big)B_k^{-1} F(\bs x^k)$$
    - **证明**：
      - 将外层迭代公式递推，再利用 $B_k^{-1}A_k = I-H_k$ 和 $(I-H_k)^{-1}$ 的泰勒展开式，变形即可
- **符号约定**：
  - $D(\bs x)，L(\bs x)，U(\bs x)$ 分别是 $F'(\bs x)$ 的对角阵、严格上三角阵、严格下三角阵


### N-SOR迭代法

- **思想**：
  - 将牛顿方程组写成形式 $A_k\bs x^{k+1} = b_k$，其中 $\begin{cases} A_k = F'(\bs x^k) \\\\ b_k = F'(\bs x^k)\bs x^k - F(\bs x^k) \end{cases}$
  - 然后对该线性方程组用SOR迭代法求解
- **N-Jacobi迭代**：
  - 设 $A_k = D_k-L_k-U_k$，取 $B_k = D_k，C_k = L_k+U_k$
  - 则此时 $H_k = D^{-1}_k (L_k+U_k)$  
  - 总迭代公式为 $$\bs x^{k+1} = \bs x^k - \Big( H^{m_k-1}_k + \cdots + H_k + I \Big)D_k^{-1} F(\bs x^k)$$
- **N-Seidel迭代**：
  - 设 $A_k = D_k-L_k-U_k$，取 $B_k = D_k-L_k，C_k = U_k$
  - 则此时 $H_k = (D_k-L_k)^{-1}_kU_k$ 
  - 总迭代公式为 $$\bs x^{k+1} = \bs x^k - \Big( H^{m_k-1}_k + \cdots + H_k + I \Big)(D_k-L_k)^{-1} F(\bs x^k)$$
- **N-SOR迭代**：
  - 设 $B_k = \cfrac{1}{w_k}\Big[ D_k-w_kL_k \Big]，C_k = \cfrac{1}{w_k}\Big[ (1-w_k)D_k + w_kU_k \Big]$
  - 则此时 $H_k = (D_k-w_kL_k)^{-1}\Big[ (1-w_k)D_k + w_kU_k \Big]$
  - 总迭代公式为 $$\bs x^{k+1} = \bs x^k - w_k\Big( H^{m_k-1}_k + \cdots + H_k + I \Big)(D_k - w_kL_k )^{-1} F(\bs x^k)$$
- **m步N-SOR迭代法**：取松弛因子 $w_k$ 和内循环次数 $m_k$ 均为常数 $w$ 和 $m$
- **m步N-SOR迭代法的收敛性定理**：
  - 设
    - $\bs x^*$ 是 $F(\bs x) = 0$ 的解
    - $F:D\subset \R^n\to \R^n$ 在点 $\bs x^*$ 的邻域 $S_0\subset D$ 内连续可导
    -  $F(\bs x^*)$ 非奇异
  - 若 $\rho\Big( H(\bs x^*) \Big) < 1$ 且 $w > 0$，则序列收敛到 $\bs x^*$，收敛因子 $\a = \rho\Big( H(\bs x^*)^m \Big)$
  - **证明**：
  - **推论**：
    - （m步N-SOR迭代）相当于（1步N-SOR迭代）的m步
    - 再由于1步迭代不需要计算上三角阵，故所以容易计算导数时，用1步更好

### SOR-N迭代法

- **思想**：先用SOR迭代计算非线性方程组，然后用牛顿法求解迭代公式
  - <font color='chartreuse'>非线性Gauss-Seidel迭代是什么意思呢？
    - 实际上线性方程组的G-S迭代法远远不止分解矩阵这么简单
    - 它实际上是把矩阵 $A$ 分解成了严格上三角 $U$ 和下三角 $D-L$ 两个部分
    - 然后在具体计算时，先用 $\bs x_{k-1}$ 求出第一个分量 $\bs x^{(1)}_k$，再用 $\bs x^{(1)}_k$ 和 $\bs x_{k-1}$ 求出第二个分量 $\bs x^{(2)}_k$，这样逐次进行下去
    - 由于是线性方程组，所以上述步骤可以直接写成矩阵乘积的形式。但它的本质其实还是用 $\bs x_{k-1}$ 的分量去逐步递推 $\bs x_{k}$
    - 所以也可以将这种思想应用到非线性方程组上。只是不能写成矩阵乘积的形式而已</font>
- **G-S-N迭代**：把非线性方程组 $F(\bs x) = 0$ 转化为求 $n$ 个一维非线性方程，然后用牛顿法分别求解
- **SOR-N迭代法**：对G-S-N迭代法引入松弛因子，然后用牛顿法分别求解
- **1步SOR-N迭代法的局部收敛定理**：
  - 设
    - $\bs x^*$ 是方程的解
    - $F:D\subset \R^n\to \R^n$ 在 $\bs x^*$ 的开邻域 $S_0$ 内连续可导
    - $D(\bs x^*)$ 非奇异
  - 若 $\rho\Big( H(\bs x^*) \Big) < 1$ 且 $w = 0$，则序列在开球 $S(\bs x^*,\d)\subset S_0$ 内有定义，且 $\bs x^*$ 是序列的吸引点
