- **误差限运算公式**：
  - 设两个近似数 $x_1,x_2$ 的误差限分别是 $\e(x_1)$ 和 $\e(x_2)$，则
  - **加法**：$\e(x_1+x_2) \leq \e(x_1) + \e(x_2)$
  - **乘法**：$\e(x_1x_2) \leq |x_1|\e(x_2) + |x_2|\e(x_1)$
  - **除法**：$\e\left( \cfrac{x_1}{x_2} \right) \leq \cfrac{|x_1|\e(x_2) + |x_2|\e(x_1)}{|x_2|^2}$
