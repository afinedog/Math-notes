# 样条函数

- **思想**：整体插值有很多问题，所以提出了分段插值的概念
    - 整体插值会出现多余的拐点，即龙格现象
    - 多项式可以通过一点确定整体，但大多数函数却不是这样
    - 综上，我们分段取不同的多项式逼近，可以解决上述问题
    - <font color='chartreuse'>从几何上来看，给出一堆高度不同的竖直木条（即插值点），用一整根木条压在上面时，两端非常容易压不住翘起来（龙格现象）。所以考虑用分段小木条分别压上去，即样条函数</font>
- **符号约定**：
  - $\mc P_n$ 表示所有不超过 $n$ 次的多项式

## 样条函数的定义

- **样条函数（分段光滑逼近）**：
  - 给定一组节点 $-\infty = x_0 < x_1 < ... < x_{N+1} = +\infty$
  - 设分段函数 $s(t)$ 满足
    - **分段多项式性**：$\forall [x_j,x_{j+1}]$ 中，$s(t)$ 是至多 $n$ 次的实系数代数多项式
    - **连续可微性**：$s(x)$ 在 $\R$ 上有 $n-1$ 阶连续导数
  - 则称 $s(t)$ 为 $n$ 次样条函数
  - <font color='chartreuse'>样条是工业上使用的一种弹性木条。当已知部分节点，需要绘画光滑曲线时，就用样条连接每两个节点，这样就能构造出一个整体光滑的分段函数
  - 一般的分段插值公式在节点处连续但不一定光滑，但样条函数可以满足这一点</font>
- **样条节点**：$\{x_i\}^N_{i=0}$
- **$n$ 次样条函数总体**：$S_n(x_1,...,x_N)$

- **自然样条函数**：
  - 若 $2n-1$ 次样条函数 $s(x)$ 在 $(-\infty,x_1]$ 和 $[x_N,+\infty)$ 都是 $n-1$ 次多项式
  - 则称为 $2n-1$ 次自然样条函数
- **$2n-1$ 次自然样条函数总体**：$\mc N_{2n-1}(x_1,...,x_N)$


### 样条函数的数学表达式

- **截断多项式**：$x^n_+ = \begin{cases} x^n & x > 0 \\ 0 & x \leq 0 \end{cases}$
  - <font color='chartreuse'>即仅支撑在正半轴上的多项式</font>
- **样条函数的一般形式**：$s(x) = p_n(x) + \sum\limits^N_{j=1} c_j(x-x_j)^n_{+}，x\in \R$
  - **证明**：
    - 设 $(-\infty,x_1]$ 上是 $n$ 次多项式 $p_n(x)$，在 $[x_1,x_2]$ 上是 $n$ 次多项式 $q_n(x)$
    - 设 $\eta(x) = q_n(x)-p_n(x)$
      - 由样条函数连续可微性，$\forall \eta^{(i)}(x_1) = 0$，从而 $(x-x_1)^n\mid \eta(x)$
      - 再由 $\eta$ 为至多 $n$ 次多项式，故 $\eta(x) = c_1(x-x_1)^n$
    - 从而 $s(x) = \begin{cases} p_n(x) & x\in (-\infty,x_1] \\ p_n(x) + c_1(x-x_1)^n  & x\in (x_1,x_2] \end{cases}$
    - 不断递推即可
- **样条函数判定定理**：$s(x)\in \mc S_n \LR \exists p_n(x)\in \mc P_n$ 和 $c_1,...,c_N\in\R$ 使得成立上式
- **自然样条函数的一般形式**：$s(x) =  p_{n-1}(x) + \sum\limits^N_{j=1} c_j (x-x_j)^{2n-1}_{+}$
  - 其中 $\sum\limits^N_{j=1} c_jx_j^k = 0，k=0,1,...,n-1$
  - **证明**：
    - 由样条函数的一般形式，以及自然样条函数的定义，其在 $[x_N,+\infty)$ 上可写成 $s(x) = p_{n-1}(x) + \sum\limits^N_{j=1} c_j(x-x_j)^{2n-1}_+$
    - <font color='chartreuse'>接下来要使它为 $n-1$ 次多项式，显然必须令求和式中 $n$ 次以上系数为 $0$</font>
    - 将求和式根据二项式展开，然后改变求和顺序，最终可变为 $$\sum\limits^{2n-1}_{i=0} (-1)^{i-1}C^i_{2n-1} x^i\sum\limits^N_{j=1} c_jx_j^{2n-1-i}$$
    - 从而只需题设的 $c_j$ 条件成立，就能得到自然性质
- **自然样条函数判定定理**：$s(x)\in \mc N_{2n-1} \LR  \exists p_n(x)\in \mc P_n$ 和 $c_1,...,c_N\in\R$ 使得成立上式
