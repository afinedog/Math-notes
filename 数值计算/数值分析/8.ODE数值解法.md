# 常微分方程初值问题的数值解法

## 基本知识

- **方程数值解法的分类**
  - **近似解析方法**：给出解的近似表达式
    - 幂级数解法
    - 傅里叶级数解法
    - 皮卡序列逼近法
  - **数值方法**：给出解在离散点上的近似值
- **常微分方程初值问题（空间格式）**
  - 设 $f(x,y)$ 在区域 $G: x\in [0,T]，|y|<\infty$ 上连续，且关于 $y$ 满足Lip条件
  - 求满足 $\begin{cases} \cfrac{dy}{dx} = f(x,y) \\\\ y(0) = y_0 \end{cases}$ 的函数 $y(x)$
- **常微分方程初值问题（时间格式）**：将 $y$ 写成 $u$，将 $x$ 写成 $t$
  - <font color='chartreuse'>在ODE中区别不大，但在PDE中具有特定物理意义，需要区分</font>

## 线性单步法

- <font color='red'>本章假设区间被 $n$ 等分</font>
- **步长**：每个等分区间的长度 $h = \dfrac{T}{N}$
- **节点**：点列 $x_n = x_0 + nh$
- **线性单步法**：
  - **线性**：迭代公式关于 $f_j$ 线性
    - <font color='chartreuse'>如果出现中间点（比如 $f_{j+\frac{1}{2}}$），那也不是线性的</font>
  - **单步**：每次计算时只需依赖前一步的值
    <!-- - <font color='chartreuse'>迭代方向比较自由，$y_n$ 的前一步可能是 $y_{n-1}$ 也可能是 $y_{n+1}$</font> -->
- **迭代公式（数值积分形式）**：$y_{n+1} = y_n + h\p(x_n,y_n,y_{n+1},h)$，其中 $\p$ 称为增量函数
  - <font color='chartreuse'>增量函数 $\p$ 本质是对 $f$ 的离散模拟，上式本质是用数值积分公式将ODE积分形式离散化</font>
  - **显式格式**：$\p$ 中不包含 $y_{n+1}$，可以直接递推
  - **隐式格式**：$\p$ 中包含 $y_{n+1}$，需要求解方程
- **迭代公式（数值微分形式）**：$\cfrac{y_{n+1}-y_n}{h} = \p$
  - <font color='chartreuse'>这里是用数值微分公式对ODE微分形式离散化。变形即得数值积分形式
  - 从该形式可以发现，一阶ODE中增量 $\p$ 需要乘 $h$，二阶ODE中增量 $\p$ 需要乘 $h^2$</font>

### 欧拉法

- **Euler法**：
  - **步骤**：
    - 由初值条件，可求出初值点的导数值 $f(x_0,y_0)$
    - 将泰勒展开式约去二阶小量，可求出 $y_1 = y_0 + hf(x_0,y_0)$
    - 不断迭代下去即可
  - **递推公式**：$y_{n+1} = y_n + hf(x_n,y_n)$
    - <font color='chartreuse'>$f$ 就是解函数 $y$ 的导数</font>
  - **几何意义**：
    - 实际上就是常微分方程中的欧拉折线。等分区间取的越小，逼近效果越良好
    - 欧拉折线是用差商去近似导数，从而以直代曲的

### 隐式欧拉法

- **隐式欧拉法（后退的欧拉法）**：使用向后差分公式替代微分
- **迭代公式**：$y_{n+1} = y_n + hf(x_{n+1},y_{n+1})$
  - <font color='chartreuse'>计算顺序是从右往左，需要解函数方程，不好</font>

### 梯形方法

- **梯形方法**：使用数值积分的梯形公式
- **迭代公式**：$y_{n+1} = y_n + \cfrac{h}{2}\Big[ f(x_n,y_n) + f(x_{n+1},y_{n+1}) \Big]$

### 综合方法

- **改进的Euler法**：梯形公式 + 预估校正法
  - **积分迭代**： $y(x_{n+1}) = y(x_n) + \dis\int^{x_1}_{x_0} f\Big( t,y(t) \Big)dt$
    - <font color='chartreuse'>这个公式需要计算积分，故仍需改进</font>
  - **梯形迭代**：$y_{n+1} = y_n + \cfrac{h}{2}\Big[ f(x_n,y_n) + f(x_{n+1},y_{n+1}) \Big]$
    - <font color='chartreuse'>这个公式需要求解函数方程，故仍需改进</font>
  - **方程迭代**：$y_{n+1}^{[k+1]} = y_n + \cfrac{h}{2}\Big[ f(x_n,y_n) + f(x_{n+1},y_{n+1}^{[k]}) \Big]$，取初值为 $y_{n+1}^{[0]} = y_n$
    - <font color='chartreuse'>一般几步即可收敛到 $y_{n+1}$</font>
- **最终迭代公式（预报-校正格式）**：$$ \begin{cases} \ol y_{n+1} = y_n + hf(x_n,y_n) & 预报值 \\\\ y_{n+1} = y_n + \cfrac{h}{2}\Big[ f(x_n,y_n) + f(x_{n+1},\ol y_{n+1}) \Big] & 校正值 \end{cases}$$
  - <font color='chartreuse'>首先用欧拉方法，得到一个近似的预报值
  - 然后将预报值代入梯形方法，则得到的精度会比原来更高</font>
- **最终迭代公式（嵌套格式）**：$$ y_{n+1} = y_n + \frac{h}{2}\left[ f(x_n,y_n) + f\Big( x_{n+1}，y_n+hf(x_n,y_n) \Big) \right] $$


### 习题

- **微积分问题转化**：求 $\dis\int^x_0 f(x)dx$ 在 $\{x_k\}$ 节点处的近似值
  - **解**：
    - 易得可等价转化为ODE初值问题：$\begin{cases} y' = f(x) \\ y(0) = 0 \end{cases}$
    - $y$ 在 $\{x_k\}$ 上的值就是 $f$ 上限函数的近似值
- **非线性单步法举例**：
  - **牛顿单步二阶法**：$y_{n+1} = y_n + hf_n + \dfrac{h^2}{2}(f_x+f_yf)_n$
    - **推导**：
      - 将泰勒展开扩大至二阶即可，再易得 $y'' = \dfrac{df}{dx} = f_x + f_yy_x$
      - <font color='chartreuse'>对于一阶ODE来说，增量函数必定只乘一个 $h$，故本方法的增量函数实际是 $\p = f + \dfrac{h}{2}(f_x + f_yf)$</font>
  - **龙格库塔方法**：含有中间点
- **二阶ODE迭代法举例**：
  - **verlet法（中心差分）**：$y_{n+1} = 2y_n - y_{n-1} + h^2f(x_n,y_n,y_n')$
    - **推导**：
      - 用二阶中心差商代替二阶导即可

### 误差分析

- <font color='red'>截断误差是方程与迭代公式的差，整体误差是数值解与真解的差
  - 线性迭代法中，整体误差</font>
- **截断误差**：因为舍去高阶小量所导致的误差
- **局部截断误差 $R_n$**：节点 $n$ 处的截断误差
  - **计算方法**：$u(x_{n+1}) = u(x_n) + h\p + R_n$
  - **欧拉法的截断误差**：$O(h^2)$
    - **证明**：
      - 由欧拉法的迭代公式得 $R_n = u(x_{n+1}) -\Big[ u(x_n) + hf(t_n,u_n) \Big]$
      - 将 $u(x_{n+1})$ 在 $t_n$ 处泰勒展开，计算得 $R_n = \dfrac{h^2}{2}u'' + O(h^3)$
  - **梯形法的截断误差**：$O(h^3)$
    - **证明**：同下
  - **改进欧拉法的截断误差**：$O(h^3)$
    - **证明**：
      - 由改进欧拉法的迭代公式得 $R_n = u_{n+1} - \Big[ u_n + \cfrac{h}{2}(f_n + f_{n+1})\Big]$
      - 将 $u_{n+1},f_{n+1}$ 在 $t_n$ 处泰勒展开，计算易得结论
      <!-- -  $R_n = \cfrac{h^3}{3!}u_n'''(\xi) - \cfrac{h^3}{4}u_n'''(\eta)$ -->
      - <font color='chartreuse'>或由积分形式得 $R_n = \dis\int^{t_{n+1}}_{t_n} f\Big( t,u(t) \Big)dt$，将其在 $t_n$ 处泰勒展开，计算易得结论</font>
- **精度**：
  - **计算方法**：$\cfrac{u_{n+1}-u_n}{h} = \p + T_n$
    - <font color='chartreuse'>也可以看作数值微分格式的截断误差</font>
  - **精度定理**：$m$ 阶ODE中，局部截断误差是 $R_{n} = O(h^{p+m})$，则具有 $p$ 阶精度
    - **证明**：比较数值积分和数值微分形式即可
  - **欧拉法的精度**：一阶
    - **证明**：
      - 计算一阶微分时，若取步长为 $h$ 的向前差分格式，则截断误差为 $$ R_h = \frac{u(x_{i+1})-u(x_i)}{h}-u'(x_i) $$
      - 将 $u(x_{i+1})$ 在 $x_i$ 处泰勒展开后得 $R_h = \dfrac{h}{2}u'' + O(h^2) = O(h)$
- **整体误差（离散误差）**：数值解与真实解的误差 $e_n = u(t_n)-u_n$
  - **误差方程**：误差的递推关系 $$e_{n+1} = e_n + h\Big[ f(t_n,u(t_n))-f(t_n,u_n) \Big] + R_n$$
    - <font color='chartreuse'>（真解的泰勒展开式）减去（数值解的递推公式）</font>
  - **误差阶数定理**：收敛的线性单步法中，整体误差比局部截断误差低一阶
    - **证明**：
      - 由于 $f$ 满足Lip条件，故由误差方程得 $|e_{n+1}| \leq (1+Lh)e_n + R$
        - <font color='chartreuse'>$L$ 表示 $\D f$ 的上界</font>
      - 不断递推得 $|e_n|\leq (1+Lh)^n|e_0| + R\sum\limits^{n-1}_{j=0} (1+Lh)^j$
        - <font color='chartreuse'>后项是等比级数加和，故还可以继续化简为分式形式</font>
      - 再将右式用指数不等式放缩即得 $|e_n|\leq e^{LT}|e_0| + \cfrac{R}{Lh}\Big[ e^{LT}-1 \Big]$
      - 再代入 $R$ 的阶数即可得到结论

### 相容性分析

- **算法的相容性**：若 $h\to 0$ 时局部截断误差 $R_h\to 0$，则称算法与方程相容
  - <font color='red'>只和算法有关，跟步长、初值无关</font>

### 稳定性分析

- **算法的稳定性**：
  - **连续定义法**：已知每一步迭代都会出现误差，若每步迭代的误差 $e_n$ 都连续依赖于初始误差 $e_0$，则称算法稳定 $$ \max\limits_{nh\leq T} |u_n-v_n| \leq C\max\limits_{0\leq j<k} |u_j-v_j| $$
    - <font color='chartreuse'>虽然说法是连续，但实际上我们只关心误差传递时的有界性</font>
  - **有界定义法**：$[0,T]$ 内，当 $h\to 0$ 时总存在 $M$ 使得 $|e_n| \leq M|e_0|$
    - <font color='chartreuse'>所以稳定性也可以叫受控性</font>
  - <font color='red'>只和算法有关，跟步长、初值无关</font>
- **欧拉法稳定性**
  - **证明**：
    - 设从两个初值 $u_0,v_0$ 算出的第 $n$ 个节点值分别为 $u_n,v_n$
    - 设 $e_n = u_n-v_n$
    - 计算易得 $e_n = e_{n-1} + h\Big[ f(t_{n-1},u_{n-1})-f(t_{n-1},v_{n-1}) \Big]$
    - 由于 $f$ 满足Lip条件，故 $$|e_n| \leq (1+Lh)|e_{n-1}|\leq (1+Lh)^n|e_0| \leq e^{nLh}|e_0| = e^{LT}|e_0|$$，从而得到连续性（**证毕**）
- **改进欧拉法的稳定性**
  - **证明**：
    - 同上即可

### 求解线性差分方程

- **线性差分方程**：
  - 设 $\{a_i(n)\}^k_{i=0}$ 是函数序列，$\{b_n\}^\infty_{n=0}$ 是序列
  - 若 $a_k(n)\neq 0, a_0(n) \neq 0$
  - 则称下列方程为 $k$ 阶线性差分方程，序列 $\{u_n\}$ 为方程的解 $$ a_k(n)u_{n+k} + ... + a_0(n)u_n = b_n $$
  - <font color='chartreuse'>这实际上就是组合数学里的线性递推公式，求解方法详见组合数学的特征法</font>
- **向前差分**：
  - $\D_+ u_n = u_{n+1}-u_n$ 称为一阶差分
  - $\D_+^2 u_n = \D_+ u_{n+1} - \D_+ u_{n}$ 称为二阶差分
  - <font color='chartreuse'>计算易得 $u_{n+2} = u_n + 2\D_+ u_n + \D_+^2 u_n$
  - 显然 $k$ 阶差分方程可被 $u_n$ 的 $1\sim k$ 阶差分完全表示</font>
- **齐方程**：若 $\forall b_n = 0$，则称为齐方程
  - **线性封闭性**：齐方程的解关于线性封闭
  - **线性无关性**：$k$ 阶齐方程具有 $k$ 个线性无关解
  - **通解性**：非齐方程的解 = 齐方程的解 + 非齐方程的特解

#### 常系数差分方程

- **等比特解**：$u_n = \z^n$
  - **解**：
    - 问题转化为求多项式 $a_k\z^k + ... + a_0 = 0$ 的根
    - 设 $C$ 是多项式的友阵，解向量 $U_n = \tvec{ u_{n+k-1} \\ \vdots \\ u_n}$，右端向量 $b_n = \tvec{\dfrac{b_n}{a_k} \\ \vdots \\ 0}$
    - 此时方程可写为向量形式 $U_{n+1} = CU_n + b_n$
    - 显然通解为 $U_n = C^nU_0 + \sum\limits^{n-1}_{l=0} C^lb_{n-l-1}$
      - <font color='chartreuse'>前项为齐方程的通解，后项为非齐方程的特解（令所有初值 $u_i = 0$）</font>
- **等比特解的性质**：
  - $\{C^n\}$ 有界 $\LR $ 多项式的根均在单位圆内（模小于 $1$），且单位圆上的都是单根

    - **证明**：
  - $\lim\limits_{n\to\infty} C^n = O \LR$ 多项式的根均在单位圆内（模小于 $1$）

#### 习题

- 求解线性差分方程 $u_{n+2}-2\mu u_{n+1} + \mu u_n = 1$，其中 $0<\mu<1$
  - **解**：
    - 特征方程 $\l^2 - 2\mu\l + \mu = 0$，特征根为 $\mu\pm i\sqrt{\mu(1-\mu)}$
    - 写成指数形式 $re^{\pm i\t}$，则齐次通解为 $u_n = r^n(C_1\cos n\t + C_2\sin n\t)$
    - 设非齐次特解 $u_n = A$，代入方程即得 $A = \cfrac{1}{1-\mu}$
    - 综上，通解为 $$ u_n = \sqrt{\mu}^n\Big( C_1\cos(n\arccos\sqrt{\mu}) + C_2\sin(n\arccos\sqrt{\mu}) \Big) + \frac{1}{1-\mu}$$
    - 易得 $n\to\infty$ 时 $u_n\to \cfrac{1}{1-\mu}$，即解渐近稳定

### 解的先验估计

- **解的先验估计**：在求出解之前对解的范围进行估计
- **Gronwall不等式（连续形式）**：
  - 设 $\eta(t)$ 是 $[a,b]$ 上的连续函数
  - 若存在非负常数 $\a,\b$ 使得 $\forall t\in [a,b]$ 都有 $\dis|\eta(t)| \leq \b + \a\int^t_a |\eta(\tau)|d\tau$
  - 则 $|\eta(t)| \leq \b e^{\a(t-a)}$
  - <font color='chartreuse'>它将（解自身的积分不等式）变为了（解的上界不等式），即证明了解存在上界</font>
  - **证明（端点延伸法）**：
    - **原始端点**：当 $t\to a$ 时，易得 $|\eta(t)| \leq \b$。而当 $|\eta(t)| \leq \b$ 时，显然结论成立
      - <font color='chartreuse'>简化题目：由连续性易得正负性不影响题设两不等式，故不妨设 $\eta(t) > 0$</font>
    - **新端点**：不妨设存在 $\eta(t) > \b$，则由连续性，必定存在 $\eta(t_0) = \b$
      - <font color='chartreuse'>再易得只有增长极快时，$\eta$ 才有可能不符合结论。但同时题设要求又限制了增长率的大小，故不妨取增长率的阈值进行讨论。这就引出了下面的函数比较法</font>
  - **证明（函数比较法）**：
    - 首先证明 $\eta(t) = \b e^{\a(t-a)}$ 时满足第一式（实际上正好取等）
    - 然后证明当 $\eta$ 的增长率大于它时均不满足第一式即可
  - **证明（直接法）**：
    - 当 $\b = 0$ 时，结论易得
    - 当 $\b\neq 0$ 时，设 $f(t) = \dis\b + \a\int^t_a |\eta(\tau)|d\tau$，则易得 $f'(t) \leq \a f(t)$，再积分即可得结论
- **Gronwall不等式（离散形式）**：
  - 设 $\eta_n$ 是序列
  - 若存在非负常数 $\a,\b$ 使得 $\forall n\geq k$ 都有 $\dis|\eta_n| \leq \b + \a h\sum^{n-1}_{j=0} \eta_j$
  - 则 $|\eta_n| \leq \b e^{\a T}(\b + \a khM_0)$，其中 $M_0 = \max\{|\eta_0|,...,|\eta_{k-1}|\}$
  - **证明（直接法）**：
    - 设 $f_n = \b + \a h\sum\limits^{n-1}_{j=0} |\eta_j|$，则 $f_n-f_{n-1} \leq \a h f_{n-1}$
    - 递推即可得 $f_n \leq (1+\a h)^{n-k}f_k \leq e^{\a T}(\b + \a khM_0)$（**证毕**）


## Runge-Kutta方法

### 显式龙格-库塔方法

- **思想**：希望在欧拉法的基础上提高精度
  - 改进欧拉法是用（数值积分的梯形公式）代替了（积分）
  - 要提高阶数，可以用（更多节点的数值积分公式）来代替它
- **一般形式的迭代公式**：$y_{n+1} = y_n + h\p$，其中 $$\begin{cases} \p(x_n,y_n,h) = \sum\limits^r_{i=1} c_iK_i \\\\ K_1 = f(x_n,y_n) \\\\ K_i = f\Big(x_n + \l_i h，y_n + h\sum\limits^{i-1}_{j=1} \mu_{ij}K_j \Big) \end{cases}$$
  - **非线性**：R-K方法中，需要计算 $f$ 在等距节点 $(x_j,y_j)$ 以外的值，所以不是线性方法
  - **单步法**
- **阶数**：$r$ 称为R-K方法的阶数
  - <font color='chartreuse'>我们希望选取 $c_i,K_i$ 使得 $r$ 阶方法有 $r$ 阶精度
  - 一阶的显式R-K方法就是欧拉法</font>

### 二阶显式R-K方法

- **分析**：
  - 此时局部截断误差 $R_{n+1} = y_{n+1}-y_n-h(c_1K_1 + c_2K_2)$
  - 为了衡量误差的阶数，将式子中的 $f(x,y)$ 在 $(x_n,y_n)$ 处进行二元泰勒展开，最终得到一个关于 $h$ 的多项式
  - 为了具有 $2$ 阶精度，必须满足 $h^3$ 前面的项系数均为 $0$
    - <font color='chartreuse'>已知 $$u_{n+1} = u_n + hu_n' + \frac{h^2}{2}u_n'' + O(h^3)$$ 和 $$K_2 = f_n + \dkh{\l_2h\pfrac{}{x} + \mu_{21}f_nh\pfrac{}{y}}f_n + O(h^2)$$
    - 因为 $u_n'' = f_x + f_yf$，故上式中 $f_x$ 和 $f_y$ 的系数都要等于 $\dfrac{1}{2}$ 才行</font>
  - 即得到一个非线性方程组 $$\begin{cases}c_1+c_2 = 1 & 要令 O(h) = 0 \\  c_2\l_2 = \dfrac{1}{2} & 要令O(h^2)=0 \\ c_2\mu_{21} = \dfrac{1}{2} & 要令O(h^2)=0\end{cases}$$
    - <font color='chartreuse'>这个非线性方程组的解不是唯一的，如果对 $c_2$ 取不同的值，会得到不同的迭代公式</font>
  - 设 $c_2 = a$，则其它参数为 $c_1 = 1-a，\l_2 = \mu_{21} = \cfrac{1}{2a}$
- **实例**：
  - **改进的欧拉法**：取 $a = \dfrac{1}{2}$
  - **中点公式**：取 $a = 1$，则得到 $$ y_{n+1} = y_n + hf\Big( x_n+\frac{h}{2} ，y_n + \frac{h}{2}f(x_n,y_n) \Big) $$
    - <font color='chartreuse'>它相当于用中矩形公式取代积分</font>

## 单步法的性质总结

- **稳定定理**：若 $\p$ 关于 $y$ 满足Lip条件，则单步法稳定
  - **证明**：同欧拉法的稳定性
  - **其它例**：欧拉法中 $f(x,y) = \sqrt{y}$ 时，它连续且局部有界，但不满足Lip条件
    - 易得误差只沿多项式阶数增长（而非指数增长），从而也稳定
- **相容定理**：$p$ 阶单步法相容 $\LR p\geq 1$
  - **证明**：同欧拉法的误差估计，计算截断误差时泰勒展开即可
- **核心定理（Lax等价定理）**：单步法中，收敛性 $\LR$ 相容性 + 稳定性  
  - <font color='chartreuse'>相容性保证局部误差收敛，稳定性保证局部扩散到整体时依然收敛，故整体收敛</font>
- **敛速估计定理**：
  - 设 $t\in [t_0,T]，h\in [0,h_0]，u\in \R$，
  - 若 $p$ 阶单步法相容且 $\p$ 关于 $y$ 满足Lip条件，则 $$ |e_n| \leq e^{LT}\Big[ (1+Lh)|e_0| + cTh^p \Big] $$
  - **证明**：
    - 由 $p$ 阶收敛，相减得误差方程 $$e_{n+1} = e_n + h\Big[ \p(t_n,u(t_n),h) - \p(t_n,u_n,h) \Big] + O(h^{p+1})$$
    - 由Lip条件 + 递推得 $$ |e_n| \leq |e_0| + cTh^p + hL\sum^{n-1}_{k=0} |e_k| $$
    - 再由Gronwall不等式即得结论

### 绝对稳定性

- **思想**：对于固定步长 $h$，当 $n\to\infty$ 时 $y_n$ 有界
  - <font color='chartreuse'>稳定性只服务于收敛性，只考虑 $h\to 0$ 的情况，故也称为零稳定性
  - 但实际工程中 $h$ 总要有一个最小值，此时零稳定性并不能解决长时间运算时的指数爆炸问题</font>
- **绝对稳定性**：
  - 设单步法对模型方程 $y' = \l y\quad (\l\in\C)$ 的迭代公式为 $y_{n+1} = E(h\l)y_n$
  - 若 $|E(h\l)| < 1$，则称该方法绝对稳定
  - <font color='chartreuse'>为什么只对模型问题定义：
    - 任何光滑ODE在足够小区域内都可线性化
    - 当 $f$ 线性时，模型问题最容易不稳定</font>
- **绝对稳定域**：使得 $|E(h\l)| < 1$ 的变量所围成的域
- **绝对稳定区间**：绝对稳定域与实轴的交集

### A-稳定

- **A稳定方法**：若绝对稳定域包含 $\{h\l \mid \text{Re}(h\l) < 0\}$，则称该方法是A稳定的

## 线性多步法

- <font color='red'>本节只讨论等距节点情况（变步长的多步法不经济）</font>
- **线性多步法**：
  - **线性**：$\p$ 关于 $f$ 线性
  - **多步**：需要用到前面多个节点的值
- **$k$ 步法（数值积分形式）**：
  - **显式格式**：$u_{n+k} = \sum\limits^{k-1}_{j=0} \a_i u_{n+j} + h\sum\limits^k_{j=0} \b_i f_{n+j}$
  - **隐式格式**：$\sum\limits^k_{j=0} \a_j u_{n+j} = h\sum\limits^k_{j=0} \b_j f_{n+j}$
  - <font color='chartreuse'>多步法中"数值微分形式"的意义不大</font>
  - **显方法**：$\b_k = 0$
    - <font color='chartreuse'>因为 $f_{n+k}$ 中含有 $u_{n+k}$ 项，所以如果没有该项，就不用解方程</font>
  - **隐方法**：$\b_k \neq 0$
    - <font color='chartreuse'>此时存在 $f_{n+k}$ 项，每步迭代都要解非线性方程</font>
- **计算方法**：
  - 首先用（和线性多步法同阶的）单步法求出前面 $k$ 个节点值
  - 然后使用线性多步法公式即可
- **局部截断误差**：$L[u_n;h] = \sum\limits^k_{j=0} \Big[ \a_j u_{n+j} - h\b_j u'_{n+j} \Big]$
  - **证明**：
    - 依次将 $u_{n+j}$ 和 $f_{n+j}$ 在 $t_n$ 处泰勒展开，然后合并即可

### 数值积分法

- **思想**：已知 $u(t_{n+1}) = u(t_n) + \dis\int^{t_{n+1}}_{t_n} f\Big( t,u(t) \Big)dt$
  - 若用 $k$ 次插值多项式代替 $f$，即可得到线性 $k$ 步迭代公式

#### Adams外插法

- **思想**：
  - 取 $t_n,...,t_{n-k}$ 为插值节点，构造牛顿向后插值公式的反转形式即得 $$ u_{n+1} = u_n + h\sum^k_{j=0} a_j \D^j f_{n-j} $$
  - 其中 $a_j = (-1)^j\dis\int^1_0 \tvec{-\tau \\ j} d\tau$
- **系数递推公式**：
  - 设母函数 $G(t) = \sum\limits^\infty_{j=0} a_jt^j$，易得 $|a_j| < 1$，故一致收敛
  - 积分级数交换得 $G(t) = \dis\int^1_0 (1-t)^{-\tau}d\tau$，计算得 $-\cfrac{\ln(1-t)}{t}G(t) = \cfrac{1}{1-t}$
  - 两端的三项都展开成幂级数得 $$ (1+\frac{1}{2}t + \frac{1}{3}t^2 + ...)(a_0 + a_1t + ...) = 1+t+t^2+...$$
  - 比较系数即可<font color='chartreuse'>（可以 $a_0$ 开始一个个算，也可以用下面的单步形式）</font> $$ a_n + \frac{1}{2}a_{n-1} + ... + \frac{1}{n+1}a_0 = 1 \quad (\forall n\in\N)$$
  - **实例**：由上式可依次计算出 $a_0 = 1，a_1 = \dfrac{1}{2}，a_2 = \dfrac{5}{12}，...$
- **局部截断误差**：$R_{n,k} = O(h^{k+2})$
  - **证明**：
    - 已知余项是插值余项的积分 $\dis\int^{t_{n+1}}_{t_n} r_{n,k}(t)dt $
    - 换元 $\tau$ 后用积分第二中值定理即可
- **一阶显式阿达姆斯方法（$k=0$）**：欧拉法
- **二阶显式阿达姆斯方法（$k=1$）**： $$u_{n+1} = u_n + \frac{h}{2}(3f_n-f_{n-1})$$


#### Adams内插法

- **思想**：
  - 取 $t_{n+1},t_n,...,t_{n-k}$ 为插值节点，构造牛顿向后插值公式的反转形式 $$ u_{n+1} = u_n + h\sum^{k+1}_{j=0} a_j^* \D^j f_{n+1-j} $$
  - 其中 $a_j^* = (-1)^j\dis\int^0_{-1} \tvec{-\tau \\ j} d\tau$
- **系数递推公式**：
  - 设母函数 $G(t) = \sum\limits^\infty_{j=0} a_j^*t^j$，易得 $|a_j^*| < 1$，故一致收敛
  - 积分级数交换得 $G(t) = \dis\int^0_{-1} (1-t)^{-\tau}d\tau$，计算得 $-\cfrac{\ln(1-t)}{t}G(t) = 1$
  - 两端的三项都展开成幂级数得 $$ (1+\frac{1}{2}t + \frac{1}{3}t^2 + ...)(a_0 + a_1t + ...) = 1$$
  - 比较系数即得 $a_0 =1$，其它项 $$ a_n + \frac{1}{2}a_{n-1} + ... + \frac{1}{n+1}a_0 = 0 \quad (\forall n\in\N^+)$$
  - **实例**：由上式可依次计算出 $a_0 = 1，a_1 = -\dfrac{1}{2}，a_2 = -\dfrac{1}{12}，...$
- **局部截断误差**：$R_{n,k} = O(h^{k+3})$
  - **证明**：
    - 已知余项是插值余项的积分 $\dis\int^{t_{n+1}}_{t_n} r_{n,k}(t)dt $
    - 换元 $\tau$ 后用积分第二中值定理即可
- **一阶隐式阿达姆斯方法（$k=0$）**：改进的欧拉法
- **二阶隐式阿达姆斯方法（$k=1$）**：$$u_{n+1} = u_{n} + \frac{h}{12}(5f_{n+1} + 8f_n - f_{n-1})$$

#### Simpson插值法

- **思想**：取积分形式 $u(t_{n+2}) - u(t_n) = \dis\int^{t_{n+2}}_{t_n} f\Big( t,u(t) \Big)dt$，然后用Simpson公式求积分

### 待定系数法

- **p阶k步法**：设局部截断误差 $L[u(t);h] = \sum\limits^k_{j=0} \Big[ \a_j u(t+jh) - h\b_j u'(t+jh) \Big]$
  - 将后面两个函数在 $t$ 处泰勒展开，设 $u^{(j)}$ 的系数为 $c_j$
  - 构造方程组 $c_0 = ... = c_p = 0，c_{p+1}\neq 0$，解出 $\a_j,\b_j$ 即可
  - <font color='chartreuse'>显然这是个 $(p+1)\times (2k+2)$ 阶线性方程组，为了保证有非零解必须 $p < 2k$</font>
- **方程通项**： $$ \begin{cases} c_0 = \a_0 + \a_1 + ... + \a_k \\\\ c_1 = (\a_1+2\a_2 + ... + k\a_k) - (\b_0 + ... + \b_k) \\\\ c_q = \frac{1}{q!}(\a_1 + 2^q\a_2 + ... + k^q\a_k) - \frac{1}{(q-1)!}(\b_1 + 2^{q-1}\b_2 + ... + k^{q-1}\b_k) \end{cases} $$
- **符号约定**：
  - 因为方程整体可相差一个系数，故取 $\a_k = 1$
  - 由于解不唯一，通常取 $\a_0,\a_1,...$ 为自由变量
  - 通过取 $\a_0$ 的值，可以令 $\b_k = 0$（显方法）或 $\b_k\neq 0$（隐方法）
- **局部截断误差**：$R_{n,k} = O(h^{p+1})$
- **整体误差**：$e_{n,k} = O(h^p)$
- **Milne法（4阶2步法）**：Simpson插值法

#### 达尔奎斯特定理

- **第一达尔奎斯特定理**：稳定的线性 $k$ 步法中，若 $k$ 为奇数则最高阶数为 $k+1$，若 $k$ 为偶数则最高阶数为 $k+2$
  - **证明**：
    - 已知理论最高阶为 $2k$，但再加上零稳定条件则添加新的非线性约束，从而阶数降低
