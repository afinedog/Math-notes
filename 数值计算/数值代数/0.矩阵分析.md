## 矩阵分析

### 向量范数

- **基本范数**：
  - 1-范数：$|a_1|+...+|a_n|$
    - 菱正方形
  - 2-范数：$\sqrt{a_1^2+...+a_n^2}$
    - 圆
  - p-范数：$\sqrt[p]{|a_1|^p + ... + |a_n|^p}$
    - 更加接近平正方形的圆
  - 无穷范数：$\max\limits_{1\leq j\leq n} |a_j|$
    - （夹逼可证其为 $p\to\infty$ 的极限值）
    - 平正方形
- **范数构造**：满足（正定性、齐次性、三角不等式）即可
  - **A乘积范数**：给定范数 $\|\cdot\|$ 和非奇异方阵A，则 $\|x\|_A = \|Ax\|$ 是新的范数
    - **证明**：2-范数直接传递过来
  - **二次型范数**：正定对称矩阵A，$\|x\|_{\sqrt{A}} = \sqrt{(Ax,x)}$ 是范数
    - **证明**：
      - 由正定对称性，存在对角矩阵 $\Lambda$，过渡矩阵 $H$ 是正交阵
      - $\sqrt{A} = H^T\sqrt{\Lambda}H$，非奇异
      - 由欧氏内积定义易得 $\|x\|_{\sqrt{A}} = \|\sqrt{A}x\|_2 = \sqrt{(Ax,x)}$
    - **理解**：用第五章的向量数值化理解
    - **本质（二次型范数）**：$\|x\|_{\sqrt{A}} = \sqrt{x^TAx}$
- **范数连续性**：范数函数是连续函数
  - **证明**：由连续定义，拆分成单位向量后放缩即可
    - $|\|x\|-\|y\|| \leq \max\{\|e_j\|\cdot \sum |x_j-y_j|\}$
- **范数等价**：$\exist \alpha_1,\alpha_2，\forall x\in R^n，\alpha_1\|x\|_{II} \leq \|x\|_I \leq \alpha_2\|x\|_{II}$
  - **本质**：夹逼得来的极限同阶性
  - **等价性定理**：有限维空间下的任意范数等价
    - **证明**：用2-范数当媒介
      - 单位球面上，新范数函数连续，故存在极值
      - 再因为 $\cfrac{x}{\|x\|_{2}}$ 在单位球面上，对其取新范数，由极值定义得两个不等式。
      - 由范数齐次性，变形即得题设不等式
- **Heine定理**：向量序列 $x^{(k)}\to 0 \Leftrightarrow$ 任意范数下，$\|x^{(k)}\|\to 0$
  - **必要性**：同实数Heine定理，范数连续性直得
  - **充分性**：取欧氏范数 + 有限范数等价性 + 夹逼定理
- **范数完备性**：向量序列极限存在 $\Leftrightarrow$ 其为Cauchy列
  - **证明**：取无穷范数 + 定义证明 + 有限范数等价性

### 矩阵范数

- **范数V,W和U相容**：$\|AB\|_{U}\leqslant \|A\|_V\|B\|_W$
- **相容范数**：与自身相容的范数
  - **本质**：满足Cauchy-Schwarz不等式，$\sum(a)(b) \leqslant (\sum a)(\sum b)$
  - **反例**：如果不满足相容性，则退化为向量范数
    - $R^{n\times n}$ 中的无穷范数不相容，取 $A = B = \begin{pmatrix} 1&1\\1&1 \end{pmatrix}$ 即可
  - 猜想：等价条件是，可以同时取到最小值
- **Frobenious范数**：矩阵形式的欧氏范数。$\|A\|_F = (\sum\limits^m_{i=1}\sum\limits^n_{j=1}a^2_{ij})^{1/2}$
  - **相容性**：自身相容，且与向量的2-范数相容
    - **证明**：
      - $\|Ax\|^2_2 = \sum\limits^m_{i=1}(\sum\limits^n_{j=1}a_{ij}x_j)^2 \leq \sum\limits^m_{i=1}\|A_i\|^2_2\|x\|^2_2 = \|A\|^2_F\|x\|^2_2$
        - 行向量分解 + Schwarz不等式
      - $\|AB\|^2_F \leq \sum\limits^p_{j=1} \|A\|^2_F\|B_j\|^2_2 = \|A\|^2_F\|B\|^2_F$
        - 列向量分解 + 三角不等式
  - **迹性**：$\|A\|^2_F = tr(A^TA)$

#### 诱导范数

- **向量范数的诱导范数**：已知向量范数 $\|\cdot\|$，则其诱导范数为 $\|A\| = \max\limits_{\|x\|=1}\|Ax\|$
  - **本质**：单位球面上的向量，经过A变换后的最大长度<font color='red'>（泛函的有界线性算子范数）</font>
  - **从属性**：其与原范数相容（$\|Ax\| \leqslant \|A\|\|x\|$）
    - **证明**：
      - *和向量范数相容*：由定义中的最大性得 $\|\frac{Ax}{\|x\|}\| \leqslant \|A\|$，变形即可得相容
      - *范数条件*：
      - *相容范数*：
- **经典诱导范数**
  - **相似范数**：乘积范数 $\|x\|_P$ 的诱导范数为 $\|A\|_P = \|PAP^{-1}\|$
    - **证明**：
      - $\max\limits_{\|x\|_P=1} \|Ax\|_p = \max\limits_{\|Px\|=1}\|PAx\| = \max\limits_{\|y\|=1} \|PAP^{-1}y\|$
      - 由诱导范数自身相容，上式 $\leqslant \max\limits_{\|y\|=1} \|PAP^{-1}\|\max\limits_{\|y\|=1} \|y\|$（**证毕**）
  - **列和范数**：1-向量范数的诱导范数 $\|A\|_1 = \max\limits_{1\leq j\leq n} \sum\limits^m_{i=1} |a_{ij}|$
    - 1-范数最大的列
  - **行和范数**：无穷向量范数的诱导范数 $\|A\|_\infty = \max\limits_{1\leq i\leq n}\sum\limits^n_{j=1} |a_{ij}|$
    - 1-范数最大的行
  - **相容系数**：$\kappa_\infty(A) = \|A^{-1}\|_\infty\cdot\|A\|_\infty$
  - **谱范数**：2-向量范数的诱导范数 $\|A\|_2 = \sqrt{\lambda}$（$\lambda$ 是 $A^TA$ 的最大特征值）
    - **谱半径**：$\rho(A) = \max\set{|\lambda_i|}$
- **诱导范数下界定理**：复矩阵 $A$ 的相容范数的下界为 $\rho(A)$
  - **证明**：
    - 取特征值 $\lambda$ 和特征向量 $v$，则 $|\lambda|\|v\| = \|\lambda v\| = \|Av\| \leqslant \|A\|\|v\|$
    - 从而 $|\lambda| \leqslant \|A\|$。再由 $\lambda$ 任意性，$\rho(A) \leqslant \|A\|$
  - **理解**：谱范数是C-S取等情况，从而是最小的
  - **推论（确界定理）**：复矩阵 $A$ 的相容范数的下确界为 $\rho(A)$
    - **证明**：设Jordan型到 $\cfrac{A}{\varepsilon}$ 的过渡阵为 $T$（即 $T^{-1}JT = A$）
      - 由Jordan型的行列结构 + 谱半径定义得
        - $\|TAT^{-1}\|_\infty = \|J\|_\infty \leqslant \max |\lambda_i| + \varepsilon \leqslant \rho(A)+\varepsilon$
        - $\|TAT^{-1}\|_1 = \|J\|_1 \leqslant \max |\lambda_i| + \varepsilon \leqslant \rho(A)+\varepsilon$
      - 因此，存在（1-范数和无穷范数下）（$T$ 乘积范数）的诱导范数，使谱范数的下确界极小性被满足：$\|A\|_T = \|TA\|_{1/\infty}\leqslant \rho(A) + \varepsilon$
    - **理解**：用Jordan阵将范数与特征值联系起来
    - **推论**：逼近范数的构造法
- **等价关系**：设 $A\in R^{m\times n}$
  - $\|A\|_2 \leqslant \|A\|_F \leqslant \sqrt{n}\|A\|_2$
  - $\frac{1}{\sqrt{n}}\|A\|_\infty \leqslant \|A\|_2 \leqslant \sqrt{m}\|A\|_\infty$
  - $\frac{1}{\sqrt{m}}\|A\|_1 \leqslant \|A\|_2 \leqslant \sqrt{n}\|A\|_1$

### 矩阵级数

- **矩阵序列的极限**：矩阵序列 $A^{(k)} = (a_{ij})_{m\times n}$
  - 若 $\lim\limits_{k\to\infty} a^{(k)}_{ij} = a_{ij}（\forall i,j）$，则 $A = (a_{ij})_{m\times n}$ 是极限
  - **本质**：每个元都逼近
- **Heine定理**
- **方阵幂级数**：部分和数列 $S_k = \sum\limits^k_{j=1} A^j$ 的极限
  - **收敛充要条件**：$A^k\to O \Leftrightarrow \rho(A)<1$
    - **必要性**：$A^kv = \lambda^kv$，矩阵数列收敛得等比级数收敛，从而 $|\lambda| < 1$（这玩意是不是也能证充分性？）
    - **充分性**：
      - 由谱半径下确界性，存在逼近相容范数 $\|\cdot\|$，使得 $\|A^k\|\leqslant \|A\|^k \to 0$
      - 由Heine定理，$A^k\to O$
  - **幂级数收敛充要条件**：$(I-A)^{-1}$ Taylor可展开 $\Leftrightarrow \rho(A) < 1$
    - **必要性**：Taylor级数收敛 $\LR$ 幂级数收敛
    - **充分性**：初等数论的等比级数因式分解法即可
  - **推论（幂级数不等式）**：若 $\exist \|A\|<1$，则 $I-A$ 非奇异，且 $\|(I-A)^{-1}\| \leqslant \frac{1}{1-\|A\|}$
    - **证明**：
      - *非奇异性*：由 $\rho(A)$ 下界性，A的特征值均小于1，从而不可能被I消去
      - 左边Taylor展开 + 三角不等式 + 相容性 = 右边
- **常用估计式**：若方阵 $A$ 非奇异，则若方阵 $E$ 满足 $\|A^{-1}E\| < 1$，有 $A+E$ 非奇异，且 $\|(A+E)^{-1}\| \leqslant \cfrac{\|A^{-1}\|}{1-\|A^{-1}E\|}$

### 习题

- **幂不等式**：由相容性易得 $\begin{cases} \|A^n\| \leq \|A\|^n \\ \|A^{-n}\|\geq \|A\|^{-n}\end{cases}$
- **逆转化式**：$\|A^{-1}\|^{-1} = \min\limits_{\|x\| = 1} \|Ax\|$
  - **相容证明**：$\|x\| = \|A^{-1}Ax\| \leq \|A^{-1}\|\cdot\|Ax\|$
    - 从而 $\|A^{-1}\|^{-1}\leq \|Ax\|\quad (\|x\|=1)$，只需 $\exist x$ 使得等号成立即可
    - 设 $A^{-1}$ 在 $y$ 方向上最长，则取 $x = \cfrac{A^{-1}y}{\|A^{-1}\|}$ 即可
  - **直接证明**：
    - 由定义，$\|A^{-1}\|^{-1} = \cfrac{1}{\max\limits_{\|x\|=1}\|A^{-1}x\|} \color{chartreuse} = \min\limits_{\|x\|=1} \cfrac{1}{\|A^{-1}x\|}$
    - 设 $A^{-1}$ 在 $x_0$ 方向最大，则 $\|A^{-1}\|^{-1} = \cfrac{1}{\|A^{-1}x_0\|}$
      - 由 $A(A^{-1}x_0) = x_0$，可得此时 $A$ 在 $x_0$ 方向最小
      - 故 $\min\limits_{\|x\|=1}\|Ax\| = \Big\|A\cfrac{A^{-1}x_0}{\|A^{-1}x_0\|}\Big\| = \cfrac{1}{\|A^{-1}x_0\|} = \|A^{-1}\|^{-1}$
    - <font color='chartreuse'>本质和上面一样，都得讨论最值向量，区别在于思考顺序不同，个人人为这个比较好想</font>
  - **范数延伸性理解**：由 $A^{-1}(Ax) = x$，可得若 $A$ 将 $x$ 延伸最多，那么为了把该向量映回来，$A^{-1}$ 必须也压缩最多
    - 取这个最值向量讨论，即可得到结果


## 矩阵约化

### QR分解

- **QR分解**：$A = QR$，其中Q是正交矩阵，R是上三角矩阵
  - **QR算法**：$Ax = b\red\Rt Rx = Q^Tb$
  - **唯一性**：非奇异矩阵 $A$ 的QR分解唯一
    - **证明**：归纳法得存在性，反证法得唯一性

#### Gram-Schmidt正交化方法

- $Q$ 是正交化的结果矩阵
- $R$ 是正交化的列变换矩阵
- **理解**：不断删去多余分量，保留正交分量

#### Givens变换

- **Givens矩阵**：平面旋转矩阵 $T_{ij}$，将 $I$ 中的 $i$ 行 $j$ 列换为二阶旋转阵
  - **几何意义**：在维度 $i,j$ 的平面上旋转 $\theta$ 角
  - **正交性**：定义易得
- **算法**：辅助角公式，把向量在 $ij$ 平面旋转到 $i$ 轴上，从而 $y_j = 0$
  - 取 $\begin{cases} a = \cfrac{x_i}{\sqrt{x^2_i + x^2_j}} = \sin\lang \vec x_i,\vec x_j \rang\\ b = \cfrac{x_j}{\sqrt{x^2_i + x^2_j}} = \cos\lang \vec x_i,\vec x_j \rang \end{cases}$，则有 $\begin{cases} y_i = \sqrt{x^2_i + x^2_j}（ij平面投影） \\ y_j = 0 \end{cases}$
- **QR分解**：$\begin{cases} Q^T = T_{n-1,n}T_{n-2,n}...T_{1n}...T_{12} \\ R = Q^TA \end{cases}$
  - <font color='red'>每一次迭代后，根据当前 $x^{(k)}\pad(k\leq n^2)$ 来构造 $T_{ij}$</font>
  - **运算次数**：
    - $\frac{4}{3}n(n^2-1)$ 次乘除运算
    - $\frac{1}{2}n(n-1)$ 次开方运算
  - **应用**：适用于稀疏矩阵


```matlab
function [a,b] = givens(x,y)
  if y == 0
    a = 1, b = 0;
  elseif abs(y) > abs(x)
    temp = x/y;
    b = 1/sqrt(1+temp^2);
    a = b*temp;
  else
    temp = y/x;
    a = 1/sqrt(1+temp^2);
    b = a*temp;
  end
end
```

#### Householder变换

- **Householder矩阵**：镜像投影矩阵。设 $v$ 是单位向量，$H = I-2vv^T$
  - **镜面对称性**：$Hx$ 是（以 $v$ 为法向量的超平面）的（$x$ 的镜像投影）
    - **理解**：
      - $2vv^Tx$ 是等腰三角形的底边，也是镜面指向 $x$ 的法向量
      - $x$ 和 $Hx$ 是两条腰
      - $x$ 在镜面的投影是高
  - **对称正交性**：
    - **对称性**：$vv^T = \begin{pmatrix} v_1^2 & v_1v_2 & \cdots & v_1v_n \\ v_1v_2 & v_2^2 & \cdots & v_2v_n \\ \vdots & \ddots \\  \end{pmatrix}$
    - **正交性**
- **算法**：
  - **求镜像平面**：$\forall \|x\|_2 = \|y\|_2$，取 $v = \cfrac{x-y}{\|x-y\|_2}$，则有 $y = Hx$
  - **退化法**：构造 $\overline{H}_m\overline{x}= \overline{H}_m\begin{pmatrix} x_{m+1} \\ \vdots \\ x_n \end{pmatrix} = \begin{pmatrix} -\alpha_{m+1} \\ 0 \\ \vdots \end{pmatrix}$
    - $\alpha_{m+1} = \|\overline{x}\|_2\cdot sgn(\overline{x})$
    - 再设 $H = diag(I_m,\overline{H}_m)$，不断进行下去
- **QR分解**：$\begin{cases} Q = H_1...H_{n-1} \\ R = Q^TA \end{cases}$
  - 构造 $H_1$ 将 $A$ 的第一列映射到 $(-\alpha_1,0,\cdots)_n^T$
  - 再构造 $H_2 = diag(1,\overline{H}_2)$，将 $A$ 第二列映射到 $(a_{12},-\a_2,0,\cdots)_{n-1}^T$
  - 不断进行下去，最终 $A$ 变为上三角矩阵 $R$，$H$ 合并得到 $Q$
  - **应用**：适用于稠密矩阵


#### Hessenberg变换

- **上Hessenberg矩阵**：$G = \begin{cases} g_{ij}\quad i\leq j+1 \\ 0，\pad 其他情况 \end{cases}$
  - $G = \begin{pmatrix} g_{11} & g_{12} & \cdots & g_{1,n-1} &  g_{1n} \\ g_{21} & g_{22} & \cdots & g_{2,n-1} & g_{2n} \\ 0 & g_{23} & \ddots & g_{3,n-1} & g_{3n} \\ \vdots & \ddots & \ddots  & g & g_{n-1,n} \\ 0 & 0 & \cdots & g_{n,n-1} & g_{nn} \end{pmatrix}$，对角元下方一对角行也不为0
- **Hess相似定理**：任意矩阵可正交相似于上Hess矩阵
  - **构造性证明**：设 $A = \begin{pmatrix} a_{11} & c^T_1 \\ b_1 & D_1 \end{pmatrix}$
    - 由Householder存在性，$\exist\overline{H}_1b_1 = (\beta_1,0,...)^T$
      - 构造 $H_1 = diag(1,\overline{H}_1)$
    - 则 $A_1 = H_1AH_1 = \begin{pmatrix} a_{11} & c_1^T\ol H_1 \\ \ol H_1 b_1 & \ol H_1D_1\ol H_1 \end{pmatrix}$
      - <font color='chartreuse'>由Householder对称正交性，这是正交相似</font>
    - 不断作用下去，即得 $(H_1\cdots H_{n-1})^TA(H_1\cdots H_{n-1}) = G$
  - **不唯一性**
- **QR分解**：$\begin{cases} Q = H_1...H_{n-2} \\ R = Q^TA \end{cases}$
    - **推论**：若A对称，则G为三对角矩阵（带宽为3的对角矩阵）
- **复杂度**：$\cfrac{10n^3}{3}$

### 奇异值分解（SVD）

- **奇异值分解**：设 $UAV = \Sigma = diag(\sigma_1,...\sigma_r,0...)$
  - 其中 $U，V$ 是酉矩阵<font color='chartreuse'>（见高等代数——非欧线性空间）</font>
- **像核对偶定理**：$(Im\ A)^\perp = Ker\ A^T$
  - **证明**：定义直得
  - **矩阵理解**：将 $AX = b$ 奇异值分解，$b\in Im\ A$ 或 $b\perp Ker\ A^T$ 时有界
  - **方程理解**：垂直是转置相乘为0，核是映射为0，故垂直向量 $=$ 转置相乘算子的核
  - **几何理解**：
- **奇异值分解定理**：
  - $Ker\ A = Span\{V_{r+1},...,V_n\}$
    - **证明**：对角阵形式下容易看出
  - $Im\ A = Span\{U_1,...,U_r\}$
    - **证明**：像核对偶定理，此时 $\ker A^T = span\{U^T_{r+1},\cdots,U^T_{n}\}$
      - 再由酉矩阵列向量的正交基性，$(\ker A^T)^\perp = span\{ U_1,\cdots ,U_r \} = Im A$
  - $A = U_1\Sigma V_1^T$
    - **证明**：由实酉方阵的逆转置性，移项即可
  - $\|A\|^2_F = \sum\limits^r_{i=1} \sigma_i$
    - **证明**：对角阵形式下易得
  - $\|A\|_2 = \sigma_1$
    - **证明**：定义直得
- **奇异值分解算法**：利用酉方阵逆转置性，用转置简化求逆运算
  - **规范解**：$x^* = V\Sigma^{-1}U^Tb = \sum\limits^r_{j=1} \cfrac{1}{\sigma_j}(u_j^Tb)v_j$
  - **推论**：规范解是按2-范数最小解
- **几何理解**：
  - **正交射影**：$S\subset R^n，P\in R^{n\times n}$
    - $\begin{cases} Im\ P = S （射影性） \\ P^T = P \\ P^2 = P \end{cases}$
    - **P是映射到S上的射影矩阵**：若S的标准正交基矩阵为 $V$，则 $P = VV^T$
      - 投影像：$\forall x\in R^n，Px \in S$
      - 投影边：$(I-P)x\perp S$
      - 射影最小定理：$\|x-Px\|_2 = \min\limits_{y\in S} \|x-y\|_2$
  - **Moore-Pernose广义逆**：$A^\dag = V_1\Sigma^{-1}U^{-1}$
    - **逆性**：$AA^\dag A = A，A^\dag AA^\dag = A^\dag$
    - **对称性**：$(AA^\dag)^T = AA^\dag$，$(A^\dag A)^T = A^\dag A$
    - **子空间射影性**：$AA^\dag = P_{Im}A，A^\dag A = P_{(Ker\ A)^\perp}A$
