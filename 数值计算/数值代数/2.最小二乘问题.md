# 第三章：最小二乘问题（LS问题）

- **方程组分类**
  - **超定方程组**：行秩 $>$ 列秩（系数矩阵是竖条）
    - <font color='chartreuse'>超定的意思是条件太多，即方程数量 $>$ 未知数数量</font>
  - **欠定方程组**：行秩 $<$ 列秩（系数矩阵是横条）
    - <font color='chartreuse'>欠定的意思是条件太少，即方程数量 $<$ 未知数数量</font>
- **思想**：
  - 超定方程组一般无解，即 $b\notin \Im A$
    - 此时退一步求（在模的意义上）最逼近（线性方程组的解）的向量
  - 欠定方程组有无穷多解，即解空间构成一个超平面
    - 此时进一步求（2范数最小的解）
  - 这两类问题都可以归为线性最小二乘问题

## LS问题

- **线性最小二乘问题**：设矩阵 $A\in \R^{m\times n}$，向量 $b\in\R^{m\times 1}$，求 $\min\limits_{y\in \R^n} \|Ax-b\|_2$
  - **变换意义**：求向量 $x$ 使得它的像距离 $b$ 最近
  - **组合意义**：求线性组合系数 $x_1,...,x_n$ 使得列向量 $A_1,...,A_n$ 的线性组合距离 $b$ 最近
  - <font color='chartreuse'>学过高代就知道这就是个求平面法向量的问题，低维情况下用解析几何 + 求二次函数极值就能做。但高维情况需要更方便的数值解法</font>
- **线性最小二乘向量**：解 $x$
- **残向量**：$r(x) =  b-Ax$
  - <font color='chartreuse'>逼近解与精确解的残差</font>

- **解存在和唯一性定理**：
  - 线性最小二乘问题的解总存在
  - 线性最小二乘问题的解唯一 $\LR \ker A = \{\bs 0\}$
  - <font color='chartreuse'>解唯一时有 $r(x) = \bs 0$</font>
  - **证明（存在性）**：正交分解即可
    - <font color='chartreuse'>首先正交分解</font>
      - 已知 $\R^n = \Im A \oplus (\Im A)^\perp$，从而有 $b = b_1 + b_2\pad (b_1\in \Im A，b_2\in(\Im A)^\perp) $
    - <font color='chartreuse'>然后勾股定理化简</font>
      - 此时 $b_1-Ax \in \Im A$，从而 $\|r(x)\|^2_2 = \|(b_1-Ax) + b_2\|^2_2$
      - 再由勾股定理，上式 $=  \|b_1-Ax\|_2^2 + \|b_2\|^2_2$
    - <font color='chartreuse'>最后除去常数项</font>
      - 由于 $A,b$ 已给定，故 $\|b_2\|_2^2$ 的值固定，即残向量极小等价于  $b_1-Ax$ 极小
      - 再由于 $b_1\in \Im A$，故 $Ax = b_1$ 存在解，从而最小二乘问题存在解
  - **证明（唯一性）**：
    - 当 $A$ 退化时，$Ax = b_1$ 不止一个解，所以符合条件的 $x$ 不唯一
    - 当 $A$ 非退化时，$\Im A$ 就是总空间 $\R^n$，故取直接取 $Ax = b$ 的解即可
- **线性最小二乘问题的解集**：$\chi_{LS}$
  - <font color='chartreuse'>非齐次线性方程组 $Ax = b_1$ 的解可能是超平面</font>
- **最小2范数解**：$2$ 范数最小的解 $x_{LS}$
  - **唯一性**：
    - **证明**：由于解集 $\chi_{LS}$ 是超平面，易得最小2范数解就是 $\chi_{LS}$ 的过原点的法向量

### 正则化

- **正则化方程组/法方程组**：$x\in \chi_{LS} \LR A^Tr(x) = 0 \LR A^TAx = A^Tb$
  - **线代证明**：
    - 由像核对偶定理，$\ker A^T = (\Im A)^\perp$，再由 $r(x)\perp \Im A$，即得 $A^Tr(x) = 0$，移项即得正则方程组
  - **内积证明**：
    - 由前面的垂直定理，当 $(Ax,Ax) = (Ax,b)$ 时最近，即 $x^TA^TAx = x^TA^Tb$
    - 再由于 $x^T$ 任意，故只可能是 $A^TAx-A^Tb = 0$
  - **数值化证明**：
    - 对于 $\|Ax-b\|^2_2$，先将其看作 $(ax-b)^2$
    - 将括号展开得 $x^TA^TAx - 2b^TAx  + b^Tb$
    - 再因式分解得 $x^T(A^TAx - 2A^Tb) + b^Tb$
    - 要使该式最小（还是到后面的最速下降法，那里用微分讲的明白）
  - **函数法（扰动法）证明**：
    - 设 $f(x) = \|Ax-b\|^2_2$
    - 若 $x\in\chi_{LS}$，则其为 $f$ 极小值点，即 $f'(x) = 0$
    - 再易得 $f(x+\a w) = f(x) + 2\a w^TA^T(Ax-b) + \a^2\|Aw\|^2_2$
      - <font color='chartreuse'>这些关系都不难计算</font>
      - 故此时 $f'(x+\a w)|_{\a = 0} = 2w^TA^T(Ax-b) = 0$，由$w$ 任意性，只能是 $A^T(Ax-b) = 0$，即得到正则化方程组
- **正则化方法的步骤**
  - 设 $C = A^TA，d = A^Tb$，转化为解方程组 $Cx = d$
  - 用平方根法计算 $C$ 的Cholesky分解，再求解方程 $Ly = d，L^Tx = y$ 即可
- **Moore-Penrose广义逆（伪逆）**：$$A^\dag = \begin{cases} (A^TA)^{-1}A^T & 超定情况 \\\\ A^T(AA^T)^{-1} & 欠定情况  \end{cases} $$
  - <font color='chartreuse'>它可看作非方阵的逆
  - 线性方程组的解是 $A^{-1}b$，法方程组的解是 $A^\dag b$</font>
  <!-- - **逆性**：
    - 若 $A$ 是方阵，则广义逆就是逆
    - 若 $A$ 不是方阵，则不存在逆，但存在广义逆 -->
  - **最小二乘性**：$A^\dag b$ 是线性最小二乘问题的解（无论超定、欠定）
    - **证明**：
      - 超定情况下，易得它是法方程的解，从而是最佳逼近
      - 欠定情况下，由高代知识得它是 $A^{-1}b$ 在 $\Im A^T = (\ker A)^\perp$ 上的正交投影，即 $\chi_{LS}$ 的法向量

### 扰动分析

- **扰动定理**：设 $\wt b = b+\d b$，其中 $b_1,\tilde{b}_1$ 分别为在 $\Im A$ 上的正交投影
  - 若 $b_1\neq 0$
  - 则 $\dis \frac{\|\delta x\|_2}{\|x\|_2} \leqslant \|A\|_2\|A^\dag\|_2\frac{\|b_1-\tilde{b}_1\|_2}{\|b_1\|_2}$
  - **证明**：
    - （条件数的定义相同）
  - **本质**：和普通扰动一样，只不过换成了广义逆
  - **推论**：若 $A$ 非奇异，则 $\Cond(A)^2 = \Cond(A^TA)$
    - **证明**：广义逆性质即可

## 数值法求解正则化方程组

- **QR法简化方程**：$R^TRx = A^Tb$
  - <font color='chartreuse'>此时已直接得到Cholesky分解</font>
  - **证明**：直接对正则化方程的 $A$ 进行QR分解即可
  - **几何意义**：
    - 由正交变换不改变长度得 $\|Ax-b\|_2 = \|Q^T(Ax-b)\|_2 = \|Rx - Q^Tb\|_2$
    - 残向量转化为 $r(x) = Rx-Q^Tb$，再代入正则化方程组即可
    <!-- - 变为 $R^TRx = R^TQ^Tb = A^Tb$ -->

### 非方阵的QR分解

- **QR分解的常用正交变换**：
  - Gram-Schmidt变换
  - Householder变换
  - Givens变换
  - Hessenberg变换
  - <font color='chartreuse'>详见矩阵分析</font>
- **超定矩阵的QR分解**：以 $A\in \R^{3\times 2}$ 为例
  - **分解公式**：
    - 此时 $Q\in \R^{3\times 2}$ 超定，$R\in \R^{2\times 2}$ 退化一阶，即 $A = [Q_1,Q_2]\bvec{R_1 \\ O} = Q_1R_1$
    - <font color='red'>虽然 $Q_1$ 不是方阵，但它的列向量依然是标准正交基，依然有 $Q^T_1Q_1 = I_2$ 和 $\|Q_1\|_2 = 1$，故也可用来求解最小二乘问题</font>
  - **计算简化**：
    - 在进行正交变换求 $Q$ 时，可以把 $A$ 的第三行忽略，只计算上侧的二阶子方阵
      - 这样虽然 $Q$ 的第三行也是 $0$，但依然有 $Q^TQ = I_2$，同时还省略了计算步骤
      - <font color='chartreuse'>这种思路就是下面的正交变换法</font>
    - 此时的 $R$ 也是只取前两行，直接截去第三行
- **欠定矩阵的QR分解**：以 $A\in \R^{2\times 3}$ 为例
  - **分解公式**：
    - 此时 $A^T$ 超定，对其作QR分解，即 $A^T = [Q_1,Q_2]\bvec{R_1 \\ O} = Q_1R_1$
    - 再转置回来即得 $A = R_1^TQ_1^T$
  - **计算简化**：
    - 在进行正交变换求 $Q$ 时，可以把 $A$ 的第三列忽略，只计算左侧的二阶子方阵
    - 此时 $R$ 的最后一列和 $A$ 相等，即 $A$ 只被处理了前两列

### 求解LS问题

- **超定情况**：
  - 设 $A$ 是超定矩阵
  - QR分解得 $A = Q_1R$，此时法方程化为 $R^TRx = R^TQ_1^Tb$
  - 求解 $Rx = Q_1^Tb$ 即可
  <!-- - <font color='chartreuse'>此时 $Q$ 也超定</font>
  - 此时 $Q$ 分块为 $[Q_1 , Q_2]$，由列向量两两正交性得 $Q_1^Tb \perp Q_2^Tb$
    - $Q_1$ 是方阵，$Q_2$ 是可忽略的部分
  - 由勾股定理得 $\|Rx - Q^Tb\|^2_2 = \|Rx - Q_1^Tb\|^2_2 + \|Q_2^Tb\|$
    - <font color='chartreuse'>显然最小二乘问题的解只与前面有关
    - 再由于 $R$ 和 $Q_1^T$ 对应于 $A$ 的线性无关部分，故前项的核为 $\bs 0$，从而下面的方程组是必定可解的</font>
  - 此时求解上三角方程组 $Rx = Q_1^Tb$ 即可 -->
- **欠定情况**：
  - 设 $A$ 是欠定矩阵
  - QR分解得 $A^T = Q_1R$
  - 此时最小二范数方程 $x = A^T(AA^T)^{-1}b$ 可化为 $R^TQ_1^Tx = b$
    - <font color='chartreuse'>设 $y = Q_1^Tx$，分别求解 $R^Ty = b$ 和 $Q_1^Tx = y$ 即可</font>

### 习题

- **Givens算法**：给出两个向量 $\|x\|_2 = \|y\|_2$，求Givens变换的相关算法，使得 $Qx = y$
  - **解**：
    - <font color='chartreuse'>只需要通过换坐标系，将 $y$ 作为新坐标系的 $e_1 = \tvec{1 \\ 0 \\ \vdots}$ 向量，然后将 $x$ 不断二维旋转即可</font>
    - 首先将 $y$ 变换为 $e_1$，设 $Q_1y = e_1$
    - 再将 $Q_1x$ 变换为 $e_1$，设 $Q_2(Q_1x) = e_1$
    - 则 $Q =  Q_1^TQ_2Q_1$
- 设 $L\in\R^{m\times n}\pad (m\geq n)$ 的前 $m$ 行是下三角矩阵，求Householder变换使得 $HL = \tvec{L_0 \\ \bs 0}$
