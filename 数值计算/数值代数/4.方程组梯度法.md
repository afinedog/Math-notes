# 第五章：共轭梯度法

- **CG法适用条件**：正定对称矩阵
- **向量的数值化**：将二次型的展开视为多项式展开
  - **分配律**：$(x+y)^T(x+y) = (x^T+y^T)(x+y) = x^Tx + y^Ty + 2x^Ty$
  - **交换律**：$(x^Ty)z = z(x^Ty)$（其它顺序都不能换）
  - **转化**：
    - $ax^2+bx+c = x^TAx + bx + c$
    - $a(x-d)^2 = (x-b)^TA(x-b)$
- **向量的泛函化**：将矩阵视为算子

## 最速下降法（DS）

- 方程组 $Ax = b$
- **二次泛函**：$\varphi(x) = x^TAx -2b^Tx$
- **极小解定理**：若 $A$ 对称正定，则方程组的解 $\LR$ 二次泛函的极小值点
  - **必要性**：
    - 设解为 $x_*$，则 $\varphi(x_*+y) = \varphi(x_*) + y^TAy$
      - <font color='chartreuse'>原式中，左项中取一个 $x_*$ 一个 $y$ 时，会和右边的 $by$ 抵消，从而只剩下左项两端的两项</font>
    - 由 $A$ 正定性，$\varphi(x_*+y) \geqslant \varphi(x_*)$，从而是极小值点
  - **充分性**：
    - $\cpfrac{\varphi}{x_i} = 2\sum\limits^n_{j=1} a_{ij}x_j - 2b_i$
    - $\grad\varphi = 2(Ax-b)$
    - 由梯度在极小值处为0，极小值点是解
  - **数值法理解**：$f(x) = ax^2-2bx$ 的极小值点 $\LR ax-b=0$ 的解
  - **本质**：该定理若将向量数值化后，简直太简单了
- **最速下降法**：满足 $\p(x_k+\a_k p_k) \leq \p(x_k+\a p_k)\quad (\forall \a\in R)$
  - 搜索方向 $p_k$
  - 步长 $\a_k$
  - 残差 $r_k = b-Ax_{k}$
  - 设 $f(\a) = \p(x_k+\a p_k)$
    - 求极值得 $\a_k = \cfrac{r^T_k p_k}{p_k^T Ap_k}$
    - 下降值 $\p(x_{k+1})-\p(x_k) = -\cfrac{(r_k^Tp_k)^2}{p^T_k Ap_k}$
- **最速下降性**：SD中，$p_k$ 永远取最快的方向 $r_k$（负梯度方向）

### DS收敛性

- **收敛性引理**：
  - 设 $A$ 的特征值为 $\{\lambda_i\}_n$，$P(t)$ 是 $t$ 的多项式
  - 则 $\|P(A)x\|_A \leqslant \max\limits_{1\leqslant i\leqslant n} |P(\lambda_i)|\|x\|_A$
  - <font color='chartreuse'>其中 $\|x\|_A = \sqrt{x^TAx}$</font>
  - **证明**：设 $\{y_i\}^n_{i=1}$ 是特征空间的标准正交基，则 $\forall x\in R^n$，有 $x = \sum\limits^n_{i=1}  \b_i y_i$
    - 从而 $\big[ x^TP(A) \big] A \big[ P(A)x \big]$
    - 由特征性，原式 $= (\sum\limits^n_{i=1}\b_i P(\l_i) y_i)^T A (\sum\limits^n_{i=1} \b_i P(\l_i)y_i) $
    - 由正交性，原式 $= \sum\limits^n_{i=1} \l_i\b_i^2 P^2(\l_i)$
    - 上界放缩得：原式 $\leq \max\limits_{1\leq i\leq n} P^2(\l_i)\cdot\sum\limits^n_{i=1} \l_i\b_i^2$
    - 用特征性再转化回来：放缩式 $= \max\limits_{1\leq i\leq n} P^2(\l_i)\cdot x^TAx$
    - 由于 $P(A)\leq \max\limits_{1\leq i\leq n} P(\l_i)$，故左边消去 $x^TP(A)$ ，右边消去 $x^TP(\l_i)$，不等号依然成立（放缩）
    - 最终得到 $A\big[ P(A)x \big] \leq \max\limits_{1\leq i\leq n} P(\l_i)\cdot Ax$
  - **理解**：
  - **本质**：特征值的确界不等式，添项 $P(A)$ 使得更加具有一般性，并采用正交规范基进行标准化
- **收敛性定理**：设 $A$ 的特征值为 $\{\lambda_n\}$
  - 则 $\|x_k-x_*\|_A \leqslant (\cfrac{\lambda_n-\lambda_1}{\lambda_n+\lambda_1})^k \|x_0-x_*\|_A$
  - <font color='chartreuse'>其中 $x_* = A^{-1}b \approx \dfrac{b}{a}$</font>
  - **证明**：
    - 由最速下降性，$\p(x_k)\leq \p(x_{k-1}+\a r_{k-1})\quad (\forall \a\in\R)$
    - 数值法配方得 $\p(x) + x_*^TAx_* = (x-x_*)^TA(x-x_*)$
      - $\color{chartreuse}ax^2-2bx+\dfrac{b^2}{a} = a(x-\dfrac{b}{a})^2$
    - 从而最速下降的单调公式转移到二次型上，即：
    - $(x-x_*)^TA(x-x_*) \leq (x_{k-1}+\a r_{k-1} - x_*)^TA(x_{k-1}+\a r_{k-1}-x_*)$
      - 右式变形为 $\big[ (I-\a A)(x_{k-1}-x_*) \big]^TA\big[ (I-\a A)(x_{k-1}-x_*) \big]$
      - $\color{chartreuse}a(x_k-\dfrac{b}{a})^2 \leq a(1-\a\cdot a)^2(x_{k-1}-\dfrac{b}{a})^2$
    - 设 $P_\a(t) = 1-\a t$，则右式变为 $P_\a(A)(x_{k-1}-x_*)$
    - 由引理，$\|x_k-x_*\|_A \leq \max\limits_{1\leq i\leq n} |P_\a(\l_i)| \cdot\|x_{k-1}-x_*\|_A\quad (\forall \a\in\R)$
    - 再由 $\min\limits_{\a}\max\limits_{\l_1\leq t\leq \l_n} |1-\a t| = \cfrac{\l_n-\l_1}{\l_n+\l_1}$，代入上式即可
  - **理解**：
- **收敛速度**：$\cfrac{\lambda_n-\lambda_1}{\lambda_n+\lambda_1}$
  - 当 $\l_1 \ll \l_n$ 时，速度极慢

## 共轭梯度法（CG）

- **优点**：防止陷入局部最优
- **初始条件**
  - 初始向量 $x_0$
  - 负梯度 $r_0$
  - 下降方向 $p_0$
  - 步长 $\a_k$
  - （当前负梯度）和（上个下降方向）张成的平面 $\pi_k = \{x_k+\xi r_k + \eta p_{k-1}\mid \xi,\eta\in R\}$
- **共轭梯度法**：
  - 第一步取 $r_0 = p_0$，进行最速下降迭代
  - 第 $k+1$ 步，下降方向取 $\pi_k$ 内 $\p$ 下降最快的方向，进行最速下降迭代
- **二次泛函的限制**：$\p|_{\pi_k} = \psi(\xi,\eta) = \p(x_k+\xi r_k + \eta p_{k-1})$
- **极小值点**：$\wt x$
  - 对 $\xi,\eta$ 求极值得二元方程组
  - 求解方程组得 $\b_{k-1} = \cfrac{\eta_0}{\xi_0} = -\cfrac{r^T_k A p_{k-1}}{p^T_{k-1} A p_{k-1}}$
  <br>
- **下降方向**：$p_k = \dfrac{1}{\xi_0} (\wt x - x_k) = r_k + \b_{k-1}\cdot p_{k-1}$
  <img src = "./图片/共轭梯度法.png">
- **模型**：$\begin{cases} x_{k+1} = x_k + \a_k p_k \quad 迭代 \\ r_{k+1} = b-Ax_{k+1} \quad 残差 \\ p_{k+1} = r_{k+1} + \b_k p_k \pad 下降方向 \\  \end{cases}$ **计算**：$\begin{cases} \a_k = \cfrac{r_k^Tp_k}{p^T_k A p_k} \\\\ \b_k = -\cfrac{r^T_{k+1} A p_k}{p_k^T A p_k} \end{cases}$

### CG性质

- **Krylov子空间**：$\mc K(A,r_0,k+1) = span\{r_0,Ar_0,\cdots,A^kr_0\}$
  - **方程意义**：初始梯度向量 $r_0$ 在系数矩阵 $A$ 映射下张成的子空间
- **CG性质**：
  - **相互垂直性**：$p_i^Tr_j = 0\quad (0\leq i<j\leq k)$
  - **残差正交性**：$r_i^Tr_j = 0\quad (0\leq i,j\leq k)$
  - **方向共轭性**：$p^T_i A p_j = 0 \quad (0\leq i,j\leq k)$
  - **空间性**：$span\{r_0,\cdots,r_k\} = span\{p_0,\cdots,p_k\} = \mc K(A,r_0,k+1)$
  - **证明**：
    - **算术证明**：数学归纳法
  - **几何理解**：
  - **推论**：共轭梯度法最多 $n$ 次便会得到结果<font color='chartreuse'>（每一次构造都是取新正交基，$n$ 维空间中仅有 $n$ 个）</font>
- **估计定理**：$\p(x_k) = \min\set{\p(x)\mid x\in \big[ x_0 + \mc K(A,r_0,k+1) \big]}$
