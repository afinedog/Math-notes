# 第四章：古典迭代解法

## 单步定长迭代法

- **单步性**：一次迭代只有一步运算（与后面的双位移QR形成对比）
- **定长性**：递推公式是一阶线性的（与后面的幂法形成对比）

### 两个经典方法

- 方程组 $Ax = b$
- 构造 $A = D-L-U$，（分别是对角、下三角、上三角矩阵），则原式可变形
  - <font color='chartreuse'>理解：变形的实质就是分裂x，然后两边一个取 $x_k$，另一个取 $x_{k-1}$
  - 而由极限的递推等式，可得若收敛即收敛到解
  - 本质：这是先知道极限的递推等式，在收敛的假设下，原等式变形得到的结果</font>
- **Jacobi迭代法**：分离 $D$ 和 $L+U$
  - **递推公式**：$x_k = D^{-1}(L+U)x_{k-1} + D^{-1}b$
    - 迭代矩阵 $B = D^{-1}(L+U)$
    - 常数项 $g = D^{-1}b$
  - **迭代法**：给定一个初值 $x_0$
- **Gauss-Seidel迭代法**：分离 $D-L$ 和 $U$
  - **递推公式**：$x_k = D^{-1}Lx_k + D^{-1}Ux_{k-1} + g$
  - 若 $(D-L)^{-1}$ 存在
  - **递推公式**：$x_k = (D-L)^{-1}Ux_{k-1} + (D-L)^{-1}b$
    - 迭代矩阵 $L_1 = (D-L)^{-1}U$
    - 常数项 $h = (D-L)^{-1}b$

### 总结

- **单步定长迭代法**：$x_k = Mx_{k-1} + g$
  - 迭代矩阵
  - 常数项
  - 初始向量
- **收敛有解性**：若迭代法收敛，则 $x_k = x_{k-1}$，从而等式变形为 $x = (I-M)^{-1}g$，即为解
- **迭代法与方程组相容**：存在非奇异矩阵 $G$，使得 $\begin{cases} G(I-M) = A \\ Gg = b \end{cases}$
- **收敛判定定理**：收敛 $\Leftrightarrow$ 相容

## 收敛判别

### 充要条件

- **极限（解）**：$x_*$
- **误差向量**：$y_k = x_k - x_*$
- **收敛充要条件**：收敛 $\Leftrightarrow M^k\to 0$
  - **证明**：由 $y_{k+1} = My_k$ 直得
  - **推论**：收敛 $\Leftrightarrow \rho(M) < 1$
    - **证明**：见矩阵分析，右式是矩阵幂收敛充要条件

#### 习题

- 迭代矩阵 $M$ 也同时写作 $B$，不想改了
- **Jacobi精确解**：若 $\rho(B) = 0$，则Jacobi迭代 $n$ 次后可得到精确解
  - **证明**：
    - 由谱范数为0，得 $B$ 的所有特征值均为0
    - 再因为 $B$ 的所有对角元均为0，故其相似于Jordan标准型 $J(0)$（对角元均为0）
    - 由上升性，$J^n = O$，故 $B^n = P^{-1}J^nP = O$
    - 从而误差 $y_n = B^n y_0 = 0$，是精确解

### 充分条件

- **通项误差定理**：
  - 若 $\|M\| = q < 1$
  - 则 $\|x_k-x_*\| \leqslant \dfrac{q^k}{1-q} \|x_1-x_0\|$
  - **证明**：
    - 由C-S不等式得 $\|x_{k+1}-x_k\| \leq q^k\cdot\| x_1-x_0\|$
    - 再由三角不等式 + 裂项得 $\|x_*-x_k\| < \dfrac{1-q^\infty}{1-q}\cdot\|x_{k+1}-x_k\|$
    - 两边综合即得结果
  - **理解**：通项误差系数就是等比和函数
- **递推误差定理**：
  - 若 $\|M\| = q < 1$
  - 则 $\|x_k-x_*\| \leqslant \dfrac{q}{1-q} \|x_{k-1}-x_{k}\|$
  - **证明**：即为上述定理推论

### 范数收敛性

- **无穷范数定理**：Jacobi和G-S迭代中
  - 若 $\|B\|_\infty < 1$
  - 则
    - $\|L_1\|_\infty < 1$
  - **误差估计引理**：
    - G-S误差估计式为：$\|x_k-x_*\|_\infty \leqslant \dfrac{\mu^k}{1-\mu} \|x_1-x_0\|_\infty$
    - 其中 $\mu = \max\Big\{\cfrac{\sum\limits^n_{j=i+1} |b_{ij}|}{1-\sum\limits^{i-1}_{j=1}|b_{ij}|} \Big\} \leqslant \|B\|_\infty < 1$
    - **证明**：
      - 设
        - 下三角矩阵 $L$ 的第 $i$ 行元之和为 $l_i = \sum\limits^{i-1}_{j=1} |b_{ij}|$
        - 同理上三角为 $u_i = \sum\limits^n_{i=j+1} |b_{ij}|$
      - 由 $l_i+u_i \leq \|B\|_\infty < 1$，得 $\dfrac{u_i}{1-l_i} \leq l_i+u_i$
      - 两边取最大值即得 $\mu < 1$
  - **定理证明**：设 $x\in R^n$ 满足 $\|L_1\|_\infty = \|L_1x\|_\infty$，$y = L_1x$
    - 则存在 $0<i<n$ 使得 $|y_i| = \|y\|_\infty$
    - 再由 $L_1$ 定义得 $y = D^{-1}Ly + D^{-1}Ux$
    - 从而 $y_i = \sum\limits^{i-1}_{j=1} b_{ij}y_j + \sum\limits^n_{j=i+1} b_{ij}x_j$
    - 取绝对值得 $\|y\|_\infty \leq \|y\|_\infty\cdot l_i + u_i$
    - 再由误差引理即得结论
- **1-范数定理**：
  - 若 $\|B\|_1 < 1$
  - 则 $\rho(L_1) < 1$
  - **误差估计引理**：
    - G-S误差估计式为：$\|x_k-x_*\|_1 \leqslant \dfrac{\wt\mu^k}{(1-\wt\mu)(1-s)} \|x_1-x_0\|_1$
    - 其中 $s = \max\sum\limits^n_{i=j+1} |b_{ij}|$，$\mu = \max\Big\{\cfrac{\sum\limits^{j-1}_{i=1} |b_{ij}|}{1-\sum\limits^{j+1}_{i=1}|b_{ij}|} \Big\} \leqslant \|B\|_1 < 1$
    - **证明**：同上

### 正定收敛性

- **正定收敛定理**：
  - 若 $A$ 对称，对角元均为正，则 Jaboci收敛 $\Leftrightarrow A、2D-A$ 均正定
    - **证明**：
      - $B = I-D^{-1}A = D^{-\dfrac{1}{2}}(I-D^{-\dfrac{1}{2}}AD^{-\dfrac{1}{2}})D^{\dfrac{1}{2}}$

      - 再由 $J$ 收敛性等价于 $\rho(B)<1$，从而其等价于 $I-B、I+B$ 的特征值均为正
      - 再由 $\begin{cases} I-B = D^{-\dfrac{1}{2}}(D^{-\dfrac{1}{2}}AD^{-\dfrac{1}{2}})D^{\dfrac{1}{2}} \\ I+B = D^{-\dfrac{1}{2}}(2I-D^{-\dfrac{1}{2}}AD^{-\dfrac{1}{2}})D^{\dfrac{1}{2}} \end{cases}$ 即得结论
  - 若 $A$ 对称正定，则G-S收敛

### 可约性+对角占优性与收敛性

- **弱严格对角占优矩阵**：$|a_{ii}| \geqslant \sum\limits^n_{\substack{j=1 \\ j\neq i}} |a_{ij}|$，至少有一个不等号成立
- **可约矩阵**：存在置换方阵 $P$，使得 $PAP^T = \tvek{A_{11} & 0}{A_{12} & A_{22}}$
  - **置换方阵**：每行每列一个元为1，其余元为0
  - **分解性**：可分矩阵的齐次方程组可化为两个低阶方程
  - **分解定义**：$W = \{1,\cdots,n\}$ 存在不相交分割 $S,T$ 使得 $a_{ij} = 0\quad (i\in S,j\in T)$
    - <font color='chartreuse'>$S$ 对应左边元，$T$ 对应右边元，$a_{ij}$ 对应右上角</font>
- **奇异定理**：$A$ 非奇异的充分条件：
  - 严格对角占优
    - **证明**：
      - 反设奇异，则 $Ax = 0$ 有非零解 $x$，可设 $|x_i| = \|x\|_\infty = 1$
      - 再由解性，$(Ax)_i = 0$，故 $a_{ii}x_i = -\sum\limits^n_{\substack{j=1 \\ j\neq i}} a_{ij}x_{j}$
      - 则 $|a_{ii}| = |a_{ii}x_i|  = |\sum\limits^n_{\substack{j=1 \\ j\neq i}} a_{ij}x_{j}| \leq \sum\limits^n_{\substack{j=1 \\ j\neq i}} |a_{ij}|$，矛盾
    - **本质**：
      - **稀疏矩阵**：哪怕其它元的绝对值均为0，对角元的绝对值也大于0，当然非奇异
      - **非稀疏矩阵**：某个对角元若要被中和，则线性系数必须有一个 $\xi_i$ 大于1，但此时第 $i$ 行的对角元就不能被中和了。故永远不可能整行被中和，即非奇异
  - 不可约对角占优
    - **证明**：反设 $A$ 奇异，$x$ 是解，不妨设 $\forall |x_i| \leq 1$
      - 则 $S = \{i: |x_i| = 1\}$，$T = \{k: |x_k| < 1\}$ 是分割
        - 由对角占优性，$T$ 非空
        - 同上得 $|a_{ii}x_i| = \sum\limits_{\substack{j\in S \\ j\neq i}}|a_{ij}||x_j| + \sum\limits_{j\in T}|a_{ij}||x_j|$
      - 由不可约性，存在 $a_{ik} \neq 0\quad (i\in S, k\in T)$
      - 由C-S不等式， $|a_{ik}x_k| < |a_{ik}|$，故 $|a_{ii}| < \sum\limits_{j\neq i} |a_{ij}|$，矛盾
    - **理解**：和上面的证法差不多
    - **本质**：
      - **稀疏矩阵**：连分块的零矩阵都不能有，当然也不可能存在退化的整行了
      - **非稀疏矩阵**：
  - **总结**：
    - 对角占优性离非奇异性其实只差一步之遥
    - 严格性和不可约性主要针对稀疏矩阵  
  - **推论**：
    - 上述 $A$ 若对称则正定
      - **证明**：
        - $\forall \l \leq 0，A-\l I$ 和 $A$ 对角占优性、可约性均相同，故非奇异
        - 再由 $\l$ 任意性，可取绝对值足够大的 $\l$，使特征值或行列式为正
      - **理解**：
        - 严格性与不可约性是元之间的关系，故改变对角元不会变化
        - 对角占优性只会被 $\l$ 扩大
    - 上述正定 $A$ 对Jacobi和G-S收敛
      - **Jacobi证明**：
        - 反设 $B$ 某特征值 $|\l|\geq 1$
          - 易得 $\l I - B = D^{-1}(\l D-L-U)$
        - 由正定性和顺序主子式定理，$D$ 非奇异
          - 再由 $\l D-L-U$ 对角占优性、可约性与 $A$ 相同，故非奇异
        - 从而 $\l I - B$ 非奇异
          - 从而方程 $\l I - B = 0$ 仅零解，与特征值条件矛盾  
      - **G-S证明**：
        - 反设 $L_1$ 某特征值 $|\l|\geq 1$
          - 易得 $\l - L_1 = (D-L)^{-1}(\l(D-L) - U)$
        - 由 $A$ 的对角占优性、可约性，得对角元不为0，故下三角阵 $D-L$ 非奇异
          - 再由 $\l(D-L)-U$ 的对角占优性、可约性与 $A$ 相同，故非奇异
        - 从而 $\l I - L_1$ 非奇异
          - 从而方程 $\l I-L_1 = 0$ 仅零解，与特征值条件矛盾
    - **总结**：就是利用了 $\l I - A$ 的对角占优性和可约性只会被扩大，从而非奇异，从而导出特征方程无解，使得特征值必须被限制，从而谱范数被限制，即收敛

## 收敛速度

### 两种速度

- **$k$ 次迭代的平均收敛速度**：$R_k(M) = \cfrac{-\ln\|M^k\|}{k}$
- **比例因子**：$\sigma = \Big( \cfrac{\|y_k\|}{\|y_0\|} \Big)^{\dfrac{1}{k}} \approx \|M^k\|^{\dfrac{1}{k}} = e^{-R_k}$
- **渐进收敛速度**：$R_\infty(M) = \lim\limits_{k\to\infty} R_k(M)$
- **渐进定理**：$R_\infty(M) = -\ln\rho(M)$
  - **互包证明**：
    - $\rho(M)\leq \|M^k\|^{\dfrac{1}{k}}$
      - 由范数幂性 $\rho(M)^k = \rho(M^k)$
      - 由范数最小性，$\rho(M^k) \leq \|M^k\|$
    - 设 $B_\varepsilon = \cfrac{1}{\rho(M)+\varepsilon} M$
      - 其幂收敛到0，故存在 $k$ 使得 $\|M^k\| \leq (\rho(M)+\varepsilon)^k$

## 超松弛迭代法（SOR）

- **松弛迭代法**：G-S迭代法中，设 $\D x = x_{k+1}-x_k$，则 $x_{k+1} = x_k + w\D x$ 即为松弛迭代法
  - **松弛因子 $w$**：大于1则超松弛，小于1则低松弛
- **简化**：$x_{k+1} = L_w x_k + w(D-wL)^{-1}b$
  - **迭代矩阵**：$L_w = (D-wL)^{-1}\big[ (1-w)D + wU \big]$
  - **加速性**：G-S的加速算法

### 收敛性

- **经典收敛定理**：
  - 收敛 $\LR \rho(L_w) < 1$
- **收敛必要条件**：收敛 $\Rt w\in (0,2)$
  - **证明**：
    - 收敛得 $\rho(L_w) < 1$，即 $|\l_1...\l_n| < 1$
      - 从而 $\det(L_w) < 1$
    - 再由 $|\det L_w| = |\det()|\cdot|\det()| = |(1-w)^n\cdot 1|$，即得 $|1-w|<1$
- **对角收敛定理**：收敛的充分条件：$w\in (0,1)$
  - $A$ 严格对角占优
  - $A$ 不可约对角占优
  - **证明**：同奇异定理
- **正定收敛定理**：$A$ 是实对称正定矩阵，$w\in (0,2)$，则SOR收敛
  - **证明**：
    - $L_w x = \l x$，展开变形得 $\l^2$
    - 设 $\begin{cases} x^*Dx = \d \\ x^*Lx = \a+i\b \\ x^*L^Tx = \a-i\b \end{cases}$
    - 分子减分母得 $w\d(\d-2\a)(w-2)$
    - 由正定性，$\d,\d-2\a > 0$，故 $w\in (0,2)$ 时 $|\l^2|<1$，从而收敛
