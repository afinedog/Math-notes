# 第四章：古典迭代解法

- **符号约定**：
  - **迭代公式的极限（方程组的解）**：$x^*$
  - **误差向量**：$e_k = x_k - x^*$
- **线性方程组的迭代解法分类**：
  - 按照迭代公式的数学形式，可分为单步法（本章）和多步法（双重步位移QR迭代）、定常法（本章）和非定常法（梯度法）
  - 按照迭代公式的推导方式，可分为矩阵分裂法（J、G-S）、优化类方法（梯度法）、多项式逼近法（Richardson迭代法）等

## 单步定常迭代法（矩阵分裂法）

- **单步定常迭代法**：
  - **单步性**：一次迭代只有一步运算
  - **定常性**：每次迭代的递推公式不变
- **矩阵分裂法**：在 $Ax = b$ 中，将 $A$ 分裂并移项后，左边的 $x$ 取为 $x_k$，右边的 $x$ 取为 $x_{k-1}$，从而构造迭代公式
  - <font color='chartreuse'>单步定常迭代法实际上等价于矩阵分裂法</font>
- **递推公式**：$x_k = Mx_{k-1} + g$
  - 迭代矩阵 $M$
  - 常数项 $g$
  - 初始向量 $x_0$
- **相容性**：若存在非奇异矩阵 $G$，使得 $\begin{cases} G(I-M) = A \\ Gg = b \end{cases}$，则称该迭代法与方程组相容
  - <font color='chartreuse'>ODE数值解中，相容的意思是（差分格式）在极限状态下能变回（原方程）
  - 这里的思想类似，相容指的是（迭代公式）能够不损失信息地变回（原方程），关键就是 $G$ 的非奇异性</font>
- **相容收敛定理**：单步定常迭代法收敛 $\LR$ 迭代法与方程组相容
  - **证明**：
    - 直接计算易得相容的定义式和 $Ax = b$ 可相互转化

### 收敛判别

- **收敛充要条件**：迭代法收敛 $\LR M^k\to 0 \LR \rho(M) < 1$
  - **证明**：
    - 易得误差方程 $e_{k+1} = Me_k$，再由矩阵分析的幂序列收敛条件即得结论
  - <font color='red'>谱半径 $\rho(M)$ 就是判别单步定常迭代法收敛性、收敛速度的唯一标准。后面的其它方法不过是这个标准的一些充分条件而已，最终证明方法都还是要回归到谱半径上</font>

### 收敛速度

- **先验误差估计**：
  - 若 $\|M\| < 1$
  - 则 $\|x_k-x^*\| \leq \cfrac{\|M\|^k}{1-\|M\|} \|x_1-x_0\|$
  - **证明**：
    - 由C-S不等式得 $\|x_{k+1}-x_k\| \leq \|M\|^k\cdot\| x_1-x_0\|$
    - 再由三角不等式 + 裂项得 $\|x_*-x_k\| < \cfrac{1-\|M\|^\infty}{1-\|M\|}\cdot\|x_{k+1}-x_k\|$
    - 两边综合即得结果
- **后验误差估计**：
  - 若 $\|M\| < 1$
  - 则 $\|x_k-x^*\| \leq \cfrac{\|M\|}{1-\|M\|} \|x_{k}-x_{k-1}\|$
  - **证明**：
    - 同上即可

### 两种收敛速度

- <font color='chartreuse'>已知单步定常迭代法的收敛性由 $M$ 的范数决定，故可用 $M^k$ 的范数大小来定义收敛速度</font>
- **平均收敛速度**：$R_k(M) = -\ln\|M^k\|^{\dfrac{1}{k}}$
  - <font color='chartreuse'>它表示前 $k$ 次迭代的收敛速度的对数平均值。值越大说明收敛速度越快
  - 它的定义依赖于具体范数</font>
- **比例因子**：$\sigma = \Big( \cfrac{\|y_k\|}{\|y_0\|} \Big)^{\dfrac{1}{k}} \approx \|M^k\|^{\dfrac{1}{k}} = e^{-R_k(M)}$
  - <font color='chartreuse'>它表示   </font>
- **渐进收敛速度**：$R_\infty(M) = \lim\limits_{k\to\infty} R_k(M)$
  - <font color='chartreuse'>它表示无穷次迭代后，最终稳定下来的收敛速度</font>
- **渐进定理**：$R_\infty(M) = -\ln\rho(M)$
  - <font color='chartreuse'>它不依赖于具体范数</font>
  - **互包证明**：
    - **左大于右**： $\rho(M)\leq \|M^k\|^{\dfrac{1}{k}}$
      - 由特征值是相似不变量得 $\rho(M)^k = \rho(M^k)$
      - 由谱范数是范数下确界得 $\rho(M^k) \leq \|M^k\|$
    - **左小于右**：
      - 由特征值的相似不变性得 $\forall \e > 0$，都有 $\lim\limits_{k\to\infty} \left( \cfrac{M}{\rho(M)+\varepsilon} \right)^k = 0$
      - 由极限保号性得存在 $k$ 使得 $\|M^k\| \leq (\rho(M)+\varepsilon)^k$

## Seidel技巧

- **Seidel技巧**：在迭代法中，后算出来值的近似效果更好，所以在计算 $x_{k}$ 的第 $i$ 分量时，总是将前 $i-1$ 个分量都取 $x_{k}$ 的，而不是 $x_{k-1}$ 的
- **Seidel迭代法**：将单步定常迭代法用Seidel技巧优化
- **递推公式**：设 $M = L+U$，其中 $L$ 是严格下三角阵，$U$ 是上三角阵，则递推公式为 $$\bs x_{k} = L\bs x_{k} + U\bs x_{k-1} + g$$，一般形式为 $$\bs x_{k} = (I-L)^{-1}U\bs x_{k-1} + (I-L)^{-1}g$$

### Seidel收敛判别法

- **无穷范数定理**：
  - 设 $M$ 是原迭代矩阵，$L_1$ 是Seidel迭代矩阵
  - 若 $\|M\|_\infty < 1$，则 $\|L_1\|_\infty < 1$
  - **误差估计引理**：
    - G-S迭代法的通项误差估计式为：$\|x_k-x_*\|_\infty \leq \cfrac{\mu^k}{1-\mu} \|x_1-x_0\|_\infty$
      - 其中 $\mu = \max\limits_i\Big\{\cfrac{\sum\limits^n_{j=i+1} |b_{ij}|}{1-\sum\limits^{i-1}_{j=1}|b_{ij}|} \Big\} \leq \|B\|_\infty < 1$
      - <font color='chartreuse'>若设
        - 严格下三角矩阵 $L$ 的第 $i$ 行元素之和 $l_i = \sum\limits^{i-1}_{j=1} |b_{ij}|$
        - 严格上三角矩阵 $U$ 的第 $i$ 行元素之和 $u_i = \sum\limits^n_{i=j+1} |b_{ij}|$
      - 则 $\mu = \max\limits_i \cfrac{u_i}{1-l_i}$</font>
    - **证**：
      - 由行和范数的定义易得 $l_i+u_i \leq \|B\|_\infty < 1$，变形得 $\cfrac{u_i}{1-l_i} < l_i+u_i$
      - 两边都取最大值即得 $\mu < 1$
  - **定理证明**：
    - 设 $x\in \R^n$ 满足 $\|L_1\|_\infty = \|L_1x\|_\infty$，$y = L_1x$
      - <font color='chartreuse'>即 $x$ 是被 $L_1$ 映射后最长的单位球面向量，$y$ 是其像</font>
    - 取 $|y_k| = \max\limits_i \|y_i\| = \|y\|_\infty$
    - 将 $L_1$ 定义式变形可得 $y = D^{-1}Ly + D^{-1}Ux$
      - 计算得对任意 $i$ 都有 $y_i = \sum\limits^{i-1}_{j=1} b_{ij}y_j + \sum\limits^n_{j=i+1} b_{ij}x_j$
        - <font color='chartreuse'>因为 $B = D^{-1}L + D^{-1}U$。前者是 $B$ 的下三角部分，后者是 $B$ 的上三角部分，它们在乘积中是相互独立的</font>
      - 两边取绝对值 + 三角不等式 + 最值放缩得 $\|y\|_\infty \leq \|y\|_\infty\cdot l_i + u_i$
        - <font color='chartreuse'>两边都取 $i = k$，则左边变为无穷范数
        - 将右边第一项的所有 $y_j$ 均放缩为 $\max\limits_j y_j = \|y\|_{\infty}$，从而提取公因式
        - 再由 $x$ 是单位球面向量，故 $\|x\|_{\infty} = 1$，即右边第二项的 $x_j$ 均放缩为 $\max\limits_j x_j = 1$</font>
    - 再由误差估计引理即得结论
      - <font color='chartreuse'>上式变形可得 $\|y\|_{\infty} \leq \cfrac{u_i}{1-l_i} = \mu < 1$</font>
    - <font color='red'>这玩意的过程没法学，记个结论得了</font>
- **1-范数定理**：
  - 若 $\|M\|_1 < 1$，则Seidel迭代法收敛
  - **误差估计引理**：
    - G-S迭代法的通项误差估计式为：$\|x_k-x_*\|_1 \leq \cfrac{\wt\mu^k}{(1-\wt\mu)(1-s)} \|x_1-x_0\|_1$
    - 其中
      - $s = \max\limits_j\sum\limits^n_{i=j+1} |b_{ij}|$
      - $\mu = \max\Big\{\cfrac{\sum\limits^{j-1}_{i=1} |b_{ij}|}{1-\sum\limits^{n}_{i=j+1}|b_{ij}|} \Big\} \leq \|B\|_1 < 1$
    - <font color='chartreuse'>此时 $\mu = \max\limits_i\cfrac{u_i}{1-l_i}$ 的定义同上，是 $L$ 和 $U$ 的行和函数
    - 而 $s$ 仅表示下三角矩阵 $L$ 的最大列和</font>

## 两个经典方法

### Jacobi迭代法

- **思路**：希望构造一个线性递推序列，使其收敛到方程组 $Ax =b$ 的解
- **思想**：
  - 已知方程组 $Ax = b$
  - 将矩阵 $A$ 分裂的和，比如将方程组化为 $(D-L-U)x = b$
    - <font color='chartreuse'>这里 $D$ 是 $A$ 的对角部分，$-L,-U$ 分别是 $A$ 的上三角、下三角部分
      - 原因1是希望拆为结构简单的矩阵，即三角阵和对角阵
      - 原因2是由戈氏圆盘定理，严格对角占优阵必定非奇异，且谱半径容易估计</font>
  - 因为 $D$ 容易求逆，故变形为 $Dx = (L+U)x + b$
  - 再写成标准形式即可
- **Jacobi迭代法**：
  - **递推公式**：$x_k = D^{-1}(L+U)x_{k-1} + D^{-1}b$
    - <font color='chartreuse'>迭代矩阵 $B = D^{-1}(L+U)$
    - 常数项 $g = D^{-1}b$</font>
- **应用条件**：$D$ 必须可逆（$A$ 的对角元均不为 $0$）
- **换序法**：
  - 可以通过添加置换矩阵 $P$ 使得 $D$ 可逆，即 $B = PAP^T$
  - 设 $y = P^Tx$，则 $By = Pb$ 和 $Ax = b$ 同解
  - 迭代求出 $y$ 后再计算 $x = Py$ 即可

### 习题

- **Jacobi精确解**：若 $\rho(M) = 0$，则Jacobi迭代 $n$ 次后可得到精确解
  - **证明**：
    - 由谱范数为 $0$，得 $M$ 的所有特征值均为0，即其相似于Jordan标准型 $J(0)$
    - 由上升性，$J(0)^n = O$，故 $M^n = P^{-1}J^nP = O$
    - 从而误差 $y_n = M^n y_0 = 0$，是精确解

## 常用收敛判别法


### 对称阵：正定性

- **正定收敛定理**：
  - 若 $A$ 对称，对角元均为正，则Jacobi迭代法收敛 $\LR A、2D-A$ 均正定
    - **证明**：
      - 计算易得 $B = I-D^{-1}A = D^{-\dfrac{1}{2}}(I-D^{-\dfrac{1}{2}}AD^{-\dfrac{1}{2}})D^{\dfrac{1}{2}}$，再设右边括号中的矩阵为 $C$
        - 即 $B$ 与 $C$ 关于过渡矩阵 $\sqrt{D}$ 相似，特征值相同
        - 再易得 $B,C$ 均对称，从而均可对角化，且特征值均为实数
        - <font color='chartreuse'>这个过程只是说明一下 $B$ 的特征值是实数而已，并不重要</font>
      - 已知Jacobi收敛性等价于 $\rho(B)<1$，即等价于 $\begin{cases} I-B = D^{-1}A \\\\ I+B = 2I-D^{-1}A \end{cases}$ 的特征值均为正。再由 $D$ 的元素均为正，得它等价于题设条件
    - <font color='red'>实际上就是把谱半径问题（特征值问题）转化为正定问题而已</font>

  - 若 $A$ 对称正定，则G-S迭代法收敛

### 对称阵：可约性/对角占优性

- **弱严格对角占优矩阵**：$|a_{ii}| \geqslant \sum\limits^n_{\substack{j=1 \\ j\neq i}} |a_{ij}|$，至少有一个不等号成立
- **置换方阵**：每行每列都有一个元素为 $1$，其余均为 $0$
- **可约矩阵**：存在置换方阵 $P$，使得 $PAP^T = \tvek{A_{11} & 0}{A_{12} & A_{22}}$
  - <font color='chartreuse'>因为 $P$ 是置换方阵，所以 $A$ 实际上只是被调换了行与列的顺序而已，也就是说 $A$ 本来就是稀疏矩阵，只是把其中的零元素聚集到了一起</font>
- **可分解矩阵**：设 $W = \{1,\cdots,n\}$，存在不相交分割 $S,T$ 使得当 $i\in S,j\in T$ 时 $a_{ij} = 0$
  - <font color='chartreuse'>$S$ 对应左边元，$T$ 对应右边元，$a_{ij}$ 对应右上角</font>
- **分解定理**：$A$ 是可约矩阵 $\LR A$ 是可分解矩阵 $\LR Ax = 0$ 可化为两个低阶方程组
  - **证明**：显然
  - <font color='red'>不可约的几何意义就是有向图的强连通性
    - 如果 $A$ 看作有向图 $T$ 的邻接矩阵，那么 $T$ 强连通 $\LR A$ 不可约</font>
- **奇异性定理**：$A$ 非奇异的充分条件：
  - 严格对角占优
    - **证明1**：戈氏圆盘定理直得
    - **证明2**：
      - 反设 $A$ 奇异，则 $Ax = 0$ 有非零解空间，在解空间中取无穷范数为 $1$ 的向量 $x$，并设 $|x_k| = \|x\|_\infty = 1$
      - 此时有 $Ax = 0$，取其中第 $k$ 个元，移项可得 $a_{kk}x_k = -\sum\limits^n_{\substack{j=1 \\ j\neq k}} a_{kj}x_{j}$
      - 则 $|a_{kk}| = |a_{kk}x_k|  = |\sum\limits^n_{\substack{j=1 \\ j\neq k}} a_{kj}x_{j}| \leq \sum\limits^n_{\substack{j=1 \\ j\neq k}} |a_{kj}|$，与严格对角占优性矛盾
    - **理解**：
      - 实际上严格对角占优是很强的性质，奇异性又基本上等价于有解性，倒腾一下关系式就出来了
      - 唯一有思考量的就是取无穷范数为 $1$ 的 $x$，它的目的是得到向量等式的同时还能消去多余的 $x_j$
    - **本质**：
      - **稀疏矩阵**：哪怕其它元的绝对值均为0，对角元的绝对值也大于0，当然非奇异
      - **非稀疏矩阵**：某个对角元若要被中和，则线性组合系数中必须有一个 $\l_i$ 大于1，但此时第 $i$ 行的对角元就不能被中和了。故永远不可能整行被中和，即非奇异
  - 不可约 + 弱严格对角占优
    - **证明1**：戈氏圆盘定理直得
    - **证明2**：
      - 反设 $A$ 奇异，则 $Ax = 0$ 有非零解空间，在解空间中取无穷范数为 $1$ 的向量 $x$
        - <font color='chartreuse'>显然两种性质的证明思路与方法都是类似的。不同点在于这里的对角占优条件是不严格的，所以我们必须把上面证明过程的（不等号）加强为（完全不等号）才能得出矛盾</font>
      - 易得 $S = \{i: |x_i| = 1\}$，$T = \{k: |x_k| < 1\}$ 是一个分割
        - <font color='chartreuse'>对 $x$ 的各个元素给出人为划分，即 $S$ 是优元集合，$T$ 是劣元集合</font>
      - 对 $Ax = 0$ 变形得对 $\forall i\in S$ 都有 $|a_{ii}x_i| \leq \sum\limits_{\substack{j\in S \\ j\neq i}}|a_{ij}||x_j| + \sum\limits_{j\in T}|a_{ij}||x_j|$
        - <font color='chartreuse'>这个式子已经很接近对角占优定义式了。如果要得出矛盾，我们希望将上式删去 $x_j$ 项的同时，再删去取等情况。显然将 $x_j$ 放缩为 $1$ 即可。但是需要证明 $T$ 非空
        - 直接反设 $T$ 为空集，则上式可将 $x_j$ 项直接退化掉。此时只能取等为 $|a_{ii}| = \sum\limits_{\substack{j\in S \\ j\neq i}} |a_{ij}|$，否则与 $A$ 的对角占优性矛盾。但是对角占优同时又要求至少有一行不取等，但此时 $S$ 就是所有行，故最终还是有矛盾</font>
      - <font color='chartreuse'>然后再考虑另外一种例外情况，即某行除对角元外其它元素均为 $0$。我们证明它可被 $A$ 的不可约性排除掉</font>
      - 由 $A$ 的不可约性，必定存在 $i\in S, k\in T$ 使得 $a_{ik} \neq 0$
        - <font color='chartreuse'>这一步完全是 $A$ 的性质引出的，和 $x$、$S$、$T$ 的定义都没有关系，但之后会用它和 $x$ 搭配
        - 此时由 $k\in T$ 得 $|a_{ik}x_k| < |a_{ik}|$，即（完全不等号）是一定存在的，可以排除所有的例外情况</font>
      - 最终，我们可将上面的不等式进行最值放缩，得到 $|a_{ii}| < \sum\limits_{j\neq i} |a_{ij}|$，与对角占优性矛盾
        - <font color='chartreuse'>将 $T$ 中的 $x_j$ 放大为 $1$，再由 $S$ 中的 $x_j = 1$，即得到最终的不等式</font>
    - **理解**：
      - 实际上用对角占优性倒腾倒腾就能得出结论了。上面证明的大部分过程都是在用不可约性来排除例外情况
    - **本质**：
      - **稀疏矩阵**：连分块的零矩阵都不能有，当然也不可能存在退化的整行了
      - **非稀疏矩阵**：对角占优说明无论怎么线性组合，某个行中必定都会出现错位元素，从而无法被归零，从而非奇异
  - **总结**：
    - 对角占优性离非奇异性其实只差一步之遥

## 松弛迭代法

- **松弛迭代法**：设 $\D x = x_{k+1} - x_k$，则将添加比例系数 $w$ 后的公式 $x_{k+1} = x_k + w\D x$ 称为松弛迭代法
  - **修正项 $\D x$**：每一步的迭代差
  - **松弛因子 $w$**：表示加速程度
    - **超松弛迭代法**：若 $w > 1$，则新值占比更多，称为超松弛迭代法
    - **低松弛迭代法**：若 $w < 1$，则新值占比更少，称为低松弛迭代法
    - <font color='red'>低松弛情况下，上一步的旧值占大多数，当前步的新值被人为缩小，故总能保持收敛性
    - 超松弛情况下，新值被人为放大，很容易造成误差变大，从而振荡发散
    - $w$ 的大小和收敛速度不是正相关的，需要结合具体谱半径情况判断</font>

### 广义Richardson迭代法

- **递推公式**：$x_{k+1} = x_k + w_kP_k^{-1}(b-Ax_k)$
- **分类**：
  - **定常情况（松弛迭代法）**：$w,P_k$ 都是常数
  - **非定常情况**：比如最速下降法、共轭梯度法等

### 标准Richardson迭代法

- **思想**：广义R的定常情况，且 $P = I$
- **递推公式**：$x_{k+1} = x_k - w(Ax_k-b)$
  - <font color='chartreuse'>迭代矩阵：$M_w = I-wA$
  - 常数项：$g = wb$</font>
- **正定收敛定理**：设 $A$ 是正定对称阵，则 $w\in (0,\dfrac{2}{\l_{\max}})$ 时R迭代法收敛
  - **证明**：
    - 设 $\l_A$ 是 $A$ 的某个特征值，则收敛等价于 $|1-w\l_A| < 1$，再将其实虚部展开即得等价于 $w\in (0,\min\dfrac{\Re \l_A}{|\l_A|^2})$
    - 再由正定对称性，特征值均为正实数，从而得到结论
  - **推论**：$w = \cfrac{2}{\l_{\min} + \l_{\max}}$ 时收敛速度最快
    - **证明**：
      - 只需取使得谱半径 $\max|1-w\l_A|$ 最小的 $w$ 即可
      - 由运筹学的最大最小定理，此时只需 $1-w\l_{\min} = -(1-w\l_{\max})$ 即可，变形即得结论

### JOR松弛法

- **思想**：广义R的定常情况，且 $P = D$
- **递推公式**：$$x_{k+1} = (I-wD^{-1}A)x_k + wD^{-1}b$$
  - <font color='chartreuse'>迭代矩阵：$B_w = I-wD^{-1}A$</font>
- **低松弛保收敛定理**：若 $A$ 对应的Jacobi迭代法收敛，则 $w\in (0,1]$ 的JOR迭代法也收敛
  - **证明**：
    - **特征值计算**：若 $B$ 的特征值为 $\l$，则 $B_w$ 的相应特征值为 $1-(1-\l)w$
      - **证**：<font color='chartreuse'>直接计算即可</font>
        - 易得 $\begin{cases} B = I - D^{-1}A \\ B_w = (1-w)I + wB \end{cases}$
        - 设 $B$ 关于 $\l$ 的特征向量为 $v$，则 $B_wv = (1-w)v + w\l v = (1-w+w\l)v$
        - 故 $v$ 在 $B_w$ 下的特征值为 $1-w+w\l$
    - 已知收敛的充要条件为 $|1-(1-\l)w| < 1$，再由题设得 $|\l| < 1$，讨论即得结论
- **正定收敛定理**：
  - 设 $A$ 是实对称矩阵，对角元均为正
  - 则JOR迭代法收敛 $\LR A，2w^{-1}D-A$ 均为正定对称矩阵
  - **证明**：
    - 仿照非松弛情况即可
- <font color='red'>JOR更多是一个过渡概念，它的提速效果有限，且没有漂亮的收敛理论</font>

### 超松弛迭代法（SOR）

- **思想**：广义R的定常情况，且 $P = D-wL$
  - <font color='red'>它不再像JOR一样直接加权外推，而是在内部增加了系数，变为 $D-wL$
  - 因为G-S迭代法的基本思想是Seidel技巧，在松弛情况下要继续沿用这一点</font>
- **递推公式**：$$x_{k+1} = L_w x_k + w(D-wL)^{-1}b$$
  - <font color='chartreuse'>迭代矩阵：$L_w = (D-wL)^{-1}\Big[ (1-w)D + wU \Big]$</font>

#### 收敛性

- **收敛必要条件**：SOR迭代法收敛 $\red\Rt w\in (0,2)$
  - <font color='chartreuse'>直接算就完了</font>
  - **证明**：
    - 由迭代法收敛得 $\rho(L_w) < 1$，即特征值乘积 $|\l_1...\l_n| < 1$，从而 $\det(L_w) < 1$
    - 计算易得 $|\det L_w| = |\det()|\cdot|\det()| = |(1-w)^n\cdot 1|$，即得 $|1-w|<1$（**证毕**）
- **收敛充分条件**：
  - **对角收敛定理**：若 $w\in (0,1)$，且 $A$ （严格对角占优）或（不可约弱严格对角占优），则收敛
    - <font color='red'>这其实就是普通的低松弛保收敛</font>
    - **证明**：
      - 和之前证明Jacobi和G-S的方法相同
  - **正定收敛定理**：若 $w\in (0,2)$，且 $A$ （正定实对称），则收敛
    - <font color='red'>正定实对称阵自带能量衰减结构，故可以适当超松弛</font>
    - **证明**：
      - <font color='chartreuse'>要证明迭代法收敛，只需证明 $L_w$ 的特征值均小于 $1$ 即可</font>
      - **特征值计算**：设 $x$ 是 $L_w$ 的特征向量，$\l$ 是对应特征值。则有 $L_w x = \l x$
        - <font color='chartreuse'> 将其展开，然后左乘 $x^T$
        - 设 $\begin{cases} x^TDx = \d \\ x^TLx = \a+i\b \\ x^TL^Tx = \a-i\b \end{cases}$
        - 则特征方程最终可变形为 $|\l|^2 = \cfrac{\Big[ (1-w)\d + w\a \Big]^2 + w^2\b^2}{(\d-w\a)^2+w^2\b^2}$</font>
      - 现在我们要证明 $|\l| < 1$
        - 计算得 $|\l|^2-1$ 的分子为 $w\d(\d-2\a)(w-2)$
        - 由 $A$ 的正定性，$D$ 的对角元均为正，从而 $\d,\d-2\a > 0$，故 $w\in (0,2)$ 时 $|\l|^2<1$，从而收敛
    - **理解**：
      - 就是算，只不过步骤看起来比较复杂。实际上处理一下就好，没有太多需要创新的地方
