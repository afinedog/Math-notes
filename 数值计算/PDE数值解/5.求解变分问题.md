# 求解变分问题


## Ritz-Galekin法

- **思想**：先在有穷维子空间求解，再逼近无穷维
- **符号约定**：
  - $U$ 表示Sobolev空间，$H = H^0$ 是 $L^2$ 空间
  - $L$ 是微分算子，$a(u,v)$ 是 $L$ 和边值条件所对应的双线性形式
  - 设 $U_n$ 是 $U$ 的 $n$ 维子空间，$\p_1,...,\p_n$ 是 $U_n$ 的一组基函数，线性表出式 $u_n = \sum\limits^n_{i=1} c_i\p_i$
- **Ritz法**：
  - **思想**：根据极小位能原理，用求偏导的方法，选取系数使得 $J(u_n)$ 取极小值
  - **推导**：
    - 此时 $J(u_n) = \dfrac{1}{2}\sum\limits^n_{i,j=1} a(\p_i,\p_j)c_ic_j - \sum\limits^n_{j=1} c_j(f,\p_j)$
    - 易得驻点方程组 $\dpfrac{J(u_n)}{c_j} = 0$ 可化为 $\sum\limits^n_{i=1} a(\p_i,\p_j)c_i = (f,\p_j)，j=1,...,n$
    - 该线性方程组的解就是 $U_n$ 上的最小值
  - <font color='chartreuse'>力学意义更明显</font>
- **Galekin法**：
  - **思想**：根据虚功原理，求解 $U_n$ 上的虚功方程，从而得到 $U_n$ 上的最佳系数
  - <font color='chartreuse'>最终线性方程组和上面相同
  - 不需要 $a(u,v)$ 对称正定</font>

### 解的收敛性

- **收敛定理**：
  - 设 $u$ 是变分问题的解，$u_n$ 是R-G方法的解
  - 则存在与 $u,n$ 无关的常数 $\b > 0$ 使得 $\|u-u_n\|_1 \leq \b\inf\limits_{v\in U_n} \|u-v\|_1$
  - **证明**：
  - **推论**：若 $\{\p_i\}^\infty_{i=1}$ 的线性组合在 $U$ 上稠密，则解收敛

### 边值条件的齐次化

- **非齐次边值条件**：
  - 对于非齐次的自然边值条件，只需修改右端即可
  - 对于非齐次的本质边值条件，应当齐次化后再使用R-G方法
- **实例**：
  - 前面的两点边值问题，当边值条件为 $u(a) = \a，u'(b) = \b$ 时
  - 右端改为 $(f)$
  - 左端改为 $u_n(x) = u_0(x) + \sum\limits^n_{i=1} c_i\p_i(x)$，其中 $\p_i(a) = 0，u_0(x)$ 是任意满足 $u_0(a) = \a$ 的函数
  - 最终R-G方程变为 $\sum\limits^n_{i=1} a(\p_i,\p_j)c_i = (f,\p_j) + p(b)\b\p_j(b) - a(u_0,\p_j)$
  - <font color='chartreuse'>二维边值问题的 $u_0(x)$ 很难给出，需要用插值法</font>
