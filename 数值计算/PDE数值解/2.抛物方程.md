# 抛物方程的有限差分法

- **驻定问题**：物理状态不随时间改变
  - **实例**：椭圆方程定解问题
- **非驻定问题**：物理状态与时间有关
  - **实例**：抛物方程、双曲方程
- **定态原理**：驻定问题（定态问题）可看作非驻定问题（发展方程问题）最终达到稳定时的情形
  - <font color='chartreuse'>$t$ 对应时间，$x$ 对应空间</font>
  - **证明**：
    - 考虑抛物方程 $u_t = a\D u + f(x)，a>0，x\in\O，0 < t < T$
    - 若解存在时间稳态，即 $t\to\infty$ 时 $u(x,t)\to \ol u(x)$，则 $u_t\to 0$。此时变为椭圆方程 $0 = a\D \ol u + f(x)$
  - <font color='chartreuse'>驻定问题只需考虑边值条件，非驻定问题需要同时考虑初值条件（$t$）和边值条件（$x$）</font>
- **初值问题**：$u(x,0) = \phi(x)，x\in\R$
- **初边值问题**：
  - 初值 $u(x,0) = \phi(x)，x\in (0,l)$
  - 第一边值 $u(0,t) = u(l,t)，t\in [0,T]$
    - <font color='chartreuse'>仅零阶微分</font>
  - 第二边值 $u_x(0,t) = \a，u_x(l,t) = \b$
    - <font color='chartreuse'>仅一阶微分</font>
  - 第三边值 $u(0,t) +\a_1 u_x(0,t) = \b_1，u(l,t) + \a_2 u_x(l,t) = \b_2$

## 一维差分格式

- **一维热方程**：$\begin{cases} u_t = au_{xx} + f(x) & t\in [0,T]，x\in (0,l) \\\\ u(x,0) = \phi(x) & x\in [0,l] \\\\ u(0,t) = 0 = u(l,t) \end{cases}$
- **矩形剖分**：
  - 设矩形区域 $\ol G = \hkh{(x,t) | 0\leq x\leq l，0\leq t\leq T}$，$(x_j,t_n)$ 处的节点为 $u^n_j$
  - 空间步长 $h = \cfrac{l}{J}$，时间步长 $\tau = \cfrac{T}{N}$


### 向前差分格式

- **向前差分格式（时间向前，空间中心）**：直接计算易得 $$ \frac{u^{n+1}_j - u^n_j}{\tau} = a\frac{u^n_{j+1} - 2u_j^n + u^n_{j-1}}{h^2} + f_j $$
- **显格式**：设**网比** $r = \cfrac{a\tau}{h^2}$，上式变形易得 $$ u^{n+1}_j = u^n_j + r(u^n_{j+1} - 2u^n_j + u^n_{j-1}) + \tau f_j$$
  - <font color='chartreuse'>可先固定 $t=n$，逐个算出 $u^n_j$，再逐个计算 $u^{n+1}_j$，不需要解线性方程组</font>
- **截断误差**：易得 $R^n_j(u) = L_h^{(1)}u(x_j,t_n) - [Lu]^n_j$ $$ = -\tau\fkh{\frac{1}{12r} - \frac{1}{2}}[\wt u_{tt}]^n_j + O(\tau^2+h^2)$$
  - <font color='chartreuse'>即 $R^n_j = O(\tau + h^2)$</font>


### 向后差分格式

- **向前差分格式（时间向后，空间中心）**：直接计算易得 $$ \frac{u^{n+1}_j - u^n_j}{\tau} = a\frac{u^{n+1}_{j+1} - 2u_j^{n+1} + u^{n+1}_{j-1}}{h^2} + f_j $$
- **隐格式**：设**网比** $r = \cfrac{a\tau}{h^2}$，上式变形易得 $$ -ru^{n+1}_{j+1} + (1+2r)u^{n+1}_j - ru^{n+1}_{j-1} = u^n_j + \tau f_j$$
  - <font color='chartreuse'>需要联立全体 $u^n_j$ 的方程，解出全体 $u^{n+1}_j$</font>
- **截断误差**：易得 $R^n_j(u) = L_h^{(2)}u(x_j,t_n) - [Lu]^n_j$ $$ = -\tau\fkh{\frac{1}{12r} + \frac{1}{2}}[\wt u_{tt}]^n_j + O(\tau^2+h^2)$$
  - <font color='chartreuse'>即 $R^n_j = O(\tau + h^2)$</font>

### 六点对称格式

- **六点对称格式**：求两种格式的算数平均得 $$ \frac{u^{n+1}_j - u^n_j}{\tau} = \frac{a}{2}\fkh{\frac{u^{n+1}_{j+1} - 2u_j^{n} + u^{n+1}_{j-1}}{h^2} + \frac{u^{n+1}_{j+1} - 2u_j^{n+1} + u^{n+1}_{j-1}}{h^2}} + f_j $$
- **隐格式**：设**网比** $r = \cfrac{a\tau}{h^2}$，上式变形易得 $$ -\frac{r}{2}u^{n+1}_{j+1} + (1+r)u^{n+1}_j - \frac{r}{2}u^{n+1}_{j-1} = \frac{r}{2}u^{n}_{j+1} + (1-r)u^{n}_j - \frac{r}{2}u^{n}_{j-1} + \tau f_j$$
  - <font color='chartreuse'>需要联立全体 $u^n_j$ 的方程，解出全体 $u^{n+1}_j$</font>
- **截断误差**：易得 $R^n_j(u) = L_h^{(3)}u(x_j,t_n) - [Lu]^n_j$，将其在 $(x_j,t_{n+\frac{1}{2}})$ 处展开即可
  - <font color='chartreuse'>即 $R^n_j = O(\tau^2 + h^2)$</font>

### 理查德森格式

- **Richardson格式（时间中心，空间中心）**：直接计算易得 $$ \frac{u^{n+1}_j - u^{n-1}_j}{2\tau} = a\frac{u^{n}_{j+1} - 2u_j^{n} + u^{n}_{j-1}}{h^2} + f_j $$
- **三层显格式**：设**网比** $r = \cfrac{a\tau}{h^2}$，上式变形易得 $$ u^{n+1}_{j} =  2r(u^{n}_{j+1} +2 u^{n}_{j} + u^n_{j-1}) + 2\tau f_j$$
  - <font color='chartreuse'>首先用二层显式差分计算 $u^1_j$，然后逐次计算即可</font>
- **截断误差**：
  - <font color='chartreuse'>即 $R^n_j = O(\tau^2 + h^2)$</font>

### 稳定性

- **按初值稳定**：数值解的范数依赖于初值 $u_0$
  - **实例**：ODE的欧拉法和多步法
- **按右端稳定**：数值解的范数依赖于右端函数 $f$
  - **实例**：椭圆方程
- **二层差分格式的矩阵表示**：设 $U^n = \tvec{u^n_1 \\ \vdots \\ u^n_{J-1}}，F = \tvec{f_1 \\ \vdots \\ f_{J-1}}$，则二层差分格式均可写为 $AU^{n+1} = BU^n + \tau F$
  - **约定**：
    - 要求 $A$ 可逆，设 $C = A^{-1}B$，则可写为**右端方程** $U^{n+1} = CU^n + \tau A^{-1}F$
    - 要求 $A,B$ 不依赖于 $n$
    - 要求 $h = g(\tau)$，满足 $g$ 连续且 $g(0) = 0$
    - <font color='chartreuse'>此时 $A,B,C$ 可看作 $\tau$ 的函数</font>
- **初值稳定定理**：若存在 $\tau_0 > 0,K >0$ 使得任意 $\tau\in (0,\tau_0]，n\tau\in (0,T]$ 都有 $\|C^{n+1}\| \leq K$，则称差分格式按初值稳定
  - <font color='red'>这里的矩阵范数是算子范数</font>
  - **证明**：
    <!-- - <font color='chartreuse'>易得该条件等价于 $\exists \tau_0,K >0$ 使得 $\forall U_0\in\R^{J-1}，\tau\in (0,\tau_0]，n\tau\in (0,T]$ 都有 $\|U^{n+1}\| = \|C^{n+1}U^0\| \leq K\|U^0\|$</font> -->
    - 仅考虑初值稳定性时，不妨令 $F = \bs 0$
      - <font color='chartreuse'>因为此时仅 $U^0$ 变化，$F$ 不变。由于差分方程是线性方程组，故两个不同初值的差分方程直接相减后形式不变，且可消去 $F$</font>
    - 由递推式得此时 $U^{n+1} = C^{n+1}U^0$，从而只需 $\|C^{n+1}U^0\| \leq K\|U^0\|$ 即可
      - <font color='red'>这里的向量范数一般取 $\|u\|^2_0 = \sum\limits^{J-1}_{j=1} hu_j^2$</font>
    - 由算子范数定义得它等价于题设条件
- **右端稳定定理**：若存在 $\|A^{-1}\| \leq K'$，则按初值稳定 $\Rt$ 按右端稳定
  - **证明**：
    - 仅考虑右端稳定性时，不妨令 $U^0 = \bs 0$
      - <font color='chartreuse'>同理，两个不同右端的差分方程直接相减后形式不变，且 $U^0$ 变为 $0$</font>
    - <font color='red'>由定义，只需存在 $\tau_0 > 0,K >0$ 使得任意 $\tau\in (0,\tau_0]，n\tau\in (0,T]$ 都有 $\|U^{n+1}\| \leq K\|F\|$</font>
    - 由递推式得此时 $U^{n+1} = \tau\Big[ C^n + C^{n-1} + ... + C + I \Big]A^{-1}F$
    - 由初值稳定性，可设 $\forall\|C^n\|\leq K''$，故 $$\|U^{n+1}\| \leq \tau(n+1) K'K''\|F\|$$
    - 再由 $n,T$ 定义得 $(n+1)\tau \leq T$，从而取 $K = TK'K''$ 即可
- **稳定定理**：差分格式的稳定性仅依赖于矩阵族 $\hkh{C^n(\tau)\mid \tau\in (0,\tau_0]，0 < n\tau \leq T}$ 的一致有界性

### 稳定性条件

- **稳定必要条件**：存在常数 $M$ 使得谱半径 $\rho(C) \leq 1 + M\tau$
  - **证明**：
    - 由谱半径下界定理 + 初值稳定性得 $\rho^n(C) \leq \|C^n\| \leq K$
    - 不妨设 $K>1，n = [\dfrac{T}{\tau}]$，则 $$\rho(C) \leq \sqrt[n]{K} \leq K^{\dfrac{r}{T-\tau}} = e^{\dfrac{r}{T-\tau}\ln K} \leq  e^{\dfrac{\ln K}{T-\tau_0}\tau} = 1+O(\tau)$$
- **稳定充分条件**：$C(\tau)$ 是正规矩阵
  - **证明**：
    - 此时 $\rho(C) = \|C(\tau)\|$，故 $$\|C^n(\tau)\| \leq \rho^n(C) \leq (1+M\tau)^n \leq (1+M\tau)^{\dfrac{T}{\tau}} \leq K $$
- **对称稳定定理**：
  - $C(\tau)$ 是对称矩阵 $S$ 的实系数有理函数 $R(S)$
  - 则差分格式稳定 $\LR \max\limits_j |R(\l^s_j)| \leq 1+M\tau$
    - <font color='chartreuse'>其中 $\l_j$ 是 $S$ 的特征值</font>
  - **证明**：
    - 由于 $R(S)$ 是实数和 $S$ 的四则运算，故和 $S$ 同阶。再由谱半径的定义即得结论

### 收敛性

- **相容条件**：若 $\lim\limits_{n\to\infty} \|\bs R^n\| = 0$，则称差分算子是边值问题的相容逼近
- **收敛定理**：
  - 若差分方程满足相容条件，且按初值稳定
  - 则差分解收敛到热方程的解，且有误差估计式 $\|\bs E^n\| \leq K\sup\limits_n\|\bs R^n\|$
  - **证明**：
    - **差分解范数估计**
      - 设一维差分格式的解可分解为 $u^n_j = v^n_j + w^n_j$，前项是零初值和非齐次右端方程的解，后项是非零初值和齐次右端方程的解
      - 由按初值稳定 + 右端稳定定理得也按右端稳定，故可设 $\|U\| \leq K\|U^0\| + \|F\|$
        <!-- - $\|V^n\| \leq K_1\|F\|，\|W^n\| \leq K_2\|W^0\| = K_2\|U^0\|$ -->
    - **差分解误差估计**：
      - 设 $u$ 是热方程的解，$u^n_j$ 是差分方程的解，误差 $e^n_j = [u]^n_j - u^n_j$
      - 已知误差方程的向量形式可写为 $E^{n+1} = C(\tau)E^n + \tau A^{-1}R^n$，故由差分解范数估计得 $\|E^n\| \leq K\sup\limits_n \|R^n\|$（**证毕**）    
  - **推论**：
    - 网比 $r\leq \dfrac{1}{2}$ 时，向前差分格式的收敛阶为 $O(\tau + h^2)$
    - 对任意网比，向后差分格式的收敛阶为 $O(\tau + h^2)$
    - 对任意网比，六点对称格式的收敛阶为 $O(\tau^2 + h^2)$

## 傅立叶判稳方法

### 初边值周期问题

- **约定**：仅考虑具有（初值和周期边值条件）的（线性常系数一维抛物方程）
- **二层差分格式的诺伊曼形式**：$\sum\limits_{m\in \mc N_1} a_m u^{n+1}_{j+m} = \sum\limits_{m\in \mc N_0} b_m u^n_{j+m}，j=1,...,J-1$
  - <font color='chartreuse'>$\mc N_0$ 和 $\mc N_1$ 是整数集</font>
  - **实例**：
    - 向前差分格式中
      - $\mc N_0 = \{-1,0,1\}，\mc N_1 = \{0\}$
      - $b_{-1} = b_1 = r，b_0 = 1-2r$
      - $a_0 = 1$
- **差分解的连续延拓**：
  - 取阶梯函数延拓 $\begin{cases} u^0(x) = \phi(x_j) \\ u^n(x) = u^n_j \end{cases}，x_{j-\frac{1}{2}} < x < x_{j+\frac{1}{2}}$
  - 易得阶梯函数平方可积，即 $u^n(x) \in L^2(0,l)$
  - 易得范数转化关系 $\|U^n\|^2 = \sum\limits^{J-1}_{j=0} h(u^n_j)^2 = \|u^n\|^2_{L^2}$
- **傅立叶展开**：
  - 差分方程连续延拓后可写为 $\sum\limits_{m\in \mc N_1} a_m u^{n+1}(x+x_{m}) = \sum\limits_{m\in \mc N_0} b_m u^n(x+x_{m})$
  - 将 $u^n$ 展成傅里叶级数后变为 $$ \sum^\infty_{p=-\infty} v^{n+1}_p \fkh{\sum_{m\in\mc N_1} a_me^{i\dfrac{2p\pi}{l}x_m} } e^{i\dfrac{2p\pi}{l}x} = \sum^\infty_{p=-\infty} v^{n}_p \fkh{\sum_{m\in\mc N_0} b_me^{i\dfrac{2p\pi}{l}x_m} } e^{i\dfrac{2p\pi}{l}x} $$
    - <font color='chartreuse'>其中 $v^n_p = \dis\frac{1}{l}\int^l_0 u^n(x)e^{-i\dfrac{2p\pi}{l}x}dx$ 是相应的傅立叶系数</font>
  - 比较对应项的系数可得 $$ v^{n+1}_p = G(ph,\tau)v^n_p $$
    - <font color='chartreuse'>其中 $G(ph,\tau) = \cfrac{\sum b_m...}{\sum a_m ...}$ 称为增长因子</font>
- **稳定性**
  - 由Parseval等式和增长因子递推式可得 $$\|u^n\|^2_{L^2} = l\sum\limits^\infty_{p=-\infty}|v_p^n|^2 = l\sum^\infty_{p=-\infty}|G^n(ph,\tau)v^0_p|^2 $$
  - 若差分格式稳定，则 $$\sum^\infty_{p=-\infty}|G^n(ph,\tau)v^0_p|^2 \leq K^2 \|u^0\|^2_{L^2}$$
  - 再由于全体阶梯函数在 $L^2$ 中稠密，故可取 $u^0(x)$ 满足 $\|u^0\|_{L^2} = 1$ 且 $\begin{cases} v^0_p = 1 \\ v^0_q = 0 & q\neq p \end{cases}$，则上式变为 $$ |G^n(ph,\tau)| \leq K\quad (0 < \tau \leq \tau_0，0 < n\tau \leq T) $$
- **冯诺依曼条件**：$|G(ph,\tau)| \leq 1+M\tau$
- **冯诺依曼定理**：差分格式稳定 $\LR G^n(ph,\tau)$ 一致有界 $\LR$ 冯诺依曼条件成立
  - **证明**：
    - $(1)\to (2)$ 上面已经证明
    - $(2)\to (1)$ 由上面过程易得
    - $(1)\to (3)$：
      - 此时 $G(ph,\tau) \leq K^{\dfrac{1}{n}}$
      - 取 $n = [\dfrac{T}{\tau}]$ 即得 $\dfrac{1}{n} \leq \dfrac{\tau}{T-\tau}$，从而 $$|G(ph,\tau)| \leq K^{\dfrac{\tau}{T-\tau}}  \leq  e^{\dfrac{\tau}{T-\tau}\ln K} = 1+M\tau$$



#### 习题

- **符号约定**：设 $\a = \dfrac{2\pi p}{l}$ 表示频率，$x_p = ph$ 表示空间网点
- **增长因子计算方法**：将傅立叶级数通项代入差分格式两端，变形即得 $v^{n+1} = Gv^n$
- **向前差分格式**：
  - 代入傅里叶级数通项 $u^n_j = v^ne^{i\a x_j}$，计算即得 $G(x_p,\tau) = 1-4r\sin^2 \dfrac{\a h}{2}$
  - 由于 $h\to 0$ 时 $\dfrac{\a h}{2}$ 稠密，故满足冯诺依曼条件等价于 $r\leq \dfrac{1}{2}$

### 纯初值问题

- **约定**：仅考虑具有（初值）的（线性常系数一维抛物方程）
- 只需将傅立叶级数替换为傅立叶变换即可

### 差分方程组

- **差分格式的诺伊曼形式**：$\sum\limits_{m\in N_1} A_m U^{n+1}_{j+m} = \sum\limits_{m\in N_0} B_m U^n_{j+m}，j=1,...,J-1$
  - <font color='chartreuse'>$A_m,B_m$ 是关于 $\tau$ 的方阵，$U$ 是列向量</font>
  - 将 $U$ 连续延拓后取傅里叶级数 $U^n(x) = \sum\limits^\infty_{p=-\infty} V^n_p e^{i\dfrac{2\pi p}{l}x}$
  - 代入后比较相应项的系数即得增长矩阵
- **等价条件**：$\hkh{G^n(x_p,\tau) \mid 0 < \tau \leq \tau_0，n < n\tau \leq T，p=0,...,J-1}$ 一致有界
  - <font color='chartreuse'>此时冯诺依曼条件取增长矩阵的欧氏模，即 $|G| \leq 1+O(\tau)$，它只是稳定的必要条件</font>
  - **证明**：
    - 易得 $G(x_p,\tau)$ 关于 $x_p,\tau$ 连续，且关于 $x_p$ 以 $l$ 为周期，故只需在周期域上研究即可

#### 习题

- **Richardson格式**：写成方程组形式 $\begin{cases} u^{n+1}_{j} =  2r(u^{n}_{j+1} +2 u^{n}_{j} + u^n_{j-1}) + w^n_j \\ w^{n+1}_j = u^n_j \end{cases}$
  - 代入傅立叶级数通项 $u^n_j = v^n_1 e^{i\a x_j}，w^n_j = v^n_2 e^{i\a x_j}$，变形即得 $$\begin{cases} v^{n+1}_2 = 4r(\cos \a h - 1)v^n_1 + v^n_2 \\ v^{n+1}_2 = v^n_1 \end{cases}$$
  - 即增长矩阵 $G(\a h) = \tvec{-8r\sin^2 \dfrac{\a h}{2} & 1 \\ 1 & 0}$
  - 易得 $\rho(G)$ 对任意 $r$ 均不满足冯诺依曼条件，故恒不稳定
- **正规定理**：对于差分方程组，当增长矩阵 $G$ 是正规矩阵时，冯诺依曼条件 $\LR$ 稳定
  - **证明**：此时 $|G| = \rho(G)$

### 二维差分格式

- **约定**：仅考虑求解域为全空间（纯初值问题）或超长方体（周期边值条件）的情况
- **二维热方程初边值问题**：$\begin{cases} u_t = a(u_{xx}+u_{yy})，(0<x,y<l，a>0) \\ u(x,y,0) = \p(x,y) \\ u(0,y,t) = u(l,y,t) = 0，u(x,0,t) = u(x,l,t) = 0\end{cases}$
- **剖分**：
  - 取均匀步长 $h = \cfrac{l}{J}，\tau = \cfrac{T}{N}$，将求解域划分为矩形网格
  - 设二阶差分算子 $$\begin{matrix} \d^2_x = u^n_{j+1,k} - 2u^n_{j,k} + u^n_{j-1,k} \\\\ \d^2_y = u^n_{j,k+1} - 2u^n_{j,k} + u^n_{j,k-1} \end{matrix}$$
  - **向前差分格式**：$\cfrac{u^{n+1}_{j,k} - u^{n}_{j,k}}{\tau} = \cfrac{a}{h^2}(\d^2_x u^{n}_{j,k} + \d^2_y u^{n}_{j,k})$
    - 此时傅立叶级数通项为 $$u^n_{j,k} = v^ne^{i(\dfrac{2\pi p}{l}x_j + \dfrac{2\pi q}{l}y_k)} $$
    - 将通项代入差分格式后变形即得 $$ G(\a h,\b h) = 1-4r\dkh{\sin^2\frac{\a h}{2} + \sin^2\frac{\b h}{2}}$$
    - 即差分格式稳定 $\LR r\leq \dfrac{1}{4}$


## 代数判稳方法

- **约定**：仅考虑差分方程组
- **利普西茨方法**：若 $G(x_p,\tau)$ 关于 $\tau$ 在 $\tau=0$ 处满足Lip条件，则稳定等价于
- **时间无关方法**：若增长因子与 $\tau$ 无关，则稳定 $\LR$
- **一致对角化定理**：若 $\forall x_p,\tau，\exists H$ 使得 $H^{-1}GH$ 是对角阵，且 $H,H^{-1}$ 关于 $p$ 和充分小的 $\tau$ 一致有界，则差分格式稳定 $\LR$ 冯诺依曼条件
- **二阶特征值方法**：
  - 设 $G$ 是二阶矩阵，$g_{ij}$ 是矩阵元，$\l_1,\l_2$ 是特征值
  - 若
    - $|\l_i| \leq 1+M\tau$
    - $\|G-\cfrac{g_{11}+g_{22}}{2}I\| \leq M\dkh{\tau + \big|1-|\l_1|\big| + |\l_1-\l_2|}$
  - 则增长矩阵族一致有界


## 分数步长法

### 交替方向隐式法（ADI）

- **剖分**：取空间步长 $h = \cfrac{l}{M}$，时间步长 $\tau > 0$，将空间域分割成 $M^2$ 个矩形
- **差分化**：
  - 第 $n$ 层到第 $n+\dfrac{1}{2}$ 层中，对 $u_{xx}$ 向后差分，对 $u_{yy}$ 向前差分
  - 第 $n+\dfrac{1}{2}$ 层到第 $n$ 层中，对 $u_{xx}$ 向前差分，对 $u_{yy}$ 向后差分
- **差分方程**：
  - <font color='chartreuse'>已知 $u^n_{j,k}$ 时，首先按行解三对角方程组求出 $u^{n+\dfrac{1}{2}}_{j,k}$，再按列解三对角方程组求 $u^{n+1}_{j,k}$ 即可</font>
- **误差估计**：
  - 将两式相减相加，可消去过渡值 $u^{n+\frac{1}{2}}_{j,k}$，得到 $$(I+\frac{1}{4}\frac{\tau^2}{h^4}\d^2_x\d^2_y)\frac{u^{n+1}_{j,k} - u^n_{j,k}}{\tau} = \frac{1}{h^2}(\d^2_x + \d^2_y)\frac{u^{n+1}_{j,k} - u^n_{j,k}}{2}$$
  - 泰勒展开可得截断误差为 $O(\tau^2 + h^2)$
- **稳定性**：
  - 改写成诺伊曼形式，发现两式可以合并成 $$(I-\frac{r}{2}\d^2_x)(I-\frac{r}{2}\d^2_y)u^{n+1}_{j,k} = (I+\frac{r}{2}\d^2_x)(I+\frac{r}{2}\d^2_y)u^n_{j,k}$$
  - 再代入傅立叶级数通项 $u^n_{j,k} = v^ne^{i(\a x_j + \b y_k)}$ 即得增长因子
  - 易得恒有 $|G| \leq 1$，故绝对稳定
