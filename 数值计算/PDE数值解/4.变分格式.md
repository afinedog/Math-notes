# 边值问题的变分形式

## 变分法原理


### 最速降线问题

- **物理问题**：给定起点 $A(0,0)$ 和终点 $B(x_1,y_1)$，求小球只在重力作用下，沿什么曲线的坡下降速度最快
- **数学模型**：
  - 由机械能守恒，$mgy = \dfrac{1}{2}mv^2$，即 $v = \sqrt{2gy}$
  - 由数分知识，弧长微元 $ds = \sqrt{1+(y')^2}dx$
  - 即时间微元 $dt = \dfrac{ds}{v}$，总时间 $T = \dis\int^{x_1}_0 dt = \int^{x_1}_0 \frac{\sqrt{1+(y')^2}}{\sqrt{2gy}}dx$，要求函数 $y(x)$ 使得 $T$ 最小
- **变分法原理**：
  - 当求泛函 $J[y] = \dis\int^b_a F(x,y,y')dx$ 的极值时，只需代入E-L方程 $\dpfrac{F}{y} - \dfrac{d}{dx}\dpfrac{F}{y'} = 0$ 后，将其转化为求解微分方程即可
  - <font color='chartreuse'>详见后面的极小位能原理</font>
- **转化**：
  - 由链式法则，E-L方程两边对 $y$ 积分得 $F-y'\dpfrac{F}{y'} = C$
  - 代入 $F = \cfrac{\sqrt{1+(y')^2}}{\sqrt{y}}$ 后计算得 $\cfrac{1}{\sqrt{y}\sqrt{1+(y')^2}} = C$
  - 两边平方后变形可写为 $y(1+(y')^2) = 2a$
- **求解ODE**：
  - 设 $y' = \cfrac{1}{\tan\frac{\t}{2}}$，则可化为 $y = a(1-\cos\t)$
  - 将上式微分得 $dy = a\sin\t d\t$，从而 $dx = \dfrac{dy}{y'} = a(1-\cos\t)d\t$
  - 两边积分得 $x = a(\t - \sin\t) + C$，从而得到参数解（摆线）
  - 最后由初值条件得 $C = 0$

### 弦平衡问题

- **物理问题**：设弦长度为 $l$，两端固定在 $A(0,0)$ 和 $B(l,0)$
  - 当有强度为 $f(x)$ 的外荷载垂直向下作用在弦上，$u(x)$ 表示弦的平衡位置，$T$ 表示弦的张力
  - **边值问题**：力学原理得平衡位置满足 $\begin{cases} -Tu'' = f(x)\quad 0<x<l & 微分方程 \\\\ u(0) = 0，u(l) = 0 & 边值条件\end{cases}$
  - **变分问题**：由极小位能原理，平衡位置的总位能最小
    - 应变能 $W_内 = \dis\frac{1}{2}\int^l_0 Tu'(x)^2dx$
    - 外力功 $W_外 = -\dis\int^l_0 fudx$
    - 总位能 $J(u) = W_内 + W_外$，只需求该泛函的最小值即可
    - <font color='chartreuse'>求值的前提是这些数学公式都有意义</font>
  - **函数空间**：
    - 当 $f\in H^0(I)，u\in H^1(I)$ 时，位能 $J(u)$ 恒有意义
    - 设 $H^1(I)$ 中全体满足齐次边值条件的函数为 $H^1_0(I)$，则变分问题可描述为求 $u\in H^1_0$ 使得 $J(u_*) = \min\limits_{u\in H^1_0} J(u)$
  - **Lp内积形式**：由边值条件 + 分部积分公式易得 $$ J(u) = \frac{1}{2}(Lu,u) - (f,u)$$

### 变分法原理（欧氏空间）

- **极小位能原理**：设矩阵 $A$ 对称正定，$J(\bs x) = \dfrac{1}{2}x^TAx - b^Tx$，则下列两问题等价
  - **极值问题**：求 $\bs x_0\in \R^n$ 使得 $J(\bs x_0) = \min\limits_{\R^n} J(\bs x)$
  - **线性方程组问题**：求方程组 $Ax = b$ 的解
  - <font color='chartreuse'>在力学中 $J$ 表示能量</font>

## Soblev空间

### 广义导数

- **紧支光滑函数类**：设 $C^\infty_0(I)$ 表示区间 $I$ 上无穷阶可微，且在端点某邻域上为 $0$ 的函数
- **广义导数**：<font color='chartreuse'>用分部积分定义的导数</font>
  - 设 $f\in L^2(I)$，若存在 $g\in L^2(I)$ 满足 $\forall \p\in C^\infty_0(I)$ 都有 $$ \int^b_a g(x)\p(x) dx =  -\int^b_a f(x)\p'(x)dx $$，则称 $g$ 是 $f$ 在 $I$ 上的广义导数
  - **实例**：$f(x) = |x|$ 的广义导数为 $g(x) = \begin{cases} -1 & [-1,2) \\ 1 & [0,1] \end{cases}$
  - **唯一性**：同一函数的广义导数几乎处处相等
    - **证明**：变分法基本引理直得
  - **无存在性**：阶梯函数
    - **证明**：
      - 设 $I = [-1,1]$，$f = \begin{cases} 0 & [-1,0] \\ 1 & (0,1) \end{cases}$
      - 设 $g$ 是 $f$ 的广义导数，则 $$\forall \p\in C^\infty_0，\int^1_{-1} g(x)\p(x)dx = -\int^1_0 \p'(x)dx = \p(0)$$
      - 即 $g(x)$ 是狄拉克函数
      - 若再取磨光子 $\p_\e = \begin{cases} e^{\dfrac{1}{(x/\e)^2-1}} & |x|<\e \\\\ 0 & 其它 \end{cases}$，易得积分范数 $\lim\limits_{\e\to 0}\|\p_\e\| = 0$
      - 再易得 $\p_\e(0) = \dfrac{1}{e}$，再由积分C-S不等式 $|\p_\e(0)| \leq \|g\|\|\p_\e\|$
- **变分法基本引理**：
  - 设 $f\in L^2(I)$
  - 若 $\forall \p\in C^\infty_0(I)$ 都有 $\dis\int^b_a f(x)\p(x)dx = 0$，则 $f(x) = 0 \ae$
  - **证明（连续函数空间）**：
    - <font color='chartreuse'>当 $f\in C(\ol I)$ 时有 $f \equiv 0$</font>
    - 反设 $f(x_0) > 0$，则由连续性，$f$ 在某个邻域 $O(x_0,\eta)$ 也为正，
    - 再取以其为支集的光滑正值函数 $\p(x) = \begin{cases} e^{\dfrac{1}{(x-x_0)^2-\eta^2}} & |x-x_0|<\eta \\\\ 0 & 其它 \end{cases}$ 即得矛盾

### 直线上二次积分的Sobolev空间

- **符号约定**：$f'$ 表示广义导数
- **1阶Sob空间**：$H^1(I) = \hkh{f\in L^2(I)\mid f'\in L^2(I)}$
  - **内积**：$(f,g)_1 = \dis\int^b_a [fg+f'g']dx$
  - **完备性**
- **m阶Sob空间**：$H^m(I) = \hkh{f\in L^2(I)\mid f',...,f^{(m)}\in L^2(I)}$
  - **内积**：$(f,g)_m = \dis \int^b_a \sum^m_{k=0} f^{(k)}g^{(k)}dx$
  - **完备性**

### 平面上二次积分的Soblev空间

- **广义偏导数**：定义同广义导数
- **1阶Sob空间**：$H^1(I) = \hkh{f\in L^2(G)\mid f_x,f_y\in L^2(G)}$
  - **内积**：$(f,g)_1 = \dis\int^b_a [fg + f_xg_x + f_yg_y]dx$


## 两点边值问题

### 极小位能原理

- **E-L方程**：$Lu = -\dfrac{d}{dx}(p\dfrac{du}{dx}) + qu = f，x\in (a,b)$
  - <font color='chartreuse'>根据极小位能原理，称下面的二次泛函 $J(u)$ 和这里的E-L方程相关</font>
- **边值条件**：$u(a) = 0，u'(b) = 0$
- **条件**：
  - $\ol I = [a,b]，f\in H^0(I)$
  - $p\in C^1(\ol I)，p(x) \geq \min\limits_{x\in I}p(x) > 0$
  - $q\in C(\ol I)，q\geq 0$
- **二次泛函**：设 $a(u,v) = \dis\int^b_a \Big[ p\frac{du}{dx}\frac{dv}{dx} + quv \Big]dx$，则由边值条件 + 分部积分公式易得 $$J(u) = \frac{1}{2}a(u,u)-(f,u)$$
  - <font color='chartreuse'>$a(u,v)$ 是双线性泛函，且满足对称性、正定性、连续性（有界性）</font>
- **变分问题**：设 $H^1_E$ 是 $H^1$ 中满足左边值条件 $u(a) = 0$ 的全体函数，则只需求 $u_*\in H^1_E$ 使得 $J(u_*) = \min\limits_{u\in H^1_E} J(u)$
- **极小位能原理**：
  - 设 $f\in C(I)$
  - 若 $u_*\in C^2$ 是上述边值问题的解，则它是 $J$ 的极小值点
  - 若 $u_*\in C^2\cap H^1_E$ 是 $J$ 的极小值点，则它是边值问题的解
  - **证明**：
- **古典解**：二次连续可微解
  - <font color='chartreuse'>边值问题不一定存在古典解</font>
- **广义解**：连续且按段连续可微的解
  - **实例**：弦平衡问题
  - <font color='chartreuse'>广义解不满足极小位能原理</font>
- **强制边值条件**：必须加入到变分问题的函数类限制中
  - <font color='chartreuse'>比如上面的左边值条件</font>
- **自然边值条件**：不必加入变分问题的函数类限制中
  - <font color='chartreuse'>比如上面的右边值条件</font>

### 虚功原理

- **虚功原理**：
  - 设 $u\in C^2$
  - 则 $u$ 是上述边值条件的解 $\LR u\in H^1_E$ 且满足 $\forall v\in H^1_E，a(u,v)-(f,v) = 0$
  - <font color='chartreuse'>物理意义上，$a(u,v)$ 是外力虚功，$-(f,u)$ 是内力虚功</font>
  - **证明**：
- **推论**：
  - 设 $u\in C^2$，则下列命题等价 $$\begin{cases} Lu = -\dfrac{d}{dx}(p\dfrac{du}{dx}) + r\dfrac{du}{dx} + qu = f \\\\ u(a) = 0 ，u'(b) = 0 \end{cases}$$
  - $u\in H^1_E$ 且满足 $$ \begin{cases} a(u,v) - (f,v) = 0 ，\forall v\in H^1_E \\\\ a(u,v) = \dis\int^b_a \Big[ p\frac{du}{dx}\frac{dv}{dx} + r\frac{du}{dx}v + quv \Big]dx \end{cases} $$
  - <font color='chartreuse'>即将方程替换为更一般的形式后也成立，此时 $a(u,v)$ 非对称正定</font>


## 二阶椭圆边值问题

### 极小位能原理（泊松方程 + 第一边值条件）

- **E-L方程（泊松方程）**：$-\D u = f(u,v)，(x,y)\in G$
  - <font color='chartreuse'>根据极小位能原理，称下面的二次泛函 $J(u)$ 和这里的E-L方程相关</font>
- **第一边值条件**：$u|_\G = 0$
- **二次泛函**：设 $a(u,v) = \dis\iint_G (u_xv_x + u_yv_y)dxdy$，则由边值条件 + 格林第一公式易得 $$J(u) = \frac{1}{2}a(u,u)-(f,u)$$
  - <font color='chartreuse'>$a(u,v)$ 是双线性泛函，且满足对称性、正定性、连续性（有界性）</font>
- **变分问题**：设 $H^1_0$ 是 $H^1$ 中满足边值条件 $u|_\G = 0$ 的全体函数，则只需求 $u_*\in H^1_0$ 使得 $J(u_*) = \min\limits_{u\in H^1_0} J(u)$
- **极小位能原理**：
  - 设 $f\in L^2(G)$
  - 若 $u_*\in C^2(\ol G)$ 是上述边值问题的解，则它是 $J$ 的极小值点
  - 若 $u_*\in C^2(\ol G)\cap H^1_0(G)$ 是 $J$ 的极小值点，则它是边值问题的解
  - **证明**：与一维类似

  - 第二/三边值条件 $\dpfrac{u}{n} + \a u|_{\G} = 0，\a\geq 0$ 是自然边值条件

### 虚功原理（泊松方程 + 混合边值条件）

- **混合边值条件**：设边界 $\G$ 分为互不相交的两部分，其中 $u|_{\G_1} = 0，\dpfrac{u}{n} + \a u|_{\G_2} = 0，\a\geq 0$
- **二次泛函**：设 $a(u,v) = \dis\iint_G (u_xv_x + u_yv_y)dxdy + \int_{\G_2} \a uvds$
- **虚功原理**：
  - 设 $u\in C^2(\ol G)$
  - 则 $u$ 是混合边值条件的解 $\LR u\in H^1_E$ 且满足 $\forall v\in H^1_E，a(u,v)-(f,v) = 0$
  - **证明**：
- **推论**：
  - 设 $u\in C^2$，则下列命题等价 $$\begin{cases} Lu = -\dfrac{d}{dx}(p\dfrac{du}{dx}) + r\dfrac{du}{dx} + qu = f \\\\ u(a) = 0 ，u'(b) = 0 \end{cases}$$
  - $u\in H^1_E$ 且满足 $$ \begin{cases} a(u,v) - (f,v) = 0 ，\forall v\in H^1_E \\\\ a(u,v) = \dis\int^b_a \Big[ p\frac{du}{dx}\frac{dv}{dx} + r\frac{du}{dx}v + quv \Big]dx \end{cases} $$
  - <font color='chartreuse'>即将方程替换为更一般的形式后也成立，此时 $a(u,v)$ 非对称正定</font>
