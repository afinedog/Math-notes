# 偏微分方程的数值解法

- **微分方程定解问题**：给定微分方程和初值或边值条件，求它的解
- **偏微分方程的数值解法**：
  - **思路**：由于计算机只能存储有限个数、进行有限次运算，故必须将连续问题离散化，最终化成有限维线性方程组
  - **网格剖分**：对求解区域进行网格剖分，用有限个网格节点代替连续区域
  - **微分算子离散化**：将微分算子用近似的线性算子代替
- **常用方法**
  - **有限差分法**：从微分形式/积分形式出发，用数值微分/数值积分公式导出线性方程组
  - **有限元法**：从变分形式出发，选取特定基函数，用Ritz-Galerkin法导出线性方程组



## 差分逼近

- **差分格式的分类**：
  - 根据差分公式的选取，可分为向前差分格式、中心差分格式、向后差分格式等
  - 根据差分方程涉及到点的数量，可分为三点差分格式、五点差分格式等

### 离散化

- **经典问题（最简形式的椭圆方程第一边值问题）**
  - $Lu = -\dfrac{d^2u}{dx^2} + qu = f，\begin{cases} a<x<b \\ u(a) = \a \\ u(b) = \b \end{cases}$
  - 其中 $q,f$ 都是 $[a,b]$ 上的连续函数，$q\geq 0$
- **节点**：
- **步长**：
- **离散化**：
  - 由泰勒公式可得 $\cfrac{u(x_{i+1})-2u(x_i)+u(x_{i-1})}{h^2} = \cfrac{d^2u}{dx^2}\biggm|_{x_i} + \cfrac{h^2}{12}\cdot\cfrac{d^4u}{dx^4}\biggm|_{x_i} + O(h^3)$
  - 故可用中心差分代替二阶微分，余项（**截断误差**） $R_i(u) = o(h^2)$
- **中心差分格式**：
  - $L_hu_i = \dfrac{u_{i+1}-2u_i+u_{i-1}}{h^2} + q_iu_i = f_i\pad (i=1,...,N-1)$，$\begin{cases} u_0 = \a \\ u_N = \b \end{cases}$
  - 其中 $q_i,f_i$ 都是节点上的取值
  - <font color='chartreuse'>从微分方程组变为 $N-1$ 阶线性代数方程组（对称三对角矩阵）</font>

#### 习题

- **用泰勒展式构造差分**：
  - **一阶微分的中心差分**：$u'(x_i) = \dfrac{u_{i+1}-u_{i-1}}{2h}$
    - **截断误差 $R_h = O(h^2)$** 
  - **二阶微分的中心差分**：$u''(x_i) = \dfrac{u_{i+1}-2u_i+u_{i-1}}{h^2}$
    - **截断误差 $R_h = O(h^2)$**
  - **三阶微分的中心差分**：$u'''(x_i) = \dfrac{-u_{i+2}+2u_{i+1}-2u_{i-1}+u_{i-2}}{2h^3}$
    - **截断误差 $R_h = O(h^2)$**

### 存在性

- **网格函数**：设 $I_h$ 表示全体网格内点，$\ol I_h$ 表示全体网格点，则 $u_h:\ol I_h\to \R，x_i\to u_i$ 称为网格函数
  - **常用范数**：
    - **最大值范数**：$\|u_h\|_C = \max\limits_{1\leq i\leq N-1}|u_i|$
    - **H0范数**：$\|u_h\|^2_0 = \sum\limits^{N-1}_{i=1}hu_i^2$
      - <font color='chartreuse'>索伯列夫空间0范数（即 $L^2$ 范数）的积分离散化</font>
    - **H1范数**：$\|u_h\|^2_1 = \|u_h\|^2_0 + |u_h|_1^2$，其中后项为 $\sum\limits^{N-1}_{i=1}h\dkh{\cfrac{u_i-u_{i-1}}{h}}^2$
      - <font color='chartreuse'>后项是H1半范数的积分离散化</font>
  - <font color='chartreuse'>易得这些范数等价，以后均用 $\|\cdot\|$ 表示</font>
- **相容条件**：设 $\mc M$ 是光滑函数类，$R_h$ 是网格函数的截断误差，若任意 $u\in M$ 都有 $\lim\limits_{h\to 0} \|R_h(u)\| = 0$，则称差分算子逼近微分算子
  - **实例**：
    - 中心差分算子满足
      - $\|R_h(u)\|_C = O(h^2)$
      - $\|R_h(u)\|_0 = O(h^2)$
      - $\|R_h(u)\|_1 = O(h)$
- **收敛性**：若存在 $h$ 使得 $u_h$ 是差分方程的解，且在某范数下满足 $\lim\limits_{h\to 0} \|u_h-u\| = 0$，则称差分解 $u_h$ 收敛到边值问题的解 $u$

### 唯一性

- **稳定性**：
  - 设差分方程 $L_hv_i = f_i\pad (i=1,...,N-1)，v_0 = v_N = 0$ 
  - 若存在正常数 $M,h_0$ 和范数 $\|\cdot\|_R$ 使得 $$\|v_h\| \leq M \|f_h\|_R\quad (0<h<h_0)$$
  - 则称该方程关于右端稳定
  - <font color='chartreuse'>即解 $v_h$ 的变化被右端函数 $f_h$ 的变化所控制</font>
- **唯一性定理**：满足稳定性的差分方程的解唯一
  - **证明**：
    - 已知差分方程是线性方程组，再由稳定性得齐次情况下仅零解，故非齐次情况下存在唯一解
- **收敛阶定理**：
  - 若
    - 边值问题的解 $u$ 充分光滑
    - 差分方程关于 $\|\cdot\|_R$ 满足相容条件，且关于右端稳定
  - 则差分解 $u_h$ 依范数 $\|\cdot\|$ 收敛到 $u$，且和 $\|R_h(u)\|_R$ 收敛阶相同
  - **证明**：
    - 设绝对误差 $e_i = u(x_i)-u_i$，由差分方程的线性可得其满足误差方程 $L_he_i = R_i(u)，e_0 = e_N = 0$
    - 由相容条件得 $\lim\limits_{h\to 0}\|R_h(u)\|_R = 0$
    - 再由稳定性 + 不等式作差即得 $\lim\limits_{h\to 0} \|e_h\| = 0$
