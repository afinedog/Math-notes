# 算术函数

## 莫比乌斯

- **Mobius函数**：$\mu(n) = \begin{cases} 1 \qquad\pad n=1 \\ (-1)^r \pad n = p_1p_2...p_r \\ 0 \qquad\pad 存在复幂因子 \end{cases}$<font color='chartreuse'>（素因子个数的指示函数）</font>
  - **单幂因子**：素因子次数均为1（$p_1p_2...p_k$）
  - **复幂因子**：存在次数大于1的素因子（$\exist\alpha_i \geq 1$）
  - **积性**：
    - **证明**：利用0和1的乘积性即可
    - <font color='chartreuse'>积性要求 $(m,n)=1$，所以不能有复幂</font>
  - **周延性**：$\sum\limits_{d\mid n} \mu(n) = \begin{cases} 1\quad n=1 \\ 0\quad n>1 \end{cases}$
    - **证明**：$左式 = \mu(1) + \mu(p_1) + \cdots + \mu(p_k) \\ \whhh\qquad + \mu(p_1p_2) + \cdots + \mu(p_{k-1}p_k) \\ \whhh\qquad + \cdots \\ \whhh\qquad + \mu(p_1p_2...p_k) \\ \whhh\quad = \sum\limits^k_{i=1} C^i_k(-1)^k$
    - **本质**：各因子数量是组合数，将其用二项式系数公式相加即可
    - **理解**：任何单幂因子数，它的全部因子里奇数偶数的数量相等（类似奇排列和偶排列数量相等）
- **Mobius筛选公式**：$\dis\sum\limits^n_{\substack{x=1 \\ (x,n)=1}} f(x) = \left[ \cfrac{1}{n} \right] = \sum\limits_{d\mid n} \left[\mu(d) \sum\limits^\frac{n}{d}_{l=1} f(dl)\right]$
  - **证明**：
    - 左边添项变为 $\sum\limits^n_{x=1} [f(x)\sum\limits_{d\mid (x,n)} \mu(d)]$
      - <font color='chartreuse'>（筛选器，根据Mobius周延性，凡是 $d \neq 1$ 的都为0，只保留 $(x,n) = 1$ 的项）</font>
    - 然后展开，再组合成 $\sum f(dl)$ 的形式，以 $\mu(d)$ 为主要因子。
    - 即可得到 $\sum\limits_{d\mid n} [\mu(d)·\sum\limits_{\substack{d\mid x \\ x=1}} f(x)]$，其即为右式（**证毕**）
  - **本质**：归纳所有 $n$ 的因子，用 $\mu(d)$ 当作筛选器
    - 类似组合数学的广义容斥原理公式
- **Mobius反转公式**：设 $F(n) = \sum\limits_{d\mid n} f(d)$，则 $f(n) = \sum\limits_{d\mid n} \mu(d)F(\frac{n}{d})$
  - **证明**
    - 同样是右式展开、重组。（其中 $\mu$ 的因子和 $f(p_i^{\alpha_i})$ 的因子是规律互斥的）
    - $f(1)·[\mu(1) + ...]$（全部的 $\sum\mu = 0$）
    - $f(p_1)·[\mu_(1) + ...]$（少了 $\mu(p_i^{\alpha_i}·K)$，但它全部为复幂因子，不影响，依然为0）
    - $f(p_1^{\alpha_1})·[\mu_(1) + ...]$（少了 $\mu(p_i·K)$，其中只有单幂因子有影响，而所有的单幂因子组合起来，正好是一个 $\sum\limits_{d\mid p_2...p_k}\mu(p_2...p_k)$ 的形式，依然为0）
    - 如此做下去，只剩下了 $f(n)\mu(1)$，**证毕**
- **积性的传递**：若 $f,g$ 是积性函数，则 $H(n) = \sum\limits_{d\mid n} f(d)g(\frac{n}{d})$ 也是积性函数
  - **证明**：拆分重组即可得结论
- **性质**：
  - $\dis\p(n) = \sum\limits_{d\mid n} \mu(d) \frac{n}{d}$
    - **证明**：
      - 易得 $\p(n) = \sum\limits^n_{k=1} \fkh{\dfrac{1}{(n,k)}}$
      - 再由周延性，$原式 = \sum\limits^n_{k=1} \sum\limits_{d\mid (n,k)} \mu(d)$
      - 设 $k=qd$，则 $原式 = \sum\limits_{d\mid n} \sum\limits^{\dfrac{n}{d}}_{q=1} \mu(d) = \sum\limits_{d\mid n}\mu(d)\sum\limits^{\dfrac{n}{d}}_{q=1}1$（**证毕**）
  - $\dis\sum\limits_{d\mid n} \frac{\mu^2(d)}{\varphi(d)} = \frac{n}{\varphi(n)}$
    - **证明**：由欧拉函数积性即可
  - $\dis\sum\limits_{d\mid n} \frac{\mu(d)}{d} = \frac{\varphi(n)}{n}$
    - **证明**：
      - 由Mobius筛选公式，$\varphi(x) = \sum\limits^n_{\substack{x=1 \\ (x,n)=1}}x$
      - 再由Mobius反转公式，可得 $\sum\limits_{d\mid n} \varphi(d) = \varphi(n)$

## 算术函数的卷积

- **狄利克雷积**：$f,g$ 是算术函数，则存在算术函数 $h(n) = (f*g)(n) = \sum\limits_{d\mid n} f(d)g(\dfrac{n}{d})$
  - **实例**：
    - 设 $N$ 是恒等映射，则 $\p = \mu * N$
    - 莫比乌斯反转公式中，$f$ 是 $F$ 的狄利克雷积
  - **卷积性**：
    - 交换律、结合律
- **恒等函数**：$I(n) = \fkh{\dfrac{1}{n}} = \begin{cases} 1，n=1 \\ 0，n>1 \end{cases}$
  - **恒等性**：$I*f = f*I = f$

### 卷积逆

<!-- 类似Fourier的卷积和反演 -->
- **狄利克雷逆**：$f^{-1}*f = f*f^{-1} = I$
  - **唯一性**：$f$
  - **递推公式**：$\begin{cases} f^{-1}(1) = \dfrac{1}{f(1)} \\\\ f^{-1}(n) = \dfrac{-1}{f(1)}\sum\limits_{\substack{d\mid n \\ d<n}} f(\dfrac{n}{d})f^{-1}(d) \end{cases}$
    - **归纳证明**：
      - $n=1$ 时，易得解唯一
      - $n$ 以下均成立时，此时 $\forall d\mid n$ 也成立等式。再由等式是加和形式，故直接分解变形即可
    - **本质**：太简单了，都不用写其实
- **单元函数**：$u(n) = 1\quad (\forall n)$
  - **单元性**：$\mu * u = I$
  - **推论**：莫比乌斯反演公式的卷积证明
    - $F = f*u$，从而 $F*\mu = (f*u)*\mu = f*I = f$

### 曼戈尔特函数

- **Mangoldt函数**：$\L(n) = \begin{cases} \log p\quad n=p^m \\ 0\qquad 其它情况 \end{cases}$
- **周延性**：$\log n = \sum\limits_{d\mid n} \L(d)$
  - **归纳证明**：
    - $n=1$ 时，易得成立
    - $n$ 以下均成立时，唯一分解得 $\log n = \sum\limits^r_{k=1}\a_k \log p_k$
      - 计算易得 $\sum\limits_{d\mid n} \L(d) = \sum\limits^r_{k=1} \sum\limits^{\a_k}_{m=1} \L(p_k^m) = \sum\limits^r_{k=1}\sum\limits^{\a_k}_{m=1} \log p_k = \log n$
  - **本质**：习题难度而已，归纳法的东西都不难
- **卷积公式**：$\L(n) = \sum\limits_{d\mid n}\mu(d)\log \dfrac{n}{d} = -\sum\limits_{d\mid n} \mu(d)\log d$
  - **证明**：由莫比乌斯反转公式即得第一式，再由周延性得第二式

## 积性函数

- **举例**；
  - 幂函数 $f_\a(n) = n^\a$，完全积性
  - 恒等函数 $I$，完全积性
  - 莫比乌斯函数，积性
  - 欧拉函数，积性
- **（完全）积性的运算传递性**：
  - **一般积** $(fg)(n) = f(n)g(n)$，**一般商** $\dkh{\dfrac{f}{g}}(n) = \dfrac{f(n)}{g(n)}$
- **性质**：
  - **积性函数首1性**：$f(1) = 1$
    - **证明**：易得
  - **分解式传递性**：（完全）积性 $\LR$ 唯一分解式满足（完全）积性
    - **证明**：易得
  - **卷积传递性**
    - **证明**：设 $f,g$ 积性，$m,n$ 是两素数，则 $h(mn) = \sum\limits_{c\mid mn} f(c)g(\dfrac{mn}{c})$
      - 互素纯性得必有 $c=ab\quad (a\mid m,b\mid n)$，此时 $(a,b)=1,(\dfrac{m}{a},\dfrac{n}{b}) = 1$
      - 故由两函数积性，将 $h(mn)$ 用 $ab$ 分解即可
    - **反例**：完全积性函数无卷积传递性
    - **推论（积性的逆卷积传递性）**：$g,f*g$ 积性，则 $f$ 积性
      - **归纳反证**：反设 $f$ 不是积性
        - $mn=1$ 时，$f(1) \neq f(1)f(1)$，从而 $f(1)\neq 1$，易得此时 $h(1)\neq 1$，卷积无积性，矛盾
        - $mn$ 以下的任何互素 $ab$ 均成立时，仿照上面公式即得 $h(mn) = h(m)h(n)-f(m)f(n)+f(mn)$，发现 $h$ 积性依赖于 $f$ 积性，矛盾
  - **卷积逆传递性**：
    - **证明**：$g*g^{-1} = I$ + 逆卷积传递性直得
- **进化定理**：积性函数 $f$ 是完全积性 $\LR f^{-1}(n) = \mu(n)f(n)\quad (\forall n\geq 1)$
  - **证明**：
    - **必要性**：设 $g(n)=f(n)\mu(n)$
      - 则计算可得 $(g*f)(n) = \sum\limits_{d\mid n}\mu(d)f(d)f(\dfrac{d}{n}) = f(n)\sum\limits_{d\mid n} \mu(d) = f(n)I(n) = 0 = I(n)$，从而 $g = f^{-1}$
    - **充分性**：只需证明 $p^\a$ 情况即可
      - 首先，由方程可直得 $\sum\limits_{d\mid n}\mu(d)f(d)f(\dfrac{d}{n}) = 0$
      - 代入 $n = p^\a$，得只有 $d=1,p$ 两项不为0，变形即可得 $f(p^\a) = f(p)^\a$
  - **实例**：
    - 欧拉函数的逆 $\p^{-1} = \mu^{-1}*\mu N = u*\mu N$
    - 从而 $\p^{-1}(n) = \sum\limits_{d\mid n} d\mu(d)$
- **积性公式**：若 $f$ 是积性函数，则 $\sum\limits_{d\mid n} \mu(d)f(d) = \prod\limits_{p\mid n} \dkh{1-f(p)}$
  - **证明**：设 $g(n) = \sum\limits_{d\mid n}\mu(d)f(d)$
    - 在 $n=p^\a$ 时，同上法筛选得 $d=1,p$ 两项为 $g(p^\a) = 1-f(p)$
    - 再用积性推广到唯一分解式，即得结论
  - **实例**：
    - 欧拉函数逆公式 $\p^{-1}(n) = \prod\limits_{p\mid n} (1-p)$

### 刘维尔函数

- **Louville函数**：设 $n=p_1^{\a_k}\cdots p_k^{\a_k}$，则 $\begin{cases} \l(1) = 1 \\ \l(n) = (-1)^{\a_1+\cdots+\a_k} \end{cases}$
- **周延性**：$\sum\limits_{d\mid n} \l(d) = \begin{cases} 1，n是完全平方数 \\ 0，其它情况 \end{cases}$
  - **证明**：周延函数设为 $g(n)$，则 $g(p^\a) = \sum\limits^\a_{k=1} \l(p^k) = 0（奇数）1（偶数）$
    - 推广到唯一分解式同理
  - **推论**：$\l^{-1}(n) = |\mu(n)|$
    - **证明**：$\l^{-1}(n) = \mu(n)\l(n) = \mu^2(n) = |\mu(n)|$

### 除数函数

- **除数函数**：$\sigma_\a(n) = \sum\limits_{d\mid n} d^\a\quad (\a\in\C, n\in\N)$
- **公式**：（见前面冯克勤）
- **逆公式**：$\sigma^{-1}_\a(n) = \sum\limits_{d\mid n}d^\a \mu(d) \mu(\dfrac{n}{d})$
  - **证明**：$\sigma_\a = N^\a*u$，从而 $\sigma^{-1}_\a = (\mu N^\a)*u^{-1} = (\mu N^\a)\mu$

### 广义卷积

- **广义卷积**：设 $F:(0,+\infty)\to \C$，其中 $F(x) = 0\pad，\forall x\in (0,1)$
  - 则 $(\a \circ F)(x) = \sum\limits_{n\leq x} \a(n)F(\dfrac{x}{n})$
  - **延拓性**：狄利克雷卷积即为整数支集上的广义卷积
  - 无交换律、结合律
- **交错结合律**：$\a\circ (\b\circ F) = (\a*\b)\circ F$
  - **证明**：直接计算
- **反转公式**：若 $\a$ 含有狄利克雷逆
  - 则 $G(x) = \sum\limits_{n\leq x} \a(n)F(\dfrac{x}{n}) \LR F(x) = \sum\limits_{n\leq x}\a^{-1}(n)G(\dfrac{x}{n})$
  - **证明**：设 $G = \a \circ F$，则 $\a^{-1}\circ G = \a^{-1}\circ(\a\circ F) = (\a^{-1}*\a)\circ F = F$
- **广义莫比乌斯反转公式**：若 $\a$ 是完全积性函数
  - 则 $G(x) = \sum\limits_{n\leq x} \a(n)F(\dfrac{x}{n}) \LR F(x) = \sum\limits_{n\leq x}\mu(n)\a(n)G(\dfrac{x}{n})$
  - **证明**：由莫比乌斯反转公式，$\a^{-1}(n) = \mu(n)\a(n)$
  - **本质**：$G = \a\circ F \LR F = (\mu\a)\circ F$（第一个是一般积）

## 算术级数

### 形式幂级数

- **运算**：设 $A(x) = \sum\limits^\infty_{n=0} a(n)x^n，B(x) = \sum\limits^\infty_{n=0} b(n)x^n$
  - **相等**：$\forall n，a(n) = b(n)$<font color='chartreuse'>（逐项相等）</font>
  - **加法**：$A(x) + B(x) = \sum\limits^\infty_{n=0} \dkh{a(n)+b(n)} x^n$<font color='chartreuse'>（逐项相加）</font>
  - **乘法**：$A(x)B(x) = \sum\limits^\infty_{n=0} \dkh{\sum\limits^n_{k=0} a(k)b(n-k) } x^n$<font color='chartreuse'>（柯西乘积）</font>
  - **逆**：若 $a(0)\neq 0$，则 $A(x)B(x) = 1$ 时互逆
- **几何级数**：$A(x) = 1 + \sum\limits^\infty_{n=1} a^nx^n = \dfrac{1}{1-ax}$
  - **逆**：$B(x) = 1-ax$
- **形式幂级数环**：
  - **零元**：$\forall a(n) = 0$
  - **1元**：$a(0) = 1$，且 $\forall n\geq 1，a(n) = 0$
