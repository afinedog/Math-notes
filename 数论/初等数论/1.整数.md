## Peano公理

- **自然数集**：设 $\N \neq \varnothing$
  - 若定义了：
    - **元素的后继**：$\forall n\in N，\exists! n^+ 与之对应$
    - **幺元素**：$\exist e\in N$，不是任何元素的后继
    - **唯一性**：一个元素最多是一个元素的后继 $a^+ = b^+ \Rightarrow a=b$
    - **归纳公理**
  - 则 $\N$ 称为自然数集合
- **性质（设集合S，由归纳公理证明S=N）**：
  - $\forall n\in N$，有 $n\neq n^+$，否则没有唯一性
  - 除 $e$ 外，任何元素都是后继元素

## 归纳法

- **归纳公理**：设 $S \subset\N$
  - 若
    - $1\in S$
    - $n\in S \Rightarrow n+1 \in S$
  - 则 $S=\N$
  - **理解**：由1作为发动钥匙，不断相加即可。（因此在自然数上成立）
- **第一数学归纳法**：设命题 $P(n)$ 定义在 $\N$ 上
  - 若
    - $n = 1$ 时，$P(1)$ 成立
    - $P(n)$ 成立 $\Rightarrow P(n+1)$ 成立
  - 则 $P(n)$ 对所有正整数成立
  - **证明**：设 $S$ 是使得 $P$ 成立的自然数子集，易得 $S$ 满足归纳公理条件，故 $S = \N$
  - **应用**：把已给条件进行变换，拼凑成归纳假设的形式
    - $n<a^n$
    - $\exist! k，a^k \leq n < a^{k+1}$
- **最小自然数原理**：
  - $\forall T\subset N$，$\exist t_0 \in T$，使得 $\forall t\in T$，有 $t_0 \leq t$<font color='chartreuse'>（T必有下确界）</font>
  - **证明（归纳公理）**：
    - 若 $T$ 包含 $0$，则 $t_0 = 0$ 即可
    - 若 $T$ 不包含 $0$，则由0的最小性，$\exist S$，使得 $\forall s\in S$，有 $s < \forall t\in T$
       - 只能有 $S\cap T = \varnothing$
         - 若 $S\cap T = \{t_1,t_2,\cdots\}$，则由 $S$ 定义，$t_1<t_2$ 且 $t_2<t_1$，不可能
         - 若 $S\cap T = \{t\}$，则由 $S$ 定义得 $t$ 就是 $t_0$，矛盾
       - $S$ 必定存在最大元，否则由归纳公理，$S=\N$，与小于 $T$ 中元素矛盾
         - <font color='chartreuse'>（由于 $S$ 定义的特殊性，它不需要最大自然数原理，从而不是循环证明）</font>
       - 同样由归纳公理得必定存在最大的 $S$
         - 即 $\exist s_0\in S$，但 $s_0+1\notin S$，并满足 $s_0+1 \in T$，从而这就是最小的 $t$（**证毕**）
   - **本质**：自然数的最小单元性，相当于弱化的（确界存在定理的数列证明）
  - **证明（第二数学归纳法）**：
    - 反设 $T$ 无最小数
    - $P：k< \forall t$
      - 则 $P(1)$ 成立
      - 设 $n$,对 $\forall k<n，P(k)$ 均成立，又因为反设，$n$ 不是 $T$ 的最小数，即 $n\notin T$，即 $P(n)$ 也成立
    - 则 $\N$ 上 $P$ 均成立，但这样的话 $T$ 无穷大，不可能存在，矛盾。（**证毕**）
  - **理解**：自然数的最小单元性（非稠密性）
- **第二数学归纳法**：
  - 若
    - $n = 1$ 时，$P(1)$ 成立
    - $\forall n>1，\forall k<n，P(k)$ 成立 $\Rightarrow P(n)$ 成立
  - 则 $P(n)$ 对 $\N$ 成立
  - **理解**：它不需要递推关系，只需要范围关系（有时候范围可以通过放缩，比递推更简单）
  - **证明（最小自然数原理）**：反设 $P$ 是不成立的集合，则存在最小数。则最小数以下均成立，但它和条件2矛盾。**证毕**
  - **等价命题**：
    - 最小自然数原理
    - 第二数学归纳法（如果不证明最小自然数原理，则无法得到第二数学归纳法）
    - **自然数集上的良序原理**：任意非空自然数集存在一个最小元
    - **归纳原理**：归纳的正整数集合只有 $\Z_+$
      - （正推已经写了，逆推回去就不写了）
    - **强归纳原理**：若截 $S_n\in A \red\Rightarrow n+1 \in A$，则 $A=Z_+$
      - （第二数学归纳法的归纳原理）
      - 详见Munkres

### 群环域

- 加法群：可进行加减运算，具有0和负数的集合
- 交换环：可进行加减乘，且满足运算律的集合
  - 无零因子环：仅0具有乘积性（向量集合的点乘运算不满足，完系不满足），从而乘法消去律成立
- 域：加减乘除运算
  - 商域：Q是Z的商域
- 扩域：R是Q的扩域

## 整除

- **因子性**：整除这一章的本质就是把整数看作有大小的因子，大整数必定由有大小的因子组成
  - 带余除法是直接体现
- **整除的基本性质**
  - 消去律
  - 传递性
  - 右线性：$a\mid \{b_i\}$，则 $a\mid (\sum\limits^k_{i=1} \a_i\cdot b_i)$
  - 大小性
- **带余除法**：$a、b\in \Z，b\neq 0$，则 $\exist! q,r$，使得 $a = bq+r$（$0\leq  r<|b|$）
  - **证明**：高代采用了归纳次数法和同构法证明存在性，反证法证明唯一性
    - 利用最小自然数原理，设 $E = \{a-bk | k\in \Z\}$，则最小的自然数就是r。接下来根据最小性证明r的范围即可
    - 唯一性则使用反证法
  - **推论**：实数可被（大于1的任意自然数进制）表出
- **理想的幺元性**：
  - 若 非空集 $S\in \Z$ 对减法封闭
  - 则 $\exist! d\in N \geq 0$，使得 $\forall s\in S = kd$
  - **理想**：满足该性质的 $S$ 称为 $\Z$ 的理想
  - **证明**：
    - 首先，减法封闭得存在幺元素0，从而存在负数，从而对加法也封闭，从而S是无限集合
    - 设 $d$ 是 $S$ 中最小自然数，反设 $m \neq kd$，则由带余除法，$m = qd + r$，再由加减封闭性，$r\in S$，与 $d$ 最小性矛盾。（**证毕**）
- **主理想**：由一个元素生成的理想
- **主理想环**：每个理想都是主理想
  - **实例**：整数集 $\Z$

### 习题

- $(a-b) | (a^n-b^n)$
  - **证明**：添项，利用线性性质消元。
    - $a^n-b^n = a^n-ab^{n-1}+ab^{n-1}-b^n \\ = a(a^{n-1}-b^{n-1}) + b^{n-1}(a-b)$
    - 后项可消去，此时 $原式 \to a(a^{n-1}-b^{n-1})$
    - 不断归纳即可
- $(2^m-1) \nmid (2^n+1)\quad (m<n)$
  - **证明**：添项，利用线性性质消元
    - $2^n - 2^{n-m} + 2^{n-m}+1 \\= 2^{n-m}(2^m-1) + 2^{n-m}+1$，消去并归纳
- 若 $n$ 是奇数，则 $\sum\limits^{n-1}_{k=1}\frac{1}{k-1} · (n-1)!$ 被 $n$ 整除
  - **证明**：首位相加即可，比如：
    - $\quad 1×2×3×……×(n-2)×(n-1)$
    - $= 1×2×3×……×(n-2)×1$
    - 它们相加得到n，又因为n是奇数，n-1是偶数，所以正好相消
- 高斯函数的分解：$[\dfrac{a}{b}] = q - [\dfrac{r}{b}] = q$

## 最大公因数

- **最大公约数**：$(a,b,...,c)$
  - 公因性、最大性
  - 存在性、唯一性
- **互素**：$(a,b,...,c) = 1$
- **最大公因数的基本性质**：
  - 交换律、自反律
  - 封闭性：$(a,b,\cdots,c) = (a,b+ax,\cdots , c+ay)\quad (x,y\in\N)$
  - 因子定理：$E =\{ ax+by+...+cz | x,y,...,z\in Z\} = \{ k(a,b,...,c)| k\in Z\}$
    - **因子证明**：
      - **必要性**（用右式代表左式）：直得
      - **充分性**：已知 $\forall k(a,b,...,c) = qa + r$
        - 再对 $r$ 和 $b$ 使用带余除法（负数也能用），使得 $r$ 不断减小
        - 直到 $r$ 不能再减小时，若 $r< \forall a,b,...c$，且 $r\nmid\forall a,b,...c$，则由 $r = k() - k_1() - k_2()...$ 得 $r\mid ()$，进而一定整除 $\forall a,b,...,c$，矛盾。（**证毕**）
    - 也可以用不等号互包性证明：
      - 由 $\Z$ 的主理想性，得到一个 $D$，证明 $d = D$ 即可
  - **裴蜀等式**：
    - 因子性的直接表现
    - 带余除法的终极形态
- **最大公因数的高级性质**
  - 结合律：$(a,b,\cdots,c) = ((a,b),\cdots,c)$
    - 理解：
      - 削弱视角：把取最大公因式看作一次削弱，而最大公因数本身不怕削弱
      - 因子视角：同上
  - 提公因式
    - 主理想视角（裴蜀等式）
  - 互素的纯性：
    - $(a,m) = (b,m) = 1 \red\Rightarrow (ab,m) = 1$
      - 裴蜀等式相乘（1的乘积性）
      - 因子不灭性（类似集合交并补的理解）
    - $c\mid ab 且 (c,b)=1 \red\Rightarrow c\mid a$
      - 裴蜀代换即可
- **欧几里得辗转相除法**：
  - 求方程 $ax+by = (a,b)$
- **最小公倍数**：
  - $(a,b)[a,b] = |ab|$
    - 整除互包性证明：$[a,b] = \dfrac{|ab|}{(ab)}$，左边整除右边（最小性）。$(a,b) = \dfrac{|ab|}{[a,b]}$，右边整除左边（最大性）
    - 先用质数得出结论，然后提公因式转化为质数

### 习题

- **广义带余除法**：$(a^m-1,a^n-1) = a^{(m.n)}-1$
  - **证明**：由线性性质，不断归纳即可
- **带余除法缩系定理（有界性）**：$ax+by = (a,b)，其中|x| < b，|y| < a$
  - **缩系法**：
    - 已知 $(a,b) \leq a、b$，则 $x$ 和 $y$ 不能同时为正。
    - 反设 $0<x<b,-a<y<0$ 时无解，而 $b<x_0<2b,-2a<y_0<-a$ 时有解
      - 则添项得 $a(x_0-b)+b(y_0-a)=(a,b)$，发现 $0< x_0-b <b$，矛盾。
    - 归纳推理到整个Z中即可
  - **带余除法**：
    - 带余除法，$x = bq + r$，则 $ar + b(aq + y) = (a,b)$
    - 则 $r<b,|aq+y| = |\frac{ar-d}{b}| < |a|$
- **最大公因式的变形问题**：
  - 裴蜀法：$(d,mn) = (d,m)(d,n)\quad$ 已知 $(m,n) = 1$
  - 括号法：互包性，唯一分解定理
- **幂性**：$(a^n,b^n) = (a,b)^n$
  - $(a,b) = 1,ab=c^n \boldsymbol{\longrightarrow} a = (a,c)^n,b=(b,c)^n$
- **通解**：$ax + by = c$，通解为 $\begin{cases} x = x_0 + \frac{b}{(a,b)}t \\ y = y_0 - \frac{a}{(a,b)}t \end{cases}$（缩系中添项）

## 唯一分解定理

- **素数有无穷个**
  - **欧几里得归纳法**：反设只有有限个素数，它们的乘积为 $\prod p$
    - 由于除了0和1外 $\forall p\in\N，p\nmid p+1$，故 $\prod p\nmid (\prod p+1)$
    - 这样，我们就生成了一个新的素数，且该过程可无限归纳下去
- **素数的单元性**：$p\mid ab\cdots c \Rightarrow \exist k$，有 $p\mid k$
  - **利用素数定义**：
    - 首先有 $mp = ab\cdots c$
    - 则由互素纯性，存在 $\dfrac{ab\cdots k_1}{m} = \dfrac{p}{k_2\cdots c}\in \N$。再由 $p$ 是素数，$k_2\cdots c$ 只能是 $1$ 或 $p$。再易得其不为 $1$
    - 再反证设不存在 $k$，则 $\forall n，\dfrac{k}{n}\neq p$，也即 $k_2\cdots c\neq p$，矛盾
  - **或者利用互素的纯性（裴蜀等式，主理想S）**
- **唯一分解定理**：归纳法
  - **存在性**：退化证明
  - **唯一性**：反证，退化，归一
- **Z是唯一分解环**
  - **非唯一分解环实例**：
    - $S = \{4x+1\mid x=0,1,2...\}$：
      - 乘法封闭性（1的乘积性（见同余））
      - 定义素数为不能分解为 $S$ 中两数之积的元素
        - 即 $x \neq 4mn+m+n\pad (\forall m,n\in S)$
      - 非唯一性：设 $p、q = 4k -1$，则它们的乘积 $pq,p^2,q^2\in S$ 且为素数（-1的乘积性），但 $(pq)^2 = pq·pq = p^2·q^2$，分解不唯一
      - （S中的素数无穷多，但是互素不具有纯性）
- **素因数标准分解**：$n = \pm p_1^{\alpha_1}\cdots p_k^{\alpha_k}$
  - 标准分解下的因子：$d = \pm p_1^{\beta_1}\cdots p_k^{\beta_k} \quad (\beta_i \leq \a_i)$
  - 标准分解下的最大公因数：$d = \pm p_1^{\g_1}\cdots p_k^{\g_k}\quad (\gamma_i = \min(\alpha_i,\beta_i))$
- **数论函数**：定义在 $N^+$ 上的函数
  - **除数函数**：$\tau(n)$ 为正约数的个数
    - $\tau(n) = (\alpha_1 + 1)\cdots (\alpha_k + 1)$：唯一分解式的全排列
  - **除数和函数**：$\sigma(n)$ 为正约数之和
    - $\sigma(n) = \dfrac{p_1^{\alpha_1 + 1} - 1}{p_1-1}\cdots \dfrac{p_k^{\alpha_k + 1} - 1}{p_k-1}$：等比数列求和，一共有k组等比数列，通过提公因式合并成乘积
  - **积性函数**：$f(mn) = f(m)f(n)$。上面两个函数都是积性函数（整体函数的形式是因子函数的乘积）

### 习题

- 解题方法：
  - **素数乘积构造**：
    - $4m-1$ 的素数有无穷多个：
      - 设 $p_k$ 为 $4k-1$ 最大的素数，$4\cdot p_1p_2...p_k - 1$。
      - 又由于 $4$ 的缩系只有 $4k+1$ 和 $4k-1$，且由 $-1$ 的乘积性，上式必有 $4k-1$ 的素因子，但又与其形式矛盾。证毕
    - 可以有任意个相邻的数都是合数：
      - 构造 $m = p_1p_2...p_k+1$，则 $m+奇数 = 含2的合数$，$m+偶数 = p_1p_2...p_k+奇数$，若奇数是素数，则有单个素因子。若奇数是合数，则有多个素因子
      - 可以一直延续到 $p_k-1$，由任意性得结果（√）
  - **二项式构造**
    - **费马数**：$F_n = 2^{2^n}+1$，其中的素数称为费马素数
      - $2^n+1$ 为素数，则其必为费马数
    - **梅森数**：$M_p = 2^p-1$，其中 $p$ 是素数
      - $m^n-1$ 为素数，则其必为梅森数
    - 最大公因式：$(a,b) = 1$,$p$ 是素数，证明 $(a+b,\frac{a^p+b^p}{a+b}) = 1或p$：换元，令 $m = a+b$ 即可
