# 素数


- **素数集**：$\P$
- **素数无穷性（分析法）**：反设只有有限个素数 $\{p_i\}^r_{i=1}$
  - 考虑 $X = \prod\limits^r_{k=1}\cfrac{1}{1-\dfrac{1}{p_k}}$，它可分解为（素数倒数的等比列）的积。
  - 即 $X \geq (1+\dfrac{1}{2}+\cdots)\cdots(1+\dfrac{1}{p_r}+\cdots)$
  - 去括号得 $右式 = \sum\limits_{n\in \mc N(K)} \dfrac{1}{n}$
    - $\mc N(K) = \{n\in\N\mid n = p_1^{e_1}\cdots p_r^{e_r}，e_i \leq K\}$
  - 由唯一分解定理，$右式 = \sum\limits^\infty_{n=1} \dfrac{1}{n}$
    - 但由于调和级数不收敛，其与 $X$ 是有限个有限数乘积，从而必定有界相矛盾
- **素数发散性**：$\sum\limits_{p\in\P}\frac{1}{p}$ 发散
  - **反证法**：反设收敛，则 $\exists N$ 满足 $\sum\limits_{p>N} \frac{1}{p} < \frac{1}{2}$
    - 设 $Q = \prod\limits_{p\leq N} p$
    - 设 $P = \sum\limits^\infty_{t=1} \dkh{\sum\limits_{p>N}\frac{1}{p}}^t < \sum\limits^\infty_{t=1} \frac{1}{2^t} = 1$
    - 易得 $\sum\limits^\infty_{n=1} \frac{1}{1+nQ} \leq \sum\limits^\infty_{t=1} \dkh{\sum\limits_{p>N}\frac{1}{p}}^t = P < 1$
    - 但又因为 $\sum\limits^K_{n=1} \dfrac{1}{1+nQ} \geq \dfrac{1}{2Q} \sum\limits^K_{n=1}\frac{1}{n}$，且右式发散，矛盾（**证毕**）
  - **放缩法**：强化定理 $\sum\limits_{p\leq N} \dfrac{1}{p} > \log\log N - 2$
    - 设 $\mf N(N) = \set{n\in\N \mid n的所有素因子均\leq N}$
      - 类似分析法，得 $\sum\limits_{n\in\mf N(N)}\dfrac{1}{n} = \prod\limits_{p\leq N}\cfrac{1}{1-\frac{1}{p}}$
      - 再由调和级数性质，$\log N\leq \sum\limits_{n\leq N+1}\dfrac{1}{n} \leq \sum\limits_{n\in\mf N(N)}\dfrac{1}{n}$
    - **引理**：$\forall v\in [0,\frac{1}{2}]，\dfrac{1}{1-v} \leq e^{v+v^2}$
      - **证明**：用导数即可
      - 令 $v = \frac{1}{p} \leq \frac{1}{2}$，则 $\prod\limits_{p\leq N}\cfrac{1}{1-\frac{1}{p}} \leq \prod\limits_{p\leq N}e^{\frac{1}{p} + \frac{1}{p^2}}$
    - 再由 $\sum\limits_{p\in\P} \frac{1}{p^2} < \sum\limits^\infty_{n=2} \dfrac{1}{n^2} < 1$
      - **引理**：$\sum\limits^N_{n=2} \dfrac{1}{n} < \int^N_2\dfrac{1}{(x-1)^2}dx \leq 1\quad (\forall N\geq 2)$
    - 综上即可得 $\log\log N \leq \sum\limits_{p\leq N}\frac{1}{p} + 1$（**证毕**）

### R-Z函数

- **黎曼泽塔函数**：$\zeta(\sigma) = \sum\limits^\infty_{n=1} \dfrac{1}{n^\sigma}$
- **欧拉积表示定理**：$\forall \sigma>1，\zeta(\sigma) = \dis\prod_{p}\cfrac{1}{1-\frac{1}{p^\sigma}}$
  - **证明**：
    - $(1-2^\sigma)\zeta(\sigma) = \sum\limits^\infty_{n=1} \dfrac{1}{n^\sigma} - \sum\limits^\infty_{n=1} \dfrac{1}{(2n)^\sigma}$
      - 错位相减得 $= \sum\limits_{n是奇数}\dfrac{1}{n^\sigma}$
      - 分离 $n=1$ 项得 $= 1+\sum\limits_{p\mid n\Rt p>2} \dfrac{1}{n^\sigma}$（大于2的奇数）
    - 其余素数（$3,5,\cdots$）同理，计算可得 $\\ (1-2^\sigma)(1-3^\sigma)\cdots(1-P^\sigma)\zeta(\sigma) = 1+\sum\limits_{p\mid n\Rt p>P} \dfrac{1}{n^\sigma}$（大于P的奇数）
    - $P\to\infty$ 即得结论

## 素数的列举

### 筛法

- **实例**：选取 $1\sim 50$ 的所有质数
  - 依次减去各个质数 （$2，3，5\cdots$）的所有倍数
  - 减到 $7$ 即可停止，因为 $7^2 = 49 \approx 50$
  - **理解**：一个合数的二重素因子分解中，必定存在一个大一个小。我们只需找到小的

### 质数生成函数

- **引理**：$\forall n\geq 1$，$\sum\limits_{p\leq n} \log p < 2n\log 2
  - **证明**：设 $M = \tvek{2m+1}{m} = \cfrac{(2m+1)(2m)\cdots(m+2)}{m!}$
    - 二项式系数得 $M < \dfrac{(1+1)^{2m+1}}{2} = 2^{2m}$
    - 设 $A(m) = \set{p\mid m+1< p\leq 2m+1}$，则 $\prod\limits_{p\in A(m)} p$ 整除 $M$
      - 则 $\sum\limits_{p\leq 2m+1} \log p - \sum\limits_{p\leq m+1} \log p = \sum\limits_{p\in A(m)} \log p \leq \log M < 2m\log 2$
    - **归纳法**：$n\leq 2$ 易得题设不等式成立
      - 设 $n=k-1$ 成立
        - 若 $k$ 为偶数，可直接退化+放缩
        - 若 $k$ 为奇数，设为 $2m+1$，则添项放缩得 $\sum\limits_{p\leq 2m+1} \log p < 2m\log 2 + 2(m+1)\log 2 = 2k\log 2$
- **勃兰特假设**：若 $n\geq 1$，则至少存在一个素数 $p\in (n,2n]$
  - **证明**：已知 $\sum\limits^\infty_{k=1} \big[ \dfrac{n}{p^k} \big]$ 是 $p$ 除 $n!$ 的最大幂
    - 设 $N = \prod\limits_{p\leq 2n} p^{k(p)} = \dfrac{(2n)!}{(n!)^2}$
      - $k(p) = \sum\limits^\infty_{m=1} \dkh{\big[ \dfrac{2n}{p^m} \big]- 2\big[ \dfrac{n}{p^m} \big]}$（奇数是0，偶数是1）
      - 易得 $p^m>2n$ 时，$k(p) \equiv 0$，故 $k(p) \leq \big[ \dfrac{\log 2n}{\log p} \big]$
    - 反设存在 $n$ 不满足题设，且 $p$ 是 $N$ 的一个素因子。
      - 若 $p\in (\dfrac{2}{3}n,n]$，此时 $k(p) = 0$，不可能
      - 若 $p\leq \dfrac{2}{3}n$
        - 由引理得 $\sum\limits_{p\mid N} \log p \leq \sum\limits_{p\leq \frac{2n}{3}} \log p\leq \dfrac{4}{3}\log 2$
        - 若 $k(p) > 1$，则由其定义域得 $p \leq \sqrt{2n}$，从而 $\sum\limits_{k(p)\geq 2} k(p)\log p \leq \sqrt{2n}\log 2n$
        - 综上，由 $\log N$ 定义放缩得 $\log N \leq \sum\limits_{k(p)=1} \log p + \sum\limits_{k(p)\geq 2} k(p)\log p \\ \leq \dfrac{4}{3}\log 2 + \sqrt{2n}\log 2n$
        - 再由 $N$ 的二项式系数意义，得 $2^{2n} \leq 2nN$
          - 最终得到 $2n\log 2 \leq \dfrac{4}{3}n\log 2 + \log 2n + \sqrt{2n}\log 2n$，它在 $n\geq 500$ 后不成立，矛盾（**证毕**）
    - **推论**：素数无穷性
      - $\pi(X) = \set{p\leq X \mid p\in\P}$
        - 设 $u_1 = 1，u_{n+1} = u_1\cdots u_n + 1$，则 $\pi(X)\geq \min\{n\mid u_n\geq X\}$
        - $(1,2^N]$ 中至少有 $N$ 个素数，故 $\pi(2^N)>N$
    - **推论**：存在一个实数 $\t$ 使得 $\left[ 2^{2^{2^{\ddots^\t}}} \right]$
      - **证明**：可设素数列 $q_n$ 满足 $2^{q_n} < q_{n+1} < 2^{q_n+1}$
        - 设函数列 $f^{(1)}(x) = \log_2 x，f^{(n+1)}(x) = \log_2(f^{(n)}(x))$
          - 设 $u_n = f^{(n)}(q_n)，v_n = f^{(n)}(q_n+1)$
          - 易得 $f^{(n)}$ 单增，故 $u_n<u_{n+1}<v_{n+1}<v_n$，从而 $u_n$ 收敛，设极限为 $\t$
        - 设 $g^{(1)}(x) = 2^x，g^{(n+1)}(x) = 2^{g^{(n)}(x)}$
          - 则 $g^{(n)}(u_n) < g^{(n)}(\t) < g^{(n)}(v_n)$
          - 从而 $\forall n，q_n < g^{(n)}(\t) < q_n+1$

### 梅森素数

- **梅森数 $M_n$**：$p$ 是比2的偶数幂多3或更少的素数时，$2^p-1$ 称为梅森数

- **引理**：若 $p$ 是素数，$q$ 是 $M_p$ 的素因子，则 $q\equiv 1 \pmod p$
  - **证明**：已知 $2^p\equiv 1\pmod q$，再由费马定理，$2^{p-1}\equiv 1\pmod q$

## 费马数

- **费马数**：$F_n = 2^{2^n} + 1$
- **欧拉反例**：$641\mid F_5$
  - **证明（猜想法）**：已知 $640\equiv 5\cdot 2^7 \equiv -1 \pmod 641$
    - 从而 $(-1)^4 \equiv 1\equiv (5\cdot 2^7)^4 \equiv 5^4\cdot 2^{28}$
    - 再由 $5^4\equiv -16\pmod 641$，得 $1 \equiv -2^{32}\equiv -2^{2^5} \pmod 641$
- **引理**：若素数 $p\mid F_n$，则 $\exists k\in\N$ 满足 $p = 2^{n+1}k+1$
  - **证明**：设 $d = gcd(2^{n+1},p-1)$，裴蜀等式 $d = 2^{n+1}a + (p-1)b$
    - 同上得 $2^d\equiv 1\pmod p$
    - 再由 $\exists 0\leq c\leq n+1$ 使得 $d = 2^c$
    - 因此只能是 $c = n+1$，从而 $p-1 = k2^{n+1}$（**证毕**）
  - **实例**：费马数 $F_5$，列举法
  - **推论**：对上述 $p$，$\exists k\in\N$ 满足 $p = 2^{n+2}k+1$
