# 勒贝格测度

- 测度是一种度量集合大小的量。随着度量标准的变化，测度也可以分为许多种。
  - 测度在一定意义上定义了集合的大小，它按照元素的体积进行偏序比较
  - 势也在一定意义上定义了集合的大小，它按照元素的数量进行偏序比较
- 本章讨论的是在 $\R^n$ 中的集合，且测度默认是勒贝格测度。但对于一般情况的集合，应当提前说明是什么测度
  - 任何测度都可定义外测度，且任何集合都有外测度，但集合不一定可测。不可测集的测度根据取法不同而不同

## 矩形的体积

- **几乎不相交**：若一族集合的内部互不相交，则称它们几乎不相交
- **广义矩形**：取 $\R^n$ 中的 $n$ 维矩形
  - 若保留其边界，则称为 $\R^n$ 中的闭矩形
  - 若删去其边界，则称为 $\R^n$ 中的开矩形
  - **实例**：$\R$ 中开矩形是开区间，$\R^2$ 中开矩形是开长方形，$\R^3$ 中开矩形是开长方体
- **有限矩形体积可加性**：
  - 若某矩形 $R$ 可分解为有限个几乎不相交矩形 $\{R_k\}^N_{k=1}$，即 $R = \mathop{\bigcup}\limits^N_{k=1}R_k$
  - 则总体积等于所有矩形体积的和，即 $|R| = \sum\limits^N_{k=1}|R_k|$
  - **证明（面积公式配凑）**：
    - 由矩形本身的性质，其可以画成网格形式
      - <font color='chartreuse'>此时的网格是不规则网格</font>
    - 对每个 $R_k$ 再次进行分割，使得每个新矩形 $R_j$ 与其相邻矩形的公共边长度相等
      - <font color='chartreuse'>此时的网格是规则网格</font>
    - 此时因为公共边长相等，故可以**逆用乘法分配律**，得到 $|R| = ab = \sum a_j \sum b_j = \sum a_jb_j = \sum |R_j|$
    - 同理，$|R_k|$ 可以分解为 $|R_j|$ 的组合（**证毕**）
  - <font color='red'>注意只能是有限个矩形，可数个矩形一般不满足体积可加性</font>

## 外测度

- **Lebesgue外测度**：
  - 设 $E\subset\R^n$，则 $E$ 的外测度定义为 $E$ 的最小矩形覆盖的体积 $$m^*E = \inf\hkh{\sum\limits^\infty_{n=1}|Q_n| : E\subset \mathop{\bigcup}\limits_{n=1}^\infty Q_n ，Q_n是 \R^n 中的开矩形}$$
  - **等价写法**：$m^*E = \inf\limits_{E\subset (\cup Q_j)}\sum\limits^\infty_{j=1} |Q_j|$
  - **不相交性**：满足以上条件的最小集族 $\{\Q_n\}$ 必定是互不相交的
  - **体积性**：
    - <font color='chartreuse'>外测度是体积概念的自然推广
      - 数分中我们学到，求不规则图形的体积时，通常找到它的边界函数，然后对该函数定积分。而定积分的本质就是取小矩形进行逼近。而如果用任意方法进行内外逼近时，小矩形总体积都相等，则称函数可积
      - 因为有些集合是没有边界的（比如有限点集），所以不妨跳过取边界函数的步骤，直接用矩形进行逼近
        - 如果用外部逼近，就是勒贝格外测度，它相当于Darboux大和的下确界
        - 如果用内部逼近，猜想可以类似定义出勒贝格内测度。但遗憾的是，我们发现没办法对点集取矩形内部逼近，即勒贝格内测度并不是一个良定义
      - 不难发现，这种推广的体积定义是否可以导出推广的积分方法呢？实际上它就是后面要学习的勒贝格积分
    - 测度概念的提出，本质是为了衡量集合的大小
      - 一些奇怪的数学问题，比如线段的长度是怎么定义的，为什么无穷个点没有体积，组成线段就有了体积。通过严格的测度论，这些问题都能得到解答
    - 在不同数学观点下，通过定义不同的测度，对集合的大小的定义也随之变化。勒贝格测度是一种最简单的朴素定义，所以我们先学这个。等到以后再学其它的测度</font>
  - **存在性**：任意集合的勒贝格外测度均存在
    - **证明**：
      - 显然任意集合都存在开父集，显然开集都可被开矩形覆盖，故任意集合都存在开矩形覆盖
      - 再由矩形体积总存在，故任意开矩形覆盖的体积和总存在
      - 再由确界存在定理，得体积和集合的下确界必定存在，从而得到结论
  - **无限性**：矩形必须是无限个<font color='chartreuse'>（无穷多意味着无穷小，才能达到任意逼近的效果）</font>
  - **实例**：
    - 单点集的覆盖为其本身，因此外测度为0
    - $\R^n$ 包含所有矩形，易得 $m^*(R^n) = \infty$
- **矩形的外测度**：设 $Q$ 是 $\R^n$ 中开矩形或闭矩形，则 $m^*Q = |Q|$
  - <font color='chartreuse'>$\R^n$ 中矩形的外测度等于其体积</font>
  - **证明**：
    - 数学归纳法（对矩形数量归纳） + 取交集法（用于合并图形），配凑得到不等式 $\sum\limits^n_{i=1}|I_i| \geq |\bar{I}|$ （有限开矩形覆盖的外测度 > 被覆盖闭矩形的外测度）
    - 由有限覆盖定理，可将不等式推广到任意开覆盖
    - **边界测度为 $0$**：由重积分知识，（任意有界集合的边界）的（最小矩形覆盖）的体积为 $0$，故 $m^*(\ol Q - Q) = 0$，从而闭矩形 = 开矩形
- **矩形外测度可加性**：
- **外测度性质**
  - **刚体运动不变性**：设 $\p$ 是刚体运动，则 $m^*E = m^*\Big( \p(E) \Big)$
    - **证明（分解法）**：
      - 已知刚体运动均可分解为平移和旋转的复合，故分别证明不变性即可
      - **平移不变性**：设 $a\in\R^n$，则 $m^*E = m^*(E+a)$
        - **证**：将 $E$ 的外测度定义中的矩形覆盖也取 $a$ 平移，易得平移下矩形体积不变。再按照定义即可证明
      - **旋转不变性**：
        - **证**：同上
  - **非负性**：$m^*E \in [0,+\infty]$
    - **证明**：
      - 易得矩形体积非负，再由矩形体积可加性，即得矩形覆盖的总体积也非负
  - **单调性**：$A\subset B \LR m^*A\leq m^*B$
    - **证明**：
      - 设 $A,B$ 的开矩形覆盖为 $Q_A，Q_B$，显然有 $Q_B\subset Q_A$，由矩形体积可加性即得结论
  - **三角不等式（次可加性）**：$m^*(\mathop{\cup}\limits^\infty_{n=1}A_n) \leq \sum\limits^\infty_{n=1}m^*A_n$
    - **证明（定义 + 容斥原理）**：
      - $\sum\limits^\infty_{n=1}m^*A_n = \sum\limits^\infty_{n=1}\sum\limits^\infty_{k=1}|Q_{n_k}| \geq |\mathop{\cup}\limits^\infty_{n=1}\mathop{\cup}\limits^\infty_{k=1}Q_{n_k}| = m^*(\mathop{\cup}\limits^\infty_{n=1}A_n)$

### 习题

- **内测度**：
  - **补集定义法**：
    - **开集的测度**：包含E的开集测度，减去E的补集的测度
      - $m_*E = (b-a)-m^*(\complement_{(a,b)}E)$
    - **闭集的测度**：包含E的闭集测度，减去E的补集的测度
  - **闭覆盖定义**：包含在E中的闭集 $\bar{I}_n$ 测度的上确界
  - **不可行性**：由刚才外测度中所述，点集的内测度不好处理（一般意义讲，其没有闭覆盖，除非重新定义）
- **极限保序性**：若对任意 $n$，$a_n < b_n$，则 $\lim\limits_{n\to\infty} a_n \leq \lim\limits_{n\to\infty} b_n$
- **级数保序性**：若对任意 $n$，$\sum\limits^n_{k=1}a_k < \sum\limits^n_{k=1} b_k$，且 $\sum\limits^\infty_{k=1} a_k,\sum\limits^\infty_{k=1} b_k$ 均收敛，则 $\sum\limits^\infty_{k=1} a_k \leq \sum\limits^\infty_{k=1} b_k$
  - <font color='chartreuse'>本质是（有限可加性）推广到（可数可加性）的条件</font>
  - **实例**：任意有限集合，其勒贝格外测度均可加
    - **证明**：若已知 $\forall N，m^*(\mathop{\cup}\limits^\infty_{n=0}E_i) \geq \sum\limits^N_{n=0} m^*|E_i|$，因为可以比较大小，所以两边都是有界量（级数收敛）。所以可以直接把右式的N变为无穷，依然是有界量，依然满足大小关系。




## Lebesgue可测

- **勒贝格可测（Cara定义法）**：设 $E\subset \R^n$
  - 若满足**Caratheodory条件**：$\forall T\subset \R^n，m^*T = m^*(T\cap E) + m^*(T\cap E^C)$
  - 则 $E$ 称为勒贝格可测集，其勒贝格外测度 $m^*E$ 称为勒贝格测度
  - <font color='chartreuse'>实际上就是：（勒贝格外测度彼此可加的不相交集族）称为（勒贝格可测集族）</font>
  - **实例**：
    - 矩形均可测
      -  **证明**：由矩形外测度是体积 + 矩形体积可加性直得结论
    - 勒贝格外测度有限的集合均可测
      - **互包证明**：
        - 设 $E$ 测度有限，
        - 由外测度三角不等式得 $m^*T \leq m^*(T\cap E)+m^*(T\cap E^C)$
        - 构造 $T$ 的开矩形覆盖 $\mathop{\cup}\limits^\infty_{n=1}I_n$，且满足 $\sum\limits^\infty_{n=1}|I_n| < m^*T+\varepsilon$
          - <font color='chartreuse'>由外测度的定义，对任意的 $\e$，总可以取到相应的 $I_n$</font>
          - 则 $m^*(T\cap E)+m^*(T\cap E^C) \leq m^*(\mathop{\cup}\limits^\infty_{n=1}I_n) \leq \sum\limits^\infty_{n=1}|I_n| < m^*T + \varepsilon$
        - 由 $\e$ 任意性，
- **等价定义（并集版Cara条件）**：设 $E\subset\R^n$
  - 若对任意 $A\subset E，B\subset E^c$，都有 $m^*(A\cup B) = m^*A + m^*B$
  - 则 $E$ 勒贝格可测
  - <font color='chartreuse'>这个定义方法更能看出（集合可测性）的本质是（不相交部分满足测度可加性）</font>
  - **证明**：
    - 设 $T=A\cup B$，则易得 $T\cap E = (A\cup B)\cap E = A$，同理 $T\cap E^c = B$
    - 验证Cara条件即可

### 可测集的性质

- **零测集可测性**：勒贝格外测度为 $0$ 的集合可测
  - **互包证明**：
    - 设 $m^* E = 0$，任取 $A\subset E，B\subset E^C$，由测度单调性 + 非负性得 $m^*A = 0$
    - 再由测度三角不等式，$m^*(A\cup B) \leq m^*A + m^*B = m^*B$
    - 再由测度单调性，$m^*B \leq m^*(A\cup B)$
    - 互包即得并集版Cara条件，从而 $E$ 可测
- **补集可测性**：若 $E$ 可测，则 $E^C$ 可测
  - **证明**：
    - 由Cara条件可直得结论
- **全集可测性**：全集 $\R^n$ 和空集均可测
  - **证明**：
    - 前面已规定 $m^*\varnothing = 0$，故空集可测，且其补集 $\R^n$ 可测

- **极限封闭性（单增）**：
  - 设 $E_i$ 是单增的可测集列
  - 则 $\lim\limits_{n\to \infty}E_i = \mathop{\cup}\limits^\infty_{i=1}E_i$ 可测，且 $m^*(\lim\limits_{i\to\infty}E_i) = \lim\limits_{i\to\infty}m^*E_i$
  - <font color='chartreuse'>上一个是互不相交的可测集列，这一个是单增的可测集列</font>
  - **证明（补分拆 + 裂项相消）**：
    - 构造分拆 $S_i = E_{i+1}-E_i$，显然其互不相交，且 $\mathop{\cup}\limits^\infty_{i=1} S_i = \mathop{\cup}\limits^\infty_{i=1} E_i$
    - 由可数可加性即得题设集合可测
    - 若 $\lim\limits_{i\to\infty}m^*E_i = +\infty$
      - 由测度三角不等式即得 $m^*(\lim\limits_{i\to\infty}E_i) = +\infty$
    - 若 $\lim\limits_{i\to\infty}m^*E_i$ 有限
      - 由可数可加性，$m^*(\mathop{\cup}\limits^\infty_{i=1} E_i) = m^*(\mathop{\cup}\limits^\infty_{i=1} S_i) = \sum\limits^\infty_{i=1} m^* S_i = m^*E_1 + \sum\limits^\infty_{i=1} m^*(E_{i+1}-E_i)$
      - 由四则运算可测性 + 有限可加性得 $m^*(E_{i+1}-E_i) = mE_{i+1} - mE_i$
      - 综上即得等式
  - **本质**：转化为互不相交情况
- **极限封闭性（单减）**：

  - 若 $\exist n_0$，使得 $mE_{n_0} < +\infty$<font color='chartreuse'>（防止全部集合都为无穷测度集）</font>
  - 则 $\lim\limits_{n\to \infty}E_i = \mathop{\cup}\limits^\infty_{i=1}E_i$ 可测，且 $m^*(\lim\limits_{i\to\infty}E_i) = \lim\limits_{i\to\infty}m^*E_i$
  - **证明（补集单增性）**：
    - 构造集列 $S_n = E_{n_0}-E_n（n>n_0）$，易得其单增且可测
    - 由集合极限定义，$\lim\limits_{n\to \infty} S_n = E_{n_0}-\mathop{\bigcap}\limits^\infty_{n=n_0+1}E_n$
    - 由测度可加性 + 单增列测度极限得 $m(\mathop{\cap}\limits^\infty_{n=n_0+1}E_n) = m(\mathop{\cap}\limits^\infty_{n=1}E_n) = \lim\limits_{n\to\infty}mE_n$
  - **本质**：单增列情况的对偶推论罢了
- **极限封闭性**：
  - 可测集列的上下极限均可测
    - **证明**：根据上下极限定义 + 可数运算可测性，集列的上下极限均可测
  - 可测集列 $E_n$ 若存在极限，则极限可测
    - **证明**：已知集列若存在极限，则上极限 = 下极限 = 极限，从而极限也可测
- **代数封闭性**：$\R^n$ 中所有可测子集构成 $\sigma$ 域
  - **证明**：由四则运算封闭性 + 极限封闭性即得结论
- **测度连续性**：
  - 设集列 $E_n$ 收敛
  - 若 $\exist K_0$，使得 $m\Big(\mathop{\bigcup}\limits_{n\geq K_0} E_n\Big) < +\infty$，则 $m(\lim\limits_{n\to \infty}E_n) = \lim\limits_{n\to \infty}m^*E_n$
  - **证明（构造单调列）**：
    - 令 $S_n = \mathop{\bigcup}\limits_{k\geq n} E_k$，则 $S_n$ 单减， $\lim\limits_{n\to \infty}m^*S_n = m(\overline{\lim\limits_{n\to\infty}}E_n)$
      - <font color='chartreuse'>这个构造方法和数列上极限的确界定义的构造方法是一样的</font>
      - 再由 $E_n$ 收敛，上式 $ = m(\lim\limits_{n\to\infty}E_n)$
      - 再因为 $E_n\subset S_n$，由测度单调性 $\lim\limits_{n\to\infty}m^*E_n \leq \lim\limits_{n\to\infty}m^*S_n = m(\lim\limits_{n\to \infty}E_n)$
    - 再设 $F_n = \mathop{\bigcap}\limits_{k\geq n} E_k$，则 $F_n$ 单增……
      - 同理得 $\lim\limits_{n\to\infty}m^*E_n \geq m(\lim\limits_{n\to \infty}E_n)$
    - 互包即得相等

## Lebesgue不可测集

### 经典不可测集

- **Vitali集**：
  - **有理平移集**：任取 $x\in(0,1)$，设 $R_x = \{\xi\in(0,1)\mid \xi-x\in \Q\}$
    - <font color='chartreuse'>集合 $R_x$ 依赖于 $x$ 的选取</font>
      - **平移性**：把 $\R$ 看作一维向量空间，则 $R_x$ 表示子空间 $\Q$ 沿向量 $x$ 的平移
      - **非空性、可数性**：显然
      - **不相交性**：若 $R_x\neq R_y$，则 $R_x\cap R_y = \varnothing$
        - **证明**：
          - 反设存在 $\xi$ 和 $x\neq y$ 满足 $\xi-x\in \Q$ 且 $\xi-y\in\Q$
          - 则由封闭性，$x-y\in\Q$，显然此时 $R_x = R_y$
          - <font color='chartreuse'>因为有理数集满足稠密性，且关于加减法封闭，故 $\Q$ 平移任意有理数向量后依然是 $\Q$，即 $\Q$ 以任意有理数为周期</font>
      - **分拆性**：$\mathop{\bigcup}\limits_{x\in (0,1)} R_x = (0,1)$
        - **证明**：
          - 显然 $\forall R_x\subset (0,1)$，故 $\mathop{\bigcup}\limits_{x\in (0,1)} R_x \subset (0,1)$
          - 反设存在 $q\in(0,1)$ 不属于任意 $R_x$，则 $\forall x\in (0,1)$，恒有 $q-x\notin \Q$
            - 但显然有 $q-q = 0\in\Q$，矛盾。故 $(0,1) \subset \mathop{\bigcup}\limits_{x\in (0,1)} R_x$
          - 互包即得结论
  - **代表元集**：取某个分拆 $\mc R = \{R_x\}$，在所有 $R_x$ 都取出一点，得到集合 $S$
    - <font color='chartreuse'>良定义性：由选择公理（超穷归纳法）易得</font>
    - **不可数性**：
      - **证明**：由于 $\ool\Q = \alef_0，\ool\R = \alef = 2^{\alef_0}$，故反证易得任意分拆 $\{R_x\}$ 都是不可数集族，从而 $S$ 不可数
      - <font color='chartreuse'>前面已知 $\Q$ 平移有理数后依然是 $\Q$，故 $S$ 不可数说明 $\Q$ 在平移任意无理数后的结果都不同</font>
    - **不可测性**：详见下面的证明
  - **代表元集的平移**：按某种顺序排列 $(-1,1)$ 上的有理数，得到数列 $Q = \{q_i\}^\infty_{i=1}$，则对任意 $n\in\N$，$S_n = \{x+q_n\mid x\in S\}$ 都是不可测集
  - **证明**：
    - **平移性**：$S_n$ 就是集合 $S$ 沿某个有理数向量 $q_n$ 的平移
    - **测度有界性**：
      - **上界（平移有界性）**：
        - 由于 $x\in(0,1)$，故 $\Q$ 的平移距离有限，即 $\mathop{\cup}\limits^\infty_{n=1}S_n \subset (-1,2)$
      - **下界（平移覆盖性）**：
        - 已知 $R_x$ 具有不相交性，而 $S$ 是 $\mc R = \{R_x\}$ 的代表元，故易得 $S_n$ 也有不相交性
          - <font color='chartreuse'>若 $n\neq m，则 S_n\cap S_m = \varnothing$
            - 证明：$x_n-x_m$ 若为有理数，则两集合相等</font>
        - 设 $S\cap R_x = \tau$
          - <font color='chartreuse'>$\tau$ 就是 $S$ 在集合 $R_x$ 中所取的点（代表元）</font>
        - 易得 $\forall x\in (0,1)$，总存在 $q_n = x-\tau$，使 $x\in S_n$
          - 即 $n$ 和 $x$ 存在一一对应，从而 $R_x$ 和 $S_n$ 存在一一对应，从而 $\ool{\mc R}= \ool{\mc S} = \alef$
          - <font color='chartreuse'>实际上就是两种用 $\Q$ 的平移来分拆 $(0,1)$ 的方法
            - $\mc R$ 是先平移，再相并
            - $\mc S$ 是先相并（然后取代表元），再平移（即取遍所有不同代表元）
            - 所以它们肯定是彼此等价且对应的</font>
          - 再由 $\mc R$ 是 $(0,1)$ 的分拆，即得 $(0,1)\subset \mathop{\cup}\limits^\infty_{n=1}S_n$
    - **测度收敛性**
      - 由测度单调性，$1 = m^*(0,1)\leq m^*(\mathop{\cup}\limits^\infty_{n=1}S_n) \leq m^*(-1,2) = 3$
      - 再由平移不变性得 $m^*S_n = m^*S$，即每个 $S_n$ 外测度相等
      - 反设 $S$ 可测，则由平移可测性得 $S_n$ 均可测
        - 再由互不相交性 + 可数可加性得  $m^*(\mathop{\cup}\limits^\infty_{n=1}S_n) = \sum\limits^\infty_{n=1}m^*S_n$
        - 等号左侧有界，故右侧级数收敛，故只能是 $\lim\limits_{n\to\infty} m^*S_n = 0$
        - 但 $S_n$ 选取的顺序是任意的，故只能是 $\forall m^*S_n = 0$，但这与 $\sum\limits^\infty_{n=1} m^*S_n \geq 1$ 矛盾
  - **理解**：
    - 证明的核心其实就是探讨 $S_n$ 的分拆性，思路的难点就是发现这两种不同的分拆方法
      - $R_x$ 是可数集，且不可数个 $R_x$ 分拆了 $(0,1)$
      - $S_n$ 是不可数集，且可数个 $S_n$ 分拆了 $(0,1)$
      - 前者容易理解，但充当证明工具很难。后者很难理解，但充当证明工具很容易。
        - 因为可数加和就是级数，关于级数已经有了一套很成熟的理论，但对于不可数加和却知之甚少
    - 数分中已经提到，无穷多个无穷小量的和不一定是0。同理，无穷多个零测集之并的测度也不一定是0。我们就是根据这个特点构造不可测集的
      - 离散性使单个 $S_n$ 测度为0，有界性使 $\bigcup S_n$ 测度不为0，故不满足可数可加性
    - 但很有意思的是，“不可数的零测集”其实也是存在的。详见后面的反例
  - **本质**：
    - 每个 $S_n$ 外测度都相等，但可数个之和却等于 $1$，所以 $m^*S_n$ 到底等于多少呢？实际上不同的 $\R$ 选取方式会导致相应 $S_n$ 的外测度不同，而具体的 $\mc R$ 是难以写出的（因为涉及到超穷归纳法，只能由选择公理判断其存在性），所以不可测集的外测度也是难以求出的（但对于每种取法所确定的 $S_n$，它的外测度肯定都是存在的）
      <!-- - 某个（选择方法）到（实数）的映射 $\sigma$ 的像 -->
  - **推论**：
    - 实际上不一定非得是有理数集，任意一个 $(0,1)$ 内满足稠密性的 $F_\sigma$ 集都可以
    - 下确界为0：代表元全部选在 $[0,\varepsilon]$ 内
    - 上确界为1：代表元取遍非零测 $F_\sigma$ 集
- **圆周上的角度分拆集**
  - **选取可数集**：
    - 在单位圆周 $C$ 上任取一点 $M$，任取无理数 $\a$
    - 选取另一点 $A_1$，设圆心角 $\angle MOA_1 = \theta_1$
    - 再选取另一点 $A_2$，使得 $\angle MOA_2 = \theta_1 + \alpha\pi$
    - 不断做下去，可以导出不重合数列 $\Theta_1 = \{\t + k\a\pi\}^{\infty}_{k=0}$
  - **可数集分拆**：
    - 在 $C$ 中挖去所有 $\{\theta_1\}$ 的点，选取新无理数 $\b$，继续构造 $\{\theta_2\}$
    - 不断做下去，最终得到数列集族 $\{\theta\}$，易得其为 $C$ 的分拆，且势为 $\alef$
  - **选取代表元集**：
    - 再从每个数列中选取第一个点，构造不可数集 $\varPhi_1$（代表元）
  - **代表元集分拆**：
    - 不断做下去，最终得到可数集族 $\Phi = \{\varPhi_n\}$
    - 易得点列 $\varPhi_k$ 的势为 $\alef$，集族 $\Phi$ 的势为 $\alef_0$
  - **验证不可测性**：
    - 若 $\varPhi_n$ 可测，满足测度可加性，而每个 $\varPhi_n$ 的元素个数相等，从而测度相等。但其总和为单位圆周，测度为1
    - 若每个 $\varPhi_n$ 的测度为0，则总测度为0
    - 若每个 $\varPhi_n$ 的测度为1，则总测度无界
    - 总之测度之和不可能为1，这就产生了矛盾，$\varPhi_n$ 不可测。

### 不可测集的性质

- **不可测集的构造方法**：
  - 代表元的选取
  - 每个不可测集，都离不开在某个父集（连续集比较方便，Borel集也不差）中构造分拆，然后在分拆中选取代表元。
    - 这个代表元的选取依赖于选择公理。不可测集的测度和选取的方式有关。
- **不可测集无可加性**：
  - 这些代表元的测度相等，但是有可数个，且总测度被父集约束为有界，从而产生矛盾
    - $S_n$ 的平移性，$\varPhi_n$ 的数列序号，都是不同的代表元选取方法，同时也导出了代表元测度相等性
- **不可测集的存在性**：
  - 在抽象实分析中会学到，抽象测度的定义是：满足非零性、单调性、可列可加性的集合函数
  - 而且若集合函数满足上面三条性质，则其一定会遇到不可加的集合，即任何测度都必定存在相应的不可测集

## Lebesgue可测集的结构

- <font color='chartreuse'>可测集，无论是可加性定义，还是等测包定义，都是很强的性质。它们规定集合必须具有一定的结构
  - 可加性定义约束了其离散点的数量，从而控制其结构
  - 等测包定义约束了其离散点的位置，从而控制其结构</font>

### 等测包

- **等测包**：若 $E\subset O$，且 $\forall \e>0，m^*(O-E) \leq \e$，则称 $O$ 是 $E$ 的等测包
- **等测核**：若 $O\subset E$，且 $\forall \e>0，m^*(E-O) \leq \e$，则称 $O$ 是 $E$ 的等测核
- **四种等测性**：
  - 强等测：差集为零测集的两集合
  - 工具等测：就是上面的定义
  - 一般等测：测度相等的两集合
  - 一般 $\e$ 等测：$\forall \e>0，m^*(E\D O) \leq \e$
  - **强弱关系**：若满足条件a的集合一定满足条件b，则称a强于b
    - 不限定集合类型时，1和2等价。限定集合类型时，1强于2
      - <font color='chartreuse'>工具等测包能用可数集列逼近法得到，但强等测包不行</font>
    - 若两集合都是可测集，1和3等价。若出现不可测集，1强于3
      - <font color='chartreuse'>由外测度三角不等式 + 外测度单调性易得结论</font>
    - 由外测度存在性，任意集合都存在一般 $\e$ 开等测包，即4是最弱的条件
  - **存在性**：
    - 显然集合本身即为等测包/核，故总存在。但限定类型的等测包/核不一定存在
    - **反例**：
      - 对闭矩形
        - 存在（强 $G_\d$ 等测包），不存在（强开等测包）
          - <font color='chartreuse'>因为定义 $G_\d$ 型集时必须依赖于可数开集列的选择，所以用工具等测包，和Vitali集类似
          - 此时外测度的（开矩形覆盖下确界）是取不到的</font>
        - 存在（工具开等测包），不存在（一般开等测包）
          - <font color='chartreuse'>已知开集 - 闭集 = 开集，但欧氏空间上不存在零测非空开集</font>
      - 对Vitali集
        - $(0,1)$ 是（工具 $\e$ 等测包），但不存在任何（一般等测包）
  - **唯一性**：
    - 集合意义下不唯一：若两集合元素完全相同，则认为它们在集合意义下相同
    - 测度意义下唯一：若两集合强等测，则认为它们在测度意义下相同
- **可测包**：包含偏序下，包含 $E$ 的最小可测集
- **可测核**：包含偏序下，含于 $E$ 的最大可测集

### Borel集族

- **左开右闭矩形**：$G = \{(x_1,...,x_n)\in R^n\mid c_i<x_i\leq d_i，i=1,...,n\}$
- **矩形分拆定理**：设 $G$ 是 $\R^n$ 中的非空开集，则其是（至多可数个）（互不相交的）（左开右闭矩形）的并集
  - **证明**：
    - **写出矩形网格**：构造互不相交的左开右闭矩形 $\{B^k_{\bd m}\}$，即设 $$B^{k}_{\bd m} = \{(x_1,...,x_n)\mid \frac{m_i}{2^k}< x_i \leq \frac{m_i+1}{2^k}，i=1,...,n\}$$
      - $i$ 是坐标维度的序号（有限个），$n$ 是总空间的维度数量
      - $\bd m$ 表示矩形的位置（有限维向量，数量为可数个）
      - $k$ 是矩形列的序号（可数个），同时表示矩形的边长
      - <font color='chartreuse'>我们把 $\{B^k_{\bd m}\}$ 看作一个二维序列，横向是关于 $\bd m$ 的，纵向是关于 $k$ 的
        - 每一行都表示 $\R^n$ 的边长为 $\dfrac{1}{2^{k}}$ 的矩形网格分割
        - 每一列都表示 $\bd m$ 位置的不同大小的矩形</font>
      - **覆盖性**：易得对于任意有限的 $n$ 和 $k$，都存在可数个 $m$ 使得 $\R^n$ 被 $\{B_k\}$ 覆盖
      - **单减性**：矩形的测度关于 $k$ 单调递减，因为 $m^*(B_k) = |B_{k}| = \dfrac{1}{2^{nk}}$
    - **空隙填充**：
      - 设 $I^{k}_j$ 表示 $\{B^k_{\bd m}\}$ 中，含于 $G$ 且和所有 $\{I^{l}_j\mid l<k\}$ 都不相交的元素
        - <font color='chartreuse'>它们用来填充 $G$ 和 $\mathop{\bigcup}\limits^{k-1}_{l=1}\mathop{\bigcup}\limits_{j} I^{l}_j$ 之间的空隙
        - 因为 $\bd m$ 至多可数，故存在一种自然数排序 $B^k_j$ 来替代 $B^k_{\bd m}$。再从中选取满足上述条件的元素，就是 $I^{k}_j$
        - $j$ 为至多可数个，但数量不确定，依赖于矩形的选取</font>
    - **验证分拆性（元素分析法）**：
      - 由 $G$ 是开集，得其中任意点 $x$ 必有邻域 $O(x,\delta)\subset G$
      - 由于 $\lim\limits_{k\to\infty} |B_k| = 0$，故必定存在 $k$ 和 $\bd m$ 使得 $x\in B^k_{\bd m} \subset O(x,\delta)$，即存在 $k$ 和 $j$ 使得 $x\in I^{k}_j$
      - 再由 $x$ 任意性即得 $G = \mathop{\bigcup}\limits_{i,k}I^k_i$
  - **理解**：实际上和Stein的证明相同。虽然思路很简单，但写出所用集合的严格通式很难，需要较长的解释
  - **本质**：前面证明了 $\R$ 上开集可分解为不相交开区间，而当维度大于 $1$ 时，开集可分解为不相交半开半闭矩形
    - 类似数分中二维反常积分可积等价于绝对可积。其可以构造一个正函数通道来体现极端情况，且不失积分区域连通性。但一维上没有通道一说，所以没有这个性质。
    - 在这里，一维若出现多个不相接的开集，则可以直接用各个不相接的开集本身充当覆盖。但二维不行，二维的各个不相接开集 $G_i$ 可以是形状不规则的，仅用矩形是不能充当 $G_i$ 本身的。（用Stein中的网格的极限小形式可以给出直观解释）
    - 而连续的开集不可以用不相交的小开集来覆盖，即使在一维上也不行。所以二维只能用半开半闭矩形进行覆盖（称为几乎不相交的矩形）
- **Borel集**：可由开集经过（至多可数个交并差运算）得到的集合
- **Borel集类 $\mathscr{B}(\R^n)$**：$\R^n$ 中全体开集经过（至多可数个交并差运算）后得到的 $\sigma$ 域，称为Borel集类
- **Borel可测定理**：$\R^n$ 中的Borel集类可测
  - **引理**：$\R^n$ 中的开集、闭集、$F_\sigma$ 型集、$G_\d$ 型集均可测
    - **证**：
      - 由矩形分拆定理，开集能被可数个矩形分拆。再由矩形可测性 + 可测集的可数可加性即得开集可测
      - 闭集都是开集的补集，由补集可测性即得可测
      - $F_\sigma$ 型集是可数个闭集的并，$G_\d$ 型集是可数个开集的交。由极限封闭性即得可测
  
### Lebesgue可测集族

- **Borel势定理**：$\R$ 上全体Borel集的势为 $\alef$
  - **证明**：由开集分解定理得全体开集的势为 $\alef$，再由Borel集的定义，其全体的势为 $\alef_0·\alef = \alef$
- **Lebesgue势定理**：$\R$ 上全体Lebesgue可测集的势 $\ms L(\R) = 2^\alef$
  - **证明（互包法）**：
    - $\ms L(\R) \leq 2^\alef$
      - 已知 $(0,1)$ 的幂集的势为 $2^\alef$，而 $\R$ 可被至多可数个开区间覆盖，所以 $\R$ 的幂集的势也为 $2^\alef$，从而 $\R$ 中可测集的势不可能大于它
    - $\ms L(\R) \geq 2^\alef$
      - 已知存在势为 $\alef$ 的可测集（Cantor集），易得其所有的子集也可测（？），故 $\R$ 中可测集的势不小于它
- **可测集等测定理**：$E$ 是可测集 $\LR$ 存在一个 $E$ 的工具开等测包
  - **证明**：
    - **充分性**：已知开集、零测集可测，由四则运算传递性即得 $E$ 可测
    - **必要性**：
      - 设 $E$ 是可测集。易得总存在一个开集 $O$ 包含 $E$
        - 由四则运算传递性，$O-E$ 可测
      - 由矩形分拆定理，总存在至多可数个矩形 $\mc I = \{I^k_j\}$ 分拆 $O$
        - 再由 $k$ 的定义，当 $k$ 足够大时，$\mc I$ 中矩形可任意小，即 $\mc I$ 可无限细
        - 故其中必定存在子集 $\mc I_1$ 满足其并是 $O-E$ 的等测包，子集 $\mc I_2$ 满足其并是 $O-E$ 的等测核
          - <font color='chartreuse'>找一个单减至 $0$ 的数列 $a_k$，再找到与 $O-E$ 测度相差 $a_k$ 的相应矩形子列即可。（就是数分中介绍过的二维数列取对角线子列法，详细过程在2.2中）</font>
      - 再由 $E$ 的任意性，将 $E$ 和 $O-E$ 互换位置即得结论
- **一般的等测定理**：任意集合都存在一个一般 $G_\d$ 等测包
    - **证明**：
      - 取所有一般 $\e$ 开等测包的交，显然它就是下确界，且符合 $G_\d$ 型集的定义
- **勒贝格测度的正则性**：任意可测集都存在一个Borel强等测核
    - **证明**：
      - 若 $E$ 无界，则构造单增无界开矩形覆盖，使得交集 $E_n$ 为单增有界可测集列。从而满足单增可测集的极限连续性，得到 $mE = \lim\limits_{n\to \infty}mE_n $
        - 假设可以找到极小真包含子集（Borel集） $F_n\subset E_n$，则 $mE = \overline{\lim\limits_{n\to\infty}}mF_n\leq mF\leq mE$（前面是单增函数极限性质，后面是测度单调性），得到 $mE=mF$
        - 最后得到 $m(E-\mathop{\cup}\limits^\infty_{n=1}F_n) \leq \sum\limits^\infty_{n=1}m(E_n-F_n) = 0$
      - 若 $E$ 有界，则利用其补集的开等测包情况即可证明
  - **推论**：（不可测集）与其（任意开父集）的差集的外测度均不为0
      - **证明**：
        - 否则由四则运算传递性得其可测，从而矛盾
