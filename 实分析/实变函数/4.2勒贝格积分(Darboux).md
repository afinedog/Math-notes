# 勒贝格积分（达布和定义法）

## 定义方法

- 简单函数：$I = \lim\limits_{n\to\infty}\sum\limits^{N_n}_{i=1}C_i^{(n)} mE_i^{(n)}$（类似定积分）
- 值域划分：$E_i = E\{x\mid l<f(x)\leq l_{i+1}\}$，$\sum\limits_{i} \xi_i mE_i$
- 定义域划分：此时要做任意分划

## 常义积分

- **常义勒贝格积分**：设 $E$ 是有界可测集，$f$ 是可测函数， $f(E)\subset (\alpha,\beta)$
  - **值域划分**：设 $D:\alpha=l_0<l_1,...,<l_n=\beta$
    - **最大段**：$\delta(D) = \max\limits_{k} |l_k-l_{k-1}|$
    - **水平集列**：$E_k = \{l_{k-1} < f(k) \leq l_k\}$
  - **达布和**：设  $\xi_k\in [l_{k-1},l_k]$，则达布和为 $S(D) = \sum\limits^n_{k=1} \xi_k m(E_k)$
  - **勒贝格可积**：若极限 $\lim\limits_{\d(D)\to 0} S(D)$ 存在，则称 $f$ 在 $E$ 上勒贝格可积
  - **勒贝格积分**： $I = \dis\int_E f(x)dm(x)$
- **达布引理0**：
  - **达布大和**：$\overline{S}(D) = \sum\limits^n_{k=1} l_k m(E_k)$
  - **达布小和**：$\underline{S}(D) = \sum\limits^n_{k=1} l_{k-1} m(E_k)$
  - $f$ 勒贝格可积 $\LR \lim\limits_{\d(D)\to 0} \ol S(D) = \lim\limits_{\d(D)\to 0} \ul S(D)$
  - **证明**：
    - 易得 $\ul S(D) \leq S(D) \leq \ol S(D)$，故题设条件等价于 $\lim\limits_{\d(D)\to 0} S(D)$ 存在
- **达布引理1**：对于任意有界有限支撑函数，当划分加细时，其大和不增，小和不减
  - **证明**：
    - 针对定义式，不断放缩即可
- **达布引理2**：$f$ 勒贝格可积 $\LR \inf\limits_D \ol S(D) = \sup\limits_D \ul S(D)$
  - **证明**：
    - 由达布引理1，易得划分越细，小和越可能大，大和越可能小，从而 $\ol S$ 的极限值就是下确界，$\ul S$ 的极限值就是上确界
- **常义可积定理**：若 $E$ 有限可测，$f$ 是 $E$ 上可测函数，则 $f$ 可积
  - **证明**：
    - 由大小和的定义得任意划分 $D$ 都有 $\ul S \leq \ol S$
    - 反设 $\sup \ul S < \inf \ol S$
  - **实例**：狄利克雷函数可积
- **积分性质**
  - **技巧本质**：（$i-1$ 和 $i$ 之间可以分离 $\delta$，从而达到正反向转化）
  - **集合可加性**：
    - **互包证明**：
      - 取 $\wt E_i^{(j)} = E_j\cap \wt E_i$，由可测可加性 + 划分不等式得 $|l_{k-1}-l_k|\leq\delta$
      - 反向不等式：利用 $l_{i-1}、f、l_i$ 的不等式
  - **线性**：
    - **证明**：定义易得数乘性，只需证可加性
      - 正向不等式：取划分之并，用上界 $\delta$ 分离+取无穷小
      - 反向不等式：取划分之并，讨论函数 $f+g$，然后分离+取无穷小即可
  - **单调性**：
    - **证明**：
      - 取 $F=f-g$，定义证明即可
  - **积分上下界不等式**：
    - **证明**：
      - 由单调性可直接推出
  - **绝对值不等式（三角不等式）**：
    - **证明**：
      - 由单调性易得 $\dis \int_E -|f|dx \leq \int_E f dx \leq \int_E |f|dx$，再由正负分解定理即得结论
  - **零测控制性**：
    - 设 $E$ 有界可测，$f$ 有界可测
    - 若
      - $f(x) \geq 0 \ae$
      - $\dis\int_E fdx = 0$
    - 则 $f(x) = 0 \ae$
    - **证明**：
      - 拉格朗日法（用水平集分割积分域）+ 积分上下界不等式
    - **理解**：必须是有界有限支撑，否则无界点可能位于无穷远处

## 反常积分

### 无界函数的反常积分

- **非负函数积分（控制列定义法）**：
  - **控制函数列**：$f_m = \min\{f(x),m\}$
  - **勒贝格可积**：若 $\dis\int_E fdx < +\infty$，则称 $f$ 勒贝格可积
    - **等价命题**：正负分解全部可积
- **一般函数积分（控制列定义法）**：
  - **积分存在（条件收敛）**：若 $f$ 的正部分和负部分至少有一个可积，则称积分存在
  - **勒贝格可积（绝对收敛）**：若 $\dis\int_E |f|dx < +\infty$，则称 $f$ 勒贝格可积


### 无限区间的反常积分

- **非负函数积分（膨胀列定义法）**
  - **膨胀列**：设闭球列 $\overline{B}_k = \{x\mid \|x\|\leq k\}$，$E_k = E\cap \overline{B}_k$
    - <font color='chartreuse'>易得 $E_k$ 有限可测，$f$ 在 $E_k$ 上可积</font>
  - **膨胀积分列**：设 $J_k = \dis\int_{E_k} fdx$，则 $\dis\int_E fdx = \lim\limits_{m\to\infty}J_m$
    - <font color='chartreuse'>由区间可加性得 $J_k$ 单增，从而存在广义极限</font>
  - **勒贝格可积**：若 $\dis\int_E fdx < +\infty$，则称 $f$ 勒贝格可积
- **一般函数积分（膨胀列定义法）**：
  - **积分存在（条件收敛）**：若 $f$ 的正部分和负部分至少有一个可积，则称积分存在
  - **勒贝格可积（绝对收敛）**：若 $\dis\int_E |f|dx < +\infty$，则称 $f$ 勒贝格可积

### 一般的反常积分

- **无限区间的无界函数**：将膨胀列和控制列按某种排序方式合并成一个即可

### 反常积分的性质


- **方法总结**：正负分解、可积转化、配凑
- **比较定理**：
  - 设 $f$ 可测，$g$ 非负可积
  - 若 $|f(x)| \leq g(x)$，则 $f$ 可积且 $\dis\int_E |f|dx \leq \int_E gdx$
  - **证明**：
    - 由数列极限的保序性直得结论
- **集合可加性**：
  - **证明**：
    - 数列的正负分解 + 区间分解 + 极限的线性传递性
- **线性**：
  - **证明**：
    - **数乘性**：系数数乘等后，只需将水平集列等比放缩，再由极限的运算性质即可得结论
    - **比较不等式**：$\{f+g\}_m \leq \{f\}_m + \{g\}_m \leq \{f+g\}_{2m}$
      - <font color='chartreuse'>由该不等式包夹，即可得到非负情况的结论</font>
    - **分解不等式**：$(f+g)^+ \leq f^+ + g^+$
      - <font color='chartreuse'>由该不等式易得一般情况的结论</font>
- **零测控制性**：几乎处处相等的函数积分值相等
  - **证明**：用拉格朗日方法，将积分域用水平集分解即可
- **单调性**
  - **证明**：
    - 设 $F = g-f$，则 $F\geq 0$。对于正负部分分别讨论比较即可

### 习题

- **黎曼积分与勒贝格积分**：有界闭区间上，若 $f$ 黎曼可积，则其也勒贝格可积，且积分值相等
  - **证明**：
    - 设下确界函数数列 $\varphi_m$，上确界函数数列 $\psi_m$，极限相等
    - 由极限可测性，大小和函数可测
    - 再因为划分极限的积分意义，得到反向不等式，从而相等，从而媒介 $f$ 可测

## 勒贝格积分的极限定理

- **Levi定理（单调收敛定理）**
  - **证明**：
    - 数列 < 极限：数列本来就是单增逼近
    - 极限 < 数列：用一致收敛的多余无穷小量，补偿零测集的缺失无穷小量
    - 积分无界时，分离为有限区间之并即可
- **勒贝格基本定理（函数项级数积分-极限换序）**：
  - **证明**：对部分和函数使用Levi定理即可
  - **推论**：
    - 设 $E_k$ 是互不相交的可测集列，$E = \mathop{\bigcup}\limits^\infty_{k=1} E_k$
    - 若 $f$ 在 $E$ 上可积
    - 则 $f$ 在每个 $E_k$ 上都可积，且满足集合可加性
    - **证明**：
- **Fatou引理**：设 $f_n$ 是 $E$ 上非负可测函数列，则 $\dis\int_E \liminf\limits_{n\to\infty} f_n \leq \liminf\limits_{n\to\infty}\int_E f_n$ 
  - **证明**：
    - 设 $g_k = \inf\limits_{n\geq k} f_n(x)$，再由Levi定理即可
  - **反例**：
    - 当 $f_n\to f$ 时，等号也不成立：$f_n(x) = \begin{cases} m & x\in (0,\dfrac{1}{m}) \\\\ 0 & x\in [\dfrac{1}{m},1] \end{cases}$
      - **证明**：
        - **点态收敛性**：$f_n\to 0 \pw$
        - 计算易得左式为 $0$，右式为 $1$

### 控制收敛定理

- **思路**：考虑对于收敛可测函数列，积分-极限可换序的最低条件
  - 由数分知识，一致收敛的函数列积分-极限可换序
  - 由Egorov定理，仅任意小的测度上不一致收敛
- **积分的绝对连续性**：
  - **证明**：
    - 首先令膨胀列的序号 $m\to\infty$，使得积分区域极小
    - 然后令控制列的序号 $l\to\infty$，使得积分区域上积分值逼近极限函数
    - 最后放缩即得结论
- **勒贝格控制收敛定理**：
  - 设 $f_n$ 是 $E$ 上的可测函数列，$g$ 在 $E$ 上可积
  - 若
    - $f_n\Rt f$
    - $|f_n(x)| \leq g(x) \ae$
  - 则 $f$ 可积，且积分和极限可换序
  - <font color='chartreuse'>只是将4.1中的几乎处处收敛弱化为了依测度收敛而已</font>
  - **证明**：
    - 由Riesz定理，$f_n$ 存在几乎处处收敛的子列
    - 由几乎处处收敛性 + $|f_{n_k}(x)| \leq g(x) \ae$ 即得 $|f(x)| \leq g(x) \ae$
      - <font color='chartreuse'>依测度收敛不能得到极限保号性，必须至少是几乎处处收敛才行</font>
      - 再由比较定理即得 $f$ 可积
    - **积分域有限时**：
      - 首先由依测度收敛性，易得 $f_n$ 的无界区域极小
      - 然后由控制函数可积性，得差函数的绝对连续性，从而积分可交换
    - **积分域无限时**：
      - 令膨胀列的序号 $m \to\infty$，逼近极限函数
      - 在 $E_m$ 上由控制函数可积性，得到作差函数的绝对连续性，从而积分可交换
- **勒贝格有界收敛定理**：
  - 设 $E$ 是有限可测集，$f_n$ 在 $E$ 上可测
  - 若
    - $f_n\Rt f$
    - 存在 $K$ 使得 $\forall n$ 都有 $|f_n(x)| \leq K \ae$
  - 则 $f$ 在 $E$ 上可积，且积分和极限可换序
  - **证明**：
    - 取控制函数 $g(x) = K$ 即可


#### 习题

- **黎曼可积充要条件**：设 $f$ 是闭区间上的有界函数，则 $f$ 黎曼可积 $\LR f$ 的不连续点是零测集
  - **证明**：
    - **必要性**：
      - 将区间 $2^n$ 等分，取大小和数列 $\ol S_n$ 和 $\ul S_n$
        - 易得数列的极限为达布大小和，从而 $\bar{f} = f = \underline{f}$ 几乎处处成立
      - 设不相等区间为 $E_0$，则 $mE_0 = 0$。再由收敛性 + 划分极小性导出 $f$ 在 $E_0^c$ 上连续即可
    - **充分性**：
      - 设不连续点集为 $E_0$，则由连续可积性得 $E_0^c$ 上可积
      - 由函数有界性，大小和的差函数在零测集上勒贝格积分为 $0$，从而大小和相等，从而黎曼可积

### 验证收敛

- 设 $0\leq s\leq 1$，则 $f_n(x) = \cfrac{(nx)^s}{1+(nx)^{s+1}}，x\in [0,1]$ 满足 $\lim\limits_{n\to\infty} I_n = 0$
  - **证明**：
    - 分类讨论 $nx$ 与1的大小关系
- 设 $f_n(x) = \cfrac{\sqrt{n^3}x}{1+n^2x^2}，x\in [0,1]$，则其满足 $\lim\limits_{n\to\infty} I_n = 0$
  - **证明**：
    - 设控制函数 $g(x) = \frac{2}{\sqrt{x}}$，把 $nx$ 和 $\frac{1}{4}$ 比较

### 等度的控制意义

- **积分等度绝对连续**：
  - 设 $E$ 是可测集，$\ms F$ 是 $E$ 上函数的某个集合
  - 若 $\forall \e > 0，\exists \d(\e) > 0$ 使得当 $A\subset E，mA<\d$ 时，对任意 $f\in \ms F$ 都有 $|\dis\int_A f(x)dx| < \e$，则称 $\ms F$ 关于积分等度绝对连续
    - <font color='chartreuse'>关于的积分绝对连续性 + 等度性</font>
- **Vitali定理**：
  - 设 $E$ 是有限可测集，$f_n$ 是可测函数列
  - 若
    - $f_n$ 满足积分等度绝对连续性
    - $f_n \Rt f$
  - 则 $f$ 可积，且积分和极限可换序
  - **证明**
    - 首先证明Cauchy收敛
    - 然后令 $f$ 为裂项级数，F为绝对值裂项级数，从而可积
    - 最后分离区间即可

## 积空间

### 积空间测度

- 乘积性：$m(A\times B) = mA·mB$
  - 矩形：直得
  - 开集：矩形覆盖 + 外部积的并分配律
  - $G_\delta$ 型集：单减开集列 + 外部积的交分配律
  - 有界可测集：极小开父集 + 外部积的并分配律（分离差集） + 差集测度为0（矩形比较）
  - 无界可测集：可数有界矩形覆盖 + 外部积的并分配律 + 可数可加性 + 乘法分配律重组
- 多维可测集类
  - **积拓扑和箱拓扑**：$\mathscr{C}_n\times \mathscr{C}_m\subset \mathscr{C}_{n+m}$
    - 讨论可测矩形即可
    - **推论**：积空间是包含可测矩形的最小 $\sigma$ 域
  - **Borel族与可测集族**：$\mathscr{B}_{n+m} \subset \mathscr{C}_n\times \mathscr{C}_m$
    - 开集是可数个左开右闭矩形之并
  - **内外逼近**：$\forall E\in \mathscr{C}_{n+m}，\exist E_1,E_2\in \mathscr{C}_n\times \mathscr{C}_m，使 E_1\subset E\subset E_2$ 且差集零测
    - Borel集和Lebesgue可测集的关系
    - 积空间和总空间相差一个零测集，而零测集不影响积分，从而总空间条件和积空间条件没差
- **截口独立性**
  - $x\in \mathscr{C}_n，y\in \mathscr{C}_m，则 E^x\in \mathscr{C}_m，E^y\in \mathscr{C}_n$
  - **证明**：
    - 设 $E=E^x\times E^y$，$\mathscr{C}$ 是满足条件的集合。其具有
      - 全集性：$R^n\times R^m \in \mathscr{C}$
      - 补传递性：$(\complement E)_x = \complement(E_x)，(\complement E)_y = \complement(E_y)$，从而 $\complement E\in \mathscr{C}$
      - 并传递性：$E^x = \mathop{\bigcup}\limits^\infty_{i=1} (E_i)_x，E^y = \mathop{\bigcup}\limits^\infty_{i=1} (E_i)_y$
    - 得 $\mathscr{C}$ 是含有所有可测矩形的最小 $\sigma$ 域
    - 相互包含得 $\mathscr{C} = \mathscr{C}_n\times \mathscr{C}_m$
  - **理解**：积具有复制性，所以其是相对较粗的拓扑

### 单调完备族

- **单调类**：单调元素的极限均封闭的集族
- **积空间最小性**：$\mathscr{C}_n\times \mathscr{C}_m$ 是 *最小的*、*含所有矩形的有限堆砌的* 单调类 $\mathscr{M}$
  - 单调性：
    - 由极限可测性，可测集是单调类
    - 从而积空间也是单调类，则有 $\mathscr{M}\subset \mathscr{C}_n\times \mathscr{C}_m$
  - 最小性：
    - 首先由交并差封闭性，得有限矩形堆砌类 $\mathscr{R}$ 是环
      - 有限性得其 *非单调类*、*非 $\sigma$ 域*
      - $\mathscr{R}\subset \mathscr{M}$
    - 对偶变换 $\Omega(E) = \{F\in \mathscr{C}_n\times \mathscr{C}_m\mid E\in \mathscr{C}_n\times \mathscr{C}_m\ 且\ E\triangle F，E\cup F\in \mathscr{M}\}$
      - **对称性**：取遍 $\mathscr{M}$ 中单个元素内部的所有二元分割 + 外部的所有全集与补集
      - **单调类**：由 $\mathscr{M}$ 的单调完备性，函数的值域（补集）也为单调类
      - 由E的任意性，E取 $\mathscr{M}$ 中有限矩形堆砌的 $M$ 补集，即得 $\mathscr{R}\subset \Omega(E)$
      - 则由 $\mathscr{M}$ 的最小性得 $\mathscr{M}\subset \Omega(E)$，从而 $\mathscr{M}$ 是环
        - （或空集的像是 $\mathscr{M}$，直得结论）
    - 取限制 $\Phi(E) = \{F\in \mathscr{C}_n\times \mathscr{C}_m\mid E\in \mathscr{M}\ 且\ E\triangle F，E\cup F\in \mathscr{M}\}$
      - 因为 $\mathscr{R}\subset \mathscr{M}$，则 $M\in \Phi(R)$，对称性得 $R\in \Phi(M)$，所以 $M\in \Phi(M)$，从而 $\mathscr{M}$ 是环
    - $\mathscr{M}$ 是 $\sigma$ 域：
      - 全集性：全集是矩形堆砌的单增极限
      - 补集封闭性：全集性 + 环封闭性
      - 可数封闭性：单调完备性
    - 再因为 $\mathscr{R}\subset \mathscr{M}$，由 $\mathscr{C}_n\times \mathscr{C}_m$ 是包含 $\mathscr{R}$ 的最小 $\sigma$ 域，得 $\mathscr{C}_n\times \mathscr{C}_m \subset \mathscr{M}$

### 累次性

- **截口可测性**：函数的截口原像等于原像集合的截口。而积空间集合截口可测
- **特征函数累次性**：
- **Fubini定理**：
  - 有界非负可测函数可累次积分
  - 截口可积且累次可积，则二元可积
  - 总空间可积则截口 a.e. 可积

## 有界变差函数

### 跳跃函数和导数

- 不连续点
  - 第一类
  - 可数个
  - 跃度有界
- **跳跃函数**：分点的跃度之和，依赖于划分
  - **Heaviside函数**：左右跃度指示函数
    - 右跃度指示 $\theta = \begin{cases} 1，x\geq 0 \\ 0，x<0 \end{cases}$，左跃度指示 $\theta_1$ 相反
  - 不连续点全体的跃度函数 $\varphi$ 单增，且 $h = f-\varphi$ 单增连续
- 四种导数：可以无界
  - 可微：导数相等且有界

### 阴影区间

- **函数的右控点（阴影区间）**：闭区间上的连续函数，$\exist x'\in (x,b)，f(x)<f(x')$
- **F.Riesz引理**：控点集是开集，且 $f(a_k)\leq f(b_k)$
- **广义右控点**：不连续点均为第一类，且左右极限也满足条件
- **强可微性**：闭区间上的单调函数 a.e. 可微
- **Fubini推论（导数变差收敛性）**：闭区间上的有限单增函数列 $f_n$，处处收敛到有限函数 $f$，则其导数 $f' = \sum\limits^\infty_{n=1} f'_n$ 在闭区间上 a.e. 成立

### N-L公式条件

- **常函数条件**：闭区间上的跳跃函数，其导数 a.e. 为0
  - **奇异函数**：非常值函数，导数 a.e. 为0
- **N-L公式条件**：闭区间上的单增函数，其导数勒贝格可积且满足N-L不等式

### 有界变差函数

- 变差：$V(\Delta,f)$
- 全变差：依赖于划分的上确界 $V^b_a (f) = \sup\limits_\triangle(\triangle,f)$
- 有界变差函数：
  - 有界性
  - 线性封闭性 + 三角不等式
  - 乘积封闭性
  - 常函数条件：有界变差函数f，若 $V(f) = 0$，则为常函数
  - 子集传递性 + 集合单调性
  - 集合可加性
  - 极限封闭性
- **Jordan分解定理**：有界变差函数 $\Leftrightarrow$ 可以分解为两个单增函数的差
  - **推论**：
    - 有界变差函数间断点性质 == 单调函数
    - 有界变差函数Riemann可积性
    - 分解不唯一性
    - 正规分解（P-N）
- 上限函数：$V^x_a(f)$
  - **零控制性**：若上限函数恒为0，则函数恒为0
- 有界变差函数与其上限函数的连续点相同
- **正向可微性**：有界变差函数 a.e. 可微，导数勒贝格可积，且满足N-L不等式
- **绝对连续**
  - 成立N-L公式
- **反向可微性**：可积函数存在上限函数，其绝对连续，且导数 a.e. $=f$
  - 分部积分法
  - 换元法
