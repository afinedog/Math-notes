# 勒贝格测度习题（徐）


## 勒贝格测度

- **勒贝格测度的正则性**：任何集合有等测包
  - **证明**：取测度极限列即可
  - **推论**：可测集的测度性质，推广到不可测集上

### 多边形的测度

- **三角形**：
  - **解1**：由测度体积定理，直接求即可
  - **解2**：由测度旋转不变性，不断将其旋转后合并，直到得到矩形
- **正多边形**：
  - **解2**：三角形的旋转并
- **圆盘**：
  - **解2**：求正n边形的测度极限
- **扇形**：
  - 角度可能是离散点集，此时满足 $m(S) \leq \cfrac{(b^2-a^2)m(E)}{2}$
    - 角度E分解成开区间列
  - 角度可测则扇形可测

### 映射与测度

- **连续变换保可测性**：设 $f$ 是连续变换，$E$ 可测，则 $f(E)$ 可测
  - **证明**：
    - 保紧性 + 零测传递性即得保可测性
- **线性变换测度公式**：设 $f$ 是非奇异线性变换，则 $m(f(E)) = det|f|·m(E)$
  - **证明**：
    - 集合分解成矩形的并集
    - 单位矩形成立，则极小矩形也成立
    - 线性变换分解为：
      - 坐标交换：易得
      - 首坐标数乘：已证
      - 首坐标和次坐标相加：
        - $x_1$ 方向上的长条变换，$y=1-x$ 左边变成左半正方形，右边变成右半正方形
        - 设A为左边，B为右边
        - $m(f(I)) = m(A)+m(B-e_1) = m(I) = 1 = det|f|$

## 可测集

- **Steinhaus定理**：设 $E$ 是可测集，则向量差点集 $F = \{x-y\mid x,y\in E\}$ 至少包含一个有界开球
  - $\lambda \in(1-\frac{1}{2^{n+1}},1)$（打通边长和面积）
  - 开矩体 $J=\{(a_1,...,a_n)\mid |a_i|<\frac{\delta}{2}\}$
  - **证明**：
    - 即证 $J$ 中点 $x_0$ 满足 $E\cap I$ 和 $(E\cap I) + \{x_0\}$ 相交
    - 已知 $m(I\cap (I+\{x_0\})) > \frac{|I|}{2^n}$（边长）
    - 从而 $m(I\cap (I + x_0))>2\lambda|I|$，从而必定相交

  - $\Psi(x) = \frac{x+\varPhi(x)}{2}$
  - 严格递增的连续函数

## Cantor集

### Cantor集的测度

- **Cantor集的外测度为0**
    - **证明（三分定义）**：由定义 $C = \mathop{\bigcap}\limits^\infty_{k=1} C_k$，已知 $C_k$ 是闭矩形的并集，且单调递减，故由可测集列的单调极限封闭性即得结论
    - **证明（进制定义）**：$m^*C = |C| =  k·(\frac{2}{3})^k \to 0$（可测集可数可加性）
    - **证明2**：用勒贝格可测的定义（外测度开等测包条件，或Caratheodory条件）证明

### Cantor映射

- **Cantor-Lebesgue函数（指示函数）**：$F:C\to\R，x \mapsto \sum\limits^\infty_{k=1} \cfrac{b_k}{2^k}$
  - 其中当 $x = \sum\limits^\infty_{k=1} \cfrac{a_k}{3^k}$ 时，相应地取 $b_k=\cfrac{a_k}{2}$
  - **理解**：
    - $b_k$ 是用逻辑01来表示康托尔集某点在三进制中第 $k$ 数位上是否为 $0$
    - $F$ 是将三进制数位换为二进制数位的映射
  - **本质**：已知Cantor集具有连续势，这其实就是给出了 $C$ 到 $\R$ 的双射
  - **存在性（良定义）**：
    - <font color='chartreuse'>通用方法是，证明定义域中等价类的像相同</font>
  - **连续性**：
    - **证明**：
  - **满射性**：$F:\mc C\to [0,1]$ 是满射
    - **证明**：上面已给出
  - **可延拓性**：$F$ 可延拓为 $[0,1]$ 上的连续函数
    - **证明**：
      - **延拓性**：易得 $\mc C^c$ 是Borel集，故 $\exists\bigcup (a,b) \cup \mc C = [0,1]$
      - **连续性**：易得 $\forall (a,b)\subset \mc C^c$，有 $F(a) = F(b)$，故定义 $(a,b)$ 上为常函数，由常函数连续性 + C-L函数连续性即得结论
- 存在单增连续双射 $F:[0,1]\to [0,1]$ 将一个Cantor集满射到另一个
  - **证明**：套用C-L函数的构造即可

### 类Cantor集

- **Cantor常数剖分 $\mc C_\xi$**：
  - 设 $\xi\in (0,1)$
  - 分 $k$ 步移除当前各集合中心位置的 $k$ 个 $\xi$ 长度开区间
  - $k\to\infty$ 时，剩下的集合就是剖分 $\mc C_\xi$
  - **实例**：$\xi = \dfrac{1}{3}$ 时，剖分为Cantor集
  - **补集Borel性**：$\mc C_\xi^c$ 是总长度为 $1$ 的开集并
    - **证明**：第k步后，集合剩余长度为 $(1-\xi)^k$
  - **零测性**：$m^*(\mc C_\xi) = 0$
    - **证明**：可测集运算性质即可
- **类Cantor集 $\wh{\mc C}$**：每次移除 $2^{k-1}$ 个长度为 $\ell_k$ 的开区间，满足 $\sum\limits^\infty_{k=1} 2^{k-1}\ell_k < 1$

- **密度**：
  - 设 $\R$ 上的集合 $E$ 可测，$x_0\in E$
  - 任取 $(a,b)$ 包含 $x_0$，若 $d_E(x_0) = \lim\limits_{(a,b)\to x_0} \dfrac{m\Big[ (a,b)\cap E \Big]}{b-a}$ 存在，则称其为 $E$ 在 $x_0$ 处的密度
- **全密点**：若 $d_E(x_0)=1$，则称 $x_0$ 是 $E$ 的全密点
