# 实变的可测函数

- **局部有界化**：有界可测集上的 a.e. 有界非零可测函数 $f$，$\forall \delta$，存在 $k_0$，使得 f上下均强有界
  - 构造值域限制列即可
- 可测函数的紧支集逼近：$\varphi_k(x) = g_k(x)\chi_{(0,k)}(x)$
- **近一致收敛**：Egorov定理的一致收敛
- 连续函数的复合传递近一致收敛性和依测度收敛性
- 一致连续的复合将近一致收敛强化为一致收敛
- a.e. 有限的可测函数可被连续函数 a.e. 逼近：依测度收敛 + R子列
  - **Dirichlet函数的逼近**：$D(x) = \lim\limits_{n\to\infty}\lim\limits_{k\to\infty} [cos(n!·2\pi x)]^{2k}$
  - Dirichlet函数不可被连续函数完全逼近
