# 实变函数的集合

## 数分中的经典反例

- 处处不连续、处处不可积的函数：Dirichlet函数
- 处处收敛，但不可积的函数项级数：无法交换极限、累次积分
- 无理点上连续，有理点上间断的单增函数：
  - **有理数列上限函数**：$\{r_n\}$ 是 $(a,b)$ 中有理数全体，收敛正项级数 $C_n$，则 $f(x) = \sum\limits_{r_n<x} C_n$ 
  - 其在无理点上连续，有理点上间断（这不就是跳跃函数？）

### 有界收敛定理

- **有界收敛定理**：设 $f_n(x)$ 是定义在 $[a,b]$ 上的可积函数
  - $|f_n(x)| \leq M$
  - $f(x)$ 可积且 $\lim\limits_{n\to\infty}f_n(x) = f(x)$
  - 则有  $\lim\limits_{n\to\infty}\int^b_a f_n(x)dx = \int^b_a f(x)dx$
- **证明**：
- 全体有理数列 $r_n$，以及 $f_n(x) = \begin{cases} 1,x=r_1,r_2,...,r_n \\ 0,其它 \end{cases}$
  - n越大，$f_n(x)$ 是1的可能性越大，从而值可能越大，因此有 $f_1(x)\leq f_2(x) \leq...$（并不是增加关系，而是可能性关系）
  - $f_n(x)$ 的极限是Dirichlet函数。因为每个函数列Riemann可积，所以其积分的极限存在。但极限函数的积分不存在
  - 不过，我们可以定义它们相等，这就超出了Riemann可积的范畴

### 微积分基本定理

- 可微且有界的函数，其导函数不可积

### 可积函数空间

- **距离函数**：每个非抽象空间中都可以定义距离
- 积分空间的距离函数：
  - 1-范数：$d(f,g) = \int^b_a |f-g|dx$
  - 2-范数：$d(f,g) = \sqrt{\int^b_a |f-g|^2dx}$
- **收敛**：$\lim\limits_{\substack{m\to\infty \\ n\to\infty}}d(f_m,f_n) = 0$
- **完备性**：$\exist f，使\lim\limits_{n\to\infty} d(f_n,f)=0$
  - 要实现完备性，必要条件是可积性。但
- **$R([0,1])$ 的不完备性**：构造类Dirichlet函数 $f(x)$，其中1区间是$\cup I_n$，满足 $r_n\in I_n，且|I_n| < \frac{1}{2^n}$
  - 它收敛，但是不可积，无极限f，所以积分空间 $R([0,1])$ 不完备

## 集合 

- **无最大基数定理**：幂集的势大于集合
  - **证明**：即证 $2^C>C$，构造恒等映射即可
  - **应用**：函数族、勒贝格可测集族
- **函数在某点的振幅**：$\overline{\lim\limits_{\substack{n\to \infty \\ \delta\to 0}}} |f(x_1)-f(x_2)|$
- 开集上的实值函数，其连续点是 $G_\delta$ 型集
  - **证明**：构造极限列 $D = \{x\mid |f(x_1)-f(x_2)|< \frac{1}{n}\}$ ，则其可数交集是 $G_\delta$ 型集
  - 换成闭集极限列也可以，因为相等的点不可能保留到最后，所以结果不变
- 收敛列的极限函数，其连续点集为 $\mathop{\bigcap}\limits^\infty_{m=1}\mathop{\bigcup}\limits^\infty_{k=1}\overset{\circ}{E_k}(\frac{1}{m})$（k是序号，m是极小值）
- **Barie第一函数类**：区间 $I$ 上连续函数列的极限函数全体 $B_1(I)$
  - 收敛列极限的级数写法：$\sum\limits^\infty_{n=1} [f_n(x)-f_{n-1}(x)]$
  - **Barie定理**：$R^n$ 上的 $F_\sigma$ 集 E，若每个闭集F无内点，则E无内点
    - **证明**：反设有内点，则 $O(x_0,\delta_0)$，且和 $F_1$ 不相交。则依次构造，和所有F不相交，且收敛到一个点。此时该点是内点但不属于所有F，从而不属于E，矛盾

### Borel集

- **类Cantor集（Hanack集）**：取 $p=\frac{1+2m}{m}$，类似Cantor集一样移去中间长度为 $\frac{1}{p^n}$ 的开区间，从而移去的总长度为 $\sum\limits^\infty_{n=1} \frac{2^{n-1}}{p^n} = m$，余下的集合 $C_p$ 是完全集，且无处稠密
- E是完全集 $\Leftrightarrow E = (\mathop{\bigcup}\limits_{n\geq 1}(a_n,b_n))^c$
  - 必要性：
  - 充分性
  - **理解**：完全集都像Cantor集一样，是挖去连续集堆砌形成的，只有这个过程才能得到独立于迭代次数的（可以递归比较的）离散点集

### 距离

- 闭集距离 $d(x_0,F)$ 可取到
- 距离函数对于点一致连续
- 函数：$f(x) = \frac{d(x,F_2)}{d(x,F_1)+d(x,F_2)}$
- **连续延拓定理**：闭集上的有界连续函数，可以延拓到整个平面上，依然满足有界性
  - **证明**：把闭集F按照值域限制 $[-M,M]$，均分成三个集合ABC
    - $g_1(x) = \frac{M}{3}·\frac{d(x,B)-d(x,A)}{d(x,B)+d(x,A)}$，则 $g_1(x)、|f(x)-g_1(x)|$ 均有界。
    - 再令 $g_2(x) = f(x)-g_1(x)$，依次构造，级数有界且一致收敛到 $f(x)$
  - **推论**：无界也成立
- $\alef = 2^{\alef_0}$：实数和自然数数列的个数等势（或者用二进制表示实数即可）
- **连续G延拓**：连续函数可延拓到 $G_\delta$ 父集H上
  - $H = \{x\notin E\mid w_f(x)=0\}$（振幅极限列）
  - 由Heine定理，可找到点列极限为 $H-E$ 上的点，由C收敛原理，满足连续延拓
- **L条件延拓**：若一维满足Lipschitz条件，则可延拓到R上，依然满足Lipschitz条件
  - $g(x) = \inf\{f(y)+M|x-y|: y\in E\}$
