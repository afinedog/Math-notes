# 实变的测度论

- 测度论Fatou：$m(\overline{\lim\limits_{n\to \infty}} E_k) \geq \overline{\lim\limits_{n\to \infty}}m(E_k)$，下极限则相反
- 极小真包含/极小真子集：**等测包/等测核**
- **外测度正则性**：任何集合有等测包
  - **证明**：取测度极限列即可
  - **推论**：可测集的测度性质，推广到不可测集上
- **第二纲集的零测集**：$\mathop{\bigcap}\limits^\infty_{k=1}\mathop{\bigcup}\limits^\infty_{n=1} (r_n-\frac{1}{2^{n+k}}, r_n+\frac{1}{2^{n+k}})$
- 正测集 $E\subset [0,1]$ 满足 $0<m(E\cap I)< m(I)$
- **矩形逼近变形**：$\lambda|I| < m(I\cap E)$ 对任意 $\lambda\in [0,1]$ 和 可测集 $E$ 成立
  - （反证 + 矩形分解即可）
- **Steinhaus定理**：可测集E的向量差点集 $\{x-y\}$ 至少包含一个有界邻域
  - $\lambda \in(1-\frac{1}{2^{n+1}},1)$（打通边长和面积）
  - 开矩体 $J=\{(a_1,...,a_n)\mid |a_i|<\frac{\delta}{2}\}$
  - **证明**：即证 $J$ 中点 $x_0$ 满足 $E\cap I$ 和 $(E\cap I) + \{x_0\}$ 相交
    - 已知 $m(I\cap (I+\{x_0\})) > \frac{|I|}{2^n}$（边长）
    - 从而 $m(I\cap (I + x_0))>2\lambda|I|$，从而必定相交

## 连续变换

- 保紧性、零测传递性得保可测性
- 非奇异线性变换：$m(f(E)) = det|f|·m(E)$
  - **证明**：
    - 集合分解成矩形的并集
    - 单位矩形成立，则极小矩形也成立
    - 线性变换分解为：
      - 坐标交换：易得
      - 首坐标数乘：已证
      - 首坐标和次坐标相加：
        - $x_1$ 方向上的长条变换，$y=1-x$ 左边变成左半正方形，右边变成右半正方形
        - 设A为左边，B为右边
        - $m(f(I)) = m(A)+m(B-e_1) = m(I) = 1 = det|f|$
  - **推论**：旋转变换保测度性

### 多边形的测度

- 三角形：旋转得矩形，二分即可
- 正多边形：三角形的旋转并
- 圆盘：求正n边形的测度极限
- 扇形：
  - 角度可能是离散点集，此时满足 $m(S) \leq \frac{(b^2-a^2)m(E)}{2}$
    - 角度E分解成开区间列
  - 角度可测则扇形可测

## 特殊可测集

- 函数传递 $\sigma$ 代数
- 连续函数传递Borel集
- 非Borel集的可测集：
  - C-L函数 $\varPhi(x)$
  - $\Psi(x) = \frac{x+\varPhi(x)}{2}$
  - 严格递增的连续函数
  - $\Psi(C) = H$，则H中的不可测集 $W$ 的原像是C的子集，且可测，且不为Borel集
- Borel集对差不封闭
- 测度空间 $(R,\mathscr{A},\mu)$
  - 依次为定义域、$\sigma$ 代数（集族）、测度定义
  - Borel集不能构成测度空间
  - 存在无等测包/核的空间
- R中Borel集的基数是c：$B(x,\delta)$，其中 $\delta\sim c，x\sim c$，从而 $B\sim c^2=c$，再由可数相并运算，得 $B\sim c$
- 若任意平移集的差集零测，则集合和补集都是零测
- 若与任意区间交集均强非零，则 $m(R-E)=0$
  - 强有界：不仅有界，而且极限上也有界，不存在任意小情况
  - 强有界对应 a.e. 有界（有限值函数）
- 不能积分解的可测集：$E = \{(x,y)\mid x-y\in C\}$
  - C是Cantor集
- 外全密点：$\lim\limits_{h\to 0} \frac{m(E\cap (x-h,x+h))}{2h} = 1$
  - 一维上的非外全密点至多为零测集
