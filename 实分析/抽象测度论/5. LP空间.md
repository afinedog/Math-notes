## 基本性质

- **共轭指数**：$\dfrac{1}{p}+\dfrac{1}{q}$（满足Holder不等式）

### $L^\infty$ 空间

- **本性有界定义**：$L^p(X,\mc M,\mu) = \{f:X\to\C\mid f\in\mc M，\|f\|_p < \infty\}$
  - **等价命题**：$f\in L^\infty\LR $ 存在有界可测函数 $g = f\ae$
    - **证明**：取 $g = f\chi_E$，其中 $E = \{x: |f(x)| \leq \|f\|_\infty\}$ 即可
    - **本质**：取限制，将无界部分（定义得这部分零测）删去，则变为有界函数，且a.e.相等
- **稠密子集**：简单函数集
- **积分上界不等式**：若 $f,g$ 均为可测函数，则 $\|fg\|_1 \leq \|f\|_1\|g\|_\infty$
  - **证明**：<font color='chartreuse'>弄懂了本性上确界的意义后，这些都是显然的了，没啥必要记下来</font>
  - **本质**：就是把 $g$ 放缩为本性上确界后提出积分
  - **推论**：若 $f\in L^1,g\in L^\infty$，则不等式取等号 $\LR$ 在 $\supp{f}$ 上 $|g(x)| = \|g\|_\infty\pad \text{a.e.}$
    - **证明**：定义直得
- **收敛条件**：$\|f_n-f\|_\infty \to 0 \LR$ 存在可测集 $E$ 满足 $\mu(E^c) = 0$，且 $f_n$ 在其上一致收敛
  - **证明**：定义直得
- **一致有界范数**：勒贝格测度和Borel测度中，对任何连续函数 $\|f\|_\infty = \|f\|_u$
  - 有界连续函数是 $L^\infty$ 闭子空间

### $L^p$ 空间关系

- **非包含性**：$L^p\not\subset L^q\pad (\forall p\neq q)$
  - **反例（勒贝格测度）**：设p积分 $f_a = \dfrac{1}{x^a}\pad (a>0)$
    - 易得 $p<\dfrac{1}{a} \LR f_a\chi(0,1)\in L^p$
    - $p>\dfrac{1}{a} \LR f_a\chi(0,\infty)\in L^p$
  - **推论**：若 $0<p<q<\infty$，则
    - $L^p\not\subset L^q \LR X$ 包含任意小的正测度集列
    - $L^q\not\subset L^p \LR X$ 包含任意大的有限测度集列
    - **证明**：考虑 $f = \sum a_n\chi_{E_n}$ 即可
- **直和关系**：若 $0<p<q<r\leq \infty$，则 $L^q \subset L^p\oplus L^r$ （$\forall f_q$ 都可表为 $f_p+f_r$）
  - **证明**：
    - 设 $f\in L^q$，$E = \{|f|>1\}$，$g = f\chi_E，h = f\chi_{E^c}$，定义得 $f = g+h$
    - 易得 $|g|^p = |f|^p\chi_E \leq |f|^q\chi_E$，取积分即得 $g\in L^p$
    - 同理 $|h|^r = |f|^r\chi_{E^c} \leq |f|^q\chi_{E^c}$，取积分即得 $h\in L^r$
    - 若 $r = \infty$，则 $\|h\|_\infty \leq 1$
  - **理解**：以1作为分水岭，因为改变次方后不可积只有两种情况：无穷处降低过慢，原点处增长过快，这两者分别可由高次（$r,E^c$）和低次（$p,E$）来解决
- **共轭关系**：若 $0<p<q<r\leq \infty$，则
  - $L^p\cap L^r \subset L^q$
    - <font color='chartreuse'>该结论易得</font>
  - $\|f\|_q \leq \|f\|^\l_p \|f\|^{1-\l}_{r}$
    - 其中 $\dfrac{1}{q} = \dfrac{\l}{p} + \dfrac{1-\l}{r}$，即 $\l = \dfrac{q^{-1} - r^{-1}}{p^{-1} - r^{-1}}$
  - **证明**：
    - 若 $r = \infty$，由本性上确界定义得 $|f|^q \leq \|f\|^{q-p}_\infty|f|^p$，易得 $\l = \dfrac{p}{q}$
      - 则 $\|f\|_q \leq \|f\|_p^{\dfrac{p}{q}}\|f\|^{1-\dfrac{p}{q}}_\infty$，即得结论
    - 若 $r < \infty$，则分离次方 + Holder不等式直得
- **数列关系**：若 $0<p<q\leq \infty$，则 $l^p(A)\subset l^q(A)$，且 $\|f\|_q \leq \|f\|_p$
  - **证明**：定义 + 共轭关系直得
  - **理解**：数列收敛比
- **有限域内的关系**：若 $\mu(X)<\infty$，$0<p<q\leq\infty$，则
  - $L^p(\mu)\supset L^q(\mu)$
  - $\|f\|_p \leq \|f\|_q\mu(X)^{\dfrac{1}{p}-\dfrac{1}{q}}$
  - **证明**：若 $q = \infty$，上界放缩直得。若 $q<\infty$，分离1应用Holder不等式直得
  - **理解**：若测度空间是有限的，则不会出现无限区间反常积分情况，那么和数列一样

## 对偶性

- **对偶范数**：设 $g\in L^p$，可定义 $L^p$ 空间中线性有界泛函 $\phi_g(f) = \int fg$
  - **算子范数**：上界为 $\|g\|_q$
  - **实例**：$p=2$，则 $L^2$ 是Hilbert空间，此时 $\phi_g(f) = \int f\ol g$
- **共轭定理**：设 $1\leq q<\infty$，$p,q$ 共轭
  - 若 $g\in L^q$，则 $\|g\|_q = \|\phi\|_g = \sup\hkh{|\int fg| : \|f\|_p = 1}$
  - **证明**：
  - **推论**：若 $\mu$ 是半有限测度，则 $q = \infty$ 也成立
- **对偶引理**：
  - 设
    - $p,q$ 共轭
    - $\Sigma$ 是具有有界支集的简单函数全集
    - $g$ 是 $X$ 上可测函数，满足 $\forall f\in \Sigma，fg\in L^{1}$
    - $M_q(g) = \sup\hkh{|\int fg| : f\in \Sigma，\|f\|_p = 1}$
  - 若 $\supp{g}$ 有限 或 $\mu$ 半有限
  - 则 $g\in L^q$，且 $M_q(g) = \|g\|_q$
  - **证明**：
- **对偶定理**：设 $p,q$ 共轭
  - 若
    - $1<p<\infty$
    - $\forall \phi\in (L^p)^*，\exists g\in L^p$ 使得 $\forall f\in L^p，\phi(f) = \int fg$
  - 则 $L^q\cong (L^p)^*$
  - **推论**：此时 $L^p$ 自反
- $L^\infty \to (L^1)^*，g\mapsto \phi_g$ 不是单射或满射
  - **反例**：
    - 若 $\mu$ 非半有限，则非单射
- **限制测度**：
  - 设
    - $X$ 是不可数集，$\mu$ 是 $(X,\mc P(X))$ 的计数测度
    - $\mc M$ 是(余)可数集的 $\sigma$ 有限测度，$\mu_0$ 是在其上的限制
    - $f\in L^1(\mu)$ 的支集最多是可数集
  - 则 $L^1(\mu) = L^1(\mu_0)$
  - **推论（无穷维对偶）**：此时 $L^\infty(\mu)$ 以有界函数组成，$L^\infty(\mu_0)$ 以最多在可数集上不是常函数的函数组成
    - 从而 $L^1(\mu_0)$ 的对偶是 $L^\infty(\mu)$
  - **实例**：$X = [0,1]$，$\mu$ 是勒贝格测度，$f\mapsto f(0)$ 是 $C(X)$ 上有界线性泛函
    - 易得 $C(X)\subset L^\infty$
    - 由H-B定理，存在 $\phi\in (L^\infty)^*$ 满足 $\phi(f) = f(0)$
    - **非原函数性**：设 $f_n\in C(X)，x\mapsto \max(1-nx,0)$，则 $\phi(f_n) \equiv 1$，但 $f_n(x)\to 0$，故由控制收敛定理，$\forall g\in L^1，\int fg\to 0$

## 分布理论

### 分布函数

- **分布函数**：$\l_f:(0,\infty)\to [0,\infty]，\a\mapsto \mu\{x: |f(x)|>\a\}$
  - **本质**：将（数理统计中的分位数）映射成（分位数右方集合的测度）
- **性质**
  - 单增右连续性
  - 下标单调性：$|f|\leq |g|\red\Rt \l_f\leq \l_g$
  - 单增收敛性：$|f_n|\nearrow |f| \red\Rt \l_{f_n}\nearrow \l_f$
  - 下标凸性：$f = g+h \red\Rt \l_f(\a) \leq \l_g(\dfrac{1}{2}\a) + \l_h(\dfrac{1}{2}\a)$
- **L-S转化定理**：若 $\l$ 有界，$\phi$ 是 $(0,\infty)$ 上非负Borel可测函数，
  - 则 $\dis\int_X \phi\circ |f|d\mu = -\int^\infty_0 \phi(\a)d\l_f(\a)$
  - **证明**：定义得 $\nu\Big( (a,b] \Big) = \l_f(b)-\l_f(a) = -\mu\Big( |f|^{-1}\big( (a,b] \big) \Big)$
    - 由Borel集定义，其可扩展到任意 $E\in\mc B_X$ 上
    - 再令 $\phi$ 分别为特征函数、简单函数等即可
  - **理解**：
- **常用形式**：若 $0<p<\infty$，则 $\int |f|^pd\mu = p\int^\infty_0 a^{p-1}\l_f(\a)d\a$
  - **证明**：
    - 若 $\l_f(\a) = \infty$，则两边均无穷
    - 若 $f$ 为简单函数，则 $\l_f$ 在 $\a$ 趋于0时有界，无穷时为0
    - 若 $f$ 为一般函数，取简单函数列 $\{g_n\}\nearrow |f|$，由单调收敛定理 + 单增收敛性即得结论

### 弱 $L^p$ 空间

- **弱积分**：$[f]_p = \Big( \sup\limits_{\a>0} \a^p\l_f(\a) \Big)^{\dfrac{1}{p}}$
  - **非范数性**：只满足正齐次性，不满足三角不等式
- **弱 $L^p$ 空间**：所有弱积分有界的函数全集
  - **拓扑向量空间性**
- **关系**：$L^p \subset L^p_\text{w}，[f]_p \leq \|f\|_p$
  - **证明**：切比雪夫不等式直得
