# 测度论

- **核心思想**：矩形覆盖 + Cauchy收敛（高低阶） + 函数极限的集合形式 + 区间分拆
  - 区间分拆方法有：
    - 补集簇（矩形、球体）：$A_n = A-\mathop{\bigcup}\limits^{n-1}_{i=1}A_i$
      - <font color='chartreuse'>叫它集合的分拆也行，主要是为了好辩认</font>
    - 开区间（不相交分割）
- **整体思路**：用最简单的矩形情况，结合收敛定理，用矩形逼近一般的集合，从而得到一般的结果

## 体积

- **几乎不相交的矩形**：矩形的内部不相交<font color='chartreuse'>（边界可能相交）</font>
- **矩形体积可加性**：
  - 若某矩形被有限个几乎不相交矩形覆盖（$R = \mathop{\cup}\limits^N_{k=1}R_k$）
  - 则总体积等于所有矩形体积的和（$|R| = \sum\limits^N_{k=1}|R_k|$）
  - **证明（面积乘积式重组）**：
    - 由矩形本身的性质，其可以分解成网格形式。
    - 对每个 $R_k$ 再次进行分割，使得每个 $R_j$ 与其相邻矩形的公共边长度相等
    - 每个 $R_j$ 可以看作有限区间的乘积，此时因为公共边长相等，可以**逆用乘法分配律**，得到 $|R| = ab = \sum a_{R_j} \sum b_{R_j} = \sum |R_j|$
    - 同理，$|R_k|$ 可以分解为 $|R_j|$ 的组合（**证毕**）
  - **推论**：
    - 互不相交的开区间测度可加
    - Cantor集外测度为0
      - **证明**：其为不相交的闭集并，故直线上其外测度可加
        - 或外测度开等测包条件，或Caratheodory条件
- **直线上的开集分解定理**：
  - **证明**：同曹广福
  - **理解**：高维空间上不满足纯开集覆盖，所以替代为左开右闭矩形
  - **$R^n$ 上开集可数矩形覆盖性**
    - **本质**：类似数分证明重积分边界为0。即利用Darboux大和与小和两端夹逼，使得边界矩形面积收缩为0

## 外测度

- **外测度**：$m^*(E) = \inf\sum\limits^\infty_{j=1} |Q_j|$
  - **存在性**：由实数的稠密性，任意集合均具有外测度
  - **无限性**：矩形必须是无限个<font color='chartreuse'>（只有无穷才能达到任意逼近的效果）</font>
  - **实例**：
    - 单点集的覆盖为其本身，因此外测度为0
    - $R^n$ 包含所有矩形，易得 $m^*(R^n) = \infty$
- **$E$ 的等测包 $O$**：$E\subset O$，且 $m^*(O\j E) = 0$
- **$E$ 的等测核 $O$**：$O\subset E$，且 $m^*(E\j O) = 0$

### 外测度和体积（部分级数与整体覆盖）

- **矩形测度公式**：$m^*E = |E|$
  - **充分性**：$m^*E \leq |E|$
    - E为闭集：已知E覆盖其本身。由外测度的下确界性直得
    - E为开集：取闭包 + 边界体积为0
  - **必要性**：$m^*E\geq |E|$
    - E为闭集：对闭矩形覆盖中的每个闭矩形 $Q_j$，构造开矩形等测包 $S_j$，其可以有限开覆盖闭矩形 $Q$。由矩形体积可加性得 $|Q| \leq \mathop{\cup}\limits^\infty_{j=1}|S_j|$ ，再由等测包，趋于极限得到闭矩形结果 $Q_j$（**级数Cauchy收敛原理**）
    - E为开集：取闭包 + 边界体积为0
  - **本质**：大矩形可被小矩形的几乎不交并逼近，再讨论下无穷小量，结论就水到渠成了
- **有界闭矩形的外部逼近证明法**：
  - **证明**：易得存在有限开覆盖，对每个开集构造矩形等测包。即得 $|E| \leq \sum |E_i| = m^*E$
  - **具体证明**：同曹广福
- **开矩形的内部逼近证明法**：
  - **证明**：首先选择被开集 $E$ 覆盖的闭集族 $E_0$，闭集族上得到矩形逼近。然后利用闭集族体积无限逼近开集体积（De Morgan公式）的性质，得到开集逼近结果
  - **具体证明**：首先内部矩形 $\sum |Q_内| < |R|$，然后外部矩形的体积等价于其边界面积 $O(k^{d-1})$，而总体积 $|R| = k^d$，因此 $\sum |Q_{内\cup 边}| < |R| + O(k^{d-1})$。k趋于无穷可消去O得到不等式
  - **优势**：利用等价量，即极限的思想解释边界体积为0性

### 外测度性质

- **外测度性质**
  - 单调性
    - **证明**：矩形覆盖法即可
  - 三角不等式/次可加性：
    - **证明**：
      - 无界情况易得
      - 有界情况：构造每个集合的闭矩形等测包覆盖，已知矩形具有三角不等式，可数可加即可
      - **理解**：矩形覆盖 + Cauchy收敛原理
  - 外测度 = 开父集外测度的下界
    - **证明**：
      - 由单调性，$m^*E \leq \inf m^*O$
      - 对任意开父集 $O$，易得存在 $E$ 的可数闭矩形覆盖 $\bigcup Q_j$，其测度和等于O的开矩形等测包 $Q_j^0$
      - 由外测度三角不等式 + 下确界最小性，$\inf m^*O \leq \sum m^*Q_j^0$
      - 每个开矩形化为闭矩形高阶小等测包，Cauchy收敛原理得 $\inf m^*O\leq m^*E$
    - **理解**：（矩形覆盖“总数量”差值）与（等测包“测度大小”差值）相互弥补
  - 不交并测度可加性
    - **证明**：
      - 首先由三角不等式得到正向不等式
      - 然后取闭矩形等测包 $Q_j$，每个矩形的直径小于距离，则该矩形只能和其中一个集合相交
      - 则 两集合测度相加，转化为不相交闭矩形相加（可加），取极限得到结果
    - **理解**：矩形覆盖 + 等测包任意小性
  - 几乎不相交的矩形 $Q_j$ 并集，总测度等于体积之和
    - **证明**：构造 $Q_j$ 闭矩形等测核 $\overline{Q_j}$，变为矩形不相交情况，由不交并测度可加性得结果，其中Cauchy收敛原理用于去掉等测核测度差值的和
    - **理解**：矩形核 + Cauchy收敛原理

## 可测集

- **Lebesgue可测**：存在开等测包（**开父集下极限**）
  - **反例**：Vitali集 $S_n$ 不可测（见曹广福）
    - **理解**：因为其范围确定，测度不定，从而没有确定的开等测包
- **可测集性质（可测集可数运算封闭性）**
  - 开集可测
    - **证明**：矩形逼近 + 对矩形取开等测包
  - 零测集均可测
    - **证明**：取下极限为0的开集列，即为等测包
  - 可数并集可测
    - **证明**：对每个集合取开父集等测包 + Cauchy收敛原理
  - 闭集可测（**有限 + 收敛 + 反向 = 无限 + 反向**）
    - **引理**：不相交的闭集 $F$ 和紧集 $K$，其距离大于0
      - **证明**：
        - 紧集中任取点 $x$，由不相交得可设 $d(x,F)>3\delta_x$
          - <font color='chartreuse'>此时两边都是一个关于x的函数，可能会收敛于0，从而下确界为0，所以必须是</font>
        - 再取紧集中有限闭覆盖 $\bigcup \ol B(x,2\delta_{x_j})$，则由有限性，$\delta_j$ 存在下界 $\delta_{min}$
        - 对 $\forall y\in F,x\in K$，三角不等式得 $|y-x| < |y-x_j|-|x_j-x| < \delta_{min}$，从而得到确定的距离正值 $\d_{min}$
      - **理解**：
        - 闭集：不相交的基本前提，用于构造距离函数
        - 紧集：用有限性导出距离函数有界性。
    - **互包证明**：
      - 取F的开等测包 $O$，在余集中取有限个闭矩形 $Q_j$，定义得其并 $K$ 为紧集，且和 $F$ 不相交，故由引理得距离大于0，从而得到不等式
        - $m^*O \geq m^*F + \sum\limits^N_{j=1}m^*Q_j$（单调性、不相交测度可加性）
        - 因为右侧收敛，左侧为定值，所以N可以推广到极限，此时得到等号
      - 再由矩形覆盖的测度三角不等式，得 $m^*(O-F) \leq \sum\limits^\infty_{j=1}m^*Q_j$，从而 $m^*(O-F) <\varepsilon$
    - **本质**：有限易得反向不等式，再由收敛推广到无穷，得到无穷反向不等式
  - 补集可测性
    - **证明**：
      - 开父集的补集 $O^c$，其开父集为 $E^c$，可测。
      - 因为E可测，所以可以设开父集列 $m^*(O_n-E) \leq \frac{1}{n}$
        - 在此基础上构造补集列 $S = \mathop{\cup}\limits^\infty_{n=1} O^c_n$，由并集可测性，S可测。
        - 若 $E^c-S$ 零测，则其可测。
        - 并集可测性，$E^c$ 可测。可测运算性，得到 $m^*(E^c-S)=0$
    - **理解**：父集列 + 取补集 + 下极限
    - **本质**：通过可测得到测度逼近开父集 $O_n$，然后转化到补集上即可
  - 可数交集可测性
    - **证明**：取每个集合的补集，将它们并起来即可
    - **本质**：不交并可测性的对偶
- **可测集可加定理**：
  - **证明**：
    - 有界情况：<font color='chartreuse'>（其实前面已经证过了，这里只是描述一下）</font>
      - **正向不等式**：不相交集合 $E_j$ 的闭集等测核 $F_j$，具有并集测度可加性。对 $F_j$ 取极限得到 $m^*E \geq \sum\limits^\infty_{i=1} m^*E_j$
      - **反向不等式**：三角不等式
    - 无界情况（**补集簇级数**）：设递增矩形 $Q_k$ 逼近R，$S_k = Q_k-Q_{k-1}$
      - 则 $E = \mathop{\cup}\limits_{j,k}(E_j\cap S_k)$<font color='chartreuse'>（表示遍历 $j$ 和 $k$ 的所有组合）</font>
      - 这些并集不相交，所以 $m(E) = \sum\limits_j\sum\limits_{k} m(E_{j,k})$。组合即可得到 $m(E) = \sum\limits_j m(E_j)$
    - **本质**：“补集簇” + “分析法”（构建补集簇分拆列，从而得到级数，应用可加性）
- **单调可测集合测度连续性（补集簇级数）**：
  - **证明**：
    - 单增（极限为并集）：构造补集簇即可，利用不相交性得到加和形式。然后再合并，并集等于原集
    - 单减（极限为交集）：反向补集簇，$E_1 = E\cup (\mathop{\cup}\limits^\infty_{i=1}S_i)$，交集反向则为并集
- **测度逼近定理**：一个可测集可以被：
  - 开集外逼近
    - **证明**：开父集列（单减）的极限
  - 闭集内逼近
    - **证明**：开父集补集列（单增）的极限
  - 有界时，紧集内逼近
    - **证明**：补集簇级数
      - 选取E的等测包闭子集F，F和原点开球族 $B_n$ 的交集为 $K_n$
      - $E-K_n$ 递减且收敛于 E-F=0
      - $K_n$ 是紧集内逼近列
  - 有界时，有限闭矩形内外均可逼近
    - **证明（放缩 + Cauchy收敛原理）**
      - 构造E的等测包闭矩形覆盖 $Q_j$。
      - 由有界得覆盖的体积收敛（**核心**），有限并为F
      - $m(E\triangle F) = m(E-F)+m(F-E)$
        - 前项由有限并化为 $m(\mathop{\bigcup}\limits^\infty_{j=N+1}Q_j)$
        - 后项直接放缩为 $m(\mathop{\bigcup}\limits^\infty_{j=1} Q_j-E)$
      - 矩形测度可加性 + Cauchy收敛原理

#### 小总结

- 方法
  - 外部开父集逼近、外部矩形覆盖（几乎不相交的闭矩形）
  - 内部闭子集逼近、内部矩形逼近
    - 如果强调不相交（如并集测度可加性），则用内部较好
    - 或者正反向不等式，一个对应外部一个对应内部
  - 高阶无穷小矩形构成零测余集或重叠部分
- Stein这里非常喜欢用矩形逼近和高阶无穷小。对于这种方法，开父集可测定义比可加性可测定义更加好用

#### Cantor性质扩展

- **三进制**：$\forall x\in [0,1]$，都可以表示为 $x = \sum\limits^\infty_{k=1}\cfrac{a_k}{3^k}\quad (a_k=0或2)$
  - **理解**：
    - $a_k$ 是三进制下小数位上的数
    - 第 $k$ 位上的数都是 $3^k$ 等分段的父段（$3^{k-1}$ 等分段）下的序号
  - **证明**：若 $a_k = 1$，则其表示的就是三等分点中的第二段，正好是被消去的段
- **Cantor-Lebesgue函数（指示函数）**：$F(x) = \sum\limits^\infty_{k=1} \cfrac{b_k}{2^k}\quad (x\in\mc C)$
  - 其中若 $x = \sum\limits^\infty_{k=1} \cfrac{a_k}{3^k}$，则 $b_k=\cfrac{a_k}{2}$
  - **理解**：
    - $b_k$ 是用逻辑01来表示三进制中某数位上是否有值
    - $F$ 是将三进制数位换为二进制数位
  - **本质**：已知Cantor集具有连续势，这其实就是给出了到 $\R$ 的双射
  - **存在性（良定义）**：
    - <font color='chartreuse'>通用方法是，证明定义域中等价类的像相同</font>
  - **连续性**：
    - **证明**：
  - **满射性**：$F:\mc C\to [0,1]$ 是满射
  - **可延拓性**：$F$ 可延拓为 $[0,1]$ 上的连续函数
    - **证明**：
      - 延拓性：易得 $\mc C^c$ 是Borel集，故 $\exists\bigcup (a,b) \cup \mc C = [0,1]$
      - 连续性：易得 $\forall (a,b)\subset \mc C^c$，有 $F(a) = F(b)$，故定义 $(a,b)$ 上为常函数，由常函数连续性 + C-L函数连续性即得结论
- **Cantor常数剖分 $\mc C_\xi$**：设 $\xi\in (0,1)$
  - 分 $k$ 步移除当前各集合中心位置的 $k$ 个 $\xi$ 长度开区间
  - $k\to\infty$ 时，剩下的集合就是剖分 $\mc C_\xi$

    - **证明**：可测集运算性质即可
- **类Cantor集 $\wh{\mc C}$**：每次移除 $2^{k-1}$ 个长度为 $\ell_k$ 的开区间，满足 $\sum\limits^\infty_{k=1} 2^{k-1}\ell_k < 1$
  - **可逼近性**：$\forall x\in\wh{\mc C}，\exists \{x_n\}\notin \wh{\mc C}$，满足 $x_n\to x$
    - 且 $x_n\in I_n$，其中 $|I_n|\to 0$ 是补集中的子区间

  - **完美性**：所有点都是聚点，不包含开区间
  - **不可数性**

## 可测函数

- **阶梯函数（连续的集合）**：$f = \sum\limits^N_{k=1} a_k\chi_{R_k}$
  - $R_k$ 是矩形，$a_k$ 是常数
- **简单函数（可测的集合）**：$f = \sum\limits^N_{k=1} a_k\chi_{E_k}$
  - $E_k$ 是可测集
- **可测函数**：只能定义在可测集上
  - **等价命题**：有限值函数，其陪域中Borel集的原像可测
  - **证明**：
    - 由可数运算可测性得到区间开闭和方向任意
    - 由a的任意性得到区间大小任意
    - 可测集可数可加性得性质传递到Borel集上
- **函数有界性**
  - **有限函数（逐点有界）**：$Im\ f = \ol\R\j\{\infty,-\infty\}$
  - **有界函数（一致有界）**：$\exists C\in \R$，使得 $\forall x，f(x)\leq C\pad $
  - **推论**：有界函数 $\subset$ 有限函数
- **连续可测性**
  - **连续函数可测**
    - **证明**：值域连续得到原像集合连续，而开集和闭集可测
  - **复合可测条件**：连续函数 $\varPhi$ 和有限值可测函数 $f$ 的复合可测
    - $(\varPhi\ \omicron\ f)^{-1} = f^{-1}\ \omicron\ \varPhi^{-1}$
    - **证明**：连续函数把连续开区间映射成连续开区间，因此原像是开集 $O$。而可测函数的像若为开集 $O$，则原像可测。
- **极限可测性（函数极限的集合表示）**
  - 函数列 $\{f_n(x)\}^\infty_{n=1}$ 可测 $\LR$ 上下界函数和函数上下极限均可测
    - $\sup\limits_n f_n(x)、\inf\limits_n f_n(x)、\mathop{\lim\sup}\limits_{n\to\infty} f_n(x)、\mathop{\lim\inf}\limits_{n\to\infty} f_n(x)$ 
    - $\{x\mid \sup f_n(x) >a\} \LR \mathop{\bigcup}\limits_n\{f_n(x)>a\}$
    - **证明**：
      - 某区域内，函数上界有最小值约束，等价于 “存在”某个函数满足最小值约束，即并集
      - 同理，函数下界有最小值约束，等价于 “任意”某个函数满足最小值约束，是交集。
      - 由可测可数可加性得到结果
    - 上下极限也不过是取函数列的一部分而已，本质是不变的。
  - 可测函数列的极限函数可测
    - **证明**：只需上下极限相等即可
- **运算可测性**
  - 正整数幂
    - **证明**：一次幂不断传递可测性
  - 有限值的线性
    - **证明**：
      - **加法**：相加后的函数值域单值约束 $a$，可以分解为两个函数分别的值域单值约束 $r$ 和 $a-r$（r是取遍有理数的集合。实数因为不可数，无法保持并集可测性。但因为是区间约束，所以只需要r具有稠密性且遍布实轴即可让区间取遍实轴）
      - **乘积**：平行四边形等式即可 $fg = \frac{1}{4}\big[(f+g)^2+(f-g)^2\big]$
- **几乎处处相等**：不相等的区域测度为0
  - **理解**：函数的可测性和其它测度性质，在 a.e. 符合条件时都可以传递
- **几乎处处收敛**：不收敛的区域测度为0
  - $f \geq \liminf\limits_{n\to \infty}f_n$
    - **理解**：将（可数个不收敛点）视为一个数列。则该数列必定存在下极限

### 简单函数逼近

- $R^n$ 上的非负可测函数 $f(x)$ 可以被一个（非负非降简单函数列）逐点逼近（点态收敛）
- **证明**：
  - **$f(x)$ 无界情况**：
    - 设 $Q_N$ 是以原点为中心的正方体，边长为 $N$
    - 构造函数 $F_N(x) = \begin{cases} f(x)，x\in Q_N 且 f(x)\leq N \\ N\quad，x\in Q_N 且 f(x)>N \\ 0\quad\ \ ，x\notin Q_N\end{cases}$
    - 易得 $N\to\infty，F_N(x)\to f(x)$
  - **值域单元化，划分定义域**
    - 对于给定的 $N,M$，设 $E_{L,M}(L) = \{x\in Q_N\mid \dfrac{L}{M}<F_N(x)\leq \dfrac{L+1}{M}\}$（**核心**）
      - 易得 $0 \leq F_N(x)\leq N \LR 0<L<NM（L\in N）$
      - <font color='chartreuse'>$E_{L,M}$ 是对 $F_N(x)$ 值域的一种划分方式，可以用于构造简单函数</font>
  - **可变简单函数，L是序号，M是单位区间**
    - 设 $F_{N,M}(x) = \sum\limits_{L}\dfrac{L}{M}\chi_{E_{L,M}}(x)$
      - <font color='chartreuse'>它具有简单函数的形式（L取相同的正整数时，值不变。具有值数有限性）
        - 类似Lagrange插值多项式的Krockoneck符号。每当L取一个值时，和中只有一项不为0</font>
      - 若 $x$ 满足 $f(x)\leq N$，则存在L、M使得 $\frac{L}{M}< f(x) \leq \frac{L+1}{M}$
        - 即 $L\in [Mf(x)-1,Mf(x))$，$L$ 只能为 $Mf(x)-1$
      - 若 $x$ 满足 $f(x)>N$
        - 即 $L\in [NM-1,NM)$，$L$ 只能为 $NM-1$
    - 综上， $F_N(x) - F_{N,M}(x) = \begin{cases} f(x)-\frac{L}{M} = \frac{1}{M} & f(x)\leq N\\ N-\frac{NM-1}{M} = \frac{1}{M} & f(x)>N \end{cases}$
  - **将区间极小化**
    - 令 $N=M=2^k$，则上式结果为 $\dfrac{1}{2^k}$
      - 令 $\varphi_k(x) = F_{2^k,2^k}(x)$，则 $\varphi_k(x)$ 是非降简单函数列，且当 $f(x)<N \LR N\to\infty$ 时点态收敛于 $f(x)$
    - 理论上， $N、M$ 为任意无穷大数列时，$F_{N,M}(x)$ 均为所求的简单函数
  - **理解**：
    - 构造分段函数 $F_N(x)$ 的目的是考虑 $f(x)$ 无界的情况。此时无法找到有界的L满足 $L=Mf(x)-1$，即所有无界L均满足限制。
      - 此时即为寻找无穷大简单函数列。
    - L充当序号，M充当划分的单元。它们共同构成了简单函数，其极限也是简单函数的极限。
      - 区间的划分方法才真正把简单函数和原函数联系起来
  - **本质**：给每个点都赋予一个简单函数，累加起来值域构成一个连续集合，从而成为可测函数
  - **应用**：这里其实就已经有了勒贝格积分的思想，也是和数分中Darboux定理最不同的地方。因为可测函数的值域连续，而定义域可能是离散的（不离散就直接Riemann积分即可）。所以对值域进行划分，形成的就是简单函数
  - **推论**：
    - 如果推广到广义实数域上，则 $F_N(x) = f(x)$ 即可
    - 非可测函数不可逼近
- $R^n$ 可测函数 $f(x)$ 可以被一个简单函数列逼近
  - **证明**：
    - 用两个筛选函数构造绝对值函数：$f^+(x)-f^-(x) = |f(x)|$
    - 非负函数 $f^+(x)、f^-(x)$ 对应两个非负单不减简单函数列
    - 令 $\varphi_k(x) = \varphi_k^{(1)}(x) - \varphi_k^{(2)}(x)$，它收敛于 $f(x)$
      - 绝对值单增：$|\varphi_k(x)| = \varphi_k^{(1)}(x) + \varphi_k^{(2)}(x)$。因为当 $f(x)>0$ 时，后者为0，当 $f(x)<0$ 时，前者为0

#### 阶梯函数逼近

- $R^n$ 上的可测函数 $f(x)$，存在一个阶梯函数列 $\{\psi_k(x)\}^\infty_{k=1}$ 点态收敛于 $f(x)$
  - **证明**：测度有限的集合E，已知可以用有限个闭矩形 $Q_j$ 内外逼近。则也可以几乎不相交的矩形 $R_j$ 逼近 ，使得 $m^*(E\triangle \mathop{\cup}\limits^M_{j=1}R_j)\leq \varepsilon$

### Littlewoods三原理

#### 总结

1. 每个可测集都几乎是有限个集合的并
2. 每个可测函数都几乎连续
   - 限制上的连续性：$E\cap O(x_0,\delta)$ 
   - 孤立点必定连续
3. 每个可测收敛函数列都几乎一致收敛

#### 证明

- **闭集逼近定理**
- **Lusin定理**：有限可测集 $E$ 上的有限值可测函数 $f$，存在一个等测核闭子集 $F$，使得 $f\mid_{F}$ 连续
  - **反例（限制连续，但原函数在限制域上不连续）**：Dirichlet函数限制在有理数域上
  - **证明**：
    - 设阶梯函数 $f_n \to f\ae$
    - 由阶梯内部的连续性 + 阶梯交界的连续性定义（**核心**），交界处不连续集的测度可放缩为 $m(E_n)\leq \frac{1}{2^n}$，则总不连续测度为0
    - 由Egorov定理，找到一致收敛等测包闭集 $A$，$\tilde{F}= A-\mathop{\cup}\limits^\infty_{n>N}E_n$
      - <font color='chartreuse'>（通过选取 $N$，使得 $\tilde{F}$ 和 $A$ 只相差一簇零测集）</font>
    - 所有的 $f_n$ 在 $\tilde{F}$ 上连续，且一致收敛，由数分知识得极限函数连续。
    - 再找一个等测包闭子集 $F$ 即可
  - **理解**：先在阶梯函数上 a.e. 连续，然后用一致收敛把连续性导到极限函数上
  - **本质**：由阶梯函数逼近，把不连续点的离散性进一步量化、分析化
- **Egorov定理**：
  - 已知有限可测集 $E$ 上的可测函数列 $\{f_k\}^\infty_{k=1}$
  - 若在 $E$ 上 $f_k \to f  \ae$
  - 则存在闭集 $A$，使得 $m(E-A)<\varepsilon$ 且 $f_k$ 在A上一致收敛
  - **证明**：
    - **函数极限的集合形式**
      - 设收敛点集 $E_k^n = E\{x\in E\mid \forall j>k，|f_j(x)-f(x)|<\frac{1}{n}\}$
      - <font color='chartreuse'>（其为可测函数列的原像集，从而可测）</font>
    - **先定k再定n，累次极限 $\to$ 高低阶**
      - 给定n，则该集合关于k单增（**核心**），因此存在可测极限E（极限可测性）
      - 由单调可测集合测度连续性， $\exist k_n$，使得 $\ m^*(E-E^n_{k_n})< \frac{1}{2^n}$
      - <font color='chartreuse'>（E测度应当有界，否则该步无法完成）</font>
    - **Cauchy收敛原理**，易得存在 $N$ 使得 $\sum\limits^\infty_{n=N}\frac{1}{2^n}<\varepsilon$
      - 交集 $\mathop{\bigcap}\limits_{n>N}E^n_{k}$ 设为 $\tilde{A}$。集合对偶律 + 三角不等式，得 $\\ m(E-\tilde{A})< \sum\limits^\infty_{n=N} m(E-E^n_{k_n}) < \varepsilon$
    - **收敛性**：继续放缩，令 $\frac{1}{n} < \varepsilon$
    - **一致收敛性**：取 $\tilde{A}$ 的等测核闭集 $A$，由 $x$ 任意性即得 $f_k$ 一致收敛性
  - **理解**：只需要一步步把区间放缩成贴近即可
    - 一共放缩三次：先保证收敛 $(\frac{1}{n})$ 的前提下，再保证差集零测（Cauchy收敛 $\frac{1}{2^n}$），最后找到等测包闭子集
    - Cauchy收敛定理的本质就是作差，如果写成 $mA = mE$ 的形式，就不用Cauchy收敛定理，而是直接用函数极限（形式不同而已）
  - **本质**：点态收敛中，不连续奇点的集合极限表示 + 可测集可加性 + Cauchy级数放缩
  - <font color='red'>反例</font>：任意小测度不是零测度（等测包/核就是任意小）
    - $f_n(x) = \begin{cases} 0 & x\in \{0\}\cup [\frac{1}{n},1] \\ 1 & x\in [\frac{1}{n+2},\frac{1}{n+1}) \\ 线性 & x\in (0,\frac{1}{n+2})\cup [\frac{1}{n+1},\frac{1}{n}) \end{cases}$
    - 其分段连续，从而可测，且一致收敛于0
      - （因为不收敛区间本身向0逼近，最终归于0，而不像依测度收敛一样乱窜）
    - 若不收敛集合 $e$ 测度为0，则任何点均为 $E-e$ 的聚点（孤立点集零测）
    - $x=0$ 是收敛集合的聚点
      - 但对 $N\to \infty$，总可以找到 $x_n\in [\frac{1}{N+k+2},\frac{1}{N+k+1})$，使得 $f_{N+k}(x_n) = 1$，与收敛的一致性相矛盾。
  - <font color='red'>反例的反例</font>：一致收敛可数子集列可以逼近得到零测度
    - Egorov得到任意小补开集，其可以分解为可数个开区间，充当子集

## Brunn-Minkowski不等式

- 集合的运算类似戴德金分割
- **（定理5.1）凸集不等式**：$m(A+B)^{\frac{1}{d}} \geq (m(A)^{\frac{1}{d}}+m(B)^{\frac{1}{d}})$
  - 凸集满足线性，则 $d$ 维空间上的凸集满足 $m((1+\lambda)A) = (1+\lambda)^dm(A)$
  - 曹广福习题中，测度满足乘积性：$m(\alpha E) = \alpha^n m(E)$
  - 可测集的和不一定可测
  - **推论**
    - **几乎不相交的矩形**：归纳法
      - *两个矩形*
      - 把A平移，分割成 $A_+、A_-$，再平移B，使得AB正负区测度比例相同。
      - 由于 $A+B\supset (A_+ +  B_+)\cup (A_- + B_-)$，则由测度单调性，再分别对正负矩形做不等式，最后合并得到AB不等式即可
      - *有限个矩形*
    - **几乎不相交的开集**：内部取几乎不相交矩形的等测核
    - **任意紧集**：首先紧集的和也是紧集（把开覆盖也加起来即可），然后取紧集的等测包开父集，开父集的和小于和的开父集（类似闭包，只考虑集合的不同部分即可）
