# 测度论

- 本章讨论的是在 $R^n$ 中的集合。对于一般集合应当使用抽象测度
  - 测度的定义是多样的。可测性的定义也是多样的。
  - 本章使用外测度 + Lebesgue可测性
  - 任何集合都有测度，但是不一定可测。不可测集的测度根据取法不同而不同
- 测度的意思是：一种度量。随着度量的不同，测度也可以分为许多种。
  - 测度在一定意义上定义了集合的大小，按照元素的体积进行偏序比较
  - 势也是在一定意义上定义了集合的大小，按照元素的数量进行偏序比较

## 外测度

- **Lebesgue外测度**：$E$ 是 $R^n$ 中的点集，$\{I_n\}^\infty_{n=1}$ 是一列开长方体，$\mathop{\cup}\limits_{n=1}^\infty I_n \supset E$，则 $inf\sum\limits^\infty_{n=1}|I_n|$ 确定一个非负的数u（或确定正无穷）
  - $m^*E = inf\{u\mid u=\sum\limits^\infty_{n=1}|I_n|,\mathop{\cup}\limits_{n=1}^\infty I_n \supset E，I_n是开长方体\}$
  - 若E无界，则u是正无穷
- **理解**：
  - 为了得到最小性，首先要满足“非重复性”，然后要满足“包裹性”，即无论集合是什么形状，都应该完全包住。如果是点集，就应该是点覆盖。如果是无限点集，就是无限点覆盖。
  - 为什么叫外测度，因为它是最小的覆盖，相当于Darboux和的最小上界Riemann和。
  - 对于点集，其并没有内测度（点不覆盖任何几何图形），所以只能定义外测度
- **外测度性质**
  - 平移不变性、非负性、单调性（$A\subset B \Leftrightarrow m^*A\leq m^*B$）
  - **三角不等式**：$m^*(\mathop{\cup}\limits^\infty_{n=1}A_n) \leq \sum\limits^\infty_{n=1}m^*A_n$
    - **证明（矩形覆盖？学stein还不学到底）**：
      - $\sum\limits^\infty_{n=1}m^*A_n = \sum\limits^\infty_{n=1}\sum\limits^\infty_{k=1}|I_{n_k}| \geq |\mathop{\cup}\limits^\infty_{n=1}\mathop{\cup}\limits^\infty_{k=1}I_{n_k}| = m^*(\mathop{\cup}\limits^\infty_{n=1}A_n)$
      - 或利用补集簇的互不相交性 和 补集簇 $\leq$ 全集
- **外测度是体积的超集**：
  - 数学归纳法（对矩形数量归纳） + 取交集法（用于合并图形），配凑得到不等式 $\sum\limits^n_{i=1}|I_i| \geq |\bar{I}|$ （有限开矩形覆盖的外测度 > 被覆盖闭矩形的外测度）
  - 有限覆盖定理，拓展到任意开覆盖
  - $m^*(\bar{I}-I) = 0$，从而闭矩形 = 开矩形

### 可测集可加性（后面还要用Carathedory条件再证一遍）

- 可加性 $\Rightarrow$ 可数可加性
  - **例子**：正项级数、可测集
  - 证明方法：若已知 $\forall N，m^*(\mathop{\cup}\limits^\infty_{n=0}E_i) \geq \sum\limits^N_{n=0} m^*|E_i|$，因为可以比较大小，所以两边都是有界量（级数收敛）。所以可以直接把右式的N变为无穷，依然是有界量，依然满足大小关系。

#### 经典不可测集

- **Vitali集**：区间上的有理分拆集
  - $x\in(0,1)，R_x = \{\xi\mid \xi\in(0,1),\xi-x是有理数\}$（依赖于x而变化）
  - **$R_x$ 非空性**：$x\in R_x$
  - **$R_x$ 不相交性**：$R_x\cap R_y \neq \varnothing \longrightarrow R_x = R_y$
  - **分拆单点集 $S$**：$(0,1)$ 可以分解成 $\{R_x\}_{x\in(0,1)}$ 的分拆，每个分拆中取一点，构成 $S$
    - 选择公理（超穷归纳法）
  - **分拆单点的平移 $S_n$**：设 $(-1,1)$ 上有理数全体构成 $\{Q_i\}^\infty_{i=1}$，则 $S_n = \{x+Q_n\mid x\in S\}$ 是不可测集
  - **证明**：
    - **有界性**：
      - **上界（平移有界性）**：
        - 平移距离有界，$\mathop{\cup}\limits^\infty_{n=1}S_n \subset (-1,2)$
      - **下界（平移完备性）**：若 $n\neq m，则 S_n\cap S_m = \varnothing$（$x_n-x_m$ 若为有理数，则两集合相等）
        - **平移集和单点集的一一对应性**：$\forall x\in (0,1)$， 设 $S\cap R_x = \tau$，则总存在 $Q_n = x-\tau$，使 $x\in S_n$
        - 由一一对应性，n和x是绑定的，$R_x$ 和 $S_n$ 是绑定的，都是 $\overline{\overline{R}}$，由分拆得 $(0,1)\subset \mathop{\cup}\limits^\infty_{n=1}S_n$
    - **收敛性**
      - 反设外测度可加，则 $1 = m^*(0,1)\leq m^*(\mathop{\cup}\limits^\infty_{n=1}S_n) \leq m^*(-1,2) = 3$
      - 但是又由平移性 $m^*S_n = m^*S$，即每个 $S_n$ 测度相等
        - 再由互不相交性得  $m^*(\mathop{\cup}\limits^\infty_{n=1}S_n) = \sum\limits^\infty_{n=1}m^*S_n$
        - 由级数收敛，得到 $m^*S_n = 0$。但无穷个0不能可加（$0·\infty$ 无意义），矛盾。
  - **理解**：$R_x$ 是基于 $Q$ 定义的不可数集（$Q$ 的超集）。且$Q$ 个 $R_x$ 相并包含了 $(0,1)$（本例中体现为 $\mathop{\cup}\limits^\infty_{n=1}S_n$）
    - 离散性使其单点测度为0，有界性使其整体测度不为0，不满足可加性
  - **本质**：$R_x$ 的互不相交性 使得 $S_n$ 有非零下界。平移使得 $S_n$ 传递了互不相交性和有上界性，同时增加了级数项相等性
    - 数分中已经提到，无穷多个无穷小量的和不一定是0，而测度论其实就是定量研究这个东西的。
  - **推论**：
    - 下确界为0：代表元全部选在 $[0,\varepsilon]$ 内
    - 上确界为1：代表元取遍非零测 $F_\sigma$ 集

#### 其它不可测集

- 圆周上的角度分拆集
  - 单位圆周上一点 $M$，选取 $A_1$ 使得夹角为 $\theta_1$，第二点夹角为 $\theta_1+\alpha\pi$（$\alpha$ 是无理数），可以导出不重合点列 $\{\theta_{1}\}$
  - 挖去 $\{\theta_1\}$ 的点，继续构造 $\{\theta_2\}$（良序原理，或选择公理），最终得到的数列集合（分拆） $\{\theta\}$ 的势为 $\alef$
  - 再从这些数列中选取第一个点，构造集合 $\varPhi_1$（代表元），从而 $\{\varPhi\}$ 的元素的势为 $\alef$，集族势为 $\alef_0$
  - 若 $\varPhi_n$ 可测，满足测度可加性，而每个 $\varPhi_n$ 的元素个数相等，从而测度相等。但其总和为单位圆周，测度为1
    - 若每个 $\varPhi_n$ 的测度为0，则总测度为0
    - 若每个 $\varPhi_n$ 的测度为1，则总测度无界
    - 总之测度之和不可能为1，这就产生了矛盾，$\varPhi_n$ 不可测。

### 勒贝格不可测性

- 代表元的选取：
  - 每个不可测集，都离不开在某个父集（连续集比较方便，Borel集也不差）中构造分拆，然后在分拆中选取代表元。
    - 这个代表元的选取依赖于选择公理。不可测集的测度和选取的方式有关。
- 无可加性
  - 这些代表元的测度相等，但是有可数个，且总测度被父集约束为有界，从而产生矛盾
    - $S_n$ 的平移性，$\varPhi_n$ 的数列序号，都是不同的代表元选取方法，同时也导出了代表元测度相等性

## 内测度

- **补集定义**：
  - **开集的测度**：包含E的开集测度，减去E的补集的测度
    - $m_*E = (b-a)-m^*(\complement_{(a,b)}E)$
  - **闭集的测度**：包含E的闭集测度，减去E的补集的测度
- **闭覆盖定义**：包含在E中的闭集 $\bar{I}_n$ 测度的上确界
- 由刚才外测度中所述，点集的内测度不好处理（一般意义讲，其没有闭覆盖，除非重新定义）

## Lebesgue可测集

- **限制定义（交集版）**：对任意 $T\subset R^n$
  - 若满足
    - **Caratheodory条件**：$m^*T = m^*(T\cap E) + m^*(T\cap\complement E)$
  - 则有
    - E为**Lebesgue可测集**
    - $m^*E$ 为**Lebesgue测度**
  - **证明**：（未完）
  - **本质**：选择 $T\supset E$ 时，可以发现 $E$ 的内外测度相等
  - **推广**：从 开长方体 $I$ 推广到 任意集合 $T$
    - **核心**：构造开长方体序列 $\mathop{\cup}\limits^\infty_{n=1}I_n\supset T$（包含），$\sum\limits^\infty_{n=1}|I_n| < m^*T+\varepsilon$（极小无重叠真包含）
    - **形式证明**：$m^*(T\cap E)+m^*(T\cap \complement E) \leq m^*(I_n) < m^*T + \varepsilon$
      - **证明**：左式化为加和形式后合并，可以变形成 $\sum\limits^\infty_{n=1}|I_n|$
    - $m^*T \leq m^*(T\cap E)+m^*(T\cap\complement E)$，使用三角不等式即可
- **限制定义（并集版）**：对 $A\subset E，B\subset E^c$
  - 只需满足 $m^*(A\cup B) = m^*A + m^*B$
  - **证明**：设 $T=A\cup B$
    - $T\cap E = (A\cup B)\cap E = A$
    - $T\cap E^c = B$
    - 按照交集版本即可证明
- 重点是，T是任意的，和E并没有任何关系

### Lebesgue可测集的性质

- **计算公式**：勒贝格可测集上的勒贝格外测度
- **补集可测性**：交集定义的对偶性直得
- **测度为0则可测**：交集 < 原集 、 交集三角不等式
  - 测度为0，补集不一定是全集，所以还是需要互包证明
- **$\varnothing、R^n$ 可测性**
- **子集不一定可测性**
- **运算可测性**
  - $S_1\cup S_2$：
    - 分割成四个集合，T1补集，T2-S1，T3-S2，T4交集
    - 取 $t_1 = (T_1\cup T_2)\cup (T_3\cup T_4)$，利用 $S_2$ 可测（也即直接取$t_1 = T$）
    - 取 $t_2 = T_1\cup T_2$，利用 $S_1$ 可测
    - 取 $t_3 = T_2\cup (T_3\cup T_4)$，利用 $S_2$ 可测
    - 最终利用 $t_3 = T\cap(S_1\cup S_2)$，$t_1 = T_1\cup t_3$得到并集可测性
  - $S_1\cap S_2 = (S^c_1\cup S^c_2)^c$
    - $S_1、S_2$ 可测，其补集均可测。并集可测，则并集的补集可测（√）
  - $S_1-S_2 = S_1\cap S^c_2$
- **有限直和性**：互不相交的可测集 $m(\mathop{\cup}\limits^n_{i=1}S_i) = \sum\limits^n_{i=1}mS_i$
  - **证明**：取 $T = \mathop{\cup}\limits^n_{i=1}S_i$，归纳得到其可测性。再取 $E = \mathop{\cup}\limits^{n-1}_{i=1}S_i$，利用可测集定义一个个分离即可
  - （其实互不相交的条件并不强，完全可以构造补集簇来达到这一点，因此该结论具有一般性）
- **可数直和性**：互不相交的可测集 $m(\mathop{\cup}\limits^\infty_{i=1}S_i) = \sum\limits^\infty_{i=1}mS_i$
  - **证明**：如果要推广到无穷的话，还是从头得用定义证明
    - 已知 $m^*T = m^*[T\cap(\mathop{\cup}\limits^n_{i=1}S_i)] + m^*[T\cap (\mathop{\cup}\limits^n_{i=1}S_i^c)]$
    - 然后一个个分离，归纳得到 $m^*[T\cap(\mathop{\cup}\limits^n_{i=1}S_i)] = \sum\limits^n_{i=1} m^*(T\cap S_i)$
    - 右项n趋于无穷，得到不等关系
      - 即 $m^*T \geq \sum\limits^n_{i=1}m^*(T\cap S_i) + m^*\big[T\cap(\mathop{\cup}\limits^\infty_{i=1}S_i)^c\big]$
    - 左项n趋于无穷，得到不等式的无穷形式。
    - 再由三角不等式，不等号互包得到 $\mathop{\cup}\limits^\infty_{i=1}S_i$ 是可测集
    - 再取 $T = E = \mathop{\cup}\limits^\infty_{i=1}S_i$，利用上述不等式互包，即可得到结论（**证毕**）
  - **推论**：可数集的测度为0（单点集的测度为0）
- **无穷交性**：若 $S_i$ 可测，则 $\mathop{\cap}\limits^\infty_{i=1} S_i$ 可测
  - **证明**：$(\mathop{\cap}\limits^\infty_{i=1}S_i)^c = \mathop{\cup}\limits^\infty_{i=1}S^c_i$
- $R^n$ 中所有可测子集构成 $\sigma$ 域
  - 可测的运算封闭性
- **可测集列的极限（上升）**：$E_i$ 是单调上升的可测集列，则 $\lim\limits_{n\to \infty}E_i = \mathop{\cup}\limits^\infty_{i=1}E_i$ 可测，且 $m^*(\lim\limits_{i\to\infty}E_i) = \lim\limits_{i\to\infty}m^*E_i$
  - **证明（补集簇 + 裂项）**：
    - 若右式 $= +\infty$ ，则由无穷直和性，得到左式 $= +\infty$
    - 若右式不为无穷，则构造补集簇 $\{S_i\}$，互不相交且等价于 $\{E_i\}$
      - $m\lim = m(\cup S) = \lim \sum mS_i = \lim [\sum m(E_i-E_{i-1}) + mE_1]$
      - 可测并集定义，得到 $m(E_i-E_{i-1}) = mE_i - mE_{i-1}$
      - 从而上式结果为 $\lim\limits_{n\to\infty} mE_n$
  - **本质**：互不相交性导出可加性
- **可测集列的极限（下降）**：增添条件 $\exist n_0$，使得 $mE_{n_0} < +\infty$（防止全部为无穷测度）
  - **证明（$E_{n_0}$ 补集单增性）**：
    - 令 $S_n = E_{n_0}-E_n（n>n_0）$，则其单增且可测
    - $\lim\limits_{n\to \infty} S_n = E_{n_0}-\mathop{\cap}\limits^\infty_{n=n_0+1}E_n$，由测度可加性 + 单增列测度极限
    - $m(\mathop{\cap}\limits^\infty_{n=n_0+1}E_n) = m(\mathop{\cap}\limits^\infty_{n=1}E_n) = \lim\limits_{n\to\infty}mE_n$
  - **本质**：单增列的对偶
- **极限可测性**：
  - 可测集列 $E_n$ 若存在极限，则极限可测
    - **证明**：
      - 根据无穷交并性 + 上下极限定义，上下极限可测
      - 上下极限都可测，则极限可测
- **测度连续性**：存在极限的集列 $E_n$
  - 若 $\exist K_0$，使得 $m(\mathop{\cup}\limits_{n\geq K_0} E_n) < +\infty$，则 $m(\lim\limits_{n\to \infty}E_n) = \lim\limits_{n\to \infty}mE_n$
  - **证明**：
    - 令 $S_n = \mathop{\cup}\limits_{k\geq n} E_k$，则 $S_n$ 单减， $\lim\limits_{n\to \infty}mS_n = m(\overline{\lim\limits_{n\to\infty}}E_n) = m(\lim\limits_{n\to\infty}E_n)$
      - 再因为 $E_n\subset S_n$，所以 $\overline{\lim\limits_{n\to\infty}}mE_n \leq \lim\limits_{n\to\infty}mS_n = m(\lim\limits_{n\to \infty}E_n)$
    - 再设 $F_n = \mathop{\cap}\limits_{k\geq n} E_k$，……
      - 同理得 $\underline{\lim\limits_{n\to\infty}}mE_n \geq m(\lim\limits_{n\to \infty}E_n)$
    - 再由上下确界大小关系，得到它们相等
  - **推论**：

## Lebesgue可测集的结构

- 可测集，无论是可加性定义，还是覆盖性定义，都是强性质。
- 可加性是约束了其离散点的位置/结构
- 覆盖性是约束了其离散点的数量/结构

### 开集可测性

- **矩形的测度**：$R_n$ 中任意开长方体可测，且 $mI = |I|$
  - **证明**：交集定义 + “分”析法（所有图形都是可测的，只要配凑式子即可）
- **左开右闭长方体**：$G = \{(x_1,...,x_n)\in R^n\mid c_i<x_i\leq d_i，i=1,...,n\}$
- **开集可数矩形覆盖引理**：$R^n$ 中的非空开集G，可以表示成最多可数个互不相交的左开右闭的长方体之并
  - **证明（不就是Stein中的网格证明……非得写这么麻烦，不用符号不行吗？）**：
    - 构造可数个长方体 $B_k = \{(x_1,...,x_n)\mid \frac{m_i}{2^k}< x_i \leq \frac{m_i+1}{2^k}，i=1,...,n\}（k\in N^+）$
      - k是集合序号（可数个），i是坐标序号（有限个），$\{all\ m_i\}\sim R$
      - 对于每一个k，$R^n$ 都可以被 $B_k$ 完全覆盖
      - 可数是指m可数
      - $B_k$ 相对于k是递减的（体积是 $2^{nk}$ 倍关系）
    - **空隙填充**：$I^{(k)}_j$ 是完全被G包含，但和 $I^{(l<k)}_j$ 互不相交的长方体（它们用来填充G和 $\cup I^{l}_j$ 之间的空隙）
      - $j\sim m_i$ 是有限个或可数个
    - G是开集，所以其内点x必有邻域 $O(x,\delta)\subset G$
      - $k\to \infty，|B_k|\to 0$，则存在 $x\in B_k \subset O(x,\delta)$，即 $x\in I_i^{(k)}$
      - 由x任意性，得到 $G = \mathop{\cup}\limits_{i,k}I^{(k)}_i$
  - **理解**：高维空间上没有开集分解性，只能用半开半闭矩形覆盖。
    - 类似数分中二维反常积分可积等价于绝对可积。其可以构造一个正函数通道来体现极端情况，且不失积分区域连通性。但一维上没有通道一说，所以没有这个性质。
    - 在这里，一维若出现多个不相接的开集，则可以直接用各个不相接的开集本身充当覆盖。但二维不行，二维的各个不相接开集 $G_i$ 可以是形状不规则的，仅用矩形是不能充当 $G_i$ 本身的。（用Stein中的网格的极限小形式可以给出直观解释）
    - 而连续的开集不可以用不相交的小开集来覆盖，即使在一维上也不行。所以二维只能用半开半闭矩形进行覆盖（称为几乎不相交的矩形）
- **可测定理**：$R^n$ 中的开集、闭集、$F_\sigma$ 型集、$G_\sigma$ 型集均为可测集
  - 开集可用可数矩形覆盖，矩形可测，可数直和性得到开集的Lebesgue测度具有可加性（可测性）
  - 闭集是开集的补集，因此可测
  - $F_\sigma$ 是可数个闭集的并、$G_\sigma$ 是可数个开集的交（它们不一定是开集或闭集）。由运算可测性得到可测
  - **Borel集类**：所有开集经过交并差运算后得到的 $\sigma$ 域，称为 $\mathscr{B}(R^n)$
  - **推论**：任意Borel集类都可测

### 可测集的结构

- 直线上Borel集的全体的势为 $\alef$
  - **证明**：直线上的可数开集覆盖定理（所有开集为实数代表元，势为 $\alef$） + $\alef_0·\alef = \alef$
- 全体Lebesgue可测集的势为 $2^\alef$
  - **证明（互包）**：
    - 首先，其势不大于 $2^\alef$，因为 $(0,1)$ 上全体子集的势是这个，而R上 $(0,1)$ 这种区间可数，所以R上全体子集的势也是这个。（任何R上集合的势不可能超过这个？）
    - 然后，其势不小于 $2^\alef$，因为已知存在势为 $\alef$ 的可测集（Cantor集），其所有的子集也可测。
- **可测集和其开覆盖的差集为零测集**
  - **证明**：对可测集，定义极小真包含开矩形覆盖 $\{I_i\}^\infty_{i=1}$。$\mathop{\cup}\limits^\infty_{i=1}I_i-E$ 是零测集，可测。所以其和E符合可加性。分解成加和形式后，由于极小真包含性，可以得到差的零测性。
  - 和Stein的定义是反过来的吗……
  - **推论**：
    - 每个可测集都存在一个开父集，使得差集为零测集（即Stein对可测集的定义）
      - 开覆盖的下确界是一个 $G_\sigma$ 型集（这就是开父集）
    - 可测集也可以内部逼近，差集为零测集
      - **证明**：
        - 若E无界，则构造单增无界开矩形覆盖，使得交集 $E_n$ 为单增有界可测集列。从而满足单增可测集的极限连续性，得到 $mE = \lim\limits_{n\to \infty}mE_n $
          - 假设可以找到极小真包含子集（Borel集） $F_n\subset E_n$，则 $mE = \overline{\lim\limits_{n\to\infty}}mF_n\leq mF\leq mE$（前面是单增函数极限性质，后面是测度单调性），得到 $mE=mF$（本质上和Stein五花八门的逼近相同）
          - 最后得到 $m(E-\mathop{\cup}\limits^\infty_{n=1}F_n) \leq \sum\limits^\infty_{n=1}m(E_n-F_n) = 0$
        - 若E有界，则利用补集的开父集即可证明（同stein）
    - 不可测集也可以找到开父集，但差集的外测度不为0，否则两个可测的并也可测
