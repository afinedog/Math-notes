## 域扩张

- **域 $K$ 的代数**：设域 $K$ 是环 $R$ 的子环，则 $R$ 是K域(作用)代数
  - **环代数性**：其也是环意义下的代数（易得模性、数乘交换性）
  - **进化性**：整环 + 有限的域(作用)代数 = 域
    - **证明**：只需可逆性，定义易得
    - 有限扩张的子环是域
- **域 $K$ 的扩张**：设域 $K$ 是域 $F$ 的子域，则 $F$ 是K域扩张
  - **向量性**：$F$ 是 $K$ 作用的向量空间，维数写为 $[F:K]$<font color='chartreuse'>（可以视为加法群的陪集指数）
  - 基中的 $1_K$ 就可以表出全部的 $K$，其它基元素都是用来表出 $F-K$ 的</font>
    - **实例**：
      - **有限扩张**（维数有限的扩张）：$\C$ 是 $\R$ 的2维有限扩张
      - **无限扩张**：$\R$ 是 $\Q$ 的无限扩张
        - **证明**：$\Q$ 可数，故其有限扩张均是可数集的有限直积，当然也可数
      - $\forall X\subset K，K(X),K[X]$ 都是无限扩张
        - **证明**：基为 $1,X,X^2,...$，从而无穷
- **（定理1.12）中间域维数公式**：$[F:K] = [F:E][E:K]$
  - **证明**：模的维数公式直得

- **X生成的子域**：$F$ 上所有包含 $X$ 的域的交
  - **X生成的子环**：$F$ 上所有包含 $X$ 的环的交
- **X在K上生成的子域**：设域 $F\geq K，X\subset F$，则 $K\cup X$ 生成的子域是 $K(X)$
  - **X在K上生成的子环**：设域 $F\geq K，X\subset F$，则 $K\cup X$ 生成的子环是 $K[X]$
    - <font color='chartreuse'>易得其为整环</font>
- **有限生成扩张**：$X= \{u_1,...,u_n\}$，则 $F$ 上的子域 $K(u_1,...,u_n)$ 是 $K$ 的有限生成扩张
  - **单扩张**：$X = \{u\}$，则 $K(u)$ 是单扩张
- **（定理1.3）生成环【域】的多项式性**：若域 $F$ 是域 $K$ 的扩张，$u,u_i\in F$，$X\subset F$，则<a id = "生成域的多项式性"></a>
  - 子环 $K[u]$ 由所有 $K$ 上多项式的像 $f(u)$ 组成<font color='chartreuse'>（即 $f\in K[x]$）</font>
  - 子环 $K[u_1,...,u_m]$ 由所有 $K$ 上m元多项式组成<font color='chartreuse'>（即 $g\in K[x_1,...,x_m]$）</font>
  - 子环 $K[X]$ 由所有 $K$ 上 $X$ 中元素的所有多项式组成<font color='chartreuse'>（即 $h\in K[x_1,...,x_n]$）</font>
  <br>
  - 子域 $K(u)$ 由所有 $K$ 上单项式的分数组成 <font color='chartreuse'>即（$f(u)/g(u) = f(u)g(u)^{-1}$）</font>
  - 子域 $K(u_1,...,u_m)$ 由所有 $K$ 上多项式的分数组成
  - **证明**：仅以最后一个举例
    - 易得每个包含 $X,K$ 的域都包含多项式的分式集 $E$，即 $K(X)\supset E$
    - 易得多项式的分式集 $E$ 是加法群和乘法群。再由可逆性和交换性得是域，再由 $K,X\subset E$ 得 $K(X)\subset E$
  - **理解**：<font color='red'>扩张的向量空间理解，就是把原域 $K$ 中的元素看作常数，基为 $1_K$，把新元素 $u\in X$ 看作基 $\{u,u^2,...\}$。基通过四则运算来生成所有新元素，最终可写成有理分式的形式
    - 域中的多项式 $f(u)\in K[u]$ 相当于基的线性组合结果，$f\in K[x]$ 相当于基的线性组合方式</font>
  - **推论（有限生成集覆盖性）**：$\forall v\in K(X)$ 或 $K[X]$，存在有限集 $X'\subset X$ 使得 $v\in K(X')$ 或 $K[X']$
    - **证明**：由于 $v$ 的次数有限，显然其系数集是有限集，从而也是有限生成集
  - **推论（生成域的复合公式）**：$K(X,x) = K(X)(x)$
    - **证明**：左边是 $\sum\limits^n_{k=1} a_{k_1k_2}X^{k_1}x^{k_2}$ 相除，右边是 $\sum\limits^m_{i=1}(\sum\limits^n_{j=1} a_jX^j)x^i$ 相除<font color='chartreuse'>（$m,n\in\N^+$）</font>，显然它们展开后相等，故相除得到的分式也相等
- **子域合成**：设 $L,M$ 是 $F$ 的子域，则 $LM$ 是 $L\cup M$ 生成的子域<font color='chartreuse'>（若生成集 $X$ 是域，则生成关系就是相乘关系）</font>
  - **合成公式**：$LM = L(M) = M(L)$
    - **证明**：
      - $L(M) = \hkh{\cfrac{\sum\limits^n_{i=0} l_{i_1,...,i_m} m_1^im_2^i...m_m^i}{\sum\limits^n_{i=0} l_{i_1,...,i_m} m_1^im_2^i...m_m^i}\biggm| l_{i_1,...,i_m}\in L，m_{j}\in M}$，由域的加乘除幂的封闭性，可得 $L(M)\subset LM$
      - $LM = \set{lm\mid l\in L，m\in M}$，显然 $L(M)$ 中取单项式即可表出 $lm$，从而 $LM\subset L(M)$
      - 另一个同理
    - **本质**：由于域的良好封闭性，多个元素线性组合（右）等价于两个元素相乘（左）
    - **推论**：若 $K\leq L\cap K(S)，S\subset K(S)$，则 $L\cdot K(S) = L(S)$
      - **证明**：本质是 $LKS = LS$
      - **实例**：$K(u,v) = K(u)(v) = K(u)K(v)$
  

### 干域（系数域的代数单扩张）

- **干域**：设 $X\subset F\geq K$，$f\in K[X]$ 是最小的首一既约多项式，$\a\in F$<a id="干域"></a>
  - 若 $F = K[\a]$，$f(\a) = 0$，则 $(F,\a)$ 称为 $f$ 的干域<font color='chartreuse'>（实际上就是 $K[x]$ 的系数域 $K$ 的代数单扩张 $K(\a)$，它实际上是将 $K[x]$ 的元素 $f$ 商等价为0，以此来得到可逆性）</font>
- **标准干域**：设 $f\in K[X]$ 是 $m$ 阶首一既约多项式，$(f)$ 是 $f(X)$ 生成的理想
  - 其干域中取 $F = K[X]/(f)，\a = X+(f)$ 时，称为标准干域
  - **证明**：
    - **$\a$ 是根**：$f(\a) \in (f)$，从而在 $F$ 中等价于0
    - **商环**：此时 $\a = X+(f) \in K[X]/(f)$
    - **商映射**：$\p:K[X]\to K[\a]，f_0(X)\mapsto f_0(\a)$，其中 $\ker\p = f(X)$
      - **证明**：
        - $K[\a] \LR f_0(\a) = \sum\limits^{m-1}_{i=0} a_i\a^i =  \sum\limits^{m-1}_{i=0} a_i\Big(X+(f)\Big)^i$
        - 由于 $nX^k(f) = nX^k\Big[ K[X]中元素乘上(\sum\limits^{m-1}_{i=0} b_iX^i) \Big] \in (f)$
          - <font color='chartreuse'>也就是说，二项式的中间项均等于 $(f)$</font>
        - 故 $原式 = \sum\limits^{m-1}_{i=1} a_iX^i + (f) = f_0(X) + (f)  \LR K[X]/(f)$，从而是商映射
          - <font color='chartreuse'>也就是说，当 $f(\a) = 0$ 时，$K[\a] = K[X]/(f)$</font>
        - 由 $\p$ 定义，$\p(f(X)) = f(\a)$。再由 $\a$ 定义即得结论
    - **可逆性**：
      - **证明**：设 $f(X)$ 既约
        - 由裴蜀等式，$\forall g\in K[X]$，都有 $a(X)f(X) + b(X)g(X) = d(X)$
        - 由既约性，$d(X) \equiv 1$
        - 再由 $f(\a) = 0$，取 $X = \a$ 即得 $b(\a) = g(\a)^{-1}$
        - 再由 $g$ 任意性，即得 $K[\a] = K[X]/(f)$ 可逆
    - 则此时 $\Big(K[X]/(f), \a\Big)$ 是域
  - **理解**：代数元素生成的环，由裴蜀等式可得可逆性
  - **本质**：将任意多项式环化为多项式域（干域），只需要将一些多项式视为等价，那么幺元 $1$ 就等价于无穷个任意次数的多项式，在此基础上，对任意多项式 $g$，总有一个适合它的 $(f)+1 = gh$，从而可构建出等价逆 $h$
     - 我们所做的只是规定某个既约的 $f$ 为0，它等价于用其根 $\a$ 代替自由未知数 $x$ 来组成多项式环（根的约束条件被转化为“相差 $f$ 的多项式等价”）。
    - 在后面的[代数扩张结构公式](#结构的性质)中，还会进行详细说明和总结
  - **推论**：
    - **维度性质**：若 $f(X)$ 是 $m$ 阶首一既约多项式，则 $K(\a) = K[X]/(f)$ 是 $m$ 维 $F$ 域扩张
      - **理解**：已知多项式均可被某个既约的 $f$ 用带余除法削至低于 $f$ 次数
        - 在干域中，我们规定某个既约的 $f$ 为0，则在带余除法过程中始终保持等价关系，最终每个多项式都等价于次数比 $f$ 小的多项式
        - 从而此时用 $m$ 个单项式 $\set{1_K,\a,...,\a^{m-1}}$ 线性组合，即可表出所有等价多项式
    - **标准唯一性**：任意干域均 $K$ 同构于标准干域
      - **证明**：K同构映射为 $\sigma:K[X]/(f)\to K[\a]，f_0(X) + (f) \mapsto f_0(\a)$，<font color='chartreuse'>实际上它就是 $\sigma:K[X]\to K[\a]，g\mapsto g(\a)$ 在商域上的限制</font>
        - **单射性**：反设不是单射，则存在 $f_1(\a) = f_2(\a)$，但 $f_1\neq f_2 + (f)$
          - 此时只能是 $f_1$ 和 $f_2$ 相差一个以 $\a$ 为根的多项式 $g$。但由 $f$ 最小性，此时 $f\mid g$，故 $g\in (f)$，矛盾
        - **满射性**：易得
  - **实例**：
    - **实多项式环的干域**：已知 $\R[x]$ 不是域
      - **求解**：易得 $\R$ 上的最小既约多项式为 $f(X) = X^2 + 1$，其根为 $i$，故干域为 $\R(i) = \R[X]/(f) = \C$
        - **代数单扩张**：$\R(i) = a_0 + a_1i + a_2 i^2+\cdots = a+bi$
          - <font color='chartreuse'>$i^2=-1$，变回实数，故合并后只剩下两项</font>
        - **标准干域**：$\R[X]/(f) = (a_0 + a_1X + a_2X^2 + \cdots )/(X^2+1) = a+bX$
          - <font color='chartreuse'>带余除法消去2次以上单项式，故最终只剩下两项</font>
        - 显然，取 $X = i$，即标准同构 $\sigma:\R[X]/(f)\to \R[i]$ 时，两者相等
      - **实例**：原不可逆多项式 $X+1 \red\Rt i+1$ 的逆为 $\dfrac{1}{i+1} = \dfrac{i-1}{2} \red\Rt \dfrac{1}{2}X-\dfrac{1}{2}$
    - **复数域多项式的实数表示**：已知一个复系数多项式，求相同根的实系数多项式
      - **求解**：首先将根求出来，$\a_1,...,\a_n$
        - 然后将共轭根分别复合，若 $n$ 为偶数，则根均共轭。若 $n$ 为奇数，则存在一个根无共轭。这时我们添加其共轭根因子 $x-\ol\a_n$
        - 最终得到的就是实系数多项式
      - **实例**：$\Big[ x-(1+i) \Big]\Big[ x-(2+i) \Big] = x^2-(3+2i)x+(1+3i)$
        - 添加两个共轭因子 $x-(1-i)$ 和 $x-(2-i)$
        - 原式变为 $(x^2-2x+2)(x^2-4x+5)$
      - <font color='chartreuse'>可以应用到复相似矩阵求特征值上</font>
    - **欧几里得算法求逆**：设 $f(X) = X^3-3X-1\in \Q[X]$，已知其既约
      - $\Q[x] = \Q[X]/(f)$ 是 $\Q$ 上向量空间，基为 $\set{1,x,x^2}$
      - <font color='chartreuse'>设 $\b = x^4+2x^3+3\in \Q[x]$，用带余除法约去 $f(X)$ 后，得余项 $r = 3x^2+7x+5$。再用欧几里得算法，得 $f$ 与 $r$ 的裴蜀等式，从而得到 $r$ 的逆</font>

  - **有理函数域**：环 $K[x_1,...,x_n]$ 的分式域就是 $K(x_1,...,x_n)$，元素为 $\dfrac{f}{g}$

#### 结构的性质

- **（定理1.5）超越扩张结构自由性**：
  - 设域 $F\geq K$，$u\in F$ 是 $K$ 上超越元素，$x$ 是未知量
  - 则存在同构 $K(u)\cong K(x)$
  - **证明**：
    - 取 $\p:K(x)\to F，f/g \mapsto f(u)/g(u)$
    - 由超越性，$\forall g(u)\neq 0$，从而是良定义的
    - 易得是域单同态。再由域的生成定理即得 $\Im\p = K(u)$，是满同态
  - **理解**：用超越元素取代未知数 $x$，可以达到一样的多项式环效果。比如在 $\Z[x]$ 或 $\Q[x]$ 中，$ax^2 + 1 = a\pi^2+1$
    - 或者说，若取 $\a$ 为超越元素，则取干域不会商去任何等价的多项式
  - **本质**：超越扩张就是普通的域生成扩张
- **（定理1.6）代数扩张结构公式**：设 $F\geq K$，$u\in F$ 是 $K$ 上代数元素，则<a id = "代数扩张结构公式"></a>
  - **干域结论**：$K[u] = K(u)\cong K[x]/(f)$<font color='chartreuse'>（其中 $f\in K[x]$ 是（整除偏序下最小的）（以 $u$ 为根的）首一既约多项式）</font>
    - **证明**：详见[干域部分](#干域)，或取商映射 $\p:K[x]\to K[u]，g\mapsto g(u)$ 即可
    - **理解**
  - **代数元素 $u$ 在 $K$ 上的次数**：$[K(u):K] = n = \deg f$（下面两个是等价命题）
    - $\{1_K,u,u^2,...,u^{n-1}\}$ 是 $K$ 作用的向量空间 $K(u)$ 的基
    - $K(u)$ 中元素均可唯一表为 $\sum\limits^{n-1}_{i=0} a_iu^i$
    - **证明**：已知 $\forall g\in K[x]$，由带余除法可设 $g = qf + r\pad (\deg r <\deg f)$
      - 两边取干域商映射得 $g(u) = q(u)f(u) + r(u)$。再由 $u$ 是 $f$ 的根，得 $g(u) = r(u)$，故 $\deg g(u) < \deg f$，再由 $g$ 任意性即得结论
    - **理解**：<font color='red'>之前在[生成域的多项式性](#生成域的多项式性)说了，$K$ 中元素看作常数，可全部由基 $1_K$ 生成。$F-K$ 中元素看作由基 $\{u,u^2,...\}$ 生成
      - 但若存在一个 $f$，使得 $f(u) = 0$，即基在线性组合 $f$ 下会归零，即在次数大于 $\deg f$ 的线性组合必定等价于某个次数小于 $\deg f$ 的线性组合。也就是说，此时（线性无关向量）基只能是 $\deg f$ 个，即扩张被限制成了有限维的。（它和[干域部分](#干域)想要表达的意思相同）</font>
  - **本质**：代数扩张相比普通的域生成扩张，多了一个限制。故需要取一次商域
  - **推论（根非基）**：$f$ 的全部根 $\{u_1,...,u_n\}$ 不一定构成基
    - **反例**：比如 $\Q(\sqrt[3]{2})$ 中，$u = \sqrt[3]{2}$ 的极小多项式为 $x^3-2$，其全部根为两个2次本原单位根，不是 $\Q(\sqrt[3]{2})$ 的基
    - **实例**：详见[有限域习题](15.域的分类.md)
  - **推论（根的共轭性）**：既约多项式的根在分裂域中成对共轭出现
    - **理解**：以 $\C = \R(i)$ 为例（实际上在高代中已解释过一遍了）

### 尺规作图

- **域 $F$ 平面**：设 $F$ 是实数域的子域，则 $P_F = \set{(c,d)\mid c,d\in F}$ 是域 $F$ 平面
  - $P_F$ 上的线：以 $P,Q\in P_F$ 为两端点的直线
  - $P_F$ 上的圆：以 $P\in P_F$ 为圆心，$PQ$ 为半径的圆
- **基本作图法**：
  - **线段 $PQ$ 的垂直平分线**
    - 分别以 $P,Q$ 为圆心，$PQ$ 为半径画两个圆，连接两圆交点即可
  - **过点 $A$ 的 $PQ$ 平行线**：
    - 作出垂直平分线 $\a$
    - 以 $A$ 为圆心，随便画一个与 $\a$ 交于两点 $C,D$ 的圆
    - 再作 $CD$ 的垂直平分线，即为平行线
  - **过点 $A$ 的 $PQ$ 垂线**：
    - 作 $PQ$ 的垂直平分线 $\a$
    - 过 $A$ 作 $\a$ 的平行线

  - **证明**：直线等价于一次多项式方程，圆等价于二次多项式方程

        - 则 $[L:K] = [L:E][E:K]，[M:K] = [M:E][E:K]$，与互素矛盾，故只能是 $L\cap M = K$
      - 此时可设 $L = K(x_1,...)，M = K(y_1,...)$，由无共同中间域性，易得基集合不交，故可取等
    - **推论**：若 $L\cap M = K$，则仅当其中一个为2时等号成立
    - **反例**：
      - $2$ 次本原单位根中，$\Q(\sqrt[3]{2})$ 和 $\Q[\zeta]$ 都是 $3$ 次扩张，交集为 $\Q$，但乘积 $\Q(\sqrt[3]{2},\z) = \Q(\z)$，是 $3$ 次扩张
  - 若 $L,M$ 都是代数扩张，则 $LM$ 也是
    - **证明**：基都是根，故基的并集也是根
  - 设 $u,v$ 分别是 $K$ 上 $m,n$ 次代数元素，则 $[K(u,v):K]\leq mn$，当次数互素时取等
    - **证明**：$K(u,v) = K(u)(v)$
      - 即 $[K(u,v):K] = [K(u)(v):K(u)][K(u):K]$，其中 $v$ 在 $K$ 上是 $n$ 次代数元素，但在 $K(u)$ 上，多项式系数得到扩展，故 $v$ 的次数可能小于 $n$，故 $[K(u)(v):K] \leq n$
        - <font color='chartreuse'>或者我们可换一个视角，$K(u)$ 中 $\deg v < n \LR \lang v \rang \cong \lang u^i \rang$，那么互素时显然不可能有同构关系，故取等</font>
      - 但是当次数互素时，由维数整除关系即得只能取等
    - **反例**：
      - 设 $u$ 是 $4$ 次代数元素，极小多项式为 $f(x) = x^4-x^2-1$，则 $u^2$ 是 $2$ 次代数元素，$K(u,u^2) = K(u)$，是 $4$ 次扩张
  - **本质**：维数相关的统统转化为多项式和基就行了
- **多项式环的超越扩张性**：$\forall f\in K(x_1,...,x_n)$，若 $f\notin K$，则是 $K$ 的超越元素
  - **证明**：利用超越元素结构自由性，归纳即可
- **代数超越对称性**：若 $v$ 是 $K(u)$ 的代数元素，$K$ 的超越元素，则 $u$ 是 $K(v)$ 的代数元素
  - **证明**：
    - 存在 $f\in K(u)[x]$ 使得 $f(v) = 0$，由 $v$ 超越性，$f$ 的系数必须包含 $u$，故可将 $f(v)$ 看作 $u$ 的多项式 $g$，$v$ 变成系数，那么就有 $g(u) = 0$ 且 $g\in K(v)$
- **基的等价变换**：若 $u$ 是 $K$ 上奇数次代数元素，则 $K(u) = K(u^2)$，从而 $u^2$ 也是奇数次代数元素
  - **证明**：
    - 设 $u$ 的最小多项式为 $f$，则 $\deg f = 2k+1$ 是奇数，$K(u)$ 有奇数个新基 $\{u,...,u^{2k+1}\}$，显然将 $u$ 替换成 $u^2$ 时基不变，故 $K(u) = K(u^2)$
- **代数扩张环**：设 $K\leq R \leq F$，若 $R$ 是整环，$F$ 是 $K$ 的代数扩张，则 $R$ 是域
  - **证明**：只需证明 $R$ 的可逆性
    - $R$ 中元素均为代数元素，
- $F\geq K$ 是代数扩张 $\LR$ 每个中间域 $E$ 上的K单同态 $\sigma:E\to E$ 都是自同构
  - **证明**：代数扩张的基是根 + K自同构是 $E-K$ 部分根集的置换，即得结论
- **超越扩张保代数性**：设 $E_1,E_2\leq F，X\subset F$
  - 若 $E_1$ 中元素均为 $E_2$ 的代数元素<font color='chartreuse'>（但不一定是代数扩张）</font>，则 $E_1(X),E_2(X)$ 同理
  - **本质**：超越扩张不改变代数元素

#### 例子

- **超越扩张无限性**：有限生成的域扩张不是有限维的
  - **反例**：由维度代数关系，显然必须是超越扩张
    - 取 $\Q[\pi] \cong \Q[x]$，基为 $\{1,x,x^2,...\}$，是无限维的
- **超越扩张非独立性**：$K\leq F$，$u,v$ 是超越元素，但 $K(u,v)\not\cong K(x_1,x_2)$
  - **反例**：设 $v\in K(u)$，则 $K(u,v) = K(u)$
    - 比如 $\Q[\pi, \pi^2] \not\cong \Q[x,y]$
