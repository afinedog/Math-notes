# 求根公式

## 有限域

- **有限域（伽罗瓦域）**：元素数量有限的域
  - **表示方法**：$n$ 个元素的有限域用 $F_n$ 或 $GF(n)$ 表示
  - **素同构性**：$n$ 为素数 $\LR F_n \cong \Z_n$
    - **证明**：由数论知识，同余类中只有缩系才有逆，只有素数同余类的缩系为其本身
- **（定理5.1）素域最小性**：设 $F$ 是域，$P$ 是其所有子域的交，则 $P$ 是域，且没有真子域
  - 若 $\char F = p$，则 $P\cong \Z_p$
  - 若 $\char F = 0$，则 $P\cong \Q$
  - **证明**：易得 $P$ 的生成元只能是 $0,1_F$
    - 再由域的运算封闭性
      - 若 $\char P = p$，则 $P = \{m1_F\mid 0\leq m\leq p\}$
      - 若 $\char P = 0$，则 $P = \set{\dfrac{m1_F}{n1_F}\mid m,n\in\Z}$
    - 此时环同态 $\p:\Z\to P，m\mapsto m1_F$ 中，易得 $\ker\p$ 是 $\char P$ 生成的主理想
      - 若 $\char P = p$，则 $\Z_p \cong \Z/\ker\p \cong \Im\p \subset P$。由于 $\Z_p$ 是域，且 $P$ 没有真子域，则只能是 $\Z_p\cong P$
      - 若 $\char P = 0$，则其为单同态，已知其可延拓到分式域 $\Q$ 上，易得此时为满射，即 $\Q\cong P$
  - **理解**：大概就是先阐述一下 $P$ 的集合写法，发现 $\Z_p$ 和 $\Q$ 都只能嵌入 $P$。再用环同态的单射性证明同构性
  - **本质**：最小的有特征域是 $\Z_p$，最小的零特征域是 $\Q$
- **（推论5.2）有限域的结构**：设 $F$ 是有限域
  - 则存在素数 $p$ 满足 $\char F = p$，且 $|F| = p^n$，$F\cong \sum\limits^n_{i=1} \Z_p$
  - **证明**：
    - 已知整环的特征为素数，故 $\char F = p$，即 $P\cong \Z_p$
    - 再由 $F$ 是 $P$ 的有限维向量空间，由向量空间自由性得 $F \cong \sum\limits^n_{i=1} \Z_p$
  - **理解**：有限域 $F$ 必定是代数扩张，从而其结构是明晰的，找到一个最小的域 $P$，然后逐步扩张即可得到 $F$ 的结构
    - 素域 $P$ 可看作平凡域 $\Z_2 = \{0,1\}$ 关于 $p$ 原根的代数扩张，而有限域 $F_{p^n}$ 是素域 $P$ 的直和
- **（定理5.3）有限域乘法结构**：有限域上所有非零元构成的乘法群 $G$ 是循环群

      - <font color='chartreuse'>交换的乘法群同构于加法群</font>

- **（推论5.4）有限域单扩张性**：若 $F$ 是有限域，则 $F$ 是其素子域 $\Z_p$ 的单扩张
  - **证明**：取 $u$ 是乘法循环群 $G$ 的生成元即可
- **（引理5.5）新生梦**：设 $\char F = p$，$r\in\N^+$，则 $\p:F\to F，u\mapsto u^{p^r}$ 是 $\Z_p$ 同态
  - 若 $F$ 有限，则是同构
  - **证明**：由新生之梦直得同态的加法乘法公式
- **（命题5.6）有限域分裂性**：设 $p$ 是素数，$n\in\N^+$，则 $F$ 是 $p^n$ 阶有限域 $\LR F$ 是 $\Z_p$ 上 $x^{p^n}-x$ 的分裂域
  - **证明**：
    - **必要性**：此时 $|G| = p^n-1$，从而每个 $u\in G$ 都是 $x^{p^n-1}-1_F$ 的根。再想要使 $0$ 为根的话，只需变为 $x(x^{p^n-1}-1_F)$
    - **充分性**：
      - 易得 $f' = -1$，从而 $(f,f') = 1$，从而 $f$ 无重根，即有 $p^n$ 个不同根
      - 设 $\p$ 是新生梦单同态，则 $u\in F$ 是 $f$ 的根等价于 $u$ 是 $\p$ 的不动点
      - 由根的四则运算封闭性，$f$ 的根集 $E$ 是 $p^n$ 阶域。再由 $\Z_p$ 是最小域得 $E\geq \Z_p$。再由 $F$ 是分裂域，定义即得 $F = \Z_p(E) = E$
  - **理解**：
    - 分裂域就是所有根生成的域。而有限域的循环结构使得其等价于（本原多项式的分裂域）
    - 有趣的是，分裂域的维数和根的数量无关，但其阶数却等于根的数量。
- **（推论5.7）有限域存在唯一性**：对任意素数 $p$ 和正整数 $n$，都存在 $p^n$ 阶有限域，且在同构意义下唯一
  - **证明**：由分裂域存在性，$\Z_p$ 上 $x^{p^n}-x$ 的分裂域总存在。再由分裂域等价于 $p$ 域，由有限次多项式的分裂域唯一性即得结论
- **（推论5.8）有限维扩张存在唯一性**：设域 $K$ 有限，则对 $\forall n\in\N^+$，总存在有限 $n$ 维扩张 $F = K(u)$，且在K同构意义下唯一
  - **证明（存在性）**：
    - 设 $K$ 的阶为 $p^r$，设 $F$ 是 $K$ 上 $f_{nr}(x) = x^{p^{nr}}-x$ 的分裂域
      - 由有限域分裂性，$F$ 为 $p^{nr}$ 阶有限域
      - 由 $\Z_p$ 的最小性，$K,F$ 分别是 $\Z_p$ 上 $f_r,f_{nr}$ 的分裂域，由 $p^r,p^{nr}$ 个不同的本原单位根组成
      - 再由分裂域的定义，扩域本原单位根的数量 $=$ 基域本原单位根的数量 ** 基域中基的次数<font color='chartreuse'>（什么垃圾描述，直接看下面的理解得了）</font>
      - 综上，$p^{nr} = |F| = |K|^{[F:K]} = (p^r)^{[F:K]}$，即 $[F:K] = n$
      - 再由有限域的单扩张性即得 $F$ 是 $K$ 的单扩张
  - **证明（唯一性）**：
    - 反设存在另一个 $F_1$，则 $[F_1:\Z_p] = n[K:\Z_p] = nr$，从而 $|F_1| = p^{nr}$
    - 由有限域分裂性 + 有限次多项式分裂域唯一性，即得 $F_1$ 和 $F$ 在同构下等价
  - **理解**：本质是两个分裂域的阶具有指数关系
    - 另一种想法是寻找本原单位根的空隙。因为 $K,F$ 分别对标 $\z_r, \z_{nr}$，而本原单位根满足 $\z_{nr}^n = \z_r$，从而可立刻得到 $[F:K] = n$
- **（推论5.9）有限域多项式存在性**：设域 $K$ 有限，则对 $\forall n\in\N^+$ 均存在 $f\in K[x]$ 是 $n$ 次既约多项式
  - **证明**：由有限维扩张存在唯一性 + 代数扩张结构公式直得
- **（命题5.10）有限维扩张伽罗瓦循环性**：若 $F\geq K$ 是有限域的有限维扩张，则 $F$ 有限，且是伽罗瓦扩张，且 $\aut_K F$ 是循环群

### 可分的程度

- **可分次数 $[F:K]_S$**：设 $F\geq K$ 是代数扩张，$S$ 是最大可分子域，则 $[S:K]$ 是 $F$ 在 $K$ 上的可分次数
  - <font color='chartreuse'>还可添加单根基的数量（下界与最大可分子域的相差维数）</font>
- **不可分次数 $[F:K]_I$**：设 $F\geq K$ 是代数扩张，$S$ 是最大可分子域，则 $[F:S]$ 是 $F$ 在 $K$ 上的不可分次数
  - <font color='chartreuse'>基中重根的数量（上界与最大可分子域的相差维数）</font>
- **（引理6.11）单同态基数传递**：设 $N\geq F\geq E\geq K$，$N$ 是 $K$ 的正规扩张
  - 若 $r$ 是E单同态 $F\to N$ 的基数，$t$ 是K单同态 $E\to N$ 的基数
  - 则 $rt$ 是K单同态 $F\to N$ 的基数
  - **证明**：不妨设基数有限
    - 设 $\{\tau_i\}^r_{i=1}$ 是所有E单同态，$\{\sigma_j\}^t_{j=1}$ 是所有K单同态
    - **延拓性**：由分裂域延拓性，都可延拓为 $N$ 自同态。
    - **组合唯一性**：若 $\sigma_i\tau_j = \sigma_a\tau_b$，则 $\sigma^{-1}_a\sigma_i\tau_j = \tau_b$。再由 $\tau$ 的E不动性，$\sigma^{-1}_a\sigma_i|_E = 1_E$，再由 $\sigma$ 定义域为 $E$，故只能是 $\sigma_a = \sigma_i$。再由单射性即得 $\tau_j  = \tau_b$
    - **组合表出性**：任取K单同态 $\sigma:F\to N$，则由定义，可设 $\sigma|_E = \sigma_i$，从而K单同态 $\sigma_i^{-1}\sigma:F\to N$ 在E上不动，从而可设 $\sigma_i^{-1}\sigma = \tau_j$。此时 $\sigma = \sigma_i\tau_j$，即得表出性
- **（命题6.12）正规扩张的可分次数性**：设 $N\geq F\geq K$ 中，$F$ 是有限维扩张，$N$ 是 $K$ 的正规扩张。
  - 则K单同态 $F\to N$ 的基数是 $[F:K]_S$
  - **证明**：
    - 设 $S$ 是 $F$ 的最大可分子域。则由分裂域延拓性，只需证明 $F\to N$ 的基数和 $S\to N$ 的基数相同即可
      - 当 $\char K = 0$ 时，$F = S$，结论易得
      - 当 $\char K = p$ 时，设 $\sigma,\tau$ 都是 $F\to N$ 的K单同态，满足 $\sigma|_S = \tau|_S$
        - 若 $u\in F$，则由可分性 $\exists u^{p^n}\in S$。此时 $\sigma(u)^{p^n} = \sigma(u^{p^n}) = \tau(u^{p^n}) = \tau(u)^{p^n}$，从而 $\sigma(u) = \tau(u)$，从而 $\sigma = \tau$
    - 此时可设 $F = S$，再易得 $[F:K]_S = [S:K]，[F:E]_S = [S:E]$，从而 $[E:K] = [E:K]_S$
  - **归纳证明**：
    - 当 $n=1$ 时，显然
    - 若 $n-1$ 以下均成立，则取 $u\in F-K$，设 $[K(u):K] = r > 1$
      - 若 $r<n$，则由归纳假设 + 单同态基数传递性即得结论
      - 若 $r = n$，则 $F = K(u)$，则所有K单同态 $F\to N$ 都完全由 $u$ 上的作用决定
        - 设 $u$ 的根式为 $f$，则至多有 $\deg f$ 个K单同态。由正规性，$f$ 在 $N$ 上分裂。再由可分性，共有 $\deg f = [F:K]$ 个K单同态
- **（推论6.13）可分维数公式**：设 $F\geq E \geq K$，则 $\begin{cases} [F:E]_S[E:K]_S = [F:K]_S \\ [F:E]_I[E:K]_I = [F:K]_I \end{cases}$
  - **证明**：前两个定理直得
- **（推论6.14）一般分解公式**：设 $f\in K[x]$ 是首一既约多项式，$F$ 是其分裂域，$u_1\in F$ 是根，则
  - $f = \Big[ (x-u_1)\cdots(x-u_n) \Big]^{\large [K(u_1):K]_I}$，其中 $n = [K(u_1):K]_S$
    - <font color='chartreuse'>根的数量为可分次数，根的重数为不可分次数</font>
    - **证明**：
      - 若 $\char K = 0$，则结论显然
      - 设 $\char K = p$，对  $\forall i$ 都有根置换 $\sigma:K(u_1)\cong K(u_i)，u_1\mapsto u_i$ 可延拓到 $F$ 中。
        - 将 $\sigma$ 作用到 $f$ 上，由唯一分解性，只能是 $(x-u_i)^{r_i} = (x-\sigma(u_1))^{r_1}$，从而 $\forall r_i = r_1$，根的重数相同，$[K(u_1):K] = \deg f = nr$，从而有 $n$ 个不同的K自同态 $K(u_1)\to F$，从而 $[K(u_1):K]_S = n$，从而 $[K(u_1):K]_I = \cfrac{[K(u_1):K]}{[K(u_1):K]_S} = \dfrac{nr}{n} = r$
  - $u_1^{[K(u_1):K]_I}$ 是 $K$ 上可分元素
    - **证明**：
      - 若 $\char K = 0$，则结论显然
      - 设 $\char K = p$，此时 $r$ 是 $p$ 的幂，故由新生梦，$f = \sum\limits^n_{i=0} a_ix^{r_i}$，从而 $u_1^r$ 是 $g(x) = \sum\limits^n_{i=0} a_ix^i \in K[x]$ 的根，即 $g(x)$ 在 $K$ 上可分，从而 $u_1^r = u_1^{[K(u_1):K]_I}$ 是在 $K$ 上可分

### 范数和迹

- **代数元素的范数和迹**：设 $F\geq K$ 是有限维扩张，$\ol K\geq F$ 是代数闭包
  - 设 $\sigma_1,...,\sigma_r$ 是所有K单同态 $F\to \ol K$
  - 对 $u\in F$
    - **范数** $N_K^F(u) = (\sigma_1(u)\sigma_2(u)...\sigma_r(u))^{[F:K]_I}$
    - **迹**：$T_K^F(u) = [F:K]_I \Big(\sigma_1(u) + \sigma_2(u) + \cdots + \sigma_r(u) \Big)$
  - **等价定义**：$\ol K$ 换成正规扩张也成立
  - **实例**：$F = \C，K = \R$，则 $\R$ 单同态 $\C\to \C$ 只有恒等和共轭，此时 $N^{\C}_{\R}(a+b) = \Big[ (a+bi)(a-bi) \Big]^{1} = a^2+b^2$
  - **相似性**：迹是范数的加法形式，故证明其中一个的性质，另一个也可以类似证明
    - **反例**：若 $F$ 不是可分扩张，则 $\char K = p$，$[F:K]_I = p^t$。此时 $\forall u\in F，T^F_K(u) = 0$，但 $N^F_K(u)$ 不一定
- **（定理7.2）乘法加法公式**：设 $F\geq K$ 是有限维伽罗瓦扩张，$\aut_K F = \set{\sigma_1,...,\sigma_n}$
  - 则 $\forall u\in F，\begin{cases} N_K^F(u) = \sigma_1(u)\sigma_2(u)...\sigma_r(u) \\ T_K^F(u) = \sigma_1(u) + \sigma_2(u) + \cdots + \sigma_r(u) \end{cases}$

  - **同构公式**：$\begin{cases} N_K^F(u)N_K^F(v) = N_K^F(uv) \\\\ T_K^F(u) + T_K^F(v) = T_K^F(u+v) \end{cases}$

  - **指数乘法公式**：若 $u\in K$，则 $N_K^F(u) = u^{[F:K]}，T_K^F(u) = [F:K]u$
