# 一般环论

- 不少教材是先学域再学环的。实际上域的学习只需要环的概念，用不到多少环的定理，里面的向量空间知识也可以用高代的线性空间去类比。所以也可以先简单了解下环、模的相关概念后，直接去看域的部分。看完后再读环的部分，对环的认识会更清晰。当然，按顺序读也是可以的。

## 环

- **环**：一个集合 $R$ 与两个封闭二元运算（加法、乘法），满足 $(R,+)$ 是阿贝尔群
  - **加法**：具有可逆性、交换律、结合律的运算
  - **乘法**：具有结合律、左右分配律的运算
    - **左右分配律**：$a(b+c) = ab + ac$、$(a+b)c = ac + bc$
    - <font color='chartreuse'>$(R,\cdot)$ 不一定是群，因为不一定有乘法幺元和逆元。而且，很有意思的是，对 $c\in R$，只有其不是素元时才有 $ba = c$（详见[欧几里得环](./7.交换环.md)）</font>
- **交换环**：满足乘法交换律的环
- **平凡环**：只有一个元素的环，等价于 $1_R = 0$<font color='chartreuse'>（实际上，大多数情况只关心含幺环）</font>
- **含幺环**：
  - **幺元**：$1_Ra = a1_R = a$
  - **零元**：$0 + a = a + 0 = a$
- **运算法则**：<font color='chartreuse'>都是分配律的推论</font>
  - 零元乘法性：$0a = a0 = 0$
    - <font color='chartreuse'>反设 $0a = b$，则由分配律 $a = (0+1)a = b+a$，矛盾</font>
  - 负数交换律、结合律：$(-a)b = a(-b) = -(ab)$
    - <font color='chartreuse'>由分配律，$ab + (-a)b = (a-a)b = 0b = 0$，故是加法逆元</font>
  - 数乘交换律、结合律：$(na)b = a(nb) = n(ab)$
    - <font color='chartreuse'>由分配律，$(na)b = (a+\cdots+a)b = ab+\cdots+ab = n(ab)$</font>
- **左零因子**：已知 $a$，若 $\exists (b\neq 0)\in R$ 使得 $ab = 0$，则 $a$ 是 $b$ 的左零因子
  - **零因子**：同时为左右零因子的元素<font color='chartreuse'>（纠结这种死板定义没有意义，实际上具体情况中取临时定义也没有问题。在这里，我们只需要知道有提出零因子概念的必要就行了）</font>
    - **实例**：$Z_6$ 中，$2$ 与 $3$ 互为零因子
  - **乘法消去律**：若 $b$ 无左零因子，则 $ab = cb \red\Rt a = c\pad (\forall a,c\in R)$
- **左可逆**：已知 $a$，若 $\exists c\in R$ 使得 $ca = 1_R$，则 $a$ 左可逆
  - **左逆**：$a^{-1}_L = c$
    - **不唯一性**：
      - **反例**：无穷实数列对分量加法构成阿贝尔群，其上自同态构成环。
        - 设 $l,r$ 分别是左右推移算子，$l_0:(a_0,...) \mapsto (a_0+a_1,a_2,...)$
        - 则 $lr = \text{id} = l_0r$，即 $r$ 左逆不唯一
        - 同理，$l$ 右逆不唯一<font color='chartreuse'>（？？$l$ 非满射，$r$ 非单射，详见泛函分析的满线性算子群）</font>
    - **唯一性条件**：若 $a$ 不是右零因子，则其左逆唯一
      - **证明**：反设 $b_1a = b_2a = 1$，则 $(b_1-b_2)a = 0$，矛盾
      - <font color='chartreuse'>$r$ 是 $(l-l_0):(a_0,...)\mapsto (a_0,0,0,...)$ 的右零因子
        - 同时也是 $p:(a_0,...)\mapsto(a_0,a_0,a_0,...)$ 的右零因子
      - $l$ 是 $(r-r_0):(a_0,...)\mapsto (a_0,0,0,...)$ 的左零因子
        - 同时也是 $q:(a_0,...)\mapsto (a_0,0,0,...)$ 的左零因子</font>
  - **可逆元（单位）**：左右均可逆的元
    - **单位群 $\U_R$**：含幺环中的所有可逆元构成乘法群
      - **实例**：复平面整数点 $\Z[i] = \set{a+bi\mid a,b\in Z}$ 的单位群为 $\lang i \rang$
        - $\Z[3i]$ 的单位群为 $\lang -1 \rang$ 
- **整环 $R$**：无零因子且 $1_R \neq 0$ 的交换环<font color='chartreuse'>（无零因子、含幺、交换）</font>
  - **实例**：整数环 $Z$
  - **反例**：偶数环 $E$ 交换但无幺元、矩阵环有幺元但不交换，连续函数环有零因子
- **可除环 $D$（体）**：非零元均可逆 且 $1_D \neq 0$ 的环<font color='chartreuse'>（可逆、含幺）</font>
  - **有限情况**：无零因子的非平凡有限环是可除环
    - **证明**：无零因子得左右逆唯一，从而可逆，
  - **实例**：实四元数集是可除环
- **域 $F$**：交换的可除环<font color='chartreuse'>（可逆、含幺、交换）</font>
  - **有限情况**：有限整环是域
    - **证明**：$R-\{0\}$ 对乘法满足消去律，且为有限幺半群。易得其为乘法群。
      - **含幺性**：由整环定义
      - **交换性**：由整环定义
      - **可逆性**：整环得无零因子性，再由有限性得可逆性
  - **反例**：$\Z$ 是无限整环，但不是可除环，从而不是域
  - **总结**：有限情况下，除环 $\subset$ 整环 $\subset$ 域

### 例子

- **非平凡无幺环**：偶数环
  - 所有整数倍数环 $\set{nk\mid n\in\Z，(k\in\N^+)\neq 1}$，以及其在同余类上的限制
- **含幺非交换环**：
  - 矩阵环 $\Q^{n\times n}、\R^{n\times n}、\C^{n\times n}$，其中可逆元是非奇异矩阵
- **含零因子有限环**：$Z_{kr}$，零因子为 $\bar k，\bar r$
  - $Z_m$ 的子环为 $\set{Z_r\cong \dfrac{m}{r}Z_m : r\mid m}$
  - $\Aut Z_m = \{1\}$
- **有限域**：$Z_p$
- **有理数域 $\Q$**：下面结论都是域论知识
  - $\Q$ 的子域只有 $\Q$（详见[有限域](15.域的分类.md)）
  - $\Q[\sqrt{2}] = \set{a+b\sqrt{2}\mid a,b\in\Q}$ 是 $\R$ 的子域，其子域为 $\Q$ 和 $\Q[\sqrt{2}]$
  - $\Aut\Q[\sqrt{2}]\cong Z_2$
    - **证明**：易得 $\sigma(1) = 1$，故在 $\Q$ 上限制为恒等映射。再设 $\sigma(\sqrt{2}) = a$，已知 $\sigma(2) = a^2 = 2$，故 $a = \pm\sqrt{2}$
- **整数域 $\R$**：
  - $\Aut \R$ 具有单调性
    - **证明**：只需证明保号性。设 $a > 0$，则存在 $b$ 使得 $a = b^2$，从而 $f(a) = f(b)^2 > 0$
  - $\Aut \R = \{1\}$
    - **证明**：易得 $f$ 在 $\Q$ 上限制为恒等映射
      - 设 $r$ 是正无理数，由完备性，可取单增收敛列 $r_n\to r$，易得 $f(r)-f(r_n) < f(\e)$。再由单调性易得 $f(\e)\to 0$，从而 $f(r) = \lim\limits_{n\to\infty}r_n = r$，即得结论
- **复数域 $\C$**
  - **矩阵表示**：取嵌入 $\sigma:\C\to GL_n(\R)，a+bi\mapsto \tvec{a & b \\ -b & a}$，本质是复数特征值的一一对应性
- **自同态群** $\End A$ 是环
  - **交换条件**：若 $A$ 是交换群，则 $\End A$ 是交换环
  - **反例**：$A = Z\oplus Z$ 时，$\End A$ 是非交换环
- **群与环的组合**：设 $G$ 是 $n$ 阶乘法群，$R$ 是环
  - **群环**：$R[G] = \sum\limits_{g\in G} R$
    - <font color='chartreuse'>（为每个 $g$ 赋予一个环 $R$ 的副本，本质上是环的直和）</font>
    - 元素：$x = \sum\limits^n_{i=1} r_{g_i}g_i$<font color='chartreuse'>（形式和，类似形式幂级数）
      - 意思是说，$r_{g_i}$ 的下标是为了区分直和中不同的 $r$，但它无法表示该项在直和中所处的序号。故要引入右边的因子 $g_i$，它没啥实际意义，就是用来表示在直和中的序号。实际中的运算我们关心的只有 $g_i$ 的系数而已</font>
    - 加法：$x+y = \sum\limits^n_{i=1} r_{g_i}g_i + \sum\limits^n_{i=1} s_{g_i}g_i = \sum\limits^n_{i=1} (r_{g_i}+s_{g_i})g_i$
    - 乘法：$R(r_is_j) = G(g_ih_j) = \dkh{\sum\limits^n_{i=1} r_ig_i}\dkh{\sum\limits^m_{j=1} s_{g_i}h_j} = \sum\limits^n_{i=1}\sum\limits^m_{j=1} (r_is_j)(g_ih_j)$
    - 幺元：$1_R\cdot e$
  - **实例**：
    - 当 $G$ 为 $n$ 阶循环群时，$R[G] = R[x]/(x^n-1)$ 是干域（详见[有限域](15.域的分类.md)）
    - 当 $G$ 为无限循环群（$\Z$）时，$R[G] = R[x]$ 是多项式环
  - **父环性**：$R$ 是 $R[G]$ 的子环
  - **交换性**：
    - 若 $G$ 有限，$R$ 交换，则 $R[G]$ 的中心是
    - $R,G$ 均交换 $\LR$ $R[G]$ 交换
  - **零因子**：
    - $G\neq \{e\}$ 时，$\Big( \sum\limits_{g\in G} g \Big)(1-h) = (1-h)\sum\limits_{g\in G} g = 0$，它们互为 $R[G]$ 的零因子
    - $G = \{e\}$ 时，$R[G]$ 无零因子 $\LR R$ 无零因子
  - **单位群**：
    - 若 $G = \set{1,g}$，则 $n+mg\in R[G]$ 可逆 $\LR \begin{cases} nx+my = 1 \\ mx+ny = 0  \end{cases}$ 有整数解，即 $n^2-m^2 = \pm 1$，从而 $R[G]$ 的单位群为 $K_4 = \set{1,-1,g,-g}$
- **实四元可除环**：[群的习题](2.群的映射.md)中也有出现
  - **实数表示**：$\set{1,-1,i,-i,j,-j,k,-k}$
  - **复矩阵表示**：所有矩阵 $\tvec{z & w \\ -\ol w & \ol z}$ 构成的环
    - 生成元系：$\tvec{1 & 0 \\ 0 & 1},\tvec{i & 0 \\ 0 & -i},\tvec{0 & 1 \\ -1 & 0},\tvec{0 & i \\ -i & 0}$


### 环与同态

- **（定理1.6）二项式定理**
- **环同态**：两个环之间的映射 $f:R\to S$ 满足 $\begin{cases} f(a+b) = f(a)+f(b) \\ f(ab) = f(a)f(b) \end{cases}$
  - **核**：同态像是 $0$ 的元素
  - 若 $R$ 含幺，则陪域 $S$ 必须含幺
    - <font color='chartreuse'>$\p:\Z\to 2\Z，1\mapsto 2$ 不是环同态，不满足乘法公式</font>
  - **实例**：
    - $Z\to Z_m，k\mapsto \ol k$ 是满同态（收缩）
    - $Z_3 \to Z_6，\bar k\mapsto \ol{4k}$ 是单同态（嵌入）
    - **群环同态**：设 $f:G\to H$ 是群同态
      - 则 $\bar{f}:R[G]\to R(H)，\sum\limits^n_{i=1} r_ig_i\mapsto \sum\limits^n_{i=1} r_if(g_i)$ 是环同态
  - **无保幺元性**：$1_R\to 1_S$ 不必总成立
    - **反例**：
      - 同态不保可逆元，则不保幺元
        - 泛函分析中，满线性映射构成群，其有右逆，无左逆
      - 陪域存在零因子，则可能不保幺元
        - 矩阵同态：$\p:R\to R^{2\times 2}，x\mapsto \tvec{x & 0 \\ 0 & 0}$，其像集无幺元（是 $R^{2\times 2}$ 的子环，但幺元不相等）
    - **充分条件**：
      - 满同态保幺元
        - **证明**：$\p(1_R)s \overset{满射}{=} \p(1_R)\p(r) = \p(1_R\cdot r) = \p(r) = s$
      - 若同态非平凡，且陪域无零因子，则映射保幺元
        - **证明**：
          - 反设 $\p(1_R) \neq 1_S$，则 $\forall \p(a)(\p(1_R)-1_S) = 0$，与无零因子性矛盾
        - **理解**：
          - 由于陪域无零因子，再由 $\p(0)$ 和 $\Im\p$ 中元素相乘均为0，故只能是 $\p(0) = 0$
          - 仿照上面的思路，由于 $\p(1_R)$ 和 $\Im\p$ 中元素相乘均为其本身，用类似数分零点存在 $\to$ 不动点存在的方法，即可得结论
        - **本质**：无零因子的意思可看作“无局部恒定元素”，即无局部的 $0$ 因子和 $1,2,...$ 因子等
          <!-- - 由上得 $\Im\p$ 中都有幺元性，只需考虑 $\p(R)\j \Im\p$ 部分 -->
  - **保可逆同态**：满足保幺元性和保逆元性
- **环范畴**：所有环构成的范畴，态射为环同态
- **环的特征**：若 $\exists n\in \Z$ 使得 $na = 0\pad (\forall a\in R)$，则 $\char R = n$
  - **本质**：环作为加法群时，元素的阶
  - **实例**：$Z_p$ 的特征为 $p$
  - **零特征环**：没有特征的环（无限阶加法阿贝尔群）
    - **实例**：$\Z、n\Z、\Z^{n\times n}$
- **（定理1.9）特征性质**
  - **幺元阶最大性**：特征 $n$ 是满足 $n1_R = 0$ 的最小正整数
    - **证明**：
      - 反设 $\char R = n$，但存在 $m<n$ 使得 $m1_R = 0$
      - 则此时存在 $a$，使得 $n$ 是使其为 $0$ 的最小系数，则 $n$ 也是 $a\cdot 1_R$ 的最小系数
        - 但是由乘法分配律，$ma\cdot 1_R = a\cdot m1_R$
        - 由 $1_R$ 性质，右式为0，但由 $a$ 性质，左式不为0，矛盾
  - **系数同态**：$\p:\Z\to R，m\mapsto m1_R$，核为 $\{k\cdot \char R\mid k\in\Z\}$
    - **证明**：同态式易得
  - **同余类性**：若 $R$ 不含零因子，则 $\char R$ 是素数
    - **证明**：反设 $\char R = mn$，则 $mna = m(na) = 0$。由于 $m<mn$，故 $na$ 只能是 $0$。再由 $n<mn$ 得要么 $a=0$，要么 $R$ 同构于同余类，且 $n$ 是 $a$ 的零因子
  - **本质**：具有特征的环可以看作同余类
- **（定理1.10）嵌入定理**：每个环 $R$ 都可以嵌入到某个含幺环 $S$ 中
  - 其中 $\char S = 0$ 或 $\char S = \char R$
  - **证明**：
    - **零特征嵌入**：
      - 设 $S$ 是阿贝尔群 $R\oplus Z$
      - 定义乘法 $(r_1,k_1)(r_2,k_2) = (r_1r_2+r_1k_2+r_2k_1，k_1k_2)$
      - 则 $S$ 是含有幺元 $(0,1)$ 的环且特征为0，环嵌入为 $r\mapsto (r,0)$
    - **等特征嵌入**：
      - 设 $S$ 是阿贝尔群 $R\oplus Z_n$（其中 $n = \char R$）
      - 定义乘法 $(r_1,\ol k_1)(r_2,\ol k_2) = (r_1r_2+r_1\ol k_2+r_2\ol k_1，\ol k_1\ol k_2)$
      - 则 $S$ 是含有幺元 $(0,\ol 1)$ 的环且特征为 $n$，环嵌入为 $r\mapsto (r,0)$
  - **理解**：
  - **本质**：每个环都可嵌入自身和整数的直和中，是环的规范逆投影

### 习题

- **环的直和**：设 $\{R_i\mid i\in I\}$ 均为含幺环，则 $\sum\limits_{i\in I} R_i$ 是含幺环，加法和乘法为各分量相加相乘。
- **$R$ 的反环**：$R^{\circ p}$ 的底层集合是 $R$，加法和乘法都换序（$a\circ b = ba$）
  - **含幺等价性**：两环幺元相同
  - **除环等价性**：由互逆可换性，两环逆元相同
  - **自反性**：反环的反环是原环
  - **同构等价性**：定义易得
- **广义可逆性**：若 $R$ 中任意非零元 $a$，都存在唯一的 $b$ 使得 $aba = a$，则其满足
  - **对称性**：$bab = b$
    - **证明**：易得此时 $(ab)^2 = ab$，即 $ab$ 是幂等元素
      - 故 $(ab)^3 = (aba)(bab) = ab$，再由 $aba = a$ 即得 $bab = b$
  - **含幺性**：
    - **证明**：此时 $(ab)a = a$，是左幺，$b(ab) = b$，是右幺
  - **无零因子性**：
    - **证明**：反设 $a$ 有右零因子，则其右逆不唯一。设 $ac_1 = ac_2 = 1$，则 $ac_1a = a，ac_2a = a$，与题设唯一性矛盾，故无零因子
  - **可逆性**：
    - **证明**：由含幺性得 $(ab-1)a = 0$，再由无零因子性，只能是 $ab = 1$，同理 $ba = 1$，从而 $b = a^{-1}$
- **新生之梦（蚌埠住了这名字）**：设 $R$ 是交换幺环，若 $\char R = p$，则 $(a\pm b)^{p^n} = a^{p^n} \pm b^{p^n}$
  - **证明**：二项式定理易得
- **幂零元素**：$\exists n$ 使得 $a^n = 0$
  - **加法封闭性**：交换环中两幂零元素的和 $a+b$ 也幂零
  - **反例**：非交换环中不成立
    - $GL_2(\R)$ 中，上升阵 $N$ 和下降阵 $N^T$ 幂零，但 $N+N^T$ 可逆
  - **推论**：环 $R$ 中不存在非零的幂零元素 $\LR a^2 = 0 \blue\Rt a = 0$
    - **证明**：
      - **必要性**：定义易得
      - **充分性**：设 $a^n = 0$
        - 若 $n$ 是偶数，则 $a^n = (a^m)^2 = 0$，由充分性条件即得 $a^m = 0$
        - 若 $n$ 是奇数，则 $a^n\cdot a = 0$，从而 $a^{\dfrac{n+1}{2}} = 0$
        - 不断重复即得结论
- **Frobenius同态**：若交换环 $R$ 满足 $\char R = p$，则 $F:R\to R，r\mapsto r^p$ 是同态
  - **证明**：
    - 由交换性，$F(ab) = (ab)^p = a^pb^p = F(a)F(b)$
    - 由新生之梦，$F(a+b) = (a+b)^p = a^p+b^p = F(a)+F(b)$
- **可逆共轭性**：幺环中，$1-ab$ 可逆 $\LR 1-ba$ 可逆
  - **证明**：设 $(1-ab)^{-1} = c$，则 $(1-ba)^{-1} = 1+bca$

#### 例子

- **全零环（交换无幺环）**：加法阿贝尔群 $G$ 上，定义 $\forall ab = 0$，则其构成环
  - **交换性**、**无幺性**、**不可逆性**
  - **零因子**：所有元素均互为零因子
- **幂集环（交换幺环）**：幂集 $\mc P(U)$ 中，$A+B = A\D B$，$AB = A\cap B$，则其构成环
  - **交换性**：易得
  - **含幺性**：幺元为全集，零元为空集
  - **零因子**：不相交的集合互为零因子
- **布尔环（幂等环）（交换幺环）**：$\forall a\in R，a^2 = a$
  - **自逆性**：$a+a = 0$
    - **证明**：已知 $(-a)^2 = (-1)a(-1)a = a^2 = a$，再由 $(-a)^2 = -a$，即得 $a = -a$

## 理想

- **$X$ 生成的理想**：设 $X\subset R$，则 $\{A_i\mid i\in I，A_i\in\I_R，X\subset A_i\}$ 是 $R$ 上包含 $X$ 的所有理想。此时 $\mathop{\bigcap}\limits_{i\in I} A_i = (X)$ 即为子集生成的理想

    - **本质**：具有乘法交换性后，双向乘法和单向乘法可合并为左乘

  - **理解**：

### 素理想（理想的大小）

- **素理想**：$P\in\I_R，P\neq R$，满足 $AB\subset P \red\Rt A\subset P$ 或 $B\subset P\quad (\forall A,B\in\I_R)$
  - **理解**：素数空隙
  - **本质**：素数生成的理想，详见[欧几里得环的性质](7.交换环.md)
- **元素判定法**：$P\in \I_R，P\neq R$，满足 $ab\in P \red\Rt a\in P$ 或 $b\in P\quad (\forall a,b\in R)$
  - **证明**：定义直得
  - **实例**：
    - 整环上的零理想 $(0)$ 都是素理想
      - <font color='chartreuse'>因为整环无零因子，故 $ab = 0 \red\Rt (a=0) \lor (b=0)$</font>
    - $Z$ 上的素数主理想 $(p) = pZ$ 是素理想
  - **反例**：$Z_6$ 上的 $(0)$ 不是素理想，因为 $\ol 2\cdot \ol 3 = \ol 0$
  - **理解**：若不是素数空隙，则两个真因子会凑出空隙。也就是说，素理想是空隙最小的理想，也即元素最多的理想
  - **本质**：素理想不是两个理想的乘积
    - <font color='red'>理想和空隙的大小是反过来的，$(a)$ 中的数字 $a$ 表示空隙。$a$ 越小则生成的理想越大
    - 而两个理想相乘时，因为空隙被放大了，结果反而更小。</font>
- **（定理2.16）素理想单元性**：
  - 若交换环 $R$ 满足 $1_R\neq 0$
  - 则 $P\in\I_R$ 是素理想 $\LR R/P$ 是整环
  - **必要性**：
    - **含幺性**：若 $P$ 是素理想，则 $P\neq R$，从而 $1_R\notin P$，则此时 $1_R+P\neq P$，即 $\ol{1_R} \neq \ol 0$
    - **无零性**：反设 $R/P$ 存在零因子
      - 则有 $(a+P)(b+P) = P$，从而 $ab\in P$
      - 再由素理想定义，只能是 $a\in P$ 或 $b\in P$，即只能是 $R/P$ 中的零元素，也即不存在 $R/P$ 的零因子
    - 综上即得 $R/P$ 是整环
  - **充分性**：
    - **非全集性**：若 $R/P$ 是整环，则 $1_R \neq 0$，从而 $1_R\notin P$，即 $P\neq R$
    - **素性**：再同上即可得 $P$ 是素理想
  - **理解**：压缩了
  - **实例**：
    - $Z/pZ = Z_p$ 是整环（域）

- **理想与可逆性**：<a id = "理想与含幺"></a>
  - 含幺环是可除环 $\LR R$ 没有真左理想
    - **证明**：反设存在真左理想，任取 $x\in I$，则 $x^{-1}x = 1_R\notin I$，与乘法封闭性矛盾
  - 若 $S$ 是无真左理想的环，则其要么是全零环（$S^2 = 0$），要么是可除环
    - **证明**：
      - 若含幺，则已证是可除环
      - 若不含幺，易得 $S$ 中所有元素的零因子集 $I = \set{a\in S\mid Sa = 0}$ 构成左理想。再由 $S$ 无真左理想，故只能是 $I = (0)$ 或 $I = S$
        - 而当 $I = (0)$ 时，由于任意元素的右零因子也是左理想，故 $S$ 中元素均无右零因子
        - 此时右乘一个元素是置换作用，而所有右乘元素作用构成的集合 $\{\sigma_i: d \mapsto dx_i\mid i\in\N^+，d,x_i\in S\}$ 是 $S$ 中的一个置换
        - 设 $e$ 是被 $d$ 置换为 $d$ 的元素，则 $ed = d$
        - 同理可得 $de = d$，从而 $e$ 是幺元，与不含幺矛盾
    - **本质**：置换法求幺元、逆元看来是一个常用思想。需要学习一下

#### 素理想

- **其它几种定义**：设 $P$ 是 $R$ 中理想，则下列条件等价
  - $P$ 是素理想
  - 若 $rRs\subset P$，则 $r\in P$ 或 $s\in P$
  - 若 $(r),(s)$ 满足 $(r)(s)\subset P$，则 $r\in P$ 或 $s\in P$

  - **证明**：
    - $(1)\to (2)$ 元素分析法 + 定义易得
    - $(1)\to (3)$
- 含幺交换环中，零元和零因子集<font color='chartreuse'>（$S = \{s\in R\mid \exist a\in R，sa = 0\}$）</font>中至少有一个素理想
  - **证明**：
    - 若环中无零因子，易得零元是素理想
      - $ab = 0$，则要么 $a = 0$，要么 $b = 0$
    - 若环中存在零因子，由交换性，易得零因子集是素理想
      - $c(ab) = acb = abc = 0$，则要么 $ca = ac = 0$，要么 $cb = bc = 0$
- 含幺交换环 $R$ 中，若理想 $A$ 含于素理想并集 $\mathop{\bigcup}\limits^n_{i=1} P_i$，则 $\exists i，A\subset P_i$
  - **证明**：
    - 首先，若 $A$ 不含于单个 $P_m$ 中，则 $A$ 与所有 $P_j$ 相交
      - 设 $a \in A\cap P_m$
      <!-- - 则 $ab\in A$，再由理想乘法封闭性，$ab \in A\cap P_i\cap P_j$ -->
      - 再分别取 $p_j\in P_j$，则由理想乘法封闭性，$\prod p_i a\in A\cap \cdots \cap P_n$
      - 从而得到引理
    - 然后，证明 $\forall j，(A\cap P_j) \not\subset \mathop{\bigcup}\limits_{i\neq j} P_i$
      - 反设 $\exists j$ 使得 $(A\cap P_j)\subset \mathop{\bigcup}\limits_{i\neq j} P_i$，则 $\forall b\in A\cap P_j，\exists k$ 使得 $b\in P_k$

      - 再由 $P_k$ 是素理想，要么 $a_k\in P_k$（定义矛盾），要么 $r_k\in P_k$（定义矛盾）

### 环与直积


- **（定理2.22）直积性质**：设 $R_i$ 是环族，$\prod\limits_{i\in I} R_i$ 是加法阿贝尔群直积
  - 加法为各元相加，乘法为各元相乘 $\{a_i\}_{i\in I}\{b_i\}_{i\in I} = \{a_ib_i\}_{i\in I}$，则其构成环
  - 含幺性、交换性可传递
  - 规范投影是满同态
  - 规范逆投影是单同态
- **环的直和**：环的弱直积是直积的子环，同时也是环的直和
- **（定理2.24）弱内直积**：
  - 若 $R$ 的理想列 $\{A_i\}^{n}_{i=1}$ 满足
    - **覆盖性**：$\sum\limits^n_{i=1} A_i = R$
    - **不相交性**：$A_k\cap \sum\limits_{i\neq k} A_i = 0\quad (\forall 1\leq k\leq n)$
  - 则存在环同构 $R\cong \prod\limits^n_{i=1} A_i$
  - **证明**：已知 $(a_1,...,a_n) \mapsto \sum\limits^n_{i=1}a_i$ 是加法阿贝尔群同构，只需证明其为环同构即可
- **环同余**：若 $a,b\in R$ 满足 $a-b\in A$，则称为**模理想 $A$ 同余**
- **（定理2.25）中国剩余定理**：
  - 设 $\{A_i\}^n_{i=1} \in \I_R，\{b_i\}^n_{i=1}\in R$
  - 若满足
    - **平方不太小性**：$R = R^2+A_i$
    - **两两覆盖性**：$R = A_i+A_j\pad (\forall i\neq j)$
  - 则存在 $b\in R$ 使得 $b\equiv b_i \pmod {A_i}$
  - **引理**：$R = A_1 + (\mathop{\bigcap}\limits_{i\neq 1}A_i)$
    - **初始条件**：$R = A_1 + (A_2\cap A_3)$
      - 由题设，$R^2 = (A_1+A_2)(A_1+A_3) = A_1^2 + A_1A_3 + A_2A_1 + A_2A_3 $
      - 由乘积缩小性，原式可改写为 $\subset A_1+A_2A_3 \subset A_1+(A_2\cap A_3)$
      - 再由题设，$R = A_1 + R^2 \subset A_1 + (A_2\cap A_3)\subset R$
        - 综上即得**初始条件**
    - **归纳假设**：$R = A_1 + (A_2\cap\cdots A_{k-1})$
      - 依然有 $R^2 = \Big(A_1 + A_2\cap\cdots\cap A_{k-1}\Big)(A_1 + A_{k}) = \cdots$
  - 同理也可得 $R = A_k + (\mathop{\bigcap}\limits_{i\neq k}A_i)$
  - 综上，$\forall k，\exists a_k\in A_k，r_k\in (\mathop{\bigcap}\limits_{i\neq k}A_i)$ 使得 $b_k = a_k + r_k$
    - 也即 $\begin{cases} r_k\equiv b_k \pmod{A_k} \\ r_k\equiv 0\pmod{A_i} & i\neq k\end{cases}$
    - 最后取 $b = r_1 + \cdots + r_n$ 即可
  - **推论**：$b$ 由模 $\bigcap A_i$ 的同余类唯一决定
    - **证明**：若 $\forall i$，都有 $c \equiv b_i \pmod{A_i}$，则也有 $b\equiv c\pmod A_i$
      - 从而由任意性，$b-c\in \bigcap A_i$，也即 $b\equiv c\pmod{\bigcap A_i}$
- **（推论2.27）商积定理**： <a id = "商积定理"></a>
  - 若 $A_1,...,A_n \in \I_{R}$
  - 则存在单同态 $\t:R/(A_1\cap \cdots \cap A_n)\to R/A_1\times \cdots \times R/A_n$
  - **证明**：由环范畴的范畴直积定理，规范商映射 $\pi_k$ 可导出商积环同态 $\t_1: R\to \prod\limits^n_{i=1} R/A_i，r\mapsto (r+A_i)^n_{i=1}$
    - 易得 $\ker \t_1 = \mathop{\bigcap}\limits^n_{i=1} A_i$。再由环同构定理，存在环单同态 $\t:R/(\cap) \to \prod R/A_i$
  - **推论**：满足中国剩余条件，则 $\t$ 是环同构
    - **证明**：
      - 由中国剩余定理可得同余关系
      - 此时 $\t(b+\cap A_i) = (b+A_i)^n_{i=1} = (b_i+A_i)^n_{i=1}$，从而是满射
    - **反例**：
      - 设 $R = \Z，A_1 = (6)，A_2 = (4)$
      - 则 $\t:R/(A_1\cap A_2)\to R/A_1\times R/A_2$ 不是满射
      - **证明**：此时 $\t:Z_{12}\to Z_6\times Z_4$，前者是12阶，后者是24阶
