# 伽罗瓦理论的应用

- **新符号**：为了方便计算，提出以下两个符号
  - $\Gal(F/K)$ 表示伽罗瓦扩张 $F\geq K$ 的伽罗瓦群 $\aut_K F$
  - $E^G$ 表示群 $G$ 在域 $E$ 上的不动点集

## 多项式的伽罗瓦群

- **多项式的伽罗瓦群**：设 $K$ 是域，$f\in K[x]$。若 $F$ 是 $K$ 上 $f$ 的分裂域，则 $\aut_K F$ 称为 $f$ 的伽罗瓦群
  - <font color='chartreuse'>其为正规扩张。若 $f$ 可分，则为伽罗瓦扩张
  - 且由分裂域唯一性，其与分裂域的选择无关</font>
- **（定理4.2）置换同构性**：设 $K$ 是域，$f\in K[x]$ 的伽罗瓦群为 $G_f$，$\deg f = n$，则
  - $G_f$ 同构于 $S_n$ 的子群
    - **证明**：设 $u_1,...,u_n$ 是分裂域 $F$ 中的不同根，则由K同构保根性，$\forall \sigma\in\aut_K F$ 是根的置换，从而可构造同构映射 $\aut_K F\to S_n$
  - 若 $f$ 是 $n$ 次既约可分多项式，则 $n$ 整除 $|G_f|$，且 $G_f$ 同构于传递子群
    - **证明**：
      - 由可分性+分裂性得 $F$ 是代数伽罗瓦扩张，从而 $[K(u_1):K] = n = \deg f$
      - 由伽罗瓦对应，$G_f$ 中存在 $n$ 阶子群，从而 $n$ 整除 $|G_f|$
      - 由标准干域唯一性，对每个 $i\neq j$ 都存在同构 $\sigma:K(u_i)\cong K(u_j)，u_i\mapsto u_j$，且由分裂域同构可延拓性，$\sigma$ 可延拓到 $F$ 上
      - 易得 $G_f$ 作用在根集上时只有一个轨道，故同构于传递子群
  - **本质**：<font color='red'>以后都将多项式伽罗瓦群视为根的置换群</font>
- **（推论4.3）二次方程的伽罗瓦群**：设 $K$ 是域，$f\in K[x]$ 是 $2$ 次既约多项式，伽罗瓦群为 $G_f$
  - 若 $f$ 可分，则 $G_f\cong \Z_2$，或 $G_f = 1$
  - **证明**：可分得其为伽罗瓦扩张，从而是闭域，即 $|G_f| = [F:K]$ 整除 $2!$，此时只有题目中那两种情况。

### 三次多项式

- **判别式**：设域 $K$ 满足 $\char K\neq 2$，则 $f\in K[x]$ 是 $n$ 次多项式，在分裂域 $F$ 上有根 $u_1,...,u_n$。设 $\D = \prod\limits_{i<j} (u_i-u_j)$，则 $D = \D^2$ 称为 $f$ 的判别式
  - **非扩张性**：$\D^2\in K$
    - **证明（对称法）**：由于不考虑符号时 $\D$ 是对称的，从而 $\D^2$ 是完全对称的。从而其系数在根置换 $\aut_K F$ 下保持不变。再由 $F$ 是伽罗瓦扩张，只能是各系数均含于 $K$，再由域的加法封闭性，即得总体也含于 $K$
    - **证明（同构法）**：$\sigma(\D^2) = \sigma(\D)^2 = (\pm \D)^2 = \D^2$，故是不动点
  - **奇偶判别性**：$\forall \sigma\in \aut_K F < S_n$
    - $\sigma$ 是偶置换 $\LR \sigma(\D) = \D$
    - $\sigma$ 是奇置换 $\LR \sigma(\D) = -\D$
    - **证明**：已知置换奇偶性与分解成对换时的数量有关
      - 也就是说，若有奇数个根被对换，不难验证原来的各项系数会取负，偶数个对换同理
  - 设 $G_f = \aut_K F$ 是 $S_n$ 的子群，则在伽罗瓦对应中，$K(\D)\LR G_f\cap A_n$
    - **证明**：
      - 伽罗瓦对应中，$K(\D)$ 对应的是 $\aut_{K(\D)} F$，而 $G_f$ 保持 $K$ 不动，$A_n$ 保持 $\D$ 不动，故它们的交集就是保持 $K(\D)$ 不动的所有自同构
  - $G_f$ 由偶置换组成 $\LR \D\in K$
    - **证明**：易得
- **（推论4.7）三次方程的伽罗瓦群**：设 $K$ 是域，$f\in K[x]$ 是 $3$ 次既约可分多项式，则伽罗瓦群是 $S_3$ 或 $A_3$
  - **证明**：
    - $|G_f|$ 整除 $3!$，且同构于置换群，故只可能是 $\braket{e}，\Z_2，\Z_3 = A_3，S_3$
    - 
  - **推论**：若 $\char K\neq 2$，则 $G = A_3 \LR \D$ 是 $K$ 中元素的平方
- **（命题4.8）三次方程伽罗瓦群判别式**：设 $K$ 是域，$\char K \neq 2,3$
  - 若 $f(x)=  x^3 + bx^2 + cx + d\in K[x]$ 在分裂域中有三个不同的根
  - 则
    - **标准化**：$g(x) = f(x-\dfrac{b}{3})\in K[x]$ 的形式为 $x^3+px+q$<font color='chartreuse'>（代入易得）</font>
    - **判别式**：$f$ 的判别式为 $-4p^3-27q^2$
  - **证明**：设 $F$ 是 $K$ 上 $f$ 的分裂域。易得 $g,f$ 判别式相同
    - 由 $g$ 根与系数关系，其三个根满足 $\begin{cases} v_1+v_2+v_3 = 0 \\ v_1v_2 + v_1v_3 + v_2v_3 = p \\ -v_1v_2v_3 = q \end{cases}$
    - 代入计算即得 $g$ 的判别式
  - **本质**：就是换元简化了一下，其实还是用的根与系数关系
  - **实例（求伽罗瓦群）**：
    - $x^3-3x+1\in \Q[x]$
      - **标准化**：已经是标准形式
      - **可分性**：易得其既约。再由 $\char \Q = 0$，其可分
      - **判别式**：判别式为 $-4(-3)^3-27(1)^2 = 81$，是完全平方数，故伽罗瓦群为 $A_3$
    - $f(x) = x^3+3x^2-x-1\in \Q[x]$
      - **标准化**：$g(x) =  f(x-\dfrac{3}{3}) = f(x-1) = x^3-4x+2$
      - **可分性**：其既约
      - **判别式**：判别式为 $148$，不是完全平方数，故伽罗瓦群为 $S_3$

### 四次多项式

- **（引理4.9）四次方程求根**：设 $K$ 是域，$f\in K[x]$ 是 $4$ 次多项式，根为 $u_i$，分裂域为 $F$
  - 易得Klein群 $V = \set{(1),(12)(34),(13)(24),(14)(23)}\lhd S_4$
  - 故由正规子群交传递性，$(V\cap G_f)\lhd G_f = \aut_K F \leq S_4$
  - 若 $\begin{cases} \a = u_1u_2 + u_3u_4 \\ \b = u_1u_3 + u_2u_4 \\ \g = u_1u_4 + u_2u_3 \end{cases}$，则子域 $K(\a,\b,\g)$ 和 $V\cap G_f$ 是伽罗瓦对应
  - 从而 $K(\a,\b,\g)\geq K$ 是伽罗瓦扩张，$\aut_K K(\a,\b,\g)\cong G_f/(G_f\cap V)$
  - **证明**：
    - 易得 $\a,\b,\g$ 在 $V\cap G_f$ 下不动，从而 $K(\a,\b,\g) \leq (V\cap G_f)'$
    - 再证明 $G_f-(V\cap G_f)$ 的点均可动
      - 例如，$\sigma = (12)$，反设 $\sigma(\b) = \b$，则 $(u_2-u_1)(u_3-u_4) = 0$，与根不同矛盾
      - 依次讨论……
- **三次预解式**：$(x-\a)(x-\b)(x-\g)\in K(\a,\b,\g)[x]$ 称为 $f$ 的三次预解式
  - <font color='chartreuse'>因为只有伽罗瓦扩张才是性质最好的，而伽罗瓦群可以先用部分性质确定的根来分解成更小阶的问题。故先搞一个中间变量（三次预解式）可以更方便地求解问题</font>
- **（引理4.10）四次方程的三次预解式**：设 $K$ 是域，$f = x^4 + bx^3 + cx^2 + dx + e\in K[x]$
  - 则 $f$ 的三次预解式为 $x^3 - cx^2 + (bd-4e)x - b^2e + 4ce - d^2\in K[x]$
  - **证明**：
    - 由韦达定理，将系数 $b,c,d,e$ 表为根 $u_1,u_2,u_3,u_4$ 的（对称）多项式
    - 由预解式定义，将 $\a,\b,\g$ 表为根 $u_1,u_2,u_3,u_4$ 的（非对称）多项式
    - 然后就变成了，求某个多项式如何用对称多项式组合得到（高代方法即可）
- **（命题4.11）四次方程的伽罗瓦群**：设 $K$ 是域，$f\in K[x]$ 是既约可分 $4$ 次多项式，伽罗瓦群为 $G$
  - 设 $\a,\b,\g$ 是 $f$ 的三次预解式，$m = [K(\a,\b,\g):K]$
  - 则
    - $m = 6 \LR G = S_4$
    - $m = 3 \LR G = A_4$
    - $m = 1 \LR G = V$
    - $m = 2 \LR G = D_4$ 或 $G\cong \Z_4$
      - 若 $f$ 在 $K(\a,\b,\g)$ 上既约，则 $G\cong D_4$，否则 $G\cong \Z_4$
  - **证明（等价性）**：
    - 由伽罗瓦扩张得 $m$ 只可能是 $1,2,3,6$，前面已证所有必要性，故只需充分性即可
    - 已知 $m = |G/(G\cap V)|$
    - 若 $G = A_4$，则 $G\cap V = V$，故 $m = \dfrac{|G|}{|V|} = 3$
    - 若 $G = S_4$，则同理 $m = \dfrac{|G|}{|V|} = 6$
    - 若 $G \cong D_4$，由于 $V$ 含于 $S_4$ 的所有西罗 $2$ 子群，故 $G/V = V$，从而 $m = \dfrac{|G|}{|V|} = 2$
    - 若 $G\cong \Z_4$，则 $|G\cap V| = 2$，从而 $m = \dfrac{|G|}{|G\cap V|} = 2$
  - **证明（分类）**
    - 由于 $K$ 是完美域，故 $f$ 在 $K(\a,\b,\g)$ 上要么既约，要么可分
    - 设 $u_1,...,u_4$ 是是分裂域 $F$ 上的根
    - 若 $G\cong D_4$，则 $G\cap V = V$。再由 $V$ 是传递子群，$G\cap V = \aut_{K(\a,\b,\g)} F$
      - 从而对每个 $i\neq j$，都存在 $K(\a,\b,\g)$ 同构 $\sigma:K(\a,\b,\g)(u_i) \cong K(\a,\b,\g)(u_j)，u_i\mapsto u_j$，从而 $u_i,u_j$ 是同一个 $K(\a,\b,\g)[x]$ 上既约多项式的根，从而 $f$ 在 $K(\a,\b,\g)$ 上既约
    - 若 $G\cong \Z_4$，则 $G\cap V = \aut_{K(\a,\b,\g)}$ 是 $2$ 阶非传递子群
      - 易得不存在上面的同构，从而 $f$ 在 $K(\a,\b,\g)$ 上不是既约的（是可分的）
        - <font color='chartreuse'>由于 $F$ 是 $K(\a,\b,\g)(u_i)$ 的分裂域，若存在同构，则其为 $\aut_{K(\a,\b,\g)}F$ 的限制。再由于 $u_i,u_j$ 不可能是同一既约多项式的根，即得不存在同构</font>
  - **实例**：
    - $f(x) = x^4+4x^2+2\in \Q[x]$
      - **判断既约可分性**：由艾氏判别法，其既约。由 $\char \Q = 0$，其可分
      - **求三次预解式**：$(x-4)(x^2-8)$，故 $\Q(\a,\b,\g) = \Q(\sqrt{2})$ 是2次伽罗瓦扩张
        - 由定理，伽罗瓦群为 $D_4$ 或 $\Z_4$
      - **判断伽罗瓦群**：再由于 $f$ 的变量可写成 $x^2$，简化后得 $f$ 在 $\Q(\sqrt{2})$ 上不既约，从而 $\Gal(\Q[\sqrt{2}]/\Q) = \Z_4$
- **（定理4.12）素数次多项式的伽罗瓦群**：设 $p$ 是素数，$f$ 是 $\Q$ 上的 $p$ 次既约多项式
  - 若 $f$ 在 $\C$ 上有两个非实数根，则 $G\cong S_p$
  - **证明**：
    - 已知 $p$ 整除 $|G|$，则由柯西阶定理，存在 $p$ 阶元素 $\sigma$，其只能是为 $p$ 轮换，设为 $(aj_2...j_p)$，则 $\sigma^k = (abi_3...i_p)$
    - 已知共轭变换下实数根不动，非实根两两交换，即 $\exists\tau = (ab)\in G$
    - 不妨设 $\tau = (12)，\sigma^k = (123...p)$。已知它们可生成 $S_p$，再由 $G\leq S_p$，即得两者同构
  - **实例（求伽罗瓦群）**：
    - $f = x^5 - 4x + 2\in \Q[x]$ 只有三个实根。由有理多项式的艾氏判别法，其既约，从而伽罗瓦群为 $S_5$

### 习题

- **不可分多项式的分裂域**：
  - 设
    - $f\in K[x]$ 在 $F$ 上分裂，分裂形式为 $f = \prod\limits^k_{i=1} (x-u_i)^{n_i}$
    - $v_0,...,v_k$ 是 $g = \prod\limits^k_{i=1} (x-u_i)$ 的系数
    - $E = K(v_0,...,v_k)$
  - 则
    - $F$ 是 $g$ 在 $E$ 上的分裂域
    - $F\geq E$ 是伽罗瓦扩张
    - $\aut_E F = \aut_K F$
  - **证明**：
- 设 $K\leq \R$，$F$ 是既约的 $f\in K[x]$ 的分裂域，$\deg f = 3$，$D$ 是 $f$ 的判别式，则
  - $D>0 \LR f$ 有三个实根
  - $D<0 \LR f$ 只有一个实根
- 设 $f\in K[x]$ 是可分三次多项式，伽罗瓦群为 $S_3$，分裂域 $F$ 上的根为 $u_1,u_2,u_3$
  - 则中间域为 $K，K(u_1)，K(u_2)，K(u_3)，K(\D)，F$
  - 对应的伽罗瓦群为 $S_3，T_1，T_2，T_3，A_3，1$
    - <font color='chartreuse'>其中 $T_i = \{(1),(jk)\mid j\neq i\neq k\}$</font>

- **求（本原单位根多项式 $\a$）的次数**
    - 设 $\z$ 是 $n$ 次本原单位根，$\a = \z^2+\z^3$
    - 由上面结论，$[\Q(\a):\Q] = \dfrac{[\Q(\z):\Q]}{|\Gal(\Q(\a)/\Q)_\z|} = \dfrac{n-1}{|H|}$，分子是本原单位根次数减1，分母是 $\a$ 的稳定子群阶数
- **求（本原单位根多项式 $\a$）的极小多项式**：
  - **求扩张维数** $[\Q(\a):\Q] = \dfrac{n-1}{|H|}$
    - **求稳定子群**：对 $1\leq i\leq n-1$，分别试探 $\sigma(\z) = \z^i$ 是否对 $\a$ 不动，即得 $H$（实际上不动的 $i$ 就是 $\a$ 中各项的次数）
    - <font color='chartreuse'>设扩张次数 = 伽罗瓦群阶数 = $m$</font>
  - **求置换轨道**：再任选（某个可动的 $i$ 对应的 $\sigma$）对 $\a = \a_1$ 作用 $m-1$ 次，即可得 $\a$ 在根置换下的的轨道 $\a_2,...,\a_{m-1}$
    - <font color='chartreuse'>此时伽罗瓦群是循环群 $\Z_m$，已知可动的 $\sigma$ 是伽罗瓦群元素，故不断取其幂，即可得极小多项式的所有根（$\Q(\a)$ 的所有基）</font>
  - **求极小多项式**：极小多项式即为 $\prod\limits^{m}_{i=1} (x-\a_i)$，用韦达定理可得其各项系数


### 例子

- 若 $\char K \neq 2,3$，则三次多项式的判别式为 $b^2(c^2-4bd) + 18bcd$
- 若 $\char K \neq 2，f\in K[x]$ 是三次方程，且判别式在 $K$ 上是完全平方式，则 $f$ 在 $K$ 上要么既约，要么分裂
- 对任意域 $K$，$x^3-3x+1$ 在 $K$ 上要么既约，要么分裂
- $S_4$ 不是6阶传递子群

## 伽罗瓦群的计算举例

- 上面给出了 $4$ 次以下多项式的计算通法。$4$ 次以上的群也可按照类似的推导，只不过比较麻烦
- 但实际应用中，按照给定群的结构，往往可以做适当的简化。这里举几个例子

### 循环扩张情况（本原单位根）

- **求本原单位根的次数**：设 $\z$ 是 $7$ 次本原单位根，即 $\z = e^{\dfrac{2\pi i}{7}}$
  - **找分裂域**：由本原单位根定义易得 $\Q[\z]$ 是 $x^7-1$ 的分裂域
  - **找基**：$\z$ 的极小多项式为 $d(x) = x^6 + ... + 1$，即 $\Q[\z]$ 是 $6$ 次代数扩张，基为 $\{1_Q, \z, ..., \z^6\}$
  - **判伽罗瓦性**：再由 $d(x)$ 可分，得 $\Q[\z]$ 是伽罗瓦扩张
  - **求伽罗瓦群**：
    - 对 $\forall \sigma\in \aut_\Q \Q[\z]$，由基的结构，可设 $\sigma(\z) = \z^i$，此时通过置换同构（作用同构），可将 $\sigma$ 用 $i$ 表示：$\Gal(\Q[\z]/\Q)\to \Z_7，\sigma\mapsto i$
- **求极小多项式**：
  - 取 $i = 3$，则由于 $3$ 是 $\Z_7$ 的原根，故 $\lang \sigma \rang = \Gal(\Q[\z]/\Q)$
  - 易得 $\sigma^3\z = \ol\z$ 是共轭变换，故 $\z + \ol\z \in\R$ 是 $\sigma^3$ 的不动点，从而也是 $\lang \sigma^3 \rang$ 的不动点
  - 则 $\Q[\z] \geq \Q[\z]^{\braket{\sigma^3}} \geq \Q[\z + \ol\z] \neq \Q$
    - <font color='chartreuse'>第一个是固定域缩小性，第二个是因为 $\sigma^3$ 的不动点可能不止有 $\z + \ol\z$，第三个是因为 $\z+\ol\z \notin \Q$</font>
    - $\Q[\z] \geq \Q$ 是 $6$ 次域扩张，$[\Q[\z]^{\braket{\sigma^3}}:\Q] = \dfrac{[\Q[\z]:\Q]}{|\braket{\sigma^3}|} = $，即 $\Q[\z + \ol\z]$ 也是伽罗瓦扩张，伽罗瓦群为 $\braket{\sigma}/\braket{\sigma^3}$
  - 再设 $\a_1 = \z+\ol \z，\a_2 = \sigma(\a_1)，\a_3 = \sigma(\a_2),...$ 是 $\z+\ol\z$ 的轨道，容易计算出全部元素
    - 则 $\z+\ol\z$ 的极小多项式为 $\prod (x-\a_i)$，由韦达定理即可计算
- **求固定域**：
  - 易得 $\b = \z + \z^2 + \z^4$ 是 $\sigma^2$ 的不动点，即为 $\braket{\sigma^2}$ 的不动点  
  - 故 $\Q[\z]$ 中和伽罗瓦子群 $\braket{\sigma^2}$ 对应的子域（固定域）可由 $\b$ 生成
  - 易得其轨道只有 $\b' = \sigma(\b)$
  - 再由于 $(\b-\b')^2 = -\sqrt{7}$，而固定域至多为 $\Q$ 的一次扩张，所以 $\b,\b'$ 的多项式组合也是不动点，从而 $\braket{\sigma^2}$ 的固定域为 $\Q[\sqrt{-7}]$
