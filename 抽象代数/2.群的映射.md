# 群在映射下的结构和性质

## 群的映射结构

### 像与核

- 在同态映射下，由乘法性质易得
  - $f(e) = e$
  - $f(g^{-1}) = f(g)^{-1}$
  - 核是子群
  - 子群的像还是子群
  - 同像等价类同余于 $\ker f\red\Rightarrow$ 单同态的核仅有 $e$
  - **广义逆定理**：$f^{-1}(f(K)) = K \LR \ker f< K$
    - 必要性：K必须包含 $e$ 的全部原像，否则逆映射的像大于K
    - 充分性：讨论即可
- 幺半群同态不一定成立 $f(e) = e$
  - **反例**：至今未得

### 商群

- **G对正规子群的商群**：$\overline{G} = G/N = \{Na\mid a\in G\}$（**该正规子群陪集的集族**）
  - **运算法则**：$(aN)(bN) = (ab)N$
  - **性质**：
    - **不相交性**：若 $Na = Nb$，则 $a=b$
      - **证明**：
        - 若 $a或b\in N$，则 $Na = N$，故不妨设 $a,b\notin N$
        - 若 $na = nb$，则由可逆性得 $a=b$
        - 若 $n_1a = n_2b$，则 $n_2^{-1}n_1 = a^{-1}b$，与 $a,b\notin N$ 矛盾
    - **剔除性**：$G/N \cong Im$
      - 其中 $Im = \{\bar a\mid a\in G，\bar a = \{g\in G\mid Ng = Na\}\}$
      - **证明**：
        - 只需 $\forall a\in G$，$f:\dkh{G/N} \to Im，N\bar{a} \mapsto \bar{a}$ 是双射且满足同构式即可（其中 $\bar a = \{g\in G\mid Ng = Na\}$）
        - 单射性：反设 $\exists \bar a_1\neq \bar a_2$，使得 $f(N\bar a_1) = f(N\bar a_2)$
          - 定义易得矛盾
        - 满射性：反设 $\exists g\in G$ 使得 $\forall a\in G，f(N\bar a) \neq \bar g$
          - 则 $\bar a\neq g$，也即 $Na \neq Ng(\forall a\in G)$，这不可能
        - 同构式：$f(N\bar a)f(N\bar b) = \bar a\bar b = Im = f(N\bar a\bar b) = f(N\bar aN\bar b)$
      - <font color='chartreuse'>我是sb，陪集等价就很显然的东西，我硬是要写证明</font>
  - **本质**：以陪集为元素构成的群，阶为 $[G:N]$
    - **平移性**：由陪集平移性，可看作N的所有不相交平移，a可看作平移的距离
    - **同态压缩性**：原来的 $N$ 在商同态下被压缩为 $e$，而其它元素变为平移距离 $\bar{a}$
    - **逆积性**：商群是直积的逆运算
  - **实例**：
    - **循环群**：$G = \Z_{10}，H = \{0,5\}$，则 $G/H \cong \Z_5$（五阶循环群）
      - $G/H = \set{H,H+1,H+2,H+3,H+4} = \set{\{0,5\},\{1,6\},\{2,7\},\{3,8\},\{4,9\}}$
    - **同余类（p阶循环群）**：$\Z/p\Z$：
      - 整数加法群：$\Z = \{0,1,-1,\cdots\}$
      - 空隙群：$p\Z = \{0,p,-p,\cdots\}$
      - 商群（等价类）（同余类）：$\Z/p\Z = \set{p\Z,\cdots,p\Z+(p-1)} \\ = \set{\{0,p,-p,\cdots\} , \cdots, \{p-1,2p-1,-1,\cdots\}} \color{chartreuse} \equiv \set{1,2,3,\cdots,p-1}\pmod p$
    - **一般线性群**：
- **商映射（群投影）**：$\pi: G\to G/N$
  - **理解**：将群中同个陪集 $Na$ 内元素压缩为陪集 $Na$（将等价类 $Na$ 看作一个元 $a$）
  - **实例**：$\pi: \Z\to\Z/p\Z，g \mapsto \bar g\pmod p$

### 同态、同构定理

- **同态基本定理**：在同态映射 $f:G\to H$ 中，像 $\subset$ H的子群，核 $\subset$ G的正规子群
  - **证明**：
    - 像：同态乘积性（群） + 映射缩小性（子群）
    - 核：（Artin）同态乘积性 + 幺元素交换性（正规）
  - **推论**：<font color='chartreuse'>也可以尝试先看看[群与范畴](./3.群与范畴.md)中的交换图，对于“规范性”的理解会更加透彻</font>
    - **规范同态** $\tilde{f}: G/\ker f\to H$<font color='chartreuse'>（是已知同态 $f$ 的衍生商同态，且不一定是满射）</font>
      - **规范性**：对每个 $N\lhd G$，均存在唯一的同态映射 $\tilde f: G/N \to H$
        - **存在性**：易得存在同态 $f$ 满足 $\ker f = N$
        - **唯一性**：$\tilde f\pi = f$，已知 $\pi,f$ 均依赖于 $N$，得其唯一性，再传递到 $\tilde{f}$
      - **性质传递**：$f$ 的单射、满射性可传递到 $\wt f$ 上
        - **单同态条件**：$\ker f = \{e\}$（此时商群为G）
        - **满同态条件**：$\Im f = G$
    - **第一同构定理**：$\bar{f}: G/\ker f \to Im\ f$ 是规范同构
      - **证明**：
        - 满射性：衍生映射缩小性 + 衍生映射像集为 $Im\ f$
        - 单射性：已知 $\ker \tilde{f} = (\ker f)/N$，因此其当 $\ker f= N$ 时为 $\{e\}$
      - **本质**：像与核的对偶关系<font color='chartreuse'>（关于对偶的理论在[模对偶关系](./9.模对偶关系.md)中有更详细讨论）（一般线性群中，正规子群是特殊线性群，商群是行列式值相同的矩阵。$\ol f$ 是行列式映射，像与核是转置正交补关系）</font>
    - **同态的商群传递性**：设 $f:G\to H$ 是同态
      - 若 $f(N)\leq M$，则 $g:G/N\to H/M$ 也是同态
      - **证明**：
        - 令 $g(aN) = f(a)M$ 即可
        - <font color='chartreuse'>同态式：$g(aNbN) = g(abN) = f(ab)N = f(a)f(b)N = f(a)Nf(b)N = g(aN)g(bN)$</font>
      - **推论（同构条件）**：当 $f(N) = M$ 且 $\ker f \subset N$ 时同构
        - **证明**：
          - $g$ 可看作复合映射 $\pi \tilde{f}$
          - 再由规范同构性质，得前者是满同态条件，后者是单同态条件
    - **第二同构定理**：商群的集合运算
      - 若 $N\lhd G，K\leq G$
      - 则 $K/(K\cap N) \cong (NK)/N$
      - **证明（Hungerford）**：
        - 设 $f:K\to (NK)/N$，由第一同构定理，只需 $\Im f = (NK)/N$，即可得 $\wt f:K/(K\cap N)\to (NK)/N$ 是同构
        - 由正规性 $\forall nkN = kn'N = kN = f(k)$，由任意性即得结论
      - **证明（有问题）**：设 $f:K\to NK，k\mapsto nk$ 为同态
        - 此时 $f(\ker f) = N\cdot \ker f = e$，即 $\ker f \subset N^{-1} = N$
        - 再由 $f$ 定义域为 $K$，从而 $\ker f = K\cap N$
        - 故此时 $\wt f$ 为 $\Big(K/\ker f = K/(K\cap N)\Big)\to NK$
          - 再由 $f(K\cap N) = N(K\cap N) = N$（？？）
          - 即 $g = \pi\wt f$ 满足商群同构条件，它就是题设的同构关系
        - <font color='chartreuse'>问题在于 $f$ 取错了，应该取的和Hungerford一样，但那样就证明手法也一样了……</font>
      - **理解**：右边的很大一部分陪集平移都是在 $N$ 在 $N$ 内部的无意义平移，真正有意义的就是N和K相交的那一部分在K中的移动而已。而把右边的K和N交叠部分剥离出来，就是左边
      - **本质**：商群的非直积运算
    - **第三同构定理**：商群的复合运算
      - 若 $N_1,N_2\lhd G，N_1>N_2$
      - 则
        - $N_1/N_2\lhd G/N_2$
        - $(G/N_2)/(N_1/N_2) \cong G/N_1$
      - **证明（Hungerford）**：
        - 正规子群性由交换性易得
        - 同构函数 $f: G/N_2\to G/N_1$
      - **理解**：正规子群的逆遗传性
  - **实例**
    - 同余类加法群
      - **同态（同余类代表元映射）**：满同态（同余类组合法则），不是单同态（$\overline{e}$）
      - **正规子群**：同余类本身
      - **商群**：代表完系，因此规范同构是恒等映射，规范单同态是满射
    - 可逆矩阵乘法群
      - **同态（行列式映射）**：满同态（行列式矩阵乘法），不是单同态（偶置换）
      - **正规子群（特殊线性群）**：$SL_n(C) \lhd GL_n(C)$
      - **商群**：行列式值为 $\N^+$ 的矩阵，规范同态是行列式映射（同构）
- **保子群定理**：
  - 令 $S(H)$ 表示某个群 $H$ 的子群集族
  - 若 $K\leq G$，$f:G\to H$ 是满同态
  - 则将子群映射为子群的衍生映射 $\varphi: K\mapsto f(K)$ 满足
    - 双射性
    - 像集与原像集为 $\ker f\cup S(G)\to S(H)$
    - 正规子群一一对应
  - **证明**：反证法 + “分”析法即可
- **N和G的中间群**：$\mu = \{K\mid N\leq K\leq G\}$
- **中间群定理**：若 $N<K<G$，则 $G/N$ 的子群均可表为 $K/N$
  - **证明**：对投影应用保子群定理
  - **推论**：
    - **正规子群的商群传递**：$K\lhd G\LR K/N\lhd G/N$

### 习题

- **正规化子传递性**：若 $N\lhd G，N\leq M\leq G$，则 $N_G(M)/N = N_{G/N}(M/N)$
  - **证明**：
    - **充分性**：已知 $x^{-1}(m_1N)x = (m_2N)，(x\in G/N)$，则可设 $x = yN$
      - 此时证 $y\in N_G(M)$ 即可
      - 已知条件转化为 $(yN)^{-1}(m_1N)(yN) = (y^{-1}m_1y)N = (m_2N)$
      - 也即 $y^{-1}m_1y = m_2$（**证毕**）
    - **必要性**：已知 $x^{-1}m_1x = m_2\pad (x\in G)$
      - 则此时 $(xN)^{-1}(m_1N)(xN) = N(x^{-1}m_1Nx)N$
      - 再由 $N$ 正规性，可化为 $N(x^{-1}m_1x)N = Nm_2N = m_2N$
      - 从而 $xN\in G/N$ 是 $M/N$ 的正规化子（**证毕**）
- **逆映射扩大性**：若 $f:G\to H$ 是同态，$M\leq G$，则 $f^{-1}\circ f(M) = \ker f\cdot M$
  - **证明**：易得
  - **推论**：若同态 $f:G\to H$ 中，$K<G$，则
    - $f^{-1}\circ f(K) = K\LR \ker f<K$
    - $f^{-1}\circ f(K) = K\cdot\ker f$
    - **证明**：逆映射扩大性直得
- **商群与阶**：
  - 设 $N\lhd G$，若 $(|g|,|G/N|) = 1$，则 $g\in N$
    - **证明**：
      - <font color='chartreuse'>反设 $g\notin N$（可有可无）</font>，取同态 $f:G\to H$，其中 $\ker f = N$
      - 则 $f(gN) = f(g)$，由同态阶缩小性，右式的阶整除 $|g|$，左式的阶整除 $|G/N|$，但由互素得只能是两边阶均为 $1$，但一阶元素只有 $e$，也即只能是 $g\in N$
    - **理解**：对于商群来说，用同态的语言去思考是一种重要的方法。这也是要提出同态基本定理和规范同态概念的原因
      - 可以说商群就是同态，同态就是商群。它们是一个问题的两种视角，如同线性变换和矩阵是也是同一概念的两种视角一样。

- **商群运算（第三同构定理推论）**：
  - 若 $N_1\lhd G_1，N_2\lhd G_2$，则
  - $(N_1\times N_2)\lhd (G_1\times G_2)$
  - $(G_1\times G_2)(N_1\times N_2) \cong (G_1/N_1)(G_2/N_2)$
  - **证明**：积运算定义即可
  - **本质**：商与直积是完全兼容的运算
- **弱直和引理**：设 $N\lhd G，K\lhd G$，若 $N\cap K = \lang e \rang，N\lor K = G$，则 $G/N\cong K$
  - **证明**：详见群与范畴的[弱直和定理](./3.群与范畴.md)
- **阿贝尔同态**：若同态 $f:G\to H$ 中，$H$ 是阿贝尔群，则任意子群 $N\supset \ker f$ 都是正规子群
  - **证明**：归纳法，只需证 $N = \lang \ker f,a \rang$ 即可
    - 即只需 $\forall g\in G，g^{-1}ag \in \lang \ker f,a \rang$ 即可
    - 此时 $f(g^{-1}ag) = f(g^{-1})f(a)f(g)$，由于像是阿贝尔群，可化为 $f(a)$，从而 $g^{-1}ag \in a\cdot \ker f$（**证毕**）
  - **理解**：交换与共轭的奇妙产物
- **有理加法商群**：$Q/Z = \set{n\cdot\dfrac{p}{q}\mid p,q,n\in Z，(p,q) = 1}$
  - **本质**：相当于把 $R$ 上的 $Q$ 简化成了 $[0,1)$ 上的 $Q$
  - $Z(p^\infty) = \set{a/b\in Q/Z \mid a,b\in Z，b = p^i}$（详见[素幂阶子群](4.阿贝尔群的结构)）
    - 加法无限循环群
    - 生成元 $\lang \ol{1/p^n} \rang$

## 置换群

- <font color='chartreuse'>本章内容在[群作用](5.有限群的结构.md)中可以看得更清楚</font>
- **置换** $\sigma$：集合 $D$ 到自身的一一映射
  - **字典**：集合 $D$
  - **文字**：集合 $D$ 的元素
  - **矩阵表示法**：$\tvek{1 & 2 & 3}{a_1 & a_2 & a_3}$，表示第1,2,3个元素被置换为第 $a_1,a_2,a_3$ 个元素
- **对称群** $S(D)$ 或 $S_n$：字典 $D$ 上所有的置换构成的乘法群，阶为 $n!$
  - **同构性**：对称群是循环群，从而 $n$ 个文字的对称群同构于 $n$ 阶群？（组合数学的锅）
  - **证明**：$a_i$ 和 $p_i = \tvek{a_1 & a_2 & \cdots & a_n}{a_ia_1 & a_ia_2 & \cdots & a_ia_n}$ 一一对应
    - 保乘法性易得
  - **实例**：$S_3 = \set{(1),(12),(13),(23),(123),(132)}$
    - <font color='chartreuse'>$3$ 个文字的排列方法共有 $A^3_3 = 3!$ 种，也就是说，把 $123$ 置换后的不同结果有 $3!$ 中，从而 $|S_3| = 3!$</font>
- **置换群 $A(D)$**：对称群的子群

### 轮换

- **长为 $t$ 的轮换**：$(a_1a_2...a_t)$，其中 $a_i\to a_{i+1}$<font color='chartreuse'>（轮换中元素均左移一格，其它元素不变）</font>
  - **轮换的运算**：从左到右
    - **相等**：若两个轮换中元素彼此位置均相同，则两轮换相等
      - **实例**：$(123) = (231) = (312)$，长为3的轮换只有两种，且互逆，阶为 $3$
    - **数量定理**：阶为 $n$ 的字典中，长为 $m$ 的不同轮换共有 $A^{m-1}_{n-1}$ 个
    - **阶定理**：$S_n$ 中，长为 $m$ 的轮换阶为 $m$
    - **无交换性**
      - **反例**：
        - $1234 \xto{(1234)} 2341 \xto{(1342)} 1423$
        - $1234\xto{(1342)}3142\xto{(1234)}4213$
    - **逆运算**：$(a_1a_2...a_t)^{-1} = (a_t...a_2a_1)$
  - **对换**：长为2的轮换
    - **轮换分解**：置换是轮换的乘法
      - **算法**：分离轨道等价类，依次讨论即可
      - **实例**：$\tvek{123456}{132564} = (1)(23)(456)$
        - <font color='chartreuse'>有三个轨道，$\{1\},\{23\},\{456\}$</font>
    - **对换分解**：轮换是对换的乘法
      - **通用公式**：
        - $(123) = (12)(13)$
        - $(1234) = (12)(34)(13)$
        - $(12345) = (12)(34)(13)(15)$
        - $(123456) = (12)(34)(56)(13)(15)$
        - $(a_1a_2...a_t)$
          - $t$ 为奇数时
            - 首先是 $(a_1a_2),(a_3a_4),\cdots,(a_{t-2}a_{t-1})$
            - 然后是 $(a_1a_3),(a_1a_5),\cdots,(a_1)(a_{t-2})$
            - 最后补上 $(a_1a_t)$
          - $t$ 为偶数时
            - 首先是 $(a_1a_2),(a_3a_4),\cdots,(a_{t-1}a_t)$
            - 然后是 $(a_1a_3),(a_1a_5),\cdots,(a_1)(a_{t-1})$
        - **理解**：第一步先解决一个数的位置（偶数序号的数），第二步再解决另一个数（奇数序号的数）的遗留问题（类似容斥原理的算法）
      - **实例**：
        - $(1342) = (13)(42)(14)$
          - $1234\xto{(1342)}4312$
          - $1234\xto{(13)}3214\xto{(42)}3412\xto{(14)}4312$
        - $(1423) = (14)(23)(12)$
          - $1234\xto{(1423)}4312$
          - $1234\xto{(14)}4231\xto{(23)}4321\xto{(12)}4312$

      - 设映射 $f: D\to G，a_1^ib_1^j\mapsto a^ib^j$，易得其为满射
      - 证其为单射，即证明 $\ker f = \{e\}$
        - 若 $a^ib = e$，则 $a^i = b$，从而 $a^{i+1} = e$，则 $a^2 = e$，与 $n\geq 3$ 矛盾
        - 若 $a^i = e$，则 $i=n\equiv 0$，得证
  - **矩阵表示**：$D_4$ 为由四阶元素 $C = \pvec{0 & 1 \\ -1 & 0}$ 和二阶元素 $D = \pvec{0 & 1 \\ 1 & 0}$ 生成的 8 阶非阿贝尔群
  - **一般表示**：$D_4 = \{e,a,a^2,a^3，b，ab,a^2b,a^3b\}$
    - **封闭性**：$ab = ba^3，a^2b = ba，a^3b = ba^2$
  - **置换表示**：$D_4 = \{(1),(1234),(13)(24),(1432),(24)(13),(14),(123),(1342)\}$

    - **绕中心旋转**：$a = (123...n)$，其中 $a^k$ 为 $\dfrac{2k\pi}{n}$ 度旋转
