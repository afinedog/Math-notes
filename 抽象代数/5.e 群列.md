# 群列

## 有限群的分解

- **三个条件**：体现有限群 $G$ 能分解的程度
  - **西罗直积条件**：$G$ 是西罗子群的直积
  - **西罗分解条件**：对 $\forall m\Bigm| |G|$，$G$ 都存在 $m$ 阶子群
  - **西罗存在条件**：若 $G = mn\pad ((m,n) = 1)$，则 $G$ 存在 $m$ 阶子群
- **关系**：$1\subset 2\subset 3$
  - **证明**：若 $G = \prod\limits^n_{i=1} P_i$，设 $|P_i| = p_i^{\a_i}$，则 $|G| = \prod\limits^n_{i=1} p_i^{\a_i}$
    - $(1)\to (2)$：第一西罗定理直得条件2
    - $(1)\to (3)$：西罗子群定义直得条件3
    - $(2)\to (3)$：废话
  - **本质**：差不多在习题里被扒干净了。另外，它其实类似于有限阿贝尔群的推导
- **例子**：
  - $p$ 群、有限阿贝尔群满足1
  - $S_3$ 满足2，不满足1
  - $A_4$ 满足3，不满足2
  

### 幂零群

- **次中心**：$C_2(G) = f_2^{-1}\Big(C(f_2(G))\Big)$，其中 $f_2:G\to G/C(G)$
  - **保正规性**：$C_2(G)\lhd G$
    - **证明**：由同态下正规性不变直得
  - **延拓性**：$C_2(G) \geq C(G)$
    - **证明**：易得 $g\in C_2(G) \LR gChC = ghC = hgC = hCgC$
      - 易得 $C$ 中元素符合条件，而 $G/C$ 中的其它可交换元素也符合条件，故只增不减
    - **理解**：$C_2(G)$ 和 $G$ 在商合并作用后是中心关系，$C(G)$ 和 $G$ 在原像是中心关系
    - **本质**：商映射保中心性 + 逆映射扩大性
  - **交换性**：$C_2(G)$ 是阿贝尔群
    - **证明**：只需证明中心以外的部分交换
      - 由定义，$gC$ 是 $G/C$ 的中心，故 $(g_1C)(g_2C) = (g_2C)(g_1C)$
      - 再由中心交换性，$(g_1C)(g_2C) = g_1g_2C = (g_2C)(g_1C) = g_2g_1C$，取 $e\in C$ 即得结论
    - **推论**：$C_{i+1}/C_{i}$ 是阿贝尔群

- **升中心列**：$\lang e \rang \leq C_1(G) \leq C_2(G) \leq \cdots$<font color='chartreuse'>（以后简称为 $C_i$）</font>
  - $f_i:G\to G/C_{i-1}$
  - $C_1 = f_1^{-1}\Big(C(G/e)\Big) = C(G)$
  - $C_{i+1}(G) = f^{-1}_{i+1}\Big(C(G/{C_{i}})\Big)$
  - **本质**：商可看作剔除，也就是说，每次迭代都会找出（除去当前中心后可交换的元素），称为 $n$ **次可交换元素**。也就是说，这是一个定量衡量可交换性的过程
    - 比如 $C_2$ 就是 $G$ 商去 $C_1$ 后的中心
- **次中心的正规化子**：$\forall 0\leq i\leq n，N_G(C_i) = G$
  - **证明**：已知 $C_i \lhd G$，故由定义直得结论
  - **理解**：
- **幂零群**：$\exists n$ 使得 $C_n(G) = G$
  - **本质**：
  - **实例**：阿贝尔群均为幂零群：$G = C(G) = C_1(G) = \cdots$
  - **反例**：中心平凡的群均为非幂零群：$C(S_n) = \lang e \rang$
- **（定理7.2）$p$ 群幂零性**：有限 $p$ 群是幂零群
  - **证明**：
    - 由商群传递性，此时非平凡商群均为 $p$ 群，再由 $p$ 群中心非平凡性得总存在非平凡中心
    - 再由第一西罗定理，可设 $|G| = p^n，C(G) = p^k$，则 $|G/C(G)| = p^{n-k}$，同理可设 $|C\Big(G/C(G)\Big)| = p^{n-k-m}$，即 $|C_2(G)| = p^{n-m}$，从而可不断归纳
      - 也就是说，若 $G \neq C_i(G)$，则 $C_i(G) \gvertneqq C_{i+1}(G)$。再由 $G$ 有限即得结论
  - **本质**：只要保持性质不变，有限群总能被有限次剔除掉所有次交换元素，从而为幂零群
- **（定理7.3）有限直积传递性**：有限个幂零群的直积幂零
  - **证明**：只需证明2阶情况即可
    - 设 $G = H\times K$，则易得 $C_i(G) = C_i(H)\times C_i(K)$
    - 设 $\pi_H,\pi_K$ 是 $H\to H/C_i(H)$
      - 则满同态 $\p:G\to G/C_i(G)$ 具有以下分解：
      - $G\xto{\pi_H\times \pi_K} H/C_i\times K/C_i \xto{\psi} \cfrac{H\times K}{C_i(H)\times C_i(K)} = G/C_i(G)$
      - 从而 $C_{i+1}(G) = C_{i+1}(H)\times C_{i+1}(K)$
    - 易得幂零性可通过此式传递
  - **本质**：映射乘积性 + 商群直积互逆性
- **（引理7.4）幂零群的正规化子条件**：幂零群 $G$ 的真子群 $H \lvertneqq N_G(H)$
  - **证明**：
    - 若 $H = C_i$，则 $N_G(H) = G$，结论显然。否则由幂零性可得存在 $n$ 使得 $C_n < H < C_{n+1}$
    - 可设 $a\in C_{n+1}，a\notin H$
      - 任取 $h\in H$
      - 由于 $C_na$ 是 $G/C_{n}$ 的中心，得 $C_nah = hC_na$
      - 再由限制正规性，$C_n\lhd H$，故 $hC_na = C_nha$
    - 因此 $\exists h'\in C_n$ 使得 $ah = h'ha$，从而 $aha^{-1} = h'h\in H$，即 $a\in N_G(H)$
      - 再由 $a\notin H$，即得 $H$ 为真子群
  - **理解**：由幂零性，存在 $C_n < H < C_{n+1}$，而 $C_{n+1}\leq N_G(H)$
  - **本质**：幂零群中，所有元素都具有一定程度的交换性。若子群 $H$ 最多存在 $n+1$ 次可交换元素，则其显然在 $n+1$ 次中心 $C_{n+1}$ 中正规，从而正规化子必定包含 $C_{n+1}$
    - <font color='chartreuse'>当 $G$ 有限时，它被强化为[wieland定理](w定理)</font>
- **（命题7.5）西罗直积性**：有限群 $G$ 是幂零群 $\LR G$ 满足条件1
  - **充分性**：已知有限西罗 $p$ 群是幂零群，且幂零性可有限直积传递，故结论易得
  - **必要性**：设 $P$ 是西罗 $p$ 子群
    <!-- - 若 $P\lvertneqq G$，则由幂零正规化子引理，$P\lvertneqq N_G(P)$ -->
    - 若 $C_n \lvertneqq P\lvertneqq C_{n+1}$，则 $G = N_G(C_n) \leq N_G \Big( N_G(P) \Big) = N_G(P) \leq G$
      - 即 $N_G(P) = G$，即 $P$ 是正规子群
      - 再由第二西罗定理，$P$ 也是唯一的西罗子群。
    - 若 $P = C_i$，则显然 $N_G(P) = G$，也可得 $P$ 唯一性
    - 设 $|G| = \prod\limits^k_{i=1} p_i^{n_i}$，则可设 $P_i$ 是对应的西罗 $p_i$ 真子群，易得其不相交，从而积为直积
  - **理解**：[西罗正规不变性](5.有限群的结构.md)由轨道唯一性反推出，西罗正规性由次中心夹逼推出，西罗唯一性由共轭封闭性推出，从而有限幂零群的每个素因子 $p_i$ 对应的西罗 $p_i$ 子群均唯一，即可直积分解
  - **本质**：夹逼法，两边都正规，则中间的 $P$ 也正规
    - 某种意义上讲，直积和交换性密切相关。把群割成独立的小块，每部分越小，越可能交换，从而每部分均可交换（整体可交换）的概率越大
    - 而反过来的结论我们早就得到了，即越交换的群越可能是直积（有限阿贝尔群均可直积分解成交换群）
      - 幂零群是次一级的交换性，其只能分解成西罗子群的直积
      - 而再次一级，没有交换性，只要满足正规链条件，也能分解成不可分群的直积
- **（推论7.6）西罗存在性**：有限幂零群满足条件3
  - **证明**：易得

### 习题

- **子群传递性**：幂零群的子群幂零
  - **证明**：设 $K<G$，只需 $C_i(K) =  K\cap C_i(G)$ 即可
    - 易得 $C(K) = K\cap C(G)$，我们将 $K$ 分为 $\begin{cases} K_1 = C_1 \\ K_2 = C_2-C_1 \\ K_3 = K-C_2 \end{cases}$ 三个部分
    - 易得 $G_2\cap K = K_2$，<font color='chartreuse'>$G_3\cap K = K_3$
    - 再易得 $K\cap \Big(G/C(G)\Big) = (K_2\cup K_3)C(K)$
    - 再由 $C_2(G) = f_2^{-1}(C(G/C_1))$，</font>即得 $C_2(K) = K\cap C_2(G)$
    - 再将 $K_3$ 分为三个部分，归纳即可
  - **理解**：看着乱，核心思想就是取限制，将性质传递，然后归纳着做。下面的动机远远比上面的手段重要
  - **本质**：幂零性本质上已经将 $G$ 通过次交换性分解，若再从里面取限制 $K$，显然只需要对每个分解取限制即可
    - 若 $K$ 最高只有 $n-k$ 级次交换元素，则最低 $C_{n-k}(K) = K$。幂零性比 $G$ 更强
- **同态传递性**：幂零群的同态像幂零
  - **证明**：设 $F:G\to H$ 是同态
    - 中心同态传递性得 $F(C_1(G)) = C_1(H)$
    - 商群同态传递性得 $F(G/C_1(G)) = H/C_1(H)$
    - 中心同态传递性得 $F(C(G/C_1(G))) = C(H/C_1(H))$
    - 乘积同态传递性得 $F(C_2(G)) = C_2(H)$
      - <font color='chartreuse'>反商群传递性的本质就是乘积传递性
      - 即 $\pi^{-1}(gC_1) = \set{gx\mid g\in L_{C(G)}，x\in C(G)}$
      - 从而 $F(\pi^{-1}(C(G/C_1(G)))) = \set{F(g)x\mid x\in C(H)} = C_2(H)$</font>
    - 归纳即可
  - **本质**：同态能传递充分性质，但不能传递等价性质。只有同构才能传递所有性质
    - 比如中心和商就是充分性质，是由 $G$ 的全局得到的局部性质，而不是由局部反推出全局性质
- **商群传递性**：幂零群的商群幂零
  - **证明**：商映射是同态，或商群同构于商代表元系（G的子群），都可得到结论
  - <font color='chartreuse'>错啦！这样岂不是（子群传递性）可直接导出（商群传递性）了吗，但这显然不可能。实际上，同构并不能传递所有性质</font>
- **正规逆传递条件**：设 $N\leq C_1$ 幂零，若 $G/N$ 幂零，则 $G$ 幂零
  - **证明**：易得 $C_n(G/N) = C_n(G)/N$
    - 再由商群幂零性得 $C_n(G/N) = G/N$，综合即得结论
  - **反例**：若改为和可解群一样的 $N\lhd G$，则结论不成立
    - 如 $A_3\lhd S_3$，但非幂零
- **阶与可换性**：有限幂零群中，若 $a^m = b^n = 1，(m,n) = 1$，则 $a,b$ 可换
  - **证明**：已知幂零群可分解为西罗子群直积
    - 易得此时 $(|a|,|b|) = 1$，故 $a,b$ 属于两个不同的西罗子群，从而独立，从而可换
- **分解视角下的正规子群**：有限幂零群 $G$ 中，任意非平凡正规子群 $N$ 满足 $N\cap C_1\neq \lang e \rang$
  - **证明**：由有限幂零性，可设 $C_n = G$，即 $N\cap C_n = N$
    - $\forall n\in N$，其共轭类 $\ol n\in C_n/C_{n-1} = C(G/C_{n-1})$，从而 $\forall g\in G，[n,g]\in N\cap C_{n-1}$
    - 反设 $N\cap C_{n-1} = \{e\}$，则 $n\in C(G)$
      - 由 $n$ 任意性即得 $N\leq C(G)$，从而 $N\cap C_{n-1} = N \neq \lang e \rang$
    - 反复进行即得结论
  - **推论**：设有限非交换幂零群 $G$ 中，$A$ 是最大正规子群。若 $A$ 是阿贝尔群，则 $C_G(A) = A$<a id="分解视角下的正规子群"></a>
    - **证明**：由 $A$ 在自身中可交换，得 $A\leq C_G(A)$
      - 要证 $C_G(A)\leq A$，首先证明其正规性
      - 任取 $g\in G，b\in C_G(A)，a\in A$
        - 由 $A$ 正规性，$g^{-1}ag\in A$，从而与 $b$ 可换
        - 则 $gbg^{-1}x = xgbg^{-1}$，从而 $b$ 在 $g$ 共轭下不变，由任意性即得 $C_G(A)\lhd G$
      - 由 $A$ 最大性，此时 $C_G(A) = G$ 或 $A$
        - 反设为 $G$，则 $A\leq C(G)$，由中心正规性，只能是 $C(G) = A$ 或 $G$。由 $G$ 非交换性得只能是 $A = C(G)$
        - 再由幂零性，$C_1<C_2$。再由 $C_2$ 正规性即得只能是 $C_2 = G$
      - 此时 $G/A = C_2/C_1$ 是阿贝尔群，由 $A$ 极大性得无非平凡子群，只能是素数阶循环群。但此时 $G = A\times Z_p$，是阿贝尔群，矛盾
- **Wieland定理**：有限群 $G$ 幂零 $\LR$ 最大真子群均正规<a id = "w定理"></a>
  - **证明**：
    - **必要性**：有限幂零性得 $G$ 是西罗子群的直积，故最大真子群均是西罗子群，直积因子当然是正规的
    - **充分性**：
    - 由[正规不变性](5.有限群的结构)的本质，若 $P$ 是 $G$ 的西罗 $p$ 子群，则 $N_G(P)$ 中每个子群的正规化子都是自身
  - **本质**：有限幂零群的最大真子群均为西罗子群（显然）

### 可解群

- **全不可换元素**：$ga\neq ag\quad(\forall g\neq a)$
- **交换子（换位子）**：$[a,b] = (ab)(ba)^{-1} =  aba^{-1}b^{-1}$
  - **本质**：不可交换元素的一个配对
- **换位子群**：$G' = \{\lang [a,b] \rang\mid a,b\in G\}$
  - **本质**：削去了 $G$ 生成元系中的（可换元素乘积 $ab = ba$）部分
    - <font color='chartreuse'>（可能有非交换子元素，如 $[a,b][b,c] = aba^{-1}cb^{-1}c^{-1}$）</font>
  - **性质**：
    - **换位性**：$ab = [a,b]ba$
    - **换位子展开公式**： $[ab,c] = a[b,c]a^{-1}[a,c]$
    - **换位子幂公式**：若 $[a,b]$ 和 $a,b$ 均可交换，则 $[a^m,b^n] = [a,b]^{mn}$
      - **证明**：由 $ab = [a,b]ba$ 得 $[a,b^n] = [a,b]^n$，再对 $m+n$ 用归纳法证明即可
  - **应用**：$|G'|$ 的大小表示 $G$ 不可交换的程度<a id="换位子群本质"></a>
    - <font color='chartreuse'>$a,b$ 可换 $\Leftrightarrow$ $a$ 是 $b$ 的共轭不动点 $\Leftrightarrow$ 换位子 $[a,b] = e$
      - 小推论：若 $a,b$ 可换，则它们的逆也可换
    - 换位子 $[a,b]$ 与 $a$ 或 $b$ 可换 $\Leftrightarrow$ $a$ 是 $b^{-1}ab$ 或 $b$ 是 $a^{-1}ba$ 的共轭不动点
      - 也即 $a$ 与 $b^{-1}ab$ 可换
    - 也就是说，换位子不一定是全不可换元素，但不断在换位子群中再取换位子群，最终不变的子群一定是全不可换元素生成的（但未必是所有全不可换元素，参考 $S_n$）
    - 换位子 $[a,b] = [a,c] \Leftrightarrow a$ 是 $c^{-1}b$ 的共轭不动点
      - 也即 $a$ 与 $c^{-1}b$ 可换
    - 换位子 $[b,a]$ 与 $[a,c]$ 可换 $\Leftrightarrow [a^{-1},b]$ 和 $[b,a]$ 关于 $cac^{-1}$ 共轭等价（没用）</font>
- **交换指示矩阵**：假设 $G$ 有 $n$ 个元素，其中第 $i,j$ 个元素是否可交换记为 $\d_{ij}$，则构成 $n\times n$ 对称矩阵（对角元均为1）
  - <font color='chartreuse'>乍一看，$|G'|$ 就是其中零元个数的一半，故换位子群的阶数最多为 $\dfrac{|G|^2}{2}-|G|$。但实际上这个数大于 $|G|$，因此没有意义</font>
- **（定理7.8）对偶性**：
  - 换位子群是正规子群
    - **证明**：设 $f\in Aut\ G$，则元素分析易得 $f(G') < G'$
      - 取 $f$ 是共轭自同构，即得 $G'\lhd G$
    - **理解**：换位子群以外的都是次可交换元素，从而 $G'$ 在 $G-G'$ 眼中是可整体交换的
  - $G/G'$ 是阿贝尔群
    - **证明**：由于 $(ab)(ba)^{-1} \in G'$，故 $abG' = baG'$，故 $G/G'$ 是阿贝尔群
  - 若 $N\lhd G$，则：$G/N$ 是阿贝尔群 $\LR G'< N$
    - **必要性**：上述过程反过来即得 $G'<N$
    - **充分性**：易得
    - **理解**：若只商去不可交换元素，剩下的就是最大的阿贝尔子群。也就是说，<font color='red'>换位子群是不可换元素构成的最小正规子群</font>
  - **实例**：$S_3/A_3 = \set{(1)A_n,(12)A_n}$
    - <font color='chartreuse'>由于置换只与自身可换，故 $S_n$ 的换位子群只能是 $n/2$ 阶的，也即 $A_n$
    - 还可得到一个推论：$S_n$ 不可解</font>
  - **总结**：$f:G\to H$ 的像是阿贝尔群 $\LR G'\subset\ker G$
- **导群**：$G^{(1)} = G'，G^{(n+1)} = \dkh{G^{(n)}}'$
  - **本质**：优不可交换性<font color='chartreuse'>（对应“次可交换性”）</font>
    - 若只保留交换子生成的子群，则可能有些元素退化为可交换。此时再次进行筛选，仍不可交换的元素就是 $G^{(2)}$
- **导群链**：$G \geq G^{(1)} \geq \cdots$
  - **保正规性**：$G^{(i+1)} \lhd G^{(i)}$
    - **证明**：已知 $G'\lhd G$，归纳即可
  - **同态传递性**：$f(G^{(n)}) = H^{(n)}$
    - **证明**：元素分析法，$\forall a,b\in G，f(aba^{-1}b^{-1}) = ghg^{-1}h^{-1} \in H'$
  - **实例**：
    - 阿贝尔群的升中心列恒为 $G$，导群链为 $A\geq \lang e \rang\geq \cdots$
    - $S_n$ 的导群链为 $S_n \geq A_n \geq \cdots$（见[习题](#对称群导群)）
    - $SL_n(F)$
- **可解群**：$\exists n$ 使得 $G^{(n)} = \lang e \rang$
  - **本质**：不可交换程度可被有限次削弱为0的群。也就是说，所有元素 $g$ 都有某个元素 $h$ 与 $g$ 可交换
    - 不可解群就是存在一个 $n$，使得 $G^{(n)} = G^{(n-1)} \neq \lang e \rang$，该群即为所有全不可换元素
    - <font color='chartreuse'>可解群还可以指示方程是否有某种解，为此需要跟[伽罗瓦理论](12.伽罗瓦理论.md)联系起来</font>
- **（命题7.10）幂零可解性**：幂零群均可解
  - **证明**：
    - 定义易得
      - $C_i$ 和 $C_i/C_{i-1}$ 都是阿贝尔群
      - $C_i(G)' < C_{i-1}(G)$
      - $C_1(G)' = C(G)' = \lang e \rang$
    - 由幂零性，可设 $C_n(G) = G$，则 $G/C_{n-1}(G)$ 是阿贝尔群，从而 $G'<C_{n-1}(G)$
    - 同理，$G^{(2)} < C_{n-1}(G') < C_{n-2}(G)$
      - <font color='chartreuse'>幂零群具有子群传递性，故 $G' = C_n(G')$，从而 $G'/C_{n-1}(G')$ 是阿贝尔群</font>
    - 不断进行下去，最终 $C^{(n)}(G) < C_1(G') = \lang e \rang$，从而可解
  - **理解**：归纳法。从最次的 $n+1$ 次可交换元素开始，逐步商掉元素，直到将 $C(G)$ 削为 $\lang e \rang$ 为止
  - **本质**：交换性与不可交换性是对偶关系，知一得二。
    - 也就是说，若群可被有限个次交换分解，当然也可以被有限个优不可交换分解，且分解方向相反
  - **反例**：可解不一定幂零，如 $S_3$
- **（定理7.11）可解传递性**：
  - **子群传递性**
    - **证明**：$K<G$，由 $G$ 可解得 $K^{(n)} \leq G^{(n)} = e$，从而 $K$ 也可解
    - **理解**：本质是导群链的单调减小性
  - **同态传递性**
    - **证明**：设 $G$ 可解，$f:G\to H$ 是同态，则
      - $\lang e \rang = f(e) = f(G^{(n)}) = H^{(n)}$，从而 $H$ 可解
    - **理解**：同态具有压缩性，可视为子群的同构
  - **正规逆传递条件**：设 $N\lhd G$ 可解，若 $G/N$ 也可解，则 $G$ 可解
    - **证明**：设 $f:G\to G/N$ 是满的商同态
      - 已知 $(G/N)^{(n)} = f(G^{(n)}) = \lang e \rang$，从而 $G^{(n)} < \ker f = N$
      - 则由子群可解性，$G^{(n)}$ 可解
      - 从而存在 $G^{(n+k)} = (G^{(n)})^{(k)} = \lang e \rang$，从而 $G$ 可解
    - **理解**：
      - 证明思路：商群可解得全不可换元素只能在 $N$ 中，再由 $N$ 可解，即得 $G$ 中无全不可换元素
      - 证明方法也是一样，可将导群列推到 $N$ 中，即得某导群 $G^{(n)}$ 可解，再易得导群可解的群可解，故直积 $G$ 可解
    - **本质**：直积可解条件

#### 特征子群

- **特征子群**：$\forall f\in\Aut G，f(H) \leq H$
  - **正规性**：$H\lhd G$
    - **证明**：取 $f$ 为共轭自同构即可
  - **本质**：群自身作用本质都是自同态（作用同态），故自同构可看作一一对应的作用（共轭/左平移），故全不变的意思就是任意一一对应作用的不动集合
    - 它是和正规子群同类型的概念，但更加严格
  - **实例**
- **全不变子群**：$\forall f\in\End G，f(H) \leq H$
  - **关系**：全不变子群 $\subset$ 特征子群 $\subset$ 正规子群
  - **实例**：
    - 导群 $G'$ 是全不变子群
      - **证明**：由同态运算式，$f([a,b]) = [f(a),f(b)]$。再由 $f(a),f(b)\in G$，得 $[f(a),f(b)]\in G'$，从而 $f(G')\leq G'$
    - 若 $G$ 是阿贝尔 $p$ 群，则幂零子群 $G[p]$ 是全不变子群
      - **证明**：同态运算式易得
  - **反例**：$C(G)$ 是特征子群，但不一定全不变
    - **证明**：由同构运算式易得特征性，
- **最小正规子群**：不包含真正规子群的正规子群
- **最大正规子群**：包含所有正规子群的真正规子群
- **（引理7.13）全不变引理**：有限群 $G$ 中，$N$ 是正规子群，$H$ 是子群
  - **正规逆遗传条件**：若 $H$ 是 $N$ 的特征子群，则 $H\lhd G$
    - **证明**：
      - 已知共轭运算是 $N$ 上的自同构
      - 再由 $H$ 是 $N$ 的特征子群，得 $\forall a\in N，aHa^{-1} = H$
      - 再由 $N$ 正规性，$\forall g\in G，gaHa^{-1}g^{-1} = a_2gHg^{-1}a_2^{-1} = gHg^{-1}$，即 $gHg^{-1} \lhd N$
        - <font color='chartreuse'>在上面的证明中，实际上只用到了 $H$ 的正规性，也就是说，我们实际上在讨论正规子群的逆遗传性。但它并不成立。也就是说，确实存在 $gHg^{-1} \neq H$，但 $gHg^{-1}\lhd N$ 的情况。接下来我们要用强化后的全不变条件，来证明这种情况此时不会出现</font>
      - 由刚才的推导，可得 $gHg^{-1}$ 是 $N$ 上的自同构，从而由全不变性即得结论
    - **本质**：进一步探索正规子群性质的边界
  - **西罗不变性**：正规西罗 $p$ 子群均为全不变子群
    - **证明**：任取 $g\in P$，设阶为 $p^n$
      - 由同态的阶整除性，可设 $|f(g)| = p^m$。再由任意性，即得 $f(P)$ 中的元素阶均为 $p$ 的幂，从而是 $p$ 群
      - 再由同态保子群关系，得 $f(P)$ 是西罗 $p$ 子群。再由同态保正规性，即得全不变性
    - **本质**：共轭不变性的本质其实是更强的全不变性
      - 证明全不变性的方法很少，$p$ 群的阶方法就是其中一种
  - **可解群的最小正规群**：若 $G$ 可解，$N$ 是最小正规子群，则 $N$ 是阿贝尔 $p$ 群
    - **证明**：
      - 易得导群 $N'$ 是 $N$ 的全不变子群，再由第一结论，$N'\lhd G$
      - 再由 $N$ 是最小正规子群 + $G$ 可解性，得只能是 $N' = \lang e \rang$，从而 $N$ 是非平凡阿贝尔群
      - 设 $P$ 是 $N$ 的非平凡西罗 $p$ 子群，由 $N$ 交换性，$P\lhd N$，从而由第二结论，其为全不变子群。再由第一结论，$P\lhd G$
        - <font color='chartreuse'>交换子群的西罗子群是全不变子群</font>
      - 再由 $N$ 最小性 + $P$ 非平凡性，只能是 $P = N$
    - **理解**：可解群的最小正规子群是交换群，这个没问题，因为导群一定出现在正规链中，而可解即所有元素均有可换性，最终结果必然是彼此均可换的群
      - 同时还是 $p$ 群，也就是说不会出现不同的阶因子。这一点可通过全不变性得到
  - **没啥关系的结论**：[作用同构条件](5.有限群的结构.md)表明，若最小正规子群 $N$ 的指数有限，则 $G$ 同构于一个置换群
- **（命题7.14）Hall**：设 $G$ 是 $mn$ 阶有限可解群（$(m,n)=1$），则
  - $G$ 满足西罗存在条件3
  - 任意两个 $m$ 阶子群同构
  - 任何 $k\mid m$ 阶子群含于一个 $m$ 阶子群
  - **证明**：好长

- **双换位子群**：设 $A,B \leq G$，$[A,B]$ 是 $[a,b]$ 生成的子群
  - **正规性**：$[A,B] \lhd A\lor B$
    - **证明**：只需 $\forall a\in A，b\in B，x\in A\lor B$，都有 $x[a,b]x^{-1}\in [A,B]$ 即可
      - 若 $x\in A$，则由换位子公式，$ = [xa,b][x,b]^{-1}\in [A,B]$
      - 若 $x\in B$，同理
    - **理解**：两个子群之间的不可换元素组成的群 $[A,B]$，商去后可换
  - **单调性**：若 $[A,G] = \lang e \rang$，则 $[A,G'] = \lang e \rang$
  - **平凡对称性**：若 $[A,G'] = \lang e \rang$，则 $[A',G] = \lang e \rang$
    - **证明**：定义易得
    - **理解**：
      - 左边表示：$A$ 与 $G$ 中的不可换元素均可换，即只能是 $A$ 没有与 $G$ 配对的不可换元素
      - 右边表示：$A$ 的不可换元素与 $G$ 可换，即只能是 $A$ 没有与 $G$ 配对的不可换元素
  - **正规子群垃圾桶性**：$A\lhd G\LR [A,G]\leq A$
    - **证明**：正规性的直接推论
    - **理解**：$[A,G]$ 就是A中的彼此不可换元素。若它们都在 $A$ 中，当然其它元素相对于 $A$ 均可换。可以说正规子群充当了装这些不可换元素的垃圾桶作用
  - **商群垃圾桶**：若 $B\lhd G，B\leq A$，则 $A/B \leq C(G/B) \LR [A,G] \leq B$

##### 其它

- **$p$ 群**：由幂零性得可解

    - 若 $G$ 中不存在阶为 $p$ 的元素，则所有元素的阶整除 $q$，从而子群阶数均整除 $q$，即 $G$ 为 $q$ 群，故可解

- **三素数可解群**：设 $p,q,r$ 是三个素数（可能相同），则 $pqr$ 阶群是可解群
  - **归纳证明**：对阶中素数归纳
    - 首先，已知 $p^3$、$pq$、$p^2q$ 阶群均可解
    - 设 $|G| = pqr\pad (p>q>r)$，只要 $G$ 不是单群，则即得存在 $N\lhd G$ 和 $G/N$ 可解，从而正规逆传递得 $G$ 可解
    - 反设 $G$ 是单群，则
      - 西罗 $p$ 子群个数为 $qr$
      - 西罗 $q$ 子群个数为 $pr$ 或 $p$
      - 西罗 $r$ 子群个数为 $pq$ 或 $p$ 或 $q$
      - 但由 $|G| \geq qr(p-1) + p(q-1) + q(r-1) + 1 \\\qquad = pqr + (p-1)(q-1) > pqr$，得三种情况均有矛盾
  - **推论**：三素数群不是单群

## 群列

### 群列分类

- **次正规列**：$G = G_0 \rhd G_1 \rhd \cdots G_n$
  - **列因子**：$G_i/G_{i+1}$
  - **长度**：真包含的数量 或 恒等列因子的数量
- **正规列**：$\forall G_i\lhd G$ 的次正规列
  - **实例**：
    - 导群列均为正规列
    - 幂零群的升中心列是正规列
- **单步细化**：在次正规列中添加一项 $G_{i}\rhd H\rhd G_{i+1}$
  - **细化**：添加了一些单步细化后的次正规列
  - **真细化**：使次正规列长度增加的细化
- **合成列**：$G_n = \lang e \rang$，且（每个列因子 $G_i/G_{i+1}$ 均为单群）的次正规列
  - **理解**：$G_{i+1}$ 是 $G_i$ 的最大正规子群
  - **性质**
    - **最细性（唯一性）**
    - **存在性**
- **可解列**：$G_n = \lang e \rang$，且（每个列因子 $G_i/G_{i+1}$ 均为阿贝尔群）的次正规列
  - **理解**：$G_{i}$ 的不可交换元素均在 $G_{i+1}$ 中
  - **性质**：
    - **细化封闭性**
  - **实例**：
    - 可解群的导群列
    - 幂零群的升中心列

### 群列关系

- **（定理8.4）群列关系**
  - **合成列存在性**：有限群均含有合成列
    - **证明**：可设 $G_1$ 是最大正规子群，则 $G/G_1$ 是单群
      - 再设 $G_2$ 是 $G_1$ 的最大正规子群，则 $G_1/G_2$ 是单群
      - 依次进行，则必定存在 $G_n = \lang e \rang$。此时 $G\rhd G_1\rhd \cdots G_n$ 就是合成列
  - **可解列细化封闭性**：可解列的细化还是可解列
    - **证明**：设 $G_{i}\rhd H \rhd G_{i+1}$，则 $H/G_{i+1}\lhd G_{i}/G_{i+1}$，从而也是阿贝尔群
      - 由第三同构定理，$G_i/H = (G_i/G_{i+1})/(H/G_{i+1})$，易得也是阿贝尔群
    - **理解**：可解列中，$G$ 必定是可解群，而细化添加的群也必然是 $G$ 的正规子群
  - **合成列最细性**：次正规列 $S$ 是合成列 $\LR S$ 没有真细化
    - **证明**：由正规性的商群传递性，细化必然满足 $H/G_{i+1}\lhd G_i/G_{i+1}$，由定义即可得结论
    - **本质**：合成列最细性
- **（定理8.5）可解列可解判定**：$G$ 是可解群 $\LR G$ 含有可解列
  - **证明**：
    - **必要性**：取导群列即可
    - **充分性**：由 $G/G_1$ 是阿贝尔群，得只能是 $G'\lhd G_1$。同理可得 $G_1'\lhd G_2$，再由导群单调性即得 $G^{(2)}\lhd G_2$。归纳可得 $G^{(i)}\lhd G_i$
      - 由可解列中 $G_n = \lang e \rang$ 即得可解性
  - **实例**：
    - 二面体群存在可解列 $D_n\rhd \lang a \rang \rhd \lang e \rang$，其中 $D_n/\lang a \rang = \lang b \rang \cong Z_2$
    - 若 $|G| = pq$，由柯西阶定理，存在 $|a|=p$，从而 $G \rhd \lang a \rang\rhd \lang e \rang$ 是可解列
- **（命题8.6）合成列可解判定**：有限群 $G$ 是可解群 $\LR G$ 含有列因子均为（素数阶循环群）的合成列
  - **证明**：
    - **充分性**：$G/G_1\cong Z_p$ 是阿贝尔群，从而该合成列是可解列，从而 $G$ 可解
    - **必要性**：
      - 若 $G_0\neq G_1$，设 $H_1$ 是包含 $G_1$ 的 $G$ 中最大正规子群
      - 若 $H_1\neq G_1$，设 $H_2$ 是包含 $G_1$ 的 $H_1$ 中最大正规子群
      - ……
      - 则细化 $G\rhd H_1 \rhd \cdots \rhd H_k\rhd G_1$ 是合成列
        - 易得此时 $H_i/H_{i+1}$ 是单群
        - 由有限可解性，$G/G_1\cong \sum Z_{n_i}$，再由 $H_k/G_1 \lhd G/G_1$ 是单群，得只能同构于某个 $Z_{p}$，不断归纳即得 $H_i/H_{i+1}$ 都是素数阶循环群
      - 对 $G_i$ 均如上构造细化合成列即可
- **次正规列等价**：$G$ 的两个次正规列的非平凡因子存在一一对应关系，使得两个因子是彼此同构的群
  - **等价性**：满足自反性、对称性、传递性
  - <font color='chartreuse'>总项数不一定相等，但彼此不同的 $G_i$ 数量一定相等</font>
- **（引理8.8）合成列最细性**：设 $S$ 是群 $G$ 的合成列，则 $S$ 的细化均等价于 $S$
  - **证明**：设 $S$ 是 $G\rhd G_1\rhd \cdots \rhd G_n = \lang e \rang$
    - 已知合成列无真细化，定义易得结论
  - **本质**：用等价语言再次阐述合成列最细性
- **（引理8.9）Zassenhaus**：设 $A^*\lhd A，B^*\lhd B$ 是为 $G$ 的四个子群，则
  - $A^*(A\cap B^*) \lhd A^*(A\cap B)$
  - $B^*(A^*\cap B) \lhd B^*(A\cap B)$
  - $A^*(A\cap B)/A^*(A\cap B^*) \cong B^*(A\cap B) / B^*(A^*\cap B)$
  - **证明**：由正规的限制传递性，$A\cap B^*\lhd A\cap B$
    - 再由正规的乘积传递性得 $A^*(A\cap B) \lhd A$
    - 设 $D = (A^*\cap B)(A\cap B^*)$，则由正规乘积传递性，$D\lhd A\cap B$
    - 若存在满同态 $f:A^*(A\cap B)\to (A\cap B)/D$ 使得 $\ker f = A^*(A\cap B^*)$，即可得到结论
      - 设 $f(ac) = Dc\quad (\forall a\in A^*，c\in A\cap B)$
      - **满射性**：反设 $ac = a_1c_1$，则 $c_1c^{-1} = a_1^{-1}a\in (A\cap B)\cap A^* = A^*\cap B < D$，从而此时 $Dc_1 = Dc$
      - **同态性**：$f(a_1c_1a_2c_2) = f(a_1a_3c_1c_2) = Dc_1c_2 = Dc_1Dc_2 = f(a_1c_1)f(a_2c_2)$
      - **核**：$ac\in\ker f \LR c\in D \LR c = a_1c_1\pad (a_1\in A^*\cap B，c_1\in A\cap B^*)$
        - 从而 等价于 $ac = (aa_1)c_1\in A^*(A\cap B)$。由任意性即得结论
- **（定理8.10）Schreier**：群 $G$ 的任意两个【次】正规列均存在等价的【次】正规细化
  - **证明**：设 $G\geq G_1\geq\cdots\geq G_n，G\geq H_1\geq\cdots\geq H_m$ 是两个正规列，$G_{n+1} = H_{m+1} = \lang e \rang$
    - 此时 $G_i = G_{i+1}(G_i\cap H_0)\geq G_{i+1}(G_i\cap H_1)\geq\cdots\geq G_{i+1}(G_i\cap H_{m+1})$ 
      - 由Zassenhaus引理，它形成一个正规列，从而是原正规列的细化
      - 设 $\begin{cases} G(i,j) = G_{i+1}(G_i\cap H_j) \\ G(i,0) = G_i \end{cases}$
      - 则 $G\geq G(0,1) \geq \cdots G(0,m)\geq G(1,0)\geq \cdots \geq G_(n,m)$ 是细化
        - 有 $(n+1)(m+1)$ 项
    - 同理，也存在 $\begin{cases} H(i,j) = H_{j+1}(H_i\cap H_j) \\ H(0,j) = H_j \end{cases}$
      - 有 $(n+1)(m+1)$ 项
    - 最后，由Zanssenhaus引理，存在下列同构关系，从而细化等价
      - $\cfrac{G(i,j)}{G(i,j+1)} = \cfrac{G_{i+1}(G_i\cap H_j)}{G_{i+1}(G_i\cap H_{j+1})} \cong \cfrac{H_{j+1}(G_i\cap H_j)}{H_{j+1}(G_{i+1}\cap H_{j})} = \cfrac{H(i,j)}{G(i+1,j)}$
- **（定理8.11）Jordan-Holder**：群 $G$ 的任意两个合成列等价
  - **证明**：已知合成列是次正规列，合成列的细化等价，合成列无真细化。从而细化均等价于自身
  - **本质**：合成列唯一性
  - **实例**：
    - $Z_n$ 的合成列均等价，故算术基本定理成立
  - **推论（唯一对应性）**：每个含有合成列的群都唯一决定一个单群列
