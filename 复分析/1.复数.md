## 复数

### 复数的表示

- **复数**：$z = x+yi\pad (x,y\in\R)$
    - 实部：$x = \Re z$
    - 虚部：$y = \Im z$
- **复平面**：设实部为 $x$ 轴，虚部为 $y$ 轴，构成的二维平面称为复平面
  - **对应性**：复数与复平面上的点（坐标向量）一一对应
  - **辐角**：复平面上，点 $z$ 的坐标向量的任意角称为 $z$ 的辐角 $\arg z$
    - **不唯一性**：一个 $z$ 可以有多个任意角，即 $\arg z = \hkh{\Arg z + 2k\pi \biggm| k\in\Z}$
  - **主辐角**：满足 $-\pi < \Arg z \leq \pi$ 的辐角
    - **计算方法**：
      - $\t = \arctan\dfrac{y}{x}$
- **复数的表示**：
  - **三角形式**：$z = r(\cos\t + i\sin\t)$
  - **指数形式**：$z = re^{i\t}$，或者写为 $|z|e^{i \arg z}$
  - **代数形式**：$z = x + iy$
  - **证明**：
    - 代数形式 $\xto{辅助角公式}$ 三角形式 $\xto{欧拉公式}$ 指数形式
    - 指数形式 $\xto{欧拉公式}$ 三角形式 $\xto{定义}$ 代数形式
  - **推论**：
    - 代数形式 $\xto{\arctan}$ 主辐角 $\xto{定义}$ 指数形式

### 欧拉公式

- **扩域微分引理**：
  - 设实函数 $f(x)$ 可微，$i\notin \R$
  - 则
    - $if(x)$ 也可微，且 $\Big[ if(x) \Big]' = if'(x)$
      - **证明**：
        - 由导数定义易得结论
    - 但是 $f(ix)$ 不一定可微，且不一定有 $\Big[ f(ix) \Big]' = f'(a)|_{a=ix}$
      - **反例**：$f(x) = x^2$
- **欧拉公式**：$e^{i\t} = \cos\t + i\sin\t$
  - **证明（泰勒展开法）**：
    - 已知 $e^x$ 的泰勒展开式，同理可建立 $e^{ix}$ 的泰勒展开式
    - 然后根据 $i$ 不同幂的正负，调整各项顺序并配凑，可建立更序级数
    - 再将该级数分离为 $\cos$ 和 $\sin$ 的泰勒展开即可
  - **证明（导数法）**
    - 设 $f(\t) = e^{-i\t}(\cos\t + i\sin\t)$，计算易得 $f'(\t) \equiv 0$，从而 $f$ 是常数
    - 再由于 $f(0) = 1$，故 $f(\t)\equiv 1$，从而得到结论
  - **证明（微分方程法）**：
    - 设 $z(\t) = \cos\t + i\sin\t$，求导得 $\cfrac{dz}{d\t} = iz$，解微分方程即得结论
  - **证明（$e$ 的定义法）**
    - 由 $e^x\sim x+1\quad (x\to 0)$ 易得 $$e^x = \Big[ \lim\limits_{n\to\infty}(1+\frac{1}{n})^n \Big]^x \ =\lim\limits_{n\to\infty}(1+\frac{x}{n})^n$$
    - 同理，$e^{i\t} = \lim\limits_{n\to\infty}(1+\frac{i\t}{n})^n$，极限存在且为复数
    - <font color='chartreuse'>引理：复数的乘积性 $z_1z_2 = |z_1||z_2|\arg(z_1+z_2)$
      - 证：
        - 用 $z = x+yi$ 计算即可
    - 引理：复数的指数交换律 $|z^n| = |z|^n$
      - 证：
        - 由复数的乘积性易得结论</font>
     - **求模长**：
       - 由指数交换律计算易得 $|e^{i\t}| = \lim\limits_{n\to\infty}\dkh{\sqrt{1+\dfrac{\t^2}{n^2}}}^n$
       - 再进行 $\dfrac{\t^2}{n^2}$ 的指数配凑，计算极限即可得模长为 $1$
     - **求辐角**：
       - 由模长为 $1$，不妨设 $e^{i\t} = \cos\a + i\sin\a$
       - 。。。好像无论怎么样都绕不开泰勒展开
  - **证明（几何法）**
    - 首先由极坐标表示法，$\C$ 中任何向量均可写为半径 + 角度的表示方法
    - 计算易得复数乘积的角度相加性
    - 由 $e$ 的定义，$(1+\frac{i\t}{n})^n$ 可看作将原始向量 $z_0=1$ 进行 $n$ 次相同的延长和旋转
      - <font color='chartreuse'>每次的延长倍率为 $\sqrt{1+(\frac{\t}{n})^2}$，旋转角度为 $\frac{\t}{n}$</font>
    - 由勾股定理，易得每次旋转都形成一个极小直角三角形
      - <font color='chartreuse'>$z_n$ 是斜边，$z_{n-1}$ 是直角边</font>
    - 当 $n\to\infty$ 时，由于累乘和累加均是一阶运算，而模长是累乘下的二阶无穷小（取对数即可看出），角度是累加意义的一阶无穷小
      - 从而模长在迭代中视为不变，但旋转在迭代中保留变化
      - 极小直角三角形此时等价于极小扇形，从而欧拉公式成立
   - **本质**：
     - 类似用积分计算圆的面积时，不能对半径积分，而应当对极小扇形积分。因为半径的弧度角为 $0$，在累加意义下依然是 $0$，可视作累加意义的高阶无穷小
     - 再举一个例子，正方形不能用平移边去逼近内切圆，从而得出 $\pi = 4$ 的结论，因为余项在迭代下不是高阶无穷小（收敛速度太慢）
  - **理解**：$e$ 乘积的指数相加性正好与复数的乘积角度相加性契合

### 复数的运算

  - **正方向**：规定逆时针为正方向，顺时针为负方向
- **辐角乘积性**：$\Arg (z_1z_2) = \Arg z_1 + \Arg z_2$
  - **证明**：
    - 写成指数形式后易得结论

## 复平面

### 复数的向量性

- **复数的模展开**：$|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2\Re(z_1·\bar z_2)$
  - **证明**：
    - 易得 $\Arg(z_1·\bar{z}_2) = \t_1-\t_2$
    - 易得 $\Re z$ 就是 $z$ 的向量在 $x$ 轴上的投影，故 $$\Re(z_1·\bar{z}_2) = |z_1||z_2|\cos|\t_1-\t_2|$$
  - **理解**：
    - 复数是向量，虽然乘法运算和向量不同，但模运算却与向量相同
      - 复数的乘积是伸缩旋转，向量的乘积是伸缩投影
    - 所以需要对乘法结果取实部，从而得到 $cos$ 形式，使之化为向量乘法


### 复平面曲线的方程

- **直线**：$z = z_1 + t(z_1-z_2)$
  - <font color='chartreuse'>参数方程写法</font>
- **圆**：$|z-z_0| = r$
- **实轴**：$\Im z = 0$

### 习题

- **复数不等式**：设 $a,b\in\C$，若 $|a|,|b|<1$，证明 $\biggm|\cfrac{a-b}{1-\bar{a}b}\biggm| < 1$
  - **证明一**：
    - 三角不等式（如果a、b是向量，就这么做。复数和向量性质类似，所以也可以这么做）
  - **证明二**：
    - 向量几何（不好用）
- **等边三角形**：
  - 设 $z_1,z_2,z_3\in \C$
  - 则它们组成复平面上的等边三角形 $\LR z_1^2+z_2^2+z_3^2 = z_1z_2+z_2z_3+z_1z_3$
  - **证明**：
    - 由旋转性，$(z_1-z_2)(cos\frac{\pi}{3}+i\sin\frac{\pi}{3}) = z_2-z_3$
    - 移项得复数分式，两边平方即得上述等式
- **三角形的内角和**：三角形的内角和是 $180\degree$
  - **证明（向量旋转法）**
  - **证明（辐角法）**：
    - 设复平面上三角形的三边为 $a,b,c$，对角分别为 $A,B,C$
    - 则易得 $\dfrac{a}{b}·\dfrac{b}{c}·\dfrac{c}{-a} = -1 \red\Rt \Arg A + \Arg B + \Arg C = \Arg(-1)$

### 区域

- **区域（可以不凸，但是必须接触）**：设 $D$ 是非空开集，若其中任意两点都可用 $D$ 中的折线连接，则称 $D$ 为区域
  - <font color='chartreuse'>实际上就是道路连通集</font>
- **闭域**：区域 $D$ 加上它的边界 $C$，闭包 $\ol D = D+C$
  - **实例**：单位圆是区域，单位圆 + 单位圆周是闭域
- **有界集的直径**：$\diam E = \sup\hkh{|z - z'| : z,z'\in E}$
  - <font color='chartreuse'>集合 $E$ 内全体线段长度的上确界</font>
- **连续曲线**：
  - **参数方程写法**：$z(t) = x(t) + iy(t) \pad (\a\leq t\leq \b)$
    - **起点**：$z(\a)$
    - **终点**：$z(\b)$
  - **重复点**：若 $z(t_1) = z(t_2) \pad (t_1 \neq t_2)$，则称 $t_1,t_2$ 是曲线 $z(t)$ 的重复点
  - **简单曲线（若尔当曲线）**：无重复点的连续曲线
  - **简单闭曲线**：若尔当曲线的闭包
- **可求长的曲线**：连续曲线 $z(t)\ (\alpha \leq t \leq \beta)$）中，若对任意递增实数列 $\{t_n\}$，点列 $\{z(t_n)\}$ 对应的折线长度 $\lim\limits_{n\to\infty}\sum\limits^{n}_{j=1}|z(t_j) - z(t_{j-1})|$ 有界，则称该曲线可求长
  - **长度**：折线长度的极限值
- **光滑曲线**：实部和虚部的导函数存在且连续
- **逐段光滑曲线**：由有限条光滑曲线衔接而成的连续曲线

### 若尔当定理

- **连通集**：设集合 $E$，至少含有两个点。若 $E$ 不是两个无公共点的非空闭集之并，则称为连通的
- **单连通区域**：
  - **连通定义法**：若区域 $D$ 内任意简单闭曲线 $C$ 全含于 $D$，则称 $D$ 为单连通区域
  - **边界定义法**：若区域 $D$ 的边界为连通集，则称为单连通区域
- **$n$ 连通区域**：有 $n$ 个连通边界的集合

## 复变函数

### 函数的多值性

- **单值函数**：一个自变量只有一个对应函数值
- **多值函数**：一个自变量存在不止一个对应函数值
  - **值域**：全体函数值的集合

### 函数的写法

- **单变量写法**：$f(z) = z^2 + z$
- **双变量写法**：
  - 设 $z = x + yi$，则 $f(z) = (x+yi)^2 + (x + yi)$
  - 若继续设为 $f(z) = u + vi$，则此时可以得到 $\begin{cases} u = u(x,y) \\ v = v(x,y) \end{cases}$
  - 最终可写为 $\begin{cases} z = x+yi \\ w = u+vi \\ u = u(x,y) \\ v = v(x,y)  \end{cases}$，简写为 $f(x,y) = u(x,y) + iv(x,y)$

### 习题

- 设函数为 $\begin{cases} z = x + yi & (1) \\ x^2 - y^2 = 4 & (2) \\ w = z^2 & (3)\end{cases}$，写出其形状
  - **解**：
    - 将 $(1)$ 代入 $(3)$ 得 $w = x^2 - y^2 + 2xyi = 4 + 2xyi$
    - 易得 $f(x,y) = 2xy$ 的值域是整个 $\R$，故结果为直线 $x=4$

### 极限

- **复变函数的极限**：若 $\forall \e > 0，\exists \d > 0$ 使得 $\forall |z-z_0| < \d$ 都有 $|f(z)-f(z_0)| < \e$，则称 $\lim\limits_{z\to z_0}f(z) = w_0$
  - **极限唯一性**
  - **极限有界性**
  - **四则运算传递性**
- **坐标极限定理**：设 $f(z) = u(x,y) + iv(x,y)$，则 $u,v$ 都有极限 $\LR f$ 有极限
    - <font color='chartreuse'>类似数分，本质还是勾股定理</font>
  - **证明（必要性）**：
    - 设 $u\to a，v\to b$，则设 $\eta = a+bi$
    - 由三角不等式易得 $|u(x,y)-a| \leq |f(z)-\eta| = \sqrt{(u-a)^2 + (v-b)^2}$
    - 再由C-S不等式即得 $|f(z) - \eta| \leq |u-a|+|v-b| < \e$，从而 $f\to \eta$
  - **证明（充分性）**：
    - 设 $\eta = a+bi$，则，$|f(z)-\eta| = \sqrt{(u-a)^2 + (v-b)^2} < \e$，简单变形放缩易得 $u-a < \e$ 且 $v-b<\e$

### 连续

- **连续复变函数**：若 $\lim\limits_{z\to z_0} f(z) = f(z_0)$，则称 $f$ 在 $z_0$ 处连续
  - **四则运算传递性**
  - **复合函数传递性**  
  - **连续曲线传递性**：连续曲线被连续函数映射成连续曲线
    - **证明**：
      - 定义即可
      - 由数分知识，连续函数将紧集映射成紧集。B-W子集判定定理

### 习题

- **无极限的函数**：设 $f(z) = \cfrac{1}{2i}(\cfrac{z}{\bar{z}} - \cfrac{\bar{z}}{z}) \ (z \neq 0)$，则其在任意点都不存在极限
  - **证明**：
    - 化简为 $f(z) = \sin 2\t = \dfrac{2xy}{x^2+y^2}$
    - 方向性证明：令 $y=x，y=0$，则 $\lim\limits_{z \to z_0} f(z)$ 结果不同
    - 跳跃性证明（×）：令 $y=\frac{1}{n}$，$x=\frac{1}{n^2}$，和令 $y=\frac{1}{n},x=\frac{2}{n}$ 结果不同（本质上还是收敛方向，这里不是指的Heine定理那个一元函数定理）

## 扩充复平面

### 复球面

- **测地投影法**：
  - 在三维坐标系中，构建一个与 $xOy$ 平面相切于原点的球 $B$
  - 坐标原点设为球的南极 $S$，其相对点设为北极 $N$
  - 易得 $xOy$ 平面上的任意一点 $z$，其和 $N$ 相连的直线 $l$ 穿过球面上某点 $P$
  - 若将 $xOy$ 平面看作复平面，则任意复数 $z$ 都可通过直线 $l$ 和球面上的交点 $P$ 一一对应。球 $B$ 的表面称为复球面
  - 北极 $N$ 对应的点称为无穷远点 $\infty$，此时复球面 $D_B$ 和扩充复平面 $\C_\infty = \C\cup \{\infty\}$ 一一对应

### 无穷远点

- **四则运算**：
  - 无穷远点没有实部和虚部
  - $\begin{cases} \infty + \infty = \infty  & \infty\cdot\infty = \infty \\\\ \infty-\infty 无意义 & \cfrac{\infty}{\infty} 无意义 \end{cases}$
  - 对于任意 $z\neq\infty$
  - 对于任意 $z\neq 0,\infty$，无穷远点 $\infty$ 与 $z$ 的四则运算结果都为 $\infty$
- **直线关系**：复平面上每一条直线都通过无穷远点
- **半平面关系**：所有半平面都不包含无穷远点
- **邻域**：无穷远点的邻域是 $O\Big((0,0),r\Big)$ 的外部，对应复球面上的球盖
- **边界性**：无穷远点是复平面唯一的边界点
- **球面无边界**：扩充复平面是唯一的无边界区域，无穷远点是其内点

### 广义连续

- **广义极限**：包含无穷远点的极限
- **广义连续**：设 $\lim\limits_{z\to z_0}f(z) = f(z_0)$，当 $z_0$ 或 $f(z_0)$ 取 $\infty$ 时，称 $f$ 在点 $z_0$ 为广义连续
- **无穷远点处的连续**：$\forall \epsilon<0，\exist \delta>0, \forall |z|>\frac{1}{\delta}，有|f(z)-f(\infty)|<\epsilon$
