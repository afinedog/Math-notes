# 复变函数9：调和函数

## 调和函数性质

- **复调和函数平均值定理**：
  - 设 $u(z)$ 在开圆 $K:|\zeta-z_0|<R$ 内是一个调和函数
  - 若 $u(z)$ 在闭圆 $\overline{K}$ 上连续
  - 则 $u(z_0) = \dis\frac{1}{2\pi} \int^{2\pi}_{0} u(z_0+Re^{i\p})d\p$
  - **本质**：圆心上的值等于圆周上值的平均数
  - **证明**：
    - 首先存在共轭调和函数，共同构成一个解析函数。
    - 解析函数在开圆内具有平均值定理（极限到闭圆即可） + 角度积分具有线性
    - 比较实虚部即可
- **极值原理**：在区域 $D$ 内是调和函数的非常数函数 $u(z)$ 在 $D$ 内不能达到极值
  - **本质**：最大/小模原理
  - **理解**：调和函数的模 $\overset{欧拉公式}{\longrightarrow}$ 解析函数的模
  - **证明**：反设达到最大值，则构成解析函数 $f$
    - 因为调和函数共轭对称性，$u(z)$ 可以充当实部也可以充当虚部
    - 则设 $u(z)$ 为实部，从而 $|e^{f(z)}| = e^{u(z)}$
      - 它在内点处达到最大值，与解析性矛盾
  - **推论**：多连通区域取割线即可变为单连通区域，再分别证明

## 调和函数在积分中的应用

### 泊松积分公式

- **泊松积分**：$\dis\int^{+\infty}_0 e^{-t^2}dt = \frac{\sqrt{t}}{2}$
  - 设
    - $f(z)$ 在圆 $K:|z|<R$ 内解析，在闭圆上连续
    - $\begin{cases} z = re^{i\p}（K的圆心） \\ \zeta = Re^{i\theta} （K的某个半径向量）\\ z^* = \frac{R^2e^{i\p}}{r}（z关于K的对称点） \end{cases}$
  - **圆上柯西积分公式**：$\dis f(z) = \frac{1}{2\pi i}\int_K \frac{f(\zeta)}{\zeta-z}d\zeta = \frac{1}{2\pi}\int^{2\pi}_0 \frac{f(Re^{i\theta})}{Re^{i\theta}-re^{i\p}}Re^{i\theta}d\theta$
  <br>
  - **圆外部柯西积分公式**：$\dis 0 = \frac{1}{2\pi i}\int_K \frac{f(\zeta)}{\zeta-z^*}d\zeta = \frac{1}{2\pi}\int^{2\pi}_0 \frac{f(Re^{i\theta})}{re^{i\theta}-Re^{i\p}}re^{i\theta}d\theta$
    - 虽然圆外部不是解析区域，但扩充z平面下，C可以看作无穷远点的外部圆周C'。而 $z^*$ 在C'内部。由外部区域的柯西积分公式，$0 = \int_K$
    - 或者可以看作 “无穷远点的留数 = 所有非解析区域的留数”，而非解析区域就是圆（本质一样）
  - 两式相减，取实部，得到：
- **圆的泊松积分**：$\dis u(r,\p) = \frac{1}{2\pi}\int^{2\pi}_0 u(R,\theta)\frac{R^2-r^2}{R^2-2Rrcos(\theta-\p)+r^2}d\theta$
- **泊松积分定理**：任何圆内调和且在圆上连续的函数，在圆内的值可以用圆上的泊松积分表示
  - **证明**：上面就是证明过程

### Dirichlet问题

- **拉普拉斯方程（二阶偏微分方程）**：$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$，通解为调和函数全体
- **第一边值问题（狄利克雷问题）**：求在区域 $D$ 内调和，且在 $\overline{D}$ 上连续的函数 $u(z)$，使得其在周线 $C$ 上取已知值 $\tilde{u}(\zeta)$
  - <font color='chartreuse'>已知边界条件，所以叫边值问题</font>
  - **解具有唯一性**
    - **证明**：反设 $u_1(z)，u_2(z)$，则差函数也调和连续，且在 $C$ 上为0
      - 由极值定理，最大值和最小值在边界上，从而均为0，所以差函数恒为0（**证毕**）
    - **本质**：解析函数内部唯一性
- **单位圆内部的解**：单位圆上的泊松积分（$R = 1$）
  - **证明**：若 $u(z)\equiv 1$，则 $\dis \frac{1}{2\pi}\int^{2\pi}_0 \frac{1-r^2}{1-2rcos(\theta-\p)+r^2}d\theta = 1$
    - **$u(z)$ 是调和函数**：
      - 此时被积函数可化为 $\dis\frac{\zeta^2-z^2}{(\zeta-z)^2}$，其裂项得 $\dis\frac{\zeta}{\zeta-z} + \overline{\frac{z}{\zeta-z}}$
      - 添项分离得 $\frac{1}{2}\big[ \frac{\zeta+z}{\zeta-z} + \overline{\frac{\zeta+z}{\zeta-z}} \big] = Re(\frac{\zeta+z}{\zeta-z})$
      - <font color='chartreuse'>从而可得下述函数：</font>
      - **Schwarz积分**：$\dis f(z) = \frac{1}{2\pi i}\int_{|\zeta| = 1} u(\zeta)\frac{\zeta + z}{\zeta - z}\cdot \frac{d\zeta}{\zeta}$
        - 其实部就是 $u(z)$
      - **证明解析性**：
    - **证明** $\dis\lim_{z\to\zeta_0} \frac{1}{2\pi} \int^{2\pi}_0 \big[ u(\zeta)-u(\zeta_0) \big]\frac{1-r^2}{1-2rcos(\theta-\p)+r^2}d\theta = 0$
      - 从而 $\forall \xi_0，u(z) \to u(\zeta_0)\quad (z\to \zeta_0)$（**证毕**）
- **上半平面的解**：
  - **本质**：共形映射成原点单位圆 $|w|<1$ 再求解
    - 此时 $R = 1，w(z) = 0$
  - **求解**：$w = \cfrac{\zeta-z}{\zeta-\overline{z}}$
    - $z$ 的像是原点，实轴的像是单位圆周
    - 设 $\theta$ 被映射成 $\tau$，$t$ 是实轴上的点，则单位圆周上的映射公式为 $e^{i\tau} = \cfrac{t-z}{t-\overline{z}}$
      - 计算得 $d\tau = \cfrac{2ydt}{(t-x)^2+y^2}$
    - 由平均值定理，$u(w(z)) = P(0) = \frac{1}{2\pi}\int^{2\pi}_0 P(e^{i\tau})d\tau$，从而得到下面的解
      - <font color='chartreuse'>$P = u\circ w^{-1}$，就是单位圆的泊松积分公式</font>
    - **上半平面的泊松积分公式**：$\dis u(z) = \frac{1}{\pi}\int_R u(t)\cdot\frac{ydt}{(t-x)^2+y^2}$
  - **理解**：设原像平面为 $Z$，共形平面为 $W$，$u$ 的像平面为 $U$
    - 已知 $W\to U$ 的解，需要求 $w^{-1}:W\to Z$，才能得到 $u:Z\to U$
