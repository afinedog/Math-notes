# 前置知识见Munkres第一章

## ZFC公理体系（Zermelo-Fraenkel Choice）

- 罗素悖论：$R = \{x是集合\mid x\in x\}$，则 $R\in R \Leftrightarrow R\notin R$

- **（空集存在公理）**：$\exist a, \forall x, x\notin a$
  - <font color='orange'> 可由无限公理推出</font>
- **外延公理**：$\forall a,x,y$，若 $a\in x \Leftrightarrow a\in y$，则 $x = y$
  - 定义相等 + 交换性
  - **本质**：集合完全由元素确定
- **无序对存在公理**：$\forall x,y,z$，$\exist a$ 使得 $ z\in a\Leftrightarrow \big[ (z\in x)\lor (z\in y) \big]$
  - 定义无序对 $a = \{x,y\} = \{\{x_1,...\},\{y_1,...\}\}$
  - 定义有序对 $(x,y) = \{\{x_1,...\},\{x_1,...,y_1,...\}\}$
  - 定义直积、映射
- **并集存在公理**：集合的所有元素的并也是集合
  - 定义并集 $a = \{x_1,...,y_1,...\}$
  - 定义有限序数集 $S_n$
    - $0 = \varnothing，1 = \{\varnothing\}，2 = \{\varnothing,\{\varnothing\}\}...$
- **（内涵公理）**：$\forall a = \{x,...\},\varphi(x)，\exist b = \{x\in A\mid \varphi(x)（成立定义交集、差集、补集）\}$
  - <font color='orange'> 可由替换公理推出 </font>
- **幂集存在公理**：$\forall x，\exist \mathcal{P}(x)，\forall a\in \mathcal{P}(x) \Leftrightarrow a\subset x$
  - 定义幂集
- **正则公理**：$\forall X\neq \varnothing,x\in X，\exist y\in X$，使得 $y\cap x = \varnothing$
  - 定义空集
  - 不存在以自身为元素的集合，解决罗素悖论
    - **证明**：反设 $a\in a$，则由无序对公理，存在 $\{a\}$
      - 由正则公理，$a\cap \{a\} = \varnothing$
      - 由内涵公理 $a\in a\cap \{a\}$，矛盾
        - 由元素定义，$a\in \{a\}$
        - 由反设，$a\in a$
  - 不存在所有集合的集合
- **替换公理（模式）**：$\forall x\in X,\exist B = \{y\mid \exist x\in A,\varphi(x,y)（成立）\}$
  - 定义映射的值域
  - 有了映射，其可以推出无穷条公理
- **无限公理**：$\exist X,\varnothing \in X$，$y\in X \Rightarrow y\cup \{y\}\in X$
  - 定义无限序数集 $\omega = \{1,2,...\}$
    - 该公理定义了归纳条件，应用数学归纳法即可
- **选择公理**：$\forall y\in Y，\exist\varphi，\forall \varphi(y)\in y$
  - 定义选择函数、超穷归纳法

### 公理的模式

- 分类：
  - 区别在哪些集合是被预设和讨论的
  - 体现在哪些公理是可被推出的
- **朴素公理模式**：$\exist X = \{x\mid 性质p(x)成立\}$
  - 将某个性质化为具体集合（子集）
  - 被罗素悖论推翻
- **替换公理模式**：$\forall p(x),a，\exist X = \{x\in a\cap Dom(p)\}$
- **子集公理模式（分离公理模式）**：已知 $A$，则 $\exist B = \{x\mid x\in A, 性质p(x)成立\}$
  - 本质是内涵公理，通过映射（性质）创建新集合
  - 替换公理模式 + 空集公理

### 数理逻辑符号

- **公式**：$\varphi(x,p_1,...p_n)$
  - **原子公式**：$\varphi(x,y): x=y或x\in y$
    - 原子公式（等于，属于）通过连接词（或且非，蕴含，等价）和量词（任意，存在）建立公式
  - **谓词**：所有逻辑符号都是谓词
  - **句子**：没有变量的公式
- **类**：$C = \{x: \varphi(x,p_1,...,p_n) （成立）\}$
  - **真类**：不是集合的类
  - 所有集合的集合
  - 所有拓扑空间、线性空间等
  - 所有群
- **宇宙**：所有集合的类 $V = \{x: x=x\}$

#### 逻辑符号形式

- **分离公理模式**：$\forall X\forall p\exist Y\forall u(u\in Y\leftrightarrow u\in X\land \varphi(u,p))$
  - 推论：$Y = \{u\in X: \varphi(u,p)\}$
- R是关系
  - **定义域**：$dom(R) = \{u: \exist v(u,v) \in R\}$
  - **值域**：$ran(R) = \{v: \exist v(u,v) \in R\}$
