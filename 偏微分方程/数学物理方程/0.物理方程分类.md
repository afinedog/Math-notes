# 物理方程分类

## 物理定律总结

### 力学

- 牛顿第二定律 $F = ma$
- 牛顿冷却定律
- 胡克定律
  - 宏观 $f = -kx$
    - <font color='chartreuse'>$f$ 是弹性体的弹力，$x$ 是弹性体的形变量，$k$ 是劲度系数</font>
  - 微观 $\sigma = E\e$
    - <font color='chartreuse'>$\sigma$ 是材料的应力，$E$ 是杨氏模量，$\e$ 是相对伸长量</font>

### 守恒

- 诺特定理：每个连续对称性都有着相应的守恒律
  - 空间平移对称性：动量守恒
  - 空间转动对称性：角动量守恒
  - 时间平移对称性：能量守恒

### 输运

- 热传导定律 $\bs q = -k\nabla u$
  - <font color='chartreuse'>$\bs q$ 是热流强度（单位时间内流过单位横截面的热量）
  - $k$ 是热传导系数，$u$ 是温度函数</font>
- 扩散定律 $\bs q = -K\nabla u$
  - <font color='chartreuse'>$\bs q$ 是粒子流强度（单位时间内流过单位面积的粒子数）
  - $K$ 是扩散系数，$u$ 是粒子分布函数</font>
