# 输运方程

## 齐次形式

- **输运方程**：$u_t + b\cdot D_x u = 0$<font color='chartreuse'>（这里的 $D$ 只对 $x$ 取微分）</font>
  - $u:\R^n\times [0,\infty)\to \R，(x,t) \to u(x,t)$
  - $b\in \R^n，t\geq 0$
- **解的性质（特征线法）**：
  - 设 $z(s) = u(x+sb,t+s)\quad (s\in\R)$
  - 则 $\dfrac{dz}{ds} = b\cdot D_x u(x+sb,t+s) + u_t(x+sb,t+s)$
    - 当输运方程成立时，它恒为0，故此时 $z(s)$ 是常函数
    - $s$ 的意义是 $(x,t)$ 在 $(b,1)$ 方向上的延伸长度，即在该方向上 $u$ 是常函数
    - **特征方程**：$\dfrac{dx}{dt} = b$
    - **特征线**：解特征方程得 $x = bt+C$
- **初值问题**：若给定初值 $u(x,0) = g(x)$
  - 则当 $s = -t$ 时取到初值 $u(x-bt,0) = g(x-bt)$
  - 而当 $s=0$ 时，由于是在特征线上，故和 $s=-t$ 的值相同
    - 即 $u(x,t) = u(x-bt,0) = g(x-bt)$
  - 即 $g(x-bt)$ 为唯一解
    - 若 $g\in C^1$，则为连续解
    - 若 $g\notin C^1$，则为弱解
- **特征线法**：对于不太标准的输运方程，需要凑微分成标准形式
  - **实例**：$\begin{cases}u_t + (1+x^2)u_x - u = 0 & \R\times [0,\infty) \\ u(x,0) = \arctan x & \R\times \{0\}\end{cases}$
    - **变换（初等因子法）**：
      - 设初等因子为 $f$
      - 要使 $(uf)_t + (1+x^2)(uf)_x = (u_t + (1+x^2)u_x - u)f$
        - 化简并比较系数得 $f_t + (1+x^2)f_x = -1$
        - 我们假设 $f$ 退化为单变量函数，可选 $x$ 或 $t$，这里选比较简单的 $t$，也即 $f = f(t)$
        - 那么等式变为 $f_t = -1$，易得 $f = e^{-t}+C$
      - 原方程两边乘上初等因子 $e^{-t}$ 得 $(ue^{-t})_t + (1+x^2)(u_xe^{-t}) = 0$
      - 设 $v = ue^{-t}$，变为 $v_t + (1+x^2)v_x = 0$
      - 易得初值条件变为 $v(x,0) = \arctan x$
    - **求特征线**：
      - 此时 $d(s) = v(x+sb,t+s)$
      - 特征线 $\dfrac{dx}{dt} = 1+x^2$，即函数曲线族 $\arctan x + C = t$
        - 在 $xt$ 坐标轴上为曲线族 $x = \tan(t-C)$
        - 即 $(x,t)$ 点和 $(-\tan C,0)$ 在同一特征线上
          - <font color='red'>确定了 $x$ 和 $t$ 时，$C$ 也就确定为 $C(x,t) = t - \arctan x$
          - 将 $t=0$ 代入曲线族方程，即得 $x|_{t=0} = x(C) = -\tan C$，再将 $\begin{cases} t = 0 \\ x = x(C) \end{cases}$ 代入初值方程得到 $v(C)$，重新代入 $C(x,t)$，即可得到解函数 $v(x,t)$</font>
          <!-- - <font color="chartreuse">（若同时将 $C(x,t)$ 和 $t=0$ 代入曲线族方程，只能得到 $x=x$）</font> -->
      <!-- - **本质**：说白了，就是两个系统：初值方程和特征线方程，它们之间的选择代入
        - 我们的目的是用特征线方程代入初值方程，且要约去 $C$ 保留 $t$
        - 数学本质只考虑一个符号是变量还是常量，以及它们的表出关系是什么。因此相对底层，比较好用，但不太好理解
        - 而数学意义就要结合几何等进行思考 -->
      - **理解**：
        - 首先确定**特征线的位置**（截距）$C(x,t)$
        - 然后代入 $t=0$ 到特征线中，求出**特征线初值位置** $x|_{t=0} = x(C)$
        <!-- - 此时不能再代入 $C(x,t)$，否则会回去。因为已经将 $t$ 取为了0，不能再把它当作未知数（变量）
          - 数学意义来看，我们此时还在 $(x|_{t=0},0)$ 这一点上，需要等到沿当前特征线平移到 $(x,t)$ 处，代换才有意义 -->
        - 将 $x(C)$ 代入初值方程时，相当于求 $(x|_{t=0},0)$ 上的值（**初值**）
        - 再将 $C(x,t)$ 重新代入时，相当于从初值位置沿特征线平移到 $(x,t)$
          - 此时 $t$ 在初值系统中变成了不定量，也就是得到的结果就是**通解**
    - **求解**：
      - 求初值 $v(-\tan C,0) = -C$
      - 沿特征线方向常值性得通解为 $v(x,t) = -C(x,t) = \arctan x - t$

## 非齐次形式

- **Duhamel原理**：详见[热方程部分](2.热方程.md)
  - 此时 $\dfrac{dz}{ds} = f(x+sb,t+s)$
  - 则 $u(x,t) - g(x-tb) = z(0) - z(-t) = \int^0_{-t} f(x+sb,t+s)ds$
  - 此即为解

### 习题

- **输运方程的变形**：
  - $\begin{cases} u_t + \vec b\cdot Du + cu = 0 & \R^n\times(0,\infty) \\ u = g & \R^n\times\{t = 0\} \end{cases}$
  - 初等因子法解得 $u = g(x-bt)e^{-ct}$
