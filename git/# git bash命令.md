# git bash命令

- 差不多算是Linux和cmd的综合

## 命令行

### 操作路径

- pwd：显示当前路径
- cd []：进入指定路径
- ls：列出当前目录文件  ll：更详细地列出目录文件
- clear：清屏

### 操作文件

- mkdir []：创建文件夹
- touch []：新建文件 
- mv [a] src [b]：移动文件a到文件夹b的地址（它俩在同一目录下）
- rm []：删除文件   rm -r []：删除文件夹

### 基本命令

- history：显示历史
- help：显示帮助
