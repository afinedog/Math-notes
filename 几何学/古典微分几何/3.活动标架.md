# 标架与曲面基本定理

- **曲线关系总结**： 
  - **参数形式**：
    - 向量导数不与原向量垂直
    - 切向量长度为 $\dfrac{ds}{dt}$
  - **弧长形式**：
    - 向量导数与原向量垂直
    - 切向量长度为 $1$
    - 曲率方程
    - 挠率方程
- **曲面关系总结**：
    - $r_u, r_v$ 张成切平面，一般不垂直，但均与 $n$ 垂直
    - $n_u, n_v$ 也张成切平面，一般不垂直，可用W变换得到

- 本章之前都是从曲面上截取两个方向的曲线，将曲线性质张成曲面性质
  - 研究曲线使用的是Frenet标架，但研究曲面必须用更一般的标架
- 本章则是直接从一般标架出发，通过标架的微小运动建立微分方程，逐点研究曲面性质（所以计算复杂了起来）

## 一般的标架

### 两种标架

- **曲面的向量场**：曲面上一点 $r(u,v)$ 出发的向量 $x(u,v)$，光滑依赖于相同参数 $u,v$
  - **实例**：
    - 切向量场 $r_u,r_v$
    - 法向量场 $n$
- **自然标架（场）**：$\{r(u,v);\pad r_u,r_v,n\}$
- **活动标架**：$\{r(u,v);\pad x_1(u,v),x_2(u,v),x_3(u,v)\}$
  - **理解**：三个处处线性无关的向量场
  - **正交标架**：三个向量场均正交
    - **存在性**：
      - **证明**：对自然标架正交化即可
    - **方向**：反向变换 $s\to -s$ 后，$e_1,e_3$ 改变符号

### 正交标架的运动方程（初步）

- **正交标架的运动方程**：由标架的基性，可设 $\dfrac{de_i}{ds} = \sum\limits^3_{j=1} q_{ij}e_j$
  - **反对称性**：由正交性，取内积即得 $q_{ij} + q_{ji} = 0$
  - **变换**：将 $$\{e_i\}$ 进行正交变换，则 $\kappa = \sqrt{q^2_{12}+q^2_{13}}$ 保持不变
  - **Frenet标架**：选取满足 $\wt q_{13} = -\sin\t\cdot q_{12} + \cos\t\cdot q_{12} = 0$ 的 $\t$ 对应的正交变换，则 $\{\wt e_i\}$ 满足挠率方程，是Frenet标架

### 自然标架的运动方程

- **符号约定**：$r(u,v) = r(u^1,u^2)$<font color='chartreuse'>（自变量上标表示序号，自变量下标表示求偏导的序号）</font>
  - $g_{\a\b} = \lang r_\a,r_\b \rang，b_{\a\b} = \lang r_{\a\b},n \rang，\a,\b = 1,2$
  - $(g_{\a\b})，(b_{\a\b})$ 分别是曲面基本形式的系数矩阵
  - $(g_{\a\b})^{-1} = (g^{\a\b})，g = \det(g_{\a\b})$
- **爱因斯坦求和约定**：一个单项式中，若一个指标字母在上标和下标中都出现一次，则该式就是对该指标的求和式
  - **实例**：
    - **链式法则**：$dr = r_\a du^\a$，$I = g_{\a\b}du^\a dv^\b，II = b_{\a\b}du^\a dv^\b$
    - **矩阵乘积**：
      - **B矩阵（W变换正交系数阵）**：$b^\b_\a = b_{\a\g}g^{\g\b}$
      - **单位矩阵**：$\d^\a_\a = g^{\a\b}g_{\b\a} = 1+1 = 2$
      - **Jacobi矩阵**：
        - 设 $a^i_\a = \cfrac{\pa \wt u^i}{\pa u^\a}，a^\a_i = \cfrac{\pa u^\a}{\pa \wt u^i}$，则此时 $J = (a^i_\a)，J^{-1} = (a^\a_i)$
      - <font color='chartreuse'>下标表示左矩阵的行序号，上标表示右矩阵的列序号，即矩阵乘积的第 $(\a,\b)$ 元</font>
    - **推论（基变换）**：
      - $(\wt g_{ij}) = (a^\a_i)(g_{\a\b})(a^\a_i)$，从而 $\wt g_{ij} = a^\a_i\cdot g_{\a\b}\cdot a^\b_j$
        - $\wt b_{ij}，g_{\a\b}$ 同理
      - $\wt \G^k_{ij} = \G^\g_{\a\b}\cdot a^\a_i\cdot a^\b_j\cdot\wt a^k_\g + \cfrac{\pa a^\a_i}{\pa \wt u^j}\cdot \wt a^k_\a$
    - **平均曲率**：$H = \dfrac{1}{2}b_{\a\b}g^{\a\b}$
- **Christoffel符号**：
  - **第一类**：$\G^\g_{\a\b} = \cfrac{1}{2}\cdot g^{\g\xi}\hkh{\cfrac{\pa g_{\a\xi}}{\pa u^\b} + \cfrac{\pa g_{\b\xi}}{\pa u^\a} - \cfrac{\pa g_{\a\b}}{\pa u^\xi}}$
    - **推导**：
      - 已知 $\{r_\g，n\}$ 构成自然标架，故可设
        - 法向量 $r_{\a\b} = \G^\g_{\a\b} \vec r_\g + C_{\a\b}\vec n$
        - 切向量 $n_\a = D^\b_\a r_\b + D_\a \vec n$
      - 将切向量 $r_\g$、法向量 $n$ 与两个等式内积，即可求出待定系数的表达式<font color='chartreuse'>（也即自然标架的运动方程）</font>
  - **第二类**：$\G_{\g\a\b} = g_{\g\xi}\G^\xi_{\a\b}$
    - **理解**：将第一类符号中的 $g$ 符号删去
- **自然标架的运动方程**：
  - $\begin{cases} \cfrac{\pa r}{\pa u^\a} = r_\a \\\\ \cfrac{\pa r_\a}{\pa u^\b} = \G^\g_{\a\b} r_\g + b_{\a\b}n \\\\ \cfrac{\pa n}{\pa u^\a} = -b^\b_\a r_\b \end{cases}$
  - **理解**：自然标架在运动时，各个方向上向量的变化，可以完全由这三个无关方程导出
  - **本质**：用标架的向量基表出微小运动。或者说，把向量的微分转化为系数的微分

  - **解法**：首先正交化，然后利用简化公式即可
- 求曲面几何量的爱因斯坦表示

## 曲面的参数表示

### 曲面结构方程

- **本质**：由曲面 $S$ 的正则性，$S$ 上某点的自然标架中，对 $r_\a$ 混合求偏导的结果相等，再结合自然标架的运动方程，即可导出以下方程
- **Gauss方程**：$\ddkh{\cfrac{\pa \G^\xi_{\a\b}}{\pa u^\g} - \cfrac{\pa \G^\xi_{\a\g}}{\pa u^\b}} + \ddkh{\G^\eta_{\a\b}\G^\xi_{\eta\g} - \G^\eta_{\a\g}\G^\xi_{\eta\b}} - \ddkh{b_{\a\b}b^\xi_\g - b_{\a\g}b^\xi_\b} = 0$
  - **理解**：$r_\xi$ 系数相等导出的式子
  <br>
- **Codazzi方程**：$\ddkh{\cfrac{\pa \G^\xi_{\a\b}}{\pa u^\g} - \cfrac{\pa \G^\xi_{\a\g}}{\pa u^\b}} + \ddkh{\G^\eta_{\a\b}\G^\xi_{\eta\g} - \G^\eta_{\a\g}\G^\xi_{\eta\b}} = 0$
  - **理解**：$n$ 系数相等导出的式子
  <br>
- **结构方程**：Gauss-Codazzi方程，由下面的理论，其可完全决定曲面的形状
  - **本质**：结构方程是标架运动方程的可积条件

### 曲面存在唯一性定理

- **唯一性定理**：同一参数域 $D$ 上的两个曲面，若在参数域上任意一点均有相同的 $I,II$，则两曲面相差一个 $E^3$ 刚体运动
  - **证明**：取 $u_0\in D$
    - 由 $I,II$ 相同得，在该点 $|r_\a| = |\wt r_{\a}|，\lang r_1,r_2 \rang = \lang \wt r_1,\wt r_2 \rang$
      - 从而自然标架 $\{r;\pad r_1,r_2,n\}$ 中，向量长度和夹角均相等，且定向相同
      - 从而两标架相差一个刚体运动
    - 再由运动方程和 $I,II$ 形式的系数具有唯一表出关系，得两标架的运动方程也相同
      - 再由运动方程是一阶PDE方程组，由初值问题解的唯一性即得 $r = \wt r$
- **存在性定理**：
  - 设 $\begin{cases} \p = g_{\a\b}du^\a du^\b \\ \psi = b_{\a\b}du^\a du^\b \end{cases}$ 是 $D$ 上两个二次微分式
  - 若 $(g_{\a\b}),(b_{\a\b})$ 对称，前者正定，且满足Gauss方程和Codazzi方程
  - 则 $D$ 上任意一点 $u_0$ 均存在邻域 $U$，使得 $U$ 上曲面以 $\p,\psi$ 为 $I,II$
  - **证明**：
    - 首先，给出自然标架的运动方程形式，由一阶PDE理论，其有解 $\LR r,r_\a,n$ 均可混合求偏导 $\LR$ Gauss-Codazzi方程成立，从而方程组可解
    - 取初值 $\begin{cases}  \end{cases}$
  - **理解**：上面所有章节的意义是，给出标架和曲面的唯一参数表示

### 习题

- 给定 $I,II$ 形式，验证是否存在该曲面，并写出曲面参数方程
  - **解法**：验证则用 $g_{\a\b}$ 代入结构方程即可，求参数方程则需合理简化后，求解微分方程组
 
## 参数的正交规范化

### 正交标架

- **规范标架**：$e_1 = \cfrac{r_u}{|r_u|}，e_2 = \cfrac{1}{\sqrt{E}}\cdot\cfrac{Er_v-Fr_u}{\sqrt{EG-F^2}}，e_3 = e_1\land e_2$
  - **本质**：将自然标架进行Schmidt正交化
- **微分系数**：
  - $dr = w_1 e_1 + w_2 e_2$
  - $dn = de_3$
- **运动方程**：
  - $\begin{cases} dr = w_1e_1 + w_2e_2 \\ de_i = \sum\limits^3_{j=1} w_{ij}e_j \end{cases}$
    - **一阶投影系数**：$w_i = \lang dr,e_i \rang$
      - **几何意义**：曲线微分在基方向上的投影
    - **二阶投影系数**：$w_{ij} = -w_{ji} = \lang de_i,e_j \rang，w_{ii} = 0$
      - **反称性**：由正交性，$\lang e_i,e_j \rang = \d_{ij}$，展开即得 $(w_{ij})$ 是反对称矩阵<font color='chartreuse'>（其实之前已经说了）</font>
      - **几何意义**：基向量微分在基方向上的投影
- **曲面基本形式**：
  - $I = \lang dr,dr \rang = w_1^2 + w_2^2$
    - **本质**：勾股定理
  - $II = -\lang dr,e_3 \rang = w_1w_{13} + w_2w_{23}$
    - **本质**：曲线微分投影到基方向上，然后再微分投影到基方向上，即二维导数的投影<font color='chartreuse'>（由渐近展开、曲率定义等前面的讨论，已知二阶导可反映曲面弯曲度）</font>
  - **曲面本身性质**：$I$ 与正交标架的选取无关，$II$ 与法向相同的正交标架选取无关
    - **证明**：即证两个基本形式的旋转不变性


### 参数变换

- **一阶微分**：设 $\bvec{w_1 \\ w_2} = A\bvec{du \\ dv}，A = \bvec{a_{11} & a_{12} \\ a_{21} & a_{22}} $
  - 由 $I$ 得 $A^TA = \bvec{E & F \\ F & G}$
  - **理解**：
    - $I$ 形式二次型下，正交基系数 $w_i$ 对应单位矩阵，而自然基系数 $EFG$ 对应系数矩阵，若它们有过渡关系，在二次型下当然就表现为合同关系
  - **本质**：$I$ 的参数系数阵合同于单位矩阵
- **二阶微分**：设 $\bvec{w_{13} \\ w_{23}} = B\bvec{w_1 \\ w_2}，B = \bvec{h_{11} & h_{12} \\ h_{21} & h_{22}} $
  - **特征定理**：
    - $B$ 的特征值是主曲率
    - $K = \det B$
    - $H = \dfrac{1}{2}\text{tr}\ B$
    - **证明**：
      - 由行列式和迹的交换性，可将 $B$ 转化为 $\wt B(AA^T)^{-1}$ 形式
      - 再由 $AA^T$ 为 $I$ 系数阵，$\wt B$ 为 $II$ 系数阵，从而 $\wt B(AA^T)^{-1}$ 即为W变换下自然基 $r_u,r_v$ 的系数阵
      - 将B特征变换后，易得 $\det B = k_1k_2 = K$，$\text{tr}\ B = \dfrac{k_1+k_2}{2} = H$
  - **映射意义**：W变换在 $e_1,e_2$ 下的系数阵
    - **证明**：
      - 由过渡矩阵在变换下不变，得 $\mc W(r_\a) = A\mc W(e_\a)$
      - 再由先前推导，$\mc W(r_\a) = \wt B(AA^T)^{-1} r_\a$，以及 $r_\a = A^{-1}e_\a$
        - 即得 $\mc W(e_\a) = A^{-1}\wt B(A^T)^{-1}e_\a$
        - 从而系数阵为 $B$
    - **本质**：将 $e_1,e_2$ 看作正交参数的 $r_u，r_v$，再结合前面的W变换定义，很容易发现这点
    - **推论**：$II = k_1w_1^2 + k_2w^2_2$

## 参数的外微分形式

- **参数空间上的外微分**：[外微分](../../数学分析/多元函数积分学/4.外微分.md)
  - <font color='chartreuse'>数分中，为了迎合重积分的思想，将外微分看作高维体积微元
  - 在古典微分几何中，外微分需要被重新赋予数学意义（说到底，它就是个工具）</font>
- **定义**：
  - **零阶外微分**：函数 $f(u,v)$
    - $df = \dfrac{\pa f}{\pa u}du + \dfrac{\pa f}{\pa v}dv$
  - **一阶外微分**：$\t = f_1du + f_2dv$
    - $d\t = df\land du$
  - **二阶外微分**：$\p = fdu\land dv$
    - $d\p = 0$
    - <font color='chartreuse'>空间局限在三维欧氏，故外微分最多定义到2阶</font>
- **推论公式**：
  - $d(df) = 0$
  - $d(f\p) = df\land \p + fd\p$
  - $d(\p f) = d\p f - \p\land df$
- **变换公式**：
  - $w_1\land w_2 = \det A\cdot du\land dv \whhh = dA$
  - $w_{13}\land w_{23} = \det B\cdot w_1\land w_2 \whh\qquad = KdA$

### 结构方程

- **一阶微分部分**：$\begin{cases} dw_1 = w_2\land w_{21} \\ dw_2 = w_1\land w_{12} \end{cases}$
  - **求解**：$d(dr) = 0$ 即可
- **二阶微分部分**：$\begin{cases} dw_{12} = w_{13}\land w_{32} = -KdA & \text{Gauss} \\ dw_{13} = w_{12}\land w_{23} & \text{Codazzi(1)} \\ dw_{23} = w_{21}\land w_{13} & \text{Codazzi(2)}\end{cases}$
  - **求解**：$d(de_\a) = 0$ 即可

### 正交参数计算公式

- 设 $u,v$ 是正交参数，则 $I = Edu^2 + Gdv^2$
  - 易得 $w_1 = \sqrt{E}，w_1 = \sqrt{G}$
  - 取 $e_1 = \dfrac{r_u}{\sqrt{E}}，e_2 = \dfrac{r_v}{\sqrt{G}}$
  - 由方程
    - $dw_1 = w_{12}\land w_2$
    - $dw_2 = w_{21}\land w_1$
  - 由外微分公式
    - $dw_1 = \sqrt{E}_vdv\land du = -\dfrac{(\sqrt{E})_v}{\sqrt{G}}du \land w_2$
    - $dw_2 = \sqrt{G}_udu\land dv = -\dfrac{(\sqrt{G})_u}{\sqrt{E}}dv \land w_1$
  - 综上即得计算公式
    - $\dis w_{12} = -w_{21} = -\frac{(\sqrt{E})_v}{\sqrt{G}}du + \frac{(\sqrt{G})_u}{\sqrt{E}}dv$
- 直接计算内积即可求出 $w_{13},w_{23}$ 
- 即得
  - $\dis dw_{12}  = \Big[-\Big(\frac{(\sqrt{E})_v}{\sqrt{G}}\Big)_v + \Big( \frac{(\sqrt{G})_u}{\sqrt{E}} \Big)_u \Big]du\land dv = Kdu\land dv$
  - $w_{13}\land w_{23} = -w_{23}\land w_{13}  = -\cfrac{LN-M^2}{\sqrt{EG}}du\land dv$
  

### 总结

- 标架的运动方程
  - 对 $r,e_i$ 求微分，用正交基 $e_i$ 表示
- 曲面的结构方程
  - 对 $w_i,w_{ij}$ 求微分，用一/二阶微分形式 $w_i,w_{ij}$ 表示
- 曲面的几何量
  - 第一基本形式
  - 面积元：$dA = w_1\land w_2 = \det A\cdot du\land dv$
  - Gauss映射面积元：$d\sigma = w_{13}\land w_{23} = Kdu\land dv$
  - 第二基本形式
  - 第三基本形式：$III = \lang de_3,e_3 \rang = w_{13}^2 + w_{23}^2$
  - Hopf不变式：$\psi = w_1w_{23} - w_2w_{13}$
