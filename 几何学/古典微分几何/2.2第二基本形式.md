# 曲面的第二基本形式


## 第二基本形式 $\lang d^2r,n \rang$

- **第二基本形式**：设 $p\in S$，则 $Q:T_p(S)\times T_p(S)\to \R，\bs v\mapsto \Bip{dN_p(\bs v),\bs v}$ 称为曲面的第二基本形式
  - **参数微分形式**：$II = \lang d^2\bs r,\bs n \rang = -\lang d\bs r,d\bs n \rang = Ldudu + 2Mdudv + Ndvdv$
    - 其中 $\begin{cases} L = \lang r_{uu},n \rang = -\lang r_u,n_u \rang \\ M = \lang r_{uv},n \rang = -\lang r_u,n_v \rang = -\lang r_v,n_u \rang \\ N = \lang r_{vv},n \rang = -\lang r_v,n_v \rang \end{cases}$
    - **证明**：对 $\lang r_u,n \rang = 0$ 和 $\lang r_v,n \rang = 0$ 分别求偏导即可
  - **几何意义**：
    - 切向量变化量在法向方向上的投影
    - 曲面的弯曲程度
      - <font color='chartreuse'>曲面运动量变化后，到原本切平面的距离变化量。其表示曲面的形状（在当前点的弯曲程度）</font>
  - **实例**：
    - 平面和柱面的 $I$ 相同，但 $II$ 不同
      - 按相同速度沿曲线运动时，曲线的长度变化速率相同，但曲线的法向变化速率不同
- **欧氏变换定理**：刚体运动下 $II$ 相等，反向刚体运动下改变符号
  - **证明**：已知 $n = r_u\land r_v$，由正交变换对外积的符号影响，$n$ 的符号取决于正反向。再由正交变换内积不变性，得 $II$ 的形式不变
- **参数变换定理**：同向参数变换不改变 $II$ 的符号，反向参数变换则会改变
  - **证明**：已知同向参数变换不改变 $n$，再由一阶微分形式不变性得微分也不变


### 法曲率

- **法曲率**：
  - **投影定义**：
    - 设 $S$ 是正则曲面，$\bs N$ 是曲面法向量，$p\in S$
    - 设 $\a$ 是曲面上经过点 $p$ 的曲线，$k$ 是曲线曲率，$\bs n$ 是曲线主法向量
    - 则 $k_n = k\cos\lang \bs n,\bs N \rang$ 称为**法截线** $\a$ 在 $p$ 处的法曲率
      <!-- - 法挠率：由于以 $w$ 为切方向的曲线有无限个，它们的挠率和更高阶导数有关，不仅仅由单点决定，从而这些曲线可以有无限种挠率，即挠率是个自由对象，故曲面没有法挠率 -->
    - <font color='red'>（曲线的曲率向量 $\bs t'$）在（曲面主法向 $\bs N$）的投影</font>
  - **内积形式**：$$k_n(w)  = \ip{\cfrac{d^2\bs r}{ds^2},\bs n}$$
    - <font color='red'>法曲率是切向量 $w$ 的函数，由切方向和切点共同决定
    - 几何意义：曲面上某个曲线的曲率（曲面沿某个切向量的弯曲程度）</font>
    - **证明**：由曲线曲率的内积表述直得结论
  - **基本形式（弧长参数）**：$$k_n(w) = \cfrac{II(w)}{I(w)} = L\dkh{\cfrac{du}{ds}}^2 + 2M \cfrac{du}{ds}\cfrac{dv}{ds} + N\dkh{\cfrac{dv}{ds}}^2$$
    - <font color='chartreuse'>它对应前面的基本形式参数微分形式</font>
    - **证明**：
      - 由法曲率的内积形式 + 第二基本形式的内积形式 + 第一基本形式的弧长意义，即得结论
  - **基本形式（时间参数）**：设切向量 $w = \l r_u + \mu r_v$，则 $$k_n(w) = \cfrac{II(w)}{I(w)} = \cfrac{L\l^2 + 2M\l\mu + N\mu^2}{E\l^2 + 2F\l\mu + G\mu^2}$$
    - <font color='chartreuse'>它对应前面的切向量长度公式</font>
    - **证明1（由基本形式导出）**：
      - 由于 $w = \bs r'(t)$，全微分公式即得 $\l = u'(t)，\mu = v'(t)$
      - 由切向量长度公式，$I(w) = E\l^2 + 2F\l\mu + G\mu^2$
      - 由第二基本形式的参数微分形式，$II(w) = L\l^2 + 2M\l\mu + N\mu^2$
      - 综上即可
    - **证明2（由内积形式导出）**：
      - 由速度-弧长公式，$\cfrac{d\bs r}{ds}$ 是单位切向量
      - 单位切向量的第二基本形式就是法曲率，故 $k_n(w) = \dfrac{II(w)}{|w|^2}$
- **Mousnier定理**：在 $p$ 点处切线相同的所有曲线，其法曲率相同
  - **证明**：显然
- <font color='red'>看起来法曲率的几何意义更直观，为什么不把它定义为第二基本形式呢？
  - 首先，法曲率是依赖于方向的量，而"基本形式"必须能够表示出该点"所有方向的信息"
  - 其次，基本形式在参数下都是二次型。因为"曲面的长度和弯曲度"可以用"切向量的长度和弯曲度"表达。而切向量是一次微分，故基本形式就是二次微分。再由齐次性 + 双线性，只能是二次型
    - 微分几何本质是通过微分得到局部同胚平面，从而转化为线性代数
    - 法曲率是零次齐次型，无量纲，故只与方向有关，与长度无关</font>


### 应用


- **曲面局部形状分类**：由顺序主子式理论，二次型 $II(du,dv)$ 的判别式为 $LN-M^2$
  - **椭圆点**：判别式为正，$II$ 正定，局部为凸平面
    - 存在极值点
    - 任何切向的法曲率符号相同，即任意点的任意方向上弯曲同向
      - <font color='chartreuse'>法曲率也是 $du,dv$ 的二次型，类比二次函数是否正定</font>
  - **双曲点**：判别式为负，$II$ 负定，局部为马鞍面
    - 不存在极值点，即一个方向上是极大/小值的话，其正交方向必定会减少/增加
    - **渐近方向**：存在两个方向使法曲率为0，划分出的对称区域上法曲率符号相同
      - <font color='chartreuse'>相当于二次函数存在两个解，再加上极值点，将二次函数分为四段，每通过一段时改变符号</font>
  - **抛物点**：判别式为0，$II$ 退化，局部为曲线
    - **渐近方向**：仅有一个方向使法曲率为0，划分出的两个区域上法曲率符号相同
      - <font color='chartreuse'>相当于二次函数和坐标轴相切</font>
  - **平点**：$L = M = N$
    - 法曲率恒为0



## 外恩加藤变换

- **曲面的Weingarten变换**：描述曲面上沿曲线运动时，主法向量的转动情况
  - **微分形式**：$\mc W = -dN_p:T_P(S)\to T_P(S)，d\bs r\mapsto (-d\bs n)$
  - **基本形式（形状算子）**：$\mc W = I^{-1}II$
    - **证明**：
  - **参数形式**： $$\mc W : T_P(S)\to T_P(S)，\Big[ \l r_u + \mu r_v \Big] \mapsto \Big[-(\l n_u + \mu n_v)\Big]$$
    - **证明（良定义性）**：
      - **陪域良定义**：易得 $\dfrac{dn}{dt} = n_u\dfrac{du}{dt} + n_v\dfrac{dv}{dt}$
        - 再由于 $n$ 是单位向量，且曲面光滑，故 $n$ 只能在单位球上连续变动，故 $dn\perp n$，从而 $n_u,n_v$ 都是切向量
      - **对应性**：由于 $n_u,n_v$ 只与曲面参数 $u,v$ 有关，参数曲线切向量 $(\dfrac{du}{dt},\dfrac{dv}{dt})$ 只与平面曲线 $(u(t),v(t))$ 的表达式选取有关，与参数速度 $t$ 的选取无关
        - 综上即得 $\dfrac{dn}{dt}$ 唯一对应一个参数约束（一个曲面曲线）。从而 $\mc W$ 是一个双射
  - **几何意义**：
    - 曲面参数 $u,v$ 表达式，本质是用两个线性无关方向上的三维曲线来表出曲面
    - $r$ 是位矢，其 $u$ 偏导是 $u$ 方向上三维曲线切向量
    - $n$ 是法向量，其 $u$ 偏导是 $u$ 方向上三维曲线转向向量 + 偏转向量
    - **推论**：$r_u$ 和 $n_u$ 的方向一般不同。因为在 $u$ 曲线上前者张成密切平面，总与 $b$ 垂直。而后者在密切平面和法平面均有投影（详见曲线理论挠率部分）
      <img src = "./图片/n导数.png">
  - **参数变换**：该变换与曲面参数选取无关
    - **证明**：设 $v = \l r_u + \mu r_v = ar_{\ol u} + br_{\ol v}$
      - 由Jacobi矩阵的基变换性，$\l = a\dfrac{\pa u}{\pa \ol u} + b\dfrac{\pa u}{\pa \ol v}，\mu = a\dfrac{\pa v}{\pa \ol u} + b\dfrac{\pa v}{\pa \ol v}$
      - 代入变换即可
    - **本质**：曲面上任意一个三维曲线，其切向量与主法向量偏移量的对应计算式系数相同
    - **理解**：因为三维曲线本身的理论就足以得出计算式了，所以和曲面理论无关
  - **自伴性**：$\lang \mc W(v),w \rang = \lang v,\mc W(w) \rang$
    - **证明**：由 $\lang r_u,n_v \rang = \lang r_{uv},n \rang = \lang r_v,n_u \rang$ 直得
    - **几何理解**：
      - 同上，可选曲线满足 $\l = \dfrac{du}{ds},\mu = \dfrac{dv}{ds}$，故 $v = \dfrac{dr}{ds}，\mc W(v) = -\dfrac{dn}{ds}$
      <br>h
      - 显然 $\lang \underset{\substack{\\ v曲线}}{\dfrac{dr}{ds}}, \underset{\substack{\\ w曲线}}{-\dfrac{dn}{ds}}\rang = \lang \underset{\substack{\\ w曲线}}{\dfrac{dr}{ds}}, \underset{\substack{\\ v曲线}}{-\dfrac{dn}{ds}}\rang$<font color='chartreuse'>（不是一个方向，从而不是法曲率）
        - 要证明就将两个内积展开即可</font>
      <br>
    - **本质**：曲面某点的 $I$ 唯一，即所有经过该点的曲线的弧长微分 $ds$ 恒定
      - 再由该点的 $II$ 唯一，即 $\dfrac{II}{ds}$ 恒定，从而得到结论

### 应用

- **法曲率的外恩加藤形式**：设 $w$ 是单位切向量，则 $k_n(w) = \Bip{\mc W(w),w}$

## 习题


- 求曲面的第二基本形式
- **自伴性**：$dN_p:T_p(S)\to T_p(S)$ 是对称线性映射
  - **证明**：
    - 由微分映射的线性易得它是线性映射
    - 设 $X$ 是参数化，则 $\{X_u,X_v\}$ 是 $T_p(S)$ 的一组基，$\a(t)$ 是相应参数曲线，则 $dN_p\Big( \a'(0) \Big) = \cfrac{d}{dt}N_p\Big( u(t),v(t) \Big)\biggm|_{t=0} = dN_p(X_u)u'(0) + dN_p(X_v)v'(0)$
    - <font color='red'>上面的中间微分不妨简写为 $N_u,N_v$，只需证明 $\lang N_u,X_v \rang = \lang X_u,N_v \rang$ 即可</font>
    - 由法向量垂直性，$\lang N,X_u \rang = \lang N,X_v \rang = 0$。分别关于 $u,v$ 求导即得等式
- **曲面基本形式的应用**
  - 设 $w = ar_u + br_v$ 是切向量，则
    - $I(w)$ 代表该点以 $w$ 为切向量的曲线的弧长微分的平方，也即 $|w|^2$
    - $II(w)$ 代表该点以 $w$ 为切向量的曲线的（单位切向量微分），也即方向导数 $\vec t_w = \dfrac{1}{\sqrt{a^2+b^2}}(at_u + bt_v) = \cos\t\cdot t_u + \sin\t\cdot t_v$
