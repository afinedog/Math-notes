# 曲面的第二基本形式


## 第二基本形式 $\lang d^2r,n \rang$

- **第二基本形式**：设 $p\in S$，则 $Q:T_p(S)\times T_p(S)\to \R，\bs v\mapsto \Bip{dN_p(\bs v),\bs v}$ 称为曲面的第二基本形式
  - **参数微分形式**：$II = \lang d^2\bs r,\bs n \rang = -\lang d\bs r,d\bs n \rang = Ldudu + 2Mdudv + Ndvdv$
    - 其中 $\begin{cases} L = \lang r_{uu},n \rang = -\lang r_u,n_u \rang \\ M = \lang r_{uv},n \rang = -\lang r_u,n_v \rang = -\lang r_v,n_u \rang \\ N = \lang r_{vv},n \rang = -\lang r_v,n_v \rang \end{cases}$
    - **证明**：对 $\lang r_u,n \rang = 0$ 和 $\lang r_v,n \rang = 0$ 分别求偏导即可
  - **几何意义**：
    - 切向量变化量在法向方向上的投影
    - 曲面的弯曲程度
      - <font color='chartreuse'>曲面运动量变化后，到原本切平面的距离变化量。其表示曲面的形状（在当前点的弯曲程度）</font>
  - **实例**：
    - 平面和柱面的 $I$ 相同，但 $II$ 不同
      - 按相同速度沿曲线运动时，曲线的长度变化速率相同，但曲线的法向变化速率不同
- **欧氏变换定理**：刚体运动下 $II$ 相等，反向刚体运动下改变符号
  - **证明**：已知 $n = r_u\land r_v$，由正交变换对外积的符号影响，$n$ 的符号取决于正反向。再由正交变换内积不变性，得 $II$ 的形式不变
- **参数变换定理**：同向参数变换不改变 $II$ 的符号，反向参数变换则会改变
  - **证明**：已知同向参数变换不改变 $n$，再由一阶微分形式不变性得微分也不变


### 法曲率

- **法曲率**：
  - **投影定义**：
    - 设 $S$ 是正则曲面，$\bs N$ 是曲面法向量，$p\in S$
    - 设 $\a$ 是曲面上经过点 $p$ 的曲线，$k$ 是曲线曲率，$\bs n$ 是曲线主法向量
    - 则 $k_n = k\cos\lang \bs n,\bs N \rang$ 称为**法截线** $\a$ 在 $p$ 处的法曲率
      <!-- - 法挠率：由于以 $w$ 为切方向的曲线有无限个，它们的挠率和更高阶导数有关，不仅仅由单点决定，从而这些曲线可以有无限种挠率，即挠率是个自由对象，故曲面没有法挠率 -->
    - <font color='red'>（曲线的曲率向量 $\bs t'$）在（曲面主法向 $\bs N$）的投影</font>
  - **内积形式**：$$k_n(w)  = \ip{\cfrac{d^2\bs r}{ds^2},\bs n}$$
    - <font color='red'>法曲率是切向量 $w$ 的函数，由切方向和切点共同决定
    - 几何意义：曲面上某个曲线的曲率（曲面沿某个切向量的弯曲程度）</font>
    - **证明**：由曲线曲率的内积表述直得结论
  - **基本形式（弧长参数）**：$$k_n(w) = \cfrac{II(w)}{I(w)} = L\dkh{\cfrac{du}{ds}}^2 + 2M \cfrac{du}{ds}\cfrac{dv}{ds} + N\dkh{\cfrac{dv}{ds}}^2$$
    - <font color='chartreuse'>它对应前面的基本形式参数微分形式</font>
    - **证明**：
      - 由法曲率的内积形式 + 第二基本形式的内积形式 + 第一基本形式的弧长意义，即得结论
  - **基本形式（时间参数）**：设切向量 $w = \l r_u + \mu r_v$，则 $$k_n(w) = \cfrac{II(w)}{I(w)} = \cfrac{L\l^2 + 2M\l\mu + N\mu^2}{E\l^2 + 2F\l\mu + G\mu^2}$$
    - <font color='chartreuse'>它对应前面的切向量长度公式</font>
    - **证明1（由基本形式导出）**：
      - 由于 $w = \bs r'(t)$，全微分公式即得 $\l = u'(t)，\mu = v'(t)$
      - 由切向量长度公式，$I(w) = E\l^2 + 2F\l\mu + G\mu^2$
      - 由第二基本形式的参数微分形式，$II(w) = L\l^2 + 2M\l\mu + N\mu^2$
      - 综上即可
    - **证明2（由内积形式导出）**：
      - 由速度-弧长公式，$\cfrac{d\bs r}{ds}$ 是单位切向量
      - 单位切向量的第二基本形式就是法曲率，故 $k_n(w) = \dfrac{II(w)}{|w|^2}$
- **Mousnier定理**：在 $p$ 点处切线相同的所有曲线，其法曲率相同
  - **证明**：显然
- <font color='red'>看起来法曲率的几何意义更直观，为什么不把它定义为第二基本形式呢？
  - 首先，法曲率是依赖于方向的量，而"基本形式"必须能够表示出该点"所有方向的信息"
  - 其次，基本形式在参数下都是二次型。因为"曲面的长度和弯曲度"可以用"切向量的长度和弯曲度"表达。而切向量是一次微分，故基本形式就是二次微分。再由齐次性 + 双线性，只能是二次型
    - 微分几何本质是通过微分得到局部同胚平面，从而转化为线性代数
    - 法曲率是零次齐次型，无量纲，故只与方向有关，与长度无关</font>


### 应用


- **曲面局部形状分类**：由顺序主子式理论，二次型 $II(du,dv)$ 的判别式为 $LN-M^2$
  - **椭圆点**：判别式为正，$II$ 正定，局部为凸平面
    - 存在极值点
    - 任何切向的法曲率符号相同，即任意点的任意方向上弯曲同向
      - <font color='chartreuse'>法曲率也是 $du,dv$ 的二次型，类比二次函数是否正定</font>
  - **双曲点**：判别式为负，$II$ 负定，局部为马鞍面
    - 不存在极值点，即一个方向上是极大/小值的话，其正交方向必定会减少/增加
    - **渐近方向**：存在两个方向使法曲率为0，划分出的对称区域上法曲率符号相同
      - <font color='chartreuse'>相当于二次函数存在两个解，再加上极值点，将二次函数分为四段，每通过一段时改变符号</font>
  - **抛物点**：判别式为0，$II$ 退化，局部为曲线
    - **渐近方向**：仅有一个方向使法曲率为0，划分出的两个区域上法曲率符号相同
      - <font color='chartreuse'>相当于二次函数和坐标轴相切</font>
  - **平点**：$L = M = N$
    - 法曲率恒为0



## 外恩加藤变换

- **曲面的Weingarten变换**：描述曲面上沿曲线运动时，主法向量的转动情况
  - <font color='red'>外恩加藤变换是逐点定义的，每换一个点就换一个形式</font>
  - **微分形式**：$\mc W = -dN_p:T_p(S)\to T_p(S)，d\bs r\mapsto (-d\bs n)$
  - **参数形式**： $$\mc W : T_p(S)\to T_p(S)，\Big[ \l r_u + \mu r_v \Big] \mapsto \Big[-(\l n_u + \mu n_v)\Big]$$
    - **证明（良定义）**：
      - **值域合法性**：<font color='red'>全体像都在陪域里</font>
        - 易得 $\dfrac{dn}{dt} = n_u\dfrac{du}{dt} + n_v\dfrac{dv}{dt}$
        - 再由于 $n$ 是单位向量，且曲面光滑，故 $n$ 只能在单位球上连续变动，故 $dn\perp n$，从而 $n_u,n_v$ 都是切向量
      - **单值性**：<font color='red'>像是唯一的</font>
        - 由于 $n_u,n_v$ 只与曲面参数 $u,v$ 有关，参数曲线切向量 $(\dfrac{du}{dt},\dfrac{dv}{dt})$ 只与平面曲线 $(u(t),v(t))$ 的表达式选取有关，与参数速度 $t$ 的选取无关
        - 综上即得 $\dfrac{dn}{dt}$ 唯一对应一个参数约束（一个曲面曲线）。从而 $\mc W$ 是一个双射
      - **几何相容性**：<font color='red'>该变换与曲面参数选取无关</font>
        - <font color='red'>因为该参数形式是根据坐标基 $\{r_u,r_v\}$ 定义的。如果不验证基变换不变性，那么它就只是一个表达式，而不是函数</font>
        - 设 $v = \l r_u + \mu r_v = ar_{\ol u} + br_{\ol v}$
        - 由Jacobi矩阵的基变换性，$\l = a\dfrac{\pa u}{\pa \ol u} + b\dfrac{\pa u}{\pa \ol v}，\mu = a\dfrac{\pa v}{\pa \ol u} + b\dfrac{\pa v}{\pa \ol v}$
        - 代入变换即可
  - **基本形式（形状算子）**：$\mc W = II\circ I^{-1}$
    - <font color='red'>内积（双线性函数）不能有逆，这里的 $I^{-1}$ 本质是它的诱导对偶同构（线性映射） $I:V\to V^*，X\mapsto I(X,\cdot)$ 的逆
    - 它具有深刻的几何意义，详见后面章节</font>
    - **证明（参数法）**：
      - 此时 $I = \bvec{E & F \\ F & G}$，$II = \bvec{L & M \\ M & N}$
      - 设 $\bvec{-n_u \\ -n_v} = W\tvec{r_u \\ r_v}$，显然 $W$ 就是外恩加藤变换的矩阵
      - 再由 $E,F,G,L,M,N$ 的内积定义，易得关系式 $II = WI$，两边取逆即可
    - **证明（内积法）**：
      - 由第二基本形式的定义，$II(\a,\b) = \lang W(\a),\b \rang = \lang \a,W(\b) \rang$
      - 再由第一基本形式的定义，$I\Big( W(\a),\b \Big) = \lang W(\a),\b \rang$
      - 由高等代数知识易得结论
- **良定义**：只要一个定义是无歧义的、逻辑上不矛盾的，就叫做良定义
  - 这是定义的基本要求，也是唯一要求
  - 根据定义的方法不同，良定义需要满足的性质也不同
  - **实例（用自变量表达式定义的函数）**：
    - 不能认为 $x^2$ 是一个函数，它只是一个表达式，我们不知道自变量是 $x$ 还是 $x^2$ 还是 $\sqrt{x}$ <font color='red'>（这叫做无歧义性）</font>
    - 也不能认为 $f(x) = x^2$ 是一个函数，因为它没有指明定义域和陪域，我们不知道 $x$ 是否可以平方运算<font color='red'>（这叫做定义域和陪域的合法性）</font>
    - 可以认为 $f:\R\to\R，x\mapsto x^2$ 是一个函数，但是我们还必须验证它是单值的<font color='red'>（这是我们在定义函数时，人为添加的性质。逻辑上其实可以单值）</font>
  - **实例（用切向量表达式定义的函数）**：
    - 上面的外恩加藤变换中，多了一个几何不变性。这是因为它的定义形式比自变量表达式形式要多绕了一个弯子（用的是切向量表达式）。而切向量与自变量之间的无歧义、无矛盾性，就是几何相容性
    - <font color='red'>数值分析中，ODE数值解的相容性实际上也是良定义的一部分</font>

### 性质

- **非平凡性**：外恩加藤变换不是伸缩变换（两个特征值一般不同）
  - **证明**：只需证明 $r_u$ 和 $n_u$ 的方向一般不同即可
    - 因为在 $u$ 曲线上前者张成密切平面，总与 $b$ 垂直。而后者在密切平面和法平面均有投影（详见曲线理论挠率部分）

- **对称线性**：$\lang \mc W(v),w \rang = \lang v,\mc W(w) \rang$
  - **证明**：由 $\lang r_u,n_v \rang = \lang r_{uv},n \rang = \lang r_v,n_u \rang$ 直得
  - **几何理解**：
    - 同上，可选曲线满足 $\l = \dfrac{du}{ds},\mu = \dfrac{dv}{ds}$，故 $v = \dfrac{dr}{ds}，\mc W(v) = -\dfrac{dn}{ds}$
    <br>h
    - 显然 $\lang \underset{\substack{\\ v曲线}}{\dfrac{dr}{ds}}, \underset{\substack{\\ w曲线}}{-\dfrac{dn}{ds}}\rang = \lang \underset{\substack{\\ w曲线}}{\dfrac{dr}{ds}}, \underset{\substack{\\ v曲线}}{-\dfrac{dn}{ds}}\rang$<font color='chartreuse'>（不是一个方向，从而不是法曲率）
      - 要证明就将两个内积展开即可</font>
- **对称线性**：$dN_p:T_p(S)\to T_p(S)$ 是对称线性映射
  - **证明**：
    - 由微分映射的线性易得它是线性映射
    - 设 $X$ 是参数化，则 $\{X_u,X_v\}$ 是 $T_p(S)$ 的一组基，$\a(t)$ 是相应参数曲线，则 $dN_p\Big( \a'(0) \Big) = \cfrac{d}{dt}N_p\Big( u(t),v(t) \Big)\biggm|_{t=0} = dN_p(X_u)u'(0) + dN_p(X_v)v'(0)$
    - <font color='red'>上面的中间微分不妨简写为 $N_u,N_v$，只需证明 $\lang N_u,X_v \rang = \lang X_u,N_v \rang$ 即可</font>
    - 由法向量垂直性，$\lang N,X_u \rang = \lang N,X_v \rang = 0$。分别关于 $u,v$ 求导即得等式

## 主曲率

- **主方向**：W变换的特征向量所对应方向
  - **存在性 + 正交性**：所有曲面点均存在两个彼此正交的主方向
    - **证明**：前面已知特征值不相同，再由对称变换即得特征值均为实数、特征向量彼此正交
- **主曲率**：主方向上的法曲率
  - **特征性**：W变换的单位特征向量的特征值
    - **证明**：$k_n(v) = \lang \mc W(v),v \rang = \lang kv,v \rang = k$
  - **极值性**：两个方向分别是法曲率最大、最小的方向
    - **证明1**：设 $\l = \dfrac{du}{dv}$，求导
    - **证明2**：由欧拉公式易得结论
    - **证明3**：由数值线性代数的极大极小原理易得结论
  - **分类**：
    - **脐点**：各向曲率相同，切向均为主方向，两主曲率相等
    - **非脐点**：存在不同法曲率，两主方向相互正交
- **欧拉公式**：
  - 设 $k_1,k_2$ 是 $P$ 点的主曲率，$e_1,e_2$ 是相应正交主方向
  - 若 $v\in T_PS$ 是单位向量，$\angle(v,e_1) = \t$，则 $P$ 点的 $v$ 方向法曲率为 $$k_n(v) = k_1\cos^2\t + k_2\sin^2\t$$
  - **证明**：由主方向正交性，$v = \cos\t e_1 + \sin\t e_2$，再直接计算即可

### 计算公式

- **系数矩阵公式（Weingarten方程）**：
  - 设曲面为 $r(u,v)$，外恩加藤变换在基 $r_u,r_v$ 下的系数矩阵为 $\bvec{a & b \\ c & d}$
  - 则 $\begin{cases} a = \cfrac{LG - MF}{EG - F^2} \\\\ b = \cfrac{ME - LF}{EG - F^2} \\\\ c = \cfrac{MG - NF}{EG - F^2} \\\\ d = \cfrac{NE - MF}{EG - F^2} \end{cases}$
  - **证明**：
    - 易得 $\begin{cases} \mc W(r_u) = -n_u = ar_u + br_v & (1)\\ \mc W(r_v) = -n_v = cr_u + dr_v & (2)\end{cases}$
    - 将 $(1),(2)$ 分别与 $r_u,r_v$ 内积得 $\bvec{E & F \\ F & G}\bvec{a & b\\ c & d} = \bvec{L & M \\ M & N}$，求解即可
- **特征方程**：$$k^2 - \cfrac{LG-2MF+NE}{EG-F^2}k + \cfrac{LN-M^2}{EG-F^2} = 0$$
  - **主曲率**：两个解 $k_1,k_2$
  - **平均曲率**：$H = \cfrac{k_1+k_2}{2} = -\cfrac{\tr(W)}{2} =  \cfrac{1}{2}\cfrac{LG-2MF+NE}{EG-F^2}$
    - <font color='red'>曲面该点弯曲度的算数平均值</font>
  - **高斯曲率**：$K = k_1k_2 =  \det(W) =\cfrac{LN-M^2}{EG-F^2}$
- **外积公式**：$$n_u\land n_v = (ad-bc)r_u\land r_v = Kr_u\land r_v$$

### 性质

- **高斯曲率是Gauss映射的面积比**
  - **证明**：
    - 由 $n$ 的连续可微性，$d\sigma \approx \lang n_u\land n_v, n \rang = KdA$
    - （面积元在Gauss映射下像）的面积：$\text{Area}(g(D)) = \dis\int_{g(D)}d\sigma  = \int_D KdA$
- **平均曲率是Gauss映射的方向**：
  - **证明**：
    - 对曲面做微小法向扰动 $\d\bs r = f\bs N$，则面积变化为 $\d A = -\dis\iint 2HfdA$
    - 也就是说，

### 应用

- **主标架**：设 $e_1,e_2$ 是 $P$ 点主方向上的标准正交基，则 $\{O,e_1,e_2,n\}$ 称为主标架
  - **计算**：
    - 设已知 $r(u,v)$，则求解 $\bvec{e_1 \\ e_2} = A\bvec{r_u \\ r_v}$，再取参数变换 $(\ol u,\ol v) = (u,v)A^{-1}$ 即可
- **曲面的二阶近似**：取主标架时，曲面局部近似于二次曲面 $r(u,v) = \cfrac{k_1u^2 + k_2v^2}{2}$
  - **证明**：
    - 易得此时 $L = k_1，M = 0，N = k_2$，再由曲面的Monge型局部展开即可
  - **近似分类**：
    - **椭圆点**：$K>0$ 时，局部近似于椭圆抛物面
      - **圆点**：$k\neq 0$ 的脐点，局部近似于球面
      - <font color='red'>此时高斯映射保定向，像球面</font>
    - **双曲点**：$K<0$ 时，局部近似于双曲抛物面
      - <font color='red'>此时高斯映射反定向，像马鞍</font>
    - **抛物点**：$K=0$ 时，局部近似于抛物柱面  
      - **平点**：$k=0$ 的脐点，局部近似于平面（可展曲面）
        - <font color='red'>此时高斯映射退化，区域变成曲线</font>


## 习题

### 第二基本形式

- 求曲面的第二基本形式
- **曲面基本形式的应用**
  - 设 $w = ar_u + br_v$ 是切向量，则
    - $I(w)$ 代表该点以 $w$ 为切向量的曲线的弧长微分的平方，也即 $|w|^2$
    - $II(w)$ 代表该点以 $w$ 为切向量的曲线的（单位切向量微分），也即方向导数 $\vec t_w = \dfrac{1}{\sqrt{a^2+b^2}}(at_u + bt_v) = \cos\t\cdot t_u + \sin\t\cdot t_v$

### 主曲率

- 求曲面切方向法曲率
- 求曲面的Gauss曲率、平均曲率
  - **法一**：求切向量、法向量，算出 $E,F,G,L,M,N$，然后用高斯曲率和平均曲率公式
  - **法二**：求切向量、法向量，算出W变换系数阵，求特征方程，利用韦达定理
- 求曲面的椭圆点、双曲点、抛物点，以及轨迹
  - **解法**：计算判别式即可

### 曲率线

- **曲率线**：若曲面 $S$ 上正则连通曲线 $\a$ 满足全体切线都是 $S$ 的主方向，则称为 $S$ 的曲率线
- **Olinde Roderigue定理**：
  - 设 $\a$ 是 $S$ 上的正则连通曲线
  - 则 $\a$ 是曲率线 $\LR$ 对任意参数表示 $\a(t)$，存在函数 $\l(t)$ 满足 $\bs N'(t) = \l(t)\a'(t)$
  - **证明**：
    - 若 $\a'(t)$ 是主方向，则其为 $dN_p$ 的特征向量。反之同理

## 第三基本形式 $\lang dn,dn \rang$

- **第三基本形式**：$III: T_p(S)\times T_p(S)\to \R，(\a,\b) \mapsto \Bip{\mc W(\a),\mc W(\b)}$
  - <font color='red'>它的几何意义不太直观，而且完全是前两个基本形式的派生量。故不是重点</font>
  - **几何意义**：$|dN_p(w)|^2 = III(w,w)$
    - <font color='red'>描述高斯映射的度量，即高斯曲率</font>
- **基本形式变换公式**：$K\cdot I - 2H\cdot II + III = 0$
  - **证明**：
    - 设 $W$ 是外恩加藤变换的矩阵，则由特征方程得 $W^2-2HW + KI = 0$
    - 两边作用于某个切向量即可
