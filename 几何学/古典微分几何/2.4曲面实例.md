## 直纹面

-  **直纹面**：单参数直线族构成的曲面 $r(u,v) = \vec a(u) + v\vec b(u)$
   - $a(u)$ 是空间曲线，$b(u)$ 是该曲线的变化方向
   - **直母线**：固定 $u$ ，变化 $v$ 时的单参数直线
   - **性质**：$N = 0$，从而 $K \leq 0$
     - $\begin{cases} r_u = a'+vb' & r_v = b \\ r_{uu} = a'' + vb'' &  r_{vv} = 0 \\ r_{uv} = r_{vu} = b' \end{cases}$
   - **实例**：双叶双曲面
 - **可展曲面**：$K\equiv 0$
   - **几何意义**：曲面上任意区域在高斯映射下的像为单点
   - **等价命题**：直纹面是可展曲面 $\LR$ 成立以下条件之一
     - $(a',b,b') = 0$（三个向量线性相关）
       - **证明**：$M\equiv 0 \LR \lang r_{uv},n \rang = \lang b'(u),n \rang \equiv 0$，从而 $b'$ 可被 $r_u = a'+vb'，r_v = b$ 表出
     - $n(u,v_1) = n(u,v_2)$ （沿着直母线的法向量不变）
       - **证明**：
   - **分类**：
     - **柱面**：$b(u)\land b'(u) \equiv 0$，曲线变化方向是一个圆（直母线平行）
     - $b,b'$ 线性无关
       - 此时可设 $a'(u) = \l(u)b(u) + \mu(u)b'(u)$
       - 再设 $\wt a(u) = a(u)-\mu(u)b(u)$
       - 则 $\wt a'(u) = (\l(u)-\mu'(u))b(u)$
     - **锥面**：$\wt a'(u) \equiv 0$，则 $r(u,v) = a_0 + (\mu(u)+v)b(u)$
       - 此时直母线均过 $a_0$
     - **切线面**：$\wt a'(u)\neq 0$，则 $r(u,v) = \wt a(u) + \dfrac{\mu(u)+v}{\l(u)-\mu'(u)}\wt a'(u)$
       - **理解**：此时直母线都是曲线 $\wt a(u)$ 的切线
       - **几何意义**：某正则曲线的切线全体构成的曲面
         - 可写为 $\vec r(v,t) = \vec r(t) + v\or r'(t)$

## 派生曲面

- **平行曲面**：若 $r(u,v)$ 没有抛物点，则 $r(u,v) + \l n(u,v)$ 是平行曲面
  - 对应点的切平面平行
- **曲率线**：以主方向为曲线切向量，构成的曲线 $\LR \dfrac{dn}{dt} \parallel \dfrac{dr}{dt}$
  - $u = C_1$ 和 $v=C_2$ 是曲率线 $\LR F = M  =0$
- **切线面**：
  - 设 $\a:I\to \R^3$ 是正则参数曲线，则 $X:I\times \R\to \R^3，(t,v) \mapsto \a(t) + v\a'(t)$ 称为它的切线面
  - 设定义域中开集 $U = \{(t,v)\mid v\neq 0\}$，则易得 $X|_U$ 是正则参数曲面
  - 容易发现，$X$ 是两个连通片组成的曲面，它们的公共边界为 $\a(I)$
