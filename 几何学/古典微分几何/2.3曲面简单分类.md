# 曲面的常见例

## 全脐点曲面

- **全脐点曲面**：每个点都是脐点
  - **等价命题**：$II = kI$（k是某函数）
- **分类定理**：全脐点曲面只有球面和平面
  - **证明（局部）**：
    - 取某点 $p\in S$，设 $X(u,v)$ 是其连通邻域 $V$ 的参数化
    - 由全脐点性，所有曲线都是曲率线。再由O-R定理，任意 $w\in T_p(S)$ 都存在可微函数 $\l:S\to \R$ 使得 $dN(w) = \l(q)w$
    - **常数性**：
      - 设 $w = a_1X_u + a_2X_v$，则有 $N_ua_1 + N_va_2 = \l(X_ua_1 + X_va_2)$
      - 这个线性方程的解空间是一维的。但由 $w$ 的任意性，其必须对任意 $a_1,a_2$ 都成立，故只能是 $\begin{cases} N_u = \l X_u \\ N_v = \l X_v \end{cases}$
      - 第一式对 $v$ 微分，第二式对 $u$ 微分，相减即得 $\l_uX_v - \l_vX_u = 0$
      - 再由于 $X_u,X_v$ 线性无关，故只能是 $\l_u = \l_v = 0$，即常数
    - **分类讨论**：
      - 若 $\l = 0$，则 $N_u = N_v = 0$，即高斯映射是常数，从而是局部平面
      - 若 $\l \neq 0$
        - 设 $Y(u,v) = X(u,v) - \cfrac{1}{\l}N(u,v)$，易得 $Y_u = Y_v = 0$，故它是定点
        - 再由于 $|X-Y|^2 = \cfrac{1}{\l^2}$，故 $X$ 表示以 $Y$ 为心，半径 $\cfrac{1}{\l}$ 的球面
  - **证明（整体）**：
    - 由 $S$ 连通性，对任意 $r\in S$ 都存在连续道路 $\a:[0,1]\to S，p\to r$
    - 由连续性，$\a(t)$ 中每一点的邻域 $V_t$，其原象组成 $[0,1]$ 的开覆盖 $\mc V$。再由有界闭集紧性，从 $\mc V$ 中可取有限覆盖 $\{U_1,...,U_n\}$。再由道路双射性，对应的 $\{V_1,...,V_n\}$ 就是 $\a(t)$ 的有限覆盖
    - 若某个 $V_i$ 是平面/球面，则由主曲率恒定性，与它相交的 $V_i$ 也是平面/球面，从而可得所有 $V_i$ 都是平面/球面。再由 $r$ 任意性即得 $S$ 是球面/平面

## 二阶渐进判别法

- **渐近方向**：若 $II(v) = 0$，则称该切方向是渐进方向
  - **几何形式**：若 $T_p(S)$ 中，某个方向的法曲率为 $0$，则称为 $S$ 在 $p$ 点的渐近方向
  - **实例**：
    - 椭圆点没有渐近方向
    - 双曲点和抛物点的渐近方向是Dupin标线的渐近线方向
- **渐近线**：所有切向都是渐近方向的曲面上曲线
- **Dupin标线**：$D_p = \hkh{w\in T_p(S)\mid II(w) = \pm 1}$
  - **方程形式**：设 $(\xi,\eta)$ 是主基下的坐标，则Dupin标线可用下列方程表出 $$ k_1\xi^2 + k_2\eta^2 = \pm 1 $$
    - **证明**：
      - 设 $(\xi,\eta)$ 是 $\{e_1,e_2\}$ 下的坐标，$\rho,\t$ 是极坐标
        - 则由欧拉公式，存在坐标转化公式 $w = \rho v$，其中 $v = e_1\cos\t + e_2\sin\t$
      - 再由二次型齐次性，$II(w) = \rho^2II(v) = k_1\xi^2 + k_2\eta^2$
  - **几何形式**：它就是第二基本形式的二次型图像，可用于判断该点邻域的二阶渐进类型
  - **实例**：
    - 椭圆点的Dupin标线是椭圆
    - 双曲点的Dupin标线是渐近线相同的两条双曲线
    - 抛物点的Dupin标线是渐近方向的两条平行直线

## 直纹面

- **单参数直线族**：$\{\bs a(t),\bs l(t)\mid t\in I\}$，其中 $\bs a(t)$ 是点向量，$\bs l(t)$ 是方向向量
  - <font color='chartreuse'>一个点和一个方向能确定一条直线（点斜式）</font>
-  **直纹面**：单参数直线族构成的曲面 $\bs r(t,\l) = \bs a(t) + \l\bs l(t)$
   - **母线**：固定 $t$ ，变化 $\l$ 时的单参数直线 $L(\l) = \bs a_t + \l\bs l_t$
   - **准线**：$\bs a(t)$
   - <font color='red'>几何意义1：母线族 $\{L(\l)\}_t$ 拼成的曲面
   - 几何意义2：准线 $\bs a(t)$ 沿母线方向滑动成的曲面</font>
- **计算方法**：
  - 写出曲面的参数形式 $r(u,v) = \Big( x(u,v)，y(u,v)，z(u,v) \Big)$
  - 如果 $x,y,z$ 可以线性分离成仅有 $u$ 和仅有 $v$ 的向量，那就是直纹面
- **实例**：
  - **平面**：$\bs a_0 + u\bs l_1(t) + v\bs l_2(t)$
  - **标准圆柱面**：设 $x^2+y^2=R^2$，则直纹写法 $\begin{cases} a(t) = (R\cos t,R\sin t,0) \\ l(t) = (0,0,1) \end{cases}$
  - **标准圆锥面**：设 $x^2+y^2=z^2$，则直纹写法 $\begin{cases} a(t) = (0,0,0) \\ l(t) = (\cos u,\sin u,1) \end{cases}$
  - **单叶双曲面**：设 $\cfrac{x^2}{a^2} + \cfrac{y^2}{b^2} - \cfrac{z^2}{c^2} = 1$，则直纹写法 $\begin{cases} a_1(t) = (a\cos t, b\sin t, 0) \\ l_1(t) = (-a\sin t, b\cos t, c) \\ l_2(t) = (a\sin t,-b\cos t,c) \end{cases}$
  - **双曲抛物面**：设 $z=xy$，则直纹写法 $\begin{cases} (t,0,0) + \l(0,1,t) \\ (0,t,0) + \l (1,0,t) \end{cases}$
  - **椭圆柱面**：设 $\cfrac{x^2}{a^2} + \cfrac{y^2}{b^2} = 1$，则直纹写法 $(a\cos t,b\sin t, 0 ) + \l(0,0,1)$
  - **抛物柱面**：设 $y = 2px$，则直纹写法 $\dkh{\cfrac{t^2}{2p},t,0} + \l(0,0,1)$

### 腰曲线

- **一般直纹面**：以后我们只讨论非柱面，且 $|\bs l(t)| = 1$ 的形式
  - <font color='red'>非柱面是保证后续计算中分母 $\bs l'\not\equiv 0$，且腰曲线唯一
  - 单位向量是保证总有 $\Bip{\bs l(t),\bs l'(t) } = 0$</font>
- **腰曲线**：直纹面上总与 $\bs l'$ 垂直的曲线
  - <font color='red'>准线可以任意选取，腰曲线是特殊的准线
  - 特别注意一点，$\bs l'$ 是对 $t$ 求导。所以它的几何意义是切换母线时，母线方向的转动量</font>
  - **参数形式**：$\bs b(t) = \bs a(t) + \l(t)\bs l(t)$，其中 $\l(t) = -\cfrac{\lang \bs a',\bs l' \rang}{\lang \bs l',\bs l' \rang}$
    - **证明**：由垂直性变形即可
  - **唯一性**：直纹面的腰曲线唯一，且只依赖于直纹面本身
    - **证明**：取两条准线，易得根据它们构造的腰曲线相同
  - **长度最短性**：腰曲线是长度最短的准线
    - **证明**：
      - 只需寻找 $L = \dis\int_L |\bs b'(t)|dt$ 最小的曲线即可。这需要用变分法
      - 由欧拉-拉格朗日方程，易得必须满足 $\Bip{\bs b',\bs l'} = 0$，即腰曲线
- **腰点（中心点）**：腰曲线上的点

### 几种特殊的直纹面

- **柱面**：母线彼此平行
  - **标准参数函数**：$\bs a(t) + \l\bs l_0$，其中 $\bs l_0$ 是母线方向
  - **腰曲线**：圆
- **锥面**：母线均交于顶点
  - **标准参数函数**：$\bs a_0 + \l\bs l(t)$，其中 $\bs a_0$ 是顶点
  - **腰曲线**：顶点
- **切线面**：某正则曲线的切线全体构成的曲面
  - **标准参数函数**：$\bs r(t) + \l \bs r'(t)$
  - **腰曲线**：母线都是原曲线 $\bs r(t)$ 的切线，腰曲线就是原曲线
- **切线面**：
  - 设 $\a:I\to \R^3$ 是正则参数曲线，则 $X:I\times \R\to \R^3，(t,v) \mapsto \a(t) + v\a'(t)$ 称为它的切线面
  - 设定义域中开集 $U = \{(t,v)\mid v\neq 0\}$，则易得 $X|_U$ 是正则参数曲面
  - 它的形状是两个连通片组成的曲面，两片的公共边界为 $\a(I)$


### 标准直纹面的性质
 
- **微分公式**：$\begin{cases} r_t = \bs a'+ \l\bs l' & r_\l = \bs l\\ r_{tt} = \bs a'' + \l\bs l'' &  r_{\l\l} = 0 \\ r_{t\l} = r_{\l t} = \bs l' \end{cases}$
  - <font color='chartreuse'>直接计算即可</font>
- **分布参数**：$\mu = \cfrac{\Bip{\bs a'\land \bs l，\bs l'}}{|\bs l'|^2}$
  - <font color='red'>由于腰曲线不依赖准线，故它也不依赖准线</font>
- **夹角定理**：设 $\t$ 是 $\bs N(t,0)$ 和 $\bs N(t,\l)$ 的夹角，则 $\tan\t = \cfrac{\l}{\mu}$
  - **证明**：
    - 计算易得 $N(t,u) = \cfrac{r_t\land r_\l}{|r_t\land r_\l|} = \cfrac{\mu\bs l' + \l\bs l'\land \bs l}{\sqrt{\l^2 + \mu^2}|\bs l'|}$，再计算夹角即可
- **高斯曲率公式**：$ K = -\cfrac{\mu^2}{(\mu^2 + t^2)^2} $
  - **证明**：由微分公式计算易得 
  - **推论（无椭圆点性）**：直纹面的 $N = 0$，从而 $K \leq 0$
  - **推论（腰曲线）**：腰曲线上高斯曲率绝对值最大


### 可展曲面

- **可展曲面**：$K\equiv 0$
  - **映射意义**：曲面上任意区域在高斯映射下的像为单点
  - **几何意义**：等距同构于平面，即可以展成平面的曲面
- **可展定理1**：
  - 可展曲面一定是直纹面
  - 直纹面可展 $\LR $ 分布参数恒为零 $\LR$ 向量 $\bs a',\bs l,\bs l'$ 线性相关 $\LR \bs N(t,\l) = \bs N(t,0)$
  - **证明**：
    - 由分布参数的性质易得结论2
    - 由于 $K = \det(W) = 0$，故可展曲面的W变换处处退化，即每点都存在唯一方向满足 $W(v) \equiv 0$，从而 $dN(v) \equiv 0$，即该方向上法向量不变
    - 设 $\g(t)$ 是上述切向量组成的曲线，求导易得 $\g',\g'',\g'''$ 共面，且 $\g' \perp \g''$，即 $\g''$ 方向恒定，即 $\g$ 是直线
    - 再由于每点都存在，故 $\g$ 族作为母线构成直纹面
- **可展定理2**：当零点离散时，可展直纹面只有柱面、锥面、切线面
  - **证明**：
    - 若 $\bs l\land \bs l' \equiv 0$，则只能是 $\bs l' \equiv \bs 0$，从而是柱面
    - 若 $\bs l\land \bs l' \neq 0$
      - 若腰曲线 $\bs b'\neq 0$，则由 $\mu \equiv 0$ 得 $\Bip{\bs b'\land \bs l,\bs l'} = 0$，再由腰曲线定义得  $\ip{\bs b',\bs l'} = 0$，综上只能是 $\bs b'$ 与 $\bs l$ 平行，即直纹面是其腰曲线的切线面
      - 若腰曲线 $\bs b' = 0$，则其在该处退化为一点，则直纹面是锥面
    - 对于 $\bs l' = 0$ 或 $\bs b'= 0$，且零点不是聚点时，我们都可以采用分片的方式，证明零点邻域内也是上述三种曲面
- **包络定理**：若曲面 $S$ 上曲线 $\a$ 的切向量都不是渐近方向，则 $S$ 沿 $\a$ 的切平面族的包络是可展曲面
  - **证明**：
  - **实例**：
    - 对于球面，赤道的切平面族的包络是柱面，其它纬线圆的切平面族的包络是锥面
