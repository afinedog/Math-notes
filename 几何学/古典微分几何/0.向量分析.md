# 向量分析

- **参考教材**：
  - 《微分几何》彭家贵（主要）
  - 《微分几何》陈维桓
  - 《曲线和曲面的微分几何》Do Carmo
- **符号约定**：
  - $\lang a,b \rang$ 或 $a\cdot b$ 都可表示两向量的内积
  - $a\land b$ 或 $a\times b$ 都可表示两向量的外积
  - $E^3$ 表示三维欧氏空间，$O$ 表示原点，$X$ 表示向量，默认取自然定向的正交标架（即右手定向）

## 向量分析：详见数分，也可见偏微分

- **向量函数**：
  - **参数形式**：$a(t) = \Big( a^1(t), a^2(t),... \Big)$
- **向量微分公式**：设 $\bd a(t),\bd b(t)$ 是向量函数，$\l(t)$ 是标量函数
  - **乘法公式**：$\dis\frac{d}{dt}(\l\bd a) = \frac{d\l}{dt}\bd a + \l\frac{d\bd a}{dt}$
    - **证明**：
      - 代入即可，详见数分
  - **内积公式**：$\dis\frac{d}{dt}\lang \bd a,\bd b \rang = \lang \frac{d\bd a}{dt}\bd b , \frac{d\bd b}{dt}\bd a \rang$
    - **证明**：
      - 代入即可，详见数分
  - **外积公式**：$\dis\frac{d}{dt}(\bd a\times \bd b) = \frac{d\bd a}{dt} + \frac{d\bd b}{dt}$
    - **证明**：
      - 代入即可，详见数分

### 向量场

- [场论](../数学分析/约束积分/场论.md)
- **向量场** $F$：向量值函数
- **标量函数** $f$
- **算子表示法**：
  - $\divt F = \lang \nabla,F \rang$
    - **证明**：直接计算即可（线性组合，得标量函数）
  - $\rot F = \nabla\land F$
    - **证明**：直接计算即可（行列式，得向量函数）
- **算子公式**
  - **双线性**：$\lang \nabla,(fF_1+gF_2) \rang = \lang \nabla,fF_1 \rang + \lang \nabla,gF_2 \rang$，外积同理
  - **分配律**：$\lang \nabla,fF \rang = f\lang \nabla,F \rang + \lang \nabla f,F \rang$，外积同理
  - **梯度的旋度**：$\nabla\land (\nabla f) = \rot(\grad f) = 0$
  - **旋度的散度**：$\lang \nabla,\nabla\land F \rang = \divt(\rot F) = 0$


## 坐标变换

- **标架**：设 $O$ 是原点，$v_1,v_2,v_3$ 是三个线性无关向量，则 $\{O;v_1,v_2,v_3\}$ 称为 $E^3$ 的一个标架
  - <font color='chartreuse'>$E^3$ 是欧氏空间，$\R^3$ 是欧氏向量空间。前者未取标架，后者取正交标架</font>
  - **定向**：当欧氏空间给出一个正交标架时，称该空间给出一个定向
  - **定向相同**：已知存在正交变换 $T:E_1\to E_2$，若 $\det T = 1$，则称两空间定向相同
- **欧氏变换**：线性空间中保持各点距离不变的变换（等距同构）
- **分解定理**：
  - 设 $X\in E^3$，$\mc T$ 是欧氏变换，则 $\mc T(X) = TX + P$，其中 $T$ 是三阶正交变换，$P\in E^3$
  - <font color='chartreuse'>欧氏变换都是旋转、平移、翻转的复合</font>
  - **代数证明**：
    - 不妨设原点 $O$ 是不动点
    - 设 $t\in (0,1)$，则 $d(O,tX) = d(O,\mc T(tX)) = $
  - **微分证明**：

  - **推论**：
    - 若 $\det T = 1$，则不存在翻转
    - 若 $\det T = -1$，则存在翻转
- **保积定理**：欧氏变换是保积变换，从而是内积空间的同构
    - <font color='chartreuse'>欧氏变换不是线性变换，从而不是线性同构。但其线性部分的行列式为 $1$，故依然是保积变换</font>
- **刚体运动**：满足 $\det T = 1$ 的欧氏变换
  - **分解定理**：刚体运动是平移和旋转的复合
- **三维欧氏变换群**：$E^3$ 中全体欧氏变换构成一个群
  - **证明**：
    - **含幺性**：易得恒等变换是欧氏变换
    - **结合律**：易得
    - **可逆性**：由平移、旋转、反射均可逆即得结论
  - **同构性**：$E^3$ 中全体标架和欧氏变换群一一对应
    - **证明**：
      - 显然任意两个正交标架都可由一个欧氏变换对应起来

## 习题

- **合同的嵌入性**：$E^3$ 上开集的等距变换是欧氏变换
- **合同的外积传递性**：设 $\mc T$ 是欧氏变换，则 $(\mc Tv)\land (\mc Tw) = \mc T(v\land w)$
