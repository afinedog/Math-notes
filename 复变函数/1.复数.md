## 复数

- 实部：$x = Re\ z$
- 虚部：$y = Im\ z$
- **实数对形式**：$(x,y)$
- **辐角（任意角）**：$tan\theta = \frac{y}{x}$，$\theta = Arg \ z$
  - **主辐角**：$-\pi < Arg\ z \leq \pi$

## 欧拉公式证明

$\begin{cases} e^{i\theta} = cos\theta + isin\theta \\ e^{i\pi}+1 = 0 \ (x=\pi) \end{cases}$

1. 泰勒展开证明：
   1. $e^x$的泰勒展开式
   2. 同理，建立$e^{xi}$的泰勒展开式
   3. 然后根据 $i$ 幂的正负性，调整各项顺序并配凑，建立更序级数
   4. 分离为cos和sin的泰勒展开即可
2. e的定义证明
   1. $e^x = \lim\limits_{n\to\infty}(1+\frac{1}{n})^{nx} \ =\lim\limits_{n\to\infty}(1+\frac{x}{n})^n$（无穷小代换 $e^x=x+1$）
   2. 同理，$e^i = \lim\limits_{n\to\infty}(1+\frac{i}{n})^n$，是一个复数
   3. 证明复数的指数交换律 $|z^n| = |z|^n$：
      1. 数学证明 $x+yi$
      2. 几何证明：
         1. 单位圆上的情况：$z^2$ 相当于把角度变为2倍，模=1相等。
         2. 非单位圆上的情况：反正其肯定在一个圆上。由辅助角公式得：$x+yi = \sqrt{x^2+y^2}(sin\theta+icos\theta)$，模 $= x^2+y^2$ 相等
   4. 对其应用指数交换律进行复数求值：$|e^i| = \lim\limits_{n\to\infty}(\sqrt{1+(\frac{1}{n})^2})^n$，再进行 $\frac{1}{n^2}$ 指数配凑，得到 $\lim\limits_{n\to\infty}e^{\frac{1}{2n}} = 1$
   5. （未完，）
3. **几何理解**
   - 复数的乘积性源于 $i^2 = -1$ 这个最本源的性质。再加上数分中也有的极坐标方法，即空间中任何向量均可写为半径+角度的表示方法，最终就会得到复数乘积的角度相加性
   - 然后，由 $e$ 的极限意义，$(1+\frac{i\theta}{n})^n$ 可看作将 $z_0=1$ 进行 $n$ 次相同的延长旋转操作
     - 延长倍率为 $\sqrt{1+(\frac{\theta}{n})^2}$，旋转角度为 $\frac{\theta}{n}$
   - 由勾股定理，易得每次旋转都形成一个直角三角形
     - $z_n$ 是斜边，$z_{n-1}$ 是直角边
   - 当 $n\to\infty$ 时，由于累乘和累加均是一阶的，而模长是累乘意义的二阶无穷小（取对数即可看出），角度是累加意义的一阶无穷小
     - 从而模长在迭代中视为不变，但旋转依然被保留
     - 极小直角三角形此时等价于极小扇形，从而欧拉公式成立
   - **本质**：类似于圆的面积积分，不能对半径积分，而应当对极小扇形积分。因为半径的角度为0，在累加意义下依然是0，可视作累加意义的高阶无穷小
     - 再举一个例子，正方形不能用平移边去逼近内切圆，从而得出 $\pi = 4$ 的结论，因为余项的在迭代意义下不是高阶无穷小（收敛速度太慢）
4. **代数理解**：e的乘积线性正好与乘积角度相加性契合

### 复数的表示方法

- 三角形式：$z = r(cos\theta + isin\theta)$
- 指数形式：$z = re^{i\theta} 或（|z|e^{i\ arg\ z}）$
- 代数形式：$z = x + iy$
- 相互转换
  1. 代数形式（辅助角公式） $\rightarrow$ 三角形式（欧拉公式） $\rightarrow$ 指数形式
  2. 代数形式（反三角函数） $\rightarrow$ 辐角 $\rightarrow$ 指数形式

### 复数的运算与表示

- $Arg\ (z_1z_2) = Arg\ z_1 + Arg\ z_2$（指数形式可以解释）
- **旋转性**：两个复数相乘，结果是模的乘积（模的交换律）和角度的相加（指数表达形式）
  - **方向**：逆时针为正，顺时针为负
  - **旋转乘数**：iz中的i
  - **Arg的主值（主辐角）**：$-\pi < arg\ z < \pi$
  - **特定值**：arg（某一个角度值）
- **模的结合律（指数交换律）**
  - $|z_1z_2| = |z_1||z_2|$
    - 旋转性理解
    - 指数形式理解
  - **棣莫弗公式（分圆多项式）**
    - $(cos\theta + isin\theta)^n = cos(n\theta) + isin(n\theta)$
  - **复数的幂方程（现在看来就是初等多值函数：根式函数的性质）**：$\sqrt[n]{z} = w$，使用指数形式进行解答（共n个解，分圆）
    - $w = w_0 + w^k$
      - $w = (\sqrt[n]{z})_k$ <font color = 'green'>（无穷多个k）</font>$= e^{i\frac{2k\pi}{n}}$：（旋转值）
      - $w_0 = \sqrt[n]{r}e^{\frac{i\theta}{n}}$：（初始值）
  - **求 $cos\ 3\theta$ 用 $cos\ \theta$ 和 $sin\ \theta$ 表示的式子**
    - 棣莫弗公式 + 实数和虚数的独立性即可
  - **计算 $\sqrt[3]{-8}$**
    - -8其实是复数，也可以转化成z的指数形式
- **共轭复数**
  - **有理运算性（线性、积性）**：$\overline{R(a,b,c,...)} = R(\overline{a},\overline{b},...)$
    - **理解**：因为正负只有两个状态，所以直接取反可以等于所有其中的运算取反
  - **复合复数w可以通过其因子复数z的共轭运算来求解**：$w = \frac{1+z}{1-z}$
    - $w = \frac{1+z}{1-z}·\frac{\overline{1-z}}{\overline{1-z}}$
    - 分离实部和虚部
    - 求模
    - （额，其实知道了z的实部和虚部，以及w(z)，那么常规手段也可以解出w。所以这题没意义）


## 复数与几何

### 复数的向量性

- **复数的模展开**：
  - $|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2Re(z_1·\bar{z_2})$
    - $arg(z_1·\bar{z}_2) = \theta_1-\theta_2$
    - $Re(z_1·\bar{z}_2) = |z_1||z_2|cos|\theta_1-\theta_2|$
  - **理解**：
    - 复数是向量，虽然乘法运算和向量不同，但模运算却与向量相同
      - 复数的乘积是伸缩旋转，向量的乘积是伸缩投影
    - 所以需要对乘法结果取实部，从而得到 $cos$ 形式，使之化为向量乘法
  - **本质**：取实部操作即为投影
- 复数不等式可以使用类向量方法解答：$|\frac{a-b}{1-\bar{a}b}| < 1 \qquad(|a|<1 且 |b|<1)$
  - 法一：三角不等式（如果a、b是向量，就这么做。复数和向量性质类似，所以也可以这么做）
  - 法二：向量几何（不好用）

### 曲线的复数方程

- 直线：$z = z_1 + t(z_1-z_2)$（参数方程，t为直线线段长度）
- 圆：$|z-z_0| = r$
- 实轴：$Im\ z = 0$

### 相关证明

- **等边三角形的复数表达**：$z_1^2+z_2^2+z_3^2 = z_1z_2+z_2z_3+z_1z_3$
  - **证明**：
    - 由旋转性，$(z_1-z_2)(cos\frac{\pi}{3}+isin\frac{\pi}{3}) = z_2-z_3$
    - 移项得复数分式，两边平方即得上述等式
- **证明三角形的内角和等于** $\pi$：向量表达，$arg(-1) = \pi$
  1. 旋转证明
  2. 辐角证明：$\frac{a}{b}·\frac{b}{c}·\frac{c}{-a} = -1 \Rightarrow arg\ A + arg\ B + arg\ C = arg(-1)$

## 复平面点集（二维空间）

- **平面点集**
  - $点z_0的\rho-邻域\qquad N_\rho(z_0)$
    - 由$|z-z_0|<\rho$所确定的平面点集
  - $点z_0的去心\rho邻域 \qquad N_\rho (z_0) - \{z_0\}$：
    - 由$0<|z-z_0|<\rho$所确定的平面点集
- **聚点**：平面上任意一点$z_0$的任意邻域都含有点集$E$无穷多个点（不必属于E）
  - 聚点集合$E'$
  - 相当于数分中的<font color='coral'>收敛值（不唯一）</font>
- **孤立点（孤立边界点）**：E中的非聚点
  - 相当于数分中的<font color='coral'>数列序号在N之前的点</font>
- **外点**：非E中的非聚点
  - 相当于数分中的<font color='coral'>数列外其它所有点</font>
- **？点**：非E中的聚点
  - 相当于<font color='coral'>数分中取不到的极限值</font>
  - E若是闭集，则它不存在
  - E若是开集，则它是闭包的边界点

### 边界

- **内点**：点集E中的点 $z_0$ 有一邻域全含于E内
  - E必须是个连续集
  - **内点——集聚性**：内点皆为聚点
    - “内”在这里的意思就是极限值了
  - **内点全体**：$S^o$
- **边界点**：$z_0$ 的**任意邻域**内，皆有含于和不含于E的点
- **边界**：点集E的全部边界点的点集  $\partial E$
- **孤立点——边界性**：孤立点（非规律点）必定是边界点（所以孤立点也可以叫孤立的边界点）
  - 边界点不一定是孤立点（连续集的边界点就不是孤立点）
  - 非连续集不是连续的图形，所以孤立就意味着边界
  - 相当于数分中<font color='coral'>极限值附近的收敛点</font>
- **开集**：点集E的点皆为内点（**连续性定义**）
  - 开集没有边界点（孤立点）
  - **开集——连续性**：开集的子集均为连续集（开集就是连续集，前者强调边界，后者强调连续）
- **闭集**：点集E中包含了所有的聚点（**连续性定义**）
  - 闭集可以有孤立点（数分中的闭集也可以有孤立点）
  - **闭集——边界性**：闭集必定包含其连续子集的边界
    - 点集E的边界 $\partial E$ 是闭集，即 $(\partial E)'\subseteq \partial E$
    - 证明：$\forall 聚点 \in 闭集$
- **有界集**：$\exist M > 0，\forall z \in E，都有|z| < M$，则称E为有界集
  - 点集E全含于一个圆之内（圆是二维平面内的有界指标）（垂直于坐标轴的直线是一维平面的有界指标）
- **闭包**：E与其聚点全体E'的并集叫做E的闭包 $\bar{E}$
  - 相当于加上了取不到的极限值（边界）
  - **闭包——闭集性**：闭包必定是闭集（闭包就是开集的扩充）
- **闭开集**：实数集R

#### 例子

- 数轴（R）是闭开集
- 空心正方形是闭集（二维邻域必定含有不包含于线的点，所以其中的点都不是内点，其不是开集）（空心正方形只有边界，所以是闭集）
- 实心正方形（含边界）是闭集（含有边界，不是开集）
- 实心正方形（不含边界）是开集（边界是聚点，不是闭集）

- **复数列的定义**：$\lim\limits_{n\to \infty} z_n = x_n+iy_n = z_0 = x_0 + y_0$
  - 需要 $|z_n-z_0| < \epsilon$

### 区域

- **区域（可以不凸  ，但是必须接触）**：
  - 非空点集D为开集
  - 且任意两点可用全部在D中的折线连接（**子集相连性**）
- **闭域**：区域D加上它的边界C（闭包 $\bar{D}$ = D+C）
  - （单位圆 $\neq$ 单位圆周）
- **有界集的E直径**：$d(E) = sup\{|z - z'| \ |z\in E,z'\in E \}$
  - 最大性（最大的线段）
- **连续曲线C**：由方程所决定的点集 $(\alpha < t < \beta)$
  - Re和Im的方程$\begin{cases}x = x(t) \\ y = y(t) \end{cases}$（参数方程），
  - $z = x(t) + iy(t)$（复数方程）
  - **起点**：$z(\alpha)$ 和**终点**
  - **重点**：$z(t_1) = z(t_2) \qquad (t_1 \neq t_2) $ 
    - $t_2$ 可以是起点或终点，$t_1$ 不行
    - 而重点是 $t_1$
  - **简单曲线（Jordan若尔当曲线）**：无重点的连续曲线
  - 简单闭曲线
- **可求长的弧**：连续弧AB（$z = z(t)\ (\alpha \leq t \leq \beta)$）中，$\forall 递增实数列\{t_n\}$，其在AB弧上对应的点列的折线 $Q_n$ 的长度
  - $I_n = \sum^{n}_{j=1}|z(t_j) - z(t_{j-1})|$ 有上界，则称弧AB可求长（如果弧的长度无限，即 $I_n$ 无上界，则称为不可求长）
  - 上确界称为AB弧的**长度**
- **光滑曲线**：实部和虚部的导数**存在**、**连续**、**且不全为0**（非初等函数）
- **逐段光滑曲线**：由**有限条**光滑曲线衔接而成的**连续曲线**（别有断点）
  - 光滑曲线是可求长的（数学分析证明），用中值定理和基本不等式化为 $\sum \sqrt{[x'(t_n)dt]^2 + [y'(t_n)dt]^2} $（导数存在即导数有界，而$dt$可取无穷小，那么取一个收敛级数即可）

### 若尔当定理

- **单连通区域（连通性定义）**：复平面上的区域D，$\forall 简单闭曲线C \in D$，内部全含于D，则称D为单连通区域
  - 可以称为“铁板一块”
  - **多连通区域（连通性定义）**：非单连通区域（空心的）
- **连续点集**：不止一个点的**闭集**E，不能划分为两个**无公共点**的**非空闭集**，则称为连续点集
  - 类Dedekind切割：<font color='coral'>一处切割只能有一个边界，除非它本来就存在多个边界</font>
  - **开集——连续性**：定义里都是闭集，不能用这个方法来定义开集的连续性。但是根据开集的定义，可以发现其连续性？？
  - **退化连续点集**：空集与一个点
- **单连通区域（连续性定义）**：区域D的边界为一个连续点集，则为单连通区域（铁板一块）
  - **多连通区域（连续性定义）**：非单连通的区域称为多连通区域（空心的）
- **二连通区域**：有两个边界（两处空心）

## 复变函数

### 函数

- **单值函数**：$w = f(z) \ (z\in E)$
- **多值函数**：几个或无穷多个 $w$ 与 $z$ 对应（与实数域上的函数不同）
  - 定义域E
  - 值域：$w$的全体所称的集合M
- 单变量形式：$w = z^2 + z$
- 多变量形式
  - 代数形式：$z = x + yi$，则 $w = (x+yi)^2 + (x + yi) = u + vi$，即分离成$\begin{cases} u = m(x,y) \\ v = n(x,y) \end{cases}$
    - 因为一共有四个变量，所以应当用四维空间来表示（画不出来）
    - 但是可以分离表示，取两个复平面，分别表示z和w。这下映射关系就成了点对点的

### 变换

- **入变换（变入）**：$w = f(z)$ 把E映入F，即 $f(E)\subseteq F$，即 $w = f(z)$是E到F的入变换
- **满变换（变成）**：如果任意 $w\in F$ 有 $z\in E$ 使得 $w = f(z)$，则称为满变换
- **反函数（逆变换）**
- **双方单值变换（一一变换）**
- 例：$\begin{cases} z = x + yi \\ x^2 - y^2 = 4 \\ w = z^2\end{cases}$
  - 得 $w = x^2 - y^2 + 2xyi = 4 + 2xyi$
  - **需要注意的一点是，在F平面上x不再是自变量，$Re(w) = u或Im(w) = v$ 才是自变量，所以在此处只需要考虑2xy的值域即可**
  - 结果是：直线 $x=4$

### 极限和连续

- 极限 $\lim\limits_{\substack{z\to z_0 \\ z \in E}}f(z) = w_0$
  - 极限唯一性
  - 极限有界性
  - 极限有值性（上面几个都可以通过定义直接得出）
  - 四则运算（商的分母不为0）
- **部分极限定理**：实部和虚部都有极限 $\Leftrightarrow$ 函数有极限
  - 类似偏极限定理，本质上还是应用了$Rt\triangle$三边的同阶性
    - $|u(x,y)-a| \leq |f(z)-\eta| = \sqrt{(u-a)^2 + (v-b)^2}$
    - $|f(z) - \eta| \leq |u-a|+|v-b|$
- 连续：
  - 四则运算
  - 复合函数
  - 部分连续定理
  - 虽然E上的定义域不一定是连续点集，但是只要存在聚点，就可以讨论连续性
  - **连续性传递**：只要E上是连续曲线，则就可以被连续变换映射成F上的连续曲线
    - 连续函数将紧集映射成紧集：B-W子集判定定理

### 证明无极限

- $f(z) = \frac{1}{2i}(\frac{z}{\bar{z}} - \frac{\bar{z}}{z}) \ (z \neq 0)$
  - 化简为 $f(z) = sin2\theta = \frac{2xy}{x^2+y^2}$
  - 方向性证明：令y=x和令y=0，$\lim\limits_{z \to z_0} f(z)$ 结果不同
  - 跳跃性证明（×）：令 $y=\frac{1}{n},x=\frac{1}{n^2}$，和令 $y=\frac{1}{n},x=\frac{2}{n}$ 结果不同（本质上还是收敛方向，这里不是指的Heine定理那个一元函数定理）

## 扩充复平面

### 复球面

- **测地投影法**：
  - 北极N（球面顶点），南极O（复平面原点，球面底点）
  - 复平面上的点z，通过直线Nz和球面上的交点P一一对应
  - 无穷远点 $\infty$（北极N对应的点），扩充复平面 $C_\infty = C+\{\infty\}$

### 无穷远点

- $\infty$ 作为一个数：和 $\infty$ 以及0的矛盾运算无意义，但是有 $\frac{b}{0}=\infty$，Re和Im、Arg无意义
  - 复平面上每一条直线都通过 $\infty$ 点，所有半平面都不包含 $\infty$ 点
  - 无穷远点的邻域是 $O((0,0),\frac{1}{\epsilon})$ 的外部，对应复球面上的球盖，去心邻域是再 $<+\infty$，对应去掉北极N
  - 无穷远点是复平面唯一的边界点。
  - 扩充复平面是唯一的无边界区域，以无穷远点为内点

### 连通区域

- 画简单闭曲线，其内部或外部仍全部含于D即可

### 广义连续

- 无穷远点可以包含在函数的定义域和值域中
- 若有 $\lim\limits_{z\to z_0}f(z) = f(z_0)$，在 $z$ 或 $f$ 取 $\infty$ 时，称 $f$ 在点 $z_0$ 为广义连续，极限叫广义极限
  - 在无穷远点连续的定义：$\forall \epsilon<0，\exist \delta>0, \forall |z|>\frac{1}{\delta}，有|f(z)-f(\infty)|<\epsilon$
