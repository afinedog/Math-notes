# 常微分方程6：线性微分方程组

- **符号约定**：
  - 下面所有函数 $y(x)$ 都是 $\R\to \R^n$ 的向量值函数，存在 $n$ 个标量函数 $f_i(x)$ 分量
  - 所有的向量都视为列向量
- **线性微分方程组**：$f_i$ 均为线性函数的微分方程组
  - **光滑性**：线性函数是光滑函数

## 解的性质

- **存在和唯一性**
  - **证明**：已知一阶线性方程满足皮卡定理，且易得皮卡定理对于向量值函数同样适用
- **线性微分方程组的标准形式**：$$\begin{cases} \dfrac{dy_1}{dx} = a_{11}(x)y_1 + a_{12}(x)y_2 + ... +  f_1(x) \\\\ \dfrac{dy_2}{dx} = a_{21}(x)y_1 + a_{22}(x)y_2 + ... +  f_2(x) \\\\ \dots \\\\ \dfrac{dy_n}{dx} = a_{n1}(x)y_1 + a_{n2}(x)y_2 + ... +  f_n(x)\end{cases}$$
- **解的标准形式**：$y(x) = \Big( y_1(x),y_2(x),...,y_n(x) \Big)^T$

### 齐次线性方程组

- **齐次线性微分方程组**：$\cfrac{\bold{dy}}{dx} = \bold{A}(x)\bold{y}$
- **解空间**：解集是线性空间 $\mathcal{S}$，维数是微分方程组的阶数
  - <font color='chartreuse'>关于函数空间的讨论详见泛函分析。在这里我们先引入一个抽象思维：任何线性空间的性质都是相似的</font>
  - **证明**：
    - **初值决定性**：若给定初值条件 $x_0\in J$ 和全部 $y_i(x_0)$，解就由曲线族退化为曲线
    - **函数映射**：设 $H: \mathbb{R^n}\to \mathcal{S}，\bd y_0\mapsto y(x)$ 表示将初值变为相应积分曲线的映射
      - **双射性**：解的存在性 $\Rightarrow$ 满射，解的唯一性 $\Rightarrow$ 单射。因此它是双射
      - **线性**：由一阶线性方程的性质直得
      - **同构性**：由双射性 + 线性即得 $H$ 是同构映射
        - 即（初值向量空间 $\R^n$）和（解函数空间 $\mc S$）是同构空间，从而维数相同
    - **初值点映射**：设 $H^{-1}: \mathcal{S} \to \mathbb{R^n}$ 表示将（积分曲线）变为（相应初值点）的映射
      - <font color='chartreuse'>初值点同时也可以看作（积分曲线）和（初值向量空间中的超平面） $\Sigma_{x_0}: \{(x,\bold{y})\mid x=x_0\}$ 的交点</font>
      - **同构性**：由同构的可逆性得它也是同构映射
- **线性无关判别式**：$\sum\limits^n_{i=1} \l_i \bs y_i(x) = 0$
  - $\R^n$ 是数作为分量，数作为系数
  - $\mathcal{S}$ 是单项式函数作为分量，数作为系数
    - 一组函数若线性相关，则：
      - 单项式对应相同
      - 单项式的系数线性相关
      - 比如 $y_1 = 2e^x + 2x^2 + 1$ 和 $y_2 = 4e^x + 4x^2 + 2$ 线性相关
- **解组**：由解函数向量组成的集合（可能线性相关）
- **基本解组（基础解系）（n阶函数矩阵）**：$n$ 个线性无关向量构成的解组
  - **存在性**：齐次线性微分方程组的解空间中一定存在 $n$ 个线性无关的解
    - **证明**：
      - 一个初值对应一个解函数，而n维空间上可找出n个线性无关的初值向量
      - <font color='chartreuse'>因为函数不能融合，所以它和齐次线性方程组不一样</font>
  - **基性**：齐次线性微分方程的基本解组可线性表出所有解
- **解矩阵 $\bold{Y}(x) = (y_{ij}(x))_{n×n}$**：解组向量排列成的矩阵
  - **基解矩阵 $\Phi(x)$**：基本解组的解矩阵
    - <font color='chartreuse'>相当于线性方程组的解空间的基矩阵</font>
  - **齐次线性方程组的通解**： $\bd y = \bold{\Phi}(x)\bold{c}$，其中 $\bd c$ 是任意常数向量
  - **基解矩阵的矩阵乘积封闭性**：设 $C$ 是 $n$ 阶可逆常数方阵，则 $\psi(x) = \varPhi(x)C$ 也是基解矩阵，且 $C$ 和 $\psi(x)$ 一一对应
    - **证明**：由基变换的可逆性即可证明
    - **本质**：矩阵乘积的本质是线性变换，线性变换的本质是线性组合，而解空间对线性组合封闭
- **朗斯基行列式 $W(x) = |y_{ij}(x)|_{n×n}$**：基解矩阵的行列式
  - **刘维尔公式**：$$W(x) = W(x_0)e^{\dis\int^x_{x_0}\tr[\bold{A}(x)]dx}$$
    - <font color='chartreuse'>它只能求行列式，不能求解</font>
    - **证明**：
      - 对 $W(x)$ 求导，由行列式求导法则，可将其展开为逐行求导行列式的和
      - 再由原ODE的定义，可将求导行的元素展开
      - 容易发现展开后的元素可被其它行削减为 $a_{ii}y_{ji}$
        - <font color='chartreuse'>即 $W'(x) = \sum\limits^n_{i=1}a_{ii}(x)W^T = tr[\bold{A}(x)]W$</font>
      - 得到一阶齐次线性微分方程 $W'(x) = tr[\bd A(x)]W(x)$，求解即得刘维尔公式
    - **理解**：常微分方程本来就有一定的微分自返性。而线性微分方程组的封闭性（迭代性），和行列式的归零性混合，得到了非常好的性质
    - **推论（平凡解）**：解组线性无关 $\LR$ $W(x) \neq 0$ $\LR$ $\exist W(x_0)\neq 0$
      - **几何证明**：
        - 只要是非平凡解，根据解空间 $n$ 维性，解矩阵就必定非退化
        - 根据某个解 $W(x_0)$ 就可知道是否为平凡解
      - **代数证明**：
        - 一阶线性微分方程的解是 $e^{-\int p(x)dx}$ ，要么 $y\equiv 0$（特解） 要么 $y \not\equiv 0$
      - <font color='red'>朗斯基行列式可用来快速判断解组的线性无关性</font>

#### 习题

- **证明向量值函数线性无关**
  - **解1（行列式法）**：求某个 $W(x_0)$，若不为 $0$ 则线性无关，反之线性相关
  - **解2（转置方程组法）**：同高代
- **讨论是否为解**：对于 $y'' + p(x)y' + q(x)y = 0$ 来说，不可能存在两个解 $\p(x) = \sin x$ 和 $\psi(x) = xe^x$
  - **证明（朗斯基法）**：
    - 转化为线性方程组，易得朗斯基行列式为 $W(x) = \vvec{\p(x) & \psi(x) \\ \p'(x) & \psi'(x)}$
    - 再由于 $W(0) = 0$，故如果这两个函数是解，那么它们线性相关。但显然不可能存在 $c$ 使得 $\sin x \equiv cxe^x$，故矛盾
  - **证明（代入法）**：
    - 将两个解代入方程，然后令 $x=0$ 会出现 $\begin{cases} p(0) = 0 \\ 2+p(0) = 0 \end{cases}$，无解，显然矛盾
- **基解矩阵唯一性**：若两齐次线性微分方程组的基解矩阵相同，则方程组必相同
  - **证明**：
    - 代入基解矩阵后，由克拉默法则法则可求出全部系数函数（详见下面高阶微分方程）

### 非齐次线性方程组

- **非齐次线性方程组**：$\cfrac{d\bd{y}}{dx} = \bd A(x)\bd y + \bs f(x)$
- **解空间**：
  - 设 $\dfrac{d\bd y}{d\bd x} = A\bd y + \bs f(x)$，其中 $\bs f(x)$ 不恒为 $0$
  - 则它有且至多有 $(n+1)$ 个线性无关解
  - <font color='chartreuse'>前面讨论的是齐次情况，这里讨论非齐次情况</font>
  - **证明**：
    - 反设存在 $n+2$ 个解 $y_1,...,y_{n+2}$
      - 设 $z_i = y_i-y_{n+2}\quad (i=1,...,n+1)$，则由线性得它是齐次形式的基本解组
      - 再由于齐次形式的解空间维数为 $n$，故 $z_i$ 线性相关，可设 $\sum\limits^{n+1}_{i=1} \l_i z_i = 0$
      - 若取 $\l_{n+2} = -\sum\limits^{n+1}_{i=1} \l_i$，则此时 $\sum\limits^{n+2}_{i=1} \l_i y_i = 0$，故线性相关
      - <font color='red'>本质是转化为齐次情况，再利用非零式</font>
    - 设 $\p_1,...,\p_n$ 是齐次形式的一组解
      - 由解的存在性定理，非齐次方程存在特解 $\p^*$
      - 易得 $y_i = \p_i + \p^*$ 线性无关，从而是一组解
        - <font color='chartreuse'>非零式变形得 $\sum\limits^n_{i=1} \l_i \p_i + (\l_1 + ... + \l_n)\p^* = 0$
        - 如果存在非零解，则特解可被齐次解线性表出，这不可能</font>
- **解的线性组合性**：$通解\bs\varphi(x) = 基解矩阵\Phi(x)\bd c + 特解\bs\varphi^*(x)$
  - **证明**：直接代入即可
- **求解方法（常数变易法）**：
  - 假定特解可写为 $\bs\varphi^*(x) = \Phi(x)\bd c(x)$
    - <font color='chartreuse'>注意这里的 $\bd c(x)$ 是函数向量，而不是常数向量
    - 这是一个非常大胆的假定，但后面的验证说明它确实是存在的</font>
  - 根据原ODE + $\Phi(x)$ 的定义，$$\frac{d\bs\varphi^*(x)}{dx} = [\Phi(x)\bd c(x)]' = \bd A(x)\Phi(x)\bd c(x) + \Phi(x)\bd c'(x) \tag{1}$$
  - 再由于 $\bs \p^*$ 是原ODE的解，故 $$\frac{d\bs\p^*(x)}{dx} = \bd A(x)\Big[ \Phi(x)\bd c(x) \Big] + \bs f(x) \tag{2}$$
  - 两式作差即得 $\bs c'(x) = \Phi^{-1}(x)\bs f(x)$，对其取积分即可
  - **特解** $$\bs\p^*(x) = \Phi(x)\int^x_{x_0} \Phi^{-1}(s)\bs f(s)ds$$
  - **通解** $$y = \Phi(x)\Big[\bd c + \int^x_{x_0}\Phi^{-1}(s)\bs f(s)ds \Big]$$
    - **初值关系**：$\bd c = \Phi^{-1}(x_0)\bd y_0$
- **推论**：求特解必须先知道齐次形式的基解矩阵<font color='chartreuse'>（这点也和向量空间的方程组一样）</font>


## 常系数线性微分方程组

- **常系数线性微分方程组**：系数矩阵 $\bold{A}(x)$ 为 $n$ 阶常数矩阵，且 $\bs f(x)$ 连续
- **符号约定**：
  - **矩阵空间** $\mathcal{M}$，范数定义为 $\|A\| = \sum\limits^n_{i,j=1} |a_{ij}|$
  - **范数完备性（详见[矩阵分析](../数值计算/数值代数/0.矩阵分析.md)）**
  - **Cauchy不等式（范数相容性）**：$\|AB\| \leqslant \|A\| · \|B\|$
  - **幂不等式**：$\|A^k\| \leqslant \|A\|^k$
  - **零次幂**：$A^0 = E$
- **矩阵的指数函数**：$e^A = \sum\limits^\infty_{k=0}\dfrac{A^k}{k!}$
  - **绝对收敛性**
    - **证明**：
      - 由于 $\|A\| = m$ 是有限数，相当于证明级数 $\sum\limits^\infty_{k=1}\dfrac{|m|^k}{k!}$ 收敛
      - 由数分知识（Stirling公式）得 $n!\sim \sqrt{n}(\dfrac{n}{e})^n$，故等价于证明 $\sum\limits^\infty_{n=1} (\dfrac{|em|}{n})^{n+\frac{1}{2}}$ 收敛。由Cauchy判别法即得结论
      - <font color='chartreuse'>绝对收敛性蕴含收敛性</font>
  - **保相似性**：$e^{P^{-1}AP} = P^{-1}e^AP$
    - **证明**：
      - 由C-H定理，$A^k$ 都可写为 $I,A,...,A^{n-1}$ 的线性组合，故 $e^A$ 可化为 $n-1$ 次多项式。再由绝对收敛性，多项式的系数有限
      - 再由相似的多项式传递性直得结论
  - **非退化性**：对任意 $A$ 都有 $e^A \neq O$
    - **证明**：
      - 将 $A$ 转化为若当型，则 $\det(e^A) = \det(e^J) = e^{\tr A} > 0$，故得到结论
  - **可逆性**：$(e^{A})^{-1} = e^{-A}$
    - **证明**：
      - 直接代入级数定义式验证即可
  - **交换律条件**：级数可交换性 $e^Ae^B = e^Be^A \LR$ 乘积线性：$e^{A+B} = e^Ae^B \LR$ 矩阵可交换性 $AB = BA$
    - **证明**：
      - $(2)\LR (3)$ 在数分的级数部分已用柯西乘积证明过
      - $(1)\LR (3)$ 由级数定义式可直得

### 基解矩阵

- **标准基解矩阵**：满足 $\Phi(0) = I_n$ 的基解矩阵
  - <font color='chartreuse'>类似自然基，是形式最简单的基础解系</font>
- **指数解定理**：$\Phi(x) = e^{xA}$ 是常系数齐次线性微分方程组的标准基解矩阵
  - **证明**：
    - 易得 $\Phi(x) = e^{xA}$ 是个关于 $x$ 的幂级数，收敛半径为无穷。即在任意有限区间上一致收敛
    - 由一致收敛级数的逐项求导法则，计算易得 $\dfrac{d\Phi(x)}{dx} = A\Phi(x)$，从而 $\Phi(x)$ 是解矩阵
    - 再由非退化性得其为基解矩阵
    - 代入 $x=0$ 易得其为标准基解矩阵
  - **理解**：齐次线性微分方程具有一定程度的自返性，和 $e$ 的性质很像。这点在前面一元线性方程也已经发现了，所以可以大胆猜测多元情况的通解也具有 $e$ 的形式
- **常系数齐次线性微分方程组的通解**：$\bold{y} = e^{xA}\bold{c}$
- **常系数非齐次线性微分方程组的通解**：$\bold{y} =  e^{xA}\bold{c} + \int^x_{x_0} e^{(x-s)A}\bs f(s)ds$

### 待定指数函数法

- **待定指数函数法**：下面我们给出各种情况的 $A$ 的通解公式，实际操作中可以直接用它们代入（当然如果要考试的话，该方法没什么用，还不如现场求Jordan标准型）

#### 可对角化的情况

- 若 $A$ 只有单重特征值（特征子空间均为一维），则此时可对角化
  - **过渡基解矩阵**：$\varPhi(x) = e^{xA}P = P·\diag(e^{\lambda_i x})$
    - <font color='chartreuse'>换元过渡为Jordan型后所求出的基解矩阵，还需要再过渡回去</font>
  - **标准基解矩阵**：$e^{xA} = \varPhi(x)\varPhi^{-1}(0) = \diag(e^{\lambda_i x})$
    - **证明**：
      - 第一式：由过渡基解矩阵的定义，$\varPhi(0) = P$
      - 第二式：由对角矩阵可交换性，消去两端的相似矩阵
  - **通解**：$\varPhi(x)\bold{c} = \begin{pmatrix} c_{11}e^{\lambda_1 x} & c_{12}e^{\lambda_1 x} & \cdots & c_{1n}e^{\lambda_1 x} \\ c_{21}e^{\lambda_2 x} & c_{22}e^{\lambda_2 x} & \cdots & c_{2n}e^{\lambda_2 x} \\ \vdots \\ c_{n1}e^{\lambda_n x} & c_{n2}e^{\lambda_n x} & \cdots & c_{nn}e^{\lambda_n x}  \end{pmatrix}_{n\times n}$
    - **证明**：由标准解矩阵 + c求得
  - **方程组的解**：$\{y_i = e^{\lambda_i x}r_i\}$
    - **证明**：
      - 设 $P = (r_1,..,r_n)$，其中 $r_i$ 是常数向量
      - 由矩阵乘积的列向量表达，$\varPhi(x) = diag(e^{\lambda_i x}r_i)$
      - （此时 $r_i$ 即为通解中的 $c_i$ ）
  - **特征向量**：$\lambda_i$ 的特征向量是 $r_i$
    - **证明**：由 $e^x$ 的导数性质，代入原方程组 $\frac{d\varPhi(x)}{dx} = A(x)\varPhi(x)$，得到 $\lambda e^{\lambda x}r = Ae^{\lambda x}r \Rightarrow (A-\lambda E)r = 0$
- 若 $A$ 存在多重特征值，但是可对角化
  - **过渡基解矩阵**：$\varPhi(x) = (e^{\lambda_1 x}r_1,...,e^{\lambda_n x}r_n)$
  - **标准基解矩阵**：某些对角元相同
  - **通解**：某些行的e相同
  - **特征向量**：某些向量属于同一个特征子空间
- 若 $A$ 存在复数特征根，但是可对角化
  - 解矩阵的形式相同，存在复数
    - **理解**：
      - 已知复数可作为实数代数方程的解，比如单位根
      - 所以微分方程同理，虽然系数都是实数，但可以有复数的解
  - **实数化定理**：实系数方程若存在解，则必定存在实数解
    - **本质**：和复相似均可化为实相似一样，两端都是实数，不可能只用复数才可以打通关系，所以必定可以用复数的某种线性组合化为实数。
  - **标准化法**：A是实数矩阵，从而由 $e^{xA} = \varPhi(x)\varPhi^{-1}(0)$，可将（复过渡基解矩阵）化为（实标准基解矩阵）
  - **线性组合法**：共轭复数解 $y_1(x)，y_2(x)$，可构造 $\begin{cases} y_{R1}(x) = \frac{1}{2}(y_1+y_2) \\ y_{R2}(x) =  \frac{1}{2}(y_1-y_2) \end{cases}$ 将其实数化
    - **共轭引理**：解函数的共轭也是解
      - **证明**：由方程计算结果是实数，得所有虚数反号并无影响

#### 不可对角化的情况

- 若 $A$ 不可对角化（存在重特征根）
  - 若 $r_0$ 是方程 $(A-\lambda_i E)^{n_i}r = 0$ 的非零解，则解 $\{r_i\} = \{(A-\lambda_i E)^ir_0\}$
  - **证明**：
    - $y = e^{\lambda_i x}(r_0 + ... + \frac{x^{n_i-1}}{(n_i-1)!}r_{n_i-1})$
    - 代入方程组，由 $f(x)·e^ax$ 的导数性质，得到 $(A-\lambda_i E)(r_0+...) = r_1+...$
    - 比较系数得到 $(A-\lambda_i E)r_j = r_{j+1}$（**Jordan型迭代公式**）
  - **推论**：解是根子空间的基

## 高阶线性微分方程

### 方程组形式

- **方程组标准形式**：设高阶线性微分方程为 $y^{(n)}(x) = \sum\limits^n_{k=0} a_k(x)y^{(k)}(x)$，则将其转化为标准方程组后
  - 系数矩阵为 $A = \begin{pmatrix} 0 & 1 \\ & 0 & 1 \\ && \ddots& \ddots \\  -a_1(x) & -a_2(x) & \cdots & -a_{n-1}(x) \end{pmatrix}$
    - <font color='red'>注意，虽然有理标准型中的Frobenius矩阵和若当标准型中的相同特征值的Jordan块一一对应，但并不是每个Frobenius矩阵的若当型都只有一个Jordan块
    - 因为有理标准型是根据最小多项式特化构造的，本质是从全体Frobenius矩阵中选出了一部分用到有理标准型中而已</font>
  - 右端向量为 $\bs f(x) = \begin{pmatrix} 0 \\ 0 \\ \vdots \\ f(x) \end{pmatrix}$
  - **标量函数组的朗斯基行列式**：$W(x) = \begin{vmatrix} \varphi_1(x) & ... & \varphi_n(x) \\ \p_1'(x) & ... & \p'_n(x) \\ \vdots &  &  \end{vmatrix}$
    - **逐行迭代性**：高阶线性微分方程转化的方程组中，每一行都是上面一行的导数，所以它仅依赖于第一行（但不是线性关系而是导数关系）
- **刘维尔公式**：$$W(x) = W(x_0)e^{-\dis\int^x_{x_0}a_1(s)ds}\quad (a<x<b)$$
  - **证明**：
    - 易得系数矩阵中 $\tr[A(x)] = -a_1(x)$
    - 故由刘维尔公式即得结论
    - <font color='chartreuse'>一阶线性微分方程的解其实也是这个形式</font>

### 二阶线性微分方程

- <font color='chartreuse'>此时刘维尔公式形式比较简单，可以用它直接推导出解。但高阶就难了</font>
- **二阶齐次线性微分方程**：$y'' + p(x)y' + q(x)y = 0$
  - **解**：
    - 设 $y(x),\p(x)$ 是两个线性无关解
    - 由刘维尔公式得 $\varphi y' - \varphi'y = Ce^{-\int p(x)dx}$
    - 易得左式可转化为导数除法的形式，得到 $\dis\frac{d}{dx}(\frac{y}{\varphi}) = \frac{C}{\varphi^2}e^{-\int p(x)dx}$，直接积分即可
    - **通解**：<font color='red'>求通解需要一个特解</font> $$y = \varphi(x)\Big[ C_1 + C_2\int^x_{x_0}\frac{e^{-\dis\int^{s}_{x_0}p(t)dt}}{\varphi^2(s)}ds \Big]$$
- **二阶非齐次线性微分方程**：$y'' + p(x)y' + q(x)y = f(x)$
  - **解**：
    - 设 $W_k(x)$ 是 $W(x)$ 中序号为 $(n,k)$ 元素的代数余子式
    - 则特解为 $$\varphi^*(x) = \sum\limits^n_{k=1} \varphi_k(x) · \int^x_{x_0}\frac{W_k(s)}{W(s)}f(s)ds$$
  - **证明（整体解法）**：常数变易法，设齐次方程组的基解矩阵为 $\varPhi(x)$
    - 根据前面的结论，只需要证明 $$\Phi(x)\int^x_{x_0}\Phi^{-1}(s)\bs f(s)ds \red\Rightarrow \sum\limits^n_{k=1} \varphi_k(x) · \int^x_{x_0}\frac{W_k(s)}{W(s)}f(s)ds$$
    - 将逆矩阵转化为伴随矩阵形式，则易得 $$左式 = \varPhi(x)\int \frac{W^*(s)}{W(s)}\begin{pmatrix} 0 \\ \vdots \\ f(s) \end{pmatrix}ds = \int\frac{f(s)}{W(s)}\varPhi(x)\begin{pmatrix} W_1(s) \\ \vdots \\ W_n(s) \end{pmatrix}ds$$
    - 上式结果是列向量，其第一个分量就是右式（**第一个分量才是解，其它都是解的微分**）
  - **证明（分步解法）**：
    - 首先根据齐次形式的解，设解为 $y = \sum C_k(x)\varphi_k(x)$
    - 然后求导，根据第二行的方程约去一些量，得到类似第二行方程，但含有 $C_k(x)$ 的新方程
    - 然后不断求导和约去，最后求导到最后一行方程，得到 $f(x) = ...$
    - 这样就得到了一个含有 $C_k^{(j)}(x)$ 的新的方程组，联立可以解出 $C_k^{(j)}(x)$，最后积分即可

#### 习题

- **函数线性方程组的求解**：
  - 设 $u(x),v(x)$ 是 $y'' + p(x)y' + q(x)y = 0$ 的一个基本解组，则它可唯一确定 $p(x),q(x)$
  - **解**：
    - 代入得 $\begin{cases} u'' + u'p + uq = 0 \\ v'' + v'p + vq = 0 \end{cases}$
    - <font color='red'>它可看作函数 $p,q$ 的线性方程组</font>
    - 由于是基本解组，故朗斯基行列式 $|W(x)|\neq 0$，从而存在唯一解
    - 高斯消元法易得 $p(x) = \cfrac{\vvec{-u'' & u \\ -v'' & v}}{W(x)}$，$q(x) = \cfrac{\vvec{u' & -u'' \\ v' & -v''}}{W(x)}$
    - <font color='chartreuse'>容易发现这就是克拉默法则</font>
- **解的零点**：
  - 设区间 $I$ 上的 $y'' + p(x)y' + q(x)y = 0$
  - 若 $\p(x)$ 是 $I$ 上的一个非零解，则
    - 其至多只有零阶零点
    - 其在有限闭区间上的零点数量有限，从而孤立
  - **证明**：
    - 反设存在高阶零点 $x_0$。前面已知二阶齐次方程的通解形式 $y(x)$，计算 $y'(x_0)$ 即可发现矛盾
    - 反设零点无限，则由凝聚定理存在聚点 $c$。此时 $\p'(c) = \lim\limits_{\D x\to 0} \dfrac{\p(c+\D x)}{\D x}\neq 0$，从而由极限保号性得 $\p$ 在 $c$ 附近存在非零区间，矛盾
- **山大真题**：
  - 二阶非齐次线性微分方程中
  - 满足 $y(0) = y(1) = 0$ 的解唯一 $\LR$ 齐次形式只有零解满足该式
  - <font color='chartreuse'>由解的线性组合性直得结论</font>
  - **证明（宝哥）**
    - **必要性**：
      - 设非齐次的唯一解为 $y_0(x)$，反设齐次形式存在非零解 $y_1(x)$
      - 则由线性组合性，$y_0(x) + y_1(x)$ 也是非齐次形式的满足条件的解，矛盾
    - **充分性**：
      - 反设 $y_1(x)，y_2(x)$ 都是非齐次形式满足等式的解，则 $y_1(x)-y_2(x)$ 是满足等式的齐次非零解，矛盾

### 常系数高阶微分方程

- **常系数高阶微分方程**：$y^{(n)}(x) = \sum\limits^n_{k=0} a_ky^{(k)}(x)$
- **基本解组**：设 $A$ 有 $s$ 个特征值，且相应代数重数为 $n_s$，则方程的基本解组为 $$\begin{cases} e^{\lambda_1x} & xe^{\lambda_1x} & ... & x^{n_1-1}e^{\lambda_1x} \\ e^{\lambda_2x} & xe^{\lambda_2x} & ... & x^{n_2-1}e^{\lambda_2x} \\ ... \\ e^{\lambda_s x} & xe^{\lambda_s x} & ... & x^{n_s-1}e^{\lambda_s x}\end{cases}$$
  - <font color='chartreuse'>本质就是找 $A$ 的初等因子</font>
  - **证明**：
    - 由 $A$ 是Frobenius矩阵，易得全体 $n-1$ 阶子式的公因子为 $1$，从而其Smith标准型为 $S(\l) = \diag(1,1,...,1,d_n)$
      - <font color='chartreuse'>即 $A$ 的初等因子组只有一个
      - 最小多项式就是特征多项式
      - 即 $A$ 的每个特征值只有一个Jordan块</font>
    - 设过渡矩阵 $P$ 的元素为 $p_{ij}$，则由 $AP = P\diag(J_i)$ 得 $\forall p_{1m_j} \neq 0$
      - <font color='chartreuse'>每个Jordan块第一行对应的P不为0，否则退化</font>
    - 然后根据 $e^{xA}P = Pe^{xJ_i}$，计算若当型即得基解矩阵 $e^{xA}P$ 的第一行元素（下面的 $*$ 表示常系数） $$\begin{cases} \Big( p_{1m_1}e^{\l_1x} \Big) & \Big( *e^{\l_1x} + p_{1m_1}xe^{\l_1x} \Big) & ... & \Big( *e^{\l_1x}+\cdots+\dfrac{p_{1m_1}x^{n_1-1}}{(n_1-1)}e^{\l_1x} \Big) \\ ... \\ \Big( p_{1m_s}e^{\l_sx} \Big) & \Big( *e^{\l_sx} + p_{1m_s}xe^{\l_sx} \Big) & ... & \Big( *e^{\l_sx}+\cdots+\dfrac{p_{1m_s}x^{n_s-1}}{(n_s-1)}e^{\l_sx} \Big) \end{cases}$$
      - <font color='red'>基解矩阵的第一行元素就是解，只求第一行就行</font>
      - 因为它们都是基解矩阵的第一行元素，利用解的线性封闭性，可消去带 * 的项，再把其余的项系数改为 $1$，即得到题设基本解组。（**证毕**）
- **特征方程理论**：
  - 设已知高阶线性ODE $a_ny^{(n)} + ... + a_1y' + a_0y = f$
  - 它的特征方程是 $$ a_n\l^n + ... + a_1\l + a_0 = 0$$
  - 设特征方程的根为 $\l_1,...,\l_n$，则原ODE的基本解组是 $e^{\l_1 x},...,e^{\l_n x}$
    - 若出现重根，则添加 $x$ 项
  - **证明**：
    - 易得该方程就是 $|\l I-A| = 0$，故由上面结论即可
    - <font color='red'>在组合数学中，数列的线性递推公式的特征法，本质是通过换元（Jordan变换），将其转化为等比递推，从而得到通项
    - 在这里也是一样，本质是通过换元（写成方程组后，系数矩阵变为Jordan型），将 $y$ 的高阶线性ODE转化为多个一阶线性ODE，从而得到通解
    - 本质都是泛函分析中，线性算子的谱分解理论</font>

    - 设 $\mu$ 是 $k$ 重特征根（不是特征根时 $k=0$），则一个特解形式为 $$\varphi^*(x) = x^kQ_m(x)e^{\mu x}$$
  - 如果非齐次项是 $\Big[ A_m(x)\cos\b x + B_l(x)\sin \b x \Big]e^{\a x}$ 的形式
    - 设 $\a+\b i$ 是 $k$ 重特征根，$n = \max\{m,l\}$，则一个特解形式为 $$\p^*(x) = x^k\Big[ C_n(x)\cos\b x + D_n(x)\sin\b x \Big]e^{\a x}$$

      - 同理，$\cos\b x$ 就取实部
  - 最后齐次通解 + 非齐次特解 = 非齐次通解</font>

  - **解（通解法）**：
    - 齐次情况下，基本解组为 $\cos\b x，\sin\b x$

  - **解（待定系数法）**：
    - **计算通解**：特征根为 $-1$（三重）
      - 基本解组为 $e^{-x},xe^{-x},x^2e^{-x}$
    - **寻找特解**：
      - $\mu = -1$ 是三重特征根，故可设有特解 $y^* = x^3(a+bx)e^{-x}$
      - 代入微分方程解得 $a = -\dfrac{5}{6}，b = \dfrac{1}{24}$
    - 综上通解为 $y = (C_1 + C_2x + C_3x^2 - \dfrac{5}{6}x^3 + \dfrac{1}{24}x^4)e^{-x}$
- $y'' + 4y' + 4y = \cos 2x$
  - **解（待定系数法）**
    - **计算通解**：特征根为 $\l = -2$（二重）
      - 基本解组为 $e^{-2x},xe^{-2x},x^2e^{-2x}$
    - **寻找特解**：
      - 此时 $\a = 0，\b = 2$，且 $\a\pm \b i$ 不为特征根
        - 故可设有特解 $y^* = a\cos 2x + b\sin 2x$
      - 代入微分方程解得 $a = 0，b = \dfrac{1}{8}$
    - 综上通解为 $y(x) = (C_1+C_2x)e^{-2x} + \dfrac{1}{8}\sin 2x$
- $y'' + y' = xe^x\sin x$
  - **解（待定系数法）**：
    - **计算通解**：特征根为 $\l = 1\pm i$
      - 基本解组为 $e^x\cos x，e^x\sin x$
    - **寻找特解**：
      - 此时 $\a = 1，\b = 1$，故 $\a\pm\b i$ 为一重特征根
        - 故可设有特解 $y^* = xe^x\Big[ (ax+b)\sin x + (cx+d)\cos x \Big]$
      - <font color='red'>由于它求导太麻烦了，故采用复数简化法</font>
        - 考虑右端为 $xe^{(1+i)x}$ 的情况，则有特解 $\p^* = xe^{(1+i)x}(ax+b)$
        - 代入微分方程解得 $a = -\dfrac{i}{4}，b=\dfrac{1}{4}$
        - 则 $y^* = \Im \p^* = \dfrac{xe^x}{4}(\sin x-x\cos x)$
      - 综上通解为 $$y(x) = C_1e^x\cos x + C_2e^x\sin x + \frac{xe^x}{4}(\sin x-x\cos x)$$
- $y'' + y = 1-\dfrac{1}{\sin x}$
  - **解（待定系数法 + 拆分法）**：
    - <font color='red'>可以将非齐次项拆成两部分，分别针对求特解，把特解加起来就是原方程特解</font>
    - **计算通解**：特征根为 $\l = \pm i$
      - 基本解组为 $\sin x，\cos x$
    - **寻找特解**：
      - 关于 $y'' + y = 1$，此时 $\mu = 0$，$m=0$，易得特解为 $y=1$
      - 关于 $y'' + y = -\dfrac{1}{\sin x}$
        - <font color='red'>猜特解是很困难的，此时不妨直接猜通解，然后一步步将其退化</font>
        - 设通解为 $y(x) = a(x)\cos x + b(x)\cos x$
        - 不妨取合适的 $a,b$ 使得 $a'(x)\cos x + b'(x)\sin x = 0$
          - 此时得到方程组 $\begin{cases} a'(x)\cos x + b'(x)\sin x = 0 \\ \\ -b'(x)\cos x + a'(x)\sin x = \dfrac{-1}{\sin x} \end{cases}$
          - 解得 $\begin{cases} a'(t) = 1 \\\\ b'(x) = -\cot x \end{cases}$
        - 计算得通解为 $y(x) = (x+C_1)\cos x + (-\ln|\sin x|+C_2)\sin x$
      - 再将其加上1即可


### 非常系数 + 齐次 + 高阶线性微分方程

- **数分得到的灵感**：$y'' - 3y'\tan x + 2y = 0$
  - **解**：
    - 将原方程变形为 $y''\cos x = 3y'\sin x + 2y$
    - 解必定为 $F = C_1y'\cos x + C_2y\sin x$
      - 对 $F$ 求导可解得待定系数 $C_1 = 1，C_2 = -2$，从而得到辅助函数


### 欧拉方程

- **欧拉方程**：$a_nx^ny^{(n)} + a_{n-1}x^{n-1}y^{(n-1)} + ... + a_{1}xy' + a_0y = f(x)$，其中 $x>0$
  - **解**：
    - 取变换 $x=e^t$，
    - 由于 $\Big[ y(e^t) \Big]' = e^ty'(e^t)$，归纳易得 $\begin{cases} y'_t = xy' \\ y''_t = xy'_x + x^2y''_x \\ y'''_t =  xy'_x + 3x^2y''_x + x^3 y'''_x\\ \cdots  \end{cases}$
  - **通式**：
    - 设微分算子 $D = \dfrac{d}{dt}$，则计算易得 $x^ny^{(n)} = D(D-1)...(D-n+1)y$
    - 代入后进行化简，转化为 $b_ny^{(n)}_t + ... + b_0y = f(e^t)$
    - <font color='chartreuse'>此时转变为高阶常系数微分方程，特征法求解即可</font>
  - **推论**：如果 $x<0$，则取变换 $x = -e^t$ 即可
- **实例1**：$$ x^2y'' + 5xy' + 13y = 0 \quad (x>0)$$
  - **解**：
    - 换元 $x = e^t$，则
      - $x^2y'' = D(D-1)y = D^2y - Dy$
      - $xy' = Dy$
    - 故原式可化为 $D^2y + 4Dy  + 13y = 0$
      - 特征多项式为 $\l^2 + 4\l + 13$，特征根为 $-2\pm 3i$
      - 基本解组为 $e^{-2t}\cos 3t，e^{-2x}\sin 3t$
    - 再变换 $t = \ln x$ 即得通解为 $$ y(x) =  \frac{C_1}{x^2}\cos(3\ln x) + \frac{C_2}{x^2}\sin(3\ln x)$$
