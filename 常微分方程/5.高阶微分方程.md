## 自治方程（驻定方程）

- **自治方程**：不包含显式自变量 $x$ 的方程，一般形式为 $F(y,\dfrac{dy}{dx},...,\dfrac{d^ny}{dx^n}) = 0$
  - **求解（常规降阶法）**：
    - **导数换元**：令 $z = \dfrac{dy}{dx}$，则由链式法则得 $$F\dkh{y,z,z\frac{dz}{dy},...,\varphi(z,\frac{dz}{dy},...,\frac{d^{n-1}z}{dy^{n-1}})} = 0$$
    - **方程退化**：此时 $y$ 变为自变量，$z$ 变为因变量，从而方程降低一阶
    - **迭代求解**：
      - 如果降阶后的方程可解，那么对中间解 $z = F(y)$ 再代入 $z = \frac{dy}{dx}$，继续进行求解
      - 如果降阶后的方程不可解，但仍是自治方程（即不含 $y$ 参数），则可再次进行换元降阶
      - 如果降阶后的方程不是自治方程，则无法继续降阶

### 自治方程的物理意义

- **动力系统**：对于自治方程组 $F(t,\dfrac{d\bs x}{dt},...,\dfrac{d^n\bs x}{dt^n}) = 0$，可以将 $t$ 看作时间，$\bs x$ 看作质点在欧氏空间的坐标，则初值问题 $\begin{cases} F(t,\dfrac{d\bs x}{dt},...,\dfrac{d^n\bs x}{dt^n}) = 0 \\\\ \bs x(t) = \bs x_0 \end{cases}$ 可以看作是描述质点运动的微分方程
- **相空间**：$\bs x$ 的值域
- **轨线**：$\bs x$ 的解函数曲线

### 加速度方程

- **二阶情况**：$\dfrac{d^2 x}{dt^2} = f(x)$
  - **解**：
    - 设 $v = \dfrac{dx}{dt}$，则方程化为 $v\dfrac{dv}{dx} = f(x)$，变量分离法得 $v^2 = 2F(x) - C_1$
      - <font color='chartreuse'>此时需要针对正负号讨论</font>
    - 此时方程退化为 $\dfrac{dx}{dt} = \pm\sqrt{2F(x)-C_1}$，变量分离法得通积分为 $$\dis\int\frac{dx}{\pm\sqrt{F(x)-C_1}} = t+C_2$$
- **位移-速度模型**：
  - **相平面**：平面 $Oxv$
  - **轨线**：中间解 $v = F(x)$ 确定的曲线族
  - **相图**：相平面上的轨线分布图
- **相图的几何作图**：
  - 已知 $v^2 = 2F(x) + C_1$
  - 首先画出 $2F(x)$ 的图形
  - $v(x)$ 是个分段函数
    - 当 $2F(x) > -C_1$ 时，轨线为 $v = \pm\sqrt{2F(x)+C_1}$
    - 当 $2F(x_1) = -C_1$ 时，轨线为单点 $(x_1,0)$
    - 当 $2F(x) < -C_1$ 时，轨线为 $v$

### 单摆问题

- **单摆方程**：$\dfrac{d^2x}{dt^2}+a^2\sin x=0$
  - **物理推导**：
    - 由牛顿第二定律 $F = ma$，结合单摆模型得 $-mg\sin x = m(l\dfrac{d^2x}{dt^2})$
    - 取 $a = \sqrt{\dfrac{g}{l}}$，变形即可
- **求解（首次积分法）**：
  - 设 $p = \dfrac{dx}{dt}$，两边同乘 $p$ 得 $p\dfrac{dp}{dt} + a^2\sin x\dfrac{dx}{dt} = 0$
  <!-- - 两端对 $t$ 积分得 $\dfrac{1}{2}p^2 - a^2\cos x = -\dfrac{1}{2}C_1$ -->
    <!-- - $\int a^2sinx \dfrac{dx}{dt}\cdot dt = a^2cosx + C$ -->
  - 两端对 $t$ 积分后变形即得**轨线方程** $\dfrac{dx}{dt} = \pm\sqrt{2a^2\cos x - C_1}$
  - 再由分离变量法即得通积分 $$
- **求解（近似法）**：
  - $\sin x \approx x$，则原方程化为线性方程，解为 $x = Asin(at + D)$
  - 振幅和速度初值有关：$A = \frac{C_1}{a}$
  - 相位和位移初值有关：$D = aC_2$
  - 即 $w = \frac{k}{m}$
  - 因此，方程的解和初值有关。$C_1$ 表示速度初值，$C_2$ 表示位移初值
  - **当单摆存在大振幅时**：设振幅为 $A$，初值为最大振幅处，则 $C_1 = \cos A$
    - 求周期：对四分之一周期进行变量分离积分
      - $\large\frac{1}{4}T = \int^A_0 \frac{dx}{\sqrt{2a^2cosx - cosA}}$
      - 发现周期和振幅有关
    - **单摆的进动（存在正加速度）**：振幅变大，周期变化，角速度发散
      - 体现在轨线上，$a$ 不变，$C_1$ 变大
  - **相图**：$\Gamma_A$ 是普通轨线，$\Gamma_1、\Gamma_2$ 是进动轨线 $\\$

### 悬链线问题

- **悬链线方程**：
  - 设 $\gamma$ 是微分弧 $ds$ 上的重力
    - 水平方向的微分弧受力方程：$H(x) = H(x+\triangle x)$
    - 竖直方向的微分弧受力方程：$V(x+\triangle x) - V(x) =\gamma·ds$
  - 综上，$\begin{cases} H(x) = H，水平方向受力 \\ V'(x) = \gamma\frac{ds}{dx}，竖直方向受力 \end{cases}$
    - 由弧长公式得 $\frac{ds}{dx} = \sqrt{1+(y'(x))^2}$
    - 由张力分解得 $y'(x) = tan\theta = \frac{V(x)}{H}$
  - **物理方程**：$y'' = a\sqrt{1+(y')^2}$，其中 $a = \dfrac{\gamma}{H}$
  - **边值问题**：在题目中
    - 已知两个零阶初值（两个边界初值）$\begin{cases} y(x_1) = y_1 \\ y(x_2) = y_2 \end{cases}$
    - 但不知道每一阶初值 $y(x_0)，y'(x_0)$
  - **求解**：
    - 二阶自治系统，可常规降价，得到含参通解 $y = \frac{1}{a}\cosh a(x+C_1) + C_2$
      - $a,C_1,C_2$ 之间存在依赖关系
    - 由边值条件，得到二阶方程组，求解可得 $C_1(a),C_2(a)$
    - 但是 $a$ 的具体值还没有确定，需要利用边值条件的约束来求取
  - **解的约束条件**
    - **约束1（题目约束：大小）**：$L > \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$
      - 弧线 > 割线
      - 可消去不合理解
    - **约束2（边值约束：等价）**：$L = \int^{x_1}_{x_2}\sqrt{1+(y')^2}dx = \frac{1}{a}\int^{x_1}_{x_2}y''(x)dx$
      - 左项是弧长积分公式，右项是悬链线物理方程的性质
      - 可得到 $a$

### 二体问题

- **二体问题**：太阳原点坐标系，地球坐标设为 $r(t) = (x(t),y(t),z(t))$
  - 万有引力定律得 $f(t) = -G\dfrac{m_sm_e}{|r(t)|^2}\dfrac{r(t)}{|r(t)|}$
  - 牛顿第二定律得 $m_er''(t) = f(t)$
  - 综上，$$\begin{cases} x'' = -\cfrac{Gm_sx}{(\sqrt{x^2+y^2+z^2})^3} \\\\ y'' = -\cfrac{Gm_sy}{(\sqrt{x^2+y^2+z^2})^3} \\\\ z'' = -\cfrac{Gm_sz}{(\sqrt{x^2+y^2+z^2})^3} \end{cases}$$

## 高阶ODE的向量表示法

- **$n$ 阶常微分方程的显函数形式**：$$\frac{d^ny}{dx^n} = F(x,y,\frac{dy}{dx},...,\frac{d^{n-1}y}{dx^{n-1}})$$
- **$n$ 阶常微分方程的标准方程组形式**：设 $y_1=y，y_2=\frac{dy}{dx}，...$，则原方程可写为 $$\begin{cases} \dfrac{dy_1}{dx} = f_1(x,y_1,...,y_n) \\\\ \dfrac{dy_2}{dx} = f_2(x,y_1,...,y_n) \\\\ \cdots \\\\ \dfrac{dy_n}{dx} = f_n(x,y_1,...,y_n) \\ \end{cases}$$
  - **知 $1$ 求 $n$ 性**：若已知 $f_i$，则 $k>i$ 时微分，$k<i$ 时积分，可导出所有 $f_k$
  - **依赖性等价定理**：各阶导数之间的微分关系 $\Leftrightarrow$ 方程组刻画的多变量依赖关系
- **多元常微分方程组的标准化**：含有多个未知数的ODE方程组，也可以分别将 $u,u',...,v,w$ 等设为 $y_i$，写成标准方程组形式
- **方程组的向量函数形式**：$\dfrac{d\bs y}{dx} = \bs f(x,\bs y)$
  - 解向量 $\bs y(x) = \tvec{y_1(x) \\ \vdots \\ y_n(x)}$
  - 分量函数 $\dfrac{dy_i}{dx} = f_i(x,\bs y) = f_i(x,y,...,y_n)$
  - 向量值函数（方程组）$\bs f(x,\bs y) = \tvec{f_1(x,\bs y) \\ f_2(x,\bs y) \\ \vdots \\ f_n(x,\bs y)}$
