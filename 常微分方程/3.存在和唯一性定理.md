# 常微分方程3：存在和唯一性定理

- 这一章大量应用积分上下界不等式，基础是皮卡序列的积分性，推导目标是其一致收敛到积分曲线
- **存在性**：无奇点
- **唯一性**：解只有一族平行曲线 $\Leftrightarrow$ 每点上只有一个函数满足微分方程
- **不唯一性**：存在奇解、特解等，

## 皮卡存在和唯一性定理

- **利普西茨条件**：$|f(x,y_1) - f(x,y_2)| \leq L|y_1 - y_2|$
    - **y连续性**：在y上，$\triangle f$ 是 $\triangle y$ 的高阶无穷小
    - **偏导数有界**：由L中值定理，变换得到 $f_y(x,\xi)\leq L$
    - **常用强化条件**：y偏导数连续，则闭区间上满足Lipschitz条件
  - **补充**：皮卡序列积分不等式 $|y_n(x) - y_0| \leq M|x-x_0|$ 是隐函数和其自变量的关系，是皮卡序列必然满足的性质
- **皮卡定理**：初值问题 $(E): \frac{dy}{dx} = f(x,y), \quad y(x_0) = y_0$
  - 若
    - $f$ 在闭矩形区域 $R:\begin{cases}|x-x_0| \leq a \\ |y-y_0| \leq b \end{cases}$ 内连续
      - （可积且一致连续，则皮卡序列极限函数—积分曲线函数可导，方程有解）
    - 对 $y$ 满足 $Lipschitz$ 条件
      - （皮卡序列收敛）
  - 则(E)在区间 $I = [x_0 - h, x_0 + h]$ 有且只有一个解
    - 其中 $\begin{cases} h = min\{ a,\frac{b}{M}\} \\ M > max|f(x,y)| \end{cases}$
- **证明（类似Newton切线迭代法，这里是级数迭代法）**：
  - 第一步：**转化为积分方程** $y = y(x_0) + \int^x_{x_0} f(x,y)dx$ 有解
    - 其中 $F(x_0,y_0)，y(x_0)$ 都是常数，可以随意设置
  - 第二步：**构造皮卡序列，证明连续性**
    - $y_{n+1}(x) = y_0 + \int^x_{x_0} f(x,y_n(x))dx \quad (x\in I,n = 0,1,...)$
    - 由积分上下界不等式，得到 $|y_n(x) - y_0| \leq M|x-x_0|$
    - 由积分的连续性传递，得 $\{y_n(x)\}$ 连续
  - 第三步：**证明皮卡序列一致收敛**
    - 等价于级数 $\sum\limits^\infty_{n=1} [y_{n+1}(x) - y_n(x)] = \int^x_{x_0} f(x,y_n)dx$ 收敛
      - （数列收敛转化为裂项级数收敛）
    - **数学归纳法证不等式** $|y_{n+2}(x) - y_{n+1}(x)| \leq \frac{M}{L} · \frac{(L|x-x_0|)^{k+2}}{(k+2)!} $
      - 由Lipschitz条件，可放缩为 $|y_{n+2}(x) - y_{n+1}(x)| = |\int(f_{k+1}-f_{k})| \leq |\int^x_{x_0} L |y_{k+1}(x) - y_k(x)|dx|$，退化一阶
      - 再代入归纳法假设，右式计算积分得 $\frac{M}{L} · \frac{(L|x-x_0|)^{k+2}}{(k+2)!}$
    - 由等比级数的W判别法，$\{y_n(x)\}$ 收敛
  - 第四步：$n\to \infty$ 时，$y(x) = y_0 + \int^x_{x_0} f(x,y(x))dx$，也就是积分方程
    - **所以皮卡序列的极限函数就是积分方程的解，在区间上一致成立**
  - 第五步：**证明唯一性**
    - 反设存在两个解 $u(x)和v(x)$，它们作差的结果是刚才的皮卡序列裂项级数，极限值为 $\frac{M}{L} · \frac{(L|x-x_0|)^{k+2}}{(k+2)!} \to 0$
  - **理解**：
    - 基本思路是由收敛的完备性证明有解（逼近法、迭代法）
    - 核心是证明收敛使用的Lipschitz条件
    - 依据是构造的皮卡序列，其差值具有“积分内部同构”的形式，可以通过积分上下界不等式简化为简单不等式，而序列迭代（重积分）可以将右侧化为等比序列，从而收敛。

#### 习题

- **利用收敛级数求解方程**：$\frac{dy}{dx} = x+y+1\quad y(0)=0$，
  - 法1：一阶线性方程，可以直接解
  - 法2：利用皮卡序列和级数。具体的皮卡序列为：
    - $y_0 = 0$
    - $y_{n+1} = \int^x_{x_0} (x+y_n+1)dx$
    - $y_{n+1}-y_n = \int^x_{x_0} (y_n-y_{n-1})dx$
    - $y_1 = x+1$，$y_2=\int^x_{x_0}(x+(\frac{1}{2}x^2+x)+1)dx,...$
    - 通项公式（多重积分运算后）：$y_n = \sum\limits^n_{k=1}\frac{x^{k+1}}{(k+1)!} + \sum\limits^n_{k=1}\frac{x^k}{k!} = (e^x-x-1)+(e^x-1)$
- **近似求解**
  - 已知区间，则可求得 $h和L$
  - 由皮卡不等式 $|\varphi_n(x) - \varphi(x)| \leq \frac{ML^n}{(n+1)!}h^{n+1} \leq g(n)$ 可根据题目中要求的精度来计选n的值
  - 然后进行皮卡序列迭代，求出 $\varphi_n(x)$ 就是近似值

### 奥斯古德条件（唯一性条件）

- **Osagood条件**
  - 函数在G内连续
  - 满足 $|f(x,y_1) - f(x,y_2)| \leq F(|y_1-y_2|)$
    - （$F(r) > 0$ 且在 $(r,+\infty)$ 连续）
  - 以0为瑕点的瑕积分 $\int^{r_1}_0 \frac{dr}{F(r)} = +\infty$
    - （在x = 0处只有一个解，$F(0) \to 0$）
- **同L条件比较**：
  - （Lipschitz条件中，$F(r) = Lr$，也就是 $\triangle f和\triangle y$ 是正比例关系）
  - （Osgoods的退化定理中，附加了一个条件 $\frac{dy}{dx} = f(y)$
- **唯一性定理**：满足Osgoods条件则解唯一
- **证明（反证）**：
  - 设 $x_0$ 是同时具有两解的点，$x_1$ 是令两个解曲线的值不同的某个点
  - $\bar{x} = sup\{x | y_1(x) = y_2(x)\}$，是两线相同的上确界（边界），则 $r(\bar{x}) = 0$
  - 设 $在[\bar{x},x_1]上，r(x) = y_1(x) - y_2(x) > 0$
  - 由常微分方程得到：$r'(x) = y_1'(x) -y_2'(x) = f(x,y_1(x)) - f(x,y_2(x)) \leq F(|y_1(x) - y_2(x)|) = F(r(x))$
  - 移项并积分x得 $\int^{r(x_1)}_0 \frac{dr}{F(r)} \leq x_1 - \bar{x}$（$dr=f(x,y_1)-f(x,y_2)$），左边无限右边有限，矛盾，**证毕**
- **理解（线素矛盾：斜率与函数值相互决定（方程），但又相互影响（曲线））**：
  - 瑕积分条件实际上就是先射箭后画靶
  - 由不等式 $\frac{f(x,y_1)-f(x,y_2)}{F(|y_1-y_2|)} \leq 1 \Rightarrow \int^{|y_1-y_2|}_{x:y_1=y_2} \frac{f(x,y_1)-f(x,y_2)}{F(|y_1-y_2|)}_{\to +\infty} \leq |y_1-y_2|_{\to +\infty}$
    - 再由 $y_1、y_2$ 的任意性，发现上式必须无意义，否则积分曲线始终无限，无意义。
    - 所以 $y_1=y_2$ 时，$\triangle f$ 必须 $=0$，或该点压根就不存在（$y_1\neq y_2$ 的地方在无穷远），总之解唯一。

### 米勒反例

- 不满足利普西茨条件的方程，其皮卡序列不一定收敛
- $F(x,y) = \begin{cases} 0，\qquad x = 0 \\ 2x，\qquad (0,1]\times (-\infty,0) \\ 2x-\frac{4y}{x}，(0,1]\times [0,x^2) \\ -2x，\qquad (0,1]\times [x^2,+\infty) \end{cases}$
- 其在 $[0,1] \times (-\infty,+\infty)$ 上连续，对y不满足L条件
- 皮卡序列 $y_n(x) = (-1)^{n+1}x^2，x\in [0,1]$ 不收敛
- 解为 $y = \frac{1}{3}x^2$ ，且唯一

#### 习题

- 已知初值时，讨论解的单向延伸唯一性
  - $y(0) = 0，\frac{dy}{dx} = |y^\alpha|$，令 $F(r) = |r|^\alpha$，证明 $\lim\limits_{r\to 0}\int = \infty$
  - $y(0) = 0，\frac{dy}{dx} = \begin{cases} 0,\quad y=0 \\ yln|y|, \quad y\neq 0 \end{cases}$，令 $F(r) = |rln|r||$
  - 使用Osgood条件
    - Osgood中的 $y_1和y_2$，原先是在区间中任意取的。但是由其证明方法可以看出，如果把 $y_2$ 取为左端点，则同样可以在 $[x_0,\beta]$ 区间内全部适用
- 若 $f(x,y)$ 在闭区域上连续，且对于y单调递减，则其在右侧上解唯一
  - **证明**：
    - 法一，利用闭区间上单调连续函数的不可导点孤立性、导数无限点孤立性，构造皮卡序列，然后利用L中值定理，得到类Lipschits条件
    - 法二：仿照Osgoods条件，使用线素矛盾来证明。
      - 反设有两解 $y_1(x),y_2(x)$，令 $x_1$ 是两解不相等的最小点
      - 设 $r(x) = y_1(x)-y_2(x)$ 在 $x_1$ 右侧 $>0，r(x_1)=0$。
      - 由方程定义和$f$单调性，$r'(x_1) = f_{y_1}-f_{y_2} < 0$，$r$ 单减，右侧 $r(x)<0$
      - 由端点效应，可以发现矛盾（单减，则函数和导数步调相反，而常微分方程又添加了函数和导数的相互决定关系，所以出现矛盾。如果单增，函数和导数步调一致，就不会矛盾了）

## 皮亚诺存在定理

- **欧拉折线**：$\frac{dy}{dx} = f(x,y)$，$f$ 连续
  - **斜率有界性**：闭区间内积分曲线的斜率有界，即 $|y-y_0| \leq M|x-x_0|$
  - 根据斜率的性质，可以构造简单折线（欧拉折线）来拟合积分曲线
- **构造方法**：
  - 设矩形区域 $R：\begin{cases} |x-x_0|\leq a \\ |y-y_0|\leq b \end{cases}$
  - 限制积分曲线在矩形区域内：
    - $|y-y_0| \leq b \Rightarrow |x-x_0| \leq \frac{b}{M}$
    - 设 $h = min\{a,\frac{b}{M}\}$，则 $(x_0-h,x_0+h)$ 内部曲线不超出 $R$
  - 两个不等式构成了一个水平角形区域（斜率不超过M，则上界是 $y=Mx$）<img src = "./图片/peano存在定理.png">
  - 在 $P_0(x_0,y_0)$ 取线素折线，延长至 $P_1$，得到 $y_1 = y_0 + f(x_0,y_0)(x_1 - x_0)$（由于M的最大性，$P_1$ 仍然在角形区域内）
  - 依次取连续的折线，得到**欧拉序列（上限函数）**：
    - $\varphi_n(x) = y_0 + \sum\limits^{s-1}_{k=0} f(x_k,y_k)(x_{k+1}-x_k) + f(x_s,y_s)(x-x_s)$
    - n是折线端点的数量，它决定欧拉折线整体的形状。
    - s是x当前所在的端点序号，它表示在某个特定形状的欧拉折线上的某个端点
    - y就是当前端点的函数值（积分曲线值）
    - 欧拉序列是函数项级数，n越大，拟合程度越高，极限函数就是解。
      - 它现在还不是皮卡序列，只是朴素的级数而不是积分。但数学本质和皮卡序列相同。后面会证明它和皮卡序列差值无穷小（）
- 类似定积分中Newton-Leibniz公式的几何割线证明

### 阿斯科利引理

- **函数序列一致有界**：$\forall n，\exist 常数M，|f_n(x)| < M$
  - **一致性**：函数序列仅有界时，$M$ 可以是 $n$ 的函数
  - （有界是针对函数，一致有界是针对序列，它们的应用域本就不同，不用纠结）
- **函数序列等度连续**：$\forall \varepsilon，\exist \delta = \delta(\varepsilon)，\forall x_1,x_2 \in I，|x_1-x_2| \leq \delta，就有|f_n(x_1) - f_n(x_2)| < \varepsilon，n = 1,2,...$
  - **等度性**：
    - 一个 $\delta(\varepsilon)$ 可以管住所有的n，即n之间不能分开来看
    - 极限函数也连续
    - 所有函数列均一致连续
- **反例**：$f_n(x) = (-1)^n + x^n$
  - 在 $[-\frac{1}{2},\frac{1}{2}]$ 上，一致有界且等度连续
  - 在 $[-1,1]$ 上，一致有界但不等度连续
    - 极限函数 $f(x) = \begin{cases}  \end{cases}$
  - 在 $[-2,2]$ 上，均不满足
    - 等比数列不收敛
- **阿斯科利引理**：函数序列在有界闭区间上一致有界、等度连续，则它有一个一致收敛的子列
  - **证明**：见泛函分析
  - 有限开区间上也成立，方法和一致连续相同，补充定义即可
  - 无限区间上不成立：$f_n(x) = sin\frac{x}{n}，x\in R$
    - 一致有界且等度连续
    - 但任何子列均振荡发散

### Peano存在定理

- **导出一致收敛**：欧拉序列在区间 $[x_0-h, x_0+h]$ 上至少有一个一致收敛的子序列
  - **证明**
    - 首先它在矩形区域中一致有界
    - 其次它是折线，且割线的斜率有界，等度连续
  - **理解**：解的存在性只取决于是否收敛，也就是我们只关心极限函数的存在性。子序列收敛足以确定一个极限函数（欧拉折线的n并不重要，重要的是 $\varphi(x)$）
- **导出皮卡序列**：欧拉折线满足 $\varphi_n(x) = y_0 + \int^x_{x_0} f(x,\varphi_n(x))dx + \delta_n(x)$。其中 $\delta_n(x) \to 0$
  - 第一步：配凑式子（易得，不是重点）
    - 由定积分定义，有 $f(x_i,y_i)(x_{i+1}-x_i) = \int^{x_{i+1}}_{x_i} f(x_i,y_i)dx，(\triangle \to 0)$
    - 添项配凑得 $\begin{cases} 右式 = \int^{x_{i+1}}_{x_i} f(x,\varphi_n(x))dx + d_n(i) \\ d_n(i) = \int^{x_{i+1}}_{x_i}[f(x_i,y_i)-f(x,\varphi_n(x))]dx \end{cases}$
    - 把它应用到整个区间，裂项累加就得到题干要求的式子，其中 $\sum\limits^{s-1}_{i=0} d_n(i) = \delta_n(x)$
    - $[x_s,x]$ 区间中的差值设为 $d_n^*$
  - 第二步：证明 $\sum\limits^{s-1}_{i=0}d_n(i) + d_n^* \to 0$
    - 由欧拉折线定义，$\begin{cases} |x-x_i| \leqslant \frac{h}{n} \\ |\varphi_n(x) - y_i| \leqslant M \end{cases}$中 $f$ 的连续性，以及区间划分无穷小，积分上下界不等式，得到 $d_n(i)<\varepsilon$
    - 由 $s$ 是有限数，得 $\delta_n(x) \to 0$
  - **理解**：
  - **本质**：级数定积分拟合
- **Peano存在定理**：函数在 $R$ 中连续，则初值问题E在区间 $[x_0-h,x_0+h]$ 上至少有一个解
  - **证明**：
    - 选取欧拉折线序列的一个一致收敛子序列，由等度连续性，其极限函数 $\varphi(x)$ 连续
    - 然后由欧拉折线的定积分拟合式，得到皮卡序列的极限形式 $\varphi(x) = y_0 + \int^x_{x_0} f(x,\varphi(x))dx$
    - 该形式下的函数 $\varphi(x)$ 就是微分方程的解
  - **理解**：皮卡定理中一致收敛（完备性导出解）需要Lipschitz条件。Peano存在定理因为阿斯科利引理可以得到一致收敛。这么说吧，欧拉折线 $\varphi_n(x)$ 就是皮卡序列中具体的 $y_n$，满足积分上下界不等式
    - 补充：皮卡序列和欧拉折线序列不是一个东西，不能用皮卡序列证明Peano存在定理（皮卡序列的一致收敛依赖于Lipschitz条件）
- **性质**：
  - 欧拉折线的一致收敛子序列均趋于 $f$ 的积分曲线
  - 不保证唯一性
  - 若函数不连续，则初值问题可能无解
    - $f(x,y) = \begin{cases} 1，\qquad 1\leqslant |x+y| < +\infty \\ (-1)^n，\frac{1}{n+1} \leqslant |x+y| \leqslant \frac{1}{n} \\ 0，\qquad |x+y| = 0 \end{cases}$
    - 则 $(E^*): \frac{dy}{dx} = f(x,y)，y(0) = 0$ 无解

## 解的延伸

### 解的延伸定理

- 已知 $\frac{dy}{dx} = f(x,y)$ 连续
- 开区域 $G$ 内某点有定义的积分曲线必定延伸到G的边界
  - 即有界闭区域 $G_1$ 内的积分曲线必定延伸到其外部
  - 即有限闭区域不可能完全包含一个积分曲线
- **证明（以右侧为例）**：
  - $\Gamma: y=\varphi(x)\quad x\in J$。
    - $\Gamma$ 为积分曲线，$J$ 为 $\Gamma$ 的最大存在区间，$x_0$ 是初值已知点。
    - 设 $J^+ = J\cap [x_0,+\infty)$，称为**右侧最大存在区间**
  1. $J^+$ 为无限区间，显然成立
  2. $J^+$ 为有限闭区间，则设其右边界为 $x_1$，在 $x_1$ 处设置闭矩形，因为初值问题中的 $f$ 在G中连续，由Peano存在定理发现其在闭矩形上有解，则延伸到外部
  3. $J^+$ 为有限半开区间，我们使用无穷闭区间逼近开区间的方式
     - 即证明
       - $\forall G_1\subset G，不能使\forall x\in J^+，有(x,\varphi(x))\in G_1$
       - $\forall G_1\subset G，\exist x\in J^+，使得(x,\varphi(x))\notin G_1$
     - 反证，构造皮卡序列 $\varphi(x) = y_0 + \int^x_{x_0} f(x,\varphi(x))dx \quad (x_0\leq x<x_1)$
       - $f = \varphi'(x)$ 连续 $\Rightarrow$ 取闭区间 $G_1$，则闭区间上的连续函数有上界 + 积分上界不等式 $\Rightarrow$ 导数 $\varphi'(x)$ 内闭一致有界 $\Rightarrow$ 开区间上导数 $\varphi'(x)$ 有界，从而在开区间上一致连续，从而G的端点极限存在，则延伸到 $G_1$ 外部，G的边界。

- **推论（局部定理）**：$f$ 在区域G内连续，且满足局部Lipschitz条件，则G内每点 $P_0$ 有唯一解，且解延伸到边界。
  - **局部L条件**：$L$ 依赖于点的选取
  - **证明**：皮卡定理 + 解的延伸定理
  - **理解**：
    - 有界闭区域内，局部L条件等于整体L条件（有限覆盖定理 + 取最大的L）
    - 开区域内，局部L条件不等于整体L条件（开区间只能被无限逼近，则L无上界）

### 最大连续存在区间

- **边界二维性**：xy平面上延伸到边界 $\Leftrightarrow$ 延伸到无穷远（x方向边界）或函数无界（y方向边界）
- **最大连续存在区间**：
    - 积分曲线不一定连续。若区间不连续，则最大连续存在区间可能有限 （如 $\tan x$）
- **初值决定性**：解的最大存在区间和初值具有决定关系（如下）
  - 一般是初值越靠近某个值，延伸能力越好。
- $\frac{dy}{dx} = x^2+y^2$（里卡蒂方程）最大存在区间有界
  - **证明**：设 $J^+ = [x_0,\beta_0)$ 是右侧最大存在区间
    - 若 $\beta_0\leqslant 0$，则区间有限。讨论 $\beta>0$ 即可
    - 取 $x_1>0$，使得区间 $[x_1,\beta_0)$ 内，有不等式 $y'(x) \geqslant x_1^2 + y^2(x)$。变形为 $\frac{y'(x)}{x_1^2 + y^2(x)} \geqslant 1$
    - 其为arctan的微分。积分得 $\frac{1}{x_1}(\arctan\frac{y(x)}{x_1} - \arctan\frac{y(x_1)}{x_1}) \geq x-x_1 \geq 0$
    - 由 $\arctan$ 有界性，$x-x_1 \leqslant \frac{2\pi}{x_1}$，有界
    - 再由 $x$ 任意性，得 $\beta_0$ 有界，从而最大存在区间有限，与解的初值有关
  - **理解**：通过取特定值，构造不等式，简化积分，得到有界函数
- $\frac{dy}{dx} = (x-y)e^{xy^2}$：解在 $[x_0,+\infty)$ 上都存在
  - **证明（观察线素/比较定理）**：
    - 首先观察得 $y=x$ 是解
      - 直线上方的函数 $k<0$，单减
      - 直线下方的函数 $k>0$，单增
    - 然后，发现在 $y=x$ 上，直线 $k=0$。所以在直线上方（左方）的函数有可能穿过直线，但直线下方（右方）的函数不可能穿过直线（否则直线上的线素 $k>0$）
    - 所以直线下方的积分曲线延伸到无穷远
- **推论**：
  - 若区域 $G$ 上满足皮卡定理，则 $G$ 内部存在区间一定连续
  - 若区域 $G$ 上只满足Peano定理，则积分曲线不一定连续，从而存在区间不一定连续

### 一阶线性微分方程的控制性

- **p比较定理**：设 $f(x,y)$ 在条形区域 $x\in (\alpha,\beta)$ 内连续
  - 若满足不等式 $|f(x,y)| \leq A(x)|y| + B(x)$
    - 其中A、B在该区间上连续
  - 则每个解的最大存在区间都是 $(\alpha,\beta)$
- **证明**：只需要证明 $\forall \beta_0<\beta$ 都不是最大存在区间即可
  - 首先设 $x_2-x_1 < x_1-x_0$
    - 防止矩形区域 $R_1$ 向左超过 $x_0$ 点。应该讨论的是右端
  - 设 $A(x)、B(x)$ 在区间上的上界为 $A_0、B_0$
    - 再设 $x_2-x_1 < \frac{1}{A_0} - 1$
  - 再设置一个矩形区域 $R_1：\begin{cases} |x-x_1|\leq x_2-x_1 \\ |y_1-y_2| \leq b_1 \end{cases}$
    - 设 $M_1 = A_0(|y_1|+b_1)+B_0+1$
      - 由题设不等式，$f(x,y) < M_1$
      - 由欧拉折线定义，存在区间半径为 $h=min\{x_2-x_1,\frac{b_1}{M_1}\}$
  - 由 $b_1$ 任意性 + 前设放缩不等式，得 $\lim\limits_{b_1\to +\infty} \frac{b_1}{M_1} = \frac{1}{A_0} > x_2-x_1$
    - y无界时，$h = x_2-x_1$
    - 所以，只要令 $x_2>\beta_0$，则解总会延伸到 $\beta_0$ 以外。由 $\beta_0$ 任意性得最大存在区间为连续边界 $[x_0,\beta_0)$（$x_0$ 为初值已知点）
- **理解**：
  - 由欧拉折线定义，解的最大存在区间半径取决于x限制和y限制中较小的那个
  - 本题核心是题设不等式中的一次项y，它使得y限制（无界 $b_1$）存在极限，从而y对存在区间限制的直接体现 $\frac{b_1}{M_1}$ 是有界的
  - 而x的限制是作差，并未受到约束，所以就把决定解的存在区间的要素锁定在了x上，即解仅依赖于x
  - 从而解的存在区间就充满了整个 $(\alpha,\beta)$ 区间。
- **本质**：增速小于一阶线性微分方程 $E$ 的曲线，最大存在区间都不小于 $E$（比较定理的前瞻）
- **推论**：一阶线性微分方程，其最大存在区间为 $f(x,y)$ 的定义域

### 习题

- 讨论解的存在区间
  - $\frac{dy}{dx} = \frac{1}{x^2+y^2}$
    - 在 $x\to \infty$ 时，$k\to 0$，且其在定义域内连续，则其并不存在无界情况
    - 再由Peano存在定理和解的延伸定理，得到其解可以延伸至无穷
  - $\frac{dy}{dx} = y(y-1)$
    - $f(x,y)$ 只与y有关，满足瑕积分条件之一，得到解的唯一性
    - $y=0、y=1$ 是其特解，所以积分曲线不会穿过 $y=0、y=1$
    - 因此若初值小于0，则右侧延伸至无穷（逼近x轴）。若初值介于0、1之间，则解不存在。若初值大于1，则左侧延伸至无穷（逼近 $y=1$）
  - $\frac{dy}{dx} = ysin(xy)$
    - $f(x,y)\leq |y|$，解延伸至定义域边界
  - $\frac{dy}{dx} = 1+y^2$：直接求解
  - $\frac{dy}{dx} = (y-3)(y+1)e^{(x+y)^2}$
    - 如果积分曲线与 $y=-1$ 或 $y=3$ 有交点，则其后与两条水平直线重合，延伸至无穷
    - 如果积分曲线处于 $y\in [-1,3]$ 条形区域中，则其有界，延伸至无穷
    - 如果积分曲线处于 $y\in (3,+\infty)$ 中，则 $k>0$，左降右升，左侧有界，延伸至无穷
  - $\frac{dy}{dx} = (x^2-y^2)f(x,y)$，且 $yf(x,y)>0\quad (y\neq 0)$
    - 水平等斜线位于 $y=x和y=-x$ 上
    - 当 $x_0<0，且|y_0|$ 足够小时，解可延伸到R边界

## 比较定理

### 第一比较定理

- 两个初值问题
  - 若导数 $f < g$，且初值点和初值均相同
  - 则初值左侧 $y_f<y_g$ ，初值右侧 $\int y_f > \int y_g$
- **证明**：端点效应
- **理解**：几何意义上很好理解

### 第二比较定理

- **最小解/最大解**：$\forall y(x)$，有  $W(x) \leq y(x) \leq Z(x)$
- **唯一性易得**
- **存在性定理**：$\exist \sigma<h$，使得区间 $[x_0-\sigma,x_0+\sigma]$ 上有最小解和最大解
  - 首先，设 $(E_m):\frac{dy}{dx} = f(x,y) + \varepsilon_m,\quad y(x_0)=y_0$，其中 $\varepsilon_m$ 是无穷小数列
      - $(E_m)$ 的解集是 $(E)$ 解集的子集，解的极限就是 $(E)$
  - 构造欧拉序列求解：
    - 设 $h_m\to h\quad (m\to +\infty)$，由Peano定理得对应区间内对应有解 $\varphi_m(x)$（欧拉序列）
    - 对其构造皮卡序列，得到 $(E_m)$ 的存在区间半径 $\sigma<h$
    - 仿照Peano存在定理证明极限存在：
      - 积分有界性得一致有界，积分上界不等式得等度连续
      - 得到 $\varphi_m(x)$ 的极限函数 $\varPhi(x)$，它就是 $(E)$ 的解全集
  - 由 $\varphi_m(x) > y(x)$，应用第一比较定理，得到 $\varPhi_m(x)$ 是左行最小解/右行最大解
  - **理解**：导数右向是增长，左向是降低。而 $(E_m)$ 方程刻画的是导数间的恒定大小关系
  - **本质**：单调收敛的结果是最值，类似实变的Fatou引理
- **第二比较定理**：
  - 若导数 $f \leqslant g$，初值点和初值均相同，则初值左侧 $\varphi(x) \geqslant \varPhi(x)$，初值右侧 $\varphi(x) \leqslant \varPhi(x)$
    - $\varphi(x)$：$f$ 右行最小解。$\varPhi(x)$：$F$ 右行最大解

### 解的延伸趋势

- $\frac{dy}{dx} = sin(xy)$ 的解可以延伸到无穷远
- **解的唯一性**：首先它满足Lipschitz条件（$L=\beta$）和p比较条件（$sin(xy)\leq xy$），从而解唯一
- **解的对称性**：它的图像关于两个对称轴对称。
- 初值 $y(0) = 0$ 时：解为 $y=0$
- 初值 $y(0)>0$ 时，要证明 $\lim\limits_{x\to +\infty}y(x) = 0$
  - **第一次放缩**：$sin(xy)\leqslant 1$
    - 积分曲线若和 $L:y=x$ 相交于 $x_1$，则 $x_1$ 右侧积分曲线 $\Gamma$ 在L下方
  - **第二次放缩**：在此假设下，再证明积分曲线在 $H:y = \frac{k}{x}$ 下方
    - **$(0,x_1)$ 区域**：要使 $x_1$ 在 $H$ 下方
      - 即 $y_H(x_1) = \frac{k}{x_1} > y_L(x_1) = x_1$
      - 得 $k$ 要满足 $x_1^2<k$
    - **$(x_1,+\infty)$ 区域**：再反设 $\Gamma$ 不总在 $H$ 下方
      - 则存在交点 $x_2 > 1$，其上斜率满足 $K_H = \large-\frac{k}{x_2^2}\normalsize < sin(x_2y_2) = K_\Gamma$
      - 直接放缩是放不出来的
    - 综上，设 $k = (2m-\frac{1}{2})\pi$ ，其中m是满足 $x_1^2<k$ 的最小正整数
      - 此时由 $\begin{cases} x_1^2<k \\ x_1 < x_2 \end{cases}$，得 $K_H > -1$
      - 计算得 $K_\Gamma = -1$
      - 所以，$k = (2m-\frac{1}{2})\pi$ 时，$\Gamma$ 始终在 $H$ 下方
  - **$x_1$ 的存在性**（$\Gamma$ 和 $L$ 相交）
    - 由于初值 $y(0)>0$，所以若不相交，必定有 $\Gamma$ 始终在L上方
    - 若存在一个始终大于 $\Gamma$ 而最终与 $L$ 相交的曲线，即可证明 $\Gamma$ 和 $L$ 相交
    - 使用反比例函数族分割平面（因为 $xy = k$）
      - 反比例函数族 $H_k: xy = k\pi$
      - 将xy平面划分区域 $G_k: (k-1)\pi < xy < k\pi$
      - 由三角函数的周期性和有界性，$\begin{cases} -1<y'(x)<0, \quad (x,y)\in G_{2n-1} \\  0<y'(x)<1, \qquad (x,y)\in G_{2n} \end{cases}$
      - 构造曲线有点难，先构造折线
        - 初值 $y(0)=y_0$
        - 折线段斜率 $u'(x) = \begin{cases}1,\quad (x,u)\in G_{2n-1} \\ 0,\quad (x,u)\in G_{2n} \end{cases}$（节点在反比例函数上）
        - 此折线取三个相邻的节点，有 $\begin{cases} y_{n+1} = y_n \\ y_{n+2} > y_n \end{cases}$
          - 由反比例函数关系得 $x_{n+2}-x_{n+1} < \frac{\pi}{y_n} = x_{n+1}-x_n$
          - 再利用折线的斜率部分区域上恒定，得 $y_{n+2}-y_n = y_{n+2}-y_{n+1} \stackrel{反比例关系}{\longrightarrow} x_{n+2}-x_{n+1}$
        - 综上我们可以求得 $\large\frac{y_{n+2}-y_n}{x_{n+2}-x_n} = \frac{x_{n+2}-x_{n+1}}{(x_{n+2}-x_{n+1})-(x_{n+1}-x_n)} < \frac{1}{2}$
      - 该折线在直线 $y-y_1 = \frac{1/}{2}(x-x_1)$ 的下方
      - 该直线与L相交，从而 $\Gamma$ 也与L相交（**证毕**）
- **理解**：本质还是比较定理，不过手法很高超

### 比较定理与存在区间

- **比较区间法**：若初值条件相同，且 $f<g$，则 $f$ 的右侧最大存在区间边界大于 $g$，左侧边界小于 $g$
- $\frac{dy}{dx} = x^2 + (y+1)^2,\quad y(0)=0$。解右侧最大存在区间为 $[0,\beta)$，证明 $\beta\in (\frac{\pi}{4},1)$
  - 先证明 $\frac{\pi}{4} \leq \beta \leq 1$
    - $x<1$ 时，由 $(y+1)^2 \leqslant x^2+(y+1)^2 \leqslant 1+(y+1)^2$，可以转化为两个可积的初值问题，比较区间法即可
  - 再证明 $\beta < 1$
    - 构造 $(E_3): \frac{dy}{dx} = (y+1)^2 < f,\quad y(\xi) = \eta$
    - 它的解的右侧最大存在区间为 $[0,\xi+\frac{1}{\eta+1})$
    - 设右侧的函数为 $C(\xi)$
      - 由 $C(0) = 1，C'(\xi)<0$，得 $\xi>0$ 时 $C(\xi)<1$
    - $\xi\to 0+$，即可把初值转到 $x_0 = \xi = 0$ 上，从而可以比较区间
  - 再证明 $\beta > \frac{\pi}{4}$
    - 构造 $(E_4): \frac{dy}{dx} = \lambda^2+(y+1)^2 > f,\quad y(0)=0$
    - 它的解的右侧最大存在区间为 $[0,\tilde{C}(\lambda))$
    - 而 $\tilde{C}(1) = \frac{\pi}{4}，\tilde{C}'(1)<0$，则 $\lambda < 1$时 $\tilde{C}(\lambda) > \frac{\pi}{4}$
    - $\lambda\to 1-$ 即可把初值迁移到 $x_0 = 1-\lambda = 0$ 上来，从而可以比较区间
