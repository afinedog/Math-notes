# 常微分方程10：首次积分

## 微分方程组的首次积分

- **首次积分**：设微分方程组的有段函数在区域 $D\in \reals^{n+1}$ 上连续，对 $y_1,...,y_n$ 连续可微
  - 若
    - 函数 $V = V(x,y_1,...,y_n)$ 在 $D$ 的某个子区域 $G$ 上连续，对 $y_1,...,y_n$ 连续可微
    - $V$ 不是常函数
    - $V$ 沿着 $G$ 内任一积分曲线 $\Gamma$ 上取常值
  - 则 $V(x,y_1,...,y_n) = C$ 即为微分方程组在 $G$ 内的首次积分
- **轨线常值性**：
  - **代数理解**：解函数成立等式，而等式可移项成为零化式，零化式可积分为常值函数，所以轨线常值性的本质就是解使得微分方程成立
  - **几何理解**：
- **首次积分法**：$n$ 阶线性方程组，若确定 $n$ 个独立的首次积分，即可用其求出通解（证明见后面）

### 实例

- **反向单位圆系统**：$\begin{cases} v_x = y - x(x^2+y^2-1) \\ v_y = -x-y(x^2+y^2-1) \end{cases}$
  - **沿圆族积分**：
    - 方程组变形得 $\dis x\frac{dx}{dt} + y\frac{dy}{dt} = d(x^2+y^2) = -2(x^2+y^2)(x^2+y^2-1)dt$
    - <font color='chartreuse'>极坐标下，令半径为常值，对角度积分</font>
    - 积分得 $\dis\frac{x^2+y^2-1}{x^2+y^2}e^{2t} = C_1$
  - **沿角度积分**
    - 方程组变形得 $\dis x\frac{dy}{dt} - y\frac{dx}{dt} = d(\arctan\frac{y}{x}) = -2(x^2+y^2)(x^2+y^2-1)dt$
    - <font color='chartreuse'>极坐标下，令角度为常值，对半径积分</font>
    - 积分得 $\arctan\frac{y}{x} + t = C_2$
  - **极坐标化**：
    - $\begin{cases} (1-\frac{1}{r^2})e^{2t} = C_1 \\ \theta + t = C_2\end{cases}$
    - 它是可解方程，从而得到通解
- **椭球面系统**：$\begin{cases} \alpha\frac{du}{dt} = (\beta-\gamma)vw \\ \beta\frac{dv}{dt} = (\gamma-\alpha)wu \\ \gamma\frac{dw}{dt} = (\alpha-\beta)uv \end{cases}（\alpha>\beta>\gamma>0）$
  - **沿椭球面 $E(C_1)$ 积分**：
    - 方程组变形得 $\alpha u\frac{du}{dt} + \beta v\frac{dv}{dt} + \gamma w\frac{dw}{dt} = 0$
    - 积分得 $\alpha u^2 + \beta v^2 + \gamma w^2 = C_1 \geqslant 0$
  - **沿椭球面 $F(C_2)$ 积分**：
    - 方程组变形得 $\alpha^2 u\frac{du}{dt} + \beta^2 v\frac{dv}{dt} + \gamma^2 w\frac{dw}{dt} = 0$
    - 积分得 $\alpha^2 u^2 + \beta^2 v^2 + \gamma^2 w^2 = C_2 \geqslant 0$
  - **求解**：
    - 两个积分得到线性方程组，解出 $u(w),v(w)$
    - 代入第三式，得到 $w$ 式，移项即得 $w$ 的首次积分
    - 三个首次积分足以确定通解
  - **几何意义**：轨线只能是奇点、闭轨、奇闭轨线
    - **奇点**：穿过坐标轴上的解只能为奇点
      - 首先，若 $u,v$ 初值为0，由原方程组得 $w'(0) = 0$，从而奇点 $(0,0,w_0)$ 是方程组的一个解
      - 再由解的唯一性，奇点上只有这一个解
      - 由原方程组易得坐标轴外无奇点
    - **闭轨线**：
      - 因为解函数满足首次积分方程，故 $E(C_1)，F(C_2)$ 的交线就是解函数
      - <font color='chartreuse'>动力系统法：</font>
    - **奇闭轨线**：含有奇点的闭轨线

## 首次积分法

- **判别法则**：$G_1$ 内连续可微的非常数函数 $\varPhi(x,y_1,...,y_n)$ 是首次积分的充要条件是
  - $\dis\pfrac{\varPhi}{x} + \pfrac{\varPhi}{y_1}f_1 + ... + \pfrac{\varPhi}{y_n}f_n = 0$ 
  - **证明**：
    - **必要性**：就是 $\varPhi$ 的全微分形式
    - **充分性**：就是全微分的重积分结果
  - **理解**：
- **降阶定理**：若已知首次积分，则可把微分公式降低一阶
  - **证明**：
- **独立的首次积分**：Jacobi行列式非0
- **首次积分法**
  - **证明**：
    - 由首次积分等式，得到隐函数 $y_n = g(x,y_1,...,y_{n-1},C)$
    - 将其代入前 $n-1$ 个式子，得到新的 $n-1$ 阶方程组
      - $\dis \frac{dy_i}{dx} = f_i(x,y_1,...,y_{n-1},g(x,...))$
    - 证其解 $\begin{cases} y_i = u_i(x) \\ \cdots \\ y_n = g(x,u_1(x),...,u_{n-1}(x),C) \end{cases}$ 为原方程组的解即可
      - 即证明满足最后一个等式即可
    - 代入 + 判别法则即可
