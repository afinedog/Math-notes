# 解对初值的依赖

- <font color='chartreuse'> $n$ 阶微分方程的初值问题的解可写为 $\cfrac{d\vec{y}}{dx} = \vec{f}(x,\vec{y},\vec{\l},...)$，其中 $\vec\l$ 是初值向量
  - 显然初值也是解函数的自变量，故可以研究初值对 $f$ 连续性的影响
  - 实际问题中，初值的测量总会存在误差。解对初值的连续依赖性说明若初值误差足够小，则解的误差也足够小
  - 同理，也可以研究解函数对ODE中参量的连续依赖性</font>
- **符号约定**：$\bs y$ 是各阶导数的向量，$\bs\l$ 是各个参数的向量
- **平移性定理**：
  - 对于一般的初值问题 $\begin{cases} \dfrac{d\bs y}{dx} = f(x,\bs y,\bs \l) \\\\ \bs y(x_0) = \bs y_0 \end{cases}$
  - 若作平移 $\begin{cases} t = x-x_0 \\\\ \bs u = \bs y-\bs y_0 \end{cases}$，则初值问题变为 $\begin{cases} \dfrac{d\bs y}{dx} = f(t+x_0,\bs u+\bs y_0,\bs \l) \\\\ \bs u(0) = \bs 0 \end{cases}$
  - 此时初值被融入参数中，也就是说我们只需要讨论初值为 $0$ 的情况下，解对参数的依赖性即可

## 解对初值的对称性

- **对称性定理**：
  - 设方程满足初值条件的解为 $y = \p(x;\ x_0,y_0)$
  - 则 $x,y$ 和 $x_0,y_0$ 可以调换位置，即 $y_0 = \p(x_0;\ x, y)$ 也是解函数
  - **理解**：
    - 本质就是把显格式的解函数换成了隐格式
    - 若曲线族的形状确定，则初值决定其位置
    - 反过来，若曲线族的形状和位置确定，则初值可被位置表出
  - **证明**：在解函数曲线上取任意一点 $(x_1,y_1)$，讨论即可

## 解对初值和参数的连续依赖性

- **解对初值和参数的连续性定理**：
  - 设含参问题 $E_\l:\begin{cases} \dfrac{d\bs y}{dx} = \bs f(x,\bs y,\bs \l) \\\\ \bs y(0) = \bs 0 \end{cases}$
  - 若

    - 对 $\bs y$ 满足Lipschitz条件
  - 则解 $\bs y = \p(x,\bs \l)$ 在 $D: \begin{cases} |x|\leq h \\ |\bs \l-\bs \l_0|\leq c \end{cases}$ 上连续（$h$ 的定义同皮卡定理）
  - **理解**：
    - 先证明皮卡序列对参数的连续性
    - 再证明皮卡序列一致收敛，从而存在唯一的、连续的极限函数
    - 从而解对参数是连续的
  - **本质**：将参数加入自变量对于皮卡定理的证明无影响，只是对存在区域加了一个约束
  - **证明**：

    - 将方程化为积分形式，并写出皮卡序列
      - 皮卡序列连续性：依然由微积分基本定理得出
        - <font color='chartreuse'>向量的元是独立的，对每个元分别讨论即可</font>
      - 皮卡不等式：用Lipschitz条件化 $\bs f$ 为 $\bs y$，再由积分上下界不等式+ 归纳假设即可
      - 由皮卡序列连续性 + 皮卡不等式即得皮卡序列一致收敛性，故存在连续的极限函数，从而存在连续的唯一解
- **无参初值问题的连续性定理**：
  - 若问题 $E$ 中不含参数，初值为 $y(x_0) = \eta$
  - 则 $E$ 的解在 $Q:\begin{cases} |x-x_0| \leq \frac{h}{2} \\ |\eta-y_0| \leq \frac{b}{2} \end{cases}$ 上连续（$h,b$ 的定义同皮卡定理）
  - <font color='chartreuse'>该结论可由上面定理直得</font>
- **拓扑变换**：点和线等的位置关系在变换下不变（数量和形状被忽略）
  - **实例**：同胚是拓扑变换

- **解的延伸定理**：
  - 若
    - 开区域 $G$ 内 $\bs f(x,\bs y)$ 连续
    - 对 $\bs y$ 满足局部Lipschitz条件
    - 解函数族中某个 $\bs y = \bs\xi(x)$ 的存在区间为 $J$
  - 则初值问题 $(E)$ 中， $\forall [a,b]\in J$，$\exist\d$，如果初值满足 $\begin{cases} a\leq x_0\leq b \\ |\bs y_0-\bs\xi(x_0)|\leq \d \end{cases}$
    - 则
      - 解在区间 $J$ 上存在
      - 解在管状闭区域 $D_\d: |\bs y_0-\bs\xi(x_0)| \leq \d$ 内连续
  - **证明**：
    - 该曲线可写为 $\Gamma = \{(x,\bs y)\mid \bs y = \bs\xi(x)，a\leq x \leq b\}$
    - 用满足局部Lipschitz条件的开圆覆盖 $\G$，得存在一个管状闭区域 $\Sigma_\sigma: \hkh{(x,\bs y)\mid |\bs y-\bs\xi(x)|\leq \sigma，a\leq x \leq b}$，使得其内有整体Lipschitz条件。再由有限覆盖定理，开圆中存在最大的 $L$
    - 构造皮卡序列 $$\bs\p_{k+1}(x;x_0,\bs y_0) = \bs y_0 + \int^x_{x_0} \bs f(x,\bs \p(x; x_0,\bs y_0))$$
      - 皮卡序列初值为 $\bs\p_0(x;x_0,\bs y_0) = \bs y_0 + \bs \xi(x) - \bs\xi(x_0)$
    - <font color='chartreuse'>接下来需要保证两点：
      - 皮卡序列不超出 $\Sigma_\sigma$，即证 $|\p_k - \xi| \leq \sigma$
      - 皮卡序列一致收敛，即证 $|\p_{k+1}-\p_k| \leq \cfrac{(L|x-x_0|)^{k+1}}{(k+1!)}|y_0-\xi_0|$</font>
    - 取正数 $\d = \dfrac{1}{2}e^{-L(b-a)}\sigma$，易得 $\d < \sigma$
      - <font color='chartreuse'>对 $k$ 使用数学归纳法证明不等式 </font>
      - $k=0$，易得不超出区域，再由 $\xi(x)$ 是解函数得满足积分方程，从而
      - $k = s$
        - 易得 $$|\bs\p_s(x) - \bs\xi(x)| = \abs{\sum\limits^s_{k=1}\Big[\bs\p_k(x) - \bs\p_{k-1}(x)\Big] + \Big[\bs\p_0(x) - \bs\xi(x)\Big] }$$
          - <font color='chartreuse'>（转化为证明差级数的收敛性）</font>
        - 由归纳假设，右式 $\dis\leq \sum^s_{k=0}\frac{(L|x-x_0|^k)}{k!}|\bs y_0-\bs\xi(x_0)|$
          - <font color='chartreuse'>一致收敛性得证</font>
        - 由 $e$ 的泰勒展开式，上面右式 $ \leq e^{L|x-x_0|}\d \leq \sigma$
          - <font color='chartreuse'>不超出管状区域得证</font>
  - **理解**：
    - 只有在管道 $\Sigma_\sigma$ 内才满足皮卡定理，从而解连续且唯一
    - 在管道外的 $J$ 上只满足Peano定理，解存在，但不一定连续
    - 在 $J$ 以外，解有可能无界，即存在区间断裂，从而不一定存在

### 用函数连续的定义证明连续依赖性（王高雄）

- **稳定性不等式**：$|\p(x) - \psi(x)| \leq |\p(x_0) - \psi(x_0)|e^{L|x-x_0|}（x\geq x_0）$
  - <font color='chartreuse'>它实际上就是解的稳定性</font>
  - **证明**：
    - 令 $V(x) = \big[ \p(x) - \psi(x) \big]^2$
    - $V'(x) \leq 2LV(x)$，即 $V(x)e^{-2Lx}$ 单调递减
    - 从而 $V(x)e^{-2Lx} \leq V(x_0)e^{-2Lx_0}$，移项开平方得即得初值不等式
  - **理解**：L条件 + e的导数结构得到单调递减性，代入初值即得不等式
- **解对初值的连续依赖定理**：（函数连续的定义） $\forall \sqrt{(\overline{x_0} - x_0)^2 + (\overline{y_0}-y_0)^2} < \d$，都有 $|\p(x;\overline{x_0},\overline{y_0}) - \p(x; x_0,y_0)| < \e$
- 即解函数 $\p$ 对于初值变量 $x_0,y_0$ 均连续
  - <img src="./图片/解对初值的连续依赖性.png">
- **证明**
  - 首先有限覆盖定理，得到以 $\p(x)$ 为中心线的管状闭区域（宽度半径为 $\eta = \min\{\e,\frac{\rho}{2}\}$）和整体L常数
  - 设两个初值 $x_0，\overline{x}_0$ 下的解函数分别为 $\p(x)$ 和 $\psi(x)$
    - 假设 $\psi(x)$ 无定义，则 $\p(x) = \psi(x)$。因此只需讨论有定义的情况
    - 再由 $\psi(x)$ 的在管状区域内的连续性得，$\psi(x)$ 的解必定延伸到管状区域的边界
  - 设 $\d_1 = \frac{1}{2}\eta e^{-L(b-a)}$，$\d_2$ 为 $\d_1$ 内 $\p$ 的差值
    - *距离约束*：令设两个初值点的距离为 $\min\{\d_1,\d_2\}$
  - 再由（整体Lipschitz条件下的初值不等式）+ 积分曲线连续性
    - 放缩可得，当距离约束成立时，$|\psi(x) - \p(x)|^2 \leq 4\d_1^2e^{2L(b-a)} = \eta^2$，**初值连续性得证**
      - 初值不等式 + 添项三角不等式，再使用距离约束，两项均放缩为 $\d_1$
      - 指数使用距离约束，放缩为 $b-a$
  - 由于 $\eta$ 是管状区域的半径，所以由上式，任意 $\psi(x)$ 均不会触碰到管道壁。再由 $\psi$ 的解延伸到边界，所以 $\psi(x)$ 必定碰到管道的底，即存在区间不小于 $[a,b]$
    - 即管道内的初值均具有连续性（**初值问题解的存在性得证**）
- **本质**：初值不等式 + $x_0,\overline{x}_0$ 的距离约束，可由定义放缩得到连续性
- **解的连续性定理**：三元函数 $y = \p(x;x_0,y_0)$ 是连续的

## 解对初值和参数的连续可微性

<!-- - **简化**：由解在初值上的连续性，只需考虑 $E:\begin{cases} \dfrac{dy}{dx} = f(x,y,\l) \\ y(0) = 0 \end{cases}$ 的解对参数 $\l$ 的连续可微性 -->
- **解对初值和参数的连续可微定理**：
  - 设含参问题 $E_\l:\begin{cases} \dfrac{d\bs y}{dx} = f(x,\bs y,\bs \l) \\ \bs y(0) = 0 \end{cases}$
  - 若
    - $\bs f(x,\bs y,\bs \l)$ 在区域 $G: \begin{cases} |x|\leq a \\ |\bs y|\leq b \\ |\bs \l - \bs\l_0| \leq c \end{cases}$ 上连续
    - $\bs f$ 对 $\bs y$ 和 $\bs \l$ 有连续的偏微商
  - 则
    - $E_\l$ 的解 $\bs y = \bs\p(x,\bs\l)$ 在区域 $D : \begin{cases} |x| \leq h \\ |\bs\l - \bs\l_0| \leq c \end{cases}$ 上连续可微
- **证明**：
  - 取 $\begin{cases} \p_0(x,\l) = 0 \\ (x,\l) \in D \end{cases}$，作皮卡序列 $$\p_{k+1}(x,\l) = \int^x_0 f(x,\p_k, \l)dx$$
  - 已知皮卡序列收敛到方程的解 $y = \p(x,\l)$
  - **皮卡序列在 $D$ 内连续可微**
    - 归纳法易得
  - **皮卡序列关于 $\l$ 的偏导**：$\dis\frac{\pa \p_{k+1}}{\pa \l} = \int^x_0\fkh{\frac{\pa f}{\pa y}(x,\p_k,\l)\frac{\pa\p_k}{\pa\l} + \frac{\pa f}{\pa \l}(x,\p_k,\l)}dx$
    - 链式法则易得
    - <font color='chartreuse'>由题设条件得 $f_y,f_\l$ 均连续</font>
  - **只需证明序列 $\l_k = \hkh{\cpfrac{\p_k}{\l}(x,\l)}$ 一致收敛即可**
    - $|\l_1|$ 有上界 $\a h$
      - 由偏导数 $f_\l$ 连续性，设其有上界 $\a$
      - 从而 $\dis\abs{\frac{\pa \p_1}{\pa \l}} \leq \abs{\int^x_0\abs{\frac{\pa f}{\pa \l}(x,0,\l)}dx} \leq\a|x|$
    - $|\l_k|$ 有上界 $e^{\a h}$ 
      - 归纳法可得 $\dis\abs{\frac{\pa \p_k}{\pa \l}} \leq \sum^k_{n=1} \frac{(\a|x|)^n}{n!} \leq e^{\a|x|}$
      - <font color='chartreuse'>本质是皮卡序列的多项式迭代积分</font>
    - **作差序列**：设 $v_{k,s} = |\l_{k+s}-\l_k| = |\dfrac{\pa \p_{k+s}}{\pa \l} - \dfrac{\pa \p_k}{\pa \l}|$
    <br>
    - **递推放缩公式**：$\dis v_{k+1,s} \leq \abs{\int^s_0 \abs{\frac{\pa f}{\pa y}(x,\p,\l)}v_{k,s}dx} + d_{k,s}(x,\l)$
      - 其中 $d_{k,s}$ 是三个微分差值的积分
    - **公式放缩**：
      - 由皮卡序列收敛性 + 偏微商连续性，$f_{y或\l}(\p)-f_{y或\l}(\p_k) < \e$，从而 $d_{k,s}(x,\l) < \e_k \to 0$
      - 再由 $f_y$ 连续性，其有上界 $\a$
      - 最终可简化为：$\dis v_{k+1,s} \leq \a\abs{\int^s_0 v_{k,s}dx} + \e_k$
    - **再次放缩**：
      - 由 $\l_k$ 有上界 $\beta$，得 $v_{k,s} \leq 2\beta$
      - **简化结果**：$v_{k+1,s} \leq \a\beta|x| + \e_k$
        - <font color='chartreuse'>递推公式简化为多项式迭代积分</font>
    - **递推得**：$$v_{k+m,s} \leq 2\beta\frac{(\a|x|)^m}{m!} + \sum\limits^{m-1}_{j=0} \e_{k+(m-1)-j}\frac{(\a|x|)^j}{j!} \leq 2\beta\frac{(\a|x|)^m}{m!} + e^{\a h}\d_k$$
      - 其中 $\d_k = \sup\{\e_n\mid n\geq k\} \to 0$
      - 易得前后项均趋于0，即 $v_{k+m,s}\to 0\quad (m,k\to\infty，\forall$ 有界量 $s$)
      - 由柯西收敛原理，$\l_k$ 一致收敛
  - **连续可微性**：
    - 已知解一定对 $x$ 连续可微
    - 由皮卡序列对 $\l$ 连续可微，且一致收敛，得极限函数对 $\l$ 也连续可微
    - 从而解在 $D$ 上连续可微
- **理解**：
  - 偏微商连续性的作用：
    - 使皮卡序列连续可微
  - 偏微商有界性的作用：
    - 简化递推放缩公式
    - 使 $\l_k$ 有上界，从而可化为多项式迭代积分，得到 $e^x$，其有界
    - 消去余项 $d_{k,s}$

### 初值可微性（王高雄）

- **解对初值的可微性定理**：若 $f(x,y)$ 和 $\frac{\pa f}{\pa y}$ 都在 $G$ 内连续，则方程的解 $\p(x,x_0,y_0)$ 在存在区间上连续可微
- **证明**：已知解对变量均具有连续性，只需证明偏导数存在且连续
  - 只需证明 $\large\frac{\pa y}{\pa x_0}$ 存在且连续，$y_0$ 方法类似
  - 设邻近初值决定的函数为 $\p(x,x_0,y_0)$ 和 $\psi = \p(x,x_0+\triangle x_0, y_0)$
  - 构造皮卡序列形式的微分：
    - $\p - \psi = \int^x_{x_0} \big[ f(x,\p)-f(x,\psi) \big] dx - \int^{x_0+\triangle x_0}_{x_0} f(x,\psi)dx$
  <br>
      - 前项对y使用L中值定理，得到 $\displaystyle\int \cfrac{\pa f(x,\p+\theta(\psi-\p))}{\pa y}(\psi - \p)dx$
  - 再由 $f$ 对 $x,y$ 的连续性，前后项转化得下述方程：
  <br>
  - $\displaystyle\frac{\psi - \p}{\triangle x_0} = \int(\frac{\pa f(x,\p)}{\pa y} + r_1)\frac{\psi - \p}{\triangle x_0}dx - f(x_0,y_0) + r_2$
  <br>
    - 第一项由 $\frac{\pa f}{\pa y}$ 的连续性得出（是函数，若对x求导，其仍保留）
    - 第二项由原函数微分得出（是常数，若对x求导，其消去）
  - 设 $z = \cfrac{\psi-\p}{\triangle x_0}$，此时该等式是一阶线性微分方程初值问题的解函数形式
    - $\begin{cases} \displaystyle\frac{dz}{dx} = \big[ \frac{\pa f(x,\p)}{\pa y} + r_1 \big] z \\ z(x_0) = -f(x_0,y_0) + r_2 \end{cases}$
    - 解为 $\cfrac{\psi - \p}{\triangle x_0}$
  - 再令 $\triangle x_0\to 0$
    - 此时 $r_1,r_2\to 0$
    - 由解对参数的连续性，得 $z$ 对 $\triangle x_0$ 连续
      - 此时极限和微分可交换：$\lim\limits_{\triangle x_0\to 0} \frac{\psi - \p}{\triangle x_0} = \frac{\pa\p}{\pa x_0}$
  - 解上述初值问题，得 $\dfrac{\pa\p}{\pa x_0} = -f(x_0,y_0) e^{\large\int^x_{x_0}\frac{\pa f(x,\p)}{\pa y}dx}$
  - 易知其为 $x,x_0,y_0$ 的连续函数（**证毕**）
- **理解**：将（皮卡序列的偏导数）转化为（一阶线性微分方程的初值问题），其解函数存在且连续，从而得到偏导数存在且连续
- **本质**：变分方程

## 变分方程

- **变分方程**：
  - 考虑方程 $\dfrac{dy}{dx} = f(x,y,\l)$，设 $\begin{cases} A(x,x_0,y_0,\l) = \dpfrac{f}{y}(x,\p,\l) \\\\ B(x,x_0,y_0,\l) = \dpfrac{f}{\l}(x,\p,\l) \end{cases}$
    - <font color='chartreuse'>通过对积分方程求关于 $x_0,y_0,\l$ 的两次偏导数，就变为下面的变分方程
    - 微分的意思是变量作微小变化，而变分的意思是函数做微小变化。</font>
  - 取 $z = \dpfrac{\p}{x_0}$，则称 $\dfrac{dz}{dx} = A(x,x_0,y_0,\l)z$ 是原方程关于 $x_0$ 的变分方程
    - <font color='chartreuse'>当 $x_0$ 微小变化时，解函数的变化情况</font>
  - 取 $z = \dpfrac{\p}{y_0}$，则称 $\dfrac{dz}{dx} = A(x,x_0,y_0,\l)z$ 是原方程关于 $y_0$ 的变分方程
    - <font color='chartreuse'>当 $y_0$ 微小变化时，解函数的变化情况</font>
  - 取 $z = \dpfrac{\p}{x_0}$，则称 $\dfrac{dz}{dx} = A(x,x_0,y_0,\l)z + B(x,x_0,y_0,\l)$ 是原方程关于 $\l$ 的变分方程
    - <font color='chartreuse'>当 $y_0$ 微小变化时，解函数的变化情况</font>


### 应用

- **应用**：求解函数对参数的偏导数时，只需求出变分方程的解
- **一阶线性方程**：$\begin{cases} \dfrac{dy}{dx} = -p(x)y + q(x) \\\\ y(x_0) = y_0 \end{cases}$
  - **解**：
