# 常微分方程7：幂级数解法

- 幂级数和导数亲和，从而和微分方程亲和

## 柯西定理

- **柯西定理**：
  - 若 $f(x,y)$
    - 在区域 $G\subset R^2$ 内解析
    - 在 $R(a,b) = \{(x,y)\mid |x-x_0|\leq a，|y-y_0|\leq b\}$ 内可以展成收敛幂级数 $$f(x,y) = \sum\limits^\infty_{i,j=0}a_{ij}(x-x_0)^i(y-y_0)^j$$
  - 则柯西问题 $(E): \frac{dy}{dx} = f(x,y)，y=y_0$ 满足以下性质
    - 满足皮卡定理条件，解存在且唯一
    - 解在 $x_0$ 附近解析，可以展成幂级数
      - $y = \sum\limits^\infty_{n=0}C_n(x-x_0)^n$
- **二元幂级数的优级数**：系数 $|a_{ij}| \leq A_{ij}$ 
  - **优函数**：优级数的极限函数（必须收敛）
- **优函数上界性**：
  - 若 $f(x,y)$ 在矩形区域 $R(\alpha,\beta)$ 上可以展成收敛幂级数
  - 则 $\exist M$，使得 $F(x,y) = \cfrac{M}{(1-\frac{x-x_0}{a})(1-\frac{y-y_0}{b})}$ 在矩形区域 $R_1(a,b) \subset R_2(\alpha,\beta)$ 上是优函数
  - **证明**：
    - 由幂级数绝对收敛性，$f(x,y)$ 的通式取绝对值 $\sum\limits^\infty_{i,j=0} |a_{ij}|a^ib^j$ 也收敛
    - 由收敛函数通项有界性，可设 $\forall a,b \in R(\alpha,\beta)，|a_{ij}|a_ib_j \leqslant M$
    - 则令 $F(x,y) = \sum\limits^\infty_{i,j=0}\frac{M}{a^ib^j}(x-x_0)^i(y-y_0)^j$
  - **本质**：幂级数收敛则绝对收敛 + 收敛域缩小换来系数增大 = 子收敛域上存在优函数
- **优函数解的解析性**：
  - $F(x,y)$ 的初值问题在区间 $(x_0-\rho,x_0+\rho)$ 内存在一个解析解
  - 其中 $\rho = a(1-e^{-\large\frac{b}{2aM}})$
  - **证明**：
    - 由 $F(x,y)$ 定义，$\frac{dy}{dx} = F(x,y)$ 是一个变量分离方程，可以直接解出
      - $y = y_0 + b-b\sqrt{1+\frac{2aM}{b}ln(1-\frac{x-x_0}{a})}$
    - 然后令 $|x-x_0| < \rho$
      - 则 $ln(1-\frac{x-x_0}{a})$ 可以展成收敛幂级数
      - 则 $\sqrt{1+s}$ 中 $s<1$，也可以展成收敛幂级数
    - 从而该解解析
- **形式解**：$y = y_0 + \sum\limits^\infty_{n=1}C_n(x-x_0)^n$
  - **展开性**：其为解函数的幂级数展开
  - **解析性**：如果幂级数是收敛的，那么形式解就是解析解
- **柯西定理**：如果函数 $f(x,y)$ 在矩形 $R_1(\alpha,\beta)$ 上可以展开成 $x，y$ 的收敛幂级数，则其初值问题在 $O(x,\rho)$ 内存在唯一的解析解
  - **证明**：
    - **存在性**：由幂级数系数（Taylor系数）的导数性，可以直接用 $f(x,y)$ 得出形式解
    - **收敛性**：
      - 由数分知识，Taylor系数为：
        - $C_1 = f(x_0,y_0) = a_{00}$
        - $C_2 = \frac{1}{2!}y''(x_0) = \large\frac{f_x + f_yy'}{2!} = \frac{a_{10}+a_{01}a_{00}}{2!}$
        - $C_n = \frac{y^{(n)}(x_0)}{n!} = P_n(a_{00},a_{01},a_{10},...,a_{n-1,0})$
      - 作优级数 $F$，展开成幂级数
        - 由优函数收敛性 + 系数 $\widehat{C}_n \geqslant |C_n|$，比较定理得形式解收敛
    - **唯一性**：Taylor展开的唯一性直得
  - **反例**：
    - 非解析方程没有形式解：$f(x,y) = \frac{y-x}{x}$
    - 非解析方程的形式解不收敛：$f(x,y) = \frac{y-x}{x^2}$
  - **推论：柯西定理的高维（方程组）形式**：因为 $C_n$ 是简单加法和乘法，且每一项都有确定的大小关系，所以可以简单应用到高维，用一个最大优级数 $G(x,y_1,...,y_n)$，即可以充当所有函数的优级数。

## 幂级数解法

### 二阶齐次线性微分方程

- 二阶齐次线性微分方程：$A(x)y'' + B(x)y'' + C(x)y = 0$
  - 其中系数函数在 $O(x_0,r)$ 内解析
  - **常点**：$A(x_0) \neq 0$，方程不退化，且可以转化为首一方程
  - **奇点**：$A(x_0) = 0$，首一方程的系数不连续，邻域内可能没有解析解
- **二阶齐次线性微分方程的幂级数解法**：
  - 若二阶齐次线性首一微分方程的系数函数 $A(x),B(x),...$ 在区间 $O(x_0,r)$ 内均可展成收敛幂级数，则在 $O(x_0,r)$ 内有收敛的幂级数解
  - **证明**：
    - 将高阶线性微分方程化为方程组标准形式，然后应用柯西定理即可
    - 系数 $C_n$ 可由初值 $C_0, C_1$ 递推求取
- **Airy方程**：$y'' = xy（x\in R）$
  - **递推解法**：
    - 首先写出 $y$ 在原点的幂级数展开形式
    - 对y逐项求二阶导，并代回方程，得到递推公式，从而得到各项系数
    - D'Alembert判别，证明收敛
    - 如果要求在某点展开的解，只需要找到相应点上的展开式即可
- **Legendre方程**：$(1-x^2)y'' - 2xy' + n(n+1)y = 0$
  - **递推解法**：
    - 易得奇点为 $x=\pm 1$，而 $x=0$ 是常点，从而 $|x|<1$ 上有幂级数解
    - 解法同上
- <font color='chartreuse'>和组合数学的递推关系很像，看完组合数学后一点也不觉得这里复杂了</font>

### Legendre多项式

- **Legendre方程的幂级数解**：$\displaystyle P_n = \frac{1}{2^n}\sum\limits^{[\frac{n}{2}]}_{k=0} \frac{(-1)^k(2n-2k)!}{k!(n-k)!(n-2k)!}x^{n-2k}$
  - 满足Ro公式
  - 具有正交性
- **Rodrigues公式**：$\displaystyle P_n(x) = \frac{1}{2^nn!}\frac{d^n}{dx^n} (x^2-1)^n$
  - **推论**：$P_n(1) = 1，P_n(-1) = (-1)^n$
- **广义Fourier级数**：$f(x) \sim \sum\limits^\infty_{n=0} a_nP_n(x)$
  - **系数**：$a_n = \frac{2n+1}{2}\int^1_{-1} f(x)P_n(x)dx$
  - **收敛性**：
    - 若 $\cfrac{f(x)}{\sqrt[4]{1-x^2}}$ 在 $[-1,1]$ 上绝对可积，且满足下列条件之一：
      - Dirichlet条件
      - Dini条件
      - Holder条件
    - 则广义Fourier级数在 $x = \xi$ 收敛到 $\cfrac{f(\xi+0) + f(\xi-0)}{2}$

## 广义幂级数解法

- **奇点处的解**：邻域内无幂级数，且初值问题可能无解
  - **实例**：
    - 欧拉方程 $x^2y'' - 2y = 0$，在奇点 $x=0$ 附近
      - 通解为 $y = C_1x^2 + \frac{C_2}{x}\quad (x\neq 0)$
    - $x^2y'' + (3x-1)y + y'= 0$，在奇点 $x=0$ 附近
      - 递推公式得形式解为 $y = C_0\sum\limits^\infty_{n=1} n!x^n$，发散
    - $x^2y'' + xy' + (x^2-\frac{1}{4})y = 0$，在奇点 $x=0$ 附近
      - 设 $y = \frac{u}{\sqrt{x}}$，则 $u'' + u = 0$ 的解为 $$\begin{cases} \displaystyle y_1 = \frac{cosx}{\sqrt{x}} = \sum\limits^\infty_{k=0} \frac{(-1)^k}{(2k)!}x^{2k-\frac{1}{2}} \\ \displaystyle y_2 = \frac{sinx}{\sqrt{x}} = \sum\limits^\infty_{k=0} \frac{(-1)^k}{(2k+1)!} x^{2k+\frac{1}{2}} \end{cases}$$
- **广义幂级数**：$\sum\limits^\infty_{n=0} C_n(x-x_0)^{n+\rho}\quad (C_0 \neq 0)$
  - **指标**：$\rho$
- **正则奇点**：构造**正则形式** $(x-x_0)^2P(x)y'' + (x-x_0)Q(x)y' + R(x)y = 0$ 后
  - 若
    - 系数函数均可在 $x_0$ 附近展成幂级数
    - $P(x)\neq 0$，$Q(x),R(x)$ 不同时为0
  - 则 $x_0$ 称为原方程的正则奇点
- **正则展开定理**：二阶齐次线性微分方程，在正则奇点的邻域内有收敛的广义幂级数解
  - **证明**：首先在邻域 $O(x_0,r)$ 内将方程写成正则形式，然后将系数函数展开为幂级数
    - <font color='chartreuse'>设系数分别是 $a_n$ 和 $b_n$</font>
    - **递推公式**：
      - 将广义幂级数解代入，由幂级数的唯一性，可得系数的递推公式
        - $\begin{cases} C_0f_0(\rho) = 0 \\ \cdots \\ \sum\limits^k_{j=1} C_jf_j(\rho + j) = 0 \\ \cdots \end{cases}$，其中 $\begin{cases} f_0(\rho) \equiv \rho(\rho - 1)+a_0\rho + b_0 \\ f_j(\rho) \equiv a_j\rho + b_j \end{cases}$
      - **指标方程**：即第一个式子，$\rho^2 + (a_0-1)\rho + b_0 = 0$
      - **指标根**：设为 $\rho_1,\rho_2$
        - 选取实部为正的指标根 $\rho_1$，其和 $C_0$ 可共同表出 $C_k$
    - **收敛性**：证明广义幂级数解 $y = \sum\limits^\infty_{k=1} C_k(x-x_0)^{k+\rho_1}$ 在正则奇点附近收敛
      - 设 $m = \rho_1 - \rho_2$，则 $Re(m) \geqslant 0$
      - 由系数函数幂级数收敛，若 $r_1<r$，则 $\exist M$，使得 $|f_j(\rho_1)| = |\rho_1a_k + b_k| \leqslant \frac{M}{r_1^k}$
      - 再由指标根的韦达定理，配凑可得 $f_0(\rho_1 + k) = k(k+m)$
    - **归纳法证明不等式**：$|C_k| \leqslant \big( \frac{M}{r_1} \big)^k$
      - 由递推公式 + 韦达配凑式，$|C_1| = \cfrac{|\rho_1a_1+b_1|}{|f_0(\rho_1+1)|} \leqslant \frac{M}{r_1|m+1|}$，成立
      - $n=k$ 时，对 $|C_k|$ 应用三角不等式，配凑成韦达配凑式，得 $\leqslant \cfrac{\sum\limits^{s-1}_{j=0} |C_j|(|\rho_1a_{s-j} + b_{s-j}| + |ja_{s-j}|)}{s^2}$
        - 将上式分子放缩为 $\sum\limits^{s-1}_{j=0} |C_j|\cdot j\cdot |\rho_1a_{s-j} + b_{s-j}|$
        - 应用归纳假设得 $\leqslant \cfrac{\frac{s(s+1)}{2}(\frac{M}{r_1})^s}{s^2} < (\frac{M}{r_1})^s$
    - 综上，当 $r_2<r_1<r$ 时，广义幂级数在 $O(x,\frac{r_2}{M})$ 收敛
  - **推论**：指标和系数可由代入法确定
    - 当 $\rho_1$ 是复数时，广义幂级数解是复级数。但由于变化的起始点都是实数，所以虚部归零，得实数解
    - 若 $m$ 是正整数，则从 $\rho_2$ 出发计算时会出现 $C_m$ 归零的情况，递推关系断裂。
    - 不过可以用刘维尔公式，从 $\rho_1$ 的广义幂级数解出发，得到新的线性无关解

### 实例

- **贝塞尔方程**：$x^2y'' + xy' + (x^2-n^2)y = 0$，其中 $n\geqslant 0$
  - **解**：
    - 正则奇点为 $x=0$
    - 代入广义幂级数解，得递推公式为 $(\rho + n + k)(\rho-n+k)C_k + C_{k-2} = 0$，其中 $C_{-1} = C_{-2} = 0$
    - 指标根为 $\rho_1 = n，\rho_2 = -n$
  - **$\rho_1$ 的解**
    - 公式为 $C_k = \begin{cases} 0，\qquad k为奇数 \\ \cfrac{(-1)^k(n+k)!}{2^{2k}n!k!}，k为偶数 \end{cases}$
    - 收敛域为 $R$
  - **第一类贝塞尔函数**：$\displaystyle J_n(x) = \sum\limits^\infty_{k=0} \frac{(-1)^k}{\Gamma(n+k+1)\Gamma(k+1)}(\frac{x}{2})^{2k+n}$
    - <font color='chartreuse'>此时将 $G_{2k}$ 写成Gamma函数</font>
    - 指标根 $\rho_1$ 对应的解
  - **$\rho_2$ 的解**
    - 公式为 $k(k-2n)C_k + C_{k-2} = 0$
    - 若 $2n$ 不为整数，则可得正常解
  - **第二类贝塞尔函数**：$\displaystyle J_{-n}(x) = \sum\limits^\infty_{k=0} \frac{(-1)^k}{\Gamma(-n+k+1)\Gamma(k+1)}(\frac{x}{2})^{2k-n} $
    - <font color='chartreuse'>nnd，不就是换了个-n吗</font>
    - $x\to 0$ 时其无界
  - **补充**：若 $2n$ 为整数 $N$，则 $C_N$ 无法递推求解
    - $N = 2s+1$
      - k为偶数时 $C_k$ 不变
      - k为奇数时
        - $N$ 前面 $C_k$ 依然为0
        - $N$ 后面 $C_k$ 依赖于 $C_N$，只要设 $C_N = 0$，后面就依然为0，依然得到第二类贝塞尔函数
    - $N = 2s$
      - 由于N前公式和N的公式矛盾，不可能从递推公式得出解
      - 取 $\alpha\to n$，则解为诺伊曼函数
  - **诺伊曼函数**：$\displaystyle Y_n(x) = \lim\limits_{\alpha\to n} \frac{J_\alpha(x)cos\alpha\pi - J_{-\alpha}(x)}{sin\alpha\pi}$

### 贝塞尔函数

- **渐进式**：$x\to\infty$ 时
  - $\displaystyle J_n(x) = \frac{A_n}{\sqrt{x}}\big[ sin(x+\alpha_n) + \omicron(1) \big]$
  - $\displaystyle Y_n(x) = \frac{B_n}{\sqrt{x}}\big[ cos(x+\alpha_n) + \omicron(1) \big]$
  - **证明**：
- **正交系**：$J_n(\beta_k t)$ 在 $[0,1]$ 上是以 $t$ 为权函数的正交系
  - **权函数**：$\displaystyle \int^1_0 t\cdot J_n(\beta_j t)J_n(\beta_k t)dt = \begin{cases} 0, \qquad\quad j\neq k \\ \tau_{n,k}>0，j=k \end{cases}$
  - $\tau_{n,k} = \frac{1}{2}\big[ J_n(\beta_k) \big]^2$
- **广义Fourier定理**：任意正交函数系均可构造相应的广义Fourier展开
