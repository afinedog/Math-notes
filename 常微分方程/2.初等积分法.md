# 常微分方程2：初等积分法

- **一阶ODE的对称形式**：$P(x,y)dx + Q(x,y)dy = 0$
- **一阶ODE的非对称形式**：$\cfrac{dy}{dx} = - \cfrac{P(x,y)}{Q(x,y)}$

## 恰当方程（全微分方程）

- **恰当方程**：如果存在一个可微函数 $\Phi(x,y)$，使得 $d\Phi = Pdx + Qdy$，则称为恰当方程
  - **通解定理**：设 $E$ 是一阶ODE，则 $E$ 存在通解 $\LR E$ 可变形为恰当方程  
    - **实例**：
      - 若将恰当方程约去一些变量，则可用积分因子法求解
  - **通积分（隐函数形式的解）**：$\Phi(x,y) = C$ 称为恰当方程的通积分

### 恰当方程的判断和求解

- **判定定理**：
  - 设函数 $P(x,y)$ 和 $Q(x,y)$ 在区域 $R: \begin{cases} \alpha < x < \beta \\  \gamma < y < \delta \end{cases}$ 上连续
  - 若有连续的一阶偏导数 $\dfrac{\partial P}{\partial y}和\dfrac{\partial Q}{\partial x}$
  - 则 $Pdx + Qdy = 0$ 是恰当方程 $\LR \dfrac{\partial}{\partial y}P(x,y) \equiv \dfrac{\partial}{\partial x}Q(x,y)$
- **证明**：
  - **必要性**：由数分知识，偏导数连续 $\red\Rightarrow$ 混合偏导数相等
    - **理解**：微分中值定理 + 多元函数连续的自变量独立性
  - **充分性**：若是恰当方程，则通积分可以写为以下两种形式
    - $\dis\Phi(x,y) = \int^x_{x_0}P(x,y)dx + \int^y_{y_0}Q(x_0,y)dy$
    - $\dis\Phi(x,y) = \int^x_{x_0}P(x_0,y)dx + \int^y_{y_0}Q(x,y)dy$
    - <font color='chartreuse'>具体过程如下（待定函数积分法）：
      - 由于 $P$ 是 $\Phi$ 的 $x$ 偏导数，故 $\Phi = \dis\int^x_{x_0} Pdx + \psi(y)$
      - 对其关于 $y$ 求微分，再由题设相等条件，转化为 $\dpfrac{\Phi}{y} = Q|^x_{x_0} + \psi'(y)$
      - 再由于 $Q$ 是 $\Phi$ 的 $y$ 偏导数，故只能是 $\psi'(y) = Q(x_0,y)$
      - 综上即得第一个形式，同理可得第二个形式</font>
    - 对两种形式分别求全微分，两式结果即为两个偏导数，从而它们相等
  - **理解（自由度视角）**：
    - 选取合适的 $\Phi$ 时，自由度并不是老老实实地一部分归于 $P$，另一部分归于 $Q$，而是（$P$ 占据了所有自由度，而 $Q$ 只是用来消去 $P$ 的残值），或（$Q$ 占据了所有自由度，而 $P$ 只是用来消去Q的残值）
    <!-- - 其实，上面这两种形式才是真正的恰当方程。一开始设的 $P(x,y)dx + Q(x,y)dy = 0$ ，只是因为未知才作的保全之策。 -->
  - **理解（几何意义）**：满足恰当性，即满足格林定理的条件，从而任意曲线积分与路径无关
    - 也就是说，任意分段光滑曲线 $L$ 都有 $\dis\int_L Pdx+Qdy = \int_{L_x}Pdx + \int_{L_y}Qdy$，其中 $L_x,L_y$ 分别是平行于 $x,y$ 轴的直线
    - 取 $L$ 的两个端点分别为 $(x_0,y_0)，(x,y)$，即将原方程的（不定积分）写成（上限函数定积分）的形式，再应用格林定理，即可得到上面两种形式

### 习题

- **求解恰当方程**：
  - **直接法**：直接写出恰当方程的通解
    - 取 $(x_0,y_0) = (0,0)$ 最简单
  - **分组凑微分法**：通过观察，把形式相似的 $x$ 和 $y$ 的单项式凑到一起，有时候可以拼出乘积的微分
     - **实例**：$2x\sin ydx + x^2\cos ydy = d(x^2\sin y)$
  - 单连通区域上的单值函数（未完）

## 变量分离方程

- **变量分离法**：若一阶ODE具有形式 $P_1(x)P_2(y)dx + Q_1(x)Q_2(y)dy = 0$
  - 则其可变形为 $\dfrac{Q_2(y)}{P_2(y)}dy = -\dfrac{P_1(x)}{Q_1(x)}dx$
  - 两边分别对一元函数求不定积分，再变形即得解
    - <font color='chartreuse'>为什么两边自变量不同，也可同时求积分？因为这里的 $y$ 实际上是 $x$ 的函数，故 $\int g(y)dy = \int g(y(x))y'(x)dx$，对 $y$ 积分就是 $x$ 的不定积分换元而已。所以实质上我们是在两边同时对两个 $x$ 的函数做不定积分</font>
- **特解**：若存在某点有 $X_1(a) = 0$，则其为原方程的解，但不能通过分离变量法得出
  - **实例**：$(x^2+1)(y^2-1)dx + xydy = 0$
    - 在 $x=0$ 时有解
    - 在 $y=\pm 1$ 时有解
    - 在 $y=0$ 时无解
  - **微分理解法**：
    - 因为在变量分离过程中，两端都除了一个函数。故只需考虑该函数恒为 $0$，即 $P \equiv 0$ 或 $Q\equiv 0$ 的情况
    - 解函数至少是一条曲线，不能是一个点
      - 故令 $x=0$ 时，相当于默认方程的解是函数 $x=0$。只需验证它是否符合等式即可
      - 易得 $y=0$ 时，等式两端变为 $-(x^2+1)dx = 0$，在 $y=0$ 上不成立，故不是解
      - 易得 $y=1$ 时，等式两端变为 $0 = 0$，故是解
  - **线素连续性理解法**：
    - ODE的解在单点上表现为线素，在区域上表现为线素场。首尾相接的连续线素共同组成全体积分曲线
      - ODE有解等价于线素场形成一个光滑曲线，即线素的斜率和位置必须连续变动
      <!-- - 沿积分曲线移动是固定 $C$，变动 $x$
      - 沿 $x=a$ 或 $y=b$ 移动是变动 $C$，固定 $x$ 或 $y$ -->
    - 取 $y=0$ 时，等式变为 $(x^2+1)dx = 0$，满足该等式的线素必须 $k = +\infty$
      - 综上全体线素为 $y=0$ 且 $k=+\infty$，其不连续，故 $y=0$ 不是ODE的解

### 解的局部唯一性（例子）

- **局部不唯一点**：若多个积分曲线 $\Phi_1,...,\Phi_k$ 在某点 $A$ 上交叉，则称其为局部不唯一点
  - <font color='chartreuse'>由于ODE只有一个，故这些积分曲线在 $A$ 点的斜率都相同（除了特殊曲线 $x=x_0$），造成解不同的原因是 $A$ 以外存在某点斜率不同</font>
- **孤立的奇异点**：$\cfrac{dy}{dx} = \cfrac{(x^2+1)(y^2-1)}{-xy}$
  - 通解为 $y^2 = 1+C\dfrac{e^{-x}}{x^2}$
  - 特解为 $x=0$ 和 $y=\pm 1$
  - **无奇异点**：解在 $\R^2$ 上均存在
  - **局部不唯一点**：在 $(0,1)$ 上，有两个特解 $x=0$ 和 $y = 1$
- **连续的奇异点**：$\cfrac{dy}{dx} = \cfrac{3\sqrt[3]{y}}{2}$
  - 通解为 $x = \sqrt[3]{y^2} + C$
  - 特解为 $y=0$
  - **局部不唯一点**：在 $x$ 轴上每一点都有无限个积分曲线重叠，解局部不唯一
- **局部唯一条件**
  - 已知微分方程 $\dfrac{dy}{dx} = f(y)$
  - 若
    - $f(y)$ 在直线 $y=a$ 的某闭邻域 $[a-\d,a+\d]$ 内连续
    - $f(y) = 0 \LR y=a$
      - <font color='chartreuse'>即 $y=a$ 是微分方程的解，且在 $a$ 以外 $f(y) \neq 0$</font>
  - 则
    - $y=a$ 上解局部唯一 $\LR \dis\abs{\int^{a\pm\d}_a \frac{dy}{f(y)}} = +\infty$
  - **证明**：
    - <font color='chartreuse'>由Peano存在定理，方程在 $y=a$ 以外有解
    - 由恰当方程通解定理，存在通积分
    - 对通积分用隐函数存在定理得，$y=a$ 以外的积分曲线可写为隐函数 $y(x)$ 和 $x(y)$</font>
    - 计算得原方程可转化为 $\dis\int^y_{y_0} \frac{dy}{f(y)} = x(y)-x(y_0)$
    - **充分性（定义法）**：
      - 此时 $a$ 是积分的唯一瑕点，故由瑕积分发散得 $$\forall A，\exists \eta，\forall y\in (a,a+\eta]\ 都有 \ x(y) > A$$
      - 同理可得邻域另一边也如此
      - 即对任意积分曲线 $y = \Phi(x)$，都不可能存在一个 $x_0$ 使得 $\Phi(x_0) = a$
    - **充分性（端点效应法）**：
      - 反设存在积分曲线与 $y=a$ 相交，即 $\Phi(x_0) = a$
      - 由积分曲线连续性，此时必定存在 $x_1 = \inf\limits_{x>x_0}\{x\mid \Phi(x) \neq a\}$
      - 参照后面Osgoods的证明
    - **必要性**：
      - 由积分曲线连续可微性得 $\lim\limits_{\e\to 0} x(\e)$ 存在
      - 反设瑕积分有界，则上面极限有界，设为 $x_1$
      - 即存在一条积分曲线在 $(x_1,a)$ 上有定义，从而解局部不唯一，矛盾
  - **理解**：
    - 首先，$y=a$ 是方程的特解
    - 然后，证明的核心是 $\dis\int^a_{a-\e} \frac{dy}{f(y)} = \int^a_{a-\e} dx$。它是 $y=a$ 到 $y=a-\e$ 这段区间内，$x$ 的积分值
      - 所以如果瑕积分发散，则任何连续积分曲线在 $y=a$ 上无定义，解只有一个$y=a$。（√）（后面会学到，这叫做解延伸到开区间边界）
      - 如果解局部唯一，反设瑕积分收敛即可
  - **实例**：
    - $\cfrac{dy}{dx} = \sqrt{|y|}$，直线为 $y=0$
      - **解**：
        - 易得瑕积分有界，故解局部不唯一
        - <font color='chartreuse'>通解为 $y = \pm\cfrac{(x+C)^2}{4}$，此时直线为它的奇解（包络）</font>
    - $\cfrac{dy}{dx} = \begin{cases} y\ln|y| &  y\neq 0 \\ 0, & y=0
    \end{cases}$， 直线为 $y=\pm 1$
      - **解**：
        - $y=1$ 上瑕积分为 $\ln(\ln y)$，无界，故解局部唯一
        - $y=-1$ 上瑕积分为 $\ln(\ln(-y))$，有界，故解局部不唯一
    - $y = \begin{cases}Ce^{-\int p(x)dx} & y\neq 0\\ 0 & y=0\end{cases}$
      - **解**：
        - 在 $y=0$ 的邻域内，指数函数连续且趋近于横线。瑕积分无界，故解局部唯一

## 一阶线性微分方程

- **一阶线性ODE**：$\cfrac{dy}{dx} + p(x)y = q(x)$
- **分类**：
  - **齐次情况**：当 $q(x) = 0$ 时
    - **解**：变量分离即得 $$y(x) = Ce^{-\int p(x)dx}$$
  - **非齐次情况**：当 $q(x) \neq 0$ 时
    - **解（积分因子法）**：
      - 设 $\mu = e^{\int p(x)dx}$，方程两边乘 $\mu$，则左边变为全微分
      - 得解为 $$y = e^{-\int p(x)dx}(C + \int q(x)e^{\int p(x)dx}dx)$$
    - **解（分步构造法）**：
      - 尝试构造 $y$ 使其满足 $y' = -p(x)y + q(x)$
      - 关于 $p(x)y$ 项，由 $e$ 的导数不变性，解中必定存在 $e^{-\int p(x)dx}$ 项
      - 关于 $q(x)$ 项，由不定积分的线性，可用常数变易法设解为 $e^{\int p(x)dx}M(x)$，然后代入求解即可

### 一阶线性微分方程的性质

- **（齐次方程）特解存在性**：齐次线性方程必定存在特解 $y=0$，且通解无零点
   - **证明**：
     <!-- - $y = \begin{cases}Ce^{-\int p(x)dx} & y\neq 0\\ 0 & y=0\end{cases}$ -->
     - 在 $y=0$ 的邻域内，指数函数连续趋于横线。瑕积分无界，从而 $y=0$ 上的解局部唯一
     - <font color='chartreuse'>类似齐次线性方程组必有零解</font>
- **解的整体存在性**：若区间 $I$ 上 $p(x)$ 和 $q(x)$ 有定义且连续，则 $I$ 上始终存在解（无奇异点）
  - **证明**：已知指数函数无奇异点，故由通解公式直得结论
- **（齐次方程）解的线性封闭性**：齐次线性方程 $E$ 的解的线性组合依然是 $E$ 的解
  - **证明**：由通解公式得同一个线性方程的不同解只有系数不同，故线性封闭
  - **推论**：解还形成一个函数线性空间（详见线性微分方程组）
- **（非齐次方程）解的线性组合性**：非齐次线性方程 $E'$ 的某一个解，可与其齐次形式 $E$ 的通解相加，从而表出所有 $E'$ 的解
  - **证明**： 由通解公式易得结论
- **解的存在唯一性**：柯西问题中，解函数存在且唯一
  - **证明**：由通解公式 + 上面对特解的讨论易得结论


### 习题

- **不定积分到定积分的转化**：
  - *一般形式*：$\int f(x)dx = \int^x_0 f(s)ds + F(0)$
  - *零点形式*：$\int f(x)dx = \int^x_{x_0} f(s)ds$（原函数零点）
  - 特殊例：$$\dis\int f(x)e^{\int g(x)dx}dx = \int^x_0 f(s)e^{\int^s_0 g(t)dt + G(0)} ds + FE_{(fe^{\int})}(0)$$
  - **推论**：
    - 齐次线性微分方程中，积分常数 $C = y(0)e^{P(0)}$
    - 非齐次线性微分方程中，积分常数 $C = y(0)e^{P(0)}$


- **周期引理**：
  - 设ODE为 $\dfrac{dy}{dx} + p(x)y(x) = q(x)$
  - 若
    - $p,q$ 都是 $2\pi$ 周期函数
    - $y(0) = y(2\pi)$
  - 则 $y(x)$ 也是 $2\pi$ 周期函数
  - **证明**：
    - 易得 $y(x+2\pi)$ 也是方程的解
      - <font color='chartreuse'>相当于把 $y(x)$ 往左侧移动了 $2\pi$，此时 $p,q$ 不变，所以方程仍然成立</font>
    - 设 $u(x) = y(x+2\pi)-y(x)$，则 $u(x)$ 为相应齐次方程的解
      - 由题设条件，$y(0)=y(2\pi)$，即此时 $u(0) = 0$
      - 由线性齐次方程解的性质得 $u(x) \equiv 0$，即 $y(x+2\pi)\equiv y(x)$（**证毕**）
    - <font color='red'>本质是齐次情况的解若有零点则恒为0</font>
- **周期解定理**：
  - 设ODE为 $\dfrac{dy}{dx} + ay = f(x)$
  - 若 $a>0$，且 $f(x)$ 为周期 $2\pi$ 的连续函数
  - 求方程的 $2\pi$ 周期解
  - **解**：
    - 其为一阶线性ODE，故通解形式为 $y(x) = Ce^{-ax} + \int^x_0 e^{-a(x-s)}f(s)ds$
    - <font color='red'>问题转化为：求使 $y$ 成为周期函数的 $C$</font>
    - 代入 $y(2\pi) = y(0)$ 即可求出 $C = \cfrac{\int^{2\pi}_0 e^{as}f(s)ds}{e^{2\pi a}-1}$
    - 将C代入通解即得 $2\pi$ 周期解

### 习题

- **积分因子法求不等式**：设 $x\geqslant 0$，若 $y' + a(x)y \leqslant 0$，则 $y(x) \leqslant y(0)e^{-\int^x_0 a(s)ds}$
  - **证明**：
    - 左边添项 $e^{\int a(x)dx}$ 合并成积分后，发现解函数单调递减，从而得到结论
  - **本质**：
    - 就是线性微分方程的解法，把等号换成不等号了而已
    - 其实应该是个数分题
- **对C的约束**
  - **周期函数的平均值**：设一阶线性ODE的系数函数都以 $w$ 为周期
    - **齐次情况**：解函数以 $w$ 为周期 $\LR$ 系数函数的积分均值 $\bar{p} = \cfrac{\int^w_0 p(x)dx}{w} = 0$
    - **非齐次情况**：方程有唯一的周期解 $\LR$ $\bar{p} \neq 0$
    - **证明**：直接利用线性微分方程的通解，以及周期性方程 $y(0) = y(w)$ 即可得出结论
      - <font color='chartreuse'>齐次情况：由通解公式，$$y(x) = Ce^{\dis\int^x_0 p(s)ds} = y(x+w) = Ce^{\dis\int^{x+w}_0 p(s)ds}$$
        - 即此时 $\dis\int^{x+w}_x p(s)ds = 0$。再由周期性即得结论
      - 非齐次情况：由通解公式，$$y(0) = C = y(w) = e^{\int p}\Big[ C + \int q(x)e^{\int p(x)dx}dx \Big]$$
        - 变形可得 $C = \dis\cfrac{e^{\int p}\int qe^{\int p}}{e^{\int p} - 1}$
        - 反设 $\ol p = 0$，则 $e^{\int p}-1 = 0$，从而 $C = \infty$，原方程无解</font>
  - **有界解**：
    - 若 $f(x)$ 在 $\R$ 上连续有界
    - 则 $y' + y = f(x)$ 在 $\R$ 上仅一个有界解
    - **证明**：
      - 易得通解为 $y(x) = \cfrac{C}{e^x} + \dis\int^x_0 f(s)e^{s-x}ds$
        - <font color='chartreuse'>解的不同仅依赖于 $C$ 的取值不同</font>
      - 设 $|f| \leq M$，则由三角不等式 + 积分上界不等式易得 $$|y(x)| \leq |Ce^{-x} + M|$$
        - <font color='red'>任何关于有界性的题目中，先考虑积分中值定理，若不行就考虑积分上界不等式</font>
        - 由于 $e^{-x}$ 无界，故只有 $C = 0$ 时才能使 $y(x)$ 有界（**证毕**）

## 初等变换法

- **初等变换法**：通过换元将方程转化为已知可解的方程<font color='chartreuse'>（变量分离方程、一阶线性方程、恰当方程等）</font>
- **实例**：
  - $\dis\frac{dy}{dx} = \frac{x^2+y^2}{2y}$
    - **解1**：令 $y^2= z$ 即可（伯努利方程）
    - **解2**：积分因子法即可
  - $\dis\frac{dy}{dx} = \frac{y}{x+y^2}$
    - **解1**：调换自变量和因变量，发现是伯努利方程
    - **解2**：积分因子法即可
  - $\dis 3xy^2\frac{dy}{dx}+y^3+x^3 = 0$
    - **解1**：令 $y^3=z$ 即可（伯努利方程）<font color='chartreuse'>（有特解 $x=0$）</font>
    - **解2**：是齐次方程
  - $\dis\frac{dy}{dx} = \frac{1}{\cos y} + x\tan y$
    - **解**：令 $z = \sin y$，则化为 $\dfrac{dz}{dx} = 1+xz$，一阶线性方程
  - $\dis\frac{dy}{dx} = y^2 + \frac{1}{x^2}$
    - <font color='chartreuse'>（这是个特殊的里卡蒂方程）</font>
    - **解1**：同乘 $x^2$，令 $y = xy$，则化为变量分离方程
- **齐次方程**：$\begin{cases} P(tx,ty) = t^mP(x,y) \\ Q(tx,ty) = t^mQ(x,y) \end{cases}$
  - **解法**：令 $y = ux$，构成变量分离方程
  - **零次齐次方程**：$\dfrac{dy}{dx} = f(\dfrac{ax+by}{mx+ny})$
      - **解**：直接令 $y = ux$
  - **可化为零次齐次方程**：$\dfrac{dy}{dx} = f(\dfrac{ax+by+c}{mx+ny+l})$
      - **解**：
        - 首先求 $\begin{vmatrix} a & b \\ m & n \end{vmatrix}$
        - 若为零，则分子分母线性无关，通过 $x、y$ 平移变换，消去 $c$ 和 $l$ 即可
        - 若不为0，则分子分母线性相关，直接取因变量 $u = ax+by$
- **伯努利方程**：$\dfrac{dy}{dx} + p(x)y = q(x)y^n$
  - **解法**：
    - 积分因子为 $\cfrac{1-n}{y^n}$，新元为 $u = \cfrac{1}{y^{n-1}}$，此时化为一阶线性方程
- **里卡蒂方程**：$\dfrac{dy}{dx} = p(x)y^2 + q(x)y + r(x)$
  - **已知特解 $\varphi_1(x)$，求通解**：
    - 令 $y = u + \varphi_1(x)$
    - 两边消项后，可化为伯努利方程 $\dfrac{du}{dx} = \big[ 2p(x)\varphi_1(x)+q(x) \big] u + p(x)u^2$
      - <font color='chartreuse'>消去了 $r(x)$</font>
  - **特殊的里卡蒂方程**：$\dfrac{dy}{dx} = -ay^2 + bx^m$
    - 若 $m=0$，则直接转化为变量分离方程
    - 若 $m=-2$，则此时次数相匹配，可令 $z=xy$，即转化为变量分离方程
    - 若 $m=-4$，则
    - 若 $m = \dfrac{4k}{2k+1}$
      - 设 $\begin{cases} x = \xi^{\frac{1}{m+1}} \\ y = \frac{b}{m+1}\frac{1}{\eta} \end{cases}$，方程变为 $\frac{d\eta}{d\xi} = -\eta^2 + \frac{b}{(m+1)^2}\xi^{\large\frac{-4k}{2k-1}}$
      - 设 $\begin{cases} \xi = \frac{1}{t} \\ \eta = t-zt^2 \end{cases}$，方程变为 $\frac{dz}{dt} = -z^2 + \frac{b}{(m+1)^2}t^{\large\frac{-4(k-1)}{2(k-1)+1}}$
      - 变换k次后，$\begin{cases} a\to \frac{1}{a^{k-1}} \\ b\to \frac{b}{(m+1)^{2k}} \\ m\to 0 \end{cases}$，可化为变量分离方程
    - **推论**：$m = \frac{-4k}{2k-1}$ 也可解

## 积分因子法

- **积分因子法**：在方程两边同乘一个函数 $\mu$（积分因子），使其变为恰当方程
- **单变量积分因子存在条件**：以 $x$ 为自变量的积分因子 $\mu(x)$ 为例
  - **解**：
    <!-- - 方程变为：$\mu(x)P(x,y)dx + \mu(x)Q(x,y)dy = 0$ -->
    - 乘 $\mu$ 后若为恰当方程，则：$\large\frac{\partial (\mu P)}{\partial y} = \frac{\partial (\mu Q)}{\partial x} \red\Rightarrow \frac{1}{\mu}\frac{d\mu}{dx} = \frac{1}{Q}(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x})$
      - <font color='chartreuse'>因为左式仅以 $x$ 为自变量，所以右式也必须仅以 $x$ 为自变量
      - 没有其它的约束条件</font>
    - 此时上面的等式是一阶齐次线性方程，故可解得 $\mu = Ce^{\large\int \frac{1}{Q}(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x})dx}$
- **积分因子变换定理**：
  - **分组法**：
    - 若方程可线性分离为 $(P_1dx + Q_1dy) + (P_2dx + Q_2dy) = 0$
    - 则对两括号内分别求积分因子 $\mu 1,\mu 2$，其公因式 $\mu$ 就是原方程的积分因子
    - **证明**：
      - **必要性**：显然不同积分因子的公因式依然是积分因子
      - **充分性**：可由引理（添项后，积分因子只小不大）得到
  - **变换法**：
    - 设乘上积分因子 $\mu$ 后的通积分为 $\Phi$
    - 则 $\mu(x,y)g\Big(\Phi(x,y)\Big)$ 也是方程的一个积分因子
      - <font color='chartreuse'>$\Phi(x,y)$ 在这里是隐函数，即 $\Phi(x,y) = C$ 的左式，并不是说它是常数</font>
    - **证明**：
      - 计算即可
      - <font color='chartreuse'>将 $\dfrac{\partial (\mu Pg)}{\partial y} = \dfrac{\partial (\mu Qg)}{\partial x}$ 展开
        - 左式：$\dfrac{\partial (\mu Pg)}{\partial y} = \dfrac{\partial \mu}{\partial y}Pg + \dfrac{\partial P}{\partial y}\mu g + \dfrac{\partial g}{\partial y}\mu P$
        - 右式同理
      - 将左右两式作差：
        - 由于 $g(\Phi(x,y))$ 是复合函数，故上面两式最后一项展开后都为 $\cfrac{\partial g}{\partial \varPhi}\mu PQ$，从而可约去
        - 前两项的 $g$ 因子可约去
        - 最后转化为普通积分因子法的判别式，在题设中它已经成立</font>
    - **理解**：通积分可约去，从而满足积分因子判别法
  - **综合应用（线性分离法）**：
    - 将方程线性分离后，求出 $\mu_1，\mu_2$，再构造 $g_1，g_2$ 让 $\mu_1 g_1(\Phi_1) = \mu_2 g_2(\Phi_2)$，则它就是积分因子

### 性质题

- **齐次积分因子**：齐次方程中，$\mu = \dfrac{1}{xP + yQ}$ 是一个积分因子
  - **证明**：换元 $y = ux$，计算即可
    - <font color='chartreuse'>由齐次性，P和Q可化为仅依赖于u的函数，此时只需要对系数分离
    - 求变换后的系数，发现可添加 $(P+uQ)$ 作为分母
      - 其仅依赖于u，可约去dx的u系数部分，且不干扰du系数，从而得到变量分离方程
      - 其反变换即为积分因子</font>
  - **本质**：齐次函数的欧拉公式
- **积分因子存在判别法**：存在积分因子 $\mu = \mu(\Phi(x,y)) \LR \cfrac{\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}}{Q\frac{\partial \Phi}{\partial x} - P\frac{\partial \Phi}{\partial y}} = f(\Phi)$<font color='chartreuse'>（即左式仅与 $\Phi$ 有关）</font>
  - **证明**：
    - 利用恰当方程的混合偏导数相等即可 $$\pfrac{\Phi}{x\pa y} = \pfrac{P}{y}\mu(\Phi) + P\frac{d\mu}{d\Phi}\pfrac{\Phi}{y} = \pfrac{Q}{x}\mu(\Phi) + Q\frac{d\mu}{d\Phi}\pfrac{\Phi}{x}$$
    - 上式可变形为 $\cfrac{d\mu}{d\Phi}\mu(\Phi) = f(\Phi)$
    - 解得积分因子为 $\mu(\Phi) = Ce^{\int f(\Phi)d\Phi}$
  - **推论**：可求出下列积分因子存在的充要条件
    - $\mu = \mu(x\pm y)$
      - **解**：此时 $\Phi = x+y$，则 $\dfrac{P_y-Q_x}{Q-P}$ 仅依赖于 $x+y$
    - $\mu = \mu(x^2+y^2)$
      - **解**：此时 $\Phi = x^2+y^2$，则 $\dfrac{P_y-Q_x}{xQ-yP}$ 仅依赖于 $x^2+y^2$
    - $\mu = \mu(xy)$
      - **解**：此时 $\Phi = xy$，则 $\dfrac{P_y-Q_x}{yQ-xP}$ 仅依赖于 $xy$
    - $\mu = \mu(\dfrac{y}{x})$
      - **解**：此时 $\Phi = \dfrac{y}{x}$，则 $\dfrac{P_y-Q_x}{yQ+xP}$ 仅依赖于 $\dfrac{y}{x}$
    - $\mu = \mu(x^\a y^\b)$
      - **解**：此时 $\Phi = x^\a y^\b$，则 $\dfrac{P_y-Q_x}{\a yQ-\b xP}$ 仅依赖于 $x^\a y^\b$
- **通积分与积分因子**：
  - 设 $P,Q$ 连续可微
  - 若 $\mu_1,\mu_2$ 是原方程的两个线性无关的连续可微积分因子，则 $\dfrac{\mu_1}{\mu_2} = C$ 是方程的通积分
  - **证明**：
    - 设 $\mu_1$ 是方程的积分因子，令 $\mu_2 = \mu_1·g(\Phi)$，则由变换法，$\mu_1$ 也是方程的积分因子
    - 再令 $g(\Phi) = \Phi$，这时 $\mu_1$ 是由 $\mu_2$ 唯一确定的。则 $\frac{\mu_1}{\mu_2} = \Phi$ 是通积分
