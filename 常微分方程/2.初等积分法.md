# 常微分方程2：初等积分法

## 恰当方程（全微分方程）

- **对称形式**：$P(x,y)dx + Q(x,y)dy = 0$
  - **非对称形式**：$\frac{dy}{dx} = - \frac{P(x,y)}{Q(x,y)}$
- **恰当方程**：如果存在一个可微函数 $\Phi(x,y)$，使得其全微分为其对称形式
  - **恰当性**：恰当方程一定存在解
  - 一般来讲，除非约去了某些值，否则可解的方程都是恰当方程（而约去的值可以用积分因子法来填充）（？那里卡蒂方程呢）
- **通积分（隐函数解）**：$\Phi(x,y) = C$，即恰当方程的解

### 恰当方程的判断和求解

- **判定定理**：设函数 $P(x,y)和Q(x,y)$
  - 在区域 $R: \alpha < x < \beta, \gamma < y < \delta$ 上连续，
  - 并且有连续的一阶偏导数 $\frac{\partial P}{\partial y}和\frac{\partial Q}{\partial x}$
  - 则微分方程是恰当方程 $\Leftrightarrow$ $\frac{\partial}{\partial y}P(x,y) \equiv \frac{\partial}{\partial x}Q(x,y)$
- **必要性**：偏导数连续性 $\Rightarrow$ 混合偏导数相等
  - **理解**：微分中值定理 + 多元函数连续的自变量独立性
- **充分性**：此时的通积分为
  - $\Phi(x,y) = \int^x_{x_0}P(x,y)dx + \int^y_{y_0}Q(x_0,y)dy$
  - $\Phi(x,y) = \int^x_{x_0}P(x_0,y)dx + \int^y_{y_0}Q(x,y)dy$
  - **理解**：上面两种情况都可以满足偏微分条件，进而满足全微分条件。但是偏微分并不是老老实实地一个归一个，而是（P占据了所有，而Q只是用来消去P的残值），或（Q占据了所有，而P只是用来消去Q的残值）
    - 其实，上面这两种形式才是真正的恰当方程。一开始设的 $P(x,y)dx + Q(x,y)dy = 0$ ，只是因为未知才作的保全之策。

## 变量分离方程

- 原方程：$X(x)Y_1(y)dx + X_1(x)Y(y)dy = 0$（$X_1和Y_1都是非主元$）
  - 分离：$\frac{X(x)}{X_1(x)}dx + \frac{Y(y)}{Y_1(y)}dy = 0$
  - 解法：由积分的同时性，分别对x和y积分即可
  - **特解**：若有 $X_1(a) = 0$，则其为原方程的解，但不是分离方程的解
    - *实例*：$(x^2+1)(y^2-1)dx + xydy = 0$
      - 在 $x=0$ 时有解
      - 在 $y=0$ 时无解
  - **微分方程定义（导数）理解法**：
    - *微分方程解的整体性*：如果令x=0，则解只能是x=0，它不能分割成x和y两个条件（否则就只是一个无价值的线素）
    - *实例*：上例中 $\frac{dy}{dx} = \frac{(x^2+1)(y^2-1)}{-xy}$
      - 如果令 $x=0$，则相当于默认方程的解是函数 $x=0$。只需检验斜率 $k=+\infty$ 是否符合等式即可
      - 同理，可发现 $y=0$ 时，$k \neq 0$，不符合等式，不是特解。
  - **微分方程定义（微分）理解法**：
    - *分式形式*：$$
    - 而变量分离方程就是类似奇点的情况 $\frac{dx}{X_1(x)} = \frac{0}{0}$
  - **线素连续性理解法**：
    - 微分方程的单元解是线素，解集是线素场（所有的原函数）
      - 沿函数移动是定C动x，沿 $x=a或y=b$ 移动是变C定x或y
      - 有解的条件是必须形成一个函数，也就是线素的斜率和位置必须相符，称为“连续”
    - 上例中取x=0，则原方程变为 $(y^2-1)dx = 0$，为了使连续的 $y$ 都能成立等式，要有 $dx = 0$，即 $y' = +\infty$
      - 综上线素为 $x=0$ 且 $k=+\infty$，其连续
    - 再取y=0，则原方程变为 $(x^2+1)dx = 0$，为了使连续的 $x$ 都能成立等式，要有 $dx = 0$，即 $y' = +\infty$
      - 综上线素为 $y=0$ 且 $k=+\infty$，其不连续
    - 所以x=0能成立，但是y=0不能成立
  - **本质**：特解一般是 $P(x,y)或Q(x,y) = 0$，即 $k=\infty或0$ 时产生。且都是和坐标轴平行的直线。

### 局部唯一性

- **孤立奇异点实例**：$\frac{dy}{dx} = \frac{(x^2+1)(y^2-1)}{-xy}$
  - 通解 $y^2 = 1+C\frac{e^{-x}}{x^2}$
  - 特解 $x=0$
  - 奇异点 $(0,\pm 1)$，解局部不存在
- **连续奇异点实例**：$\frac{dy}{dx} = \frac{3}{2}\sqrt[3]{y}$
  - 通解 $y^2 = (x+C)^3$
  - 特解$\begin{cases} x < c \\ y=0 \end{cases}$（该段上有无限个积分曲线的重叠，解局部不唯一）
- **瑕积分定理**：已知微分方程 $\frac{dy}{dx} = f(y)$，
  - 若
    - $f(y)$ 在直线 $y=a$ 的某邻域 $(a-\varepsilon,a+\varepsilon)$ 内连续
    - $f(y) = 0 \Leftrightarrow y=a$
  - 则
    - $y=a$ 上解局部唯一 $\Leftrightarrow |\int^{a\pm\varepsilon}_a \frac{dy}{f(y)}| = +\infty$
  - **证明**：
    - 方程可转化为 $\int^y_{y_0} \frac{dy}{f(y)} = x-x_0$
      - 左端原函数设为 $G(y) = x$，则方程为 $G(y)-G(y_0) = x-x_0$
    - **充分性**：反设瑕积分不发散，则由N-L定理，存在两点 $(x_1,a+\varepsilon)，(x_0,a)$ 位于同一积分曲线 $G(y) = x$ 上
      - 该曲线显然与 $y=a$ 不同，从而解局部不唯一
    - **必要性**：若解局部唯一，则解只有 $y=a$
      - 从而 $\forall x\in R，G(a) = x$ 无解，即 $G(y)$ 在 $y=a$ 处无定义或极大
      - 再由 $f(y)$ 局部连续性，其可积，从而 $G(y)$ 有定义，那么只能是极大反常积分
  - **理解**：
    - 首先，$y=a$ 是一条积分曲线，是方程的特解
    - 然后，由积分的同时性，$\int^a_{a-\varepsilon} \frac{dy}{f(y)} = \int^a_{a-\varepsilon} dx$，是指 $y=a$ 到 $y=a-\varepsilon$ 这段区间内，x的积分值
      - 所以如果瑕积分发散，则任何连续积分曲线在 $y=a$ 上无定义，解只有一个$y=a$。（√）（后面会学到，这叫做解延伸到开区间边界）
      - 如果解局部唯一，反设瑕积分收敛即可
  - **本质**：
    - 右侧条件表示 $\int^{a+\varepsilon}_a dx = +\infty$，即靠近 $y=a$ 时 $x$ 趋向于无穷，也就是取不到，只能作为渐近线。
    - 如果能取到，那么切线会导致 $y=a$ 上的点的解（线素）局部不唯一
  - **实例**：
    - $\frac{dy}{dx} = \sqrt{|y|}$，直线为 $y=0$
      - 瑕积分不发散，其上均为切线
    - $\frac{dy}{dx} = \begin{cases} yln|y|, y\neq 0 \\ 0, \qquad y=0
    \end{cases}$， 直线为 $y=\pm 1$
      - $y=1$ 上瑕积分为 $ln(lny)$，发散
      - $y=-1$ 上瑕积分为 $ln(ln(-y))$，不发散
    - $y = \begin{cases}Ce^{-\int p(x)dx}，y\neq 0\\ 0，y=0\end{cases}$
      - 在 $y=0$ 的邻域内，指数函数连续且趋近于横线。瑕积分发散，$y=0$ 上的解局部唯一

## 一阶线性方程

- $\frac{dy}{dx} + p(x)y = q(x)$
  - **齐次线性方程**：$q(x) = 0$
    - **解**：变量分离得 $y(x) = Ce^{-\int p(x)dx}$
  - **非齐次线性方程**：$q(x) \neq 0$
    - **解**：积分因子法，设 $\mu = e^{\int p(x)dx}$，左边变为全微分
        - 得解为 $y = e^{-\int p(x)dx}(C + \int q(x)e^{\int p(x)dx}dx)$
    - **构造法思路**：构造 $y' = -p(x)y + q(x)$
      - $p(x)y$：由e的导数不变性，存在 $e^{-\int p(x)dx}$ 项
      - $q(x)$：由积分线性，可用常数变易法设解为 $e^{\int p(x)dx}M(x)$

### 线性微分方程的性质

1. **平凡解**：解要么恒为0，要么恒不为0
   - $y = \begin{cases}Ce^{-\int p(x)dx}，y\neq 0\\ 0，y=0\end{cases}$
   - 在 $y=0$ 的邻域内，指数函数连续且趋近于横线。瑕积分发散，从而 $y=0$ 上的解局部唯一
2. **解的整体存在性**：方程的解都在$p(x)和q(x)$有定义且连续的整个区间上存在（没有奇异点）
3. **同一方程中，不同解的线性组合性**：不同解指C不同，而C是因式
4. **同一非齐次方程的解与通解**：C是因式内的一个单项式
5. **初值问题中解存在且唯一**：类似线性方程组的解

- **不定积分到定积分的转化**：
  - *一般形式*：$\int f(x)dx = \int^x_0 f(s)ds + F(0)$
  - *零点形式*：$\int f(x)dx = \int^x_{x_0} f(s)ds$（原函数零点）
  - 特殊例：
    - $\int f(x)e^{\int g(x)dx}dx = \int^x_0 f(s)e^{\int^s_0 g(t)dt + G(0)} ds + FE_{(fe^{\int})}(0)$
  - 微分方程中，常数项被微分消去


### 周期函数的线性微分方程

- 微分方程 $\frac{dy}{dx} + ay = f(x)$，其中a>0，且 $f(x)$ 为周期 $2\pi$ 的连续函数，求方程的 $2\pi$ 周期解
  - **解**：通解为 $y(x) = Ce^{-ax} + \int^x_0 e^{-a(x-s)}f(s)ds$
    - 问题转化为：求C使得y成为周期函数
    - **引理**：所有 $x$ 的周期等价于 $\forall x_0（比如0）$ 的周期成立
      - **证明**：
        - $y(x+2\pi)$ 也是方程的解：$\frac{dy_1}{d(x+2\pi)} + ay_1 = f(x+2\pi)$
          - 相当于把y往x轴左侧移动了 $2\pi$，因为微分不变，所以 $f(x)$ 也不变，所以方程仍然成立
        - 设 $u(x) = y(x+2\pi)-y(x)$，则 $u(x)$ 为相应齐次方程的解
        - 再由 $y(0)=y(2\pi)$ 时 $u(0) = 0$，由解的结构，得 $u(x) \equiv 0$，即 $y(x+2\pi)\equiv y(x)$（**证毕**）
    - 然后再代入 $y(2\pi) = y(0)$ 求出 $C = \cfrac{\int^{2\pi}_0 e^{as}f(s)ds}{e^{2\pi a}-1}$
    - 将C代入通解即得 $2\pi$ 周期解
  - **理解**：
  - **本质**：周期和C相关

## 初等变换法

- 转化为**变量分离方程**/**一阶线性方程**/**恰当方程**
  - $\frac{dy}{dx} = \frac{x^2+y^2}{2y}$：令 $y^2= z$
  - $\frac{dy}{dx} = \frac{y}{x+y^2}$：调换自变量和因变量
  - $3xy^2\frac{dy}{dx}+y^3+x^3 = 0$：令 $y^3=z$
  - $\frac{dy}{dx} = \frac{1}{cosy} + xtany$：令 $z = siny$
  - （里卡蒂方程）$\frac{dy}{dx} = y^2 + \frac{1}{x^2}$，同乘 $x^2，令y = xy$，构成变量分离方程
- **齐次方程**：$\begin{cases} P(tx,ty) = t^mP(x,y) \\ Q(tx,ty) = t^mQ(x,y) \end{cases}$
  - **解法**：令 $y = ux$，构成变量分离方程
  - **零次齐次方程**：
    - $\frac{dy}{dx} = f(\frac{ax+by}{mx+ny})$
      - 直接令 $y = ux$
  - **可化为零次齐次方程**
    - $\frac{dy}{dx} = f(\frac{ax+by+c}{mx+ny+l})$
      - 求解 $\begin{vmatrix} a & b \\ m & n \end{vmatrix}$
      - 上下线性无关，则将x、y平移变换，消去c和l
      - 上下线性相关，则直接 $u = ax+by$
- **伯努利方程**：$\frac{dy}{dx} + p(x)y = q(x)y^n$
  - **解法**：换元成为一阶齐次线性方程
    - 积分因子为 $(1-n)y^{-n}$，新元为 $u = y^{1-n}$
- **里卡蒂方程**：$\frac{dy}{dx} = p(x)y^2 + q(x)y + r(x)$
  - **已知特解 $\varphi_1(x)$，求通解**：
    - 令 $y = u + \varphi_1(x)$
    - 两边消项得伯努利方程 $\frac{du}{dx} = \big[ 2p(x)\varphi_1(x)+q(x) \big] u + p(x)u^2$（消去 $r(x)$）
  - **特殊的里卡蒂方程**：$\frac{dy}{dx} = -ay^2 + bx^m$
    - $m=0$，则直接转化为变量分离方程
    - $m=-2$，次数匹配，令 $z=xy$，则转化为变量分离方程
    - $m = \frac{4k}{2k+1}$
      - 设 $\begin{cases} x = \xi^{\frac{1}{m+1}} \\ y = \frac{b}{m+1}\frac{1}{\eta} \end{cases}$，方程变为 $\frac{d\eta}{d\xi} = -\eta^2 + \frac{b}{(m+1)^2}\xi^{\large\frac{-4k}{2k-1}}$
      - 设 $\begin{cases} \xi = \frac{1}{t} \\ \eta = t-zt^2 \end{cases}$，方程变为 $\frac{dz}{dt} = -z^2 + \frac{b}{(m+1)^2}t^{\large\frac{-4(k-1)}{2(k-1)+1}}$
      - 变换k次后，$\begin{cases} a\to \frac{1}{a^{k-1}} \\ b\to \frac{b}{(m+1)^{2k}} \\ m\to 0 \end{cases}$，可化为变量分离方程
    - **推论**：$m = \frac{-4k}{2k-1}$ 也可解
  - **高阶微分方程转化为里卡蒂方程**：$y'' + p(x)y' + q(x)y = 0$
    - **解**：令 $y = e^{\int u(x)dx}$ 即可
    - **理解**：利用e的还原性达到分离因式 $e^{\int u(x)dx}$ 的目的

## 积分因子法

- **单变量积分因子存在条件**：以 $\mu(x)$ 为例
  - 积分因子方程：$\mu(x)P(x,y)dx + \mu(x)Q(x,y)dy = 0$
  - 恰当方程条件：$\large\frac{\partial (\mu P)}{\partial y} = \frac{\partial (\mu Q)}{\partial x} \rightarrow \frac{1}{\mu}\frac{d\mu}{dx} = \frac{1}{Q}(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x})$
    - 右式必须仅以x为自变量
    - 积分因子为 $ Ce^{\large\int \frac{1}{Q}(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x})dx}$
- **积分因子变换定理**：
  - *分组定理*：方程线性分离后，分别求积分因子 $\mu 1,\mu 2$，其公因式 $\mu$ 就是原方程的积分因子
    - **证明**：必要性易得，充分性可由引理（添项后，积分因子只小不大）得到
  - *变换定理*：若 $\mu P(x,y)dx + \mu Q(x,y)dy = d\Phi(x,y)$ 成立，则 $\mu(x,y)g(\Phi(x,y))$ 也是方程的一个积分因子
    - $\Phi(x,y)$ 在这里是隐函数形式，即 $\Phi(x,y) = C$，并不是指它是常数。
    - **证明**：
      - 将 $\frac{\partial (\mu Pg)}{\partial y} = \frac{\partial (\mu Qg)}{\partial x}$ 展开
        - P式：$\frac{\partial (\mu Pg)}{\partial y} = \frac{\partial \mu}{\partial y}Pg + \frac{\partial P}{\partial y}\mu g + \frac{\partial g}{\partial y}\mu P$
        - Q式同理
      - 约去相等项：
        - $g(\Phi(x,y))$ 是复合函数，P式、Q式最后一项都为 $\frac{\partial g}{\partial \varPhi}\mu PQ$，可约去
        - 前两项的g因子可约去
        - 最后转化为普通积分因子法的相等式，题设中它已经成立。**证毕**
    - **理解**：几何意义暂时未知，代数意义就是全微分可约去，从而满足积分因子判别法
- **应用**：
  - **线性分离法**：求出 $\mu_1，\mu_2$，构造 $g_1，g_2$ 让 $\mu_1 g_1(\Phi_1) = \mu_2 g_2(\Phi_2)$
  - **齐次积分因子**：齐次方程中，$\mu = \frac{1}{xP + yQ}$ 是一个积分因子
    - **证明**：换元 $y = ux$，计算即可
    - **理解**：
      - 由齐次性，P和Q可化为仅依赖于u的函数，此时只需要对系数分离
      - 求变换后的系数，发现可添加 $(P+uQ)$ 作为分母
        - 其仅依赖于u，可约去dx的u系数部分，且不干扰du系数，从而得到变量分离方程
        - 其反变换即为积分因子
    - **常用例**：$(ydx-xdy)-(x^2+y^2)dy = 0$，$\mu = \frac{1}{x^2+y^2}$
  - $\exist 积分因子\mu = \mu(\varphi(x,y)) \Leftrightarrow \cfrac{\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}}{Q\frac{\partial \varphi}{\partial x} - P\frac{\partial \varphi}{\partial y}} = f(\varphi)$，即自变量恒定？
    - 证明：利用恰当方程的混合偏导数定理
  - 令 $\mu_1 = \mu_2·g(\phi)$，由定理，$\mu_1$ 也是积分因子。再令 $g(\phi) = \phi$，这时 $\mu_1$ 是由 $\mu_2$ 唯一确定的。则 $\frac{\mu_1}{\mu_2} = \phi$ 是通积分

### 等角轨线族（隐函数求导）（2.4的课后题）

- **积分曲线族**：每个微分方程确定一个曲线族。
  - $Pdx + Qdy = 0$ 确定定义域上每点的切线
  - 积分后（隐函数形式）为 $\Phi(x,y) = C$
  - 沿着函数移动，定C变x。沿着坐标轴移动，定x变C
    - 给定x和y，则可知道具体的曲线常数C。因此，C可看作是x,y的函数
    - C也可由初值决定，具体对方程的影响见后面的解对初值的依赖部分
- **几何曲线族**：每个等式确定一个曲线，等式的集合确定一个曲线族
  - $\Phi(x,y,C) = 0$，其中C就是可变常数，表示同族曲线的不同位置
- **等角轨线族**：两个曲线族，它们的任一曲线的交点，切线都成定角
  - **求解**：已知曲线族 $\Phi(x,y,C) = 0$，求 $\alpha$ 等角轨线族
    - 写出整体的微分方程 $Pdx + Qdy = 0$（同族曲线的切线走向相同）
      - 可消去C的情况：
        - 函数具有周期性，则可以根据 $y(0) = y(\omega)$，以及y的解来求出C。**微分方程只有一个周期解**
        - 等角曲线族，此时C成为 $C(x,y)$。C可以表示初值，可以表示
      - 不可消去C的情况：
        - 微分方程变形，得到二阶隐函数的导数
          - $\frac{dy}{dx} = H(x,y) = \frac{P(x,y)}{Q(x,y)} = -\cfrac{\Phi_x(x,y,C(x,y))}{\Phi_y(x,y,C(x,y))}$
        - 本式中并不代入 $C(x,y)$（一旦代入就是0）
    - 解方程 $\begin{cases} \frac{dy}{dx} = \tan(\theta + \alpha) \\ H(x,y) = \tan\theta \end{cases}$
      - 解出 $\varPsi(x,y) = K$，即得等角轨线族 $\varPsi(x,y,K)$
