- **Cayley定理**：过 $n$ 个有标志顶点的树的数目等于 $n^{n-2}$
  - 用 $n-1$ 条边连接 $n$ 个顶点的连通图个数为 $n^{n-2}$
  - **证明（一一对应法）**：
    - 给树叶编号为 $1\sim n$
    - **删除方法**
      - 首先找到标号最小的顶点 $a_1$，其邻接点设为 $b_1$。消去 $a_1$ 和边 $a_1b_1$，则 $b_1$ 成为新顶点。
      - 重复 $n-2$ 次，则最后剩下一条边，一棵树对应序列 $b_1,b_2,...,b_{n-2}$ 
    - **恢复方法**
      - 已知数列 $1,...,n$ 和节点编号序列 $b_1,...,b_{n-2}$，则在数列中找到一个不出现在节点编号序列中的数，设为 $a_1$，从而得到边 $(a_1,b_1)$
      - 重复至编号集为空集，剩下最后一条边
    - 由于取编号任意，因此共有 $n^{n-2}$ 种方案

- **排列数**：$P(n,r)$
- **组合数**：$C(n,r)$
- **圆周排列数**：$\cfrac{P(n,r)}{r}$
  - **证明**：由于排列长度为 $r$。则任选一个排列，其平移 $r$ 个位置，在圆周排列时相等，但在普通排列中不相等
  - 每个排列数量缩水 $r$ 倍，就是圆周排列

### 排列生成算法

- **等价性**：每种算法对应排列的一种现实意义
- **序数法**：添项分离组合得递推公式
  - 一阶递推 $n! = (n-1)(n-1)! + (n-1)!$
  - 完全展开 $n!=  \sum\limits^{n-1}_{k=1} k\cdot k! + 1$
  - **意义**：对应序数的排列种类数量
  - **推论（除公因算法）**：已知 $m$ 是排列数的线性组合，如何求系数
    - $m = \sum\limits^{n-1}_{k=1}a_k\cdot k!$
    - 由于排列数公因数相邻，因此首先估计最大序数，然后从它开始不断相除，每次的余数为对应排列数系数
    - **现实意义**：$a_i$ 表示右方数位中比（当前位的数）小的（位数个数）
- **字典序法**：定义见高代
- **换位法**：定义箭头排列 $\overleftarrow{1}\overleftarrow{2}\overleftarrow{3}\overleftarrow{4}$
  - **活动状态的数**：箭头所指方向的相邻数比它小
  - **算法**：将最大的活动数 $a$ 和相邻小数换位，比 $a$ 大的数均改变方向，直到没有数处于活动状态

### 条件组合问题

- **重复组合定理**：$n$ 个元素中取 $r$ 个，且允许重复，则组合数为 $C^{r}_{n+r-1}$
  - **证明（一一对应法）**：假设有重复组合为 $(a_1,...,a_r)$，则变换为 $(a_1,a_2+1,...,a_r+r-1)$
    - 可证它们互不相同。考虑极端情况（所有元均相等）即可
    - 此时原元素扩充到 $n+r-1$ 个，当作不重复组合来做
  - **证明（备用元法）**：这多出来的 $r-1$ 个元素就是允许重复的备用元
    - 因为一个元素重复1次后，其余元素的可重复次数必定减少1，所以备用元是所有元素共享的。
    - 且只要一个元素不同，对应的组合就不一样，所以备用元可以看作不同元素。
- **不相邻组合定理**：$n$ 个元素中取 $r$ 个不相邻元素，组合数为 $C^r_{n-r+1}$
  - **证明（一一对应法）**：设不相邻组合为 $(b_1,...,b_r)$，则变换为 $(b_1,b_2-1,...,b_r-r+1)$，与不重复组合对应
  - **证明（删除元法）**：这减少的 $r-1$ 个元素就是相邻的删除元
    - 删除 $r-1$ 个不相邻元素后，剩余部分无论怎么选 $r$ 个，都不会相邻
    - 且删除后组合数和原数量相等（互包证明即可）
- **举例**
  - 分配问题：
    - 线性方程的解个数 $C^b_{n-b+1}$
    - 将b个相同的球放进n个不同的盒子里，允许重复放入

### 网格模型下的组合意义

- 二维网格 $(m,n)$ 中，对角移动有 $C^m_{m+n}$ 种方案
- **组合数对称公式**：$C^m_{m+n} = C^n_{m+n}$
- **组合数相加公式**：$C^r_n = C^r_{n-1} + C^{r-1}_{n-1}$
  - 分离法：先去除一个，分别作为已选和未选，对剩下n-1个进行组合
  - 网格法：同理
- **组合数累加公式**：$C^r_{n+r+1} = \sum\limits^r_{k=0} C_{n+k}^{k}$
  - 分离法：不含 $a_1$、含 $a_1$ 不含 $a_2$，含 $a_1,a_2$ 不含 $a_3$……
  - 网格法：
    - 左式是 $(0,0)$ 到 $(n+1,r)$ 的路径数
    - 右式是 $(0,0)$ 到 $(n,k)$ 的路径数
    - 第 $n$ 列中，到达右上角的路径只有一种（因为不能经过已计算的点）
- **组合数反位累加公式（朱世杰恒等式）**：$C^{r+1}_{n+1} = \sum\limits^{n-r}_{k=0} C^r_{r+k}$
  - 令 $s = n-r$，则 $C^{r+1}_{s+r+1} = \sum\limits^s_{k=1}C^{r}_{r+k}$
- **组合数乘法公式**：$C^l_nC^r_l = C^r_nC^{n-r}_{l-r}（l\geqslant r）$
  - 重复法：先取 $l$ 个组合，再从 $l$ 中取 $r$ 个
    - 它比直接取 $r$ 个多了重复数 $C_{n-r}^{l-r}$
    - 包含某个 $r$ 的所有 $l$ 都视为重复，等价于总体中剩下的 $n-r$ 个中取 $l-r$ 个的数量
- **二次反位乘法公式**：$C^r_{m+n} = \sum\limits^r_{k=1} C^k_mC^{r-k}_n$
  - 组合法：其实就是分步取组合
  - 网格法：
  - 二项式法：$(1+x)^m(1+x)^n = (1+x)^{m+n}$
- **二次正位乘法公式**：$C^m_{m+n} = \sum\limits^m_{k=0} C^k_{n}C^k_m$
  - 组合法：同上
  - 二项式法：$(1+x)^n(1+\frac{1}{x})^m = \frac{1}{x^m}(1+x)^{m+n}$
- **组合数二项式公式**：$\sum\limits^n_{k=0} C^k_n = 2^n$
  - 集合法：
    - 左式：n个元素填入集合的所有可能情况
    - 右式：遍历n个元素，选择是否填入集合，一次两种选择，共选择n次
  - 网格法
- **组合数奇偶公式**：奇数和等于偶数和
  - 二项式定理法：直接代入1和-1
  - 一一对应法：只看元素a，若含有a则为奇数，不含有a为偶数。而含有与不含有的组合数相等。再由a任意性即可

## 组合级数

### Wallis公式

- $\cfrac{\pi}{2} = \cfrac{2}{1}\cdot\cfrac{2}{3}\cdots\cfrac{2n}{2n-1}\cdot\cfrac{2n}{2n+1}$
  - **证明**：
  - 设级数 $\large I_n = \int^{\frac{\pi}{2}}_0 sin^nxdx$
    - 分部积分得递推公式 
      - $I_n = \frac{n-1}{n}I_{n-2}$
    - 从而通项公式
      - $I_n = \begin{cases} \large\frac{(k-1)!!}{k!!}\normalsize I_1，k为奇数 \\  \large\frac{(k-1)!!}{k!!}\normalsize I_0，k为偶数\end{cases}$
  - 夹逼法
    - $sin^nx$ 递减，从而 $I_{2n+1} < I_{2n} < I_{2n+1}$
    - 阶乘移项相消，得 $1< () < \frac{2n+1}{2n}$
    - 从而 $\large\frac{[(2n-1)!!]^2}{[(2n)!!]^2}\frac{1}{2n+1} \to \frac{\pi}{2} $
  - **推论**：
    - 改变移项方式，还可得其它几个 $\frac{\pi}{2}$ 的极限
    - 设级数 $p_n = 1-\frac{1}{(2n)^2}$
      - 部分积公式 $P_n = \large\frac{[(2n-1)!!]^2}{[(2n)!!]^2} \normalsize\cdot (2n+1)$
  <br>
      - 有关系 $\frac{\pi}{2}P_n = \frac{I_{2n}}{I_{2n+1}}$
  <br> 
      - 而 $I_n$ 收敛于1，所以 $\prod\limits^{\infty}_{n=1}(1-\frac{1}{(2n)^2}) = \frac{2}{\pi}$

### Stirling公式

- **相对误差**：$\lim\limits_{n\to\infty} \large\frac{\sqrt{2\pi n}(\frac{n}{e})^n}{n!}\normalsize = 1$
- **绝对误差**：$\lim\limits_{n\to\infty} \sqrt{2\pi n}(\frac{n}{e})^n - n! = \infty$
- 设级数
  - $A_n = \int^n_1 \ln xdx = n\ln n- n + 1$
    - 曲线 $y=\ln x$ 的积分面积
  - $t_n = \ln(n!) - \frac{1}{2}\ln n$
    - 连接 $y=\ln x$ 整数点的折线的面积（直角梯形面积级数）
  - $T_n = \ln(n!) - \frac{1}{2}\ln n + \frac{1}{8}$
    - 整数点切线直角梯形面积（直角梯形面积级数）
- 上下梯形夹逼，$t_n < A_n < T_n$
  - 设 $b_n = A_n - t_n$
    - $b_n = n\ln n -n+1 - \ln(n!) + \frac{1}{2}\ln n$
    - 变形得 $n! = e^{1-b_n}\sqrt{n}(\frac{n}{e})^n$
  - $0<b_n<\frac{1}{8}$，其为单调有界数列，存在极限
    - 则 $e^{1-b_n}$ 也有极限，代入Wallis公式，得极限为 $\sqrt{2\pi}$
    - 从而得到Stirling公式
