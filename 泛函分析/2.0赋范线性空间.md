## 抽象的线性空间

- **线性流形（仿射子空间）**：线性空间 $V$ 中，子空间 $E_0$ 沿某个向量 $x_0$ 平移后的集合
  - **符号**：$E = E_0 + x_0 = \{e+x_0 \mid e\in E_0\}$
  - **非线性**：线性流形对线性运算不封闭
    - **例子**：不过原点的平面不是线性子空间
  - **非完备性**：若 $x_0\notin V$，则 $E$ 不是线性子空间
  - <font color='chartreuse'>前面说了度量空间中并没有直线的概念，但通过定义线性流形，我们可以在线性空间中定义直线（即一维线性流形）
    - 如果能定义直线，那么也就能相应地定义集合、空间、泛函的凸性，从而对最佳逼近问题进行一些简化
    - 注意此时两点间的直线未必代表最短路径。这要求直线同时也是测地线才行</font>
- **线性包**：某个向量族的全体有穷线性组合结果

### 次线性泛函

- **准范数（准模）**：满足下列条件的泛函 $\|\cdot\| : \mathscr{X}\to \R$<font color='red'>（正定性、完备性、消负性、三角不等式）</font>
  - 正定性：$\|\t\| = 0，\|x\| > 0$
  - 系数与向量完备性：$\lim\limits_{\a_n\to 0}\|\a_n x\| = 0，\lim\limits_{\|x_n\|\to 0}\|\a x_n\| = 0$
    - <font color='chartreuse'>虽然这里只说明了 $0$ 处的极限，但由线性可得结论能拓展到整个空间。例如只需定义 $\beta_n = \a_n\pm \dfrac{A}{\|x\|}$ 即可表示任意极限值 $A$</font>
  - 消负性：$\|-x\| = \|x\|$
    - <font color='chartreuse'>齐次性的弱化版</font>
  - 三角不等式
- **范数（模）**：满足齐次性的准范数<font color='red'>（正定性、齐次性、三角不等式）</font>
  - <font color='chartreuse'>齐次性蕴含了完备性和消负性</font>
- **半范数（半模）**：正定性弱化为非负定性的范数<font color='red'>（非负定性、齐次性、三角不等式）</font>
  - 非负定性：$P(x)\geq 0$
  - <font color='chartreuse'>半范数和范数仅有非负定和正定的区别，这导致半范数的核不只有 $\t$</font>
- **次线性泛函**：线性空间到实数集的映射 $P:\mathscr{X}\to \reals$，满足<font color='red'>（正齐次性，三角不等式）</font>
  - 次可加性：$P(x+y) \leq P(x)+P(y)$
  - 正齐次性：$P(|\l| x) = |\l|P(x)\quad (\l\in\K)$
  - **实例**：
    - 范数是一种次线性泛函
    - 半范数是一种次线性泛函
  - **反例**：
    - 准范数不是次线性泛函

#### 关系

- 次线性泛函（次可加、正齐次） < 半范数（非负定、次可加、齐次） < 范数
  - <font color='chartreuse'>前者少个非负定性和齐次性，后者少个正定性</font>
- 准范数（正定、次可加、消负、完备） < 范数（正定、次可加、齐次、完备）
  - <font color='chartreuse'>少个齐次性</font>

### 准赋范线性空间

- **$F^*$ 空间**：具有准范数的线性空间
  - **度量性**：$F^*$ 空间是度量空间，具有准范数诱导的度量
    - **证明**：定义易得
  - **基本列**：依准范数收敛的点列，即 $\|x_n - x_m\|_F < \e$
- **Frechet空间**：完备的 $F^*$ 空间
  - **实例**：
    - 数列线性空间，准范数为 $\|x\| = \dis\sum\limits^\infty_{n=1} \frac{1}{2^n}\frac{|x_n|}{1+|x_n|}$
      - <font color='chartreuse'>本质是无穷维欧氏空间，其完备性也是由欧氏空间完备性传递的</font>
    - 连续函数线性空间 $C(\reals^n)$，准范数为 $\|u\| = \dis\sum\limits^\infty_{k=1} \frac{1}{2^k}\frac{\max\limits_{|x|\leq k}|u(x)|}{1 + \max\limits_{|x|\leq k}|u(x)|}$

### 赋范线性空间

- **$B^*$ 空间**：具有范数的线性空间
  - **度量性**：$B^*$ 空间是度量空间，具有范数诱导的度量
    - **证明**：定义易得
  - **基本列**：依范数收敛的点列，即 $\|x_n - x_m\| < \e$
- **Banach空间**：完备的赋范线性空间
  - **实例**：
    - 任意连续函数空间 $C(M)$，取无穷范数
    - 欧氏空间 $\R^n$，取2-范数（内积范数）
- **$p$ 次绝对可积函数空间 $L^p(\Omega,\mu)$**：
  - **定义**：测度空间 $(\Omega,\mathscr{B},\mu)$ 中，所有满足 $|u(x)|^p$ 勒贝格可积的可测函数 $u(x)$
  - **等价关系**：若两函数几乎处处相等，则认为它们等价
  - **范数**：$\|u\| = \dis\dkh{\int_\Omega |u(x)|^p du}^{\dfrac{1}{p}}$
  - **赋范线性**：
    - 线性封闭性：由实变函数知识，勒贝格积分具有线性
    - 正定性：显然
    - 齐次性：由实变函数知识，勒贝格积分具有线性
    - 三角不等式：由实变函数的Minkowski不等式即可
  - **完备性**：由实变函数的Riesz-Fisher定理即可
- **$p$ 次绝对可加数列空间 $\ell^p$**：
  - **定义**：在 $L^p$ 中，取 $\Omega = \natnums$，$\mu(\{n\}) = 1(\forall n\in \N)$，则所有满足 $\sum\limits^\infty_{n=1}|u_n|^p$ 收敛的数列构成 $\ell^p$
    - <font color='chartreuse'>当每个元素的测度均为 $1$（即在积分中的权重均为 $1$）时，数列求和可以看作数列在该测度下的积分</font>
  - **范数**：$\|u_n\| = \sqrt{\sum\limits^\infty_{n=1}|u_n|^2}$
- **本性有界可测函数空间 $L^\infty(\Omega,\mu)$**：
  - **定义**：测度空间 $(\Omega,\mathscr{B},\mu)$ 中，若测度 $\mu$ 对于 $\Omega$ 满足 $\sigma$ 有限，则所有本性有界可测函数 $u(x)$ 构成 $L^\infty$
  - **本性有界**：$\exists M >0$ 使得 $\mu\Big( \{x\in\O \mid |f(x)| \geq M\} \Big) = 0$
  - **范数（本性上确界）**：$\|u\| = \esup\limits_{x\in \Omega} |u(x)| =  \inf\limits_{\substack{\mu(E_0) = 0 \\ E_0 \subset \Omega}} \Big[\sup\limits_{x\in(\Omega-E_0)} |u(x)|\Big]$
    - <font color='chartreuse'>删去一个最大零测集 $E_0$ ，使得余集上 $|u(x)|$ 的上确界最小</font>
    - **等价命题**：$\inf\Big\{ a\geq 0: \mu\{|u(x)|>a\} = 0 \Big\}$
      - **本质**：找到某个界限，使得在界上的点总测度为0。再取这样界限的下确界
    - **单调性**：
      - 直接取包含序关系下的最大零测集即可，即取该a.e.有界函数的a.e.上确界
  - **赋范线性**
    - 正定性：由绝对值正定性易得
      - <font color='chartreuse'> $\|u(x)\| = 0$ 时，构造 $E_n$ 使得 $\sup< \varepsilon_n\to 0$，取并集即得 a.e. 为0 </font>
    - 齐次性：易得改变系数不改变函数绝对值的形状
    - 三角不等式：
      - 寻找 $E_0,E_1$，使得 $\sup\limits_{x\in \Omega-E_0} |u(x)| \leq \|u(x)\| + \frac{\varepsilon}{2}$
      - 再由 $\inf$ 最小性即得三角不等式
    - 线性：同上
  - **完备性**
    - **证明**：
      - 设集列 $Z_{n,p}$，在其余集上 $|u_{n+p}(x) - u_{n}(x)| \leq \|u_{n+p}-u_n\| + \frac{1}{2^{n+p}}$
      - 若 $u_n$ 是范数基本列，则取集列并集，可得余集上存在 $u$ 是$u_n$ 的欧氏极限
      - 那么由范数定义，易得 $u$ 也是 $u_n$ 的范数极限
- **有界数列空间 $\ell^\infty$**：
  - **定义**：取 $\Omega = \natnums$，$\mu(\{n\}) = 1(\forall n\in \N)$，则全体满足 $|u_n|\pad (\forall n\in \N)$ 有界的数列构成 $\ell^\infty$
  - **范数**：$\|u_n\| = \sup\limits_{n\geqslant 1} |u_n|$<font color='chartreuse'>（零测集只有空集）</font>
- **准Sobolev空间 $C^k(\overline{\Omega})$**
  - **定义**：（$\R^n$ 中有界连通开区域 $\Omega$ 的闭包）上（直到 $k$ 阶均连续可偏导）的全体函数（$k\in \natnums$）
  - **范数**：$\|u\| = \max\limits_{|\a|\leq k}\max\limits_{x\in\overline{\Omega}} |\partial^\a u(x)|$
    - 其中 $\begin{cases} \a = (\a_1,...,\a_n) \\ |\a| = \a_1+...+\a_n \\ \partial^\a u(x) = \cfrac{\partial^{|\a|}}{\partial x_1^{\a_1}...\partial x_n^{\a_n}}u(x)  \end{cases}$
    - **本质**：总阶数 $k$ 以下的所有类型偏导数的绝对值上确界
  - **赋范线性**
    - 正定性：绝对值正定性
      - <font color='chartreuse'>若 $\|u\|=0$，即k阶以下偏导数均为0，则取0阶导数即得 $u=0$</font>
    - 齐次性：偏导数齐次性 + 系数不改变函数形状
    - 三角不等式：函数三角不等式
      - <font color='chartreuse'>两函数分别取最大点，肯定比一块取最大点要大</font>
    - 线性：同上
  - **完备性**：
    - 若 $u_n$ 是基本列，则 $\partial^{|\a|}u_n(x) \to v_\a(x)$ 一致收敛
    - 此时只需证 $v_\a = \partial^\a v_{\theta}$ 即可
    - 归纳法：
      - $v_{(1,0,...,0)} = \pfrac{}{x_1} v_{(0,0,...,0)}$，由基本列必一致收敛，其易得
      - 然后归纳到所有 $\a_i$ 即可
- **$H^{m,p}(\Omega)$（Sobolev空间）**：
  - **定义**：
    - 设 $\R^n$ 中有界连通开区域 $\Omega$，$m\in \natnums，p\in [1,\infty)$
    - $C^m(\overline{\Omega})$ 中范数 $\|u\|_{m,p} = \Big( \sum\limits_{|\a|\leq m} \int_{\Omega}|\partial^\a u(x)|^pdx \Big)^{\large\frac{1}{p}}$
    - 然后将范数有界的子集完备化，即得Sobolev空间
- **连续函数空间**：$C(M)$ 中，定义上界范数 $\|x\| = \sup\limits_{t\in M} |x(t)|$

#### 小总结

- 证明完备性的方法：取范数基本列，得其欧氏空间完备，再传递到范数完备

### 习题

- **球延拓定理**：若某个与线性相容的性质在 $B^*$ 空间的单位球上成立，则在全体空间上也成立
  - **反例**：
    - 由于单位球的有界性，故与度量相关的性质无法拓展
    - 由于单位球的非紧性，故与紧集相关的性质无法拓展
- **不完备的子空间**：
  - 有限维空间均完备
  - 无穷维空间中
    - $C[0,1]$ 的子空间 $P[0,1]$ 不完备
      - **证明**：由W定理，任意连续函数可被多项式逼近。设 $e^x = \lim\limits_{n\to\infty} p_n$，则显然此时的 $p_n$ 是基本列，但极限不封闭
    - $\ell^p$ 的有限支子空间不完备
      - **证明**：详见度量空间部分
- **积空间完备性传递**：
  - 设
    - $\funcx = \funcx_1\times\funcx_2$ 是两个 $B^*$ 空间的积
    - 定义范数 $\|x\| = \max(\|x_1\|_1,\|x_2\|_2)$
  - 若两空间完备，则积空间也完备
  - **证明**：
    - 设 $\funcx$ 中基本列为 $\{\bd x_n\}\to \bd x$
    - 由基本列，$\|\bd x_m-\bd x_n\| = \max(\|x_{1m}-x_{1n}\|_1, \|x_{2m}-x_{2n}\|_2)\leq \varepsilon$
    - 从而 $\{x_{1n}\}，\{x_{2n}\}$ 也是基本列
    - 由因式空间完备性，极限 $x_1\in\funcx_1，x_2\in \funcx_2$，从而 $x_1\times x_2 = \bd x\in \funcx$
- **类Cauchy收敛定理**：$B^*$ 空间完备 $\LR \forall \{x_n\}$，若 $\sum\limits^\infty_{n=1} \|x_n\| < \infty$，则 $\sum\limits^\infty_{n=1} x_n$ 收敛
  - **证明**：
    - **必要性**：
      - 由数分知识，范数列收敛可得 $\|x_n\|\to 0$，从而是基本列
      - 由完备性，极限存在，且 $\|x\| = 0$。由正定性得极限为 $\t$
      - 由三角不等式， $\|\sum\limits^\infty_{n=1} x_n\| \leq \sum\limits^\infty_{n=1} \|x_n\|$
    - **充分性**：
      - 设 $\{x_n\}$ 是基本列，只需证明存在收敛子列。
      - 由基本性，存在子列满足 $\|x_{n_k}-x_{n_{k+1}}\| < \dfrac{1}{2^k}$ 
      - 写为累减形式：$y_k = y_1 + \sum\limits^k_{i=1} (y_{i+1}-y_i)$，其范数级数收敛，从而 $\{y_k\}$ 收敛
    - **理解**：点列只要收敛，则极限必定存在于空间中。基本列只满足了各项间差的极小性，并不满足到某点的收敛性
    - **本质**：作差级数而已

## 范数的等价性

- **范数强弱**：若对任意点列 $\{x_n\}$，其在范数1下收敛 $\Rightarrow$ 在范数2下收敛，则称范数1强于范数2
  - <font color='chartreuse'>本质上还是对阶进行比较</font>
- **等价范数**：互为强范数
- **范数比较定理**：范数1强于2 $\LR \exist C>0$，使得 $\forall x，\|x\|_2 \leq C\|x\|_1$
  - **证明**：
    - **充分性**：定义易得
    - **必要性**：
      - 反设不成立，则对于任意 $x_n$，都存在 $C_n\to +\infty$ 使得 $\|x_n\|_2 \leq C_n\|x_n\|_1$
      - 设2-范数下单位向量 $y_n = \cfrac{x_n}{\|x_n\|_2}$，即 $\|y_n\|_2 = 1$
        - 再由C无界性，此时 $\|y_n\|_1 = \cfrac{\|x_n\|_1}{\|x_n\|_2} < \cfrac{1}{C_n}\quad (\forall n\in \natnums)$
        - 从而 $\|y_n\|_1\to 0$，再由1比2强，其在范数2下也趋于 $0$，与其单位向量的定义矛盾
  - **推论（范数等价比较定理）**：详见数值线性代数-矩阵分析部分
- **坐标空间 $\K^n$**：赋范线性空间 $\funcx$ 中某个基 $\{e_i\}$ 下的坐标构成的空间
  - **坐标映射** $T(x) = (\xi_1,...,\xi_n)$
  - **$\K^n$ 范数（数组范数）**：取坐标后的2-范数
  - **同构性**：任意 $\K^n$ 均同构于欧氏空间 $\R^n$
- **范数相容定理**：有穷维线性空间下的范数均与 $\K^n$ 范数等价
  - **证明**：
    - 设范数映射 $p: \K^n\to \R，\xi \mapsto \|\sum\limits^n_{j=1}\xi_je_j\|$
      - <font color='chartreuse'>它将向量坐标映射成 $\funcx$ 的向量范数</font>
    - **连续性**：由三角不等式 + C-S不等式易得 $p$ 是连续映射
    - **最值性**：再由单位球面 $S$ 是紧集，得 $S$ 上 $p$ 有最值，由正定性得最值为正
    - **范数等价性**
      - 任取 $\xi\in \K^n$，由范数齐次性得 $p(\xi) = |\xi|p(\dfrac{\xi}{|\xi|})$
      - 对 $p$ 的最大值、最小值不等式变形，即可得 $\min p\cdot |\xi| \leq p(\xi) \leq \max p\cdot |\xi|$
  - **本质**：类似矩阵范数的证明……
  - **推论**：
    - 两个维数相同的 $B^*$ 空间，代数同构，拓扑同胚
      - **证明**：
        - 代数同构性在高等代数中已证
        - 拓扑同胚性由范数相容定理得所有范数彼此等价，再由度量拓扑的性质即可得到
    - 有穷维 $B^*$ 空间是 $B$ 空间
      - **证明**：已知其同构于 $\R^n$，再由坐标映射连续性，得其可传递完备性
    - $B^*$ 的有穷维子空间是闭子空间
      - **证明**：该子空间为 $B$ 空间，即满足完备性，从而由完备子空间的闭集性即得结论
- **次线性泛函相容定理**：有穷维 $B^*$ 空间上的次线性泛函，若满足正定性，则其等价于范数
  - **证明**：
    - 只需证明 $C_1\|x\| \leq P(x) \leq C_2\|x\|\quad (\forall x\in \funcx)$
    - 仿照上面范数相容定理进行证明即可
  - **理解**：有穷维得一致连续性，正定性得最值非零，正齐次性即可变形最值不等式

## 商空间

- **商空间**：$\funcx/\funcx_0$ 表示 $\funcx$ 中子空间 $\funcx_0$ 的所有等价类组成的空间
- **差等价商空间**：$B^*$ 空间中，设 $(x-x')\in \funcx_0$ 是等价关系
  - **等价类** $[x]$
    - **线性**：定义加法和数乘
    - **几何意义**：平面 $\funcx_0(d)$ 可平移。若给定其到原点的距离 $d$，则端点在该平面上的向量等价
      - 加法表示 $d$ 相加，数乘表示 $d$ 伸缩
      - 减法：对于两个平面 $d_1<d_2$，将 $d_1$ 平面上某点改为原点，指向 $d_2$ 的向量即为差向量
      - 不同的 $\funcx_0$ 对应不同方向的平面
  - **商范数** $\|[x]\|_0 = \inf\limits_{y\in [x]} \|y\|$
    - **几何意义**：等价类平面到原点的距离 $d$（垂直向量）
- **商空间传递性**：$(\funcx/\funcx_0,\|\cdot\|_0)$ 的赋范线性和完备性均可继承于 $(\funcx,\|\cdot\|)$
  - **证明**：
    - **赋范线性**：
      - 正定性、齐次性易得
      - 三角不等式：在等价类中取原范数趋向于下确界的点列 $\begin{cases} \|x_n\|\to \|[x]\|_0 \\ \|y_n\|\to \|[y]\|_0 \end{cases}$
    - **零元素**：设 $\|[x]\|_0 = 0$，原空间中可取 $x_n\in [x]$，且 $\|x_n\| \to 0$
      - 由闭集完备性，$x = \lim\limits_{n\to\infty} (x-x_n) \in \funcx_0$，此时 $[x]$ 即为商空间中零元素
    - **完备性**：
      - 设 $\{[x_n]\}$ 是基本列，则可取子列使 $\|[x_n-x_{n+1}]\|_0 \leq \frac{1}{2^{n+1}}$
        - <font color='chartreuse'>平面间距越来越小</font>
      - 存在 $y_{n,n+1}\in[x_n-x_{n+1}]$ 满足范数 $\leq \|[x_n-x_{n+1}]\|_0 + \frac{1}{2^{n+1}} \leq \frac{1}{2^n}$
        - <font color='chartreuse'>平面间距缩小的同时，间距空间中逐渐逼近垂直向量的斜向量</font>
      - 再设 $\begin{cases} y_1 = x_1 \\ y_{n+1} = y_n-y_{n,n+1} \end{cases}$
        - <font color='chartreuse'>$y_n-y_{n+1} = y_{n,n+1}$，即任选一个初始向量的基础上，每次都增加一个逐渐逼近垂直向量的斜向量</font>
      - 由三角不等式，$\|y_n-y_{n+p}\| = \sum\limits^p_{k=1} \|y_{n+k-1,n+k}\| \leq \frac{1}{2^n}$，即其为基本列
      - 由原空间完备性，$y_n$ 存在极限 $y$
      - 由 $[y_{n+1}] = [x_{n+1}]$，$[y]$ 即为 $[x_n]$ 的极限
    - **理解**：取垂直向量列 $d_{n,n+1}$ 不好吗？搞不懂……

### 习题

- **Lipschitz空间**
- **商空间**：$\funcx/\funcx_0$ 中
  - $\forall x\in[x]$，有 $\inf\limits_{z\in\funcx_0} \|x-z\| = \|[x]\|_0$
    - **证明**：互包即可
      - 由于 $(x-z)\in [x]$，故由商范数下确界性，左 $\leq$ 右
      - 再由 $(x-[x])\in\funcx_0$，故由左式下确界性，右 $\leq$ 左
    - **理解**：垂直向量即为原点最小逼近元
  - $\varphi: \funcx\to\funcx/\funcx_0，x\mapsto [x] \color{red} = x+\funcx_0$ 是连续线性映射
    - **证明**：$\forall x\in\funcx，\exist d\perp\funcx_0，\varphi(x) = x+d+\funcx_0$
      - **连续性**：设 $x_n\to x$，则相应存在 $d_n\to 0$，故 $\p(x_n)\to x+\funcx_0$
      - **线性**：易得 $\p(x+y) = (x+d_x)+(y+d_y)+\funcx_0 = \p(x)+\p(y)$
    - **理解**：商映射是非单射，将同一平面内的点均映射成该平面
  - $\forall [x]\in\funcx/\funcx_0，\exist x\in\funcx$，使得
    - $\varphi(x) = [x]$，且 $\|x\| \leq 2\|[x]\|_0$
    - **证明**：由下确界最大性易得结论
    - **理解**：很无聊
