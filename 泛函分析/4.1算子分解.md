# 算子的分解

## 正交投影算子

- **正交投影算子**：
  - **定义**：
    - 设 $\ms X$ 是Hilbert空间， $M$ 是闭线性子空间
    - 取正交分解 $x = y+z\pad (y\in M)$，则 $P_M: \ms X\to M，x\mapsto y$ 称为正交投影算子
  - **线性**：易得
  - **连续性**：易得
  - **单位性**：$\forall M\neq \{\t\}，\|P\| = 1$

### 正交投影算子的性质

- 设 $L,M$ 是Hilbert空间上的闭线性子空间，$P_M$ 表示 $M$ 上的正交投影算子
- **单性质**：$P$ 是有界线性算子，则 $P$ 是正交投影 $\LR P$ 满足幂等性和自共轭性
  - **必要性**：
    - **幂等性**：$P^2_M = P_M$
      - **证明**：易得 $P_M|_M = I$，从而性质易得
    - **自共轭性**：$(P_Mx,y) = (x,P_My)$
      - **证明**：把左边的 $y$，右边的 $x$ 分解成关于 $M$ 正交投影即可
  - **充分性**：
    - **闭线性子空间**：
      - 由幂等性，$\forall y\in M，(I-P)y = Px-P^2x = 0$，从而 $M\subset N(I-P)$
      - 再由 $\forall y\in N(I-P)$，$(I-P)y = 0$ 得  $y = Py\in M$
      - 从而 $M = N(I-P)$。再由线性有界性，核为闭线性子空间
    - **正交投影性**：$x = Px + (I-P)x$
  - **理解**：利用了 $R(I-P_M) = M^\perp$ 进行构造
- **双性质**：
  - **正交性**：$L\perp M\LR P_LP_M = 0$
    - **必要性**：由自共轭性，$(P_LP_Mx,y) = (P_Mx,P_Ly) = 0$
    - **充分性**：由正交分解定理 + 自共轭性直得 $(P_Mx,P_Ly) = (P_LP_Mx,y) = 0$
  - **正交补性**：$L = M^\perp \LR P_L + P_M = I$
    - **必要性**：
    - **充分性**：
  - **可交换性**：$P_LP_M = P_{L\cap M} \LR P_LP_M = P_MP_L$
    - **必要性**：
    - **充分性**：
- **映射性质**：
  - 正交投影算子是弱压缩映射
    - **证明**：
      - 由勾股定理易得 $\|P_Mx-P_My\| \leq \|x-y\|$

## 连续线性算子的正交分解

### 连续线性泛函的内积表示

- **Hilbert空间的Riesz表示定理**：
  - 设 $f$ 是Hilbert空间 $\ms X$ 上的连续线性泛函
  - 则存在唯一的 $y_f\in \ms X$，使得 $\forall x\in \ms X，f(x) = (x,y_f)$
  - **证明**：
    - 不妨设 $f\neq 0$
    - **存在性**：
      - 由连续线性得 $\ker f$ 为真闭子空间
      - 由正交分解定理，存在单位向量 $x_0\perp \ker f$。正交分解式为 $$\forall x\in \ms X，\exists \a\in\K，y\in\ker f，使得 x = \a x_0 + y$$
        - <font color='chartreuse'>由 $y\in \ker f$ 可得 $f(y) = 0$</font>
        - 正交分解式两边作用 $f$，计算得 $\a = \dfrac{f(x)}{f(x_0)}$
        - <font color='chartreuse'>由 $x_0\perp M$ 可得 $(y,x_0) = 0$</font>
        - 正交分解式两边对 $x_0$ 做内积，计算得 $\a = (x,x_0)$
      - 综上，$f(x) = \a f(x_0) = (x,\overline{f(x_0)}x_0)$，取 $y_f = \overline{f(x_0)}x_0$ 即可
    - **唯一性**：反设易得
  - **理解**：
    - $y_f$ 是核的法向量方向上，长度等于（单位向量像的长度）的向量
    - $\a$ 是核的正交分解式中，法向量方向分量的长度
  - **几何理解**：
    - 任意向量均可被正交分解为 （核上的向量）+（垂直于核的向量）
      - 核上向量的像为0，故只需要考虑垂直于核的向量
      - 内积具有投影性，故原向量 $x$ 与单位法向量 $y_f$ 的内积即为 $f$ 的值
  - **方程理解**：
    - 连续线性泛函的等值面是线性泛函方程组的解（解空间是平行于核的超平面）
    - 从而每个向量的泛函值仅由垂直于等值面的分量决定
  - **本质**：
    - 利用 $\ker f$ 将Hilbert空间切割成两个部分，从而
    - 打通了内积代表的几何意义和泛函代表的代数意义的关系
    - 类似高等代数中，内积的同构表示
  - **实例**：
    - 在 $\R^3$ 中，R表示式为 $f(\bold x) = ax + by + cz = (\vec{n},x)$
  - **推论**：
    - （算子范数）$\|f\| = \|y_f\|$（内积范数）
      - **证明**：由R表示式直得结论
      - **理解**：
        - $x_0$ 方向是 $f$ 像最大的方向（因为任何其它方向的向量都可被分解一部分到 $M$ 中，而 $M$ 的像为0）

### 共轭双线性函数的内积表示

- **内积表示定理**：
  - 设 $\ms X$ 是Hilbert空间，$a(x,y)$ 是 $\ms X$ 上的共轭双线性函数
    - <font color='chartreuse'>不一定满足正定性、交换性，故不是内积</font>
  - 若 $\exist M>0$，使得 $a(x,y) \leqslant M\|x\|\|y\|$
    - <font color='chartreuse'>满足弱形式的C-S不等式</font>
  - 则存在唯一的 $A\in \op(\ms X,\ms X)$，满足
    - $a(x,y) = (x,Ay)$
    - $\|A\| = \sup\limits_{\substack{(x,y)\in \ms X\times \ms X \\ x,y\neq \theta}} \cfrac{|a(x,y)|}{\|x\|\|y\|}$
  - **证明**：
    - 固定 $y\in \ms X$，则 $a$ 退化为连续线性泛函 $f_y$
    - 由R表示定理，$\forall x\in \ms X$，存在唯一的 $z\perp \ker f_y$，使得 $a(x,y) = (x,z)$
      - <font color='chartreuse'>此时的 $z$ 和 $y$ 是一一对应关系</font>
    - 设 $A: y\mapsto z$，则 $\forall x,y\in \ms X，a(x,y) = (x,Ay)$
      - **双射性**：由定义易得
      - **线性**：
        - $\Big( x,A(\l y_1+\mu y_2) \Big) = a(x,\l y_1+\mu y_2) = \l a(x,y_1) + \mu a(x,y_2) \\ = \l (x,Ay_1) + \mu (x,Ay_2)$
        - 再由 $A$ 的双射性，即可得线性
      - **有界性**：
        - $\|Ay\| = \|z\| = \sup\limits_{x\in \ms X\j\{\t\}} \cfrac{|a(x,y)|}{\|x\|} \leq M\|y\|$，从而得有界性
          - <font color='chartreuse'>在R表示式中 $a(x,y) = (x,z)$ 中，取与 $z$ 共线的 $x$，即得上面的分式
          - 而显然这样的 $x$ 会使内积最大，从而也使 $a(x,y)$ 最大。再由共轭双线性，也可使该分式最大</font>
      - **算子范数**：由范数齐次性易得结论
  - **本质**：
    - Riesz表示定理中揭示了 $f$ 与 $y_f$ 的一一对应关系。本定理则表明该双射还具有线性和连续性
    - 双线性函数只不过是为了揭示这个对应关系的必要工具罢了（将 $y$ 固定后，$a$ 就变成了 $f_y$。也就是说存在一个 $f_y$ 和 $y$ 的双射）

### 习题

- **正交投影表出性**：
  - 设
    - $f_1,...,f_n$ 是Hilbert空间上的有界线性泛函
    - $M = \mathop{\bigcup}\limits^n_{k=1} \ker f_k$
    - $y_0$ 是 $x_0$ 在 $M$ 上的正交投影
  - 则
    - $\exist \{y_k\}^n_{k=1} \in (\ker f_k)^\perp$，$\{\a_k\}^n_{k=1}\in\K$，使得 $y_0 = x_0 - \sum\limits^n_{k=1} \a_ky_k$
  - **证明**：
    - 由正交分解定理，存在 $z_0 \in M^\perp$ 使得 $x_0 = y_0 + z_0$
    - 易得只需证 $z_0$ 可被 $(\ker f_k)^\perp$ 线性表出即可
      - 已知Hilbert空间上存在正交规范基 $\{z_k\}$，使得 $x = \sum\limits^n_{k=1} (x,z_k)z_k$
      - 由 $(P^\perp)^\perp = \ol{\span P}$ 得 $\ol{\span\{z_k\}} = M^\perp$，证毕

### Laplace方程

- **特殊的Dirichlet边值问题**
  - 设 $\Omega\subset \reals^n$ 是一个有界开区域，$f\in L^2(\Omega)$
  - **边值问题**：$\begin{cases} -\Delta u = f \\ u|_{\partial\Omega} = 0 \end{cases}$
  - **弱解**：若 $u\in H^1_0(\Omega)$，满足 $\forall v\in H^1_0(\Omega)，\dis\int_\Omega \nabla u\cdot \nabla vdx = \int_\Omega fvdx$，则称 $u$ 为边值问题的弱解
    - **证明**：Green公式
  - <font color='chartreuse'>Hamilton算子：$\nabla = i\dpfrac{}{ x} + j\dpfrac{}{ y} + k\dpfrac{}{z}$
  - Laplace算子：$\Delta = \nabla^2 = (\dpfrac{}{x})^2 + (\dpfrac{}{y})^2 + (\dpfrac{}{z})^2$</font>
- **弱解存在和唯一定理**：$\forall f\in L^2(\Omega)$，
  - **证明**：
    - **存在性**：由庞加莱不等式，可定义 $H^1_0(\Omega)$ 上的内积 $(u,v)_1 = \int_\Omega \nabla u\cdot \nabla vdx$
      - 再由C-S不等式，$|\int_\Omega f\cdot vdx| \leq C\|f\|\cdot \|v\|_1$
      - 从而 $g:H^1_0(\Omega)\to \reals， v\mapsto \int_\Omega f\cdot vdx$ 是连续线性泛函
      - 由R表示定理，$\exist u_0$ 使得 $g(v) = (u_0,v)_1$，其即为弱解
    - **唯一性**：反设 + 内积双线性直得

### 变分不等式

- **变分不等式**：
- **有解定理**：
  - 设 $C$ 是 $H^1_0(\Omega)$ 中的闭凸子集
  - 若 $f\in L^2(\Omega)$
  - 则 下列不等式存在唯一解 $u_0^*\in C$
    - $\int_\Omega \nabla u_0^*\cdot \nabla(v-u_0^*)dx \geqslant \int_\Omega f\cdot (v-u_0^*)dx\quad (\forall v\in C)$
  - **证明**：
    - 由弱解存在定理，该式可化为 $(u_0^*-u_0,v-u_0^*)_1\geq 0\quad (\for v\in C)$
    - 等价于 $u_0^*$ 是 $u_0$ 在 $C$ 上的最佳逼近元，从而唯一
  - **推论（矩阵形式）**
- **障碍问题**：已知 $\psi(x)\in C(\ol \Omega)$， 当 $C = \{v(x)\in H^1_0(\Omega)\mid v(x)\leq \psi(x)\}$
  - **物理意义**：u是薄膜位移，f是外力，$\psi$ 是障碍

### 测度论

- **Radon-Nikodym定理**：设 $(\Omega,\mathcal{B},\mu), (\Omega,\mc B, \nu)$ 是两个 $\sigma$-有限测度
  - 若 $\nu$ 关于 $\mu$ 绝对连续<font color='chartreuse'>（$\forall E\in \mc B，\mu(E)=0 \red\Rightarrow \nu(E) = 0$）</font>
  - 则 存在关于 $\mu$ 的可测函数 $g$，满足：
    - $g(x)\geqslant 0\quad a.e.\ \mu$
    - $\forall E\in \mc B，\nu(E) = \int_E g(x)d\mu$
  - **证明**：
    - $\bs{\mu < \infty}$ 时
      - $L^2(\Omega,\mu+\nu)$，范数为 $\|u\|^2 = \int_\Omega u^2(x)d(\mu+\nu)$
        - 设 $l(u) = \int_\Omega ud\mu$，由C-S不等式，$\|l(u)\| \leq \mu(\Omega)^{\dfrac{1}{2}}\|u\|$，即 $u$ 对于 $\mu$ 的积分有界
      - 由R表示定理，$\exist v$，使得 $\int_\Omega u\cdot d\mu = \int_\Omega uv\cdot d(\mu+\nu)$
        - 从而得**测度转化公式** $\dis\int_\Omega u(1-v)\cdot d\mu = \int_\Omega uv\cdot d\nu$
      - $\bs{0<v(x)\leq 1}\quad a.e.\ \mu$
        - 设 $F = \{x\in\Omega\mid v(x)\leq 0\}$
        - 取 $u(x) = \chi_F(x)$ 代入测度转化公式
        - 得 $\mu(F) = \int_F vd(\mu+\nu)\leq 0$，即 $\mu(F) = 0$
        - 同理得另一端结论
      - 设 $g(x) = \cfrac{1-v(x)}{v(x)} \geq 0$，其关于 $\mu$ 可测
        - 取 $u(x) = \cfrac{\chi_E(x)}{v(x)+\frac{1}{n}}$ 代入测度转化公式
        - 由 $\nu$ 绝对连续 + $\nu > 0\ae \mu$，得 $\nu > 0\ae \nu$
        - 则由单调收敛定理，$n\to\infty$ 时即得结论
    - $\bs{\mu(\Omega) = \infty}$ 时
      - 由 $\sigma$ 有限性，取单增空间列 $\{\Omega_n\}$，极限为 $\Omega$
        - 由有界时成立，$E\subset \Omega，\exist g_n，\nu(E\cap\Omega_n) = \int_{E\cap\Omega_n} g_nd\mu$
        - $g_n(x) = g_{n+1}(x)$，极限为 $g(x)$
        - 由单调收敛性得 $\nu(E) = \lim\limits_{n\to\infty} \int_{\Omega_n\cap E}d\nu  = \int_E g\cdot d\mu$
  - **理解**：
    - 总空间有界时，由R表示定理得测度转化公式，从而可构造结论
    - 总空间无界时，取有界空间列逼近即可
  - **本质**：测度转化定理。绝对连续关系下，一个测度是另一个的积分函数
