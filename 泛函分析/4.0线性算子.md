# 线性算子

- 本章思路：介绍线性算子的一些非常好的性质
  - 一般的算子：闭集上，连续性强于有界性
    - 但线性算子两者等价，故大多用有界性证连续性
  - 一般的算子：开集上，连续性和有界性无关
    - 但线性算子可用连续性得正逆算子有界等价性，故可得范数等价性，从而推出一系列有界定理

## 线性算子

- **线性算子**：具有线性的算子
  - **定义域**：线性算子的定义域必须是子空间
    - **证明**：
      - 如果定义域不满足线性封闭性，则 $T(ax+by) = aTx+bTy$ 中，$T(ax+by)$ 无法被定义，从而不可能是线性算子
  - **实例**：
    - **实向量算子** $\ms A: \R^n\to \R^m，\tvec{x_1 \\ \vdots \\ x_n}_{n\times 1} \mapsto \tvec{\sum\limits^n_{j=1}t_{1j}x_j \\ \vdots \\ \sum\limits^n_{j=1}t_{mj}x_j}_{m\times 1}$
      - <font color='chartreuse'>其实向量也是一种函数
        - 首先，它同构于线性函数空间
        - 其次，它可以看作将n个数变为向量的多元映射</font>
    - **微分算子** $\pa_x: C^\infty(\overline{\Omega})\to C^\infty(\overline{\Omega})，u\mapsto \dpfrac{u}{x}$
      - <font color='chartreuse'>在数分中已经学过，微分运算本质也是微分向量空间中的线性映射</font>
      <!-- - **微分算子的多项式** $P(\partial_x) = \sum\limits_{|\a|\leqslant m} a_\a(x)\partial^\a_x$ -->
    - **傅里叶变换算子** $F: L^1(\R)\to L^\infty(\R)，u\mapsto \dis\int_\R e^{ix\cdot t}u(x)dx$
      - <font color='chartreuse'>由傅立叶变换的线性得其为线性算子</font>
- **线性泛函**：
  - **实例**：
    - **多元实值函数**
    - **区域积分** $I: C(\overline{\Omega})\to \R，x \mapsto \dis\int_\Omega x(\xi)d\xi$
    - **区域微分** $D_{\xi_0}: C^\infty(\Omega)\to \R，u\mapsto \partial^\a u(\xi_0)$
      - <font color='chartreuse'>先变为高阶偏导函数，再取某导函数的某个值</font>
  - **反例**：
    - $x\mapsto \dis\int_\O x^2(\xi)d\xi$ 是泛函，但不是线性泛函

## 连续性与有界性

- **连续线性算子**：
  - 设 $\ms X,\ms Y$ 是 $F^*$ 空间，$T:\ms X\to \ms Y$ 是线性算子
    - <font color='chartreuse'>在 $F^*$ 空间中，即可定义连续性（连续性不依赖于具体的范数）</font>
  - 若对任意的定义域中点列 $\{x_n\}$，都有 $x_n\to x_0 \red\Rt Tx_n\to Tx_0$，则称 $T$ 是连续的
    - <font color='chartreuse'>由于 $F^*$ 空间没有完备性，故不能使用数分的方法来直接定义连续性
    - 但是还有一个定义连续的方法：Heine定理。通过它可以在任意度量空间中定义连续性</font>
- **连续延拓定理**：线性算子在定义域内处处连续 $\LR$ 在零向量处连续
  - <font color='chartreuse'>任意一点的连续线性算子都可以无限延拓</font>
  - **证明**：
    - 只需证明充分性即可
    - 设 $y_n = x_n-x_0$，显然有 $y_n \to \t$，从而由零向量处连续得 $Ty_n \to \t$
    - 再由算子线性得 $T(x_n-x_0) = Tx_n-Tx_0$，即 $Tx_n\to Tx_0$，从而 $T$ 连续
  - **本质**：
    - 线性是个很好的性质，它决定了整个对象的结构都是相似的、恒定的
    - 所以一些性质若在一点满足，则为了满足对象的线性结构，它就必须在整个对象上都满足
- **有界线性算子**：
  - 设 $\ms X,\ms Y$ 是 $B^*$ 空间，$T:\ms X\to \ms Y$ 是线性算子
  - 若满足 $\forall x\in\ms X$，都存在 $M\geqslant 0$ 使得 $\|Tx\|_\ms Y \leqslant M\|x\|_\ms X$，则称 $T$ 有界
  - <font color='chartreuse'>有界性依赖于范数，所以有齐次性</font>
- **连续有界等价定理**：$B^*$ 空间上，线性算子连续性等价于有界性
  - **证明**：
    - 由收敛列有界性，易得 保收敛性 $\Rightarrow$ 保有界性
    - 由有界性得压缩性，从而 保收敛性 $\Leftarrow$ 保有界性
    - <font color='chartreuse'>对于所有的线性算子，我们都可以认为连续性等价于有界性</font>
  - **本质**：
- **有穷连续性**：定义在有穷维 $B^*$ 空间上的线性算子均连续
  - **证明**：
    - 由有穷维线性空间同构于 $\K^n$，得此时可用矩阵表示线性算子
    - 再由有穷维线性空间的范数等价性，只需证明矩阵范数的连续性即可（详细证明见数值线性代数-矩阵分析）
  - **理解**：线性本来就连续，重要的是空间性质要好。而有穷维 $B^*$ 空间具有完备性，满足要求

### 习题

- **连续延拓定理**：
  - 设 $\ms X$ 是 $B^*$ 空间，$\ms Y$ 是 $B$ 空间，$T:\ms X\to\ms Y$ 是连续线性算子
  - 则 $T$ 能唯一地连续延拓到 $\overline{D(T)}$ 上，且算子范数相等
  - <font color='chartreuse'>它是闭图像定理的引理</font>
  - **构造性证明**：
    - 由闭包完备性，$\forall x\in\overline{D(T)}，\exist \{x_n\}\in D(T)$ 使得 $x_n\to x$
      - <font color='chartreuse'>集合是其闭包的稠密子集</font>
    - 由 $T$ 有界性 + $\ms Y$ 完备性可得 $Tx_n\to y$
      - <font color='chartreuse'>此时 $\|Tx_{n+p}-Tx_n\| \leqslant M\|x_{n+p}-x_n\|$，即其为基本列</font>
    - 从而可定义 $T_1:\overline{D(T)}\to \ms Y，x\mapsto y$
      - 易得是连续延拓，且范数相等，具有唯一性
  - **理解**：
    - 用像的点列定义的延拓算子，由像空间完备性 + 闭包完备性得连续性可以传递到闭包上
    - 再由点列极限的趋近性，用该法新定义的点，其上的像值趋近于 $D(T)$ 上的某点列。从而上确界相同，即算子范数相同
    - 再由极限唯一性得延拓唯一性
  - **本质**：依赖于像空间完备性所定义的延拓所具有的性质

### 反例（不连续的线性算子）

- **无界算子**
  - 设 $X$ 是无限维赋范线性空间，$\{e_i\}_{\in I}$ 是Hamel基，则线性泛函 $f(e_i) = i\|e_i\|$ 不连续
    - **证明**：
      - 取一列发散到无穷远处的 $e_i$ 即可
- **无限维空间中的微分算子**：
  - 在 $C^\infty[0,1]$ 中，取上确界范数，则微分算子不连续
    - **证明**：
      - 设 $x_n = \dfrac{\sin(nx)}{n}$，微分算子对其不满足保收敛性
- **非一致收敛空间的积分算子**：
  - 在 $C[0,1]$ 中，取 $L^1$ 范数，则取值泛函 $\d_a: f\mapsto f(a)$ 不连续
    - **证明**：
      - 构造尖峰函数列 $f_n(x) = n\max\{1-n|x-a|,0\}$，取值泛函对其不满足保收敛性

    - **证明**：
      - 构造仅第 $n$ 项不为 $0$ 的序列 $\{e_n\}$，则单位球上存在不收敛列

## 算子空间

- **有界线性算子空间 $\op(\ms X,\ms Y)$**：全体 $\ms X\to\ms Y$ 的有界线性算子，且 $D(T) = \ms X$
  - <font color='red'>因为要定义算子范数时，最好将单位球包含在定义域内，所以让定义域是全空间</font>
  - **算子范数**：$\|T\| = \sup\limits_{x\in \ms X\j\{\the\}} \cfrac{\|Tx\|}{\|x\|} = \sup\limits_{\|x\| = 1} \|Tx\|$<font color='chartreuse'>（由范数的齐次性容易证明）</font>
    - **几何意义**：单位向量的最大像
      - <font color='chartreuse'>它表示算子延展向量的能力大小</font>
    - **代数意义**：（算子延展后的范数）除去（原像本身的范数加成）
      - <font color='chartreuse'>有界算子的范数必定有界</font>
- **有界线性泛函空间 $\op(\ms X,\K)$**：同上
  - **对偶性**：本空间也可写作 $\ms X^*$，即 $\ms X$ 的对偶空间
- **完备性定理**：
  - 设 $\ms X$ 是 $B^*$ 空间，$\ms Y$ 是 $B$ 空间
  - 若在 $\opxy$ 上定义线性运算 $(aT_1 + bT_2)x = aT_1x + aT_2x$
  - 则此时 $\opxy$ 在算子范数 $\|T\|$ 下构成Banach空间
  - **证明**：
    - **线性空间**：由连续性的线性传递性易得结论
    - **范数**：三大性质均可由 $\ms Y$ 中范数的性质导出
    - **完备性**：
      - 设 $T_n$ 为基本列，则其有极限 $T$，此时 $T_nx\to Tx$
        - <font color='chartreuse'>由 $\ms Y$ 的完备性得 $Tx\in\ms Y$</font>
      - 只需证 $T\in \opxy$，即 $T$ 具有线性与连续性（有界性）
        - **线性**：<font color='chartreuse'>处理起来很麻烦，但思想很简单。就略去不写了</font>
        - **有界性**：当 $\|x\| = 1$ 时，由收敛性得 $\|Tx\| \leqslant \|T_nx\| + 1$。故由算子范数定义可得 $\|T\| \leqslant \|T_n\|+1$，从而有界
  - **理解**：证明的核心无非是完备性。由于算子范数需要使用像空间的范数，故要求 $\ms Y$ 是完备的。然后再由算子范数本身的良好性质即可得到结论

### 习题

- **保有界性**：有界线性算子 $\LR$ 将有界集映射为有界集的线性算子
  - <font color='chartreuse'>本质就是证明有界算子的范数必定有界</font>
  - **证明**：定义易得
- **C-S不等式**：
  - 在 $\opxy$ 中
    - $\|Ax\| \leq \|A\|\cdot \|x\|$
    - $\|AB\| \leq \|A\|\cdot \|B\|$
  - **证明**：
    - <font color='chartreuse'>仿照数值线性代数中的方法证明即可</font>
- **极值原理**：实 $B^*$ 空间上的非零实值线性泛函 $f$ 不能在开球中心取到极值
  - <font color='chartreuse'>类似复变函数中解析函数的最大模原理。但两原理不是包含关系</font>
  - **证明**：
    - 反设 $B(x_0,r)$ 中心取极小值
    - 此时对任意 $\d\in (B-x_0)$，由极小性得 $f(x_0+\delta) = f(x_0) + f(\delta) > f(x_0)$
    - 即 $f$ 在 $B(\t,r)$ 上正定，显然它与线性矛盾
    - <font color='chartreuse'>若满足线性，则 $f(-x) = -f(x)$，但这与正定性矛盾</font>
  - **理解**：用二元实值函数举个例子。设 $f(x,y) = kx + by$，则函数图形为平面，而平面无极值。它可一直延伸。
- **最大模原理**：
  - 设 $A\in \opxy$，则算子范数有以下等价定义
  - $\|A\| = \sup\limits_{\|x\|\leq 1} \|Ax\|$
    - **互包证明**：
    - **反证**：反设 $\|z\| < 1$，且 $\|Az\| > \|Ax\|$
      - 由C-S不等式，$\|Az\| \leq \|A\|\|z\| = \|Ax\|\cdot\|z\| \leq \|Ax\|$，矛盾
    - **理解**：即原像的范数越大，像的范数越大。因为是连续线性，故不会出现中央极大值，故一定程度上单调
  - $\|A\| = \sup\limits_{\|x\| < 1} \|Ax\|$
    - **互包证明**：
    - **理解**：上确界在边界上，取不到，但可以通过闭包获得
  - $\|A\| = \inf \hkh{M \biggm | \cfrac{\|Ax\|}{\|x\|} \leq M，\forall x\neq \t}$
    - **互包证明**：
    - **理解**：上界的下确界 = 上确界
  - **总结**：线性算子对原像范数单增？
- **倍率原理**：
  - 设 $f\in\op(\ms X,\R)$，则其范数有以下等价定义
  - $\|f\| = \sup\limits_{\|x\| = 1} f(x)$
    - **证明**：
  - $\sup\limits_{\|x\|<\d} f(x) = \d\|f\|\quad (\forall \d>0)$
    - **证明**：
- **非单射性**：线性算子不一定是单射
  - **证明**：
    - 易得 $f$ 不是单射 $\LR\{\t\}\subsetneqq \ker f$，显然有很多反例
  - **反例**：
    - $C[0,1]$ 上的积分算子
- **核的闭性**：
  - 若 $f$ 是线性泛函，则 $f\in\ms X^*\LR \ker f$ 是闭线性子空间
  - **证明**：
    - **必要性**：易得
      - **线性**：算子线性直得核的线性
      - **闭集性**：设 $\{x_n\}$ 是基本列，由 $\ms X$ 完备性，$\|x_m-x_n\| < \e$ 时，存在极限 $x$，只需证 $x\in \Ker f$
    - **充分性**：
      - **反证法**：反设不连续，则 $\exist x_n\to x$，但 $f(x_n)\nto f(x)$
      - **正证法**：由于闭空间完备性，$\exist x_n\to\t$，$f(x_n) = 0$，故 $f(x_n)\to 0$，在零向量处连续，从而在定义域内处处连续

## 同构

- **同构**：设 $\ms X,\ms Y$ 是 $B^*$ 空间，若存在 $\sigma\in \opxy$ 是连续双射，且逆映射也连续，则称 $\ms X,\ms Y$ 同构
  - <font color='chartreuse'>显然 $B^*$ 空间的同构都是同胚</font>
- **泰勒同构**：
  - 设 $\ms X$ 是 $B$ 空间，$u\in\ms L(\ms X)$，$I$ 是恒等映射
  - 若 $\|u\| \leq 1$，则存在 $v\in \ms L(\ms X)$ 使得 $(I-u)v = v(I-u) = I$
  - <font color='chartreuse'>即 $I-u$ 和 $v$ 连续互逆，从而是同构映射</font>
  - **证明**：
    - 将 $I-u$ 展开成映射的泰勒级数即可
