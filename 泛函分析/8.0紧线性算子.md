# 线性算子的紧性

- **符号约定**：
  - 默认为复 $B$ 空间
  - $B_u$ 表示单位球

## 紧线性算子

- **紧算子**：$B$ 空间上的线性算子 $A$，满足单位球像的闭包 $\ol{A(B_u)}$ 是紧集
  - **等价定义**：
    - **紧定义法**：有界集像的闭包为紧集
    - **紧含定义法**：有界集的像含于某个紧集中
    - **完全有界定义法**：有界集的像是完全有界集
    - **收敛子列定义法**：有界点列的像存在收敛子列
- **紧算子的性质**：
  - **连续性（有界性）**
    - **证明**：由范数连续性，$\|Ax\| \leq \sup\limits_{x\in B_u} \|Ax\|\cdot \|x\|$
    - **反例**：恒等算子 $I:l^2\to l^2$ 连续但不紧
      - **证明**：
        - 设 $e_n$ 是全体仅第 $n$ 分量为 $1$，其余均为 $0$ 的向量，则全体 $e_n$ 的集合有界，但不存在收敛子列，故不是紧的
        - <font color='chartreuse'>本质就是有界闭集不一定是紧集</font>
  - **值域可分性**
    - **证明**：
      - 易得 $\Im A = \mathop{\bigcup}\limits^\infty_{n=1} A(nB_u) =  \mathop{\bigcup}\limits^\infty_{n=1} nA(B_u)$
      - 由单位球像的列紧性 + Hausdorff定理 + 完全有界可分定理即得结论
  - **闭限制传递性**：设 $\ms X_0$ 是闭线性子空间，则 $A|_{\ms X_0}$ 是紧算子
    - **证明**：
      - 由拓扑结论，紧集的闭子集也是紧集，再由紧算子定义即得结论
  - **复合传递性**：紧算子和有界线性算子的任意次序复合都是紧的
    - **证明**：
      - 由连续线性算子的保有界性 + 保紧性直得结论
- **紧算子空间**：$\cxy$
  - **线性封闭性**
  - **完备性**
    - **证明**：
      - 设 $T_n$ 是紧算子列，收敛到 $T$
      - 对 $\ol{T_n(B_u)}$ 取有穷 $\frac{\e}{2}$ 网 $\{y_1,...,y_m\}$，则 $\ol{T(B_u)} \subset \mathop{\bigcup}\limits^\infty_{i=1} B(y_i,\e)$
      - 其存在有穷网，从而是紧集
    - **推论**：$\cxy$ 在 $\opxy$ 中是闭集

## 全连续算子

- **全连续算子**：$A\in\opxy$ 满足：若 $x_n\xto{w}x$ ，则 $Ax_n\to Ax$
  - <font color='chartreuse'>弱收敛等价于强收敛</font>
- **紧 $\LR$ 全连续定理**
  - 紧算子是全连续算子
  - 若 $\ms X$ 是自反空间，则全连续算子是紧算子
  - **证明**：
    - **必要性**：反设，则 $\exist x_0,n_i$，使得 $\|Ax_{n_i} - y\| \geq \e_0$
      - 由共鸣定理，$x_n$ 有界
      - 再由 $A$ 是紧算子，$x_{n_i}$ 的像中存在收敛子列，设为 $Ax_{n_i}\to z$
      - 由 $\forall y^*\in\ms Y^*，\lang y^*,Ax_{n_i}-y \rang = \lang A^*y^*, x_{n_i}-x \rang\to 0$
        - 得 $Ax_{n_i}\xto{w} y$，从而 $y=z$，矛盾
    - **充分性**：
      - 由E-S定理，若 $x_n$ 有界，则有子列 $x_{n_i}\xto{w}x$
      - 由A全连续得 $Ax_{n_i}\to Ax$，从而其为紧算子
- **紧共轭定理**：$T\in\cxy \LR T^*\in\cyx$
  - <font color='chartreuse'>紧算子的共轭算子也是紧的</font>
  - **证明**：
    - **必要性**：
      - 设 $\p_n(y) = \lang y_n^*,y \rang\quad (\forall y\in\ol{T(B_1)})$
      - **一致有界性**：$|\p_n(y)| \leq \|y_n^*\|\cdot \|y\| \leq \|T\|\quad (\forall y\in \ol{T(B_1)})$
      - **等度连续性**：$|\p_n(y)-\p_n(z)| \leq \|y_n^*\|\cdot\|y-z\|\leq \|y-z\|$
      - 从而 $\{\p_n(y)\}$ 在 $C(ol{T(B_1)})$ 存在收敛子列，即 $\{T^*y_n\}$ 存在收敛子列
    - **充分性**：
      - 同上得 $T^{**}\in \mf C(\ms X^{**},\ms Y^{**})$
      - 再由自反性得 $T = T^{**}|_\ms X$，由紧算子限制传递性即得结论

### 实例

- **单积分**：
  - **定义域**：有界闭集 $\Omega\subset \R^n$
    - $K\in C(\Omega\times\Omega)$
    - $\ms X = \ms Y = C(\Omega)$
  - **紧算子**：$T: C(\Omega)\to C(\Omega)，u\mapsto \int_\Omega K(x,y)u(y)dy\quad (\forall u\in C(\Omega))$
  - **证明**：
    - 设 $M = \max\limits_{x,y\in\Omega} |K(x,y)|$，则 $\|Tu\| \leq M\|u\|$
    - 由 $K$ 定义得一致连续性，$\forall \e>0，\exist \delta>0$，使得 $\forall y\in \Omega$，若 $|x-x'|<\d$，则有 $|K(x,y)-K(x',y)|<\e$
      - 此时 $|(Tu)(x) - (Tu)(x')| \leq \e\|u\| mes(\Omega)$
- 额额
  - **定义域**：有界开区域 $\Omega\subset \R^n$
  - **紧算子**：$A\in \ms L(H_0^1(\Omega))$，对给定的 $C\in\R$，满足 $\|Au\|_{H^1_0(\Omega)} \leq C\|u\|_{L^2(\Omega)}$
  - **证明**：
    - 由后面的Rellich定理，$\ell: H^1_0(\Omega)\to L^2(\Omega)$ 是紧嵌入
    - 由 $A: L^2(\Omega)\to H^1_0(\Omega)$ 连续，则由紧算子单复合传递性即得


## 紧算子的有穷秩逼近

- **有穷秩算子**：$T\in\opxy$，且 $\dim \Im T$ 有限
  - <font color='chartreuse'>高代中，线性映射的秩定义为像空间的维数。此处沿用了该定义</font>
  - **紧性**：有穷秩算子是紧算子
    - **证明**：
      - 由保有界性，单位球的像有界。再由有限维空间上紧集等价于有界闭集，即得结论
- **有穷秩算子空间**：$F(\ms X,\ms Y)$
  - **线性封闭性**：易得
  - **不完备性**
    - **反例**：设 $P_n$ 将Hilbert空间投影到前 $n$ 维子空间，则 $n\to\infty$ 时，按强算子拓扑收敛到恒等映射，但无穷维空间中按算子范数不收敛
- **秩1算子**：$y\otimes f:\ms X\to \ms Y， x\mapsto \lang f,x \rang y$，其中  $f\in\ms X^*，y\in\ms Y$
  - <font color='chartreuse'>秩1算子可以看作张量积</font>
  - **秩1性**：$\dim\Im y\otimes f = 1$
    - **证明**：
      - 显然像含于直线 $\span\{y\}$ 中，故秩为 $1$
- **秩1分解定理**：$T\in F(\ms X,\ms Y) \LR \exist y_i\in\ms Y,f_i\in\ms X^*$，使得 $T = \sum\limits^n_{i=1} y_i\otimes f_i$
  - <font color='chartreuse'>有穷秩算子都是秩1算子的和</font>
  - **证明**：
    - **充分性**：此时 $\Im T = \span\{y_1,\cdots,y_n\}$，故维度有限
    - **必要性**：
      - 由有穷秩性，像可被有限个基线性表出为 $Tx = \sum\limits^n_{i=1} l_i(x)y_i$
        - <font color='chartreuse'>只需要证明 $l_i$ 是有界线性泛函，即可表示为内积形式</font>
      - **线性**：$T$ 线性 + 坐标 $l_i$ 的唯一性
      - **连续性**：易得 $\|Tx\|$，$\sum\limits^n_{i=1} |l_i(x)|$ 都是 $\Im T$ 上的范数，再由有穷性得两范数等价，从而 $l_i$ 有界
        - <font color='chartreuse'>实际上取 $f_i = l_i$ 即可</font>
      - 综上，$\forall i，\exist f_i\in \ms X^*$，使得 $\lang f_i,x \rang = l_i(x)$**（证毕）**
  - **本质**：将有穷秩算子分解为秩1算子的和
  - **几何意义**：
    - 将 $x$ 逐个映射为 $Tx$ 在每个维度上的分量，然后将其相加即可（相当于像的正交分解）
- **有穷秩逼近定理**：<font color='chartreuse'>有穷秩算子在紧算子空间中稠密，从而可以逼近任意紧算子</font>
  - 设 $\ms X$ 是Hilbert空间，则 $\ol{F(\ms X,\ms Y)} = \cxy$
    - **证明**：
      - **必要性**：已知有穷秩算子是紧算子
      - **充分性**：
        - 由 $\ol{T(B_u)}$ 是紧集，故由豪斯多夫定理得存在有穷 $\dfrac{\e}{2}$ 网将其覆盖，即 $$\ol{T(B_u)} \subset \mathop{\bigcup}\limits^n_{i=1}B(y_i,\frac{\e}{2})$$
        - 设 $E_\e = \span\{y_1,\cdots,y_n\}$，$P_\e$ 是 $E_\e$ 的正交投影算子
          - 则由 $\dim E_\e = n$ 即得 $P_\e T\in F(\ms X,\ms Y)$
          - <font color='chartreuse'>正交投影具有良好性质，但只有希尔伯特空间才能构造</font>
        - 由网性质，$\forall x\in B_u，\exist y_i$ 使得 $\|Tx - y_i\| < \dfrac{\e}{2}$
          - 再作用正交投影算子即得 $\|P_\e Tx - y_i\| < \dfrac{\e}{2}$
          - 最后由三角不等式即得 $\|Tx-P_\e Tx\| < \e$
          - <font color='chartreuse'>本质其实是因为希尔伯特空间的结构简单，可以用维数分解，所以用紧算子</font>
  - 设 $\ms X$ 是可分Banach空间，若存在Schauder基，则 $\ol{F(\ms X)} = \mf C(\ms X)$
    - **引理**：Schauder基的坐标 $C_n(x)$ 是连续泛函
      - **证明**：
        - 设 $S_Nx = \sum\limits^N_{n=1} C_n(x)e_n$，则可定义上确界范数 $\square x\square = \sup\limits_{N\in\natnums} |S_N x|$
        - **完备性**：
          - 由Schauder基的定义易得
        - **范数等价性**：
          - $\|x\| = \lim\limits_{N\to\infty} \|S_Nx\| \leq \square x\square$
          - 由等价范数定理，它们等价
        - 作差 + 三角不等式得 $$\|C_n(x)e_n\| = \|S_nx - S_{n-1}x\| \leq 2\square x\square \leq 2M_1\|x\|$$
        - 再由C-S不等式即得 $C_n(x)$ 有界，从而连续
    - **定理证明**：
      - 由共鸣定理，$\|S_N\|\leq M$ 有界
      - 设 $T\in\cx$，任取 $\e > 0$
        - 由 $\ol{T(B_u)}$ 是紧集，豪斯多夫定理得存在有穷 $\cfrac{\e}{3M+1}$ 网，设为 $\{y_1,...,y_m\}$
        - 由Schauder基的定义，$\forall j，\exists N$ 使得 $\|y_j-S_Ny_j\| < \cfrac{\e}{3}$
        - 再由 $S_N$ 有界性即得 $\|S_N(Tx) - S_N(y_j)\| < \cfrac{\e}{3M+1}$
        - 结合上述三个不等式即得 $$\forall x\in B_u，\|Tx-(S_NT)x\| < \e$$
      - 此时 $T_\e = S_NT$ 是秩 $m$ 算子，且 $\|T-T_\e\| < \e$（**证毕**）

### 习题

- **Hilbert-Schmidt算子**：
  - 设矩形区域 $I = [a,b]^2$，函数 $k(t,s)\in L^2(I)$
  - 则 $K:x(t)\mapsto \dis\int^b_a k(t,s)x(s)ds$ 是紧线性变换
  - **证明**：
    - 易得 $\phi(t) = (b-a)^{-\dfrac{1}{2}}\exp\dkh{2\pi in\cfrac{t-a}{b-a}}$ 是 $L^2[a,b]$ 的一族正交基
    - 故易得 $\phi_{n,m} = \phi_n\ol{\phi_m}$ 是 $L^2(I)$ 的一族正交基
    - 若设 $K_N = \dis\sum_{|n|,|m|\leq N} \lang k,\phi_{n,m} \rang\phi_{n,m}$，则易得 $\|K_N-K\|\to 0\quad (N\to\infty)$
    - 由于 $\dim\Im K_N = 2N+1$，故 $K_N$ 是紧的，从而 $K$ 是紧的
- **紧算子的核可分性**：
  - 设 $H$ 是Hilbert空间，$T$ 是紧算子，则 $(\ker T)^\perp$ 可分
  - **证明**
    - 设 $\{e_\a\}_{\a\in A}$ 是 $(\ker T)^\perp$ 的正交基
      - 由算子紧性得 $\forall \{\a_n\in A\}$ 都有 $\lim\limits_{n\to\infty} Te_{\a_n} = 0$
      - 即 $A$ 至多可数，从而由Hilbert可分定理即得 $(\ker T)^\perp$ 可分
- **逼近定理**：
  - 设 $\ms {X,Y}$ 是 $B$ 空间，$T\in\cxy$
  - 则 $\forall \e > 0$，存在 $\Im T$ 的有穷维子空间 $M$ 满足 $\forall x\in\ms X$ 都有 $$\inf\hkh{\|Tx-m\|\biggm| m\in M} \leq \e\|x\|$$
  - **证明**：
  - **几何意义**：
    - 总能找到一个距离值域足够近的有限维子空间
    - $\e$ 越小，$M$ 的维数越大
    - <font color='chartreuse'>被后面的像核直和定理完爆</font>
