# 谱

## 特征算子

- 闭线性算子 $A: \funcx\to\funcx$
  - 特征值
  - 特征向量
- **预解方程**：$(\l I - A) x = \t$
- **A的预解集**：$\rho(A) = \{\l\in\Complex\mid (\l I-A)^{-1}\in\op(\funcx)\}$
  - **正则值**：$\l$
  - **非特征性**：有穷维时，$\forall \l\in\Complex$ 若不为A的特征值，则 $(\l I-A)^{-1}\in\op(\funcx)$
- **A的谱集**：$\sigma(A) = \Complex - \rho(A)$<font color='chartreuse'>（特征值的集合）</font>
  - 预解方程系数 $\l$ 的分类：
    - $\l$ 是特征值：$(\l I-A)^{-1}$ 不存在
      - **点谱 $\sigma_p(A)$**：<font color='red'>$\sigma(A)$ 为至多可数个点</font>
    - $\l$ 是正则值：$(\l I-A)^{-1}$ 存在
      - **空谱 $\varnothing$**：<font color='red'>$\rho(A)$ 为空集</font>
        - 此时 $R(\l I-A) = \funcx$
      - **连续谱 $\sigma_c(A)$**：<font color='red'>$\rho(A)$ 为极限点集</font>
        - 此时 $R(\l I-A)\neq \funcx$，$\ol{R(\l I-A)}= \funcx$
      - **剩余谱 $\sigma_r(A)$**：<font color='red'>$\rho(A)$ 为集合</font>
        - 此时 $\ol{R(\l I-A)}\neq \funcx$
  - **谱公式**：$\sigma(A) = \sigma_p(A) \cup \sigma_c(A) \cup \sigma_r(A)$
    - 将 $\l$ 取遍 $\R$ 即可

### 实例

- $\funcx = L^2[0,1]$，二阶微分取反算子 $A: u(t)\mapsto -\dfrac{d^2}{dt^2}u(t)$
  - Fourier展开后，$(Au)(t) = \sum\limits^\infty_{n=-\infty} (2n\pi)^2 u_ne^{2\pi int}$
  - 设 $D(A) = \{u\in\funcx\mid Au\in\funcx\}$，则 $A$ 成为闭线性算子
  - 谱集为点谱：$\sigma(A) = \sigma_p(A) = \{(2n\pi)^2\mid n\in\natnums\}$
  - **证明**：检验易得 $(2n\pi)^2\in \sigma_p(A)$
    - 设 $\l\neq (2n\pi)^2$
      - 则 $\forall f\in L^2[0,1]，(\l I - A)u(t) = f(t)$ 有唯一解
      $\\ \begin{cases} u(t) = \sum\limits^\infty_{i=-\infty} \cfrac{C_n}{(2n\pi)^2 - \l}e^{2\pi int} \\ C_n = \int^1_0 f(t)e^{-2\pi int}dt \end{cases}$
        - <font color='chartreuse'>由于 $\{f(t)\mid f\in L^2[0,1]\} = R(\l I-A) = \funcx$，故谱集为空？</font>
      - 再由分母有界性，设 $M_\l = \sup\limits_{n\in\mb Z}\cfrac{1}{(2n\pi)^2 - \l^2}$
        - 则 $\|u\|^2 \leq M_\l^2\sum |C_n|^2 = M_\l\|f\|^2$
        - 从而 $u\in\opx$，其为非特征值，此时 $\forall \l\in\rho(A)$
      - 综上，$\sigma(A) = \sigma_p(A)$
  - **理解**：二阶微分与Fourier级数的契合，导致其谱集均为特征值
- $\funcx = C[0,1]$，数乘算子 $A: u(t)\mapsto t\cdot u(t)$
  - 有界线性
  - 谱集为剩余谱 $\sigma(A) = \sigma_r(A) = [0,1]$
  - **证明**：
    - 若 $\l\notin [0,1]$，则 $(\l-t)^{-1}\in\opx$
      - 则 $R-[0,1]\subset \rho(A)$，即 $\sigma(A)\subset [0,1]$
    - 若 $\l\in [0,1]$，则 $\l-t$ 可能为0，从而 $(\l-t)^{-1}$ 不连续
      - 故预解方程只有解 $u=\t$
      - 且此时 $(\l-t)u = \bd 1$ 的解 $(\l-t)^{-1}$ 不连续
        - 从而恒等映射 $\bold 1\notin \ol{R(\l I-A)}$
        - 从而谱集为剩余谱，即 $[0,1]\subset \sigma_r(A)$
    - 综上即可得结论
- $\funcx = L^2[0,1]$，数乘算子 $A: u(t)\mapsto t\cdot u(t)$
  - 有界线性
  - 谱集为连续谱 $\sigma(A) = \sigma_c(A) = [0,1]$
  - **证明**：
    - 易得 $\bold 1\notin R(\l I-A)$
    - 再由 $t=\lambda$ 外的小邻域外，为平方可积函数，$\ol{R(\l I-A)} = L^2[0,1]$

## 线性算子的谱论

- **预解式**：算子值函数 $R_\l(A): \rho(A)\to\opx，\l\mapsto(\l I-A)^{-1}$
- **谱论定理**：
  - 预解集 $\rho(A)$ 是开集
  - 预解式 $R_\l(A)$ 是 $\rho(A)$ 内的算子值解析函数<font color='chartreuse'>（复定义域为开区域，内部解析，边界连续）</font>

### 开集性

- **压缩引理**：
  - 若 $T\in\opx，\|T\| < 1$
  - 则
    - $(I-T)^{-1}\in\opx$
    - $\|(I-T)^{-1}\| \leq \cfrac{1}{1-\|T\|}$
  - **C-S证明**：
    - 设 $S: x \mapsto y+Tx$
    - 由范数C-S不等式，得 $S$ 是压缩映射，从而 $\forall y$，存在唯一不动点 $x_0$，满足 $\|x_0\|\leq \cfrac{\|y\|}{1-\|T\|}$
    - 由于 $(I-T)^{-1}: y\mapsto x_0$，故 $\|x_0\| = \|(I-T)^{-1}y\| \geq \|(I-T)^{-1}\|\cdot \|y\|$
      - 从而得题设不等式
  - **Nueman级数证明**：
    - 易得 $x = \lim\limits_{n\to\infty} S^n y = \sum\limits^\infty_{k=0} T^ky$
    - 故 $\|T\| < 1$ 时，$(I-T)^{-1} = \sum\limits^\infty_{k=0} T^k$
  - **推论**：
    - **闭线性算子的预解集是开集**
      - **证明**：设 $\l_0\in\rho(A)$
        - 变形得 $\l I-A = (\l_0 I-A)\big[ I+(\l-\l_0)(\l_0 I-A)^{-1} \big]$
          - 中括号内设为 $B$
        - 设 $r = \|(\l_0 I-A)^{-1}\|^{-1}$
          - 则 $\l\in O(\l_0,r)$ 时，$B^{-1}\in\opx$
          - 从而 $(\l I-A)^{-1} = B^{-1}R_{\l_0}(A)\in\opx$
        - 即 $\lambda\in\rho(A)$
