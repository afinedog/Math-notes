# Riesz-Fredholm理论

- **符号约定**：
  - 空间默认为 $B$ 空间
  - $\d_{ij}$ 表示Kronecker符号

## 紧算子方程

- **紧算子方程**：设 $A\in\cx$，$T = I-A$，则 $Tx = y$ 称为紧算子方程
  - **共轭方程**：$T^*x = y$ 称为原方程的共轭方程

### 常见的紧算子方程

- **代数方程**：$y\in\R^n$
  - **有解条件**：方程有解 $\LR \ker T^*\perp y$
      - <font color='chartreuse'>实际上就是高代的结论，只不过定义域不是有限维的</font>
  - **解的分类**：
    - 非齐次情况：唯一解
    - 齐次情况：若 $Tx = \t$ 存在非零解，则存在解空间
  - **共轭关系**： 齐次时，共轭方程的解空间维数相同
- **积分方程**：设 $x,y,f,g\in L^2[0,1]$，则以下两方程互为共轭 $$x(t) = \int^1_0 K(t,s)x(s)ds + y(t)$$  $$f(t) = \int^1_0 K(t,s)f(s)ds + g(t)$$
  - <font color='chartreuse'>积分方程对应的紧算子就是Hilbert-Schmidt算子</font>
  - **解的情况**：
    - 非齐次情况：若右端向量 $y\in L^2[0,1]$，则方程有唯一解
    - 齐次情况：若右端向量 $y = \t$，则方程有非零解
  - **共轭关系**：
    - 解的情况相同
    - 齐次方程的解空间维数有限且相同
  - **有解条件**：
    - 齐次时，有以下定理
      - 原方程有解 $\LR \dis\int^1_0 f(t)y(t)dt = 0$
      - 共轭方程有解 $\LR \dis\int^1_0 g(t)x(t)dt = 0$

## Fredholm结论

- **Fredholm结论**：积分方程中
  - **像与核的互补性**：$\ker T = \{\t\}\red\Rt \Im T = \ms X$
  - **同谱同维性**：$\sigma(A) = \sigma(A^*)$，$\dim\ker T = \dim \ker T^* < \infty$
  - **像与核的共轭转置性**：$\Im T = \ker(T^*)^\perp$，$\Im (T^*) = \ker T^\perp$
  - <font color='chartreuse'>Fredholm结论针对积分方程，而R-F定理针对一般的紧算子方程</font>
- **Riesz-Fredholm定理**：
  - 设 $\ms X$ 是 $B$ 空间，$A\in \mf C(\ms X)，T = I-A$
  - 则
    - **谱集相同**：$\sigma(T) = \sigma(T^*)$
    - **维数相同**：$\dim \ker T = \dim \ker(T^*) < \infty$
    - **共轭转置性**：
      - $\Im T = \ker(T^*)^\perp = \hkh{x\in\ms X\mid f(x) = 0，\forall f\in \ker(T^*)}$
      - $\Im(T^*) = \ker T^\perp = \hkh{f\in\ms X^*\mid f(x) = 0，\forall x\in \ker T}$

### 闭值域算子

- **闭值域算子**：值域是闭集的算子
- **闭值域引理**：若 $A\in\cx$，则 $T = I-A$ 是闭值域算子
  - **证明**：
    - 设核的商映射 $\wt T: (\ms X/\ker T)\to\ms X，[x] \mapsto Tx$
    - **线性**：易得
    - **有界性**：易得
    - **可逆性**：易得
    - **逆连续性**：
      - 反设不连续，则 $\exist [w_m]\nto \t，\wt T[w_n]\to \t$
      - 此时存在子列 $\norm{[w_{m_n}]} \geq \varepsilon$
      - 将子列单位化 $[x_n] = \cfrac{[w_{m_n}]}{\|[w_{m_n}]\|}$
      - 则 $\forall x_n\in [x_n]$，都有 $\|x_n\| < 2$
      - 由 $A$ 是紧算子，存在像收敛子列 $\{Ax_{n_k}\} \to z$
        - 由 $(I-A)x_n\to \t$，得 $x_{n_k}\to z$
        - 从而 $Tz = \t$，即 $[z] = [\t]$
        - $\|[x_{n_k}]\| = \|[x_{n_k}-z]\| \leq \|x_{n_k}-z\| \to 0$，与 $[\|[x_{n_k}]\|] = 1$ 矛盾

### R-F定理的证明

- **谱集相同定理**：设 $T\in\ms L(\ms X)$，则 $\sigma(T) = \sigma(T^*)$
  - **证明**：
    - 只需证明 $T^{-1}\in\opx \LR (T^*)^{-1}\in\ms L (\ms X^*)$ 即可
      - <font color='chartreuse'>由 $T$ 的任意性即得对 $\l I-T$ 也成立，从而谱集相同</font>
    - **必要性**：由 $(T^*)^{-1} = (T^{-1})^*$ 易得结论
    - **充分性**：
      - 由必要性结论得此时 $(T^{**})^{-1} \in \ms L(\ms X^{**})$
        - 再由自然映射等距同构性即得 $T$ 是闭值域双射
        - <font color='chartreuse'>此时 $\Im T$ 是闭线性子空间</font>
      - 反设存在 $x_0\in\ms X\j\Im T$，则由H-B定理，存在单点延拓 $f$ 满足 $$\begin{cases} f(x_0) = \|x_0\| \\ f(x) = 0 & \forall x\in\Im T \end{cases}$$
        - <font color='chartreuse'>因为 $x_0\notin \Im T$，由闭集性易得该方向上的向量也不在 $T$ 中。再由于单点延拓在其它方向的取值可以任意小，这么取确实是合理的</font>
        - 此时 $\forall y\in\ms X，0 = \lang f,Ty \rang = \lang T^*f,y \rang$，由向量判零定理，只能是 $T^*f = 0$，即 $f\in \ker T^*$
        - 再由充分性条件，$T^*$ 可逆，从而只能是 $f = \t$，矛盾
  - **本质**：
    - <font color='chartreuse'>这个定理完爆了。。</font>
- **像核互补定理**：设 $A\in\cx$，$T = I-A$，则 $\ker T  = \{\t\} \red\Rt \Im T = \ms X$
  - **证明**：
    - 反设不然，则空间序列 $\begin{cases} \ms X_0 = \ms X \\ \ms X_k = T(\ms X_{k-1}) \end{cases}$ 是单减序列
    - 由R引理，存在 $\ms X_0$ 单位球面上的点列 $\{y_k\in X_k\}$ 满足 $\dist(y_k,\ms X_{k+1}) \geq \dfrac{1}{2}$
    - 易得 $\forall p,n\in\N$ 都有 $$ \|Ay_n - Ay_{n+p}\| = \|y_n - Ty_n + Ty_{n+p}-y_{n+p}\|\geq \frac{1}{2} $$
    - <font color='chartreuse'>因为由单调性可得 $Ty_n-Ty_{n+p}+y_{n+p}\in \ms X_{n+1}$，再由上述距离不等式即可</font>
    - 也就是说，此时单位球在 $A$ 下的像不是列紧集，与 $A$ 的紧性矛盾
- **维数有限引理**：设 $A\in\cx$，$T = I-A$，则 $\ker T，\ker T^*$ 维数有限
  - **证明**：
    - 设 $B_1$ 是 $\ker T$ 上的闭单位球，则由 $T$ 定义易得 $B_1 = \ol{AB_1}$
    - 再由于 $A$ 是紧算子，即得 $B_1$ 是紧集
    - 再由单位球的非紧性，此时只能是 $\ker T$ 维数有限。共轭情况同理
- **直和引理**：
  - 设 $\{x_1,...,x_n\}$ 是 $\ker T$ 的基
  - 则存在闭线性子空间 $\ms X_1$ 满足 $\ms X = \span\{x_1,...,x_n\}\oplus \ms X_1$，共轭情况同理
  - **证明**：
    - 由单点延拓定理，存在 $g_1, ..., g_n\in\ms X^*$ 满足 $g_{i}(x_j) = \d_{ij}$
      - <font color='chartreuse'>闭线性子空间 + 单点延拓函数的任意小性</font>
    - 取 $\ms X_1 = \mathop{\bigcap}\limits^n_{i=1} \ker g_i$ 即可
      - **闭集性**：
        - **证**：由闭集的有限交封闭性易得结论
      - **线性子空间性**：
        - **证**：由 $\ker g_i$ 的定义，易得线性封闭性
      - **不相交性**：$\span\{x_1,...,x_n\}\cap \ms X_1 = \{\t\}$
        - **证**：由 $g_i$ 定义易得
      - **互补性**：$\forall x\in \ms X，x-\sum\limits^n_{i=1} g_i(x)x_i \in \ms X_1$
        - **证**：
          - 显然 $g_i$ 只会保留 $x$ 在 $\span$ 中的第 $i$ 维度分量，而 $\span$ 的其它分量均被化为 $0$
          - 那么删去 $\span$ 的所有维度分量后，$x$ 的剩余部分被 $\forall g_i$ 都映射为 $0$，从而由定义它位于 $\ms X_1$ 中
- **Kronecker引理**：
  - 设 $\{f_1,...,f_m\}$ 是 $\ker T^*$ 的基
  - 则存在 $y_1,...,y_m\in \ms X$ 使得 $f_i(y_j) = \d_{ij}$
  - **证明**：
    - 设 $V:\ms X\to \K^m，x\mapsto \Big( f_1(x),...,f_m(x) \Big)$
      - **线性**：
        - **证**：定义易得
      - **连续性**
        - **证**：由分量函数连续性易得
      - **满射性**：
        - **证**：
          - 反设不是满射，则存在 $\a$ 使得其在 $\Im V$ 上的分量为 $0$
          - 此时 $\forall x\in\ms X，\ip{\sum\limits^m_{j=1}\a_jf_j，x} = 0$，由向量判零定理，$\sum\limits^m_{j=1}\a_jf_j = \t$，即 $\{f_j\}$ 线性相关，矛盾
    - 既然 $V$ 是满射，那么题设结论也就不难证明了
- **维数相等定理**：
  - **证明**：
    - 设 $\{x_1,...,x_n\}$ 是 $\ker T$ 的基，$\{f_1,...,f_m\}$ 是 $\ker T^*$ 的基
    - 反设 $n<m$，考虑映射 $$\wt T : \Big[ \span\{x_1,...,x_n\}\oplus \ms X_1 \Big] \to \Big[ \span\{y_1,...,y_n\}\oplus \Im T \Big]$$ $$\wt T\dkh{\sum^n_{i=1} c_ix_i + y} =  \sum^n_{i=1} c_iy_i + Ty$$
      - **变换性**：
        - **证**：由直和引理，原象是 $\ms X$ 全体。再由定义，像也是 $\ms X$ 全体
      - **满射性**：
        - **证**：
    - 从而 $m\leq n$，即 $\dim \ker T^* \leq \dim\ker T$
      - 同理 $\dim \ker T^{**} \leq \dim\ker T^*$
      - 再由于 $\dim \ker T \leq \dim\ker T^{**}$，互包即得维数相等
- **像核共轭转置引理**：若 $T\in \opx$，则 $\ol{\Im T} = (\ker T^*)^\perp$ 
  - <font color='chartreuse'>这里是像的闭包，还不是想要的结果</font>
  - **证明**：
    - **必要性**：
      - 由定义，$\forall f\in\ker T^*，f(Tx) = (T^*f)x = 0$，从而 $Tx\in\ker f$，即 $\Im T\subset $
    - **充分性**：
      - 反设存在 $x_0\in \Big[ (\ker T^*)^\perp\j \ol{\Im T} \Big]$
      - 由H-B定理，存在单点延拓 $f\in\ms X^*$ 满足 $\begin{cases}   f(x_0) = \|x_0\|  \neq 0 \\\\ f(x) = 0 \qquad \forall x\in\ol{\Im T}   \end{cases}$
        - 此时 $\forall x,f(Tx) = 0$，由向量判零定理，$T^*f = \t$
        - 再由 $x_0$ 定义即得 $f(x_0) = 0$，矛盾
- **像核共轭转置定理**：
  - **证明**：
    - 由闭值域引理，此时 $\Im T$ 是闭集，故 $\Im T = (\ker T^*)^\perp$
    - 再由维数相等定理，$$\dim \ker T^{**} = \dim\ker T^* = \dim\ker T$$
      - 再由自然映射的等距同构性即得 $\ker T = \ker T^{**}$，从而 $$\Im T^* = (\ker T^{**})^\perp = (\ker T)^\perp$$

### 维数定理

- **像核直和定理**：
  - 设 $A\in\cx$，$T = I-A$
  - 则存在闭线性子空间 $\ms X_1$ 和有限维子空间 $\ms Y_1$ （$\dim \ms Y_1 = \dim \ker T$）满足 $$\ms X = \ker T \oplus \ms X_1 = \ms Y_1 \oplus \Im T$$
  - **证明**：
    - 由直和引理 + 像核共轭转置定理即得结论
- **余维数**：$\codim M = \dim(\ms X-M)$
- **像核维数定理**：
  - 设 $A\in\cx，T = I-A$
  - 则 $\dim\ker T = \codim\Im T < \infty$
  - **证明**：
    - 由R-F定理直得结论
