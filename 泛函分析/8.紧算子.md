# 线性算子的紧性

## 紧算子

- **紧算子**：$B$ 空间上的线性算子 $A$，满足单位球像的闭包 $\ol{A(B_u)}$ 是紧集
  - **连续有界性**
    - **证明**：由范数连续性，$\|Ax\| \leq \sup\limits_{x\in B_u} \|Ax\|\cdot \|x\|$
  - **线性封闭性**
  - **闭性**
    - **证明**：设 $T_n$ 是紧算子列，收敛到 $T$
      - 对 $\ol{T_n(B_u)}$ 取有穷 $\frac{\varepsilon}{2}$ 网 $\{y_1,...,y_m\}$，则 $\ol{T(B_u)} \subset \mathop{\bigcup}\limits^\infty_{i=1} B(y_i,\varepsilon)$
      - 其存在有穷网，从而是紧集
  - **值域可分性**
    - **证明**：$R(A) = \mathop{\bigcup}\limits^\infty_{n=1} nA(B_u)$
      - 由列紧性，可分定理得可分性
  - **闭线性子空间上的限制传递性**：
  - **单复合传递性**：两个算子的复合中，若有一个是紧的，则复合是紧的
  - **等价命题**：
    - 有界集像的闭包为紧集：
    - 有界点列的像列紧：
- **紧算子集合**：$\cxy$
- **全连续算子**：$A\in\opxy$ 满足：若 $x_n\xto{w}x$ ，则 $Ax_n\to Ax$
- **紧 $\LR$ 全连续定理**
  - 紧算子是全连续算子
  - $\funcx$ 是自反空间的全连续算子是紧算子
  - **证明**：
    - **必要性**：反设，则 $\exist x_0,n_i$，使得 $\|Ax_{n_i} - y\| \geq \varepsilon_0$
      - 由共鸣定理，$x_n$ 有界
      - 再由 $A$ 是紧算子，$x_{n_i}$ 的像中存在收敛子列，设为 $Ax_{n_i}\to z$
      - 由 $\forall y^*\in\funcy^*，\lang y^*,Ax_{n_i}-y \rang = \lang A^*y^*, x_{n_i}-x \rang\to 0$
        - 得 $Ax_{n_i}\xto{w} y$，从而 $y=z$，矛盾
    - **充分性**：
      - 由E-S定理，若 $x_n$ 有界，则有子列 $x_{n_i}\xto{w}x$
      - 由A全连续得 $Ax_{n_i}\to Ax$，从而其为紧算子
- **紧共轭定理**：$T\in\cxy \LR T^*\in\cyx$
  - **证明**：
    - **必要性**：设 $\p_n(y) = \lang y_n^*,y \rang\quad (\forall y\in\ol{T(B_1)})$
      - **一致有界性**：$|\p_n(y)| \leq \|y_n^*\|\cdot \|y\| \leq \|T\|\quad (\forall y\in \ol{T(B_1)})$
      - **等度连续性**：$|\p_n(y)-\p_n(z)| \leq \|y_n^*\|\cdot\|y-z\|\leq \|y-z\|$
      - 从而 $\{\p_n(y)\}$ 在 $C(ol{T(B_1)})$ 存在收敛子列，即 $\{T^*y_n\}$ 存在收敛子列
    - **充分性**：同上得 $T^{**}\in \mf C(\ms X^{**},\ms Y^{**})$
      - 再由自反性得 $T = T^{**}|_\funcx$，由紧算子限制传递性即得结论

### 实例

- **单积分**：
  - **定义域**：有界闭集 $\Omega\subset \R^n$
    - $K\in C(\Omega\times\Omega)$
    - $\funcx = \funcy = C(\Omega)$
  - **紧算子**：$T: C(\Omega)\to C(\Omega)，u\mapsto \int_\Omega K(x,y)u(y)dy\quad (\forall u\in C(\Omega))$
  - **证明**：
    - 设 $M = \max\limits_{x,y\in\Omega} |K(x,y)|$，则 $\|Tu\| \leq M\|u\|$
    - 由 $K$ 定义得一致连续性，$\forall \varepsilon>0，\exist \delta>0$，使得 $\forall y\in \Omega$，若 $|x-x'|<\d$，则有 $|K(x,y)-K(x',y)|<\varepsilon$
      - 此时 $|(Tu)(x) - (Tu)(x')| \leq \varepsilon\|u\| mes(\Omega)$
- 额额
  - **定义域**：有界开区域 $\Omega\subset \R^n$
  - **紧算子**：$A\in \ms L(H_0^1(\Omega))$，对给定的 $C\in\R$，满足 $\|Au\|_{H^1_0(\Omega)} \leq C\|u\|_{L^2(\Omega)}$
  - **证明**：
    - 由后面的Rellich定理，$\ell: H^1_0(\Omega)\to L^2(\Omega)$ 是紧嵌入
    - 由 $A: L^2(\Omega)\to H^1_0(\Omega)$ 连续，则由紧算子单复合传递性即得

### 紧算子构造

- **有穷秩算子**：$T\in\opxy$，且值域维度 $dim\ R(T)$ 有穷
  - **紧性**：有穷秩算子是紧算子
    - **证明**：
- **有穷秩算子空间**：$F(\funcx,\funcy)$
- **秩1算子**：$y\otimes f:\funcx\to \funcy， x\mapsto \lang f,x \rang y\qquad (f\in\funcx^*，y\in\funcy)$
- **秩1分解定理**：$T\in F(\ms X,\ms Y) \LR \exist y_i\in\ms Y,f_i\in\ms X^*$，使得 $T = \sum\limits^n_{i=1} y_i\otimes f_i$
  - **证明**：
    - **充分性**：此时 $R(T) = span\{y_1,\cdots,y_n\}$，故维度有限
    - **必要性**：用基表示得 $Tx = \sum\limits^n_{i=1} l_i(x)y_i$
      - **线性**：$T$ 线性 + 坐标 $l_i$ 的唯一性
      - **连续性**：易得 $\|Tx\|$，$\sum\limits^n_{i=1} |l_i(x)|$ 都是 $R(T)$ 上的范数，再由有穷性得等价，从而 $l_i$有界
      - 从而 $\exist f_i\in \funcx^*$，使得 $\lang f_i,x \rang = l_i(x)\quad (\forall x,i)$**（证毕）**
  - **本质**：将有穷秩算子分解为秩1算子的张量积
- **有穷秩进化定理**：
  - 若 $\funcx$ 是Hilbert空间，则 $\ol{F(\funcx,\funcy)} = \cxy$
    - **证明**：
      - **必要性**：紧性已得
      - **充分性**：由 $\ol{T(B_u)}$ 是紧集，故存在有穷 $\dfrac{\varepsilon}{2}$ 网覆盖 $\ol{T(B_u)} \subset \mathop{\bigcup}\limits^n_{i=1}B(y_i,\dfrac{\varepsilon}{2})$
      - 设 $E_\varepsilon = span\{y_1,\cdots,y_n\}$，$P_\varepsilon$ 是 $E_\varepsilon$ 的正交投影算子
        - 则 $P_\varepsilon T\in F(\funcx,\funcy)$
      - 由网性质，$\forall x\in B_u，\exist y_i$ 使得 $\|Tx - y_i\| < \dfrac{\e}{2}$
        - 从而 $\|P_\e Tx - y_i\| < \dfrac{\e}{2}$
        - 从而 $\|Tx-P_\e Tx\| \leq \e$
  - 若 $\funcx$ 是可分Banach空间，则若存在Schauder基，则 $\ol{F(\funcx)} = \mf C(\funcx)$
    - **引理**：Schauder坐标 $C_n(x)$ 是连续泛函
      - **证明**：设范数 $\square x\square = \sup\limits_{N\in\natnums} |S_N x|\quad (S_Nx = \sum\limits^N_{n=1} C_n(x)e_n)$
        - **完备性**：定义易得
        - **范数等价性**：
          - $\|x\| = \lim\limits_{N\to\infty} \|S_Nx\| \leq \square x\square$
          - 由等价范数定理，它们等价
        - 作差 + 三角不等式，$\|C_n(x)e_n\| = \|S_nx - S_{n-1}x\| \leq 2\square x\square \leq 2M_1\|x\|$
        - 再由C-S不等式即得 $C_n(x)$ 有界，从而连续
    - **定理证明**：由共鸣定理，$\|S_n\|\leq M$ 有界

## 紧算子方程

- **线性方程**：两种共轭表示
  - $Tx = y$
  - $T^Tx = y$
  - **有解条件**：$y\in \R^n$ 时
    - 方程有解 $\LR \lang z,y \rang = 0，\forall z\in \{z\in \R^n\mid T^T z = \t\}$
  - **理解**：当方程有解，则右端向量是矩阵的线性组合，从而向量 $z$ 和 $y$、$T$ 的垂直关系是等价的
  - **解的情况**：
    - 唯一解
    - 齐次方程有非零解，存在解空间
- **积分方程**：两种共轭表示
  - $x(t) = \int^1_0 K(t,s)x(s)ds + y(t)$
  - $f(t) = \int^1_0 K(t,s)f(s)ds + g(t)$
  - $x,y,f,g\in L^2[0,1]$
  - **解的情况**：
    - 非齐次：右端向量 $y\in L^2[0,1]$，则有唯一解
    - 齐次：$y = \t$，则有非零解
  - **有解条件**：
    - $\int^1_0 f(t)y(t)dt = 0$

### Fredholm结论

- 设像是 $R(T)$，核是 $N(T)$
- **闭值域算子**：值域是闭集
- **闭值域引理**：若 $A\in\cx$，则 $T = I-A$ 是闭值域算子
  - **证明**：设核的商映射 $\wt T: \funcx/N(T)\to\funcx，[x] \mapsto Tx$
    - 易得有界性、线性、可逆性
    - **逆连续性**：反设不连续，则 $\exist [w_m]\nto 0，\wt T[w_n]\to 0$
      - 存在子列 $\|[w_{m_n}]\| \geq \varepsilon$
      - 将子列单位化 $[x_n] = \cfrac{[w_{m_n}]}{\|[w_{m_n}]\|}$
      - 则 $\forall x_n\in [x_n]$，都有 $\|x_n\| < 2$
      - 由 $A$ 是紧算子，存在像收敛子列 $A\{x_{n_k}\} \to z$
        - 由 $(I-A)x_n\to \t$，得 $x_{n_k}\to z$
        - 从而 $Tz = \t$，即 $[z] = [\t]$
        - $\|[x_{n_k}]\| = \|[x_{n_k}-z]\| \leq \|x_{n_k}-z\| \to 0$，与 $[\|[x_{n_k}]\|] = 1$ 矛盾
- **Fredholm结论**：
  - **像与核的互补性**：$N(T) = \{\t\}\red\Rt R(T) = \funcx$
  - **共轭同维性**：$\sigma(A) = \sigma(A^*)$，$dim\ N(T) = dim\ N(T^*) < \infty$
  - **像与核的共轭性**：$R(T) = N(T^*)^\perp$，$R(T^*) = N(T)^\perp$
- **Riesz-Fredholm结论**：
  - 设 $\funcx$ 是B空间，$A\in \mf C(\funcx)，T = I-A$
  - 则
    - $\sigma(T) = \sigma(T^*)$
      - **强化**：有界即可
      - **必要性**：由 $(T^*)^{-1} = (T^{-1})^*$ 易得
      - **充分性**：
    - $dim\ N(T) = dim\ N(T^*) < \infty$
    - $R(T) = N(T^*)^\perp = \{x\in\funcx\mid f(x) = 0，\forall f\in N(T^*)\}$
    - $R(T^*) = N(T)^\perp = \{f\in\funcx^*\mid f(x) = 0，\forall x\in N(T)\}$

  - $0\in\sigma(A)$

## 紧算子共轭

- **共轭对称算子**：$A\in\op(H)，(Ax,y) = (x,Ay)$
  - **自伴性**：$A = A^*$
  - **共轭线性**：
    - $(A+B)^* = A^*+B^*$
    - $(\a A)^* = \ol\a A^*$
  - **自反性**：
  - **穿脱性**：$(AB)* = B^*A^*$
  - **闭包性**：$\sigma(A^*) = \ol{\sigma(A)} = \{\ol \l\mid \l\in\sigma(A)\}$
- **对称算子实例**：
  - 欧氏空间：对称方阵
  - 酉空间：Hermite方阵
  - $L^2(\Omega,\ms B,\mu)$：

    - 对称算子：$A: u(x)\mapsto \int_\Omega K(x,y)u(y)d\mu(y)$
