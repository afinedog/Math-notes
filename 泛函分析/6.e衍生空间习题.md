
### Riesz表示定理

- **连续函数空间的Riesz表示定理**：
  - 若 $M$ 是一个Hausdorff紧空间
  - 则 $\forall f\in C(M)^*$ 有唯一的复值Baire纲测度 $\mu$，满足 $\lang f,\p \rang = \int_M \p(m)d\mu$
  - **证明**：
  - **存在性**：$\mu$ 存在
    - 设 $\p(m) = \chi_E(m)\quad (E\subset M)$
    - 再设单调递减且极限为空集的集列 $\{E_n\}\subset [0,1]$，则 $\mu(E_n) = \int_M \chi_E(M)d\mu$
      - $C(M)$ 是 $L^\infty(M)$ 的一个闭子空间
    - 由Hahn-Banach定理，给定 $f\in C(M)^*$，存在保范延拓 $\widetilde{f}\in L^\infty(M)^*$ 满足 $\begin{cases} <\widetilde{f},\chi_E> > 0 \\ <\widetilde{f},\p> = <f,\p> \\ \|\widetilde{f}\| = \|f\| \end{cases}$
    - 因为 $\chi_{E_n}\in L^\infty(M)$，故可设 $\begin{cases} \mu(E_n) = <\widetilde{f},\chi_{E_n}> \\ \mu(0) = <\widetilde{f},\chi_{E}> = 0 \end{cases}$
  - **表示性**：$\mu$ 完全可加，绝对值有限，且满足题设
    - **完全可加性**：
      - 有限个集合：
        - 设 $\l_k = \begin{cases} \cfrac{\ol\mu(E_n)}{|\mu|(E_n)}，|\mu|(E_n) \neq 0 \\ 0，其它 \end{cases}$
        - $<\widetilde{f},\chi_E> = <\widetilde{f},\sum\limits^n_{k=1} \l_k\chi_{E_k}> = \sum\limits^n_{k=1} \mu(E_k)\quad (E\subset \mathop{\bigcup}\limits^n_{k=1} E_k)$
        - 对左边取上确界，得 $<\widetilde{f},\chi_E> = \mu(E) \leq \mu(\mathop{\bigcup}\limits^n_{k=1} E_k)$，即 $\mu(\mathop{\bigcup}\limits^n_{k=1} E_k) \leq \sum\limits^n_{k=1} \mu(E_k)$
      - 可数个集合：
        - 由于 $\chi_E$ 是紧的，由有限覆盖性，$\exist n$，使得 $\mu(\mathop{\bigcup}\limits^\infty_{k=1} E_k) \leq \mu(\mathop{\bigcup}\limits^\infty_{k=1} E_k) = \sum\limits^n_{k=1} \mu(E_k) \leq \sum\limits^\infty_{k=1} \mu(E_k)$
    - **有限性**：$|\mu(E)| \leq |\mu(\mathop{\bigcup}\limits^\infty_{k=1} E_k)| \leq \sum\limits^n_{k=1} \mu(E_k) = \sum\limits^n_{i=1} \l_k \chi{E_k} = (\widetilde{f},\l_k\chi_{E_k}) \leq \|\tilde{f}\|\|\l_k\chi_{E_n}\|_{L^\infty} \leq \|\tilde{f}\| = \|f\|$，从而 $|\mu|(E)<\infty$
    - **题设**：
      - $\forall \p\in C(M)^*,\varepsilon>0$，取 $E_j$ 满足 $\begin{cases} \p(m) - \p(m') < \cfrac{\varepsilon}{2\|\tilde{f}\|}\quad (\forall m,m'\in E_j) \\ |\int_M \p(m)d\mu - \sum\limits^n_{j=1} \p(m_j)\mu(E_j)| < \dfrac{\varepsilon}{2} \end{cases}$
      - 令 $\p_\D = \sum\limits^n_{j=1} \p(m_j)\chi_{E_j} + \p(0)\chi_{\{0\}}$
      - 则有 $|<f,\p> - \int_M\p(m)d\mu| \leq \|\tilde{f}\|\|\p-\p_\D\|_{L^\infty} + |\sum\limits^n_{j=1}\p(m_j)\mu(E_j)-\int_M \p(m)d\mu| < \varepsilon $
      - 由 $\varepsilon$ 的任意性即得结论
  - **唯一性**：反设有两个满足条件的测度 $\mu_1,\mu_2$
    - 对 $K\subset M，\forall \varepsilon>0，\exist U\supset K，\mu_1(U)-\mu_2(U) \leq \varepsilon$
    - $\mu_1(K) = \int_M\chi_k d\mu_1 \leq \int_M\p d\mu_1 = <f,\p> = \int_M \p d\mu_2 \leq  = \int_M \chi_U d\mu_2 \leq \mu_2(U) \leq \mu_1(K) + \varepsilon$
    - 同理可得反向不等式，从而它们相等

### Runge定理

- **Runge定理**：
  - 设
    - $K$ 是复平面上的紧子集
    - $E$ 是 $\Complex_\infty-K$ 的子集，与每一个连通分量相交
  - 若 $f$ 是 $K$ 邻域内的任意解析函数
  - 则 存在有理函数列 $f_n$，极点均在 $E$ 内，且在 $K$ 上一致收敛到 $f$
  - **证明思路**：
    - 设 $K$ 上的连续函数空间 $C(K)$，$R(K,E)$ 是极点在 $E$ 内的有理函数集在 $C(K)$ 中的闭包。
    - 易得 $R(K,E)$ 是 $C(K)$ 的闭线性子空间
    - 从而 $f\in R(K,E)$ 可由 $\forall F\in C(K)^*，F(g) = 0(\forall g\in R(K,E)) \Rt F(f) = 0$ 得出
    - 再由Riesz表示定理， $C(K)^*$ 是 $K$ 上的完全可加复值测度 $M(K)$ 组成的
    - 从而转化为证明 $\forall \mu\in M(K)，\int_K gd\mu = 0(\forall g\in R(K,E)) \Rt \int_K fd\mu = 0$
  - **引理**：对 $\forall \mu\in M(K)$，设 $\widehat{\mu}(w) = \int_K \cfrac{d\mu(z)}{w-z}$
    - 则 $\forall R>0$，均有：
      - $\widehat{\mu}\in L^1(B(\bd 0,R))$
      - $\widehat{\mu}$ 在 $\C_\infty- K$ 上解析
      - $\widehat{\mu}(\infty) = 0$
    - **证明**：
      - $\widehat{\mu}\in L^1(B(\bd 0,R))$
        - 由定义，$|\widehat{\mu}(w)| \leq \int_K \cfrac{d|\mu|(z)}{|w-z|}$
        - 则 $\int_{B_R} |\widehat{\mu}(w)|dxdy \leq \int_{B_R}\int_K \cfrac{d|\mu|(z)}{|w-z|}dxdy$
        - 由Fubini定理，积分顺序可交换
          - 设 $\rho > R+\max\limits_{z\in K}$
          - 将$B_R$ 放缩为 $B(z,\rho)$，积分 $\leq 2\pi\rho|\mu|(K) < \infty$，从而可积
      - $\widehat{\mu}$ 在 $\C_\infty- K$ 上解析
        - 由于 $K$ 是紧集，故可积分内求微商
      - $\widehat{\mu}$ 在 $\{\infty\}$ 解析，且 $\widehat{\infty} = 0$
        - 由于 $K$ 是紧集，当 $|w|\to \infty$ 时，积分号下取极限得 $\widehat{\mu}(w) \to 0$，从而无穷远点是可去奇点
  - 由引理得 $(\dfrac{d}{dw})^n \widehat{\mu}(w_0) = n! \int_K (z-w_0)^{-n+1} d\mu(z)\quad (\forall w_0\in \C-K)$
  - 在无穷远点附近的展开 $\widehat{\mu}(w) = -\dfrac{1}{w}\sum\limits^\infty_{n=0} \int_K (\dfrac{z}{w})^n d\mu(z) = -\sum\limits^\infty_{n=0} \dfrac{a_n}{w^{n+1}}$
    - $a_n = \int_K z^n d\mu(z)$
  - **恒零性**：若 $\exist \mu\in M(K)$，使得 $\int_K g(z)d\mu(z) = 0$（$g$ 是极点在 $E$ 内的有理函数），则 $\widehat{\mu}(w) = 0\quad (\forall w\in\C_\infty - K)$
    - 而 $\forall w_0 \in E$，设 $\Omega(w_0)$ 是 $\C_\infty-K$ 中含 $w_0$ 的分量，则由关于 $E$ 的假设，有 $\C_\infty - K = \mathop{\bigcup}\limits_{w_0\in E} \Omega(w_0)$
    - 若 $w_0\neq \infty$，由微分与g积分，得 $\widehat{\mu}$ 在 $w_0$ 的各阶导数均为0，从而 $\widehat{\mu}$ 在 $\Omega(w_0)$ 内恒为0
    - 若 $w_0= \infty$，则由展开式与g积分，$\widehat{\mu}$ 在 $\Omega(w_0)$ 内恒为0
    - 综上，恒零性成立
  - $K$ 的某邻域 $G$ 内解析的任意函数 $f$
    - 其存在含于 $G-K$ 内的折线 $\gamma_1,\g_2,....,\g_n$，使得 $f(z) = \sum\limits^\infty_{k=1} \dfrac{1}{2\pi i}\int_{\g_k} \dfrac{f(w)}{w-z}dw$
    - 由Fubini定理，$\int_K f(z)d\mu(z) = \sum\limits^\infty_{k=1} \dfrac{1}{2\pi i}\int_K\int_{\g_k} \dfrac{f(w)}{w-z}dwd\mu(z) = \sum\limits^\infty_{k=1} \dfrac{1}{2\pi i}\int_{\g_k} f(w)\widehat{\mu}(w)dw = 0$
      - 由恒零性得出。
  - 从而定理得证
