# 共轭空间

- 为啥要把共轭空间和弱收敛放在一起：
  - 因为自反空间拥有延拓性、对偶对称性
  - 而弱收敛又是将原像性质传到函数性质上的媒介之一
- **符号约定**：
  - $\var(g)$ 表示复函数 $g$ 的全变差
- **线性泛函的内积表示法**：设 $x\in \ms X$，$f$ 是 $\ms X$ 上的线性泛函，则 $f(x) = \lang f,x \rang$
  - <font color='chartreuse'>由于线性泛函本身也是 $B^*$ 空间的元素，故 $f(x)$ 可看作双线性函数，从而可用内积的形式来表达</font>
  - **对称性**：$x$ 同时也可以看作 $f$ 的泛函

## 共轭空间（对偶空间）

- **共轭空间**：$B^*$ 空间 $\ms X$ 上的全体连续线性泛函称为 $\ms X$ 的共轭空间 $\ms X^*$
  - <font color='chartreuse'>由单点延拓定理，连续线性泛函的数量足够多，足以构成一个空间</font>
  - **第一泛函**：$\ms X^*$ 上的元素称为第一泛函
  - **共轭范数**：$\|f\| = \sup\limits_{\|x\|=1}|f(x)|$
    - <font color='chartreuse'>共轭空间的范数可被原空间诱导</font>
  - **完备性**：共轭空间在共轭范数下是 $B$ 空间
    - **证明**：
      - 只需证明有界线性泛函列的极限仍然有界即可，而这是显然的

### $C[0,1]$ 的共轭空间

<!-- - **连续函数共轭空间**：$C[0,1]$ 的共轭空间 -->
- **有界变差空间**：全体无截距、右连续、有界变差的函数 $$BV[0,1] = \hkh{g:[0,1]\to \C \Biggm| g(0) = 0， g(t) = g(t+) ， \var(g) < \infty }$$
  - **范数**：$\|g\|_v = \var(g)$
    - <font color='chartreuse'>范数是全变差</font>
  - **完备性**：
- **变差共轭定理**：$C[0,1]^* = BV[0,1]$
  - <font color='chartreuse'>等距同构意义下的相等</font>
  - **等距同构映射**：$\p:C[0,1]^*\to BV[0,1]，f\mapsto v$，其中 $f(x) = \dis\int^1_0 x(t)dv(t)$
    - <font color='chartreuse'>这里的 $x$ 是 $C[0,1]$ 上的任意函数</font>
  - **证明**：
    - **构造 $v$**：
      - 由H-B定理，$\forall f\in C[0,1]$，存在 $L^\infty[0,1]$ 上的保范延拓 $F$
      - 则取 $v(s) = F(\chi_{[0,s]})$ 即可
        - <font color='chartreuse'>等距性：详见下面证明
        - 满射性：由题设积分关系式即得
        - 线性：由题设积分关系式 + 微分线性 + 积分线性即得</font>
    - **有界变差性**：
      - 取 $[0,1]$ 的分划 $\{t_k\}^n_{k=1}$
      - 易得 $$\sum^n_{k=1} |v(t_k)-v(t_{k-1})| = F\dkh{\sum^n_{k=1} \e_k(\chi_{[0,t_k]} - \chi_{0,t_{k-1}})}$$
      - 由C-S不等式，上式 $\leq \|F\|_{\infty}\cdot\|...\|_\infty$
        - 由延拓的保范性，$\|F\|_{\infty} = \|f\|$
        - 再易得 $\forall n\in\N，\|...\|_{\infty} = 1$
        - 最后由 $n$ 任意性，推广到极限，即得全变差有界性
    - **积分关系式**：
      - 设 $x\in C[0,1]$
      - 设 $y(s) = \sum\limits^n_{k=1} x(t_k)\Big[ \chi_{[0,t_k]}(s) - \chi_{[0,t_{k-1}]}(s) \Big]$
      - 设 $F(y) = \sum\limits^n_{k=1} x(t_k)\Big[ v(t_k)-v(t_{k-1}) \Big]$
      - 由 $x$ 的连续性，计算易得当划分极小时 $\|y- x\|_{\infty}\to 0$，再由 $F$ 的连续性即得 $F(y)\to F(x)$
      - 由R-S积分的定义，即得题设积分关系式
    - **等距性**：
      - 由积分上下界不等式，$|f(x)| \leq \|x\|\var(v)$，再由定义即得 $\|f\| \leq \var(v)$
      - 在上面有界变差性的证明中，已经得到 $\var(v) \leq \|f\|$
      - 综上，$\|v\| = \var(v) = \|f\|$

### Hilbert空间的共轭空间

- **共轭同构**：等距同构的共轭线性映射
  - <font color='chartreuse'>一般的同构要求线性，这里的同构要求共轭线性</font>
- **Hilbert共轭定理**：设 $H$ 是Hilbert空间，则 $H^*$ 与 $H$ 等距同构
  - **证明**：
    - 由R表示定理，$\forall f\in H^*，\exists y_f\in H$ 使得 $\lang f,x \rang = \lang y_f,x \rang$，且 $\|y_f\| = \|f\|$
      - <font color='chartreuse'>实际上 $f$ 和 $y_f$ 是一一对应的</font>
    - 构造映射 $\tau:H^*\to H，f\mapsto y_f$
      - **共轭线性**：由内积的共轭双线性易得
      - **等距性**：由上面结论即得等距性
  - **实例**：
    - $\R^n$
    - $\C^n$
    - $L^2[a,b]$
    - $l^2$


## 第二共轭空间（对偶空间的共轭空间）

- **第二共轭空间**：设 $\ms X$ 是 $B^*$ 空间，则 $\ms X^*$ 的对偶空间 $\ms X^{**}$ 称为 $\ms X$ 的第二共轭空间
  - <font color='chartreuse'>连续线性泛函的连续线性泛函</font>
  - **第二泛函**：设 $x\in \ms X$，则 $X: \ms X^*\to \R，f \mapsto \lang f,x \rang$ 称为第二泛函
    - **线性**：$(X_1+X_2)f = f(x_1) + f(x_2)$
  - **第二共轭范数**：$\|X\| = \sup\limits_{f\neq \bd 0} \cfrac{|X(f)|}{\|f\|} =  \sup\limits_{\|f\|=1} |X(f)|$
  - **完备性**：
- **C-S不等式**：$|X(f)|\leq \|f\|\cdot \|x\|$
  - **证明**：
    - 易得 $\cfrac{|X(f)|}{\|f\|} \leq \sup\limits_{f\neq \bd 0} \cfrac{|X(f)|}{\|f\|} = \sup\limits_{\|f\|=1} |f(x)| =  \|x\|$
- **内压性**：$\|X\|\leq \|x\|$
  - **证明**：
    - 由C-S不等式，$\|X\| = \sup\limits_{\|f\| = 1}\|X(f)\|\leq \sup\limits_{\|f\| = 1}\|f\|\cdot \|x\| = \|x\|$
  - **理解**：
    - $\|f\| = 1$ 表明 $f$ 的最大延展方向也只能保持范数不变，即 $f$ 是不增映射，那么显然此时 $|X(f)| = |f(x)| \leq \|x\|$

## 自反空间

- **自然映射**：$T: \ms X\to \ms X^{**}，x\mapsto X$，且满足等距性
  - **线性**：$T(x_1+x_2) = X_1 + X_2$
    - **证明**：
      - 设 $x = x_1+x_2$，则 $\lang T(x_1+x_2),f \rang = Xf = f(x) = f(x_1) + f(x_2)$
      - 由第二泛函的线性得 $(X_1+X_2)f = f(x_1)+f(x_2)$
      - 故等式两边相等（**证毕**）
  - **单射性**：
    - **证明**：
      - 反设 $Tx = Ty = X$，则 $\forall f\in\ms X^*$，都有 $f(x) = f(y) = X(f)$
      - 再由向量判零定理，得只能是 $x=y$
    - <font color='chartreuse'>如果只考虑某个 $f$ 的话，那么 $X$ 可以是等值面 $H^{f(x)}_f$ 上的所有点
    - 但要求对全体 $f\in\ms X^*$ 都成立，那么就是全体上述等值面的交集，显然只能是 $x$</font>
  - **非满射性**：
- **自然映射扩张定理**：$\ms X$ 和 $\ms X^{**}$ 的某个子空间等距同构
  - <font color='chartreuse'>实际上是和自然映射的像空间等距同构</font>
  - **证明**：
    - **同构性**：
      - $T$ 线性：已知
      - $T$ 单射性：已知
      - <font color='chartreuse'>实际上等距映射必为单射，所以也没必要提前得到单射</font>
    - **等距性**：
      - 已知第二泛函具有内压性，即 $\|X\|\leq \|x\|$
      - 由单点延拓定理，$\forall x$，$\exists f\in \ms X^*$ 使得 $\|f\| = 1$，$f(x) = \|x\|$
        - <font color='chartreuse'>其实就是 $f(x) = \|x\|$ 从单点延拓到整体</font>
        - 则此时 $\|x\| = |X(f)| \leq \|X\|\cdot\|f\| = \|X\|$
      - 综上，$\|x\| = \|X\|$，得 $T$ 等距性
  - **本质**：
    - <font color='red'>对称性：若 $f$ 是 $x$ 的泛函，则 $x$ 也可看作 $f$ 的泛函，即第二泛函
    - 此时 $f(x)$ 可写为内积形式 $\lang f,x \rang$，且满足双线性</font>
- **自反空间**：自然映射是满射的 $B^*$ 空间
  - <font color='chartreuse'>等距同构于第二共轭空间的空间</font>
  - **实例**：$L^p\quad (1<p<\infty)$
    - **第一泛函**：$\dis f: L^p\to\R，u\mapsto \|u\| = \dkh{\int_\Omega |u(x)|^pdu}^{\frac{1}{p}}$（只是一个示例，当然还有其它取法）
    - **第二泛函**：$\dis X: (L^p)^*\to \R，f\mapsto \lang f,u \rang = \dkh{\int_\Omega |u(x)|^pdu}^{\frac{1}{p}}$
    - **自然映射**：$T: L^p\to (L^p)^{**}，u\mapsto X$
    - **证明**：
      - 由Holder定理，$(L^p)^* = L^q$，从而 $(L^p)^{**} = (L^q)^* = L^p$
  - **反例**：
    - $L^1$
    - $L^\infty$
  - **理解**：
    - 即使自然映射是恒等映射，也不一定是满射
    - 比如 $\ms X$ 不完备，但 $\ms X^{**}$ 完备的情况
