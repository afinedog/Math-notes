# 共轭算子（转置矩阵）

## 共轭算子

- **共轭算子**：$B^*$ 空间中，设 $T\in\opxy$，若 $T^*:\funcy^*\to\ms X^*$ 满足转置性，则称为 $T$ 的共轭算子
  - **转置性（内积写法）**：$\forall f\in\funcy^*，\forall x\in\ms X$，都有 $\lang Tx,f \rang = \lang x,T^*f \rang$
    - <font color='chartreuse'>容易发现，这就是高等代数中的伴随矩阵（转置矩阵）</font>
  - **转置性（映射写法）**：$\forall f\in\funcy^*，\forall x\in\ms X$，都有 $f(Tx) = (T^*f)x$
    - <font color='chartreuse'>当算子进行交换时，应当取共轭</font>
  - **实例**：
    - **矩阵**：$\lang f,Ax \rang = \lang A^Tf,x \rang$
      - 其中 $A:\K^m\to \K^n$，$A^T:\K^n\to\K^m$，$f_j = (\bd x_{n\times 1})_j$
- **共轭算子的性质**：
  - **唯一性**：
    - **证明**：
      - 反设不唯一，则 $\lang Tx,f \rang = \lang x,T^*_1f \rang = \lang x,T^*_2f \rang$，即 $\forall x,f$ 都有 $\ip{x, (T_1^*-T_2^*)f } = 0$
      - 由向量判零定理得只能是 $T^*_1 = T^*_2$
  - **线性**：
    - **证明**：由内积的共轭双线性易得对 $\forall x\in\ms X$ 都有 $$\lang T^*(f+g),x \rang = \lang f+g,Tx \rang = \lang f,Tx \rang + \lang g,Tx \rang = \lang T^*f,x \rang + \lang T^*g,x \rang$$
      - 再由 $x$ 的任意性，即得 $T^*(f+g) = T^*f + T^*g$
  - **连续性**
    - **证明**：
      - 设 $f\in\ms Y^*$，且 $g(x) = f(Tx)$，由转置性即得 $T^*(f) = g$
        - <font color='chartreuse'>有界线性算子的算子范数必然有界，所以只需证明 $g$ 有界即可</font>
      - 由C-S不等式，$|f(Tx)| \leq \|f\|\cdot\|T\|\cdot\|x\|$，从而 $g$ 有界，即 $g\in \ms X^*$
      - 从而 $T^*\in\op(\funcy^*,\ms X^*)$
  - **内压性**：$\|T^*\| \leq \|T\|$
    - **证明**：
      - 设 $g = fT$
        - 易得 $\|T^*f\| = \sup\limits_{\|x\|=1}\|(T^*f)x\| = \sup\limits_{\|x\|=1}\|f(Tx)\| = \|g\|$
          - <font color='chartreuse'>我们不能直接处理约分后的 $\|T^*\|$，更好的办法是先处理 $\|T^*f\|$，得到中间不等式，然后再用它处理约分后的式子</font>
        - 再由C-S不等式，$\|g\| \leq \|T\|\cdot\|f\|$
          - <font color='chartreuse'>实际上这里的内压性证明思路和前面的第二泛函内压性是相同的，只不过比它多绕了一个弯</font>
        - 综上，$\|T^*f\| \leq \|T\|\cdot \|f\|$
      - 此时就有 $\|T^*\| = \sup\limits_{\|f\|=1}\|T^*f\| \leq \sup\limits_{\|x\|=1}\|T\|\cdot\|f\| = \|T\|$（**证毕**）

## 共轭映射

- **共轭映射**：$*:\opxy\to\opyx，T\mapsto T^*$
  - <font color='chartreuse'>将算子变为其共轭算子的映射</font>
  - **共轭线性**：设 $a,b\in\K，A,B\in\opxy$，则 $(aA+bB)^* = \ol aA^* + \ol bB^*$
    - **证明**：
      - 定义易得（或内积的共轭双线性易得）
  - **穿脱原理**：$(AB)^* = B^*A^*$
    - **证明**：定义易得
  - **可逆性**：若 $A^{-1}$ 存在且有界，则 $(A^*)^{-1}$ 存在且有界，且 $(A^*)^{-1} = (A^{-1})^*$
    - **证明**：定义易得
  - **等距同构性**：
    - **证明**：
      - 由内压性已知 $\|T^*\| \leq \|T\|$
      - 在 $Tx$ 处进行单点延拓可得到 $f$
        - <font color='chartreuse'>即 $f$ 是保范映射，且 $f(Tx) = \|Tx\|$</font>
      - 由定义 + C-S不等式得 $$\|Tx\| = f(Tx) = (T^*f)x \leq \|T^*f\|\cdot \|x\| \leq \|T^*\|\cdot\|x\|$$
      - 综上，$\|T^*\| = \|T\|$
- **共轭保范延拓定理**：$T^{**}$ 是 $T$ 在 $\ms X^{**}$ 上的保范延拓，即 $\|T^{**}\| = \|T\|$
  - **证明（延拓性）**：
    - 设 $U: \ms X\to\ms X^{**}$，$V: \funcy\to\funcy^{**}$ 都是自然映射，$f\in\ms Y^*$
    - 则 $\lang T^{**}Ux,f \rang = \lang Ux,T^*f \rang = \lang T^*f,x \rang = \lang f,Tx \rang = \lang VTx,f \rang$
      - <font color='chartreuse'>第一个等号：转置交换性
      - 第二个等号：由等距同构性，$Ux$ 和 $x$ 在 $T^*f$ 的作用下相同
        - （把“在 $f$ 作用下相同”这个概念进行扩展，可以引出弱收敛的概念）
      - 第三个等号：转置交换性
      - 第四个等号：由等距同构性，$VTx$ 和 $Tx$ 在 $f$ 的作用下相同</font>

### 自共轭算子

- **自共轭算子**：设 $H$ 是Hilbert空间，$A\in\ms L(H)$，若 $A = A^*$，则称为自共轭算子
  - <font color='chartreuse'>只有Hilbert空间上才存在自共轭算子</font>
  - **实例**：
    - $\R^{n\times n}$ 上的自共轭算子：对称方阵
    - $\C^{n\times n}$ 上的自共轭算子：Hermite方阵
    - $L^2(\R)$ 上的乘法算子
      - 设乘法算子 $F:x(t)\to f(t)x(t)$，其中 $f\in L^\infty$
      - 则 $F$ 自共轭 $\LR f$ 是实函数
      - **证明**：
        - 易得 $(Fx,y) = \dis\int_\R fx\ol y = \int_\R x\ol{(\ol fy)} = (x,F^*y)$
        - 故当且仅当 $\ol f = f$ 时满足自共轭性
    - $L^2(\Omega,\ms B,\mu)$ 上的积分算子：
      - 设 $K\in L^\infty(\Omega\times\Omega,d\mu)$，$K(x,y) = K(y,x)$
      - 则 $\dis A: u(x)\mapsto \int_\Omega K(x,y)u(y)d\mu(y)$ 是自共轭算子
      - **证明**：
    - **投影算子**：Hilbert空间上的正交投影算子是自共轭算子
      - **证明**：
        - 易得 $(P_Mx,y) = (x_M,y_M+y_{M^\perp}) = (x_M,y_M) = (x,P_My)$
- **自共轭算子的性质**：
  - **封闭性**：自共轭算子关于线性运算、复合运算封闭
  - **闭集性**：全体自共轭算子是闭集
    - **证明**：
      - 设自共轭算子列 $A_n\to A$，只需证明 $(Ax,y) = (x,Ay)$ 即可
      - 线性添项 + C-S不等式易得 $$|(Ax,y) - (x,Ay)| \leq 2\|A_n-A\|\cdot\|x\|\cdot\|y\|\to 0$$
  - **乘积交换性**：$AB$ 是自共轭算子 $\LR AB = BA$
    - **证明**：
      - 由自共轭定义即可
  - **二次型定理**：$A$ 是自共轭算子 $\LR \forall x\in H$，$(x,Ax)\in\R$
    - **证明**：
      - **必要性**：由自共轭性 + 内积的共轭双线性，$(x,Ax) = (Ax,x) = \ol{(x,Ax)}$，从而是实数
      - **充分性**：按照极分解计算可得  $4(x,Ay) = 4(Ax,y)$，从而自共轭
  - **自共轭范数定理**：若 $A$ 是自共轭算子，则 $\|A\| = \sup\limits_{\|x\| = \|y\| = 1} (Ax,y) = \sup\limits_{\|x\| = 1} (Ax,x)$
    - **证明**：
      - 即证 $\sup\limits_{\|x\| = 1} \|Ax\| = \sup\limits_{\|x\| = 1} (Ax,x)$
      - 由内积范数定义，$\|Ax\| = \sqrt{(Ax,Ax)}$
      - 计算较复杂，从略
- **Cartesian分解定理**：设 $H$ 是Hilbert空间，$T\in \op(H)$，则存在唯一的自共轭算子 $A,B$ 满足 $T = A+iB$
  - **证明**：
    - **存在性**：取 $A = \cfrac{T+T^*}{2}，B = \cfrac{T-T^*}{2i}$ 即可
    - **唯一性**：
      - 反设不唯一，则此时 $(A_1-A_2) = i(B_1-B_2)$
      - 即 $\forall x\in H$，都有 $\Big( (A_1-A_2)x,x \Big) = \Big( i(B_1-B_2)x ,x \Big)$
      - 由自共轭的线性封闭性 + 二次型定理，此时左边是实数，右边是虚数，故只能为 $0$
  - **实例**：
    - $\C$ 中，设 $T(z) = z$，则 $z = x+iy$

- **卷积算子**：
  - 设 $K\in L^1(\R)$，算子 $T:f\mapsto (K*f)$，其中 $f\in L^p(\R)，1\leq p\leq \infty$
  - 则
    - $T \in \ms L\Big(L^p(\R)\Big)$
    - 设 $\wt K(x) = K(-x)$，则共轭算子 $T^*:f\mapsto (\wt K*f)$
  - **引理（Young不等式）**：$\|K*f\|_p \leq \|K\|_1\cdot\|f\|_p$
  - **证明**：
    - 由Fubini定理，变形易得 $$\int_\R \Big[ (K*f)(x) \Big] g(x)dx = \int_\R \Big[ (\wt K*g)(y) \Big] f(y)dy$$
    - 即 $\lang K*f,g \rang = \lang \ol K*g,f \rang$，从而 $T^* = \wt K*$
  - **推论**：
    - 若将 $\R$ 换为 $\C$，则 $T^*:f\mapsto (\wt{\ol K}*f)$

## 习题

- **B自反传递性**：B空间 $\ms X$ 自反 $\LR\ms X$ 的共轭空间自反
  - **必要性**：设 $x_0^{***}\in \ms X^{***}$，即证 $\exist x^*\in\ms X^*$，使得 $\lang x_0^{***}, x^{**} \rang = \lang x^{**},x_0^* \rang$
    - 设自然映射为 $J$，则 $J^*: \ms X^{***}\to\ms X^*$，左式 $= \lang x_0^{***},Jx \rang$
    - **限制等价性**：原式 $= \lang J^*x_0^{***},x \rang = \lang x_0^*,x \rang$
      - **理解**：本质上就是把第二共轭空间的算子限制到了原空间中。而因为空间自反，所以限制后等价
    - 再由限制等价性，原式 $=\lang x^*,x^{**} \rang$，即得右式
  - **充分性**：第二共轭空间延拓性 + B空间闭集性 + Pettis定理直得
  - **本质**：必要性是限制等价，充分性是嵌入等价
    - 限制等价只需要自反
    - 嵌入等价则需要闭集性（完备性）
  - **推论**：这么说我的猜想没错，第二共轭空间大于原空间的情况中，肯定有完备性的区别
- **闭集传递性**：设 $\ms X$ 是 $B^*$ 空间，$T$ 是自然映射
  - 则 $R(T)$ 是闭集 $\LR\ms X$ 完备
  - **证明**：
    - **必要性**：取自然映射的共轭算子，利用其逆性
    - **充分性**：自然映射连续嵌入性直得
  - **本质**：B空间自反传递性的一个子命题罢了
