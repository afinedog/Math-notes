# 衍生空间

- 为啥要把共轭空间和弱收敛放在一起：
  - 因为自反空间拥有延拓性、对偶对称性
  - 而弱收敛又是将原像性质传到函数性质上的媒介之一

## 共轭空间（对偶空间）

- **$\funcx$ 的共轭空间**：$B^*$ 空间 $\funcx$ 上的连续线性泛函全体 $\funcx^*$
  - **共轭范数**：$\|f\| = \sup\limits_{\|x\|=1}|f(x)|$
  - **完备性**：其构成 $B$ 空间

### 实例

- $L^p[0,1]\quad (1\leq p<\infty)$ 的共轭空间
  - 若 $\begin{cases} \frac{1}{p}+\frac{1}{q} = 1 ，p>1\\ q = \infty，\wh p=1 \end{cases}$
  - 则 $L^p[0,1]^* = L^q[0,1]$
  - **证明（详见抽象测度论folland）**：
    - 设 $f\in L^p[0,1]$，$g\in L^q[0,1]$
    - **第一泛函**：$F_g: L^p[0,1]\to \R ，f\mapsto \int^1_0 f(x)g(x)d\mu$
      - **线性**：易得
      - **连续性**：由Holder不等式易得
    - **同构映射**：$h: L^q[0,1]\to L^p[0,1]^*，g\mapsto F_g$
      - **线性**
      - **连续性**：由 $\|F_g\|\leq \|g\|$，$h$ 有界（连续）
      - **等距性**
  - **推论**：$h$ 是共轭空间的自然映射（等距同构）
  - **本质**：$F_g$ 类似欧氏空间中的固定一个元后的内积

### 第二共轭空间（对偶空间的共轭空间）

- **$\funcx$ 的第二共轭空间**：$B^*$ 空间的连续线性泛函的连续线性泛函 $\funcx^{**}$
  - **第二泛函 $X\in\funcx^{**}$**：
    - **定义**：给定 $x\in \funcx$，可定义 $X: \funcx^*\to \R，f \mapsto \lang f,x \rang$
    - **对称性**：若 $f$ 是 $x$ 的泛函，则 $x$ 也可看作 $f$ 的泛函，即第二泛函
  - **线性运算**：$(X_1+X_2)f = f(x_1) + f(x_2)$
  - **范数**：$\|X\| = \sup\limits_{f\neq \bd 0} \cfrac{|X(f)|}{\|f\|} =  \sup\limits_{\|f\|=1} |X(f)|$

### 自反同构性

- **自然映射**：$T: \funcx\to \funcx^{**}，x\mapsto X$
  - **等距同构性**：原空间和（第二共轭空间的某个子空间）等距同构
    - **证明**：
      - **同构性**：
        - $T$ 线性：只需证 $T(x_1+x_2) = X_1 + X_2$ 即可
          - 由映射一值性，只需证明像相同
          - $\lang T(x_1+x_2),f \rang = Xf = f(x)$。由 $X$ 定义即得 $x = x_1+x_2$
          - $(X_1+X_2)f = f(x_1)+f(x_2)$
        - $T$ 非单射性：
          - 事实上，$T$ 可以将 $x$ 映射成 $f$ 的 $f(x)$ 等值面 $\funcx/N(f)$ 中的任意一个向量
          - 但是不要紧，再加上一个等距性，就限定了像的范围，从而蕴含了单射性
      - **等距性**：
        - 由C-S不等式，$\|X\| = \sup\limits_{\|f\| = 1}\|X(f)\|\leq \|f\|\cdot \|x\| = \|x\|$
        - 由H-B定理，每个 $x\in\funcx$ 存在单点延拓函数 $f$ 使得 $\|f\| = 1$，$\lang f,x \rang = \|x\|$
          - 则此时 $\|x\| = X(f) \leq \|X\|\cdot\|f\| = \|X\|$
        - 综上，$\|x\| = \|X\|$，得 $T$ 等距性
    - **理解**：该形式的映射可以有很多种，但只有等距同构的才是自然映射
    - **自反空间**：自然映射是满射<font color='chartreuse'>（原空间和第二共轭空间同构）</font>
      - **实例**：$L^p\quad (1<p<\infty)$
        - **对偶空间元素**：$f: L^p\to\R，u\mapsto$
          - 举例：$\|u\| = (\int_\Omega |u(x)|^pdu)^{\frac{1}{p}}$（当然还有其它映射）
        - **第二泛函**：$X: L^{p*}\to \R，f\mapsto \lang f,u \rang \color{red} = $
        - **自然映射**：$T: L^p\to L^{p**}，u\mapsto X$
      - **反例**：$L^p\quad (p=1,\infty)$
      - **理解**：
        - 即使自然映射是恒等映射，也不一定是满射
        - 比如 $\funcx$ 不完备，但 $\funcx^{**}$ 完备

#### 习题

- **Hilbert空间自反**
  - **满射证明**：Riesz表示定理得 $f(x) = (x,y_f)\quad (\forall x\in\funcx)$，从而 $y_f$ 和 $f$ 一一对应
    - 仿照欧氏空间的证明，即可得到自反性
    - **推论**：Hilbert空间的共轭空间均与自身等距同构
  - **内积证明**：同下
- **有限维 $B^*$ 空间自反**
  - **证明**：
    - 设 $\{e_k\}$ 是共轭空间规范正交基，$\{f_k\}$ 是任意一组基
    - $\lang x^{**},f \rang = \lang x^{**},\sum\limits^n_{k=1} \lang f,e_k \rang f_k\rang = \sum\limits^n_{k=1} \lang f,e_k \rang\lang x^{**},f_k \rang = \lang f,\sum\limits^n_{k=1} \lang x^{**},f_k \rang e_k \rang$
    - 设 $x = \sum\limits^n_{k=1} \lang x^{**},f_k \rang e_k$，则上式为 $\lang x^{**},f \rang = \lang f,x \rang$，得到一一对应关系，从而自反
  - **理解**：展开 $f$ 将内积对换，从而将 $x^{**}$ 转化为内积形式，即得到 $x$，从而建立对应关系
  - **本质**：用基来使问题标准化。由于研究对象是 $x$，故使用的是 $f$ 的基


### 共轭算子（转置矩阵）

- **$T\in\opxy$ 的共轭算子**：$B^*$ 空间中，$T^*:\funcy^*\to\funcx^*$ 满足转置性
  - **转置性**：$\forall f\in\funcy^*,x\in\funcx，f(Tx) = (T^*f)x$
    - <font color='chartreuse'>将算子映射成其逆</font>
    - **共轭形式**：$\lang Tx,f \rang = \lang x,T^*f \rang$
  - **推论**：
    - **$T^*$ 连续线性**
      - **证明**：设 $g(x) = f(Tx)$
        - 线性易得
        - 由 $f(Tx) \leq \|f\|\cdot\|T\|\cdot\|x\|$ 得 $g$ 有界（连续）
          - 且 $\|T^*\| \leq \|T\|$
        - 再由转置性，$T^*: f\mapsto g$，从而 $T^*\in\op(\funcy^*,\funcx^*)$
    - **等距同构性**：$*:\opxy\to\opyx，T\mapsto T^*$ 等距同构
      - **证明**：
        - **$*$ 线性**：
        - **$*$ 等距性**：
          - $\|T^*\| \leq \|T\|$ 已知
          - 对 $Tx$ 单点延拓得 $f$
            - $\|Tx\| = f(Tx) = (T^*f)x \leq \|T^*f\|\cdot \|x\| \leq \|T^*\|\cdot\|x\|$ 
    - **自反扩张性**：$T^{**}$ 是 $T$ 在 $\funcx^{**}$ 上的延拓，且 $\|T^{**}\| = \|T\|$
      - **证明**：
        - 设自然映射 $U: \funcx\to\funcx^{**}$，$V: \funcy\to\funcy^{**}$
        - 则 $\lang T^{**}Ux,f \rang = \lang Ux,T^*f \rang = \lang T^*f,x \rang = \lang f,Tx \rang = \lang VTx,f \rang$
          - <font color='chartreuse'>1：原像 $\xto{U}$ 延拓原像 $\xto{T^{**}}$ 像 $\xto{f}$ 原像
          - 2：原像 $\xto{U}$ 延拓原像 $\xto{T^*f} $
          - 3：
          - </font>
          - 共轭性（$T^*f = fT^{**}$） + 延拓性（$Ux = x$） + 共轭性（$T^*f = fT$） + 延拓性（$VTx = Tx$）
          - <font color='chartreuse'>第三个共轭性也等价于 $T^{**}x = Tx$</font>
        - 则 $T^{**}(Ux) = V(Tx)$
      - **理解**：利用共轭性，不断交换的同时可以不断消去或添加星号
      - **本质**：自然映射延拓性的传递
  - **实例**：
    - **矩阵**：
      - $A:\fk^m\to \fk^n$
      - $A^T:\fk^n\to\fk^m$
      - $f_j = (\bd x_{n\times 1})_j$
      - **共轭关系**：$\lang f,Ax \rang = \lang A^Tf,x \rang$
- **自然映射的共轭算子**：$J^*: \funcx^{***}\to\funcx^*$
  - **证明**：设 $f:\funcx^{**}\to \funcy$，则由共轭性 $fJ = J^*f$
    - 左边是复合关系，$\funcx\to\funcx^{**}\to\funcy$
    - 右边是作用关系，$J^*$ 的定义域在对偶空间上，故不是复合映射。为了与左边等同，需要 $J^*f:\funcx\to\funcy$，即 $J^*f\in \funcx^*$，即得结论
  - **理解（消共轭性）**：自然映射将算子 $x$ 映射成 $x^{**}$，故自然共轭映射将自然映射变成它的逆算子，即将 $x^{**}$ 映射成 $x$ 的算子，故减少了星号（加一个减两个，总体减了一个）

## 无穷维下的收敛

### 弱收敛

#### 点列收敛性

- **弱收敛**：依连续线性泛函收敛
  - $B^*$ 空间 $\funcx$ 中的点列 $\{x_n\}$
  - 若 $\forall f\in\funcx^*，\lim\limits_{n\to\infty} f(x_n)\to f(x)$
  - 则称该点列弱收敛
- **弱极限**：$x_n\xto{w} x$
  - **唯一性**：若存在则唯一
    - **证明**：反设不唯一，则 $\forall f\in\funcx^*，f(x) = \lim\limits_{n\to\infty} f(x_n) = f(y)$
      - 由零元判别法得 $x-y = \t$
- **强收敛**：依范数收敛
  - **强弱关系**：
    - 强极限必定是弱极限
    - 有穷维上，强弱收敛等价
    - **证明**：
      - 若强收敛，由 $\forall f\in\funcx^*，|f_n(x)-f(x)| \leq \|f\|\cdot \|x_n-x\|\to 0$，从而弱收敛
      - 有穷维则存在基 $\{e_i\}$
      - 每个 $\{x_n\}$ 对应一组系数 $\{\xi^{(n)}_i\}$，即证等价于按坐标收敛
    - **反例**：$L^2[0,1]$ 中，$x_n = \sin n\pi t$
      - 由Riemann-Lebesgue定理，$\forall f，\lang f,x_n \rang = \int^1_0 f(t)\sin n\pi tdt\to 0$
      - 即 $x_n\xto{w} \t$，弱收敛
      - 再由 $\|x_n\| = \dfrac{1}{\sqrt{2}}$，不强收敛
      - **理解**：$sin$ 的积分消项性
      - **本质**：函数不收敛，但反常积分可收敛的情况
- **Mazur定理（凸集的弱极限完备性）**：
  - 设 $\funcx$ 是 $B^*$ 空间，$x_n\xto{w} x_0$
  - 则 $\forall \varepsilon > 0，\exist \l_i\geq 0，\sum\limits^n_{i=1} \l_i = 1$，使得 $\|x_0 - \sum\limits^n_{i=1}\l_i x_i\|\leq \varepsilon$
  - **证明**：设 $M = \ol{co(\{x_n\})}$，则其为闭凸集
    - 反设 $x_0\notin M$，则由Ascoli定理，$\exist f\in\funcx^*，\a\in\reals$，使得 $\forall n\in\natnums，\\ f(x_n) < \a < f(x_0)$，与弱收敛矛盾
  - **理解**：若弱极限不在凸包内，则可被 $f$ 分离，且有中间值，无法逼近
  - **本质（Ascoli引理的直接推论）**：弱极限可被凸组合逼近，即凸集上的弱极限一定在凸集内部
- **弱收敛定理（共轭双射定理）**：设 $\funcx$ 是 $B^*$ 空间，$M^*$ 是$\funcx^*$ 的稠密子集
  - $x_n\xto{w} x \LR \|x_n\|$ 有界，且 $\forall f\in M^*$ 满足 $\lim\limits_{n\to\infty} f(x_n) = f(x)$
  - **证明**：
    - 将 $x_n$ 看作定义在 $\funcx^*$ 上的有界线性泛函（单值映射）
    - 应用Banach-Steinhaus定理即可
- <font color='chartreuse'>连续线性泛函也可以有类似的结论，在性质前加 $*$</font>

#### 算子收敛性总结

- $B^*$ 空间内的 $T_n,T\in\opxy$
  - **一致收敛**：若 $\|T_n-T\|\to 0$，则 $T_n \tto T$
  - **强收敛**：若 $\|(T_n-T)x\|\to 0\quad (\for x\in\funcx)$，则 $T_n \to T$
  - **弱收敛**：若 $\lim\limits_{n\to\infty} f(T_nx) = f(Tx)\quad (\for x\in\funcx,f\in\funcy^*)$，则 $T_n\xto{w} T$
  - **反例**
    - 强收敛但不一致收敛：
      - $l^2$ 上的左推移算子 $T$（将坐标向左移动一格）
      - 设 $T_n = T^n$，则 $T_n\to 0$，但 $T_n\not\tto 0$
      - **证明**：设自然基为 $e_n$，则
        - $\|T_n\| \geq \|T_n(e_{n+1})\| = 1$
        - $\|T_nx\| = \Big(\sum\limits^\infty_{i=1} |x_{n+i}|^2 \Big)^{\dfrac{1}{2}}\to 0$
    - 弱收敛但不强收敛：
      - $l^2$ 上的右推移算子 $S$
      - 设 $S_n = S^n$，则 $S_n\xto{w} 0$，但 $S_n\nto 0$
      - **证明**：
        - $\|S_nx\| = \|x\|\nto 0$
        - $|\lang f,S_nx \rang| = |\sum\limits^\infty_{i=1}y_{i+n}x_i| \leq \Big(\sum\limits^\infty_{i=1} |x_{n+i}|^2 \Big)^{\dfrac{1}{2}} \|x\|\to 0$

- **Banach可分定理**：$B^*$ 空间 $\funcx$，若其共轭空间可分，则其也可分
  - **证明**：
    - **对偶单位球面为 $S_1^*$ 可分**：
      - 由 $\funcx^*$ 可分，$\exist f_n\in\funcx^*$，使 $\forall f\in S_1^*，\exist \{n_k\}^\infty_{k=1}$ 满足 $\lim\limits_{k\to\infty} f_{n_k} = f$
      - 设 $g_n = \cfrac{f_n}{\|f_n\|} \in S_1^*$
        - 由三角不等式，$\|f-g_{n_k}\| \leq \|f-f_{n_k}\| + |1-\|f_{n_k}\|| \to 0$
        - 从而 $\{g_n\}$ 是 $S_1^*$ 的可数稠密子集
    - **单位球某点列线性包 $\funcx_0$ 可分**：
      - 由 $\|g_n\| = \sup\limits_{\|x\|=1} g_n(x) = 1$，得 $\exist x_n\in \funcx$ 满足 $\begin{cases} \|x_n\| = 1 \\ g_n(x_n) \geq \frac{1}{2} \end{cases}$
      - 设 $\funcx_0 = \ol{span\{x_n\}}$，可数稠密子集为 $span\{x_n\}$
    - **线性包极大性** $\funcx_0 = \funcx$：
      - 反设 $\exist x_0\in\funcx-\funcx_0$，则由流形延拓定理，$\exist f_0\in S_1^*$ 满足 $f_0(x) = 0\\ (\forall x\in\funcx_0)$<font color='chartreuse'>（也就是 $\funcx_0$ 的距离函数）</font>
      - 则 $\|g_n-f_0\| \geq \|g_n\|+\|f_0\| =  |g_n(x_n)| \geq \frac{1}{2}$，与 $\{g_n\}$ 稠密性矛盾
  - **理解**：可分性等价于（某点列可逼近所有点）（存在可数基）
    - 首先我们知道，单位球的线性包一定可以张成整个空间（空间与单位球基数相同）
      - 故可以将整个空间化简为讨论单位球
      - 单位球上的像就是算子范数，故可将对偶表出性和原空间表出性联系起来
    - 然后，单位球上的可数集不一定能张成整个空间

      - 流形延拓的作用就是找一个核在可数集上的泛函 $f$。由于可数集在共轭空间中是基，但在 $f$ 下完全丧失表出的能力，故产生矛盾。
      - <font color='chartreuse'>虽然证明不是这个意思，但大差不差，只是手法和顺序的区别而已</font>
  - **本质**：流形延拓将向量和算子和极大性联系起来
