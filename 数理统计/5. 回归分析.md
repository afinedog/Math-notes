# 回归分析

- **回归模型**：
  - 因变量（随机变量）$Y$ 与自变量（样本变量） $\{X_k\}$ 存在相关关系
  - 相关关系可用函数表示，其它因素的影响视为随机误差
  - **回归函数**：$f(X_1,...,X_k)$
  - **回归方程**：$Y = f(X_1,...,X_k) + \mu$

## 一元线性回归

- **一元线性回归模型**：$Y_i = \b_0 + \b_1 X_i + \mu_i$
  - **随机误差项**：$\mu_i$，假定为正态分布 $N(0,\sigma^2)$
  - **总体回归方程**：$E(Y_i) = \b_0 + \b_1 X_i$
- **样本回归模型**：已知样本 $\{X_j,Y_j\}$ 后，$Y_j = \b_0 + \b_1 X_j + e_j$
  - **残差项**：$e_j$
  - **样本回归方程**：$\hat{Y_j} = \hat\b_0 + \hat\b_1 X_j$

### 最小二乘估计

- **最小二乘法**：将（使得误差平方和最小的估计）作为最优估计
  - **偏差平方和**：$Q(\b_0,\b_1) = \sum\limits^n_{i=1} \mu^2_i = \sum\limits^n_{i=1} (Y_i-\b_0-\b_1 X_i)$
  - **正规方程组**：$\begin{cases} \cpfrac{Q(\hat\b_0,\hat\b_1)}{\hat\b_0}  = 0 \\\\ \cpfrac{Q(\hat\b_0,\hat\b_1)}{\hat\b_1} = 0 \end{cases}$
  - **解**：$\begin{cases} \hat\b_1 = \cfrac{n\sum\limits^n_{i=1} X_iY_i - \sum\limits^n_{i=1} X_i \sum\limits^n_{i=1} Y_i}{n\sum\limits^n_{i=1} X_i^2 - (\sum\limits^n_{i=1} X_i)^2} = \cfrac{\sum\limits^n_{i=1} (X_i-\ol X)(Y_i-\ol Y)}{ \sum\limits^n_{i=1} (X_i-\ol X)^2 } = \cfrac{S_{xy}}{S_{xx}} \\\\ \hat\b_0 = \cfrac{1}{n}(\sum\limits^n_{i=1} Y_i - \hat\b_1\sum\limits^n_{i=1} X_i ) = \ol Y - \hat\b_1\ol X\end{cases}$
- **最小二乘定理**：
  - $D(Y_i) = \sigma^2$
    - **证明**：由定义，原式 $= E(Y_i-EY_i)^2$
      - 由回归方程，得原式 $ = E(\mu_i)^2$
      - 由假设 $\mu_i\sim N(0,\sigma^2)$，原式 $ = E(\mu_i-E\mu_i)^2 = D(\mu_i) = \sigma^2$
  - $\hat\b_0，\hat\b_1$ 是无偏估计
    - **证明**：
      - 首先 $\sum\limits^n_{i=1} (X_i-\ol X) = \sum\limits^n_{i=1} (Y_i-\ol Y) = 0$
        - 只需要将算术均值与加和形式互化，便可以融入和融出 $\sum$
      - 然后 $\sum\limits^n_{i=1} (X_i - \ol X) X_i = \sum\limits^n_{i=1} (X_i-\ol X)^2$
      - 有了这两个性质，直接计算均值即可
  - $cov(\ol Y,\hat\b_1) = 0$
  - $D(\hat\b_0) = \sigma^2(\cfrac{1}{n} + \cfrac{\ol X^2}{S_{xx}})$
  - 
  - $cov(\hat\b_0, \hat\b_1) = -\ol X\cfrac{\sigma^2}{S_{xx}}$
  - 
  - $D(\hat\b_1) = \cfrac{\sigma^2}{S_{xx}}$
  - **理解**：
    - 利用 $\{X_i\}$ 均为常数，$\{Y_i\}$ 均为非常数即可
- **残差平方和**：$S_e = Q(\hat\b_0,\hat\b_1) =\sum\limits^n_{i=1} e_i^2 = \sum\limits^n_{i=1} (Y_i-\hat Y_i)^2$
- **方差回归定理**：$E(S_e) = (n-2)\sigma^2$
  - **证明**：
  - **剩余方差**：残差的方差 $\hat\sigma^2_e = \cfrac{S_e}{n-2}$ 是无偏估计

### 显著性检验

- **假设**：$H_0: \b_1 = 0$
- **总偏差平方和**：$S_T = \sum\limits^n_{i=1} (Y_i-\ol Y)^2 = S_{yy}$
- **回归平方和**：$S_R = \sum\limits^n_{i=1} (\hat Y_i - \ol Y)^2$
- **平方和分解定理**：$S_T = S_R + S_e$
  - $S_e \sim \sigma^2\chi^2(n-2)$
  - $S_R \sim \sigma^2\chi^2(1)$
  - **检验统计量**：$F = \cfrac{S_R}{S_e/(n-2)}$，在 $H_0$ 成立时 $\sim F(1,n-2)$

### 预测

#### 点预测

- **点预测**：将 $X$ 的给定值 $X_0$ 代入样本回归方程，计算估计值作为点预测值
- **无偏性**

#### 区间预测

- **区间预测**：求出点预测值后，在置信度 $\a$ 下，求 $Y_0$ 落在以 $\hat Y_0$ 为中心的什么区间

### 控制

- **控制**：找出 $X$ 的一个区间 $[X_L,X_U]$，使随机变量以至少 $1-\a$ 的概率在 $[Y_L,Y_U]$ 内
