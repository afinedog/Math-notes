# 典型分布

- 参考教材：山大胡发胜讲义

## 经常会用到的量

- **算术平均值**：$\overline{X} = \dfrac{1}{n}\sum\limits^n_{i=1}X_i$
- **算术方差**：$S^2_n = \dfrac{1}{n}\sum\limits^n_{i=1} (X_i-\overline{X})^2$
- **无偏方差**：$S^2 = \dfrac{1}{n-1}\sum\limits^n_{i=1} (X_i-\overline{X})^2$
- **算术协方差**：$S^2_{XY} = \dfrac{1}{n}\sum\limits_{i=1} (X_i-\ol X)(Y_i-\ol Y)$

## 常见的分布

- **简单随机样本**：
  - 代表性：等可能抽取
  - 独立性：独立抽取

### 卡方分布

- **卡方分布 $\chi^2$**：设 $\{X_i\}^n_{i=1}\sim N(0,1)$ 且相互独立，则 $\chi^2 = \sum\limits^n_{i=1} X_i^2$ 服从卡方分布
  - **再生性**：$Y，Z$ 相互独立时， $Y+Z \sim \chi^2(n_1+n_2)$
    - **证明**：
      - 由于 $Y,Z$ 服从卡方分布，它们等价于 $\sum\limits^{n_1}_{i=1} X_i^2$，$\sum\limits^{n_2}_{j=1} X^2_j$
        - 其中 $\forall X_k\sim N(0,1)$ 且相互独立
      - 从而 $Y+Z = \sum\limits^{n_1+n_2}_{k=1} X_k^2$，由卡方分布定义即得 $Y+Z\sim \chi^2(n_1+n_2)$
    - **本质**：独立同分布即为随机变量的等价关系
  - **概率密度函数**：$\chi^2(x,n) = \begin{cases} \large\frac{1}{2^{\frac{n}{2}} \Gamma(\frac{n}{2})} x^{\frac{n}{2}-1}e^{-\frac{x}{2}}，x>0 \\ \\\large 0\whh\wh\quad ，x\leqslant 0\end{cases}$
    - **证明**：数学归纳法
      - $n=1$，$\chi^2(x,1) \sim [N(0,1)]^2$，分布函数映射定理 + 积分换元即可
      - $n=k$，分解为 $Y\sim \chi^2(n-1)，Z\sim \chi^2(1)$，相加后由伽马函数的性质求密度函数即可
  - **均值**：$E(X) = \sum\limits^n_{i=1} E(X^2_i) = \sum\limits^n_{i=1}D(X_i) = n$
  - **方差**：$D(x)= \sum\limits^n_{i=1}D(X^2_i) = \sum\limits^n_{i=1}\big[ E(X^4_i) - (EX_i^2)^2 \big] = 2n$
    - 正态分布均值可配凑为Gamma函数
    - $E(X_i^4) = 2\int^{+\infty}_0 x^4\cdot f(x)dx = \dfrac{2}{\sqrt{\pi}}\Gamma(\dfrac{5}{2}) = 3$
- **柯赫仑定理**：$Q_j$ 是秩为 $n_j$ 的 $X_1,...,X_n$ 的二次型
  - 若 $Q = \sum\limits^k_{j=1}Q_j = \sum\limits^n_{i=1} X^2_i$
  - 则
    - $Q_j$ 相互独立
    - $Q_j\sim \chi^2(n_j) \Leftrightarrow \sum\limits^k_{j=1} n_j = n$
  - **证明**：
    - **必要性**：由卡方分布再生性直得
      - **本质**：分量对应证总体对应
    - **充分性**：
      - 将 $Q_i$ 化为标准型
        - $Q_i = Y_i^T\cdot B_{n_i\times n_i}\cdot Y_i = \sum\limits^{n_i}_{i=1} b_{ij}Y^2_{ij}$
          - 其中 $B$ 为标准型矩阵（$B_i = diag(\pm 1)$）
      - 再由 $Q$ 定义，$Q = X^TIX = \sum\limits^k_{i=1} Y^T_i B_{n_i\times n_i}Y_i$
        - 右式也是一个二次型
        - 再由 $\{B_i\}_k$ 均为对角矩阵，得右式可重合为 $Y^TBY$，且 $b_{kk} = \pm 1$
      - 由坐标变换存在性，可设 $Y = AX$
        - 则 $Q = X^TX = X^T(A^TBA)X$
        - 从而 $A^TBA = I$，即标准型矩阵 $B$ 正定
        - 再由 $b_{kk} = \pm 1$ 即得 $A$ 是正交矩阵
      - 再由 $X_i\sim N(0,1)$，得 $X\sim N(0,I)$，且 $Y\sim N(0,AA^T = I)$
        - 从而 $Y$ 的各分量 $Y_i$ 独立且服从标准正态分布，从而 $Q_i$ 独立且服从卡方分布
      - **本质**：总体对应证分量对应
  - **理解**：
    - 若一组二次函数 $Q_j(X_1,...,X_{n_j})$ 之和等于 $X_1^2+...+X^2_n$，则这些二次函数相互独立，且
  - **本质**：$X_k^2$ 的二次型分量之和，若总体 $Q$ 对应卡方分布，则各分量 $Q_j$ 也对应卡方分布

### $t$ 分布

- **自由度为 $n$ 的 $t$ 分布 $T\sim t(n)$**：$\begin{cases} X\sim N(0,1) \\ Y\sim\chi^2(n) \\ T =\frac{X}{\sqrt{Y/n}} \end{cases}$
  - 等价于 $T = \cfrac{X}{\sqrt{\dfrac{X^2 + ... + X^2}{n}}}$
  - **本质**：标准正态分布变量 $X$ 除以其平方均值 $\sqrt{Y/n}$
<br>

- **密度函数**：$\dis t(x,n) = \frac{\Gamma(\frac{n+1}{2})}{\Gamma(\frac{n}{2}\sqrt{n\pi})} (1+\frac{x^2}{n})^{\large -\frac{n+1}{2}} $
  - 关于平方均值 $Y$ 对称
  - 原点极限 $\lim\limits_{|x|\to 0}t(x,n) = 0$
  - 阶数极限 $\lim\limits_{n\to\infty}t(x,n) =  \frac{1}{\sqrt{2}\pi} e^{-\frac{x^2}{2}}$（30以上就几乎等于正态分布）
  - 无穷远处略高于正态分布
  - $n>2$ 时，$EX = 0，DX = EX^2 = \frac{n}{n-2}$

### $F$ 分布

- **自由度是 $(m,n)$ 的 $F$ 分布**：$F\sim f(m,n) = \begin{cases} X\sim \chi^2(m) \\ Y\sim\chi^2(n) \\ F =\frac{X/m}{Y/n} \end{cases}$
  - **本质**：两个（标准正态分布变量的算术均值）之比
<br>

- **密度函数**：$f(x,m,n) = \begin{cases} \dis\frac{\Gamma(\frac{m+n}{2})}{\Gamma(\frac{m}{2})\Gamma(\frac{n}{2})}(\frac{m}{n})(\frac{m}{n}x)^{\frac{m}{2}-1}(1+\frac{m}{n}x)^{-\frac{m+n}{2}}\ \ , x>0 \\  \\ \qquad\qquad\qquad 0 \whh\whh\qquad\quad ,x\leq 0 \end{cases}$
<br>

- **倒数性**：$\dfrac{1}{F(m,n)}\sim F(n,m)$
  - **证明**：分布的映射传递性直得
- **正态平方性**：$t^2(n) \sim F(1,n)$
  - **证明**：$T^2 = \cfrac{X^2/1}{Y/n}$，再由分布的映射传递性即得结论

### 分位数

- **下侧 $\alpha$ 分位数**：分布函数值为 $\alpha$ 的点，$P(X\leqslant y_\alpha) = \alpha$
- **上侧 $\alpha$ 分位数**：分布函数值为 $1-\alpha$ 的点，$P(X > y_\alpha) = \alpha$
  - <font color='chartreuse'>用的比较多</font>
- **离散分位数**：离散分布下
  - 上侧分位数：$x_\a = \inf\{x\mid P(X>x)\}$
  - 下侧分位数：$x_\a = \inf\{x\mid P(X\leqslant x)\}$
- **无分位数**：
  - **近似定理**：
    - $\sqrt{2\chi^2(n)} - \sqrt{2n-1}\overset{L}{\to} N(0,1)$
      - **证明**：
      - **推论**：$\chi^2(n) \approx \frac{1}{2}(\mu_n + \sqrt{2n-1})^2$
    - $t(n) \to N(0,1)\quad (n\to\infty)$
      - **推论**：$t_n\approx \mu_n$

## 抽样分布

- **抽样分布**：某个样本中，各个 $X_i$ 的统计量的分布
  - 若总体均服从同一个分布，则显然（每个样本的分布）与（总体分布）相同
    - 则抽样分布为 $n$ 个独立同分布随机变量的联合分布
- **求解方法**：
  - **精确法**：总体分布已知，则求取 $L(X_1,...,X_n)$ 即可
  - **渐进法**：求取 $\lim\limits_{n\to\infty} L(X_1,...,X_n)$ 即可。此时一些较小的量被忽略，从而形式和计算被简化

### 正态总体的性质

- **正态分布单样本性质**：设总体服从正态分布，抽取一次，样本为 $X_1,...,X_n$，则其性质如下
  - **算术均值分布**：$\overline{X} = \dfrac{1}{n}\sum\limits^n_{i=1}X_i \sim N(\mu,\frac{\sigma^2}{n})$
    - **证明**：
      - **分布**：由样本和总体的分布相同 + 分布的映射传递性
      - **统计量**：均值、方差线性组合公式即可
  - **错位性**：$\cfrac{nS_n^2}{\sigma^2} \sim \chi^2(n-1)$
    - **证明**：
      - 设 $Y_i$ 为 $X_i$ 的标准化
        - 则由算术均值分布得 $\sqrt{n}\ \ol Y \sim N(0,1)$
      - **2-偏移量总和**：$Q_1 = \dfrac{nS_n^2}{\sigma^2} = \sum\limits^n_{i=1} \dfrac{(X_i-\ol X)^2}{\sigma^2} = \sum\limits^n_{i=1} (Y_i-\ol Y)^2$
      - **单卡方分布**：$Q_2 = (\sqrt{n}\ \overline{Y})^2$
      - 计算可得 $Q_1 + Q_2 = \sum\limits^n_{i=1}Y_i^2$，服从卡方分布
      - 两个二次型的秩为 $n-1$ 和 $1$
        - 由标准型的存在性，求解 $Q_i = X^TI'X = 0$ 即可得标准型的秩
      - 则由柯赫仑定理得，$Q_1,Q_2$ 相互独立且服从卡方分布
    - **理解**：可以通过分布性质来理解统计量二次型的秩
    - **本质**：由于算术均值分布不是一般的正态分布，故其等式会发生一定错位
  - $X\overline{X}$ 与 $S^2$ 独立，且 $\cfrac{\sqrt{n}(\overline{X}-\mu)}{S} \sim t(n-1)$
    - **证明**：
      - **独立性**：
        - $S^2 = \dfrac{\sigma^2 Q_1}{n-1}$
        - $\ol X = \sigma\sqrt{\dfrac{Q_2}{n}} + \mu$
        - **本质**：$S^2$ 可用 $Q_1$ 表出，$\overline{X}$ 可用 $Q_2$ 表出，则独立性被映射传递
      - **统计量**：分布的映射传递性，计算即可
- **正态分布多样本性质**：设总体为正态分布，抽取两次，样本为 $X_1,...,X_m$，$Y_1,...,Y_n$
  - $\cfrac{S^2_X / \sigma^2_1}{S^2_Y / \sigma^2_2} \sim F(m-1,n-1)$
    - **证明**：错位性直得
  - 总体方差 $\sigma_1^2 = \sigma_2^2$ 时
    - $\cfrac{(\overline{X} - \overline{Y}) - (\mu_1-\mu_2)}{S_W\sqrt{\frac{1}{m} + \frac{1}{n}}} \sim t(m+n-2)$
      - 其中 $S_W^2 = \cfrac{(m-1)S^2_X + (n-1)S^2_Y}{(m-1)+(n-1)}$
    - **证明**：
      - 由正态分布再生性，$\ol X-\ol Y\sim N(\mu_1-\mu_2,(\dfrac{1}{m}+\dfrac{1}{n})\sigma^2)$
      - 再计算配凑即可
  - 样本量 $m=n$ 时，$\cfrac{\sqrt{n}\big[ (\overline{X} - \overline{Y}) - (\mu_1-\mu_2) \big]}{S} \sim t(n-1)$
    - 其中 $S^2 = S_X^2 + S_Y^2 + \cfrac{2n}{n-1}S_{XY}^2$
    - **证明**：由样本量相等，可设 $Z_i = X_i-Y_i$
      - 由正态分布可加性，$Z_i\sim N(\mu_1-\mu_2,\sigma^2_1+\sigma^2_2)$
      - 计算易得 $S_Z^2 = S^2$，故由单样本结论即可得多样本结论

### 顺序统计量

- **顺序统计量**：取一组样本 $X_1,...,X_n$，在某个统计量下的相应观测值为 $x_1,...,x_n$
  - 将观测值从小到大排列为 $x_{(1)},...,x_{(n)}$，则对应的 $X_{(1)},...,X_{(n)}$ 就是顺序统计量
  - **独立同分布**：样本和总体的分布相同，且相互独立

### 最值的分布

- **最大值的分布**：$F(x) = P(\mathop{\bigcap}\limits^n_{i=1} \{X_{(i)} \leq x\})$
  - 由独立同分布性，**分布函数** $= \prod\limits^n_{i=1} P(X\leq x) = \big[ F(x) \big]^n$
  - **密度函数**：$f(x) = nf(x)\big[ F(x) \big]^{n-1}$
- **最小值的分布**：$F(x) = 1-P(X_{(1)}>x)$
  - 由独立同分布性，**分布函数** $ = 1-\prod\limits^n_{i=1}P(X_i>x) = 1-\big[ 1-F(x) \big]^n$
  - **密度函数**：$f(x) = nf(x)\big[ 1-F(x) \big]^{n-1}$
- **最大最小值向量 $(X_{(1)},X_{(n)})$ 的联合分布**：
  - $F_{1,n}(x_1,x_n) = P\vek{X_{(1)}\leq x_1 \\ X_{(n)} \leq x_n}$
  - 添项作差得到 $P(X_{(n)}\leq x_n) - P\vek{X_{(1)}>x_1 \\ X_{(n)}\leq x_n}$
    - 前项可直接求得
    - 后项化为交集，然后再利用独立性乘法公式，得到 
      - $P(\mathop{\bigcap}\limits^n_{j=1}(x_1 < X_j\leq x_n)) = \prod\limits^n_{j=1} P(X_1<X_j \leq X_n)$
  - **分布函数** $F(x_1,x_n) = [F(x_n)]^n - [F(x_n)-F(x_1)]^n$
  <br>
  - **密度函数** $f(x_1,x_n) = \begin{cases} n(n-1)f(x_1)f(x_n)\big[ F(x_n)-F(x_1) \big]^{n-2}，x_1\leq x_n \\ \whhh 0 \whhh\whh\pad ，x_1>x_n \end{cases}$
- **极差的分布**：$R_n = X_{(n)}-X_{(1)}$
  - **分布函数**：
  - **密度函数**：
    - $f(x) = \begin{cases} \dis n(n-1)\int_R f(t+x)f(t)\big[ F(t+x)-F(t) \big]^{n-2}dt，x\geq 0 \\ \whhh\whh 0 \whhh\whh ，x<0 \end{cases}$

### 顺序统计量的分布

- **n维随机向量微分法**：$g(\vec{y}) = \sum\limits^m_{i=1}f(\vec{x_i})|J_i|d\vec{y}$（分段函数形式）
- **顺序统计量的总联合密度（独立同分布）**：$(X_{(1)},...,X_{(n)})$
  - $P(X_{(1)} = x_1,...,X_{(n)} = x_n) = P(\mathop{\bigcup}\limits_{i\in I} \{X_{i_1}=x_1,...,X_{i_n}=x_n\}) = n!P(X_1 = x_1,...,X_n = x_n)$
    - <font color='chartreuse'>大小排序仅依赖于 $x_i$ 的值，和随机变量的序号无关，所以直接取 $X$ 的全排列即可</font>
  - 独立乘法公式，得联合密度为 $n!f(x_1)...f(x_n)dx_1...dx_n$
- **某个统计量的概率密度**：$X_{(k)}$

  <br>
  - 再由独立乘法公式，即得 $g_k(x) = nC^{k-1}_{n-1} [F(x)]^{k-1} [1-F(x)]^{n-k}f(x)dx$
  <br>
- **任意二个统计量的联合密度**：$(X_{(i)},X_{(j)})(i<j)$
  <br>
  - $P(X_{(i)} = x, X_{(j)} = y) = \\$ <br> $ \vek{C^{i-1}_nC^{j-i-1}_{n-i+1}C^{n-j}_{n-j+2}C^1_2} · \dkh{[F(x)]^{i-1}·[F(y)-F(x)]^{j-i-1}·[1-F(y)]^{n-j}}\cdot f(x)f(y)\\$ 

#### 习题

- 均匀分布的联合密度：$g(x_1,...,x_n) = \begin{cases} \cfrac{n!}{a^n}，0<x_1<...<x_n<a \\ 0，其他情况 \end{cases}$
- 求使得 $Q_1 = \sum\limits^n_{i=1} (x_i-a)^2$ 和 $Q_2 = \sum\limits^n_{i=1} |x_i-b|$ 均最小的 $(a,b)$
  - **解**：
    - $\sum\limits^n_{i=1} (x_i-a)^2 = \sum\limits^n_{i=1} (x_i-\ol x)^2 + \sum\limits^n_{i=1} (\ol x-a)^2$
    - 从而 $Q_1$ 在 $\ol x = a$ 时最小
    - $Q_2(b)= \sum\limits^n_{i=1} |x_{(i)}-b|$
      - 若 $b\leq x_{(1)}$，则 $Q_2(b) = \sum\limits^n_{i=1} x_i-nb$，单增
      - 若 $b\leq x_{(1)}$，则 $Q_2(b) = nb - \sum\limits^n_{i=1} x_i$，单减
      - 若 $b\in(x_{(1)},x_{(n)})$，则可设 $b\in (x_{(k)},x_{(k+1)})$
        - $Q(b) = \sum\limits^k_{i=1} \big[ b-x_{(i)} \big] + \sum\limits^n_{i=k+1} \big[ x_{(i)}-b \big]$
        - $Q'(b) = 2k-n$
        - 从而 $b = M_c$ 中位数时 $Q(b)$ 最小


## 经验分布函数

- 总体分布函数未知时，抽取样本的观测值为 $x_1,...,x_n$，按升序排列为 $x_{(1)},...,x_{(n)}$
  - 设互不相同的共 $k$ 个，重数为 $n_1,...,n_k$
- **经验分布函数**：$F_n(x) = \begin{cases} 0，\whh x < x^*_{(1)} \\ \dfrac{n_1+...+n_l}{n}，\quad x^*_{(l)} \leqslant x\leqslant x^*_{(l+1)} \\ 1，\whh x\geqslant x^*_{(k)} \end{cases}$
- **格列汶科定理**：
  - 设 $D_n = \sup\limits_{x\in R} |F_n(x)-F(x)|$
  - 则有 $P(\lim\limits_{n\to\infty} D_n = 0) = 1$
  - <font color='chartreuse'>经验分布函数依分布收敛于原分布函数</font>
  - **证明**：设 $Y_i = I(X_i\leq x)$（示性函数）
    - 由示性函数的期望性质，$EY_i = F(x)$
    - 再由示性函数的二值性，$EY_i^2 = EY_i = F(x)$
    - 故 $DY_i = F(x)\big[ 1-F(x) \big]$
    - 再由 $F_n(x) = \dfrac{1}{n}\sum\limits^n_{i=1} Y_i = \ol Y$，即得结论
  - **推论**：
    - $E\big[ F_n(x) \big] = E\ol Y =  F(x)$
    - $D\big[ F_n(x) \big] = D\ol Y =  F(x)\big[ 1-F(x) \big]$
