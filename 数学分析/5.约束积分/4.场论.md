# 场论

- **数量场（向量场）**：在区域 $\Omega$ 内的时刻t，每个点都有确定的数值 $f(x,y,z,t)$ 与它对应，则称 $f$ 为 $\Omega$ 上的数量场（或向量场）
- **稳定场**：不随时间的变化而变化
- **不稳定场**：时间是场强函数的自变量

## 梯度

- **等值面**：场在其上不变的曲面
- **梯度场**：数量场产生的向量场 $grad f = f_xi + f_yj + f_zk$

## 通量和散度

- **源**：产生流体的源头
- **汇**：排出流体的漏洞
- **通量**：
  - $已知向量场 \overrightarrow{a} = P(x,y,z)i + Q(x,y,z)j + R(x,y,z)k$
  - 则曲面积分 $\varPhi = \iint\limits_{\Sigma}\overrightarrow{a}dS$ 为 向量场沿指定侧通过曲面的通量
- **M点的散度**：$div\ a(M) = \frac{\partial P}{\partial x}(M) + \frac{\partial Q}{\partial y}(M) + \frac{\partial R}{\partial z}(M)$
- **几何意义**：
  - 散度是标量，三个函数合在一起
  - **散度的Gauss公式（闭合曲面）**：$\iiint\limits_\Omega div\ adV = \iint\limits_{\partial\Omega} a·dS$
    - **理解**：
      - $\iint Pdydz \to \iint \vec{P}dS$（P垂直于dS）
      - P、Q、R相加得到a
    - 场线通过单位体积边界的通量（曲面积分）
    - 某个曲面点上 表示 场向源头外发散程度的量
    - 向量场沿垂直于通过平面的方向的导数，即为散度div。因此散度积分需要在曲面积分的基础上再加上一个垂直方向的积分，即构成体积积分
  - $div\ a(M) = \lim\limits_{\Omega\to M}\cfrac{\iint\limits_{\partial\Omega}a·dS}{V_\Omega}$
    - 通量关于体积的变化率
- 散度与源头
  - **正源点**：$div\ a(M) >0$
  - **无源场**：$\forall M，div\ a(M) = 0$ （所有场线均回到源头，不发散）
    - 磁场线均闭合，磁场是无源场
    - **证明**：磁场的高斯定理
    - 原点点电荷的电场，$div\ E = 0(r\neq 0)$（穿入必定穿出，不可能漏走）
- **散度场**：数量场 $div\ a$

## 向量线

- **流线**：运动轨迹（切线是速度）
- **向量线**：每点的切线方向和场线切线方向相同
  - **向量线方程**：$\frac{dx}{P(x,y,z)} = \frac{dy}{Q(x,y,z)} = \frac{dz}{R(x,y,z)} = \frac{t_切}{t_M}$
    - 分子是向量线的切向量坐标，分母是M点的场向量坐标
- **电磁学Gauss定律**
  - **证明**：$div\ E = 0$

## 环量和旋度

- M点若流体旋转，则 $v = v_0 + \omega × r$
- 速度场可以直接求出旋转速度：$2\omega = (\frac{\partial v_z}{\partial y} - \frac{\partial v_y}{\partial z})i + (\frac{\partial v_x}{\partial z} - \frac{\partial v_z}{\partial x})j + (\frac{\partial v_y}{\partial x} - \frac{\partial v_x}{\partial y})k = B$
- **旋度**：$\rot a(M) 或 \text{curl}\ a(M) = \begin{vmatrix} i&j&k \\ \frac{\partial}{\partial x}&\frac{\partial}{\partial y}&\frac{\partial}{\partial z} \\ P&Q&R \end{vmatrix}_M = B(M)$
- **几何意义**：
  - 旋度是矢量
  - 场线绕某点的旋转量（环路积分）
  - **旋度的Stokes公式**： $\iint\limits_{\Sigma} B·dS = \int\limits_{\Gamma}v·ds$
    - 右式称为沿定向曲线的**环量**
  - **环量面密度**：$\|\rot a(M)\| = \lim\limits_{\Sigma\to M}\cfrac{\int\limits_{\partial \Sigma}a·ds}{S_\Sigma}$
    - 环量关于面积的变化率
    - 单位面积边界的环量
  - 旋度方向环量面密度最大，为 $\|\rot a(M)\|$
  - **证明**：
    - 单位面积内的环量 $\sigma$，由积分中值定理 + 连续函数，得到 $\sigma = \|\rot a(M)\|\cdot\cos(\rot a(M),\vec{n})$
- **旋度场**：向量线a产生的向量场 $\rot a$
- **无旋场**：$\rot a = 0$（电场线不闭合，电场是无旋场）
  - 闭合路径（周线）积分为0

## Hamilton算子

- **Hamilton算子**：$\nabla = i\frac{\partial}{\partial x} + j\frac{\partial}{\partial y} + k\frac{\partial}{\partial z}$（向量算子）
  - $\nabla f = grad\ f$
  - $\nabla · a = div\ a$
  - $\nabla × a = rot\ a$
- **Laplace算子**：$\Delta = \nabla·\nabla = (\frac{\partial}{\partial x})^2 + (\frac{\partial}{\partial y})^2 + (\frac{\partial}{\partial z})^2$（标量算子）
  - **调和函数**：满足 $\Delta u = 0$
- **算子点乘公式**：$\nabla ·(g\nabla f) = \nabla g·\nabla f + g\Delta f$
  - **证明**：偏导数乘法公式
- **Green第一公式**：$\iiint\limits_{\Omega} (\nabla f·\nabla g + f\Delta g)dV = \iint\limits_{\partial \Omega} f\frac{\partial g}{\partial n}dS$
  - 其中g是向量值函数，n是曲面单位法向量（方向余弦） $(cos\alpha,cos\beta,cos\gamma)$，方向导数为 $\frac{\partial g}{\partial n} = grad\ g\cdot n$
  - **证明**：点乘结合后，使用Gauss公式化为曲面，然后利用H算子的梯度性得到导数向量形式
- **Green第二公式**：$\iiint\limits_{\Omega} (f\Delta g - g\Delta f)dV = \iint\limits_{\partial \Omega}(f\frac{\partial g}{\partial n} - g\frac{\partial f}{\partial n})dS$

## 旋度、散度运算律

- 线性：点乘和叉乘的线性
- 点乘（散度）、叉乘（旋度）分配律：导数乘法公式
- 散度（梯度）的旋度、旋度的散度均为0

## 保守场和有势场

- **有势场**：$\exist grad\ U = a$（梯度场）
- **势函数**：$V = -U$
- **保守场**：a中曲线积分和路径无关
- 一维单连通区域：闭曲线可以不经过区域外连续收缩成一点（以曲线为边界作一个完全含于区域内的曲面）
  - 空心球是一维单连通，但不是二维单连通
- 二维单连通区域：闭曲面可以……（以曲面为边界……）
  - 圆环是二维单连通，但不是一维单连通
