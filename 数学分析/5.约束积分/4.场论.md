# 场论

- **符号约定**：
  - 若不加说明，则所处空间为 $\R^3$
  - $i,j,k$ 表示三个线性无关的向量
  - 规定流出为正方向，流入为负方向
- **向量场**：若在任意时刻 $t$，区域 $\Omega$ 内每个点都有确定的 $f(x,y,z,t)$ 与它对应，则称 $f$ 为 $\Omega$ 上的向量场或场强函数
  - <font color='chartreuse'>$f$ 可以是标量（散度场、磁场），也可以是向量（梯度场、速度场）</font>
- **稳定场**：不随时间的变化而变化的向量场（时间 $t$ 不是场强函数 $f$ 的自变量）
- **不稳定场**：时间 $t$ 是场强函数 $f$ 的自变量

## 梯度

- **等值面**：场强函数在其上为常函数的曲面
- **梯度场**：场强函数为 $\grad f = f_xi + f_yj + f_zk$ 的向量场
- **梯度定理**：$\forall M\in \O$，$\grad f(M)$ 的方向和（$M$ 所处等值面的法向量）方向相同
  - **证明**：
    - 由定义，等值面上的法向量为 $\or n = f_xi + f_yj + f_zk$
    - 由梯度定义，$\grad f = \cpfrac{f}{\vec n}\or n$

## 通量和散度

- **源点（正源）**：产生流体的源头（只有出向量，没有入向量的点）
- **汇点（负源）**：排出流体的漏洞（只有入向量，没有出向量的点）
- **通量**：
  - 若已知向量场 $\overrightarrow{a} = P(x,y,z)i + Q(x,y,z)j + R(x,y,z)k$，且分量函数可连续偏导
  - 则曲面积分 $\varPhi = \dis\iint\limits_{\Sigma}\or adS$ 为 向量场沿指定侧通过曲面的通量
  - <font color='chartreuse'>场强在曲面上的第二类积分</font>
- **散度**：$\dis\divt \or a(M) = \frac{\partial P}{\partial x}(M) + \frac{\partial Q}{\partial y}(M) + \frac{\partial R}{\partial z}(M)$ 称为 $M$ 点的散度
  - **代数意义**：散度是标量，是三个分量函数的对应偏导数合在一起的值
  - **几何意义**：点 $M$ 处场线流出的速度
    - <font color='chartreuse'>在有向场中，偏导数表示该点场强增加（流体流出）的速度，将三个方向上流出的速度相加，就是整体流出的速度（速度是标量，能直接相加）
    - "散"的意思不是几何上的发散（场线指向不同的方向），而是物理上的散出（流体流出该区域）</font>
- **散度的Gauss公式（闭合曲面）**：$\dis\iiint\limits_\Omega \divt \or adV = \iint\limits_{\partial\Omega} \or a\cdot dS$
  - **证明**：
    - 直接计算即可
  - **本质**：
  - 场线通过单位体积边界的通量（曲面积分）
  - 某个曲面点上 表示 场向源头外发散程度的量
  - 向量场沿垂直于通过平面的方向的导数，即为散度div。因此散度积分需要在曲面积分的基础上再加上一个垂直方向的积分，即构成体积积分
- **散度是通量的点密度**：$\divt a(M) = \lim\limits_{\Omega\to M}\cfrac{\dis\iint\limits_{\partial\Omega}a·dS}{V_\Omega}$
  - **本质**：
    - 散度是通量关于体积的变化率
    - 散度是极小体积的通量
    - 散度方向的通量密度最大，为 $\|\divt \or a(M)\|$
  - **证明**：
    - 计算即可（应用勒贝格微分定理）
- **无源场**：若 $\forall M\in\R^3，\divt \or a(M) = 0$，则称该向量场为无源场
  - <font color='chartreuse'>所有场线均闭合（回到源头），不发散，故散度总为 $0$</font>
  - **实例**：磁场线均闭合，故磁场是无源场
  - **证明**：
    - 由磁场的高斯定理易得结论
    - 对于点电荷的电场，$\divt E = 0(r\neq 0)$（穿入必定穿出，不可能漏走）
- **散度场**：场强函数为散度 $\divt \or a$ 的场

## 向量线

- **流线**：运动轨迹（切线是速度）
- **向量线**：每点的切线方向和场线切线方向相同
  - **向量线方程**：$\dis\frac{dx}{P(x,y,z)} = \frac{dy}{Q(x,y,z)} = \frac{dz}{R(x,y,z)} = \frac{t_切}{t_M}$
    - 分子是向量线的切向量坐标，分母是M点的场向量坐标
- **电磁学Gauss定律**
  - **证明**：$div\ E = 0$

## 环量和旋度

- **环量**：
  - 若已知向量场 $\overrightarrow{a} = P(x,y,z)i + Q(x,y,z)j + R(x,y,z)k$，且分量函数可连续偏导
  - 设 $\G$ 是定向曲线，则曲线积分 $\Psi = \dis\iint\limits_{\G}\or ads$ 为 向量场沿有向曲线的环量
- **旋度**：$\rot \or a(M) = \begin{vmatrix} i&j&k \\ \dfrac{\partial}{\partial x}&\dfrac{\partial}{\partial y}&\dfrac{\partial}{\partial z} \\ P&Q&R \end{vmatrix}_M$
  - <font color='chartreuse'>也可写为 $B(M)$ 或 $\text{curl}\ (M)$</font>
  - **几何意义**：
    - 旋度是矢量
    - 场线绕某点的旋转量（环路积分）
- **旋度的Stokes公式**：$\dis\iint\limits_{\Sigma} B·dS = \int\limits_{\pa\Sigma}v·ds$
  - **证明**：
    - 同Gauss公式，计算即可
- **环量的面密度**：$\|\rot a(M)\| = \lim\limits_{\Sigma\to M}\cfrac{\dis\int\limits_{\partial \Sigma}a·ds}{S_\Sigma}$
  - **本质**：
    - 旋度是环量关于边界线的变化率
    - 旋度是极小曲线边界的环量
    - 旋度方向的环量线密度最大，为 $\|\rot a(M)\|$
  - **证明**：
    - 单位面积内的环量 $\sigma$，由积分中值定理 + 连续函数，得到 $\sigma = \|\rot a(M)\|\cdot\cos(\rot a(M),\vec{n})$
- **旋度场**：场强函数为 $\rot a$ 的向量场
- **无旋场**：$\rot a = 0$ 的场
  - <font color='chartreuse'>所有场线均发散（不回到源头），不旋转，故旋度总为 $0$</font>
  - **实例**：电场线不闭合，电场是无旋场
  - 任意周线上的场强函数积分为 $0$

## Hamilton算子

- **Hamilton算子**：$\dis\nabla = i\pfrac{}{x} + j\pfrac{}{y} + k\pfrac{}{z}$（向量算子）
  - $\nabla f = \grad f$
  - $\nabla \cdot a = \divt a$
  - $\nabla \times a = \rot a$
- **Laplace算子**：$\dis\Delta = \nabla·\nabla = (\frac{\partial}{\partial x})^2 + (\frac{\partial}{\partial y})^2 + (\frac{\partial}{\partial z})^2$（标量算子）
  - **调和函数**：满足 $\Delta f = 0$ 的函数 $f$
- **算子点积公式**：$\nabla \cdot(g\nabla f) = \nabla g\cdot\nabla f + g\Delta f$
  - **证明**：
    - 由偏导数乘法公式，计算易得
- **Green第一公式**：$\dis\iiint\limits_{\Omega} (\nabla f·\nabla g + f\Delta g)dV = \iint\limits_{\partial \Omega} f\frac{\partial g}{\partial \or n}dS$
  - <font color='chartreuse'>$g$ 是向量值函数，$\or n$ 是曲面的单位法向量 $(\cos\alpha,\cos\beta,\cos\gamma)$
  - 方向导数为 $\cpfrac{ g}{n} = \grad g\cdot n$</font>
  - **证明**：
    - 点乘结合后，使用Gauss公式化为曲面，然后利用H算子的梯度性得到导数向量形式
- **Green第二公式**：$\dis\iiint\limits_{\Omega} (f\Delta g - g\Delta f)dV = \iint\limits_{\partial \Omega}(f\frac{\partial g}{\partial n} - g\frac{\partial f}{\partial n})dS$
  - **证明**：
    - 详见偏微分方程

## 保守场和有势场

- **有势场**：若存在函数 $U$ 满足 $\grad U = \or a$，则 $\or a$ 称为有势场
  - <font color='chartreuse'>可以化为梯度场的场</font>
  - **势函数**：$V = -U$
- **保守场**：若场强的曲线积分和路径无关，则称为保守场

- **场等价定理**：若 $P,Q,R$ 在单连通区域 $\O$ 上连续可偏导，则保守场 $\LR$ 有势场 $\LR$ 无旋场（电场）
  - **证明**：仿照曲线积分与路径无关的定理证明即可
- **原函数定理**：
  - 设 $P,Q,R$ 在单连通区域 $\O$ 内连续，$dU = Pdx + Qdy + Rdz$
  - 则 $\forall A,B\in\O$ 都有 $\dis\int_{\arc {AB}} Pdx + Qdy + Rdz = U|_{B}-U|_A$
