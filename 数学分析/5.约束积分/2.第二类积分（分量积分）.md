# 第二类积分（分量积分）

- **区别**：第一类积分是标量函数的曲线积分，第二类积分是向量值函数的曲线积分
  - 只有第二类积分才有投影法，第一类积分只有换元法（因为只有第二类积分才有1-形式、2-形式）
- **应用例**：
  - 场论
  - 复变函数积分
    - <font color='chartreuse'>因为 $1$ 和 $i$ 是域 $\C$ 上仅有的两个线性无关的元，故任意复函数可写为 $f(z) = u(x,y) + v(x,y)i$，即向量值函数。故其曲线积分只能是第二类</font>
- <font color='chartreuse'>碎碎念之教科书不要事无巨细
  - 尤其是古典微分几何这种学科，几何意义非常完备，相应的一个公式的任意变形都可以做出几何方面的解释。而我们知道公式变形是无穷无尽的，难道你也要写上无穷无尽的解释吗？显然不是，这东西当成留白就好了，只写最标准最常用的东西（即只对1-形式和2-形式这种做出解释就好）</font>

## 第二类曲线积分

### 向量值函数的曲线积分

- **符号约定**
  - 设积分曲线 $L$ 是定向的光滑曲线，两端点为 $A,B$
  - 被积向量值函数的统一写法：
    - **向量和形式**：$\vec f(x,y,z) = P(x,y,z)\vec{i} + Q(x,y,z)\vec{j} + R(x,y,z)\vec{k}$
    - **方程组形式**：$\vec f(x,y,z) = \begin{cases}f_1 = P(x,y,z) \\ f_2 = Q(x,y,z) \\ f_3 = R(x,y,z) \end{cases}$
- **单位切向量（方向余弦）**：$\tau = (\cos\alpha,\cos\beta,\cos\gamma)$
  - <font color='chartreuse'>曲线 $L$ 的单位切向量
  - 曲线方程 $\vec r(x,y,z)$ 的（三个分量函数）的导数
  - （不是梯度，向量值函数没有梯度）</font>
- **微分向量** $ds$ 满足 $\begin{cases}dx = \cos\alpha ds \\ dy = \cos\beta ds \\ dz = \cos\gamma ds \end{cases}$
- **第二类曲线积分（加权形式）**：$$\int\limits_L \vec f(x,y,z)\cdot\tau ds = \int\limits_L \Big[P(x,y,z)\cos\alpha + Q(x,y,z)\cos\beta + R(x,y,z)\cos\gamma \Big]ds$$
  - <font color='chartreuse'>（内积）加权性：本质是把（三个分量函数）按照（当前点的方向余弦）进行加权，再把（三个加权函数）的第一类曲线积分加起来
  - 参数函数形式：
    - 将三个分量函数写成速度指数 $t$ 的参数形式，结合弧长微分公式，即可化为关于 $t$ 的一元积分
  - 应用：转化为三个第一类曲线积分</font>
- **常用形式（1-形式）**：$$\int\limits_L \vec f(x,y,z)\cdot\tau ds = \int\limits_L P(x,y,z) dx + Q(x,y,z) dy + R(x,y,z)dz$$
  - <font color='chartreuse'>投影性：本质是将 $L$ 上各点的函数值投影到 $L_x,L_y,L_z$ 上，再分别在三个直线上求一元定积分，再加起来
    - 设 $L_x$ 是 $L:y=f(x)$ 在 $x$ 轴上的投影，$L_x$ 两端点为 $b>a$
    - 则 $\dis\int_{L} P(x,y)dx = \int^b_a P(x,f(x))dx$
  - 隐函数形式：
    - 曲线 $L$ 实际上也规定了约束关系，即隐函数 $y(x),z(x)$
    - 将隐函数代入1-形式，即可转化为关于 $x$ 的一元积分
  - 应用：转化为三个一元积分</font>
- **参数形式**：$$\int\limits_L f(x,y,z)·\tau ds = \int^a_b \Big[P(t)x'(t) + Q(t)y'(t) + R(t)z'(t) \Big]dt$$
  - **证明**：
    - $ds = \sqrt{x'^2(t) + y'^2(t) + z'^2(t)}dt$
    - $\tau = (\cos\alpha, \cos\beta, \cos\gamma) = \cfrac{1}{\sqrt{x'^2(t) + y'^2(t) + z'^2(t)}}(x'(t),y'(t),z'(t))$
  - **本质**：
    - 导数充当方向余弦
    - 速度指数 $t$ 形式与弧长指数 $s$ 形式的转换公式（换元即可）<font color='chartreuse'>（速度指数一般是自定义的参数变量，目的是为了让形式更简单）
    - 一般来说，如何换元是由 $L$ 的形式决定的</font>

### 性质

- **方向性**：
- **线性**
- **路径可加性**
- <font color='chartreuse'>微分向量 $\or{dr}$ 和向量微分 $d\vec r$ 的区别：
  - 微分向量是 $\vec\tau ds = (dx,dy,dz)$，是（标量微分）乘一个（方向向量）
  - 向量微分是对向量（向量值函数）做微分运算，结果是一个垂直于原向量（向量值函数）的微分向量</font>
- <font color='red'>只需要考虑一个方向余弦
  - 第二类曲线积分 $\dis\int_L \vec f\cdot\vec\tau ds$ 似乎应该有两个方向余弦，即 $\vec f$ 一个，$\vec\tau$ 一个。但实际上我们只需要考虑 $\tau$ 的方向余弦即可，因为 $\vec f$ 的分量已经给出来了（$P,Q,R$）。分开计算 $P,Q,R$ 的过程，实际上就是应用了 $\vec f$ 的方向余弦的过程。
  - 另外提一嘴，向量值函数的方向余弦也是向量值函数，算起来很麻烦的
- 向量值函数不是向量
  - 就像虽然标量的乘法和向量的数乘写法都一样，但其实它们并不是一个东西。虽然它们具有相同的性质（交换律），但是定义域和值域不同。为此，泛函中发明了算子的概念，即将定义域不同但性质相同的运算符都归类为一种算子（算子的本质还是映射，比如标量乘法 $:\R\times\R \to \R$，向量数乘 $:\R^n\times\R\to \R^n$，都可写成映射的形式。我们将它们统称为乘法算子）
  - 后面会学到微分向量空间的微分运算（详见[外微分](./5.外微分.md)章节）。其实微分就是一种算子。但因为它可定义在不同的微分向量空间中，而且随着定义域不同，值域也不同。所以我们无法像对一般的映射一样，用一个式子就能写出它的所有定义域和值域的映射关系，故把它称为算子，将其与映射区分开来
  - 但因为算子本质还是多个映射的综合，所以我们可以像处理函数一样处理算子
    - 比如前面的多元泰勒公式、多元莱布尼茨求导公式，都出现了 $(\dpfrac{}{x}+\dpfrac{}{y})^nf$ 这种式子。实际上前面两个就是对 $x$ 求偏导算子和对 $y$ 求偏导算子。我们把它当成函数一样进行运算处理（对算子定义加减乘除，并且去括号和移项等操作也完全一样）。这其实是挺自然的一种想法，我们在小学先学了实数的四则运算，然后初中又学了多项式和函数的四则运算。这个都能过渡过来，没理由我们对运算符也定义四则运算时就接受不了吧。反正运算性质都是一样的，写成一样的形式也无不可
      - 当然算子的运算其实也是一种算子，这就是泛函会把人绕晕的一个地方，hh
    - 实际上这也是迈向高阶数学课所必须的一个操作。后面偏微分方程中甚至连分量函数都不要了，而且默认所有的函数都是向量值函数，标量函数就是只有一个分量不为0的向量值函数。也就是所谓的“趋于抽象和大一统”。其实物理也是一样，场论里麦克斯韦方程组，用Hamilton算子和叉乘写成的形式可能很简洁，但一旦展开就是灾难（而实际上叉乘形式反而更加符合现实中的物理意义一些，你展开后鬼才能看出哪个东西代表哪个意义）
  - 回到刚才来，向量值函数虽然不是向量，但因为我们能像对向量一样处理它，所以也把它写成了向量的形式。但是它们之间仍然是有区别的，比如刚才举的分量函数并不是向量函数乘方向余弦的例子</font>

### 习题

#### 求第二类曲线积分

- **直接法（加权法）（隐函数法）（投影法）（分解法）**：一般是 $L$ 方程较为简单时才比较方便
- **参数法（换元法）**：通用方法，但一般比较麻烦
- **实例**：$\int\limits_L ydx + xdy$，其中 $L$ 是 $O(0,0)，A(1,0)，B(1,2)$ 围成的直角三角形
  - **解（直接法）**：易得 $OA$ 段退化为 $\int^1_0 0dx + x(d0) = 0$，$OB$ 段同理
  - **解（隐函数法）**：$OB$ 段方程为 $y=2x$，故 $\int^1_0 2xdx + \int^1_0 xd(2x) = 2$

#### 性质题

- **相反对称性**：$I = \dis\int_L f(x,y)dx + g(x,y)dy$，其中 $L$ 是上半圆弧，逆时针为正方向
  - **解**：
    - 若 $f$ 是奇函数，则左右对称区域积分相反，从而 $dx$ 积分为0
    - 若 $g$ 是偶函数，则逆时针向上（正）积分设为 $I_1$，再逆时针向下（负）积分值也是 $I_1$，两者符号相反，从而 $dy$ 积分为0
- **轮换对称性**：无用
- **放缩**：
  - 设 $L:x^2+y^2=r^2$，以逆时针为正方向，$I(r) = \dis\int_{L(r)} \frac{xdy-ydx}{(x^2+y^2+xy)^2}$
  - 则 $\lim\limits_{r\to +\infty} I(r)= 0$
  - **证明**：
    - 将1-形式转化为内积形式，辅助角公式得 $|分子|\leq r$
    - 放缩易得分母 $\geq r^4$
- **简单但有用**：设 $A(x_1,y_1)，B(x_2,y_2)$ 是平面上两点，$f(x,y)$ 是二元函数，则 $$ k\int^{x_2}_{x_1} f\Big( x，k(x-x_1)+y_1 \Big)dx = \int^{y_2}_{y_1} f\Big( \frac{1}{k}(y-y_1)+x_1 ， y \Big)dy $$

## 第二类曲面积分

### 曲面的侧

- **双侧曲面**：法向量从 $P$ 点出发，不绕过边界进行移动，最后回到 $P$ 点时方向不变的曲面
  - **实例**：常见曲面基本都是双侧的
  - **反例**：莫比乌斯环是单侧曲面
- **坐标转化公式**：设曲面 $\Sigma$ 有隐函数方程和参数函数两种形式，则 $F_x = J(x) = \cpfrac{(y,z)}{(u,v)}$
  - **证明**：详见前面的基础微分几何部分
- **曲面的法向量**：
  - **曲面法向量**：$\vec{n} = \pm r_u × r_v = \pm\Big( |J(x)|,|J(y)|,|J(z)| \Big) = \pm\grad F$
    - <font color='chartreuse'>某点法向量的正负取决于其与正方向的夹角
    - 法向量的正负性和内外性不是一个概念，相关但不相同
      - 正负性指的是该点微分（如 $dxdy$）的正负性，由曲面和坐标系两者决定
      - 内外性仅由曲面自身决定
      - 外法向量可以是正的，也可以是负的，这取决于曲面的形状和在坐标系中的位置，需要具体分析
      - 但是，若某点外法向量是正的，则内法向量一定是负的。这也是向量正负号和标量正负号能混用的原因</font>
  - **单位法向量**：$\overrightarrow{n} = (\cos\alpha,\cos\beta,\cos\gamma) = \dfrac{1}{\pm\sqrt{EG-F^2}}\Big( |J(x)|,|J(y)|,|J(z)| \Big)$
    - <font color='chartreuse'>在这里“单位”既是曲面的单位面积，也是指向量的单位长度</font>
- **方向余弦的几何意义**：$dx\land dy = \cos\gamma dS$（该向量平行于z轴）
  - <font color='chartreuse'>（方向余弦）就是以微分向量形式表示的有向面积（单位曲面）之间的夹角
  - 由于曲面只有两侧，故正方向也只有两种取法
      - 对于曲线1-形式，若切向量 $ds$ 与 $x$ 轴正方向成锐角，则 $dx$ 部分的积分为正号
      - 对于曲面2-形式，若法向量 $\vec n$ 与 $z$ 轴正方向成锐角，则 $dxdy$ 部分的积分为正号</font>
  - **实例**：
    - 标准正方体：若外侧为正方向，则
      - （曲面上侧的法向量）平行于（z轴正方向），故 $dxdy$ 为正
      - （曲面下侧的法向量）平行于（z轴负方向），故 $dxdy$ 为负
    - 球体：若外侧为正方向，则
      - （下半表面的外侧法向量）与（z轴正方向）成钝角，故 $dxdy$ 为负
      - （上半表面的外侧法向量）与（z轴正方向）成锐角，故 $dxdy$ 为正
- **法向量的连续性**：若方向余弦连续，则法向量符号不变。即（积分点处的法向量）在运动时不会转到另一侧

### 第二类曲面积分

- **第二类曲面积分（加权形式）**：$$\iint\limits_\Sigma f\cdot\vec ndS = \iint\limits_\Sigma \Big[P\cos\alpha + Q\cos\beta + R\cos\gamma \Big] dS$$
- **常用形式（2-形式）**：$$\iint\limits_{\Sigma}f\cdot \vec ndS = \iint\limits_\Sigma \Big[Pdydz + Qdzdx + Rdxdy \Big]$$
  - <font color='chartreuse'>积分会消除自变量，故 $dydz$ 就是把 $yz$ 都消掉，自然只剩下自变量 $x$，等价于 $dy\land dz = dx$ 的效果
  - 后面在古典微分几何中，1-形式和2-形式是主要研究对象。如果到时候看不懂了可以到本章来回顾一下</font>
- **参数形式**：$$\iint\limits_{\Sigma}f\cdot\vec ndS = \iint\limits_{D}\Big[ P(u,v)J(x) + Q(u,v)J(y) + R(u,v)J(z) \Big]dudv$$

## 习题

### 求第二类曲线积分

- **直接法（加权法）（投影法）（分解法）**：求出方向余弦，转化为第一类曲面积分，最终转化为二重积分（曲面能画出来时用它比较简单）
  - <font color='chartreuse'>形状投影等价于微分投影：
    - 把 $dxdy$ 除以 $\cos\g$ 变为 $dS$，然后 $dS$ 乘 $\sqrt{EG-F^2}$ 再变为 $dxdy$（顺便把曲面 $\Sigma$ 变为平面投影 $\Sigma_{xy}$） 的过程称为微分投影
    - 把曲面 $\Sigma$ 直接乘 $\cos\g$ 变为 $\Sigma_{xy}$ 的过程称为形状投影
    - 这两个过程是等价的，即可以用形状投影法，也可以用微分投影法</font>
- **参数法（换元法）**：通法，但一般不用

- **退化性**：若曲面积分 $\dis\int_{\Sigma} fdxdy$ 中 $f$ 只与 $x,y$ 有关，则其退化为二重积分 $\dis\int_{\Sigma_{xy}} fdxdy$
  - **证明**：两个积分唯一的不同点就在于曲面面积和平面面积不等。但若使用形状投影法，则发现曲面面积乘上方向余弦 $\cos\g$ 实际上就是平面面积
