# 第二类积分（分量积分）

- **理解**：弧长被拆为了多个方向，所以每个方向上并不需要弧长，只需要导数作为方向余弦就行了
  - $dS$ 和 $(cos,cos,cos)$ 的相消，其实就说明了这点
  - $t$ 可以看作三维空间中的时间维度（显然它和xyz线性相关），$x'(t)dt$ 就是 $t$ 在 $x$ 方向上的分量

## 第二类曲线积分

### 向量值函数的第一类曲线积分

- **符号约定**
  - 设积分曲线 $L$ 是定向的光滑曲线，两端点为 $A,B$
  - 被积向量值函数的统一写法：
    - **向量和形式**：$f(x,y,z) = P(x,y,z)\vec{i} + Q(x,y,z)\vec{j} + R(x,y,z)\vec{k}$
    - **方程组形式**：$f(x,y,z) = \begin{cases}f_1 = P(x,y,z) \\ f_2 = Q(x,y,z) \\ f_3 = R(x,y,z) \end{cases}$
- **单位切向量（方向余弦）**：$\tau = (\cos\alpha,\cos\beta,\cos\gamma)$
  - 本质是 $r(x,y,z)$ 的偏微分
- **弧长微分** $ds$ 满足 $\begin{cases}dx = \cos\alpha ds \\ dy = \cos\beta ds \\ dz = \cos\gamma ds \end{cases}$
- **第二类曲线积分**：
  - $\int\limits_L f(x,y,z)·\tau ds = \int\limits_L [P(x,y,z)cos\alpha + Q(x,y,z)cos\beta + R(x,y,z)cos\gamma ]ds$
  - **1-形式的积分**
  - **实例**：$\int\limits_L ydx + xdy$
    - **求解**：利用线性分离各项，转化为第一类曲线积分

### 性质

- **方向性**：
- **线性**
- **路径可加性**
- **速度指数 $t$ 形式与弧长指数 $s$ 形式的转换公式（换元即可）**：
  - $ds = \sqrt{x'^2(t) + y'^2(t) + z'^2(t)}dt$
  - $\tau = (cos\alpha, cos\beta, cos\gamma) = \cfrac{1}{\sqrt{x'^2(t) + y'^2(t) + z'^2(t)}}(x'(t),y'(t),z'(t))$
  - 上述两式合并为
    - $\dis\int\limits_L f(x,y,z)·\tau ds = \int^a_b [P(t)·x'(t) + Q(t)·y'(t) + R(t)·z'(t)]dt$
  - **本质**：导数充当方向余弦
  - <font color='chartreuse'>速度指数一般是自己设的量</font>

## 第二类曲面积分

### 曲面的侧

- **双侧曲面**：法向量从 $P$ 点出发，不绕过边界进行移动，最后回到 $P$ 点时方向不变的曲面
- **曲面的法向量**：
  - **法向量的方向**：$\vec{n} = \pm r_u × r_v$
    - 取曲面的上侧 $\Rightarrow$ 方向余弦分量 $cos\gamma$ 符号为正（同理可类推曲面的右侧和前侧）
    - 方向余弦符号 $\Rightarrow$ 积分的符号
  - **法向量的连续性**：方向余弦连续，则符号不变，法向量不会转到另一侧
  - **单位法向量**：$\overrightarrow{n} = (cos\alpha,cos\beta,cos\gamma) = \frac{1}{\pm\sqrt{EG-F^2}}(|J(x)|,|J(y)|,|J(z)|)$
    - <font color='chartreuse'>（在这里“单位”指的是曲面的单位面积，而非向量长度）</font>

### 第二类曲面积分

- $\iint\limits_\Sigma f·ndS = \iint\limits_\Sigma [Pcos\alpha + Qcos\beta + Rcos\gamma]dS$
- **微分向量形式**：$dx\land dy = cos\gamma dS$（有向面积）
  - **2-形式**：$\iint\limits_{\Sigma}f·dS = \iint\limits_\Sigma [Pdydz + Qdzdx + Rdxdy]$
    - <font color='chartreuse'>积分会消除自变量，故 $dydz$ 就是把 $yz$ 都消掉，自然只剩下自变量 $x$，等价于 $dy\land dz = dx$ 的效果</font>
