## 曲线积分 $\leftrightarrow$ 二重积分

- **简单闭曲线（Jordan曲线）**：两端点值相等，且自身不相交的曲线
- **单连通区域**：D的简单闭曲线可以不经过D外的点，连续地收缩为单点
  - <font color='chartreuse'>实际上是拓扑同胚于单点集的区域。区域的 $k$ 连通性在复变函数中还要用，但直到代数拓扑中才能彻底讲明白</font>
- **多连通区域**：
- **诱导定向**：若点沿 $\pa D$ 移动时，$D$ 始终在左边，则称该方向为正方向。该定向方式称为诱导定向
  - <font color='chartreuse'>外部边界的正方向是逆时针，内部边界的正方向是顺时针</font>
- **标准区域 $D$**：可用两函数表示 $\pa D$ 的区域 $\LR$ 任意（平行于 $x$ 轴和 $y$ 轴的直线）最多与 $\pa D$ 交于两点的区域
  - <font color='chartreuse'>（可以是凹区域）</font>
  - **形式**：
    - $\{(x,y)\mid y_1(x)\leq y \leq y_2(x)，a\leq x\leq b\}$
    - $\{(x,y)\mid x_1(y)\leq x \leq x_2(y)，c\leq y\leq d\}$
- **Green公式（二重积分的N-L公式）**：
  - 设 $D$ 是（分段光滑的简单闭曲线）围成的单连通区域
  - 若函数 $P,Q$ 在 $D$ 上有连续偏导数
  - 则 $\dis\int\limits_{\partial D}Pdx+Qdy = \iint\limits_{D} (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})dxdy$（取诱导定向）
  - **本质**：
    - 实际上就是二维的N-L公式
    - 若取 $y=[0,1]$，则退化为一维的N-L公式。而由于区域的连通性，x轴、y轴积分最终和边界最终合为一点
    - 在xy轴（一维）上应用N-L公式，然后把一维相加（边界曲线积分）就是二维的N-L公式
  - <font color='red'>确定符号：以 $D$ 为圆形区域为例：
    - 此时诱导定向为逆时针
    - 设左上、右上、左下、右下边界分别为 $D_1,D_2,D_3,D_4$
    - 对 $Q_x$ 关于 $dx$ 应用N-L公式
      - 结果为 $\Big[ Q|_{D_4(y)}-Q|_{D_3(y)} \Big]$ 和 $\Big[ Q|_{D_2(y)}-Q|_{D_1(y)} \Big]$（都是关于 $y$ 的一元函数）
    - 对 $P_y$ 关于 $dy$ 应用N-L公式时
      - 结果为 $\Big[ P|_{D_2(x)}-P|_{D_4(x)} \Big]$ 和 $\Big[ P|_{D_1(x)}-P|_{D_3(x)} \Big]$（都是关于 $x$ 的一元函数）
    - 考虑对 $Q$ 关于 $dy$ 边界积分
      - 在 $D$ 的左半边界，顺着正方向时 $y$ 减小，故符号为负
      - 在 $D$ 的右半边界，顺着正方向时 $y$ 增大，故符号为正
      - 即 $\dis\int_{\pa D}Qdy = \int_{D_4}Qdy - \int_{D_3}Qdy + \int_{D_2}Qdy - \int_{D_1}Qdy$
    - 考虑对 $P$ 关于 $dx$ 边界积分
      - 在 $D$ 的上半边界，顺着正方向时 $x$ 减小，故符号为负
      - 在 $D$ 的下半边界，顺着正方向时 $x$ 增大，故符号为正
      - 即 $\dis\int_{\pa D}Pdx = -\int_{D_2}Pdx + \int_{D_4}Pdx - \int_{D_1}Pdx + \int_{D_3}Pdx$
    - 比较符号，发现 $Q$ 的N-L结果和边界积分结果符号相同，$P$ 则相反，故右式中 $Q_x$ 符号为正，$P_y$ 符号为负
  - 二连通区域情况</font>
  - **证明**：
    - **标准区域**：
      - 右式进行累次积分，分离dx、dy，对偏导数使用N-L公式得 $$\iint\limits_{D}\frac{\partial P}{\partial y}dxdy = \int^b_aP(x,y_1(x))dx - \int^a_b P(x,y_2(x))dx$$
      - 再由二重积分的投影法，右式等于 $\dis= -\int\limits_{\partial D}P(x,y)dx$。Q同理
    - **有限块标准区域**：积分区域可加性直得结论
    - **一般情形**：用多边形割线逼近区域边界
    - **有限复连通区域**：割为两个单连通区域即可 $$(\int\limits_L+\int\limits_l)Pdx+Qdx = \int\limits_{\partial D}Pdx+Qdy$$
- **内积形式**：$$\iint\limits_{D} (\frac{\partial F}{\partial x} + \frac{\partial G}{\partial y})dxdy = \int\limits_{\partial D} \Big[F\cos(\vec{n},x)+G\cos(\vec{n},y)\Big] ds$$
  - <font color='red'>为什么重积分的N-L公式会转化为第二类积分，而不是第一类呢？
  - 以二重积分为例，由累次积分公式，它是先积长条，再将长条积为整个区域。这个过程是个"反投影"过程，而不是"反拉直"过程</font>
- **方向判别法**：
  - 由偏微分方程的高斯格林定理，$\dis\int_D P_xdxdy = \int_{\pa D}P\nu^x ds$
    - 若设 $(dx,dy) = ds(\cos\a,\cos\b)$，则易得 $\nu = (-\cos\b,\cos\a)$，从而 $\nu^xds = -dy$

### 计算例


- **用格林公式求区域面积**：$S = \int\limits_{\partial D}xdy = -\int\limits_{\partial D}ydy = \frac{1}{2}\int\limits_{\partial D}xdx-ydy$
  - <font color='chartreuse'>一般都是需要应用边界方程的等式，从而实现换元化简的区域。比如星形线、双扭线等</font>
- **极限区域法**：设 $P,Q$ 连续可偏导，若对任意位置的上半圆弧都有 $\dis\int_L Pdx+Qdy = 0$，则 $P = Q_x \equiv 0$
  - **证明**：
    - 将 $L$ 补成上半圆周，使用格林公式，再分别使用积分中值定理，约分后即得 $$\frac{\pi r}{2}(Q_x-P_y)|_{P_0} + 2P(\xi,y_0) = 0$$
    - 取 $r\to 0+$ 即得 $P(x_0,y_0) = 0$。再由圆心的任意性得 $P\equiv 0$，再由上式即得 $Q_x = 0$

### 习题

#### 第一类转化为第二类

- <font color='chartreuse'>算是学习偏微分方程的必备预习内容？但它却并没有出现在数分教材中，而是考研书中。实际上我觉得应该把它归类为偏微分第一章的课后习题</font>
- **法切引理**：
  - 设 $r$ 是单位切向量，$n$ 是单位法向量
  - 若设 $r = (\cos\a,\cos\b)$，则 $n = \pm(-\cos \b,\cos\a)$
  - **证明**：取二阶旋转变换即可
- **例1**：设 $L$ 是平面上封闭光滑曲线，$n$ 是 $L$ 的外法向量，$l$ 是任意方向向量，则 $\dis\int_L \cos(l,n)ds = 0$
  - **证明**：
    - <font color='chartreuse'> 因为 $L$ 是封闭曲线，故任意 $\vec n(x_1,y_2)$，都必定存在 $(x_2,y_2)\in L$ 使得 $\vec n(x_1,y_1) = -\vec n(x_2,y_2)$。所有的相反向量两两配对，最终积分结果就是0（当然这只是口头说明，实际必须得用坐标来做，哪怕是n维欧氏空间）</font>
    - 设 $n = ( \cos\a, \cos\b )，l = (a,b)$，展开得 $I = \dis\int_L  \frac{a\cos\a + b\cos\b}{\sqrt{a^2+b^2}}ds$
    - 由法切引理，可转化为 $I = \dis\int_L \frac{adx - bdy}{\sqrt{a^2+b^2}}$，满足恰当性
  - <font color='red'>诸如此类的题，一般都是把方向导数替换为 $x_i$ 的函数。在此过程中会产生方向余弦，其和 $ds$ 合并即转化为第二类曲线积分</font>
- **例2**：设 $L$ 是平面上不经过原点的封闭光滑曲线，$n$ 是 $L$ 的外法向量，$r = (x,y)$，计算 $\dis\int_L \frac{\cos(r,n)}{|r|}ds$
  - **解**：
    - 设 $n = (\cos\a,\cos\b)$，则展开得 $\dis\int_L \cfrac{x\cos\a + y\cos\b}{x^2+y^2}ds$
    - 再由法切引理，不妨设切向量 $t = (-\cos\b,\cos\a)$，则转化为 $\dis\int_L \frac{xdy-ydx}{x^2+y^2}$，满足恰当性，易得为 $0$

### 格林公式与极坐标换元


- **上界放缩**：
  - 设 $f(x,y)$ 在圆周 $D:(0,0)，r\leq a$ 内部连续可偏导
  - 若 $f|_{\pa D} = 0$，则 $$\biggm| \iint_{D} f(x,y)dxdy \biggm| \leq \frac{\pi}{3}a^2\max\limits_D \sqrt{f_x^2+f_y^2}$$
  - **证明**：
    - 已知 $f = (xf)_x - xf_x$，对右边使用格林公式，再由 $f$ 在边界处为 $0$ 得 $$题设左式 = \dis\frac{1}{2}\iint_D (xf_x+yf_y)dxdy$$
      - <font color='red'>本质是对 $f$ 和 $\dfrac{x^2+y^2}{4}$ 使用第二格林恒等式</font>
    - 再对被积函数使用C-S不等式得 $\leq \dis\frac{1}{2}\iint_D \sqrt{f_x^2+f_y^2}\sqrt{x^2+y^2}dxdy$
    - 再用积分上界不等式提取出 $f$ 部分即可
- **极坐标换元1**：
  - 设 $D$ 是原点为心、半径为 $a$ 的闭圆，$L = \pa D$ 为其边界
  - 若已知 $f_{xx} + f_{yy}$ 的值，计算 $I = \dis\int_{L} f_r ds$
  - **证明**：
    - 由格林公式，$\dis\iint_D \D fdxdy = \int_L f_xdy-f_ydx$，设为 $G(a)$
      - 再由极坐标换元，得右式可变为 $\dis \int^{2\pi}_0 (f_x\cos\t + f_y\sin\t) d\t = \int^{2\pi}_0 f_rd\t$
      - 再由第一类曲线积分公式即得可变为 $\dis\int_L f_rds$
    - <font color='chartreuse'>由拉普拉斯算子的高斯格林公式可得 $$\dis \iint_{D} \D f d\bs x = \int_{\pa D}\frac{\pa f}{\pa \nu}ds = \int_{\pa D} f_xdy - f_ydx$$</font>
    - <font color='red'>这题最难的地方是要把（参数函数积分）逆向转化为（第一类曲线积分）</font>
- **极坐标换元2**：
  - 设 $D$ 是原点为心、半径为 $a$ 的闭圆，$L = \pa D$ 为其边界，$f$ 二阶连续可偏导
  - 若已知 $f_{xx} + f_{yy}$ 的值，计算 $I = \dis\iint_{D} (xf_x + yf_y)dxdy$
  - **证明**：
    - 取极坐标换元得 $I = \dis\int^a_0 rdr \int^{2\pi}_0 \Big[ f_x(r\cos\t)  + f_y(r\sin\t)  \Big]d\t$
    - 由于 $dx = -r\sin\t d\t$，故 $$ I = \int^a_0 rdr\int_{L_r} f_xdy - f_ydx$$
      - 其中 $L_r$ 表示半径为 $r$ 的单位圆
    - 前面已知右边积分为 $G(r)$，故 $I = \dis\int^a_0 rG(r)dr$
  - <font color='red'>这题巧妙的地方有两点
    - 首先，它极坐标换元后，又逆向转化为（第二类曲线积分）
    - 然后，它极坐标换元后，是先分离了一部分出去，才使得剩下的部分正好满足格林公式（直接用格林公式做不出来）</font>

## Green定理

- **格林定理**：设 $D$ 是单连通区域，$P,Q$ 连续可偏导，则下列命题等价
  - **封闭归零性**：对 $D$ 内任意分段光滑闭曲线 $L$，都有 $\int\limits_L Pdx+Qdy = 0$
  - **不定积分性**：$D$ 内任意曲线积分值都与路径无关
  - **原函数存在性**：存在通积分 $U(x,y)$
  - **恰当性**：$\dfrac{\partial P}{\partial y} = \dfrac{\partial Q}{\partial x}$
  - **证明**：
    - $1\to 2$：类似复变函数，端点确定为 $a,b$ 时，取任意 $L_1,L_2$ 都能组成闭曲线，故两者积分值相等
    - $2\to 3$：构造上限函数 $U(x,y) = \dis\int^{(x,y)}_{(x_0,y_0)}Pdx + Qdy$，中值定理 + 上限函数连续性，得两个偏导数相等，从而得到全微分相等
    - $3\to 4$： 
      - 若存在通积分，则 $dU = Pdx + Qdy$，即 $P = U_x，Q = U_y$。再由偏导数均连续即得混合偏导数相等
    - $4\to 1$
      - 若满足恰当性，由格林公式即得封闭归零性
- **求通积分**
  - 用最简单的积分路径——正方形积分
  - 得到恰当方程的求解结果：$\varPhi(x,y) = \int^x_{x_0}P(x,y)dx + \int^y_{y_0}Q(x_0,y)dy + C$
- **非封闭曲线的Green公式**：$$\int\limits_{\overset{\frown}{AB}}Pdx+Qdy = U(x_B,y_B) - U(x_A,y_A)$$
  - **证明**：上限函数相减，与其下端点无关
  - **理解**：就是全微分形式的N-L公式
- **复周线定理**：同复变函数


### 计算例（恰当方程）

- **复周线法**：设 $a,b > 0$，$L$ 是分段光滑闭曲线。讨论 $I = \dis\int_L \frac{xdy-ydx}{ax^2+by^2}$
  - **解**：
    - 被积函数仅存在一个瑕点 $(0,0)$
    - 易得 $P_y = Q_x$，故任意不包围原点的 $L$，都有 $I = 0$
    - 若 $L$ 包围原点，由复周线法，在 $L$ 围成的区域内，任取包含原点的周线 $L'$，都有 $\dis\int_L Pdx+Qdy = \int_{L'} Pdx+Qdy$
      - 为了化简积分，取 $L':ax^2 + by^2 = r^2$，则 $I = \dis\frac{1}{r^2}\int_{L'} xdy-ydx$，计算即得 $I = \dfrac{\pi r^2}{\sqrt{ab}}$
- **综合题**：设 $L$ 是不过原点的简单闭曲线，求 $$I = \int_L \frac{e^y}{x^2+y^2}\Big[ (x\sin x + y\cos x)dx + (y\sin x-x\cos x)dy \Big]$$
  - **解**：
    - 被积函数仅存在一个瑕点 $(0,0)$
    - 易得 $P_x = Q_y$，故任意不包围原点的 $L$，都有 $I = 0$
    - 若 $L$ 包围原点，由复周线法，取 $L':x^2 + y^2 = r^2$，则 $I = \dis\frac{-2}{r^2}\iint_{D'} e^y\cos xdxdy$
      - <font color='red'>如果取极坐标换元，就变为前面定积分和后面含参变量积分都提到的泰勒展开法积分。其实它也可以用极限区域法来做</font>
      - 取极限区域 $r\to 0+$，则由被积函数连续性 + 积分第一中值定理得 $$I = \lim_{r\to 0+} \frac{-2}{r^2} (e^\eta\cos\xi)(\pi r^2) = \lim_{(\eta,\xi)\to (0,0)} -2\pi(e^\eta\cos\xi) = -2\pi$$

- **简化曲线积分**：若满足恰当性
  - **法1**：$L$ 端点不变时，可将其改为较简单的形状（比如分段直线 $L_x$ 和 $L_y$）
  - **法2**：直接用ODE的知识求通积分，然后端点相减
- **求被积函数（高数题。。。）**：利用恰当性 + ODE直接求即可
  - **实例**：设 $\p$
- **恰当中值定理**：
  - 设 $L$ 是有限段光滑闭曲线，$f$ 连续，$P_y = Q_x$
  - 则存在通积分 $U$，且 $\dis\int_L f(U)(Pdx+Qdy) = 0$
  - **实例**：
    - $\dis\int_L f(x^2+y^2)(xdx+ydy) = 0$
    - **证明**：
      - 换元 $u = x^2 + y^2$，由连续性得可写成 $I = \dis\frac{1}{2}\sum^n_{i=1}\int^{u_i}_{u_1} f(u_i)du_i = \frac{1}{2}\int^{u_n}_{u_1} f(u)du$
      - 再由 $u_1 = u_n$ 即得为 $0$


## 曲面积分 $\LR$ 三重积分

- **三维单连通区域**：若 $\Omega\subset \R^3$ 内的任意封闭曲面所围成的立体仍然属于 $\Omega$，则称 $\Omega$ 为单连通区域
- **三维复连通区域**
- **Gauss公式**：设 $\Omega$ 是单连通区域，$P,Q,R$ 在其内可微 $$\iiint\limits_{\Omega}(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z})dxdydz = \iint\limits_{\partial \Omega} Pdydz + Qdzdx + Rdxdy$$
  - **理解**
  - **证明**：累次积分（分离合并法）
    - 以 $R$ 为例，$\dis\iiint\limits_{\Omega}\frac{\partial R}{\partial z}dxdydz = \iint\limits_{\Omega_{xy}}dxdy\int^{z_2(x,y)}_{z_1(x,y)} \frac{\partial R}{\partial z} dz$ 
      - <font color='chartreuse'>利用累次积分公式，将（三重积分）分离为（垂线积分）和（平面投影积分）</font>
    - 用N-L公式化为 $\dis\iint\limits_{\Sigma} R\Big( x,y,z_1(x,y) \Big)-R\Big( x,y,z_2(x,y) \Big) dxdy$
      - <font color='chartreuse'>对垂线积分应用N-L公式。其中 $\Sigma$ 是 $\Omega$ 在 $z$ 轴两端的曲面边界</font>
    - 调整符号得 $\dis\iint\limits_{\Sigma_2}R + \iint\limits_{\Sigma_1}R = \iint\limits_{\partial\Omega}R(x,y,z)dxdy$
      - <font color='chartreuse'>跟格林公式相同的讨论，发现可以融合为一个曲面积分</font>
  - **体积计算公式**

### 习题

- **高斯公式简化计算**：$I = \dis\iint_S axdydz + (z+a)^2dxdy$，其中 $S$ 是上半单位球
  - **解**：
    - <font color='chartreuse'>如果采用投影法，则 $(z+a)^2$ 会变成复杂函数
    - 如果采用高斯公式法，则平面上 $(z+a)^2$ 退化为 $a^2$，容易计算</font>
- **复周线法（三维版本）**：设 $r = \sqrt{ax^2+by^2+cz^2}$，$S$ 是任意不过原点的闭曲面，求 $$I = \iint_{S} \frac{x}{r^3}dydz + \frac{y}{r^3}dzdx + \frac{z}{r^3}dxdy$$
  - **解**：
    - 易得被积函数仅以原点为奇点，再易得 $P_x + Q_y + R_z = 0$，故
      - 若 $S$ 不包围原点，则 $I = 0$
      - 若 $S$ 包围原点，则由复周线得可取椭球面 $S:ax^2+by^2+cz^2 = R^2$
        - 此时转化为求 $\dis\frac{1}{R^3}\int_{S} xdydz + ydzdx + zdxdy$，换元即可
- **极限区域法**：
  - 设 $P,Q,R$ 连续可偏导
  - 若对任意的上半球面，第二类曲面积分均为 $0$，则 $P_x+Q_y = 0$ 且 $R=0$
  - **证明**：
    - 由高斯公式，任意上半球 $V$ 都有 $\dis\iiint_V (P_x+Q_y+R_z)dxdydz = -\iint_D Rdxdy$，其中 $D$ 是 $V$ 的底面
    - 再由连续可偏导得可微，从而连续，从而积分中值定理就有 $$\frac{2\pi r^3}{3}(P_x+Q_y+R_z)|_{P_0} + \pi r^2R|_{P_0} = 0$$
    - 约分后取 $r\to 0$ 就有 $R(P_0) = 0$，再由任意性即得结论
- **复球面法**：
  - 设 $r = \sqrt{ax^2+by^2+cz^2}$，求 $I = \dis\iint_S \frac{xdydz + ydzdx + zdxdy}{\sqrt{r^3}}$（$S$ 是不经过原点的封闭球面）
    - **解**：
      - 计算易得 $P_x  + Q_y + R_z = 0$
      - 若 $S$ 不包围原点，则由高斯公式得 $I = 0$
      - 若 $S$ 包围原点，则将 $S$ 转化为极小椭球 $ax^2+by^2+cz^2 \leq \e^2$
        - 此时 $I = \dis \frac{1}{\e^3}\iint_{S_0} xdydz + ydzdx + zdxdy$，方向取外侧
        - 再由高斯公式即可
- **补体法**：
  - 设 $r = \sqrt{x^2+y^2+z^2}$，求 $I = \dis\iint_S \frac{xdydz + ydzdx + zdxdy}{r^3}$，其中 $S: x^2+2y^2+3z^2=1\quad (z\geq 0)$，取上侧
    - <font color='chartreuse'>这里不是封闭球面，且直接补体会造成瑕点。考虑用对称性</font>
    - 由于 $\dfrac{z}{r^3}$ 是关于 $z$ 的奇函数，故 $I$ 是封闭椭球面积分的一半
    - 再由高斯公式 + 内部有瑕点，可转化为 $S_0:x^2+y^2+z^2 \leq \e^2$ 上积分的一半

### 第一类转化为第二类

- $S$ 是封闭光滑曲面，$l$ 为已知向量，则 $\dis\iint_S \cos(n,l)dS = 0$
  - **解**：
    - 由向量形式易得可转化为 $\dis\iint_S \frac{xdydz + ydzdx + zdxdy}{\sqrt{x^2+y^2+z^2}}$，再用高斯公式即可

### 高斯公式与格林恒等式

- 设 $f(x,y,z)$ 是 $k$ 次齐次函数，$V$ 是原点心单位球，则 $$\iint_S fdS = \frac{1}{k}\iiint_V \D fdxdydz$$
  - **证明**：
    - 由高斯公式，右式可转化为 $\dis\frac{1}{k}\iint f_xdydz + f_ydzdx + f_zdxdy$
    - 再由于球面的法向量为 $(x,y,z)$，故可转化为 $\dis\frac{1}{k}\iint (xf_x+yf_y+zf_z)dS$
    - 最后由齐次函数的欧拉公式即得结论
- **调和函数的广义均值定理**：
  - 设 $f$ 是调和函数，$S$ 是光滑封闭曲面，$\vec{r} = (x-x_0,y-y_0,z-z_0)$，则 $$f(x_0,y_0,z_0) = \frac{1}{4\pi}\iint_S \fkh{f\frac{\cos(\vec r,\vec n)}{r^2} + \frac{1}{r}\pfrac{f}{\vec n}} dS$$
  - <font color='chartreuse'>如果 $S$ 是球面，就是普通的均值定理</font>
  - **证明**：
    - <font color='red'>逆天配凑</font>
    - 易得 $\dfrac{1}{r}$ 的方向导数为 $-\cfrac{\cos(\vec{r},\vec{n})}{r^2}$，且其为调和函数
    - 故右式转化为 $$I = \frac{1}{4\pi}\iint_S \dkh{\frac{1}{r}\pfrac{f}{\vec n} - f\pfrac{(\frac{1}{r})}{\vec{n}}}dS $$
    - 由第二格林恒等式，可转化为极小球面上的积分，再由均值定理类似的方法即得结论

## 曲线积分 $\LR$ 曲面积分

- **曲面的边界**：空间曲线
- $\partial\Sigma$ **的诱导定向**：逆时针的右手定则
- **Stokes公式**：设 $\Sigma$ 是分段光滑曲面，$P,Q,R$ 在其上有连续偏导数，则 $$\hspace{-3em}\int\limits_{\partial\Sigma}Pdx+Qdy+Rdz = \iint\limits_{\Sigma}(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z})dydz + (\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x})dzdx + (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})dxdy$$
  - 即  $$\begin{cases} \dis\int\limits_{\partial\Sigma} P(x,y,z)dx = \iint\limits_{\Sigma}\frac{\partial P}{\partial z}dzdx - \frac{\partial P}{\partial y}dydx \\\\ \dis\int\limits_{\partial\Sigma} Q(x,y,z)dy = \iint\limits_{\Sigma}\frac{\partial Q}{\partial z}dzdy - \frac{\partial Q}{\partial x}dxdy \\\\ \dis\int\limits_{\partial\Sigma} R(x,y,z)dz = \iint\limits_{\Sigma}\frac{\partial R}{\partial x}dxdz - \frac{\partial R}{\partial y}dydz \end{cases}$$
  - **理解**：空间中的Green公式
  - **证明**：以标准曲面和 $P$ 为例，
    - 由第二类曲线积分的投影法，$\dis\int\limits_{\partial\Sigma}P(x,y,z)dx = \int\limits_{\partial \Sigma_{xy}}P(x,y,z(x,y))dx$
    - 在 $xy$ 平面上对其应用Green公式，化为二重积分即可
  - **行列式形式**：$\dis\int\limits_{\partial\Sigma}Pdx+Qdy+Rdz = \iint\limits_{\Sigma} \begin{vmatrix}\cos\alpha & \cos\beta & \cos\gamma \\ \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\P & Q & R \end{vmatrix}dS$
    - <font color='chartreuse'>方向由右手定则确定，即右手伸出大拇指，四指旋转方向是 $\pa\Sigma$ 正方向，拇指方向是 $\Sigma$ 正方向</font>

#### 计算例

- **形状位置的曲线**：
  - 设 $L:\begin{cases} x+y+z=2 \\ |x|+|y|=1 \end{cases}$，从 $z$ 轴上方往下看是逆时针，求 $$I = \dis\int_L (y^2-z^2)dx + (2z^2-x^2)dy + (3x^2-y^2)dz$$
  - **解**：
    - 易得围成的曲面法向量都是 $(1)$ 的一部分，故单位法向量恒为 $\dfrac{1}{\sqrt{3}}(1,1,1)$
    - Stokes计算得 $I = \dis-\frac{2}{\sqrt{3}}\iint_S (4x+2y+3z)dS$
    - 由平面方程 + 轮换对称性得可化为 $ I = -4\sqrt{3} S$
      - <font color='red'>这一步是关键，否则这题就太难算了</font>
    - 易得 $S$ 在平面的投影为 $D:|x|+|y|\leq 1$，拉直后计算其面积即可
- **维维安尼曲线**：
  - 设 $L:\begin{cases} x^2+y^2+z^2=4x \\ x^2+y^2=2x \\ z\geq 0 \end{cases}$，从 $z$ 轴上方往下看是逆时针，求 $$I = \dis\int_L (y^2+z^2)dx + (x^2+z^2)dy + (x^2+y^2)dz$$

#### 习题

- 设 $L$ 是光滑封闭曲线，$P,Q,R$ 一阶连续可偏导，则 $$ \abs{\int_L Pdx + Qdy + Rdz} \leq S\max\limits_S \sqrt{(R_y-Q_z)^2 + ...}$$
  - **证明**：
    - 由Stokes公式转化成内积形式，再对被积函数用C-S不等式 + 积分上界不等式即可


### Stokes定理

- **Stokes定理**：设 $\Omega\subset\R^3$ 是单连通区域，若 $P,Q,R$ 在其上可连续偏导，则以下条件等价
  - **封闭归零性**：对任意闭合光滑曲线 $L$，$\dis\int_L Pdx + Qdy + Rdz = 0$
  - **不定积分性**：对任意分段光滑曲线 $L$，$\dis\int_L Pdx + Qdy + Rdz$ 与路径无关
  - **势函数存在性**：存在 $u$ 使得 $du = Pdx + Qdy + Rdz$
  - **恰当性**：$P_y = Q_x，Q_z = R_y，R_x = P_z$
  - **证明**：和格林定理相同


#### 习题

- **恰当例子**：$I = \dis\int_L (x^2-yz)dx + (y^2-xz)dy + (z^2-xy)dz$
  - 其中 $L$ 是 $A(1,0,0)\to B(1,0,2)$ 的任意光滑曲线
- **证明恰当性**：
  - 设 $P,Q,R$ 二阶连续可微，三个行列子式分别为 $a,b,c$
  - 则存在 $\R^3$ 上的线性函数 $f,g,h$ 使得 $(P-f)dx + (Q-g)dy + (R-h)dz$ 是全微分
  - **证明**：
    - 证明全微分的本质就是证明是恰当方程，直接计算易得取 $f=bz,g = cx,h=ay$ 即可
