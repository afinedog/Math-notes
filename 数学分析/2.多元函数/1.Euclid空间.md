# Euclid空间上的基本定理

- 集合的定义：连续性定义、大小性定义

## 向量空间

- **高维实数空间**：$\R^n =\set{(x_1,x_2,...,x_n)\mid \forall x_i\in \R}$
  - **向量（点）**：该集合中的元素写作 $\bd x$ 或 $\vec x$，它可被看作空间中的一个点<font color='chartreuse'>（度量空间中的元素可看作点）</font>，也可看作空间中的一个向量<font color='chartreuse'>（线性空间中的元素可看作向量）</font>
    - <font color='chartreuse'>事实上，高维实数空间具有非常良好的性质。它同时是拓扑空间、度量空间、可测空间、线性空间、内积空间，还可以定义 $p$ 范数。</font>
  - **分量（坐标）**：$x_i$ 称为向量 $\vec x$ 的第 $i$ 个分量，或点 $\bd x$ 的第 $i$ 个坐标
  - **零向量（原点）**：$\bd 0 = \vec{0} = (0,0,...,0)$
- **Euclid空间**：定义了内积运算的线性空间
  - <font color='chartreuse'>欧几里得空间又称欧氏空间。它在高阶数学课程中经常用来提供平凡且容易理解的例子
  - 由高等代数的内容，所有的线性空间都同构于 $\R^n$，故一般称欧氏空间时都可以用 $\R^n$ 指代</font>

- **Euclid范数**：$\| x \| = \sqrt{\langle x,x \rangle}$
  - **几何意义**：$\vec{x}$ 到 $\vec{0}$ 的距离
  - **性质**：
    - **正定性**
    - **对称性**
    - **三角不等式**：$|x-z| \leq |x-y| + |y-z|$
      - **证明**：两边平方，然后线性分解、组合，得到$\langle x-y,y-z \rangle \leq |x-y||y-z|$（由均值不等式得成立）
      - 多元的三角不等式，可以使用均值不等式证明（平方关系是桥梁）

## 向量空间的点列极限

- **邻域 $O(x_0,\d)$**：与 $x_0$ 的距离小于 $\d$ 的所有点组成的集合
  - 邻域的中心
  - 邻域的半径
- **坐标极限**：
  - 设 $x_k = (x_1^k, x_2^k, ... x_n^k)$，$a = (a_1,a_2,...,a_n)$，则 $\lim\limits_{k\to \infty}x_k = a \LR \forall\lim\limits_{k\to \infty}x_i^k = a_i$
  - **充分性**：$n\e = \sum\limits^n_{i=1} |x_i^k-a_i| > |x_k-a|$（三角不等式）
    - <font color='chartreuse'>所以只能在有限维空间 $\R^n$ 上成立，在无限维空间上不成立</font>
  - **必要性**：$\e > |x_k-a| > |x_i^k-a_i|$ 直得结论
  - **理解**：因为是同阶无穷小，所以彼此等价<font color='chartreuse'>（实际上是泛函分析中的定理，$1$ 范数和 $2$ 范数等价）</font>）
- **极限的性质**
  - 唯一性
  - 有界性
  - 无保序性、保号性、夹逼性

### 点的分类

- <font color='chartreuse'>下面概念的严格定义详见拓扑学
  - 实际上不少人推荐先学点集拓扑再学这一章，但我觉得从容易理解的概念入手更符合学习曲线。不过，在粗浅、不严谨的知识中耗费太多精力就不应该了，这也是我比较后悔的一点
  - 这一章的定理用元素分析法基本都能做</font>
- **几种特殊的点**
  - **聚点**：若 $x_0$ 的任意邻域均与集合 $S$ 相交，则称其为 $S$ 的聚点
  - **内点**：若 $x_0\in S$ 存在一个邻域完全含于集合 $S$ 中，则称其为 $S$ 的内点
    - **集合的内部**：内点组成的集合
    - 内点一定是聚点，反之不成立
  - **边界点**：若 $x_0$ 的任意邻域均与 $S$ 部分相交，部分不相交，则称其为 $S$ 的边界点
    - **集合的闭包**：开集与其边界的并集称为开集的闭包<font color='chartreuse'>（）</font>
  - **孤立点**：若 $x_0\in S$ 存在一个邻域完全与集合 $S$ 不相交，则称其为 $S$ 的孤立点
    - 孤立点一定是边界点，反之不成立
- **集合分类**：
  - **开集**：所有点都是内点的集合
  - **闭集**：包含所有聚点的集合
- **聚点极限性**：$x$ 是 $S$ 的聚点 $\LR \exist {x_k}\in S$ 使得 $x_k\neq x$ 且 $\lim\limits_{k\to\infty} x_k = x$
  - **充分性**：
    - 设 $\d_k = |x_k-x|$，构造一个邻域列 $O(x,\d_k)$
    - 由于 $\d_k\to 0$，故 $x$ 的任意邻域均含于某个 $O(x,\d_k)$ 中
    - <font color='chartreuse'>这里的很多证明思想在拓扑中经常用，比如构造邻域列。但是用数分的实数思维 + 元素分析法也能做</font>
  - **必要性**：同上，构造邻域列
- **邻域是开集**：邻域中的点都是内点
  - **证明**：定义即可
    - 设邻域为 $O(x_0,\d)$，则其中某点 $y$ 满足 $|y-x| = q < \d$
    - 故取邻域 $O(y,\dfrac{\d-q}{2})$，其即为完全包含于 $O(x_0,\d)$ 的集合，从而由定义 $y$ 是内点  
- **开闭对偶性**：设 $S$ 是 $\R^n$ 上的点集，则 $S$ 是闭集 $\LR S^c$ 是开集
  - **证明**：
    - **必要性**：$S$ 包括所有聚点 $\rightarrow$ $S^c$ 不包含 $S$ 的聚点  $\rightarrow$ 元素分析法即可
    - **充分性**：因为闭集情况多，所以闭集判定一般都用反证法
      - 反设 $S$ 不是闭集，定义即可

### 无限个集合的情况

- **De Morgo公式**：
- **开集的并封闭性**：任意一组开集 $S_\alpha$ 的并集 $\mathop{\cup}\limits_{\alpha}S_\alpha$ 都是开集
  - **证明**：定义直得<font color='chartreuse'>（因为并集运算不会改变内点原先的 $\delta$ 邻域）</font>
- **闭集的交封闭性**：任意一组闭集 $S_\alpha$ 的交集 $\mathop{\cap}\limits_{\alpha}T_\alpha$ 是闭集
  - **证明**：
    - **聚点的交传递性**：如果点 $a$ 在 $S_1$ 中是聚点，在 $S_2$ 中也是聚点，则其在交集中也是聚点<font color='chartreuse'>（因为交运算可以全部继承原先的邻域列）</font>
    - 如果点 $a$ 在 $S_1$ 中是聚点，在 $S_2$ 中不是聚点，则其在交集中不是聚点（不继承邻域）
    - 因此 $\mathop{\cap}\limits_{\alpha}T_\alpha$ 依然包括所有的聚点
  - （也可以使用 De Morgan公式由(1)推(2)）
- **开集的有限交封闭性**：任意有限个开集 $S_1,S_2,...,S_k$ 的交集 $\mathop{\cap}\limits_{i=1}^k S_i$ 是开集
  - **反例**：无限个开集的交集可能是一个点，而单点集是闭集
  - **证明**：定义即可
- **闭集的有限并封闭性**：任意有限个闭集 $T_1,T_2,...,T_k$ 的并集 $\mathop{\cup}\limits_{i=1}^k T_i$ 是闭集
  - **反例**：无穷个闭区间 $[a+\d_n,b-\d_n]$，若 $\d_n\to 0$，则由定义，其并集为开集 $(a,b)$
  - **证明**：定义即可

## Euclid空间上的基本定理

- 这些定理都可以用坐标极限分解为 $\R$ 上的实数系基本定理，故证明略去

### 闭矩形套定理

- **闭矩形套定理（二维情况）**：
  - 设 $\D_k = [a_k,b_k]\times [c_k,d_k]$ 是 $\R^2$ 上一列闭矩形
  - 若
    - **矩形逐渐缩小**：$\D_{k+1}\subset \D_k$
    - **矩形面积极限为0**：$\lim\limits_{k\to\infty}\sqrt{(b_k-a_k)^2+(d_k-c_k)^2} = 0$
  - 则
    - 存在唯一的点 $\bd a = (\xi,\eta)\in \mathop{\bigcap}\limits^\infty_{k=1} \D_k$
    - $\lim\limits_{k\to\infty} a_k = \lim\limits_{k\to\infty} b_k = \xi$
    - $\lim\limits_{k\to\infty} c_k = \lim\limits_{k\to\infty} d_k = \eta$
- **Cantor闭区域套定理**：
  - 设 $\{S_k\}$ 是 $\R^n$ 上的非空闭集列
  - 若
    - **闭集逐渐缩小**：$S_k \supset S_{k+1}$
    - **面积极限为0**：$\lim\limits_{k\to\infty} \diam S_k = 0$
  - 则
    - 存在唯一的点 $\bd x\in \mathop{\bigcap}\limits^\infty_{k=1} S_k$
- **应用**：证明三角形三条中线交于一点
  - **证明1**：
    - 中线的几何意义之一，是将三角形分割两块面积相等的子三角形
    - 而三条中线交于一点时，会将三角形分隔为六块面积相等的子三角形
    - 设 $x$ 点与 $A,B,C$ 三点的连线将三角形分割后的最大子区域面积为 $f(x)$，易得其连续。由闭区域上连续函数的最值定理即得存在性（貌似要用到后面的结论）
  - **证明2**：
    - 设 $\D_1 = \D ABC$，取其三个中位线，设中位线三角形为 $\D_2$
    - 不断重复这个过程，由于中线均含于三角形内部，故 $\D_{k+1}\subset \D_{k}$
    - 由闭区域套定理，其最终收缩为一个点，该点即为重心

### 凝聚定理

- **凝聚定理**：$\R^n$ 上的有界点列必有收敛子列
  - **推论**：$\R^n$ 上的有界无限点集必定有一个聚点

### Cauchy收敛原理

- **基本点列**：若 $\forall \e>0，\exist N$ 使得 $\forall m,n>N$ 都有 $|\bd x_m-\bd x_n| < \e$，则称为基本列
- **柯西收敛原理**：$\R^n$ 上的点列收敛 $\LR$ 其为基本点列

### 有限覆盖定理

- **开覆盖**：若点集 $S$ 含于一列开集 $\{S_k\}$ 之并，则称 $\{S_k\}$ 为 $S$ 的开覆盖
- **紧集**：任意开覆盖中都存在有限子覆盖的集合
  - <font color='chartreuse'>关于提出紧集的意义，详见[拓扑学](../../拓扑学/点集拓扑/第3章下：紧致性.md)</font>
- **Heine-Borel定理**：在 $\R^n$ 上，紧集 $\LR$ 有界闭集
  - **必要性**：
    - **有限个开区间之并**是有限的，所以**紧集是有界集**
    - 如果紧集是有限个点的集合，那么既然存在开覆盖（已经覆盖），则必然满足有限子覆盖（把没有点的覆盖都约去）。所以**紧集是无限点集（存在聚点）**
    - **逼近构造法（反证）（用于构造点集列）**：$设某点a\notin S，开集U_n = \{x|\quad|x-a|>\frac{1}{n} \}$
      - 如果a是紧集的聚点，那么$U_n$如果是开覆盖，它必须是无限并集（这个构造比较巧妙），所以不是有限子覆盖，与紧集矛盾
      - 而由紧集定义中有限子覆盖的任意性 $\rightarrow$**紧集外不能有聚点**$\rightarrow$ **紧集是闭集**
    - 综上，紧集是有界闭集
  - **充分性**：
    - 因为无限个开集的并集还是开集，所以 $\mathop{\cup}\limits_{n}U \supsetneq S$
    - 接下来用反证法。<font color='red'>还是老套的证明思路</font>
      - **B-W定理**：对开覆盖的端点（闭包的边界）设立点列，那么它必定收敛于一个点。
        - 如果这个点在S外部，那么必定存在无限个可约去的子覆盖，变为有限子覆盖
        - 另外，这个点不可能在S内部
        - 另外，这个收敛点会成为一个聚点，与闭集包含所有的聚点相矛盾（√）
      - **闭矩形套定理**：选择不可被有限覆盖的区域，直到收缩成一个点，然后发现该点可以被有限开覆盖（√）
- **紧集性质**：
  - 有界性
  - 闭集
  - 无限子集必定有聚点
    - 有界闭集 $\rightarrow$ 聚点：B-W定理
    - 聚点 $\rightarrow$ 有界闭集：反证，如果S是开集，那么我偏偏设{$x_k$}的收敛点（聚点）在边界点上。这就构成矛盾
    - 紧集 $\LR$ 聚点：有限开覆盖，而开集中所有点都是聚点

## 习题

1. 证明极限性质：使用$\epsilon$的**任意性**和**极小性**即可
2. 证明极限运算法则：使用三角不等式和配凑
3. 求 $S'$ 聚点集合（**导集**）、$S^o$ 内部、$\partial S$ 边界集合、$\bar{S}$ 闭包：根据定义即可
- 证明Euclid空间基本定理：互相证明，寻找情况（比如B-W定理就找点列，这个和实数系上区别不大）（有时需要用偏极限定理来导入实数系上的引理）
- **聚点传递性**：设 $M$ 为 $x_n$ 的聚点集，则 $M$ 的聚点也是 $x_n$ 的聚点
  - **证明**：定义放缩即可
  - **推论**：$\R$ 中不存在聚点集为 $M = \{\dfrac{1}{n}\}^\infty_{n=1}$ 的数列 $x_n$
    - **证明**：$0$ 是 $M$ 的聚点，但不是 $x_n$ 的聚点，矛盾
    - **理解**：$M$ 的聚点集为一个区间，其含有不可列个点。而由实变习题知，可数集的聚点至多可数，故矛盾
