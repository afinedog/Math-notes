# 偏导数在几何中的应用

- **曲线**切向量是部分Jacobi矩阵的情况：
  - 两个曲面相交，此时利用Jacobi矩阵的相交性（方程组要同时满足两个方程）
- **曲面**法向量是部分Jacobi矩阵的情况：
  - 两个曲线

- 曲线没有grad、Jacobi。
- 曲线（相交式）的Jacobi是切向量
  - 先有了曲线Jacobi是切向量，直得曲面Jacobi是切向量，才有了曲面grad是法向量
- 曲面的grad是法向量，曲面没有Jacobi。
- 曲面（参数）的Jacobi是法向量，$grad_{u/v}$ 是切平面向量

  - <font color='chartreuse'>实际上微分流形中的光滑定义应该是无穷阶连续可微。这里适当减弱了一下</font>

- **割线**：连接 $P_0(x_0,y_0)$ 和 $P_1(x_0+\D x,y_0+\D y)$ 的直线
  - **割线方程（参数形式）**：$\cfrac{x-x(t_0)}{x(t_1)-x(t_0)} = \cfrac{y-y(t_0)}{y(t_1)-y(t_0)} = \cfrac{z-z(t_0)}{z(t_1)-z(t_0)}$
- **切线**：割线的极限（$\D x,\D y\to 0$）
    - **切线方程（参数形式）**：$\dis\frac{x-x(t_0)}{x'(t_0)} = \frac{y-y(t_0)}{y'(t_0)} = \frac{z-z(t_0)}{z'(t_0)}$

      - 割线和切线的关系就是差分和微分的关系</font>
  - **切向量**：$\vec{r'}(t_0) = (\vec{x'}(t_0),\vec{y'}(t_0),\vec{z'}(t_0))$<font color='chartreuse'>（即曲线在某点 $P_0$ 处沿切线的方向向量）</font>

### 法平面

- **曲线 $\Gamma$ 在 $P_0$ 的法平面**：过点 $P_0$ 且与切线垂直的平面
- **法平面方程（法式方程）**
  - **参数形式**：$x'(t_0)(x-x_0) + y'(t_0)(y-y_0) + z'(t_0)(z-z_0) = 0$
    - <font color='chartreuse'>切向量是切线的方向向量，同时是法平面的法向量</font>
  - **隐函数形式**：$(x-x_0) + f'(x)(y-y_0) + g'(x)(z-z_0) = 0$
    - <font color='chartreuse'>参数形式的两边同除 $x'(t_0)$ 即可</font>

### 曲线与曲面的关系

- **曲线表示为两个曲面的交**：
  - 若 $\begin{pmatrix} F_x & F_y & F_z \\ G_x & G_y & G_z \end{pmatrix}$ 在 $P_0$ 点满秩
    - <font color='chartreuse'>否则退化成一个曲面 $\LR$ 两个grad向量共线</font>
  - 则曲线可表示为 $\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}$（没有参数导数形式，只能用隐函数导数形式。）
- **隐函数形式**：对该向量值函数的隐函数求导，梯度向量即为切线向量
  - **切向量（两个曲面切平面的交）**：$\vec{\tau} = (\cfrac{\partial (F,G)}{\partial(y,z)}, \cfrac{\partial (F,G)}{\partial(z,x)},\cfrac{\partial (F,G)}{\partial(x,y)})$
    - **证明**：
      - 由行列式的性质，易得 $\vec \tau\cdot \grad F(P_0) = \vec\tau\cdot \grad G(P_0) = 0$，即该向量与两个切平面的梯度向量（法向量）均垂直
  - **法平面（两个曲面的grad向量（法向量）的平面）**：$$\frac{\partial (F,G)}{\partial(y,z)}(x-x_0)+\frac{\partial (F,G)}{\partial(z,x)}(y-y_0)+\frac{\partial (F,G)}{\partial(x,y)}(z-z_0) = 0$$
    - **证明**：
      - 只需证明其与两个曲面的法向量 $\grad F(P_0)，\grad G(P_0)$ 均垂直即可
    - 垂直 $\Leftarrow$ 曲面的隐函数版本定义（0） $\Leftarrow$ 隐函数的导数和J有关 $\Leftarrow$ 链式法则

## 空间曲面

### 曲面的隐函数方程——法向量与切平面

- **曲面的隐函数方程**：$F(x,y,z) = 0$，其中 $F$ 具有连续的偏导数，且 $J_F$ 不退化
  - <font color='chartreuse'>隐函数的形式比显函数更加简便灵活，所以这里直接用隐函数代指曲面</font>
- **曲面的法向量**：若曲面在点 $P_0$ 的任何一条切线都与一个向量垂直，则该向量称为该曲面在 $P_0$ 的法向量
  - **梯度垂直性**：曲面的梯度向量 $\grad F(P_0) = (F_x,F_y,F_z)$ 是其法向量
  - <font color='chartreuse'>这里所说的隐函数梯度向量是三维的，前面所提的显函数梯度向量是二维的</font>
  - **证明（方程法）**：$F(P_0) = 0$ 中对 $t$ 求导得 $$F_x(P_0)x'(t) + F_y(P_0)y'(t) + F_z(P_0)z'(t) = 0$$，即切向量和梯度向量垂直
  - **证明（微分法）**：在曲面上任取一点 $P(x,y,z)$，由定义 $F(P)-F(P_0) = 0$
    - 若 $P\to P_0$，则有 $F_x(x-x_0) = F_y(y-y_0) = F_z(z-z_0) = \D F = 0$，从而切向量与梯度向量垂直
- **曲面的切平面**：过曲面上 $P_0$ 点且与该点法向量 $\grad F(P_0)$ 垂直的平面<font color='chartreuse'>（该点法向量对应的曲面）</font>
  - **切平面方程**：$F_x(x-x_0) + F_y(y-y_0) + F_z(z-z_0) = 0$
- **曲面的法线**：过曲面上 $P_0$ 点且与该点切平面垂直的直线<font color='chartreuse'>（该点法向量对应的直线）</font>
  - **法线方程**：$\cfrac{x-x(t_0)}{F_x} = \cfrac{y-y(t_0)}{F_y} = \cfrac{z-z(t_0)}{F_z}$
- **光滑曲面**：具有连续变动的切平面的曲面（随着切点在曲面上的变化而连续变动）
- **切平面的唯一性**：$F$ 可微 $\LR$ 可以用切平面函数唯一指代 $F$

### 曲面的参数方程

- **曲面的参数方程**：$\begin{cases} x = x(u,v) \\ y = y(u,v) \\ z = z(u,v) \end{cases} \quad (u,v)\in D$
- **导数**：
  - **逆映射存在性**：由于曲面光滑，故 $x,y,z$ 连续可导，从而存在逆映射
  - **曲面标准化**：把 $F(x,y,z) = 0$ 写成显函数形式：$f(x,y) = z\Big( u(x,y),v(x,y) \Big)$，即化成标准曲面
    - 参数形式可以直接求出 $z_u、z_v$ 两个外部导数
    - 利用逆映射定理，可以求出 $u_x、u_y、v_x、v_y$ 四个内部导数。
    - 链式法则即得 $z_x、z_y$。即可写出法向量，从而写出切平面
  - **导函数公式（隐函数形式）**：$$\begin{cases}
    \dis\frac{\partial f}{\partial x} = -\frac{\partial (y,z)}{\partial(u,v)}/\frac{\partial (x,y)}{\partial(u,v)}
    \\ \\
    \dis\frac{\partial f}{\partial y} = -\frac{\partial (x,z)}{\partial(u,v)}/\frac{\partial (x,y)}{\partial(u,v)}
  \end{cases}$$
  - **导函数公式（参数形式）**：乘一个 $z'(u,v) = \dfrac{\partial (x,y)}{\partial(u,v)}$ 即可
- **法向量（参数形式）**：$\vec{n} = (\cfrac{\partial (y,z)}{\partial(u,v)}, \cfrac{\partial (z,x)}{\partial(u,v)}, \cfrac{\partial (x,y)}{\partial(u,v)})$
  - 同时n还可以当向量值函数的方向余弦
  - **微分面积公式**：曲面中，$|dS| = \|\vec{du}×\vec{dv}\| = |\vec{n}||du||dv|$
    - 叉积的向量意义 $\LR$ 面积意义
- **法向量（隐函数形式）**：$\vec{n} = -(\cfrac{\partial (y,z)}{\partial(u,v)}/\cfrac{\partial (x,y)}{\partial(u,v)}，\cfrac{\partial (z,x)}{\partial(u,v)}/\cfrac{\partial (x,y)}{\partial(u,v)}，1)$

- **Lyapunov不等式**：设 $D$ 为凸的有界闭区域，曲面方程为 $z = f(x,y)$ 存在有界的二阶导数
  - 设 $\p(P_1,P)$ 为曲面在 $P,P_1$ 两点法线的夹角，$d(P_1,P)$ 为两点的距离
  - 则 $P- P_1\to \bs 0$ 时，$\p(P_1,P) \leq cd(P_1,P)$
  - **证明**：
    - $\sin^2\p = \cfrac{|\bs n(P_1)\times \bs n(P)|^2}{|\bs n(P_1)|^2|\bs n(P)|^2} \leq |\bs n(P)\times \bs n(P_1)|^2$
    - 设上界为 $M$，则由多元中值定理，$|f_x(P_1)-f_x(P)| \leq 2Md(P,P_1)$，对 $y$ 同理
      - 配凑 + 三角不等式得 $|f_x(P_1)f_y(P) - f_x(P)f_y(P_1)| \leq 4M^2d(P,P_1)$
    - 结合前面的叉积不等式，即得 $\sin^2\p \leq (1+2M^2)8M^2d^2$
    - 已知 $d\to 0$ 时 $\p\in [0,\dfrac{\pi}{2}]$ ，故易得 $0\leq \p(P,P_1)\leq \dfrac{\pi}{2}\sin\p \leq cd(P,P_1)$
- **正交曲面存在性**：设 $a>b>c>0$，则 $\R^3$ 中任意一点 $M$ 处，都有三个二次曲面通过并彼此正交
  - 方程为 $$\frac{x^2}{a^2-\l^2_i} + \frac{y^2}{b^2-\l^2_i} + \frac{z^2}{c^2-\l^2_i} = -1$$
  - **证明**：
    - **存在性**：
      - 只需要证明存在3个 $\l_i$ 即可
      - 将题设式子化为多项式得 $F(\l^2) = ...$，其为 $\l^2$ 的三次多项式，且在 $[c,b]，[b,a]，[a,+\infty)$ 端点异号，即仅有三个不同实根
    - **正交性**：
      - 易得法向量为 $\bs n_i = (\dfrac{2x}{a^2-\l_i^2}，\dfrac{2y}{b^2-\l_i^2}，\dfrac{2z}{c^2-\l_i^2})$
      - 彼此做内积即可验证正交性
