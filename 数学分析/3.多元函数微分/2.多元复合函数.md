# 多元复合函数

## 偏导数链式法则

- **多元复合函数**：
  - 设
    - $g(u,v) = \Big( x(u,v)，y(u,v) \Big)$ 是 $2$ 元 $2$ 维向量值函数
    - $f(u,v)$ 是 $2$ 元函数  
  - 则 $f$ 和 $g$ 的复合函数为 $(f\circ g)(u,v) = f\Big( x(u,v),y(u,v) \Big)$

- **多元函数的链式法则**：
  - 若 $g$ 可导，$f$ 可微
    - <font color='chartreuse'>$g$ 没有偏导，故条件为至少可导
    - 当 $u$ 变化时，$x$ 和 $y$ 都发生变化，所以 $f$ 必须满足可微性</font>
    - **充分条件**：$f$ 存在连续的偏导数
      - <font color='chartreuse'>因为（偏导数不连续但 $f$ 可微）的情况只可能出现在孤立点上，所以一般都使用存在连续的偏导数作为 $f$ 可微的条件替代</font>
  - 则 $f\circ g$ 可偏导，且偏导数为 $$(fg)_u(u_0,v_0) = f_x(x_0,y_0)x_u(u_0,v_0) + f_y(x_0,y_0)y_u(u_0,v_0)$$

    - <font color='chartreuse'>仿照一元函数的链式法则，配凑 + 定义即可</font>

### 极坐标下的偏导数

- 设 $z = f(x,y) = g(r,\t)$，则
  - **用极坐标表示标准坐标** $$ \begin{cases} f_x = g_r\cfrac{x}{\sqrt{x^2+y^2}} + g_\t \cfrac{y}{x^2+y^2}  \\\\ f_y = g_r\cfrac{y}{\sqrt{x^2+y^2}} + g_\t \cfrac{-x}{x^2+y^2} \end{cases} $$
  - **用极坐标表示标准坐标** $$ \begin{cases} g_r = &   f_x\cos \t + f_y\sin\t \\\\ g_{\t} = & r\Big[ -f_x\sin \t + f_y\cos\t \Big] \end{cases} $$
- **常用公式**：
  - $z_x^2 + z_y^2 = z_r^2 + \cfrac{z_\t^2}{r^2}$
    - <font color='chartreuse'>如果用 $f,g$ 表示的话更容易看出区别，用 $z$ 表示的话更容易看出联系</font>
  - $xz_x + yz_y = rz_r$

## 一阶全微分的形式不变性

- **一阶微分的形式不变性**
  - 不管 $u$ 是中间变量还是自变量，对 $f$ 求一阶微分都有相同的形式
    - u是自变量：$df = f'(u)du + \omicron(du)$
    - u是中间变量：$df = f'(u)·u'(x)dx + \omicron(dx) = f'(u)du + \omicron(du)$
      - <font color='chartreuse'>因为复合函数的两个函数f、u都必须连续，f(u(x))才可导，所以$\omicron()$里面始终是阶数最高的无穷小。而 $u$ 可微时 $du = o(dx)$</font>
- **一阶全微分的形式不变性**
  - 当 $x,y$ 是自变量时，$$df = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dy + \omicron(\sqrt{dx^2+dy^2})$$
  - 当 $u,v$ 是自变量时，$$df = (\frac{\partial f}{\partial x}·\frac{dx}{du}du + \frac{\partial f}{\partial x}·\frac{dx}{dv}dv) + (\frac{\partial f}{\partial y}·\frac{dy}{du}du + \frac{\partial f}{\partial y}·\frac{dy}{dv}dv) + \omicron(\sqrt{du^2+dv^2})$$
    - 通分即得 $$df = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dy + \omicron(\sqrt{dx^2+dy^2})$$
    - <font color='chartreuse'>实际上，前面链式法则的证明核心就是“可导等价于 $\omicron(\sqrt{(\D x)^2+(\D y)^2})$ $\sim \omicron(\D u)$，所以前者可以取代后者”</font>

- **方法不同，思想相同**：设 $z = f(x,y)$ 有连续偏导数，若 $xf_x(x,y) + yf_y(x,y) = 0$，则 $z$ 为常函数

    - 转化为极坐标，则易得 $\dfrac{\partial f}{\partial r} = 0$，即 $f$ 的值与 $r$ 无关。任选 $r=0$，查看各个方向上是否相同即可
    - 由偏导连续性得可微性，故所有方向极限相同，故 $f(\theta) = f(0)\quad ((x,y)\to 0)$
  - **证明（不换元法）**：
    - 分别取 $x = 0，y = 0$，易得 $f_x(x,0) = f_y(0,y) = 0$，即 $(0,0)$ 处偏导数均为 $0$
    - 再由偏导连续性得可微性，从而 $(0,0)$ 附近所有方向导数均为 $0$
    - 任取一个直线方向，设该方向上函数退化为 $g(x) = f(x,kx)$，则 $g'(x) = f_x +k f_y$
      - 我们惊喜地发现，此时 $xg'(x)$ 就是题设的式子形式，其恒为 $0$，再由 $x$ 任意性即得 $g'(x) \equiv 0$，即任何直线方向上 $f$ 都是常函数。这样，我们就得到了极坐标换元的灵感。

- **Lagrange公式**：设 $\begin{cases} u = f(z) \\ z = x+y\p(z) \end{cases}$，则 $\dfrac{\pa^n u}{\pa y^n} = \dfrac{\pa^{n-1}}{\pa x^{n-1}}\Big[ \p^n(z)\dpfrac{u}{x} \Big]$
  - **证明**：归纳即可
    - <font color='chartreuse'>要先搞懂依赖关系。虽然由题设关系式，$z,x,y$ 中任意变量都是另外两个的函数，但显函数形式已知的只有 $x(y,z) = y\p(z) - z$ 和 $y(x,z) = \dfrac{z}{\p(z)} - \dfrac{x}{\p (z)}$。显然前者更容易求导</font>

- **齐次函数退化引理**：$k$ 次齐次函数的偏导数是 $k-1$ 次齐次函数
  - **证明**：
    - 对齐次函数定义式两边关于 $x$ 求导即可
- **齐次函数的微分性质**：设 $f(x,y)$ 是 $n$ 阶可微的 $k$ 次齐次函数，则 $$\Big[ x\pfrac{}{x} + y\pfrac{}{y} \Big]^n f = k(k-1)\cdots (k-n+1)f$$

      - **证明**：计算出V行列式的值，直接求即可

      - **证明**：易得 $u$ 是 $\dfrac{n(n-1)}{2}$ 次齐次函数，由欧拉定理直得结论

## 微分中值定理

### 多元函数的微分中值定理

- **凸区域**：若区域是凸集，则称为凸区域
  - 设 $D \subset \R^n$ 是区域
  - 若
    - 连接 $D$ 中任意两点的线段都完全属于 $D$
    - 对 $\forall x,y\in D，\l\in[0,1]$，都有 $x + \lambda(y-x)\in D$
  - 则称 $D$ 为凸区域
- **L中值定理**：
  - 设二元函数 $f(x,y)$ 在凸区域 $D$ 上可微
    - <font color='chartreuse'>这里的L中值定理本质是定义在道路上的，所以必须取包含整个道路的凸区域</font>
  - 则 $\forall \D x,\D y，\exists \t\in (0,1)$ 满足 $$f(x_0+\D x, y_0+\D y) - f(x_0,y_0) \\ \ \\ = f_x(x_0+\theta\D x, y_0+\theta\D y)\D x + f_y(x_0+\theta\D x, y_0+\theta\D y)\D y$$
    - <font color='chartreuse'>如果分别对 $x$ 轴和 $y$ 轴使用中值定理，得到的不是题设形式，而且也不能保证中间点位于凸区域中
    - 所以必须走对角线道路，不能走正方形边界道路</font>
  - **证明**：
    - 取端点为 $x,y$ 的线段，由 $D$ 是凸区域得其仍然在 $D$ 中
    - 构造一元函数 $g(t) = g(x_0) + t\Big[ g(y_0)-g(x_0) \Big]$，应用一元L中值定理即可
      - <font color='chartreuse'>这个函数本质就是 $f$ 和道路函数 $\g$ 的复合</font>
- **导数的增长率意义**：区域 $D$ 中，$f_x、f_y\equiv 0\LR f$ 为常函数
  - **证明**：
    - **充分性**：显然
    - **必要性（端点效应法）**：
      - <font color='chartreuse'>一元函数中，是使用L中值定理，把增长值变为导数中值
      - 多元函数中，同样也用L中值定理的话，就需要讨论道路，且构造局部凸区域（即邻域）</font>
      - 设 $(x_0,y_0)\in D$，只需证明任意道路上均为常函数即可
      - 设 $\gamma(t)$ 是 $(x_0,y_0) \to (x,y)$ 的 $D$ 中道路
       - 取 $$t_0 = \sup\hkh{s\in [0,1] \biggm| f(\gamma(t)) = f(\gamma(0)), t\in [0,s] }$$
       - 反设不是常函数，则 $t_0<1$
         - 由 $f$ 连续性得 $f(\g(t_0)) = f(\g(0))$
         - 由 $D$ 是区域得存在 $O(\g(t_0),\d)\subset D$
         - 由邻域是凸集 + L中值定理 + 偏导数为 $0$ 得邻域内 $f$ 是常函数
         - 故此时 $t_0+\dfrac{\d}{2}$ 也在集合中，与 $t_0$ 确界性矛盾
  - **推论**：
    - 若 $f_x(x,y) \equiv 0$，则 $f(x,y)$ 是关于 $y$ 的一元函数 $\p(y)$
    - 若 $f_{xy}(x,y) \equiv 0$，则 $f(x,y) = f(x,0) + f(0,y) - f(0,0)$

### 多元函数的泰勒公式

- **多元Taylor公式**：
  - **第 $k$ 项**：$$\cfrac{\big[ (x-x_0)\frac{\partial}{\partial x}+(y-y_0)\frac{\partial}{\partial y} \big]^k}{k!}f(x_0,y_0)$$
  - **Lagrange余项**：$$R_k = \frac{\Big[(x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y}\Big]^{k+1}}{(k+1)!}f(x_0+\theta\D x,y_0+\theta\D y)$$
    <!-- - 跟一元函数的L中值一样，中值是导数中值而不是自变量微分中值 -->
  - **证明**：
    - 构造辅助函数 $\varphi(t) = f(x_0+t\D x，y_0+t\D y)$
    - 由莱布尼茨公式得 $$\varphi^{(k)}(t) = (\frac{\partial}{\partial x}\D x + \frac{\partial}{\partial y}\D y)^kf(x_0+t\D x,y_0+t\D y)$$
      <!-- - （链式法则，先计算再代入，有$\partial u = \partial x$） -->
    - 对该一元函数在 $t=1$ 处进行泰勒展开，并构建L余项即可
  - **理解**：
    - 全微分公式表明，多元微分可以是一元微分的线性组合，从而跟多元中值定理一样，多元Taylor公式也可用道路函数证明
    - <font color='chartreuse'>还是和外微分的思想有很深的联系</font>

### 习题

- 设 $F$ 在 $\R^3$ 中有连续的一阶偏导数
  - 若恒有 $y\dpfrac{F}{x} - x\dpfrac{F}{y} + \dpfrac{F}{z} \geq \a > 0$
  - 则沿着曲线 $\begin{cases} x = -\cos t \\ y = \sin t \\ z = t \end{cases}$ 趋于正无穷时，$F\to +\infty$
    - **证明（裴礼文）**：
      - 已知一元泰勒公式 $\Phi(t) = \Phi(0) + \Phi'(\tau)t$，代入后易得结论
    - **证明（我）**：
      - 沿着曲线时，$F(x,y,z)$ 退化为 $\p(t) = F(-\cos t,\sin t,t)$
      - 求导易得 $\p'(t)$ 刚好是题设条件的形式，即 $ \p'(t) \geq \a$，故当然有 $\lim\limits_{t\to+\infty} \p(t) = +\infty$
- **曲线导数的单调性**：
  - 设
    - $\G$ 是可微曲线
    - $f(x,y)$ 在包含 $\G$ 的区域上连续可微，在 $\G$ 上恒为0
  - 则 $f$ 沿 $\G$ 的切向量的方向导数均为0
  - **证明**：计算即可
    - <font color='chartreuse'>设 $\G$ 的图像函数为 $y = g(x)，x\in [a,b]$
    - 计算可得 $f$ 方向导数为 $\cfrac{h'(x)}{\sqrt{1+(g')^2}}$，其中 $h(x) = f\Big( x,g(x) \Big) \equiv 0$</font>
- **向量值函数的单调性**：
  - 设 $D$ 是 $\R^2$ 中的凸区域，$T:D\to \R^2$ 是连续可导的映射
  - 若 $T$ 的Jacobi矩阵非退化，则 $T$ 为单射
  - **证明**：
    - 不妨设 $T(x,y) = (X,Y) = (f(x,y),g(x,y))$。反设 $T$ 不是单射，即存在两点 $P_1,P_2$ 的值相同，即 $f(x_1,y_1) = f(x_2,y_2)$，$g$ 同理
    - 设
      - 道路函数 $P(t) = (x_1+t\D x,y_1+t\D y)$
      - 道路复合函数 $W(t) = f(P(t))\D x + g(P(t))\D y$
      - 由连续可导性，$W$ 连续可导，再由 $W(0) = W(1)$，故由L中值定理，可得 $(\D x, \D y) J(P(\xi)) \tvec{\D x \\ \D y} = 0$
      - 由偏导数连续性，$|J|\neq 0$，则二次型非退化，仅零解，$P_1 = P_2$，矛盾
    - <font color='chartreuse'>本质还是二元L中值定理的证明</font>
  - **推论**：推广到 $\R^n$ 依然成立
