# 多元复合函数

## 偏导数链式法则

- **多元复合函数**：
  - 设
    - $g(u,v) = \Big( x(u,v)，y(u,v) \Big)$ 是 $2$ 元 $2$ 维向量值函数
    - $f(u,v)$ 是 $2$ 元函数  
  - 则 $f$ 和 $g$ 的复合函数为 $(f\circ g)(u,v) = f\Big( x(u,v),y(u,v) \Big)$

- **多元函数的链式法则**：
  - 若 $g$ 可导，$f$ 可微
    - **反例**：
      - <font color='chartreuse'>如果 $f$ 可偏导但不可微，则可能链式法则的结果为0，但实际导数不为 $0$（像Taylor展开一样无法借贷）</font>
    - **充分条件**：$f$ 存在连续的偏导数
      - <font color='chartreuse'>因为偏导数不连续但 $f$ 可微的情况只集中在一个点上，所以一般都使用存在连续的偏导数作为 $f$ 可微的条件替代</font>
  - 则 $f\circ g$ 可偏导，且偏导数为 $$f_u(u_0,v_0) = f_x(x_0,y_0)x_u(u_0,v_0) + f_y(x_0,y_0)y_u(u_0,v_0)$$

    - 仿照一元函数的链式法则，配凑 + 定义即可，
    - 由偏导数定义，首先将 $\dfrac{\partial z}{\partial u}(u_0,v_0)$ 写为 $$ \lim\limits_{\D u\to 0} \frac{f\Big( x(u_0+\D u,v_0),y(u_0+\D u,v_0) \Big) - f\Big( x(u_0,v_0),y(u_0,v_0) \Big)}{\D u}$$
    - 再化成 $x_0$ 形式：$$\lim\limits_{\D u\to 0} \frac{f(x_0+\D x,y_0+\D y) - f(x_0,y_0)}{\D x}\frac{\D x}{\D u}$$
    - 若两个极限都存在，则可配凑为 $$f_x(x_0,y_0)\lim\limits_{\D u\to 0}\frac{\D x}{\D u} + f_y(x_0,y_0)\lim\limits_{\D u\to 0}\frac{\D y}{\D u}$$
    - 我们证明后面的极限是存在的。为此，考虑 $x(u,v)$ 和 $y(u,v)$ 的全微分的误差量 $\alpha(\D x,\D y)$ 与 $\D u$ 的阶数关系 $$\frac{\alpha\sqrt{\D x^2+\D y^2}}{\D u} = \alpha * \frac{|\D u|}{\D u} * \sqrt{(\frac{\D x}{\D u})^2 + (\frac{\D y}{\D u})^2}  = \alpha*A \to 0$$

  - **理解**：
      - 由 $f$ 可微得到可以转化成 $x_0$ 形式
      - 由 $x,y$ 可偏导得到 $\frac{\D x}{\D u}、\frac{\D y}{\D u}$ 是有界量

### 极坐标下的偏导数

- 用极坐标偏导数表示代数偏导数（利于C.-R.方程的形式）$$\begin{cases} \dis\frac{\partial u}{\partial x} = \frac{\partial u}{\partial r}\frac{dr}{dx} + \frac{\partial u}{\partial \theta}\frac{d\theta}{dx} = \frac{\partial u}{\partial r}\cos\theta - \frac{\partial u}{\partial \theta}\frac{\cos\theta}{r} \\\\ \dis\frac{\partial u}{\partial y} = \frac{\partial u}{\partial r}\frac{dr}{dy} + \frac{\partial u}{\partial \theta}\frac{d\theta}{dy} = \frac{\partial u}{\partial r}\sin\theta + \frac{\partial u}{\partial \theta}\frac{\sin\theta}{r} \end{cases}$$
- 用代数偏导数表示极坐标偏导数（利于计算极坐标偏导数）$$\begin{cases} \frac{\partial u}{\partial r} = u_x\cos\theta + u_y\sin\theta \\\\ \frac{\partial u}{\partial \theta} = u_x(-r\sin\theta) + u_y(r\cos\theta) \end{cases} $$

## 一阶全微分的形式不变性

- **一阶微分的形式不变性**
  - 不管 $u$ 是中间变量还是自变量，对 $f$ 求一阶微分都有相同的形式
    - u是自变量：$df = f'(u)du + \omicron(du)$
    - u是中间变量：$df = f'(u)·u'(x)dx + \omicron(dx) = f'(u)du + \omicron(du)$
      - <font color='chartreuse'>因为复合函数的两个函数f、u都必须连续，f(u(x))才可导，所以$\omicron()$里面始终是阶数最高的无穷小</font>
- **一阶全微分的形式不变性**
  - 当 $x,y$ 是自变量时，$$df = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy + \omicron(\sqrt{dx^2+dy^2})$$
  - 当 $x,y$ 是自变量时，$$df = (\frac{\partial z}{\partial x}·\frac{dx}{du}du + \frac{\partial z}{\partial x}·\frac{dx}{dv}dv) + (\frac{\partial z}{\partial y}·\frac{dy}{du}du + \frac{\partial z}{\partial y}·\frac{dy}{dv}dv) + \omicron(\sqrt{du^2+dv^2})$$
    - 通分即得 $$df = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy + \omicron(\sqrt{dx^2+dy^2})$$
    - <font color='chartreuse'>实际上，前面链式法则的证明核心就是“可导等价于 $\omicron(\sqrt{(\D x)^2+(\D y)^2})$ $\sim \omicron(\D u)$，所以前者可以取代后者”</font>

- **齐次函数的Euler定理**：设 $f(x,y)$ 可微，则 $f$ 是 $k$ 次齐次函数 $\LR xf_x + yf_y = kf(x,y)$
  - **证明**：
    - **必要性**：对齐次函数定义式的 $t$ 求导，然后代入 $t=1$ 即可
      - <font color='chartreuse'>大二上第一次做的时候完全想不到。但是现在大四上一眼秒了。为什么呢？单纯就是对数学形式更加熟悉罢了，考察一个做题的熟练度</font>
    - **充分性**：
      - <font color='chartreuse'>考虑齐次函数的几何意义。高代里已经说明了，齐次方程的解空间图像过原点，同理易得齐次函数的图像也过 $(0,0,0)$。当然了，这只是齐次函数的性质之一，还不是它的等价条件
      - 另外，高代里仅仅讨论的是线性函数，在这里我们讨论的是可微函数，性质更差一些
      - hh，几何意义果然没那么简单，与其费劲去描述它，不如直接用数学形式来做</font>
      - 对 $g(t) = f(tx,ty)$ 关于 $t$ 求导，易得 $g'(t) = \dfrac{kg(t)}{t}$，解ODE得 $g(t) = Ct^k$
      - 即 $g(t) = t^kg(1)$，对任意 $x,y$ 成立，从而 $f(x,y)$ 是齐次函数
  - **推论**：推广到 $n$ 维空间，就是 $\forall \vec x_0\in\R^n，\vec x_0\cdot D f(\vec x_0) = kf(\vec x_0)$
- **齐次函数的微分性质**：设 $f(x,y)$ 是 $n$ 阶可微的 $k$ 次齐次函数，则 $$\Big[ x\pfrac{}{x} + y\pfrac{}{y} \Big]^n f = k(k-1)\cdots (k-n+1)f$$
  - **证明**：Euler定理 + 归纳法
    - 或对齐次函数的定义式 $n$ 次求导即可

## 微分中值定理

### 多元函数的微分中值定理

- **凸区域**：若区域是凸集，则称为凸区域
  - 设 $D \subset R^n$ 是区域
  - 若
    - 连接 $D$ 中任意两点的线段都完全属于 $D$
    - 对 $\forall x,y\in D，\l\in[0,1]$，都有 $x + \lambda(y-x)\in D$
  - 则称 $D$ 为凸区域
- **中值定理**：
  - 设二元函数 $f(x,y)$ 在凸区域 $D$ 上可微
  - 则 $f(x_0+\D x, y_0+\D y) - f(x_0,y_0) = f_x(x_0+\theta\D x, y_0+\theta\D y)\D x + f_y(x_0+\theta\D x, y_0+\theta\D y)\D y$
  - **证明**：
    - 凸区域保证仍然在D中
    - 可微保证两点之间距离可以用两个正交偏导完全表示
    - 然后制造t（$\lambda$）作为一元函数，就可以使用一维中值定理来证明二维
- **导数的增长率意义**：$f_x、f_y\equiv 0\red\Rightarrow f$ 为常函数
  - **证明**：
    - 一维空间中，是使用Lagrange中值定理，把极限的导数变成通常的中值。
    - **凸区域上的常值性**：由中值定理，函数值恒为 $ 0 \Rightarrow$ 偏导数值恒为 $0$
    - **道路上的常值性**：设 $\gamma(t)$ 是 $D$ 中道路
     - 设 $t_0 = \sup\set{s\in [0,1] \mid f(\gamma(t)) = f(\gamma(0)) = f(x_0,y_0), t\in [0,s] }$
       - 这句话中，既构造了一个集合$f(\gamma(t)) = f(x_0,y_0)$，又指出了它的上界。
     - 再证明：如果$t_0<1$，则由连续性（开区域上）和偏导数为零得$t_0+\D t$取代$t_0$成为新的上界，再根据$t_0$的任意性得到$t_0 = 1$，所以道路上的常值性得证
     - **本质**：一维空间存在上下界，因此可以利用全微分来微观求宏观

### 多元函数的泰勒公式

- **多元Taylor公式**：$$f(x_0 + \D x, y_0 + \D y) = \mathop{\sum}\limits_{k=1}^{n}\frac{f^{(k)}(x_0)}{k!}(x-x_0)^k + R_k$$
  - **第 $k$ 项**：$$\cfrac{\big[ (x-x_0)\frac{\partial}{\partial x}+(y-y_0)\frac{\partial}{\partial y} \big]^k}{k!}f(x_0,y_0)$$
  - **Lagrange余项**：$$R_k = \frac{\Big[(x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y}\Big]^{k+1}}{(k+1)!}f(x_0+\theta\D x,y_0+\theta\D y)$$
    <!-- - 跟一元函数的L中值一样，中值是导数中值而不是自变量微分中值 -->
  - **证明**：
    - 构造辅助函数 $\varPhi(t) = f(x_0+t\D x，y_0+t\D y) = f(u,v)$
    - 计算易得 $\varphi^{(k)}(t) = (\frac{\partial}{\partial x}dx + \frac{\partial}{\partial y}dy)^kf(x_0+t\D x,y_0+t\D y)$
      <!-- - （链式法则，先计算再代入，有$\partial u = \partial x$） -->
    - 对该一元函数在 $t=1$ 处进行泰勒展开，并构建L余项即可
  - **理解**：全微分公式表明，多元微分可以是一元微分的线性组合，从而跟多元中值定理一样，多元Taylor公式也可以构造一元的线性组合系数函数t（$\lambda$）
    - <font color='chartreuse'>还是和外微分的思想有很深的联系</font>

### 多元微分不等式

- 设 $f$ 在 $P_0$ 的邻域 $U$ 内存在三阶偏导数
  - 若
    - 偏导数绝对值均不超过 $M$
    - $P_1,P_2$ 是 $U$ 内两点，关于 $P_0$ 对称，距离为 $l$
  - 则 $$\biggm|\frac{f(P_1)-f(P_2)}{2l} - \pfrac{f(P_0)}{\or{P_0P_1}}\biggm| \leq \frac{\sqrt{2}}{3}Ml^2$$
