# Fourier级数性质

## 分析性质

- **有限项性**：若 $f(x)$ 可积，则 $\lim\limits_{n\to \infty}a_n/b_n = 0$
  - **证明**：Fourier系数的积分形式 + Riemann定理
- **逐项积分定理**：有限个第一类不连续点（不要求收敛性）
  - **证明**：构造 $F(x) = \int^x_c\big[f(t) - \frac{a_0}{2}\big]dt$
    - 周期为 $2\pi$，存在左右导数，所以可以展开成收敛Fourier级数
    - F级数系数分部积分（积cosnx导F(x)），由三角函数积分循环性，得到 $\begin{cases} A_n = -\frac{b_n}{n} \\ B_n = \frac{a_n}{n} \end{cases}$
    - 因此得到 $F(x)、F(c)=0$ 的展开式，相减变化得到 $F(x) = \sum\limits^\infty_{n=1}\int^x_c(a_ncos\ nt + b_nsin\ ntdt)$
- **Fourier级数必要条件**：$\sum\limits^\infty_{n=1}\frac{b_n}{n}$ 收敛
  - **证明**：
    - 首先根据积分性质，即使Fourier级数不收敛，其积分级数也收敛。所以要保证积分级数的收敛性
  - 然后 $F(0)$ 需要收敛，**证毕**
- **逐项微分定理**：$f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，不可导点为有限个，且 $f'(x)$ 可积或绝对可积
  - **证明**：f'级数系数分部积分，由三角函数积分循环性，得到
    - $a_0' = 0$，$a'_n = nb_n$，$b'_n = -na_n$
    - $f'(x)\sim \sum\limits^\infty_{n=1} (-a_nnsinnx + b_nncosnx)$

## 逼近性质

- S是线性空间，定义内积运算，赋范模。
- T是n维子空间，一组正交基为 $T = span\{\varphi_1,\varphi_2,...,\varphi_n\}$
- **最佳平方逼近元素**：$\|x-x_T\| = \mathop{min}\limits_{y\in T} \|x-y\|$。（$x_T$ 是 $x$ 在 $T$ 中的投影向量）
  - **唯一存在性**：
    - **证明**：
      - $\|x-y\|^2$ 求最小值：y分解成正交基，再设出 $x$ 的投影和基 $\varphi_k$ 的模比例 $c_k = \frac{(x,\varphi_k)}{(\varphi_k,\varphi_k)}$，配凑平方可得 $\|x\|^2 - \sum\limits^n_{k=1}c_k^2 + \sum\limits^n_{k=1} (c_k-d_k)^2\|\varphi_k\|^2$
      - **存在估计式**：$\|x-x_T\|^2 = \|x\|^2 - \|x_T\|^2 = \|x\|^2 - \sum\limits^n_{k=1} c^2_k \|\varphi_k\|^2$
  - **充要条件**：$(x-x_T)\perp T$
    - **证明**：$(x-x_T,\varphi_k) = 0$
- **Fourier级数空间内积**：$(f,g) = \frac{1}{\pi} \int^\pi_{-\pi} f(x)g(x)dx$
  - **Fourier系数**：$a_k = (f,coskx)$，$b_k = (f,sinkx)$
  - **经典模**：$\|1\|^2 = 2$、$\|coskx\|^2 = \|sinkx\|^2 = 1$
- **三角级数空间**
  - T为n阶三角多项式（有限项三角级数）全体的集合
  - **三角正交基**：$T = span\{1,cosx,sinx,cos2x,...,sinnx\}$
- **平方逼近性质**：
  - 最佳平方逼近元素为部分和函数 $S_n(x)$
  - 余项为 $\|f-S_n\|^2 = \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx - \big[\frac{a_0}{2} + \sum\limits^\infty_{k=1}(a_k^2 + b_k^2)\big]$
  - **证明**：
    - $\|1\|^2 = 2$、$\|coskx\|^2 = \|sinkx\|^2 = 1$
    - 积分内展开乘积，提取常数，利用经典模即可
- **Bessel不等式（范数非负性）**：
  - 若 $f(x)$ 在 $[-\pi,\pi]$ 上可积或平方可积，则 $\frac{a_0^2}{2} + \sum\limits^{\infty}_{k=1} (a_k^2+b_k^2) \leq \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx$
  - 
- **Parseval恒等式**：
  - 若 $f(x)$ 在 $[-\pi,\pi]$ 上可积或平方可积，则 $\frac{a_0^2}{2} + \sum\limits^{\infty}_{k=1} (a_k^2+b_k^2) = \frac{1}{\pi}\int^\pi_{-\pi}f^2(x)dx$
  - 
- **平方收敛**：$\lim\limits_{n\to \infty} \|f(x)-\psi_n(x)\|^2 = 0$，则称 $\{\psi_n(x)\}$ 按范数 $\|\cdot\|$ 平方收敛于 $f(x)$
  - **平方收敛性**：可积或平方可积的函数 $f(x)$ ，其Fourier部分和数列平方收敛于自身
  - **Weierstrass第二逼近定理**：周期为 $2\pi$ 的连续函数 $f(x)$，存在三角多项式序列 $\{\psi(x) = \frac{A_0}{2} + \sum\limits^n_{k=1}(A_kcoskx + B_ksinkx)\}$ 一致收敛

## 等周问题

- **等周问题**：周长相等的闭曲线图形面积最大的是圆周
- **Wirtinger不等式**：
  - 若
    - $f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，$\int^\pi_{-\pi} f(x)dx = 0$
    - 除了有限个点均可导，不可导点的单侧导数存在
    - 导数可积或平方可积
  - 则
    - $\int^\pi_{-\pi} f^2(x)dx \leq \int^\_{-\pi}f'^2(x)dx$。等号成立时 $f(x) = acosx+bsinx$
  - **证明**：
    - 首先 $f(x)$ 可以Fourier展开，且点态收敛。$a_0=0$
    - 然后其导数可以Fourier展开
    - 由Parseval等式，得到它们的模
    - 模作差得到 $\pi\sum\limits^\infty_{k=2}(n^2-1)(a_k^2+b_k^2) \geq 0$，$n>1$ 时 $a_n、b_n=0$ 则等号成立
- **等周定理**：$L$ 是长度，$A$ 是面积，则 $A\leq \frac{L^2}{4\pi}$，等号成立时是圆周
  - **证明**：
    - 首先作弧长的参数方程 $\begin{cases} x=x(s)\\y=y(s) \end{cases}$，且首尾相等
    - 变量代换 $s=\frac{L}{2\pi}t+\frac{L}{2}\ （t\in [-\pi,\pi]）$
    - 因为是封闭曲线，所以 $\int^\pi_{-\pi}x(t)dt = 0$（闭曲线重心是原点）
    - 弧长微分公式 $(\frac{ds}{dt})^2 = \frac{L^2}{4\pi^2} = x'^2(t)+y'^2(t)$
    - 取定积分
    - 面积积分公式 $A = \int_C xdy = \int^\pi_{-\pi}\varphi(t)\psi'(t)dt$
    - $\frac{L^2}{2\pi}-2A$，配凑得到Wirtinger不等式和平方积分，得到 $\geq 0$，因而得到不等关系
