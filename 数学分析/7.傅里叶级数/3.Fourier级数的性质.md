# Fourier级数的性质

## 分析性质

- **有限项性**：若 $f(x)$ 可积，则 $\lim\limits_{n\to \infty}a_n = \lim\limits_{n\to\infty} b_n = 0$
  - **证明**：E-F公式 + Riemann定理即可
- **上限函数收敛定理**：若 $f$ 可积，则其上限函数 $F$ 的三角级数收敛
  - **证明**：
    - 设 $F(x) = \dis\int^x_c\big[f(t) - \frac{a_0}{2}\big]dt$
    - 易得其周期为 $2\pi$，存在左右导数，故可展开为三角级数
  - **推论**：对已知的三角级数，其收敛的必要条件是 $\sum\limits^\infty_{n=1}\dfrac{b_n}{n}$ 收敛
    - **证明**：由 $F(x)$ 的三角级数收敛性，取 $x=0$ 易得结论
- **逐项积分定理**：
  - 若 $f(x)$ 可积或绝对可积，且三角级数收敛，则其三角级数可以逐项积分
  - **证明（$f$ 仅有限个第一类不连续点）**：
    - <font color='chartreuse'>此时上限函数 $F$ 仅在有限个点上不可微，故可分部积分</font>
    - 对其上限函数 $F$ 的傅立叶系数分部积分（积 $\cos nx$，导 $F(x)$），则由三角函数的积分循环性，得到 $\begin{cases} A_n = -\dfrac{b_n}{n} \\ B_n = \dfrac{a_n}{n} \end{cases}$
    - 写出 $F(x)、F(c)=0$ 的展开式，相减后变形即可得到逐项积分公式 $$F(x) = \sum\limits^\infty_{n=1}\int^x_c(a_n\cos nt + b_n\sin nt)dt$$
- **导函数收敛定理**：若 $f$ 连续，分段可导，则导函数的三角级数收敛
  - <font color='red'>在黎曼积分的意义下探讨这个太折磨了，还是详见傅里叶分析专题吧</font>
- **逐项微分定理**：
  - 设 $f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，分段可导
    - <font color='chartreuse'>易得此时 $f'(x)$ 可积或绝对可积</font>
  - 则其三角级数可以逐项微分
  - **证明**：
    - 对 $f'$ 的三角级数的系数分部积分（积 $f'$ 导 $\cos nx$），由三角函数积分循环性得 $a_0' = 0$，$a'_n = nb_n$，$b'_n = -na_n$
    - 故得到逐项微分公式 $$f'(x) = \sum\limits^\infty_{n=1} (-a_nn \sin nx + b_nn\cos nx)$$

## 逼近性质

### 周期可积空间

- <font color='chartreuse'>讲道理，这里其实就是泛函分析的东西了</font>
- **周期可积空间**：
  - 设 $S$ 是 $[-\pi,\pi]$ 上可积或平方可积的全体函数，易得其为线性空间，可定义内积运算，赋范模
  - 设 $T$ 是其中的 $n$ 维子空间，一组正交基为 $\{\p_1,\p_2,...,\p_n\}$
- **最佳平方逼近元素**：若 $\|x-x_T\| = \min\limits_{y\in T} \|x-y\|$，则 $x_T$ 称为 $x$ 的最佳逼近
  - **存在性和唯一性**：
    - **证明**：
      - <font color='chartreuse'>直接对 $\|x-y\|^2$ 求最小值即可</font>
      - 先将 $y$ 分解成正交基的线性组合，系数设为 $d_k$
      - 再设出 $x$ 的投影和基 $\p_k$ 的模比例 $c_k = \cfrac{(x,\p_k)}{(\p_k,\p_k)}$
      - 利用内积的双线性 + 基的正交性易得 $$\|x-y\|^2 = \|x\|^2 - \sum\limits^n_{k=1}c_k^2\|\p_k\|^2 + \sum\limits^n_{k=1} (c_k-d_k)^2\|\p_k\|^2$$
        - 也就是说，当且仅当 $c_k = d_k$ 时为最小值
  - **余项估计式**：$\|x-x_T\|^2 = \|x\|^2 - \|x_T\|^2 = \|x\|^2 - \sum\limits^n_{k=1} c^2_k \|\p_k\|^2$
    - **证明**：令 $c_k = d_k$，代入上式即可
  - **几何意义**：$x_T$ 是 $x$ 在 $T$ 中的投影向量，即 $(x-x_T)\perp T$
    - **证明**：
      - **充分性**：代入 $d_k = c_k$ 易得 $(x-x_T,\p_k) = 0$
      - **必要性**：若 $(x-y,\p_k) = 0$，则双线性分解易得 $d_k = c_k$
      - <font color='chartreuse'>学过泛函分析的话，这些都是非常基础的手法</font>
- **周期可积空间的内积**：$\dis(f,g) = \frac{1}{\pi} \int^\pi_{-\pi} f(x)g(x)dx$
  - **Fourier系数**：$a_k = (f,\cos kx)$，$b_k = (f,\sin kx)$
  - **常用函数的模**：$\|1\|^2 = 2$、$\|\cos kx\|^2 = \|\sin kx\|^2 = 1$

### 三角级数（子）空间

- **三角级数空间**：设 $T$ 为 $n$ 阶三角多项式（有限项三角级数）全体的集合，易得其为线性空间
  - **三角正交基**：$T = \span\{1,\cos x,\sin x,\cos 2x,...,\sin nx\}$
- **傅立叶最佳平方逼近定理**：
  - 周期可积空间中元素 $f$ 在 $T$ 中的最佳平方逼近元素为三角级数的部分和函数 $S_n(x)$
  - 余项为 $\dis\|f-S_n\|^2 = \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx - \Big[\frac{a_0^2}{2} + \sum\limits^n_{k=1}(a_k^2 + b_k^2)\Big]$
    - **证明**：
      - 最佳逼近：作内积，发现垂直
      - 余项计算：易得 $\|1\|^2 = 2$、$\|\cos kx\|^2 = \|\sin kx\|^2 = 1$，利用内积双线性展开即可
- **Bessel不等式（范数非负性）**：
  - 若 $f(x)$ 在 $[-\pi,\pi]$ 上可积或平方可积，则 $\dis\frac{a_0^2}{2} + \sum\limits^n_{k=1} (a_k^2+b_k^2) \leq \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx$
    - <font color='chartreuse'>B不等式是有穷情况的结论，P恒等式是无穷情况的结论</font>
  - **证明**：
    - 已知 $(f-S_n)\perp S_n$，故由内积范数得 $\|S_n\| + \|f-S_n\| = \|f\|$，再由 $\|f-S_n\| \geq 0$ 即得结论
  - **推论**：
    - $\sum\limits^\infty_{n=1} |a_nb_n| \leq \sum\limits^\infty_{n=1} \dfrac{a_n^2+b_n^2}{2}$，故绝对收敛
- **Parseval恒等式（勾股定理）**：
  - 若 $f(x)$ 在 $[-\pi,\pi]$ 上可积或平方可积，则 $\dis\frac{a_0^2}{2} + \sum\limits^{\infty}_{k=1} (a_k^2+b_k^2) = \frac{1}{\pi}\int^\pi_{-\pi}f^2(x)dx$
  - **证明（分析法）**：
    - 由收敛判别法，此时 $\lim\limits_{n\to\infty} a_n = \lim\limits_{n\to\infty} b_n = 0$，故 $\|f-S_n\|^2 \to 0$，从而成立等号
  <!-- - **证明（互包法）**：
    - 三角不等式得 $\|f\| \leq \|f-S_n\|+\|S_n\| \to \lim\limits_{n\to\infty}\|S_n\|$
    - 再由贝塞尔不等式即得结论 -->
  - **证明（代数法）**：
    - 由内积双线性得 $(f,f) = (f-S_n,f-S_n) + 2(f,S_n) - (S_n,S_n)$
      - 第一项为 $0$
      - 由最佳逼近的正交性得 $(f,S_n) = (S_n,S_n)$，故得到等式
  - <font color='chartreuse'>不难发现它就是函数空间的勾股定理</font>
- **平方收敛**：若 $\lim\limits_{n\to \infty} \|f(x)-\psi_n(x)\|^2 = 0$，则称 $\{\psi_n(x)\}$ 按范数 $\|\cdot\|$ 平方收敛于 $f(x)$
  - **平方收敛性**：可积或平方可积的函数 $f(x)$ ，其三角部分和数列平方收敛于自身
    - **证明**：
      - 定义式 + Parseval恒等式即可
  - <font color='red'>傅立叶级数的逐点收敛性很难判别，但依范数平方收敛性却非常容易满足</font>
- **依范数收敛**：若 $\lim\limits_{n\to \infty} \|f(x)-\psi_n(x)\| = 0$，则称 $\{\psi_n(x)\}$ 按范数 $\|\cdot\|$ 收敛于 $f(x)$
  - **Weierstrass第二逼近定理**：对任意周期为 $2\pi$ 的连续函数 $f(x)$，都存在三角多项式序列 $\hkh{\psi(x) = \dfrac{A_0}{2} + \sum\limits^n_{k=1}(A_k\cos kx + B_k\sin kx)}$ 依范数收敛到 $f$
    - **证明**：详见数值分析

### 一致收敛性

- **一致收敛判别（可积情况）**：
  - 设 $f(x)$ 在 $[-\pi,\pi]$ 上可导，且导函数可积
  - 则 $f$ 的三角级数绝对收敛，且一致收敛到 $f$
  - **证明**：
    - 由可导性易得三角级数的点态收敛性
    - 由逐项可导定理 + Bessel不等式，级数 $\sum\limits^\infty_{n=1} (n^2a_n^2 + n^2b_n^2)$ 收敛
    - 再由放缩 + 均值不等式，$|a_n\cos nx + b_n\sin nx| \leq \dfrac{1}{n}|na_n| + \dfrac{1}{n}|nb_n| \leq \dfrac{1}{n^2} + \dfrac{1}{2}(n^2a_n^2 + n^2b_n^2)$
    - 比较判别法即得绝对收敛，W判别法即得一致收敛
- **一致收敛的非连续反例**：
  - 设 $f(x)$ 在 $[-\pi,\pi]$ 上连续，且 $f(\pi) = f(-\pi)$
  - 若 $f$ 的三角级数在 $[-\pi,\pi]$ 上收敛，则必然收敛于 $f$
  - **证明**：
    - 由前面收敛判别法 + 连续性易得结论
  - **推论（否命题）**：
    - 设 $f$ 可积，三角级数一致收敛
    - 若不连续，或 $f(-\pi)\neq f(\pi)$，则三角级数不一致收敛到 $f$

## 等周问题

- **等周问题**：周长相等的闭曲线图形面积最大的是圆周
- **Wirtinger不等式**：
  - 若
    - $f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，$\int^\pi_{-\pi} f(x)dx = 0$
    - $f$ 分段光滑
    - <font color='chartreuse'>易得 $f'(x)$ 可积或平方可积</font>
  - 则
    - $\dis\int^\pi_{-\pi} f(x)^2 dx \leq \int^{\pi}_{-\pi}f'(x)^2dx$。等号成立时 $f(x) = a\cos x+b\sin x$
  - **证明**：
    - 首先易得 $f(x)$ 的三角级数点态收敛，且 $a_0=0$
    - 再易得 $f'(x)$ 的三角级数点态收敛，且可以逐项微分
    - 两个Parseval等式作差得到 $\pi\sum\limits^\infty_{k=2}(n^2-1)(a_k^2+b_k^2) \geq 0$（**证毕**）
      - <font color='chartreuse'>$n>1$ 时等号成立条件为 $a_n、b_n=0$</font>
- **等周定理**：
  - 设 $\G$ 是 $\R^2$ 上分段光滑的简单闭曲线，$L$ 是长度，$A$ 是面积，则 $A\leq \dfrac{L^2}{4\pi}$，等号成立时是圆周
  - **证明**：
    - 首先作弧长的参数方程 $\begin{cases} x=x(s)\\y=y(s) \end{cases}$，其中 $s\in [0,L]$
      - <font color='chartreuse'>由封闭性得其首尾相等</font>
      - 变量代换 $ s=\dfrac{L}{2\pi}t+\dfrac{L}{2}$，其中 $t\in [-\pi,\pi]$
        - <font color='chartreuse'>为了将定义域改为 $[-\pi,\pi]$，从而应用定理</font>
      <!-- - 由封闭性得 $\dis\int^\pi_{-\pi}x(t)dt = \int^\pi_{-\pi}y(t)dt = 0$
        - <font color='chartreuse'>换元相当于对 $\G$ 进行平移，重心变为原点</font> -->
      - 由弧长微分公式得 $\dis(\frac{ds}{dt})^2 = \frac{L^2}{4\pi^2} = x'^2(t)+y'^2(t)$
      - 取定积分即得 $$ \frac{L^2}{2\pi} = \int^{\pi}_{-\pi} \Big[ x'(t)^2 + y'(t)^2 \Big] dt$$
    - 由面积积分公式得 $$\dis A = \int_C xdy = \int^\pi_{-\pi}x(t)y'(t)dt$$
    - 综上，变形易得 $$\frac{L^2}{2\pi}-2A = \int^{\pi}_{-\pi} \Big[ x'(t)^2-x(t)^2 \Big]dt +  \int^{\pi}_{-\pi} \Big[ x'(t)-y'(t) \Big]^2 dt $$
      - 前项是Wirtinger不等式，非负
      - 后项是平方积分，非负
      - 从而得到所需不等式
