# Fourier级数性质

## 分析性质

- **有限项性**：若 $f(x)$ 可积，则 $\lim\limits_{n\to \infty}a_n = \lim\limits_{n\to\infty} b_n = 0$
  - **证明**：Fourier系数的积分形式 + Riemann定理
- **一致收敛定理（连续情况）**：设 $f(x)$ 在 $[-\pi,\pi]$ 上连续
  <!-- - 以 $2\pi$ 为周期，即 $f(-\pi) = f(\pi)$ -->
  - 若 $f$ 的三角级数在 $[-\pi,\pi]$ 上一致收敛，则必然收敛于 $f$
  - **证明**：
  - **推论（否命题）**：
    - 设 $f$ 可积，三角级数一致收敛
    - 若不连续，或 $f(-\pi)\neq f(\pi)$，则三角级数不一致收敛到 $f$
- **逐项积分定理**：若 $f(x)$ 可积或绝对可积，且三角级数收敛，则其三角级数可以逐项积分
  - **证明（$f(x)$ 只有有限个第一类不连续点）**：
    - 构造 $F(x) = \dis\int^x_c\big[f(t) - \frac{a_0}{2}\big]dt$
      - 易得其周期为 $2\pi$，存在左右导数，故可展开为三角级数
    - 对其三角级数的系数分部积分（积 $\cos nx$，导 $F(x)$），则由三角函数的积分循环性，得到 $\begin{cases} A_n = -\dfrac{b_n}{n} \\ B_n = \dfrac{a_n}{n} \end{cases}$
    - 写出 $F(x)、F(c)=0$ 的展开式，相减后变形即可得到逐项积分公式 $$F(x) = \sum\limits^\infty_{n=1}\int^x_c(a_n\cos nt + b_n\sin nt)dt$$
- **收敛必要条件**：对已知的三角级数，其收敛的必要条件是 $\sum\limits^\infty_{n=1}\dfrac{b_n}{n}$ 收敛
  - **证明**：
    - 首先根据积分性质，即使Fourier级数不收敛，其积分级数也收敛。所以要保证积分级数的收敛性
  - 然后 $F(0)$ 需要收敛，**证毕**
- **逐项微分定理**：
  - 设 $f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，不可导点仅为有限个
    - <font color='chartreuse'>易得此时 $f'(x)$ 可积或绝对可积</font>
  - 则其三角级数可以逐项微分
  - **证明**：
    - 对 $f'$ 的三角级数的系数分部积分（积 $f'$ 导 $\cos nx$），由三角函数积分循环性得 $a_0' = 0$，$a'_n = nb_n$，$b'_n = -na_n$
    - 故得到逐项微分公式 $$f'(x) = \sum\limits^\infty_{n=1} (-a_nn \sin nx + b_nn\cos nx)$$

## 逼近性质

### Riemann可积空间

- <font color='chartreuse'>讲道理，这里其实就是泛函分析的东西了</font>
- **Riemann可积空间**：下面设 $S$ 是（$\R$ 上可积或平方可积的全体函数），易得其为线性空间，可定义内积运算，赋范模
  - $T$ 是其中的 $n$ 维子空间，一组正交基为 $\{\varphi_1,\varphi_2,...,\varphi_n\}$
- **最佳平方逼近元素**：若 $\|x-x_T\| = \min\limits_{y\in T} \|x-y\|$，则 $x_T$ 称为 $x$ 的最佳逼近
  - **存在性和唯一性**：
    - **证明**：只需要对 $\|x-y\|^2$ 求最小值即可
      - 先将 $y$ 分解成正交基的线性组合，系数设为 $d_k$
      - 再设出 $x$ 的投影和基 $\varphi_k$ 的模比例 $c_k = \cfrac{(x,\varphi_k)}{(\varphi_k,\varphi_k)}$
      - 利用内积的双线性易得 $$\|x-y\|^2 = \|x\|^2 - \sum\limits^n_{k=1}c_k^2\|\p_k\|^2 + \sum\limits^n_{k=1} (c_k-d_k)^2\|\varphi_k\|^2$$
        - 也就是说，当且仅当 $c_k = d_k$ 时为最小值
  - **余项估计式**：$\|x-x_T\|^2 = \|x\|^2 - \|x_T\|^2 = \|x\|^2 - \sum\limits^n_{k=1} c^2_k \|\varphi_k\|^2$
    - **证明**：令 $c_k = d_k$，代入上式即可
  - **几何意义**：$x_T$ 是 $x$ 在 $T$ 中的投影向量，即 $(x-x_T)\perp T$
    - **证明**：
      - **充分性**：代入 $d_k = c_k$ 易得 $(x-x_T,\varphi_k) = 0$
      - **必要性**：若 $(x-y,\p_k) = 0$，则双线性分解易得 $d_k = c_k$
      - <font color='chartreuse'>学过泛函分析的话，这些都是非常基础的手法</font>
- **Riemann可积空间内积**：$\dis(f,g) = \frac{1}{\pi} \int^\pi_{-\pi} f(x)g(x)dx$
  - **Fourier系数**：$a_k = (f,\cos kx)$，$b_k = (f,\sin kx)$
  - **常用函数的模**：$\|1\|^2 = 2$、$\|\cos kx\|^2 = \|\sin kx\|^2 = 1$

### 三角级数（子）空间

- **三角级数空间**：设 $T$ 为 $n$ 阶三角多项式（有限项三角级数）全体的集合，易得其为线性空间
  - **三角正交基**：$T = span\{1,\cos x,\sin x,\cos 2x,...,\sin nx\}$
- **平方逼近性质**：
  - Riemann可积空间中元素 $f$ 在 $T$ 中的最佳平方逼近元素为三角级数的部分和函数 $S_n(x)$
  - 余项为 $\dis\|f-S_n\|^2 = \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx - \Big[\frac{a_0}{2} + \sum\limits^n_{k=1}(a_k^2 + b_k^2)\Big]$
  - **证明**：
    - $\|1\|^2 = 2$、$\|coskx\|^2 = \|sinkx\|^2 = 1$
    - 积分内展开乘积，提取常数，利用经典模即可
- **Bessel不等式（范数非负性）**：
  - 若 $f(x)$ 在 $[-\pi,\pi]$ 上可积或平方可积，则 $\dis\frac{a_0^2}{2} + \sum\limits^{\infty}_{k=1} (a_k^2+b_k^2) \leq \frac{1}{\pi}\int^\pi_{-\pi} f^2(x)dx$
  - **证明**：
    - 由上面的平方逼近性质，只需证明 $n\to\infty$ 的情况即可
  - **推论**：
    - $\sum\limits^\infty_{n=1} |a_nb_n| \leq \sum\limits^\infty_{n=1} \dfrac{a_n^2+b_n^2}{2}$，故绝对收敛

## 等周问题

- **等周问题**：周长相等的闭曲线图形面积最大的是圆周
- **Wirtinger不等式**：
  - 若
    - $f(x)$ 在 $[-\pi,\pi]$ 上连续，$f(-\pi) = f(\pi)$，$\int^\pi_{-\pi} f(x)dx = 0$
    - 除了有限个点均可导，不可导点的单侧导数存在
    - <font color='chartreuse'>易得 $f'(x)$ 可积或平方可积</font>
  - 则
    - $\dis\int^\pi_{-\pi} f^2(x)dx \leq \int^{\pi}_{-\pi}f'^2(x)dx$。等号成立时 $f(x) = a\cos x+b\sin x$
  - **证明**：
    - 首先易得 $f(x)$ 的三角级数点态收敛，且 $a_0=0$
    - 再易得 $f'(x)$ 的三角级数点态收敛，且可以逐项微分
    - 两个Parseval等式作差得到 $\pi\sum\limits^\infty_{k=2}(n^2-1)(a_k^2+b_k^2) \geq 0$（**证毕**）
      - <font color='chartreuse'>$n>1$ 时等号成立条件为 $a_n、b_n=0$</font>
- **等周定理**：$L$ 是长度，$A$ 是面积，则 $A\leq \frac{L^2}{4\pi}$，等号成立时是圆周
  - **证明**：
    - 首先作弧长的参数方程 $\begin{cases} x=x(s)\\y=y(s) \end{cases}$，且首尾相等
    - 变量代换 $s=\frac{L}{2\pi}t+\frac{L}{2}\ （t\in [-\pi,\pi]）$
    - 因为是封闭曲线，所以 $\int^\pi_{-\pi}x(t)dt = 0$（闭曲线重心是原点）
    - 弧长微分公式 $(\frac{ds}{dt})^2 = \frac{L^2}{4\pi^2} = x'^2(t)+y'^2(t)$
    - 取定积分
    - 面积积分公式 $A = \int_C xdy = \int^\pi_{-\pi}\varphi(t)\psi'(t)dt$
    - $\frac{L^2}{2\pi}-2A$，配凑得到Wirtinger不等式和平方积分，得到 $\geq 0$，因而得到不等关系
