# Fourier收敛判别法

- Fourier级数形式统一，可以融合成为一个函数
- **赋权振荡性**：
- 常用函数：
  - $\varphi(u) = \varphi_\sigma(u,x)$（Fourier级数作差 $f-g$）
  - $\psi(u) = \varphi_\sigma(u,x)\cdot \frac{1}{u\ 或\ 2sin\frac{u}{2}}$
  - $D(u) = \frac{f(x\pm u)-f(x\pm)}{u}sinpu$

## 级数融合函数

### 形式变换

- **$sin+$ 形式**：$S_m(x) = \frac{1}{\pi}\int^\pi_{-\pi}f(t)\cdot \big[\frac{1}{2} + \sum\limits^m_{n=1}cosn(t-x)\big]dt$
  - 因为Fourier系数的积分形式一致，可以把 $\int^\pi_{-\pi}f(x)cosnxdx$ 线性合并。
  - 
- **$sinf$ 形式**：$S_m(x) = \frac{1}{\pi}\int^\pi_{-\pi} f(x+u)\Large\frac{sin\frac{2m+1}{2}u}{2sin\frac{u}{2}}du$
  - 积化和差 + 代换 $t-x=u$（周期函数的区间等价性）
  - （公式可用复变函数证明）
- **Dirichlet积分形式**：$\large S_m(x) = \frac{1}{\pi}\int^\pi_0\big[f(x+u)+f(x-u)\big]\frac{sin\frac{2m+1}{2}u}{2sin\frac{u}{2}}du$
  - 这是区间分离后的常见形式
- **$t/u$ 的意义**：
  - t表示积分变量，x表示当前点变量
  - 原本x在积分外，是独立于t的。
  - 进入积分后，x对于t也相当于一个系数
  - 而t是定积分，并不能对其求导和求积（在外界看来，t不存在）
    - u也是定积分，不可被求导和求积

### 收敛原理

- **收敛判别原理（作差形式）**：
  - Fourier级数收敛 $\LR \lim\limits_{m\to\infty}\int^\pi_0\varphi_\sigma(u,x)\frac{sin\frac{2m+1}{2}u}{2sin\frac{u}{2}}du = 0$
  - $\varphi_\sigma(u,x) = f(x+u) + f(x-u) - 2\sigma(x)$
- **证明**：（利用 $sin+$ 形式）
  - **积分化一**：$\frac{2}{\pi}\int^\pi_0(\frac{1}{2} + \sum\limits^m_{n=1}cosnu)du = 1$
  - **积分并入**：$S_m(x)-\sigma(x) = \frac{1}{\pi}\int^\pi_0 \cdot\ \varphi_\sigma(u,x)\cdot\ \frac{sin\frac{2m+1}{2}u}{2sin\frac{u}{2}}du$

## 赋权振荡函数

### 可积归零性

- **Riemann引理**：函数 $\psi(x)$ 在 $\big[a,b\big]$ 上可积或绝对可积，则 $\lim\limits_{p\to +\infty} \int^b_a \psi(x)sinpxdx = \lim\limits_{p\to +\infty} \int^b_a \psi(x)cospxdx = 0$
- **证明**：
  - 有界情况：
    - 首先取划分 $\{x_i\}^n_{i=1}$，每个划分中 $\int\psi(x)dx$ 下确界为 $m_i$，振幅为 $w_i$
    - 设 $p$ 满足 $\frac{2}{p}(\sum\limits^n_{i=1}|m_i|) < \frac{\e}{2}$
    - 因此 $|\int^b_a \psi(x)sinpxdx|$
      - 分离成各个划分区间
      - 添项 $m_i$，分离出 $\int w_i\cdot sinpx$ 和 $m_i\int sinpx$
      - 放缩，$sinpx<1$ 和 $\frac{1}{p}\int^{x_i}_{x_{i-1}}sinpx\ d(px) < \frac{2}{p}$，得到 $<\e$
  - 无界情况（奇点）：
    - $\psi(x)$ 绝对可积，奇点无界区间内 $\int^{b}_{b-\eta}|\psi(x)|dx < \e$
  - **理解**：对偶添项分离法
    - $\int sinpx$ 的极小性 + $m_i$ 的有界性
    - $sinpx$ 的有界性 + $\int w_i$ 的极小性
  - **本质**：$\varphi(x)$ 是振荡权值函数，$sinpx$ 是振荡周期函数。振荡周期极小，权值相对连续，则相互抵消为0

### 非邻域的可积性（无奇点性）

- **局部性原理**：Fourier级数在x点的收敛性只与 $(x-\delta,x+\delta)$ 的性质有关
  - **证明（sinf形式）**：
    - 由 $f$ 可积性 + 可积的运算传递性 + 区间内无奇点，$\psi(x,u) = \frac{f(x+u)+f(x-u)}{2sin\frac{u}{2}}$ 关于 $u$ 在 $\big[\delta,\pi\big]$ 可积或绝对可积
    - 由Riemann引理，$m\to +\infty$ 时 $\int^\pi_\delta \psi(x,u)sin\frac{2m+1}{2}u\ du$ 为0
    - 因此前方积分归零，收敛性只与 $u\in \big[0,\delta\big]$ 上的积分有关，即只与 $x\in(x-\delta,x+\delta)$ 上的积分有关。**（证毕）**
  - **理解**：局部区域对于 $\frac{\triangle}{2sin\frac{u}{2}}$ 是奇点，而非奇点区域可积，可归零
- **奇点等价变换**：若 $\varphi(u)$ 在 $\big[0,\delta\big]$ 上可积或绝对可积，则 $\lim\limits_{m\to \infty}\int^\delta_0 \varphi(u)\frac{sin\frac{2m+1}{2}u}{2sin\frac{u}{2}}du = \lim\limits_{m\to \infty}\int^\delta_0 \varphi(u)\frac{sin\frac{2m+1}{2}u}{u}du$
  - **证明**：
    - 作系数差函数 $g(u) = \begin{cases} \frac{1}{2sin\frac{u}{2}} - \frac{1}{u}，u>0 \\ 0，\ \quad u=0 \end{cases}$
    - 作差 $\int^\delta_0 \psi(u)g(u)sin\frac{2m+1}{2}u\ du = 0$**（证毕）**
  - **本质**：只要保持奇点性质，$sinu$ 还是 $u$ 没有区别

### 奇点可积条件

- **Dini条件**：$\frac{\varphi_\sigma(u,x)}{u}$ 关于 $u$ 在 $\big[0,\delta\big]$ 上可积或绝对可积
  - **证明**：$\varphi_\sigma$ 作差性 + Riemann引理 + 奇点等价变换
  - **必要条件（弱化条件）**：第一类不连续点上，必有 $\lim\limits_{u\to 0}\varphi_\sigma(u,x)\to 0$，才能保证Dini条件成立
    - **证明**：如果是有界量，则 $\int^\delta_0 \frac{A}{u}du = -\infty$，$\varphi_\sigma(u,x)$ 不可积。所以至少是无穷小量
    - **推论**：$\varphi_\sigma(x)=0 \longrightarrow \sigma(x) = \frac{f(x+)+f(x-)}{2}$
- **理解**：奇点区域可积，则阶数必须一致（类似可导）
- **本质**：必要条件过强，对原函数限制过大，从而引发反推
- **应用**：Dini条件是对u方面的限制
- **点态收敛判别定理**：若在 $x_0$ 处满足Dini条件，则 $S_m(x_0)$ 收敛于 $\sigma(x_0) = \begin{cases} f(x) \qquad\quad 连续点 \\ \frac{f(x+)+f(x-)}{2}\quad 第一类不连续点 \end{cases}$

### Dirichlet引理

- **Dirichlet引理**：设 $\varphi(u)$ 在 $\big[0,\delta\big]$ 上单调，则成立 $\lim\limits_{p\to +\infty}\int^\delta_0 \frac{\varphi(u)-\varphi(0+)}{u}sinpudu = 0$
  - **证明（单增情况）**：
    - $(0,\eta\big]$ 区间：
      - $0\leq \varphi(u)-\varphi(0+) < \e$
      - 积分第二中值定理，分为 **端点函数** $\varphi(u)-\varphi(0+)$ 和 **积分函数** $g(u) = \frac{sinpu}{u}$，得到 $|\int^\eta_0 I| < |\int^{p\eta}_{p\xi}\frac{sinu}{u}du|·\e$
      - Dirichlet反常积分有界性，得到 $\int D(u)\to 0$
    - $\big[\eta,\delta\big]$ 区间：
      - 有界区间，可积或绝对可积
      - Riemann引理得到 $\to 0$
  - **本质**：极小0区间 + 端点中值 + D反常积分有界
  - **等价形式**：$\lim\limits_{p\to +\infty}\int^\delta_0 \varphi(u)\frac{sinpu}{u}du = \frac{\pi}{2}\varphi(0+)$
    - 线性分离Dirichlet积分部分

## 其它判别法

- 设 $f(x)$ 在 $\big[-\pi,\pi\big]$ 上可积或绝对可积，且满足下面两个条件之一，则Fourier级数收敛于 $\sigma(x) = \frac{f(x+)+f(x-)}{2}$
- **Dirichlet-Jordan判别法**：在x的某个邻域 $O(x,\delta)$ 上是 **分段单调有界函数**
  - **分段单调函数**：在 $\big[a,b\big]$ 上存在有限个点 $a=x_0<x_1<...<x_N=b$，使得函数在每个区间上单调
  - **证明**：$\begin{cases} 0\Rightarrow x \\ u\Rightarrow \pm u \end{cases}$，$D(u) + D(-u) = 0+0$ 即得结果
- **Dini-Lipschitz判别法**：在x处满足Holder条件
  - **Holder条件**：x是f的连续点或第一类不连续点，则存在 $L>0、\alpha\in (0,1\big]$，使得成立 $|f(x\pm u)-f(x\pm)| < Lu^\alpha$
  - **证明**：
    - $\frac{|f(x\pm u)-f(x\pm)|}{u} < \frac{L}{u^{1-\alpha}}$
    - 得到 $\frac{\varphi_\sigma(u,x)}{u}$ 在 $\big[0,\delta\big]$ 可积或绝对可积
    - 由Riemann引理，得到 $f_m(x)-\sigma(x) = 0$
  - **推论（可导条件）**：x处两个单侧导数都存在
    - Holder条件 < Lipschitz条件 < 可导条件
