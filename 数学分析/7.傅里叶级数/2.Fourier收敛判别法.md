# Fourier收敛判别法

- 这一章有点偏微分方程的思想，所以比较难懂一点
- **收敛必要条件**：本章默认 $f$ 是以 $2\pi$ 为周期，在 $[-\pi,\pi]$ 上可积或绝对可积的函数
- **本章所使用的形式函数**：即为了形式简洁而设出的中间函数
  - **预备函数**：$\varphi(x,u) = \varphi_\sigma(u,x) = f(x+u)+f(x-u)-2\sigma(x)$
    - <font color='chartreuse'>为了表明下面两种赋权震荡函数在奇点附近的可积性相同，必须研究先 $\p$ 的性质</font>
  - **赋权震荡函数**：$\psi(x,u) = \dis\varphi_\sigma(u,x)\cdot \frac{1}{u\ 或\ 2\sin\frac{u}{2}}$
    - <font color='chartreuse'>为了对下面的被积函数使用Riemann引理，必须先研究 $\psi$ 的可积性</font>
  - **被积函数**：$\psi(x,u)\sin\dfrac{2m+1}{2}u$
    - <font color='chartreuse'>表示部分和函数 $S_m(x)$ 和级数收敛到的函数 $\sigma(x)$ 在邻域 $O(x,\pi)$ 积分中的差值
    - 如果它的积分为 $0$，则级数收敛</font>
  - **判别函数**：$D(x,u) = \cfrac{f(x+u)-f(x+)}{u}\sin(pu)$
    - <font color='chartreuse'>它的本质是被积函数的拆分，即当 $\sigma(x) = \dfrac{f(x+)+f(x-)}{2}$ 时，被积函数就等于 $D(u)+D(-u)$</font>

## 部分和函数 $S_m(x)$

### 形式变换

- **余弦级数形式**：$S_m(x) = \dis\frac{1}{\pi}\int^\pi_{-\pi}f(t)\Big[ \frac{1}{2} + \sum^m_{n=1}\cos n(t-x) \Big]dt$
  - **证明**：
    - 代入E-F公式后，利用积分线性 + 三角函数诱导公式即可
- **Dirichlet核卷积形式**：$S_m(x) = \dis\frac{1}{\pi}\int^\pi_{-\pi} f(x+u)\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}}du$
  - **证明**：
    - 将余弦级数形式进行和差化积，再代换 $t-x=u$ 即可<font color='chartreuse'>（由于是周期函数，所以积分上下限不变）</font>
- **Dirichlet积分形式**：$\dis S_m(x) = \frac{1}{\pi}\int^\pi_0\big[f(x+u)+f(x-u)\big]\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}}du$
  - **证明**：
    - 将积分区间分离成 $[-\pi,0]$ 和 $[0,\pi]$，再换元合并即可
    - <font color='chartreuse'>为什么非要把（对称区间的积分）分离成（半径区间的积分）呢？因为我们主要是研究 $u=0$ （被积函数唯一奇点）附近的性质，从而判别是否收敛的</font>
- **$t$ 和 $u$ 的意义**：
  - $t$ 表示积分的中间变量<font color='chartreuse'>（$t$ 只有在上述定积分中才有意义）</font>
  - $x$ 表示当前点<font color='chartreuse'>（表示当前 $f$ 所处的点）</font>
  - $u$ 表示将当前点 $x$ 和积分中间变量 $t$ 组合后，形成的新中间变量
    - <font color='chartreuse'>$u$ 只有在上述定积分中才有意义，表示在邻域 $O(x,\pi)$ 中进行定积分时的中间变量</font>
  - 定积分不是函数，并不能对其求导和求积
    - <font color='chartreuse'>在 $S_m(x)$ 看来，$t$ 和 $u$ 都不存在，它们既不是参量也不是变量，只有一个具体的定积分结果</font>

### 收敛原理

- **收敛判别原理（作差形式）**：
  - 设 $\varphi_\sigma(u,x) = f(x+u) + f(x-u) - 2\sigma(x)$
    - <font color='chartreuse'>刚才已经说过，$t$ 和 $u$ 都不存在。所以这里的 $\p$ 只是一个为了形式简洁而提出的形式函数（或称为中间函数）。它也没有具体的意义</font>
  - 则 $f(x)$ 的三角级数收敛到某函数 $\sigma(x) \LR \dis\lim_{m\to\infty}\int^\pi_0\p_\sigma(u,x)\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}}du = 0$
  - **证明**：
    - 利用余弦级数形式即可
    - **积分化一**：
      - 直接计算易得 $\dis\frac{2}{\pi}\int^\pi_0(\frac{1}{2} + \sum^m_{n=1}\cos nu)du = 1$
      - <font color='chartreuse'>也就是说，$\sigma(x) = \dis\frac{2}{\pi}\int^\pi_0 \sigma(x)\Big[ \frac{1}{2} + \sum^m_{n=1}\cos nu \Big]du$</font>
    - **积分并入**：
      - 将定积分线性合并，易得 $S_m(x)-\sigma(x) = \dis\frac{1}{\pi}\int^\pi_0 \p_\sigma(u,x)\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}}du$，从而得到结论

## 赋权振荡函数 $\psi(x,u)\sin\dfrac{2m+1}{2}u$

### 可积归零性

- **Riemann引理**：
  - 若函数 $\psi(x)$ 在 $\big[a,b\big]$ 上可积或绝对可积
  - 则 $\dis\lim_{p\to +\infty} \int^b_a \psi(x)\sin(px)dx = \lim_{p\to +\infty} \int^b_a \psi(x)\cos(px)dx = 0$
  - <font color='chartreuse'>用前面反常积分习题的方法来做即可</font>
- **证明（函数有界时）**：
  - 首先取划分 $\{x_i\}^n_{i=1}$，设每个划分区间中 $\int\psi(x)dx$ 下确界为 $m_i$，振幅为 $w_i$
  - 对于 $\dis\abs{\int^b_a \psi(x)\sin(px)dx}$
    - 分离成各个划分区间
    - 添项 $m_i$，将其分离成 $\int w_i\cdot \sin(px)$ 和 $m_i\int \sin(px)$ 两项
    - 进一步放缩
      - 前项用 $\sin(px) < 1$ 放掉三角函数，由可积性得 $<\dfrac{\e}{2}$
      - 后项直接计算得 $<\dfrac{\e}{2}$
        - <font color='chartreuse'>换元积分法 $\dis\frac{1}{p}\int^{x_i}_{x_{i-1}}\sin(px) d(px) < \frac{2}{p}$
        - 故取 $p$ 足够大时有 $\dfrac{2}{p}\sum\limits^n_{i=1}|m_i| < \dfrac{\e}{2}$</font>
    - 最终得到 $<\e$
- **证明（函数存在奇点时）**：
  - 不妨设 $b$ 是唯一奇点
  - 有界区间内由上述结论可得 $<\dfrac{\e}{2}$
  - 由 $\psi(x)$ 绝对可积性，无界区间内 $\dis\int^{b}_{b-\eta}|\psi(x)|dx < \frac{\e}{2}$，三角函数放缩 $<1$ 即得结论
- **理解**：很简单的分区间法而已。反常积分章节就应该学明白的（当然是对于大四学生而言，大一这个学的要死要活的）
  - $\int sinpx$ 的极小性 + $m_i$ 的有界性
  - $sinpx$ 的有界性 + $\int w_i$ 的极小性
- **本质**：$\varphi(x)$ 是振荡权值函数，$sinpx$ 是振荡周期函数。振荡周期极小，权值相对连续，则相互抵消为0

### 非邻域的可积性（无奇点性）

- **局部性原理**：三角级数在 $x$ 点的收敛性只与某个任意小邻域 $(x-\delta,x+\delta)$ 的性质有关
  - **证明**：
    - 已知 $\psi(x,u) = \cfrac{\p(x,u)}{2\sin\frac{u}{2}}$ 仅以 $u=0$ 为奇线，即其关于 $u$ 在 $\big[\delta,\pi\big]$ 可积或绝对可积
    - 故由Riemann引理，$\dis\lim_{m\to+\infty}\int^\pi_\delta \p(x,u)\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}} du = 0$
    - 因此收敛性<font color='chartreuse'>（$S_m(x)-\sigma(x)$ 是否为 $0$）</font>只与 $u\in \big[0,\delta\big]$ 上的积分有关，而这个积分只与 $f(x)$ 在 $(x-\delta,x+\delta)$ 上的性质有关。**（证毕）**
  - **理解**：局部区域对于 $\frac{\triangle}{2sin\frac{u}{2}}$ 是奇点，而非奇点区域可积，可归零
- **奇点等价变换**：
  - 设 $\p(u)$ 在 $\big[0,\delta\big]$ 上可积或绝对可积
  - 则 $\dis\lim_{m\to \infty}\int^\d_0 \p(u)\frac{\sin\frac{2m+1}{2}u}{2\sin\frac{u}{2}}du = \lim_{m\to \infty}\int^\d_0 \p(u)\frac{\sin\frac{2m+1}{2}u}{u}du$
  - **证明**：
    - 作系数差函数 $g(u) = \begin{cases} \cfrac{1}{2\sin\frac{u}{2}} - \cfrac{1}{u} &， u>0 \\ 0 &， u=0 \end{cases}$，易得其在 $[0,\d]$ 连续，从而可积
    - 作差，由Riemann引理可得 $\dis\int^\d_0 \Big[ \p(x,u)g(u) \Big]\sin\frac{2m+1}{2}u du = 0$**（证毕）**
  - **本质**：若仅研究奇点 $x=0$ 附近的可积性，则 $\sin u$ 和 $u$ 没有区别，完全可以等价看待

### 奇点可积条件

- **Dini收敛定理**：
  - 若 $f(x)$ 满足**Dini条件**：$\psi(x,u) = \cfrac{\varphi_\sigma(x,u)}{u}$ 关于 $u$ 在 $\big[0,\delta\big]$ 上可积或绝对可积
    - <font color='red'>即 $\dfrac{f(x+u)+f(x-u)-\sigma(x)}{u}$ 在任意小邻域内可积</font>
  - 则 $f(x)$ 在 $x_0$ 处收敛到 $\sigma(x)$
  - **证明**：
    - 综合以上三个引理（$\varphi_\sigma$ 作差性 + Riemann引理 + 奇点等价变换），结论是显然的
  - **推论（必要条件）**：在 $f(x)$ 的第一类不连续点上，必须保证 $\lim\limits_{u\to 0}\varphi_\sigma(x_0,u)\to 0$，Dini条件才可能成立
    - **证明**：如果 $\lim\limits_{u\to 0}\p(x_0,u) = a > 0$，则 $\dis\int^\d_0 \frac{a}{u}du = \infty$，显然不满足Dini条件。所以至少是无穷小量
  - **推论（收敛函数）**：$\varphi_\sigma(x)=0 \red\Rt \sigma(x) = \frac{f(x+)+f(x-)}{2}$
  - **理解**：在奇点区域可积，则必须是更高阶的无穷小量
  - **本质**：必要条件过强，对原函数限制过大，从而引发反推
- **点态收敛判别定理**：
  - 若 $f(x)$ 在 $x_0$ 处满足Dini条件
  - 则 $S_m(x_0)$ 收敛于 $\sigma(x_0) = \begin{cases} f(x) & f(x) 的连续点 \\\\ \dfrac{f(x+)+f(x-)}{2} & f(x) 的第一类不连续点 \end{cases}$
  - 分段可导

## 其它判别法

### D-J判别法（分段单调性）

- **Dirichlet引理**：设 $\varphi(u)$ 在 $\big[0,\delta\big]$ 上单调，则 $\dis\lim_{p\to +\infty}\int^\d_0 \frac{\varphi(u)-\varphi(0+)}{u}\sin(pu)du = 0$
  - **证明**：以单调递增函数为例，分区间讨论积分的值
    - $(0,\eta\big]$ 区间：
      - 由单调性，易得存在 $u\in (0,\eta)$ 使得 $0\leq \varphi(u)-\varphi(0+) < \e$
      - 应用积分第二中值定理，将被积函数分为以下两个部分
        - **端点函数** $\varphi(u)-\varphi(0+)$
        - **积分函数** $g(u) = \cfrac{\sin(pu)}{u}$
      - 换元放缩易得 $I = \dis\biggm|\int^\eta_0 []du\biggm| < \biggm|\int^{p\eta}_{p\xi}\frac{\sin u}{u}du\biggm|\e$。再由Dirichlet反常积分的有界性，即得 $I\to 0$
    - $\big[\eta,\delta\big]$ 区间：
      - 其为有界区间，且赋权震荡函数可积或绝对可积。故由Riemann引理即得结论
  - **本质**：极小0区间 + 端点中值 + D反常积分有界
  - **等价形式**：$\dis\lim\limits_{p\to +\infty}\int^\delta_0 \frac{\p(u)}{u}\sin(pu)du = \frac{\pi}{2}\varphi(0+)$
    - **证明**：线性分离后，应用前面Dirichlet反常积分的结果即可
- **分段单调函数**：在 $\big[a,b\big]$ 上存在有限个点 $a=x_0<x_1<...<x_N=b$，使得函数在每个区间上单调
- **Dirichlet-Jordan判别法**：
  - 若 $f(x)$ 在点 $x$ 的某个邻域上分段单调且有界
  - 则三角级数收敛于 $\sigma(x) = \dfrac{f(x+)+f(x-)}{2}$
  - **证明**：
    - 此时满足Dirichlet引理条件，即被积函数 $D(u) + D(-u) = 0+0 = 0$，从而收敛

### D-L判别法

- **Holder条件**：
    - $x$ 是 $f$ 的连续点或第一类不连续点
    - 存在 $L>0、\alpha\in (0,1\big]$，使得 $|f(x\pm u)-f(x\pm)| < Lu^\alpha$
    - <font color='chartreuse'>Holder条件 < Lipschitz条件 < 差商一致有界
    - Holder条件  < 单侧导数均存在</font>
- **Holder映射**：满足Holder条件的映射
  - **一致连续性**
- **Dini-Lipschitz判别法**：
  - 若 $f(x)$ 在点 $x$ 处满足Holder条件，则三角级数收敛于 $\sigma(x) = \dfrac{f(x+)+f(x-)}{2}$
  - **证明**：
    - 由条件易得 $D(\pm u) = \dfrac{|f(x\pm u)-f(x\pm)|}{u} < \dfrac{L}{u^{1-\alpha}}$
    - 此时由比较判别法，$\psi(x,u) = \dfrac{\varphi_\sigma(x,u)}{u}$ 在 $\big[0,\delta\big]$ 可积或绝对可积。再由Riemann引理即得结论


### Dirichlet判别法（分段光滑性）

- **分段连续**：$f$ 只有有限个间断点
- **分段可导**：$f$ 除有限个间断点 $\{x_i\}$ 外均可导
  - **实例**：
    - $x\sin\dfrac{1}{x}$ 连续（√），连续可导（×），处处可导（√），分段光滑（×），导函数可积（√）
- **分段光滑**：$f$ 只有有限个极值点
  - **等价命题**：
    - 导函数分段连续
    - 分段可导，且单侧导数均存在
  - **实例**：
    - $|x|$ 分段光滑
    - 阶跃函数分段光滑
- **Dirichlet收敛定理**：若 $f$ 可积或绝对可积、分段连续、分段光滑，则三角级数点态收敛
  - **证明**：
    - 易得分段光滑函数在有界闭区间上满足Lipschitz条件，从而由D-L判别法得结论
