# Fourier变换

## 非周期函数的三角级数

### 三角级数的复指数形式

- **圆频率**：记 $\dfrac{\pi}{T}$ 是圆频率
- **频率变换**：设 $w_n = \frac{n\pi}{T}$
  - <font color='chartreuse'>周期变换是把 $T$ 变成 $\pi$ 周期。（周期改变，函数本身改变）
  - 频率变换是把 $sin\ nx$ 变成 $sin\ w_nx$。（形式改变，函数本身不变）</font>
  - 本质都是换元，但是关注点不同
- **Fourier级数的复指数形式**：
  - **欧拉公式重组形式**：$$f_T(x) = \frac{a_0}{2} + \sum\limits^\infty_{n=1}(\frac{a_n-ib_n}{2}e^{iw_nx} + \frac{a_n+ib_n}{2}e^{-iw_nx})$$
  - **复数指数形式系数**：$$c_n = a_n-ib_n = \frac{1}{T}\int^T_{-T}f_T(t)e^{-iw_nt}dt \\\ \\ c_{-n} = a_n+ib_n = \frac{1}{T}\int^T_{-T}f_T(t)e^{iw_nt}dt$$
  - **复数指数形式**：$$f_T(x) = \frac{1}{2}\sum\limits^{+\infty}_{n=-\infty}c_ne^{iw_nx}$$
    - **本质（欧拉公式）**：三角形式 $\to$ 指数形式

### 无穷周期变换

  - 设 $\D w = w_n - w_{n-1} = \frac{\pi}{T}$
  - 将E-F公式代入展开式，变形为 $\dis f(x) = \frac{1}{2\pi}\sum\limits^{+\infty}_{n=-\infty} \big[ \int^T_{-T}f_T(t)e^{-iw_nt}dt \big] \cdot e^{iw_nx}\D w$
    - 令 $T\to \infty$，即 $\D w\to 0$
    - 但是前面变成了一个关于 $w$ 的函数 $\varphi(w) = \frac{1}{2\pi}\int^{+\infty}_{-\infty}f(t)e^{-iwt}dt\ e^{iwx}$，而Riemann和要求是一个随着n而改变的数列
      - R反常积分的Riemann和形式：$\sum\limits^{\infty}_{i=-\infty}f(i\D x)\D x$
      - 周期函数 $f_T(x)$ 变成了周期无穷函数（非周期函数）$f(x)$
      - $w_n = \frac{n\pi}{T}$ 变成了同样大的 $w = \frac{A}{T}$（无穷消有穷）
      - 因此，现在的形式相当于 $\frac{\infty}{\D x}\cdot f(\D x)\D x$
      - 区别在于，这里的 $\D x$ 并不是固定的，而是随着i变化而变化大小
    - 在这里做了妥协
    - 化为Riemann积分 $f(x)\sim \frac{1}{2\pi}\int^{+\infty}_{-\infty}\big[\int^{+\infty}_{-\infty}f(t)e^{-iwt}dt\big] e^{iwx}dw$
- **Fourier变换**：$$\wh f(w) = F[f](w) = \int^{+\infty}_{-\infty}f(t)e^{-iwt}dt$$
  - <font color='chartreuse'>方括号内的函数 $f$ 是原本的函数，变换后的函数 $\wh f(w)$ 是Fourier系数</font>
  - **意义**：F级数的复数指数形式系数 $c_n$
- **Fourier积分**：$$\frac{1}{2\pi}\int^{+\infty}_{-\infty}dw\int^{+\infty}_{-\infty}f(t)e^{iw(x-t)}dt$$
  - 总函数
  - **意义**：原函数的F级数展开
- **Fourier逆变换**：$$f(x) = F^{-1}[\wh f](x) = \frac{1}{2\pi}\int^{+\infty}_{-\infty}\hat f(w)e^{iwx}dw$$
  - **意义**：逆变换和变换组合正好形成F积分，也就是 $f(x)$

## 傅里叶变换的性质

- Fourier变换的自变量是x，不定参量是w
  - 相对于变换来讲，可以控制的是基频w，所以是 $F[f](w)$
  - 相对于函数来讲，函数被x定义

### Fourier积分性质

- **分段可导**：除有限个点外均可导，在不可导点的左右极限存在，且导数极限也存在
- **Fourier积分定理**：函数在R上绝对可积，在任何闭区间上分段可导，则 $F(x) = \frac{f(x+)+f(x-)}{2}$
  - F级数分段单调有界则收敛，F积分分段可导则收敛
- **Fourier积分的三角形式**：
  - 把 $e^{iw(x-t)}$ 再欧拉展开，得到的函数存在 $sinw(x-t)$ 和 $cos(w-t)$，对于w是奇函数和偶函数。再因为对w进行R上积分，所以奇函数被约去，只留下偶函数
  - $f(x) = \frac{1}{\pi}\int^{+\infty}_0dw\int^{+\infty}_{-\infty}f(t)cosw(x-t)dt$
- **Fourier正弦变换积分**：$f(x)$ 是奇函数
  - $f(x) = \frac{2}{\pi}\int^{+\infty}_0 \big[ \int^{+\infty}_0 f(t)sinwt·dt \big] sinwx·dw$
  - 正弦变换和逆正弦变换的复合

### Fourier变换性质

- **线性**：$F[af+bg] = aF[f] + bF[g]$
- **位移性**：
  - $F[f(x\pm x_0)](w) = F[f](w)e^{\pm iwx_0}$
  - $F^{-1}[f(x\pm x_0)](w) = F[f](w)e^{\mp iwx_0}$
  - 符号相反
  - **证明**：换元即可（无穷区间不影响），指数乘积线性
- **时间尺度性**
  - $F[f(ax)] = \frac{1}{|a|}\hat f(\frac{w}{a})$：换元
- **频率尺度性**
  - $F[\frac{1}{a}f(\frac{c}{a})] = \hat f(aw)$：换元
- **微分性质**：
  - **函数的微分（内部、x）**：R上有连续导数，f与导数绝对可积，$\lim\limits_{x\to\infty}f(x)=0$ $\longrightarrow$ $F[f'] = iw\cdot F[f]$
    - **证明**：分部积分，$e^{-iwx}$ 的导数性质
  - **变换的微分（外部、w）**：$f(x)、xf(x)$绝对可积，则 $F[-ix\cdot f] = (F[f])'$
    - **证明**：含参变量积分可导性
- **积分性质**：$f$ 与 $\int^x_{-\infty}f(t)dt$ 在R上绝对可积，则 $F[\int^x_{-\infty}f(t)dt] = \frac{1}{iw}F[f]$
  - **证明**：绝对可积性得到 R上f积分为0，即极限为0。再反用可微性质即可

## 实际意义

### 卷积

- **卷积**：$(f*g)(x) = \int^{+\infty}_{-\infty}f(t)g(x-t)dt$
  - 对称性
- **卷积的Fourier变换**：$F[f*g] = F[f]\cdot F[g]$
- **Parseval等式**：$f$ 绝对可积，且平方收敛，则 $\int^{+\infty}_{-\infty}[f(x)]^2dx = \frac{1}{2\pi}\int^{+\infty}_{-\infty}|\hat f(w)|^2dw$

### 应用

- 求解微分方程：利用Fourier级数的微分性质，把 $y(x)$ 和 $f(x)$ 展开后化为统一形式。然后把系数凑出一个Fourier变换，利用卷积定义得到卷积形式结果

### 物理意义

- 基频 $w = \frac{\pi}{T}$（最小的频率，波长最长的波）
- 谐频 $nw$
- 谐波振幅 $A = \sqrt{a_n^2+b_n^2} = |c_n|$（任意角公式）
- Fourier级数可以看作是波长不断变小的无穷多个谐波的叠加，用来逼近函数图像。
  - 基频被函数的周期T所决定。当 $T\to \infty$，基频无限小，最大波长无限大
  - 此时任何一个实数都可以看作一个波的长度，Fourier级数可以看成积分形式
- **基频表出性**：Fourier逆变换，可以看作w波（基频波）的Fourier积分
    - **证明**：
      - $F^{-1}[f] = \frac{1}{2} \big[ \int^{+\infty}_{-\infty} \frac{f(w)}{\pi} \cdot e^{iwx}dw \big]$
      - 形似F积分，n恒等于1
      - $\frac{|\hat f(w)|}{\pi} = \frac{\frac{1}{T}|\int^T_{-T}f(x)|e^{-iwx}dx|}{\D w} = \frac{|c_1|}{\D w} \sim |c_1|$
      - 形似基频振幅
