# Fourier展开

- **三角级数（Fourier级数）**：$f(x) = \frac{a_0}{2} + \sum\limits^\infty_{n=1}(a_n\cos nx + b_n\sin nx)$
- $f$ 可积或绝对可积（反常积分下绝对可积）

## 周期为 $2\pi$ 的函数

- **计算Fourier系数**
  - 左式进行定积分 $\int^\pi_{-\pi} f(x)\cos mxdx$
  - 根据三角积分正交性，右式计算结果为 $a_m\pi$
  - 移项求得Fourier系数（$b_m$ 方法同理）
- **Euler-Fourier公式**：**Fourier系数** 推导结果x
  - $a_n = \frac{1}{\pi}\int^\pi_{-\pi} f(x)\cos nxdx$
  - $b_n = \frac{1}{\pi}\int^\pi_{-\pi} f(x)\sin nxdx$

### 奇偶函数

- $f(x)$ 为奇
  - $a_n$ 为奇函数，定积分为0
  - $b_n$ 为偶函数，定积分为2倍正值
  - **正弦级数**：$\sum\limits^\infty_{n=1} b_n\sin nx$
- $f(x)$ 为偶
  - $a_n$ 为偶函数，定积分为2倍正值
  - $b_n$ 为奇函数，定积分为0
  - **余弦级数**：$\frac{a_0}{2} + \sum\limits^\infty_{n=1} a_n\cos nx$

### 周期延拓

- Fourier级数虽然需要函数具有周期，但实际上，只有在 $[-\pi,\pi]$ 上定义的函数才可以直接展开。
- 我们默认R上的函数进行了周期延拓，也就是其它区间上的结果和 $[-\pi,\pi]$ 上相同

### 任意周期函数

- **周期变换**：$f(x)$ 以2T为周期，令 $t = \frac{\pi}{T}x$，$\varphi(t)$ 周期变为 $2\pi$
- $f(x) = \frac{a_0}{2} + \sum\limits^\infty_{n=1}(a_ncos\frac{n\pi}{T}x + b_nsin\frac{n\pi}{T}x)$
- Fourier系数：
  - $a_n = \frac{1}{\pi}\int^{\pi}_{-\pi}\varphi(t)cosntdt = \frac{1}{T}\int^T_{-T}f(x)cos\frac{n\pi}{T}xdx$
  - 
  - $b_n = \frac{1}{\pi}\int^\pi_{-\pi}\varphi(t)sinntdt = \frac{1}{T}\int^T_{-T}f(x)sin\frac{n\pi}{T}xdx$
  - 
  - $\varphi(t)$ 是设出来的中间函数，并不需要考虑
