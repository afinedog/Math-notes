# 反常重积分

## 无界区域的反常积分

- **积分区域的必要条件**：无界区域 $D$，其单侧边界由有限条光滑曲线组成
- **可积函数的必要条件**：函数 $f(x,y)$ 在 $D$ 中有界，且在 $D$ 可求面积的子区域上可积
- **割线区域**：任取一条面积为0的曲线 $\Gamma$，将 $D$ 分成无界和有界两个子区域。其中有界子区域记为 $D_\Gamma$
  - **割线到原点的距离**：$d(\Gamma) = \inf\set{\sqrt{x^2+y^2}\mid (x,y)\in \Gamma}$

### 反常重积分

- **反常二重积分收敛**：若 $d(\Gamma)\to +\infty$，即 $D_\Gamma \to D$ 时,$\dis\iint\limits_{D_\Gamma}f(x,y)$ 的极限存在，则称其在 $D$ 上可积（反常二重积分收敛）
  <!-- - 它不是$\infty$，而是趋近于无限的任意大$A$（无穷大的逼近定义） -->
- **Heine定理**：
  - 设
    - $f(x,y)$ 是无界区域 $D$ 上的非负函数
    - $\{\Gamma_n\}$ 是一列曲线，其割出的有界子区域 $D_n$ 单调递增，且 $\lim\limits_{n\to\infty} d(\G_n) = +\infty$
  - 则反常二重积分收敛在 $D$ 上收敛 $\LR$ 积分数列 $\dis\iint_{D_n} f(x,y)dxdy$ 收敛
  - **证明**：
    - **必要性**：由反常积分收敛的定义直得
    - **充分性**：利用夹逼法则
      - 确定曲线 $\Gamma$，设割线到原点的最大距离为 $\rho = \sup d(\Gamma)$
        <!-- - <font color='chartreuse'>因为是逼近定义，所以总有阶数之差。这里就是先定下曲线阶数，再提升曲线数列极限的阶数</font> -->
      - 由积分数列收敛性，对任意 $\e>0$，都存在 $N>0$ 使得积分数列的值在 $(I-\e,I)$ 之间。因此取 $d(\G) > \rho(\G_N)$ 时就有 $$I-\e \leq \iint\limits_{D_N} f(x,y)dxdy \leq \iint\limits_{D_\Gamma} f(x,y)dxdy \leq I$$
      - 再由距离列 $\to+\infty$ 得，对于任意给定的 $\G$，都存在 $n>0$ 使得 $d(\G_n) > \rho(\G)$。再由 $f$ 的非负性得积分数列单增，从而此时有 $$\iint\limits_{D_\Gamma} f(x,y)dxdy \leq \iint\limits_{D_n} f(x,y)dxdy \leq I$$
      - 综上，由夹逼法即得反常二重积分收敛于 $I$
  - **理解**：
    - 阶数的大小关系为 $\e < N < \G < n$，也就是说我们先取定 $\e$，然后再取一个依赖于它的 $N(\e)$，然后再取一个 $\G(N)$，最后再取一个 $n(\G)$ ，就使得上述不等式均成立
- **$p$ 积分**：
  - **积分区域**：$D = \{(x,y)\mid a^2\leq x^2+y^2 < +\infty\}$
  - **被积函数**：$f(x,y) = \cfrac{1}{r^p}$
  - **收敛性质**：在 $p>2$ 时收敛，在 $p\leq 2$ 时发散
  - **证明**：
    - 设曲线 $\Gamma_\rho$ 割出一个有界圆环，半径分别为 $a$ 和 $\rho$
    - 对 $f$ 极坐标变换+累次积分得：$$\iint\limits_{D_\rho} f(x,y)dxdy = \int^{2\pi}_0 d\theta \int^\rho_\alpha r^{1-p}dr = 2\pi\int^\rho_\alpha r^{1-p}dr$$，化为一重 $p$ 积分，由其性质即得结论
  - **理解**：
    - 极坐标系是最接近一重积分的坐标系，其中 $\theta$ 类似常数，而圆中不同方向r相等
    - 收敛界限不是1而是2，是因为变量代换产生了一个r因子
    - 也就是说，n重积分的收敛界限就是n了
- **比较判别法**
- **绝对可积定理**：$f(x,y)$ 在 $D$ 上可积 $\LR |f(x,y)|$ 在 $D$ 上可积
  - **充分性**：正负函数分离+积分三角不等式+比较判别法即可
  - **必要性**：
    - 反设条件收敛但不绝对收敛，则正负函数中至少有一个发散至无穷。不妨设正函数积分发散至无穷
    - 寻找距离列单增趋于无穷的曲线族 $\{\Gamma_n\}$，使得其划分的区域 $\{D_n\}$ 满足 $$\iint\limits_{D_{n+1}} f^+(x,y)dxdy > 2\iint\limits_{D_n} |f(x,y)|dxdy + n$$
      - <font color='chartreuse'>由正函数积分发散性，该曲线族是存在的</font>
      - 由积分三角不等式得：$$\iint\limits_{D_n} f^+(x,y)dxdy < \iint\limits_{D_n} |f(x,y)|dxdy$$
      - 再由区间可加性，将上两式组合后得：$$\iint\limits_{D_{n+1}-D_n} f^+(x,y)dxdy > \iint\limits_{D_n} |f(x,y)|dxdy + n$$
    - 然后对 $D_{n+1}-D_n$ 区域做分割，从中筛选出 $f$ 正定的区域 $P_n$
      - 放缩易得 $$\iint\limits_{D_n\cup P_n} f(x,y)dxdy \geq -\iint\limits_{D_n} |f(x,y)|dxdy + \iint\limits_{P_n} f^+(x,y)dxdy$$
      - 再由前面的式子得 $右式 > n-1 \to +\infty$，从而 $f$ 的反常积分不收敛，与题设矛盾
    - 要得到不可积，构造矛盾，还需要把区域连通，更加贴合定义
      - 所以构造充分细小的“走廊区域” $\Sigma$ 连接 $D_n$ 和 $P_n$，且 $\Sigma$ 上的 $f$ 积分有界
      - 易得连通区域上的积分 $\iint\limits_{D_n\cup \Sigma\cup P_n}f(x,y)dxdy$ 依然发散，与 $f(x)$ 的反常积分条件收敛彻底矛盾 **（证毕）**
  - **理解**：
    - **二维的多向性**：若可积，则可以单独分离正函数区域，与负函数积分有界的走廊，其不失连通性，则必须也保持可积（**二重积分定义的加强性**：割线$\Gamma$可以任意构造），所以就是绝对可积
    - （一维函数若分离则不连通，二维函数分离后仍然可以构造充分小的走廊使之连通）
<!-- - **Cauchy判别法**：将函数换元为极坐标系后，再与 $p$ 积分比较 -->
- **累次积分公式**：
  - 设 $f$ 在无限半闭矩形区域 $D = [a,+\infty)\times [c,+\infty)$ 内连续
  - 若 $f$ 的累次积分和绝对值累次积分都收敛
  - 则 $f$ 在 $D$ 内可积且可累次积分
  - **应用**：反常变量的维持与不维持（先合并再极限、先极限再合并）
    - 不累次：用R代指无穷大，只求一次极限
    - 累次：先求y极限，再求x极限
- **Poisson积分（二重积分 $\LR$ 一重积分的平方）**：$I = \dis\int^{+\infty}_0 e^{-x^2}dx$
  - **解**：
    - 我们发现，在二维情况下可进行极坐标换元求解 $$\iint\limits_{R^2} e^{-(x^2+y^2)}dxdy = \iint\limits_{D} e^{-r^2}rdrd\theta = \pi$$
    - 再利用累次积分公式得：$$\iint\limits_{R^2} e^{-(x^2+y^2)}dxdy = \int^{+\infty}_{-\infty}e^{-x^2}dx\int^{+\infty}_{-\infty}e^{-y^2}dy \\\ \\\whhhh = (\int^{+\infty}_{-\infty}e^{-x^2}dx)^2 = \pi$$
    - 最后由被积函数对称性得 $I = \dfrac{\sqrt{\pi}}{2}$

## 无界函数的反常积分

- **奇点**：若 $f$ 在 $P_0$ 上有定义，但在 $P_0$ 任何去心邻域上无界，则称 $P_0$ 为 $f$ 的奇点
- **反常积分收敛**：
  - 设
    - $\gamma$ 为面积为0、内部含有奇点的闭曲线，$\sigma$ 为其内部区域
    - $\rho(\gamma) = sup\{|P-P_0| \mid P\in\gamma\}$
  - 若 $\lim\limits_{\rho(\gamma)\to 0}\dis\iint\limits_{D\setminus \sigma}f(x,y)dxdy$ 存在，则称 $f$ 在 $D$ 上的反常积分收敛
- **奇线**：若 $f$ 在 $D\setminus\Gamma_0$上有定义，但在任何包围 $\Gamma_0$ 的区域上无界，则称曲线 $\G_0$ 为 $f$ 的奇线
- **$p$ 级数**：奇点为 $(0,0)$
  - **证明**：以 $\rho$ 代指无穷小，累次积分，只需要计算r极限即可

### 习题

- **一重反常积分化为二重累次积分**：$\dis\int^1_0 \frac{\arctan\ x}{x\sqrt{1-x^2}}dx$
  - **解**：
    - 首先将该函数的一部分写为 $\int^1_0 f(x,y)dy$ 的形式，从而转化为单位矩形上的二重积分
    - 可以构造 $\dis\frac{\arctan\ x}{x} = \int^1_0 \frac{dy}{1+x^2y^2}dy$
    - 则原式转化为 $\dis\int^1_0dx\int^1_0\frac{dy}{(1+x^2y^2)\sqrt{1-x^2}}$
    - 再将 $x$ 和 $y$ 的积分顺序互换，做变量代换 $x=\cos\theta$ 。由有理函数可积性即可
- **奇线积分**：$\dis\iiint\limits_{\Omega}\frac{dxdydz}{\sqrt{1-x^2-y^2-z^2}}$，其中 $\Omega = \{(x,y,z) \mid x^2+y^2+z^2 \leq 1\}$
  - **解**：
    - 做球面坐标变换 $\varphi$ 和 $\theta$，进行累次积分
    - 或直接累次积分，（这个积分的边界比较巧，正好可以消掉）
    - 计算得 $I = \begin{cases} \cfrac{1}{m!}\pi^{m+1}，& n = 2m+1 \\\\ \cfrac{2^m}{(2m-1)!!}\pi^m，&  n = 2m\end{cases}$
