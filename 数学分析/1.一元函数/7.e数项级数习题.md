# 裴礼文（数项级数）

## 求和

- **错位相减法**：
  - **类等比级数**：$\sum\limits^\infty_{i=1} p(n)q^n$。其中 $p(n)$ 是 $n$ 的多项式
- **裂项级数法**：
  - **分式裂项**：设 $x_{n+1} = x_n(x_n+1)$，求 $\sum\limits^\infty_{n=1} \dfrac{1}{x_n+1}$
    - **解**：
      - 此时 $\dfrac{1}{x_{n+1}} = \dfrac{1}{x_n} - \dfrac{1}{x_n+1}$，故级数 $= \sum\limits^\infty_{n=1}\Big[ \dfrac{1}{x_n}-\dfrac{1}{x_{n+1}} \Big] = \dfrac{1}{x_1}$
  - **等比和的裂项**：$\sum\limits^\infty_{i=1}\dfrac{x^{2n}}{1-x^{2n+1}}$
  - **反三角裂项公式**：$\arctan a + \arctan b = \dfrac{a+b}{1-ab}$
  - **反三角级数裂项**：
    - **解**：$\arctan\dfrac{1}{2k^2} = \arctan\cfrac{\frac{1}{2k-1}-\frac{1}{2k+1}}{1+\frac{1}{2k-1}\frac{1}{2k+1}} = \arctan \dfrac{1}{2k-1} - \arctan\dfrac{1}{2k+1}$
  - **根式裂项**
  - **分式裂项**：
    - $\sum\limits^\infty_{n=1} \dfrac{1}{n(n+1)...(n+k)}$
    - $\sum\limits^\infty_{n=1} \Big[ \dfrac{1}{n+\frac{1}{2}}-\dfrac{1}{n+1} \Big]$
      - **解**：由级数的柯西判别法，$\sum\limits^n_{k=1} f(k) - \dis\int^n_1 f(x)dx \to \g(f)$ 收敛
        - 故原式 $= \g_1-\g_2 + \ln\dfrac{4}{3}$
          - 易得 $\g_2 = \g-1$
  - **和函数裂项**：$x_n = \sum\limits^n_{k=1} \dfrac{1}{\sqrt{k}} - 2\sqrt{n}$，则 $\lim\limits_{n\to\infty} x_n$ 存在
    - **证明（裂项级数判敛）**：取裂项级数 $x_n = \sum\limits^n_{k=1} (x_{k}-x_{k-1}) + x_1$，则 $x_n$ 的收敛性就是该级数的收敛性<font color='chartreuse'>（实际上就是前面极限部分的调和级数判敛法）</font>
- **一些裂项公式**：
  - $nn! = (n+1)!-n!$
  - $\dfrac{n}{(n+1)!} = \dfrac{1}{n!}-\dfrac{1}{(n+1)!}$
  - $\sum\limits^n_{k=1} kC^k_n = n2^{n-1}$
    - **证明**：
      - 可以用组合数公式 $kC^k_n = nC^{k-1}_{n-1}$
      - 也可以用二项式定理 + 求导
  - $\sum\limits^n_{k=1} k^2C^k_n = n(n+1)2^{n-2}$
    - **证明**：
      - $nx(1+x)^{n-1}$ 展开后求导
  - $\sum\limits^n_{k=1} (C^k_n)^2 = C^n_{2n}$
    - **证明**：
      - $(1+x)^n(1+x)^n = (1+x)^{2n}$ 两边展开
  - $\sum\limits^n_{k=1} kC^k_n(1-x)^{n-k}x^k = nx$
    - **证明**：
      - 可以用组合数公式 $kC^k_n = nC^{k-1}_{n-1}$
- **部分和法**：
  - 设 $\sum\limits^\infty_{n=1} a_n = A$，则 $\lim\limits_{n\to\infty} \dfrac{a_n + 2a_{n-1} + ... + (n-1)a_2 + na_1}{n} = A$
    - **证明**：
      - 可化为 $\lim\limits_{n\to\infty}\dfrac{1}{n}\sum\limits^n_{k=1}S_k = A$。由无穷消有穷得显然成立
- **方程法**：（其实错位相减法就是方程法的一种）
  - $\sum\limits^\infty_{k=1} q^k\cos k\a$
    - **解（方程法）**：将 $S_n$ 乘 $2q\cos\a$，积化和差，可得 $S_n$ 的方程。求解即可
    - **解（复变函数法）**：复等比级数取实部即可
- **更序级数**：对于绝对收敛的级数，求其更序级数
  - **子列法**：利用收敛级数的结合律，分别求子列的和后再相加即可


## 常用二级结论

- **交错级数求和**：将级数 $1-\dfrac{1}{2}+\dfrac{1}{3}-\cdots$ 的各项重排，使得 $p$ 项为正，$q$ 项为负，则新级数和为 $\ln 2+\dfrac{1}{2}\ln\dfrac{p}{q}$
- **递推比较判别法**：对两个正项级数，极限状态下 $\dfrac{a_{n+1}}{a_n} \leq \dfrac{v_{n+1}}{v_n}$
  - **证明**：累乘法得 $a_n \leq \dfrac{a_N}{b_N}b_n$
- **递推p判别法**：设 $a_n$ 是正项级数，$\a\in (0,1]，\b>1$
  - <font color='chartreuse'>实际上是Rabbe判别法的引理，你看方法都差不多，只不过少了构造伯努利不等式放缩</font>
  - 若 $\dfrac{a_{n+1}}{a_n} \geq (1-\dfrac{1}{n})^\a$，则 $\sum\limits^\infty_{n=1} a_n$ 发散
    - **证明**：
      - 此时等价于 $\dfrac{a_n}{n^\a}$ 递增，柯西收敛原理易得其发散
  - 若 $\dfrac{a_{n+1}}{a_n} \leq (1-\dfrac{1}{n})^\b$，则 $\sum\limits^\infty_{n=1} a_n$ 收敛
    - **证明**：
      - 此时等价于 $\dfrac{a_n}{n^\b}$ 递减，柯西收敛原理易得其收敛


## 敛散性

### 通项判敛

- 判别法即可
- $\sum\limits^\infty_{n=1} \Big[ \sqrt[n]{a} + \dfrac{1}{\sqrt[n]{a}} - 2 \Big]$
  - **解**：由 $e^x$ 的麦克劳林公式得 $\sqrt[n]{a}-1 \sim \dfrac{\ln a}{n} + \dfrac{\ln^2 a}{2n^2}，\dfrac{1}{\sqrt[n]{a}}-1 \sim -\dfrac{\ln a}{n} + \dfrac{\ln^2 a}{2n^2}$，故原式 $\sim \dfrac{\ln^2 a}{n^2}$ 收敛

### 性质判敛

- **广义柯西收敛原理**：
- 若正项级数 $a_n$ 发散，则 $\sum\limits^\infty_{n=1} \cfrac{a_n}{S_n}$ 发散
  - **证明（讨论法）**：
    - 若 $a_n\to +\infty$，则 $\cfrac{a_n}{S_n}\sim \cfrac{1}{n}$ 发散
    - 若 $a_n$ 有界震荡，则设 $m < a_n < M$，则 $\cfrac{a_n}{S_n} > \cfrac{m}{nM}$ 发散
    - 若 $a_n\to 0$，则 $\cfrac{a_n}{S_n} + \cdots + \cfrac{a_m}{S_m} \geq \cfrac{S_m-S_n}{S_m} = 1-\cfrac{S_n}{S_m}$ 有界，故由柯西收敛原理得发散
      - <font color='chartreuse'>貌似并不需要讨论……最后一个方法直接能证明所有情况</font>
- 设 $a_n$ 单增，则 $\sum\limits^\infty_{n=1} \cfrac{1}{a_n}$ 收敛 $\LR \sum\limits^\infty_{n=1}\cfrac{n}{S_n}$ 收敛 
  - **证明**：
    - 柯西收敛原理即可
- 若正项级数 $a_n\searrow 0$ 且 $\sum\limits^n_{k=1} (a_k-a_n)$ 关于 $n$ 有界，则 $a_n$ 收敛
  - **证明**：
    - 设 $\sum\limits^n_{k=1} (a_k-a_n) \leq M$
    - 任取 $m>n$，则 $\sum\limits^n_{k=1} a_k-na_m = \sum\limits^n_{k=1} (a_k-a_m) \leq \sum\limits^m_{k=1} (a_k-a_m) \leq M$
    - 令 $m\to +\infty$，则 $na_m\to 0$，从而左式化为 $S_m\leq M$
    - <font color='chartreuse'>还是柯西收敛原理，不过 $m$ 取成了 $n$ 的高阶无穷大</font>
    <!-- - 题设式子即为 $na_n-S_n\in (m,M)$ -->
- 若 $a_n > 0$，则 $\sum\limits^\infty_{n=1}\cfrac{a_n}{(1+a_1)...(1+a_n)}$ 收敛
  - **证明**：
    - 若 $a_n$ 有界，则原式 $\leq \dfrac{a_n}{(1+\min a_n)^n} \sim \dfrac{1}{(1+\d)^n}$。易得后者级数收敛
    - 若 $a_n$ 无界，则由柯西收敛定理，原式 $\sim \dfrac{a_n}{(1+a_n)^m}\sim \dfrac{1}{a^{m-1}_n}$。易得后者级数收敛
- 设正项级数 $a_n$ 单增，则 $a_n$ 有界 $\LR \sum\limits^\infty_{n=1} \Big( 1-\dfrac{a_n}{a_{n+1}} \Big)$ 收敛
  - **证明**：Sapagof判别法易得结论
- **Kummer判别法**：
  - 设 $a_n,b_n > 0$
  - 若 $\exist \a > 0$ 使得 $\dfrac{b_n}{b_{n+1}}a_n - a_{n+1} \geq \a$，则 $\sum\limits^\infty_{n=1} b_n$ 收敛
  - 若 $\sum\limits^\infty_{n=1}\dfrac{1}{a_n}$ 发散，且 $\dfrac{b_n}{b_{n+1}}a_n - a_{n+1} \leq 0$，则 $\sum\limits^\infty_{n=1} b_n$ 发散
- **Sapagof判别法**：设 $a_n$ 是单减的正项级数
  - 则 $\lim\limits_{n\to\infty} a_n = 0 \LR \sum\limits^\infty_{n=1} \Big( 1-\dfrac{a_{n+1}}{a_n} \Big)$ 发散
  - **证明**：
    - **必要性**：当 $\lim\limits_{n\to\infty} a_n = a > 0$ 时，放缩 + 比较判别法易得收敛
    - **充分性**：当 $\lim\limits_{n\to\infty} a_n = 0$ 时，易得 $\lim\limits_{p\to\infty} \dfrac{a_{N+p+1}}{a_{N+1}} = 0$，故存在 $p_0$ 使得其小于 $\dfrac{1}{2}$
      - 则 $\sum\limits^{N+p_0}_{k=N+1} \Big( 1-\dfrac{a_{k+1}}{a_k} \Big) \geq \sum\limits^{N+p_0}_{k=N+1} \dfrac{a_k-a_{k+1}}{a_{N+1}} > \dfrac{1}{2}$，由柯西收敛原理得其发散
- **Fabo数列的倒数列**：达朗贝尔判别即可
- **积分判别法**：若 $f$ 为单减正定函数，$\lim\limits_{x\to +\infty}\dfrac{e^xf(e^x)}{f(x)} = \l$
  - 则 $\sum\limits^\infty_{n=1} f(n)$ 在 $\l < 1$ 时收敛，$\l > 1$ 时发散

### 子列判敛

- **子列收敛定理**：设 $\lim\limits_{n\to\infty} a_n = 0$，且 $S_n$ 有收敛子列 $S_{pn}$，则级数收敛
  - **证明**：定义法（柯西收敛原理）即可

### 递推公式判敛

- 达朗贝尔、Rabbe、Bentrad判别法
- **方程思路**：设 $x_{n+1} = \dfrac{1}{2}(1-x_n^2)$，易得极限为 $A = \sqrt{2}-1$，则 $\sum\limits^\infty_{n=1} (x_n-A)$ 收敛
  - **证明**：达朗贝尔判别法，$\dfrac{|x_{n+1}-A|}{|x_n-A|} = \dfrac{|\dfrac{1}{2}(1-x_n^2) - \dfrac{1}{2}(1-A^2)|}{|x_n-A|} = \dfrac{|x_n+A|}{2} < 1$
- **方程递推**：设 $a_n$ 有界，$b_n$ 单调有界
  - 若 $\a a_{n+1} + \b a_n = b_n$，且 $\a > \b > 0$
  - 则 $a_n$ 收敛
  - **证明（累加法）**：
    - 转换为递推公式 $a_{n+1} = -\dfrac{\b}{\a}a_n + \dfrac{1}{\a} b_n$，则由累加法可得其通项。再由A-D判别法即可
- 设 $\sum\limits^\infty_{n=1} a_n$ 收敛，$\sum\limits^\infty_{n=1} (b_{n+1}-b_n)$ 绝对收敛，则 $\sum\limits^\infty_{n=1} a_nb_n$ 收敛
  - **证明**：Abel变换

### 其它放缩方法

- **基本不等式放缩**：若正项级数 $a_n$ 收敛，则 $\dfrac{\sqrt{a_n}}{n^p}$ 在 $p> \dfrac{1}{2}$ 时收敛
  - **证明**：$\dfrac{\sqrt{a_n}}{n^p} \leq \dfrac{1}{2}\Big( a_n + \dfrac{1}{n^{2p}} \Big)$
  - **反例**：但 $p=\dfrac{1}{2}$ 时，令 $a_n = \dfrac{1}{n\ln^2(n+1)}$ 发散
- **扩充函数判敛**：设正项级数 $a_n$ 收敛，若 $f(x) = \sum\limits^\infty_{n=1} a_n|\sin nx|$ 满足Lip条件，则 $\sum\limits^\infty_{n=1} na_n$ 收敛
  - **证明**：
    - 易得 $f$ 单增，故 $\sum\limits^n_{k=1}a_k|\sin kx| \leq |f(x)-f(0)| \leq L|x|$
    - 令 $x\to 0$，求极限即得 $S_n = \sum\limits^n_{k=1} ka_k \leq L$，从而收敛
- **放缩化简 + 夹逼判敛**：设 $a_0 = 1，a_{n+1} = a_ne^{\dfrac{1}{a_n}}$，则 $p>2$ 时 $\sum\limits^\infty_{n=1} \dfrac{a_n-n}{n(\ln n)^p}$ 收敛
  - **证明**：
    - 归纳法易得 $a_n \geq 1$
    - $e^x$ 放缩易得
      - $a_{n+1} \geq a_n(1 + \dfrac{1}{a_n}) = a_n + 1$
        - <font color='chartreuse'>即 $a_n \geq n+1$</font>
      - $a_{n+1} \leq a_n(1 + \dfrac{1}{a_n} + \dfrac{1}{a_n^2}) = a_n + 1 + \dfrac{1}{a_n}$
        - <font color='chartreuse'>即 $a_{n+1} \leq a_n + 1 +\ln\dfrac{n+1}{n}$
        - 累加即得 $a_n < n+2+\ln n$</font>
    - 再由之前结论，对 $\dfrac{1}{n(\ln n)^p}$ 判敛，夹逼即得结论
- **Abel变换**：若正项级数 $a_n$ 收敛，则 $\lim\limits_{n\to\infty} \dfrac{\sum\limits^n_{k=1} ka_k}{n} = 0$
  - **证明**：
    - Abel变换化为 $\lim\limits_{n\to\infty}\Big( S_n-\dfrac{S_1+\cdots + S_{n-1}}{n} \Big) = 0$
- **综合应用**：若正项级数 $a_n$ 收敛，则 $\lim\limits_{n\to\infty} \cfrac{n^2}{\frac{1}{a_1} + \cdots + \frac{1}{a_n}} = 0$
  - **证明**：
    - 由基本不等式变形得 $$0 < \cfrac{n^2}{\frac{1}{a_1} + \cdots + \frac{1}{a_n}} \leq \frac{n}{\sqrt[n]{n!}}\sqrt[n]{(a_1)(2a_2)...(na_n)} \leq \frac{n}{\sqrt[n]{n!}}\frac{a_1 + 2a_2 + ... + na_n}{n}$$
    - 由之前结论，最右式为 $0$，夹逼即得结论
- **等比方向放缩**：设正项级数 $a_n\searrow 0$，则 $\sum\limits^\infty_{n=1} a_n$ 与 $\sum\limits^\infty_{m=1} 2^ma_{2^m}$ 收敛性相同
  - **证明**：
    - 设前者的部分和为 $S_n$，后者为 $T_n$
    - 将 $S_n$ 往等比数列的方向放缩，易得 $S_{2^m} > \dfrac{1}{2}T_{m}$，且 $S_{2^m} \leq a_1 + a_2 + T_{m-1}$
    - 故 $S_n$ 有上界等价于 $T_n$ 有上界



## 收敛的性质应用

### 重排放缩法

- 设 $u_n$ 加括号后得到的 $v_n$ 收敛，
  - 若每个括号内符号相同，则 $u_n$ 收敛
    - **证明**：
      - 已知 $\lim\limits_{k\to\infty} v_k = \lim\limits_{k\to\infty} \sum\limits^{n_k}_{i=n_{k-1}+1} u_i = 0$
      - 由子列性质，对任意 $n$，都存在 $k$ 使得 $n_{k-1} < n < n_k$
      - 故 $|U_n - V_{k-1}| = |U_n-U_{n_{k-1}}| \leq |v_k|\to 0$
      - 从而一方收敛可得另一方收敛
  - 若 $\lim\limits_{n\to\infty} u_n = 0$，且每个括号内项数小于 $L$，则 $u_n$ 收敛
    - **证明**：
      - 此时 $|U_n - V_{k-1}| = |U_n-U_{n_{k-1}}| \leq \sum\limits^L_{j=1}|u_{n_{k-1}+j}|\to 0$

### 求级数列极限

  - **解**：由达朗贝尔判别法， $\sum\limits^\infty_{n=1} x_n$ 收敛，故 $x_n\to 0$

### 正负级数

- **正负项**：设 $S_n$ 条件收敛，则 $\lim\limits_{n\to\infty} \dfrac{S^+_n}{S_n^-}$ 存在
  - **证明**：
    - <font color='chartreuse'>若绝对收敛，则 $\lim\limits_{n\to\infty}S_n^+ = \lim\limits_{n\to\infty}S_n^- = 0$，未定</font>
    - 若条件收敛，则 $S_n^-\to \infty$，从而原式化为 $\dfrac{S_n}{S_n^-} -1 \to -1$
- **发散级数的子列**：若 $a_n$ 正定非降，且 $\sum\limits^\infty_{n=1}a_n$ 发散，则 $\lim\limits_{n\to\infty}\cfrac{a_2+a_4+...+a_{2n}}{a_1+a_3+...+a_{2n-1}} = 1$
  - **证明**：
    - 若 $a_n$ 有界，则 $a_n\to A$，由夹逼法易得结论
    - 若 $a_n$ 无界，则同上
  - **推论（三角子列）**：设 $a_n\searrow 0$，$\sum\limits^\infty_{n=1} |a_n\sin n|$ 发散，则D判别法易得 $\sum\limits^\infty_{i=1} a_n\sin n$ 收敛，从而条件收敛

### 正项作差法

- **泰勒公式转化为平方项**：讨论 $\ln\Big[ 1+\dfrac{(-1)^{n-1}}{n^p} \Big]$ 的收敛性
  - **解**：
    - 先求 $\dfrac{(-1)^{n-1}}{n^p}$ 的条件收敛、绝对收敛性

      - <font color='chartreuse'>因为 $x-\ln x \sim \dfrac{x^2}{2}$，即将 $\ln x$ 的条件收敛性，转化为 $x$ 和正项级数 $\dfrac{x^2}{2}$ 的收敛性</font>

### 夹逼法

    - **证明**：
