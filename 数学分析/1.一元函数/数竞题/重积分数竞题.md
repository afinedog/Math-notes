# 重积分数竞题

### Taylor展开

- $f_{xx}^2 + f_{xy}^2 + f_{yy}^2 \leq M$，已知 $f(0,0) = f_x(0,0) = f_y(0,0) = 0$，证明 $|\iint\limits_{x^2+y^2 \leq 1}f(x,y)dxdy| \leq \frac{\pi\sqrt{M}}{4}$
  - **解**：这么好的条件，一眼Taylor展开，然后积分中值（估值）定理即可

### 交换积分次序

- $\int^{2\pi}_{0}xdx \int^{2\pi}_x \frac{sin^2t}{t^2}dt$
  - 一眼交换次序消去分母
  - 或者分部积分，令 $f(x) = \int^{2\pi}_x \frac{sin^2t}{t^2}dt$，$\frac{1}{2}x^2f(x)|^{2\pi}_{0} = 0$
