# 级数数竞题

## 判别法

- $\sum\limits^\infty_{n=1}\frac{1+\frac{1}{2}+...+\frac{1}{n}}{(n+1)(n+2)}$ 判别敛散性
  - 设分子为 $a_n$，需要转化为 $lnn$
    - 方法1：积分比较 $a_n < 1+\int^n_1\frac{1}{x}dx$
    - 方法2：单调有界数列收敛/Stolz定理
  - 然后比较判别法即可
- $a_n = lnf(a_{n-1})$，且 $f'(x) < mf(x)\quad (0<m<1)$，则 $\sum (a_n-a_{n-1})$ 收敛
  - 递推公式 + L中值定理（利用了 $ln$ 导数的数学形式）
- $I_n = \int^\frac{\pi}{4}_0 tan^nxdx$
  - $I_n + I_{n+2} = \frac{1}{n+1}$
  - $I_{n-2} < I_n < I_{n+2}$
