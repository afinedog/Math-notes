# 积分

## 不定积分

- **原函数**：若区间 $I$ 上有 $F'(x) = f(x)$，则称 $F$ 是 $f$ 在 $I$ 上的原函数
  - **不唯一性**：存在参量 $C$
- **不定积分**：$f$ 的全体原函数称为 $f$ 的不定积分 $\int f(x)dx$
  - **被积函数**：$f$
  - **积分变量**：$x$
- **性质**：
  - **线性**：由导数线性易得

### 积分方法

- **第一类换元积分法（拼凑法）**：<font color='chartreuse'>通过删去被积函数的部分因子，将其化为简单形式</font>
  - 若存在 $g$ 使得 $f(x) = \wt f\Big[ g(x) \Big]g'(x)$，其中 $\wt f$ 的积分易求
  - 那么设 $u = g(x)$，则积分结果为 $\wt F\Big[ g(x) \Big] + C$
  - **实例**：
    - $\int f(2x) dx = \dfrac{1}{2}\int f(u)du = \dfrac{1}{2}F(2x)$
- **第二类换元积分法（展开法）**：<font color='chartreuse'>通过将 $x$ 展开，来将被积函数化为简单形式</font>
  - 若存在 $\p$ 使得 $f(x) = \wt f\Big[ \p(t) \Big]\p'(t)$，其中 $f(\p(t))\p'(t)$ 的积分易求
  - 那么设 $x=\p(t)$，则积分结果为 $\wt F\Big[ \p^{-1}(t) \Big] + C$
  - **实例**：
    - **三角换元**：$\int \sqrt{a^2-x^2}dx$ 中，设 $x = a\sin t$ 即可化简被积函数
- **分部积分公式**：$$\int uv' dx = uv - \int vu'dx$$<font color='chartreuse'>前项是只积不导的结果。后项是积一个、导一个的结果</font>
  - **证明**：逆用导数的乘法公式即可
  - **实例**：
    - **指数-三角循环**：$\int e^x\sin x dx$

### 习题

- **垃圾题**：设 $p(x)= \sum\limits_{i=0}^{n}a_ix^i$，若系数满足 $\sum\limits_{i=1}^n \dfrac{a_i}{(i-1)!} = 0$，则 $\int{p(\frac{1}{x})e^xdx}$ 是初等函数
  - **证明**：
    - 首先看出系数条件就是用来积分的，而且是单项式积分。所以首先把p(x)线性分割
    - 分部积分得 $\int e^xf(x)dx = e^x\int f(x)dx - \int e^x(\int f(x)dx)dx$，可以化为 $q_k(x)+\int \frac{e^x}{x}dx$
    - 最后由系数条件，把最后一个无初等函数积分的项给消掉

### 有理函数的不定积分

- **因式分解定理（实根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x) = (x-\a)^kq_1(x)\pad (\a\in\R)$
  - 则存在实数 $\l$ 与多项式 $p_1(x)$ 使得 $$\frac{p(x)}{q(x)} = \frac{\l}{(x-\a)^k} + \frac{p_1(x)}{(x-\a)^{k-1}q_1(x)}$$
  - **证明**：
    - <font color='chartreuse'>目的是分解出可以积分的项，并且降次</font>
    - $\Large\frac{p(x)}{q(x)} = \frac{\frac{p(\alpha)}{q_1(\alpha)}}{(x-\alpha)^k} + \frac{p_1(x)}{(x-\alpha)^{k-1}q_1(x)}$
      - 由加法构造的自由性，设 $p(x) = \frac{p(\alpha)}{q_1(\alpha)}q_1(x)+(x-\alpha)p_1(x)$，然后同时除一个$q(x)$
      - 本质就是利用加法构造，分离出一个 $() + (x-\alpha)·()$
      - 接下来具体的式子利用分式隔离性和根与因式的关系得到。而自由度由没有明确定义的$p_1(x)$撑着
- **因式分解定理（复根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x)$ 有 $l$ 重共轭复根 $\a = \b+i\g$，余式为 $q^*(x)$
  - 则存在实数 $\mu,\nu$ 和多项式 $p^*$ 使得 $$\frac{p(x)}{q(x)} = \frac{\mu x+\nu}{\Big[ (x+\a)(x-\a) \Big]^l} + \frac{p^*(x)}{\Big[ (x+\a)(x-\a) \Big]^{l-1}q^*(x)}$$
  - **证明**：
- **推论**：
  - 有理函数的不定积分都是初等函数
    - **证明**：上面两个因式分解定理的结果都是初等可积的
  - 三角函数的不定积分都是初等函数
    - **证明**：通过万能诱导公式，都可把三角函数化为 $\tan \dfrac{x}{2}$ 的形式，此时 $$\int R(\sin x,\cos x)dx = \int R(\frac{2t}{1+t^2},\frac{1-t^2}{1+t^2})\frac{2}{1+t^2}dt$$ 是有理函数积分，从而初等可积

    - <font color='chartreuse'>奇数次幂可直接分解，偶数次需要用平方差公式</font>

- **常用的奇葩微分**：

#### 根式分式（裂项和换元结合）

- **换元裂项法**：把根式变成多项式，或变成复合三角函数式
  - 添项裂项：$\Large\frac{x^2}{\sqrt{1+x-x^2}} = \sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}} + \frac{x-\frac{1}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}} + \frac{\frac{3}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}}$，然后三角换元即可
  - t最好是齐次根式分式：$\Large\frac{1}{\sqrt[3]{(x-2)(x+1)^2}} = \frac{1}{x+1}\sqrt[3]{\frac{x+1}{x-2}}$。令$t = \sqrt[3]{\frac{x+1}{x-2}}$即可

#### 分式三角函数（平方则诱导，一次则万能或升幂）

- **化一公式、升降幂公式后导数配凑、换元导数配凑**
- $\dis\int\frac{dx}{a+b\sin x}$ 类型
  - **解**：用 $t = \tan\cfrac{x}{2}$ 代换
- $\dis\int\frac{dx}{a+b\sin^2x}$ 类型：
  - **解**：用 $t = \cfrac{1}{\tan x}$ 代换
- **万能公式法**：$$\int\frac{dx}{1+\sin x+\cos x} = \int\frac{\dfrac{1}{\cos^2\frac{x}{2}}}{2\tan\frac{x}{2}+2}dx = \int\frac{d(\tan\frac{x}{2})}{2\tan\frac{x}{2}+2}$$
- **直接换元法**：$$\int\frac{dx}{1+\sin x+\cos x} = \int\frac{d(x+\frac{\pi}{4})}{1+\sin (x+\frac{\pi}{4})}$$
- 三角函数加减法，使内部系数统一：$$\dis\int\frac{dx}{\sin(x+a)\cos(x+b)} = \frac{1}{\cos(a-b)}\int\frac{\cos[(x+a)-(x+b)]}{\sin(x+a)\cos(x+b)}dx$$，拆解后裂项即可
- **换元法 + 诱导公式**：$$\dis\int\frac{1}{\sin^3x+\cos^3x}dx = \frac{1}{\sqrt{2}}\int\frac{1}{\cos(\frac{\pi}{4}-x)[\frac{1}{2}+\sin^2(\frac{\pi}{4}-x)]}$$

### 抽象不定积分

- **导数法表示抽象积分**：设 $p_n(x)$ 是 $n$ 次多项式，则 $$\dis\int \frac{p_n(x)}{(x-a)^{n+1}}dx = \sum\limits^n_{k=0} \frac{p^{(k)}(a)}{k!}\int \frac{dx}{(x-a)^{n-k+1}}$$
- 设 $R(u,v,w)$ 是三个变量的有理函数，求 $\dis\int R(x,\sqrt{a+x},\sqrt{b+x})dx$
  - **解**：只需通过换元，将三个自变量化为有理函数即可
- **隐函数的不定积分**：设 $(\dfrac{y}{x})^2(x-y)=1$，求 $y(x)$ 的原函数
  - **解**：
    - 设 $\cfrac{y}{x} = t$，则 $\begin{cases} x = \dfrac{1}{t^2(1-t)} \\\\ y=\dfrac{1}{t(1-t)} \end{cases}$
    - 此时可以写出原函数的参数形式 $Y(t) = \dis\int y(t)dt = \ln t + \ln (1-t) + C$<font color='chartreuse'>（至于显函数形式，需要求 $x(t)$ 的反函数，比较难）</font>

## 定积分

- **分割法求面积**：$S_n = \sum\limits_{i=1}^n \frac{1}{n}·(\frac{i-1}{n})^2$。$i-1$可以换成中值$\xi$，或者右端点值$i$，即可积
- **黎曼可积**：
  - 设 $f$ 是 $[a,b]$ 上有界函数
    - 取**划分** $P:a = x_0 < x_1 < ... < x_n = b$
    - 在**划分区间**中取点 $\xi_i\in [x_{i-1},x_i]$
    - **区间长度**设为 $\D x_i$
    - 设**最大长度** $\l = \max\limits_{1\leq i\leq n} \D x_i$
  - 若 $\l\to 0$ 时，**黎曼和**的极限 $\lim\limits_{\l\to 0} \sum\limits^n_{i=1} f(\xi_i)\D x_i$ 存在，且与 $P$ 和 $\xi_i$ 的取法无关
  - 则称 $f$ 在 $[a,b]$ 上Riemann可积
- **定积分**：$\lim\limits_{\l\to\infty} S_n$ 称为 $f$ 在 $[a,b]$ 上的定积分 $I = \int^b_a f(x)dx$
  - **积分上限**：$b$
  - **积分下限**：$a$

### 达布和

- **Darboux大和/小和**：若黎曼和的每个划分区间中 $\xi_i$ 都取最大值/最小值，则称其为达布大和/小和
  - **确界性**：达布大和为 $\sup S_n$，达布小和为 $\inf S_n$
- **达布划割引理**：若原划分中添加新分割点，则大和不增，小和不减<font color='chartreuse'>（大和可能减，小和可能增）</font>
  - **证明**：用黎曼和定义讨论大小即可
- **达布确界引理**：
  - 设
    - $\ol S$ 是所有划分的达布大和，$\ul S$ 是所有划分的达布小和
    - $M$ 和 $m$ 是 $f$ 在 $[a,b]$ 中的上/下确界
  - 则 $\forall P_1,P_2$ 都有 $m(b-a) \leq \underline{S}(P_2) \leq \bar{S}(P_1) \leq M(b-a)$
  - **证明**：
    - 由确界定义易得最前和最后两个不等式
    - 要证中间的不等式，只需取细于 $P_1,P_2$ 的划分 $P_3$，则由达布划割引理就有 $\ul S(P_2) \leq \ul S(P_3) \leq \ol S(P_3) \leq \ol S(P_1)$
- **达布定理**：
  - 设所有的划分中，$l$ 是最小的大和 $\inf \ol S$，$L$ 为最大的小和 $\sup\ul S$
  - 则对任意 $[a,b]$ 上有界的函数 $f$ 都有 $\begin{cases} L = \lim\limits_{\l\to 0}\ol S(P) \\ l = \lim\limits_{\l\to 0}\ul S(P) \end{cases}$
    - <font color='chartreuse'>最大的小和是无限分割的大和，最小的大和是无限分割的小和，若极限值相等，说明无限分割时大小和相等，从而可积</font>
  - **证明**
    - 由 $L$ 是确界，给出一个无限接近上界的划分 $P'$，从而引出 $\e$
    - 在 $P'$ 的基础上，定义无穷小量 $\delta = \min\{\D x_i'，k\e\}$，以其为最大区间长，构造无穷细划分 $P$
    - 在 $P$ 中插入新分点，构成 $P^*$，它起引出上下界，组合$\delta$的作用
    - 将 $\bar{S}(P) - L$ 裂项为下面三个式子的和
      - $[\ol{S}(P) - \ol{S}(P^*)] < (M-m)\d \sim \e$
        - <font color='chartreuse'>在每个划分区间中，进行上下界放缩，设 $k = \dfrac{1}{(p-1)(M-m)}$ 即得 $\e$</font>
      - $[\ol{S}(P^*) - \ol{S}(P')] < 0$
        - <font color='chartreuse'>$P^*$ 更细，由划割引理，上式是负数</font>
      - $[\ol{S}(P') - L] = \e$
        - <font color='chartreuse'>由确界定义引出 $\e$ 之一</font>
      - 综上即得 $L = \ol S(P) = \lim\limits_{\l\to 0} \ol S$
  - **理解**：还是不断放缩的过程，和前面的本质没什么差别

### 黎曼可积条件

- 设 $f$ 在 $[a,b]$ 上有界，则以下命题等价
  - $f$ 黎曼可积
  - **最小大和 = 最大小和**：$\sup \ul S = \inf \ol S$
  - **无限细分割中，大和=小和**
  - **黎曼和的极限存在**：$I = \lim\limits_{\l\to 0}\sum\limits^n_{i=1} f(\xi_i)\D x_i$
- **证明**：
  - **必要性**：夹挤易得
  - **充分性**：首先可以取 $|M-f(\xi_i)| < \e$，然后 $|b-a|\e \to 0$即可得到 $|\ol{S}(P)-\sum\limits^n_{i=1}f(\xi_i)\D x_i| < \e$
- **本质**：无限分割的$\D x$是高阶无穷小（相对于$\D f(\xi)$而言）（数学本质就是振幅$\omega$极限为0）
  - 闭区间上只有有限个不连续点的有界函数可积：单独给每个不连续点一个无穷小区间，设$\delta = \frac{\e}{4k(M-m)}$
    - k用来消去k个不连续点的累加
    - M-m用来消去有界的不连续点造成的影响
    - $\e$用来引出无穷小量
    - 连续区间共有k+1个，不连续区间共有k个，共有2k+1个子区间。（4不用管）
- **振幅和**：设划分区间的振幅为 $w_i = M_i-m_i$，则 $f$ 黎曼可积 $\LR f$ 的振幅和 $\lim\limits_{\l\to 0}\sum\limits^n_{i=1} w_i\D x_i = 0$

### 习题

- **闭区间上连续函数可积**
  - **证明**：
    - 由连续性，$\forall \e>0，\exists \d > 0$，取 $\l<\d$ 时有 $\forall |w_i|<\e$
    - 则此时振幅和 $\lim\limits_{\l\to 0}\sum\limits^n_{i=1} w_i\D x_i < \e\sum\limits^n_{i=1} \D x_i = (b-a)\e \to 0$（**证毕**）
- **闭区间上单调函数可积**（闭区间 + 单调蕴含有界条件）
  - **证明**：
    - 设 $f$ 的最值为 $M$ 和 $m$，在所有划分区间中都取 $w_i = |M-m|$，则此时振幅和为 $\e|M-m|\to 0$（**证毕**）
- **闭区间上仅有限点不连续的有界函数可积**
  - **证明**：
    - 设不连续点为 $\{x_i\}^m_{i=1}$，取它们的一共 $m$ 个邻域，使得邻域外的点均连续
    - 则转化为证明这 $m$ 个邻域内的振幅和为 $0$
    - 由于 $f$ 有界，故取每个邻域的振幅取为 $w_i = M-m$，再合理设置邻域半径即可得振幅和为 $0$（**证毕**）
- **闭区间上不连续点收敛的有界函数可积**：
  - 设 $[a,b]$ 上有界函数 $f$ 的不连续点为 $\{x_n\}^\infty_{n=1}$，若 $\lim\limits_{n\to\infty} x_n$ 存在，则 $f(x)$ 可积
  - **证明**：
    - 由收敛性，设不连续点极限为 $x_0$，则存在一个 $O(x_0,\d)$，使得其外的不连续点有限，则只需讨论该邻域内的可积性即可
    - 再由 $f$ 有界性，放缩即可得振幅和为 $0$（**证毕**）
- **复合可积性**：若 $f$ 在 $[a,b]$ 上可积，$A\leq f\leq B$，$g$ 在 $[A,B]$ 上连续，则 $g\circ f$ 在 $[a,b]$ 上可积
  - **证明**：和连续函数复合传递性的证明相似。化为黎曼和语言即可
- **黎曼函数可积**：
  - **证明**：分成两种区间，讨论其振幅和
    - 一种是 $R(x) < \dfrac{\e}{2k}$
      - 此时黎曼和的分母单元无穷小。虽然这样的区间有无限个，但总体区间长度是有限的，故放缩可得振幅和无穷小
    - 一种是 $\D x_i < \dfrac{\e}{2k}$
      - 此时分母单元有界，从而振幅有界。但这样的区间只有有限个，且区间长度无穷小，故放缩可得振幅和无穷小
- **狄利克雷函数不可积**：
  - 微观上来看，它处处不连续，且振幅处处有界，所以找不到可以以直代曲的无穷小区间。
  - 宏观上来看，它是离散的点组成的，相当于让你求无限条直线的面积
- **$L^p$ 范数的极限**：设 $f,g$ 在 $[a,b]$ 上连续，且 $f\geq 0，g>0$
  - 则 $$ \lim\limits_{n\to\infty} \dkh{\int^b_a \Big[ f(x) \Big]^ng(x)dx}^{\dfrac{1}{n}} = \max_{a\leq x\leq b} f(x)$$
  - **证明（泛函方法）**：
    - 由 $g$ 有界，故由积分上下界不等式 + 夹逼法可删去 $g$，转化为泛函分析的范数关系式 $\lim\limits_{n\to\infty} \|f\|_{L^n(\R)} = \|f\|_{L^\infty(\R)}$
  - **证明（数分方法）**：
    - <font color='chartreuse'>其实这就是数列的奥特曼极限的无穷形式。尝试仿照奥特曼极限用夹逼法做</font>
    - 设 $\max f = M$
    - 首先由积分上界不等式，左式 $\leq \lim\limits_{n\to\infty} \sqrt[n]{|M|^n(b-a)} = M$
    - 再由积分第一中值定理，存在 $\xi\in [a,b]$ 使得左式 $= \lim\limits_{n\to\infty}\sqrt[n]{|f(\xi)|^n(b-a)}$
      - <font color='chartreuse'>再由于 $f$ 有界，故 $\forall n$ 都存在 $\d(n)>0$ 使得 $|f(\xi)|^n \geq \d(n)|M|^n$
      - 从而左式 $\geq \lim\limits_{n\to\infty}\sqrt[n]{ \d(n)|M|^n(b-a)}$（错的，这个右式是小于 $M$ 的）
      - 但是证明思路肯定是没问题的。我应该把重心放在对 $\xi$ 位置的讨论上</font>
    - 奥特曼极限的原理是，项的值越大，其取幂时的增长速度越快。且幂极限之间的差值远远超过其它项。所以最终取极限时，最大项的权值会膨胀到无穷大。
    - 积分和有限数列和的区别是，有限和是离散对象，每项的初始权值都是 $1$。而积分是连续对象，每项的初始权值是无穷小。所以相比奥特曼极限，我们在这里还需要多考虑一个无穷小量 $\D x_i$。不过既然该结论成立，也就是需要我们证明，这个无穷小量的加入是无伤大雅的

### 定积分的性质

- 以下均设 $f,g$ 在 $[a,b]$ 上可积
- **线性**
  - **证明**：黎曼和具有线性 + 极限具有线性 = 定积分也有线性
- **乘积可积性**
  - **证明**：黎曼和 + 三角不等式（仿照极限的乘法）
  - **推论**：乘积的定积分 $\neq$ 定积分的乘积
<!-- - **有界性**
  - **证明**：废话，极限存在可不就是有界吗 -->
- **连续性**：$\int f(x+h) \to \int f(x) \quad (h\to 0)$
  - **证明**：
    - 只需证明 $[a,a+h]$ 和 $[b,b+h]$ 上相等即可
    - 由于这两个区间都是无穷小，故振幅和极小，从而相等
- **保序性**：若 $[a,b]$ 上恒有 $f(x)\geq g(x)$，则任意子区间 $I$ 上有 $\dis\int_I f(x)dx \geq \int_I g(x)dx$
  - **证明**：由黎曼和的保序性直得结论
- **绝对可积性**：$\dis|\int^a_b f(x)dx| \leq \int^a_b|f(x)|dx$
  - **证明**：讨论黎曼和即可
- **区间可加性**：$\dis\int^a_b f(x)dx + \int^b_c f(x)dx = \int^a_c f(x)dx$
  - **证明**：讨论黎曼和即可
- **积分第一中值定理**：若 $[a,b]$ 上恒有 $f\leq M$，则 $\dis\int^a_b f(x)dx \leq M(b-a)$
  - **证明**：讨论黎曼和即可

## 微积分基本定理

- **上限函数**：$F(x) = \dis\int^x_0 f(t)dt$
  - **原函数性**：$F$ 是 $f$ 的原函数的一种
    - **证明**：导数定义 + 微分中值定理 + $f$ 的连续性
  - **推论**：当 $f(x)$ 连续时 $F$ 可微，且 $F'(x) = f(x)$
    - **证明**：由原函数定义直得
- **微积分基本定理**：设 $f$ 在 $[a,b]$ 上连续，$F$ 是其上的原函数，则 $\dis\int^b_a f(x)dx = F(b)-F(a)$
  - **证明**：由微分的唯一性，原函数之间只会相差一个常数
    - 故可设 $F(x) = \dis\int^x_0 f(x)dx + C$
    - 直接验证等式即可

## 定积分与几何计算

- **积分的同时性**：
- **积分的有阶性**：
  - 积分实际上积的是极小的三维体，而不是二维图形。dx就是那个极小值
  - 举个例子，用积分计算圆的面积时，$dS$ 是极小的扇形，而不是半径线段
  - 另一个很经典的例子，不可用平移法证明正方形和其内接圆的周长相等，因为（圆的弧长微分 $dl$）与（正方形两边的微分 $dx+dy$）的差值 $\D$ 始终不是（前两者的无穷小量），故在积分过程中不可忽略

### 计算图形面积

- **参数函数形式**：$\begin{cases} y = y(t) \\ x=x(t) \end{cases}$ 在 $t\in [a,b]$ 上的图像面积为 $S = \dis\int^b_a y(t)dt$
- **显函数形式**：$f(x)$ 在 $[a,b]$ 上的图像面积为 $S = \dis\int^b_a f(x)dx$
  - **证明**：由定积分的定义直得
- **极坐标形式**：$r = r(\t)$ 在 $[\a,\b]$ 上的图像面积为 $S = \dis\frac{1}{2}\int^\beta_\alpha r^2(\theta)d\theta$
  - **证明**：
    - 取角度极小的曲边三角形，由正弦定理，其面积为 $dS = \dfrac{1}{2}r(\t)r(\t+d\t)d\t$
    - 由 $r$ 的光滑性，$r(\t+d\t) = r(\t)$
    - 对这些三角形的面积取积分得 $S = \dis\int^\b_\a dS = 原式$

### 计算曲线弧长

- **显函数形式**：$L = \dis\int \sqrt{1^2+[f'(x)]^2}dx$
- **参数函数形式**：$\dis\int \sqrt{[x'(t)]^2+[y'(t)]^2}dt$
  - **证明**：
    - <font color='chartreuse'>只需证明：曲线取极小部分时，该部分可以近似看为直线（以直代曲）
      - 核心在于证明（曲线与直线的长度差）是（曲线长度）的无穷小量，从而在（对曲线长度积分）时，该差值可忽略不计</font>
    - 设曲线长度的微分为 $dl$，将 $dl$ 两端连接起来的直线长度为 $dL$
    - 由勾股定理，$dL = \sqrt{dx^2+dy^2}$
    - 写成坐标形式即为 $\ol{P_{i-1}P_i} = \sqrt{[x(t_i)-x(t_{i-1})]^2 + [y(t_i)-y(t_{i-1})]^2}$
    - 由微分中值定理，$\ol{P_{i-1}P_i} = \sqrt{x'(\xi_i)^2 + y'(\eta_i)^2}dt$
    - <font color='chartreuse'>再由 $x(t),y(t)$ 的光滑性，$x'(\xi_i)-x'(t_i) << x'(t_i)$
      - 证：两式作差后，利用三角不等式化为 $x$ 和 $y$ 的振幅和 $x(\eta_i)-x(\xi_i)$。再由 $x(t)$ 和 $y(t)$ 可积即得振幅和为0</font>
- **极坐标形式**：$\dis\int \sqrt{r^2(\theta) + [r'(\theta)]^2}d\theta$
  - **证明**：
    - 做变量代换 $\begin{cases} y = r\sin\t \\ x = r\cos\t \end{cases}$ 即可

## 反常积分

### 分类

- **反常积分**：存在以下两种情况的积分
  - 积分区域无界
  - 被积函数无界
- **收敛积分**：若反常积分有界，则称为收敛
  - **无限区域**：设 $f$ 在 $[a,+\infty)$ 有定义，且任意有限区间 $[a,A]$ 上可积
    - 若 $\lim\limits_{A\to +\infty} \dis\int^A_a f(x)dx$ 存在，则称 $\dis\int^{+\infty}_a f(x)dx$ 收敛
    - <font color='chartreuse'>由于这里还没有广义实数的概念，故 $+\infty$ 不能当成一个数来讨论。必须用 $A\to +\infty$ 时的性质来指代</font>
  - **无界函数**：设 $f$ 在 $x=b$ 的某个左邻域内无界，且 $\forall \eta\in (0,b-a)$，$f(x)$ 在区间 $[a,b-\eta]$ 上有界可积
    - 若极限 $\lim\limits_{\eta\to 0+} \dis\int^{b-\eta}_a f(x)dx$ 存在，则称 $\dis\int^b_a f(x)dx$ 收敛
    - <font color='chartreuse'>同上，需要用定积分极限的收敛性来定义反常积分的收敛性</font>
  - **反例**：被积函数无界，积分区域无界，但反常积分收敛
    - $f(x) = \begin{cases} n+1, & x\in[n,n+\frac{1}{(n)(n+1)^2}] \\ 0, & x\in (n+\frac{1}{(n)(n+1)^2},n+1) \end{cases}$
    - <font color='chartreuse'>函数值相对于划分区域极小</font>
  - **互化定理**：通过换元，上面两种反常积分可以相互转化
- **积分性质**：
  - **保序性**：
  - **区间可加性**：
  - **无乘积可积性**
- **导数性质**：设反常积分收敛
  - 若被积函数有界，则只能是积分区域无界，易得此时 $\lim\limits_{x\to +\infty} f'(x) = 0$
  - 若被积函数无界，则存在无穷个不可导区间
- **一致连续性质**：反常积分收敛
  - 由于是极限积分，所以连续就是一致连续，一致连续则函数有界
  - 但被积函数有界不一定一致连续
- **双端收敛**：若要得到 $\dis\int_\R f(x)dx$ 收敛，必须得是 $(-\infty,0)$ 和 $(0,+\infty)$ 上均收敛才行
- **Cauchy主值**：若 $\lim\limits_{A\to+\infty}\dis\int^A_{-A} f(x)dx = \lim\limits_{A\to +\infty} \Big[ F(A)-F(-A) \Big]$<font color='chartreuse'>（两侧无穷远处的阶数相同）</font>，则称其为反常积分的柯西主值 $(\text{cpv})\int^{+\infty}_{-\infty} f(x)dx$
  - **实例**：中心对称函数存在柯西主值
- **主值判别法**：$f\geq 0$ 时，$\dis\int_\R f(x)dx$ 收敛 $\LR \dis\text{(cpv)}\int_\R f(x)dx$ 收敛
  - **证明**：
    - **必要性**：由于函数非负，故实轴上积分收敛等价于正无穷远和负无穷远处积分均收敛。显然此时柯西主值是收敛的
    - **充分性**：
    - <font color='chartreuse'>不过如果函数非负，那么实轴上收敛时两端可以是都发散的。</font>
    <!-- - **必要性**：由反常积分收敛的定义，此时 $\lim\limits_{A\to -\infty}\lim\limits_{B\to\infty} \dis\int^B_A f(x)dx$ 存在且有限，故由函数极限的Heine定理易得柯西主值收敛

### 习题

- $I = \dis\int^{\dfrac{\pi}{2}}_0 \ln(\sin x)dx$
  - **解**：
    - 换元 $t = \frac{x}{2}$
    - 使用升幂公式 $\sin x = 2\sin\dfrac{x}{2}\cos\dfrac{x}{2}$ 后裂项，再由 $\sin$ 和 $\cos$ 关于 $\dfrac{\pi}{2}$ 的对称性，重新组合后得到方程，求解得 $I = \dfrac{\pi}{2}\ln 2$
- $I = \dis\int^{+\infty}_0 \frac{dx}{(1+x^2)(1+x^\alpha)}$
  - **解**：以 $x=1$ 为界，分离两个区间
    - 无穷部分换元 $x = \frac{1}{t}$，则变为有界区域反常积分，并且可以重组消项
- $I = \dis\int^{+\infty}_0 \frac{\ln x}{1+x^2}dx$
  - **解**：
    - 设 $u = \arctan x$，则 $I = \dis\int^{\dfrac{\pi}{2}}_0 \ln(\tan u)du$
    - 将 $\tan$ 分解成 $\sin$ 和 $\cos$ 的两个积分，应用之前结论即可
- $\lim\limits_{n\to\infty} \dfrac{\sqrt[n]{n!}}{n}$
  - **解（夹逼法）**：
    - 均值不等式得 $原式 \leq \dfrac{1}{2}$
    - 若能证明 $n!\geq (\dfrac{n}{2})^n$ 即可，但很遗憾它不成立
  - **解（积分法）**
    - 取对数化为 $\lim\limits_{n\to\infty} \dfrac{1}{n}\sum\limits^n_{i=1} \ln\dfrac{i}{n} = \dis\int^1_0 \ln xdx = -1$
    - 故极限为 $\dfrac{1}{e}$
    - 受此启发，转化为 $e$ 的定义应该也能做，但没必要了

### 比较判别法

- **Cauchy收敛原理**：
  - $\dis\int^{+\infty}_a f(x)dx$ 收敛 $\LR\forall \e>0，\exist A_0\geq a，\forall A,A'\geq A_0$ 有 $|\dis\int^{A'}_A f(x)dx| < \e$
  - **证明**：由函数极限的C收敛原理直得
  - <font color='chartreuse'>柯西收敛原理的威力在于，它把反常积分收敛问题的讨论范围缩小到了无穷范围。也就是说我们不再需要关心（有限长度区间）或（函数有界区域）内的情况</font>
  <!-- - 因为原函数没法计算，积分具有作差性，所以Cauchy收敛原理就独当一面了 -->
  - **绝对收敛**：若被积函数取绝对值后，反常积分依然收敛，则称为绝对收敛
  - **条件收敛**：反常积分收敛，但不绝对收敛
- **比较判别法**：设 $[a,+\infty)$ 上，$f,g$ 均非负，且 $\lim\limits_{x\to+\infty} \dfrac{f(x)}{\p(x)} = l$
  - 则 $l\in (0,+\infty)$ 时，两函数的反常积分同时收敛、同时发散
  - **证明**：定积分的保序性直得
  - **实例**：$p$ 积分 $\dis\int^{+\infty}_a \frac{dx}{x^p}$，当 $p>1$ 时收敛，其余情况发散
- **Cauchy判别法**：将函数与 $p$ 积分比较

### 乘积判别法

- **积分第二中值定理**：若在 $[a,b]$ 上 $f$ 可积，$g$ 单调，则存在 $\xi\in[a,b]$ 满足 $$\dis\int^b_a f(x)g(x)dx = g(a)\int^\xi_a f(x) + g(b)\int^b_\xi f(x)$$
  - **证明**：<font color='chartreuse'>显然需要用上限函数和积分中值定理</font>
    - 不妨设 $f$ 连续，$g'$ 可积
    - 设 $f$ 的上限函数为 $F$，对左式分部积分（积 $f$）化为 $F，g，g'$ 的式子
      - 第一项 $F(x)g(x)\biggm |^b_a = g(b)\dis\int^b_a f(x)dx$
      - 再由 $g$ 的单调性，$g'$ 符号不变，故由积分第一中值定理，第二项转化为 $\dis\Big[ g(b)-g(a) \Big]\int^\xi_a f(x)dx$
    - 综上即得结论
  - **理解**：
    - 直接套用积分第一中值定理的结果为 $$\int^b_a f(x)g(x)dx = g(\xi)(\int^\xi_a f(x)dx + \int^b_\xi f(x)dx)$$
    - 只要保持自由度不变，就可以将一些未知量确定下来
      - 若 $g(x)$ 单增，且 $g(a) \geq 0$，则可化为 $g(b)\int^b_\xi f(x)dx$
      - 若 $g(x)$ 单减，且 $g(b) \geq 0$，则可化为 $g(a)\int^\xi_a f(x)dx$
      - 上面都是把 $g(\xi)$ 的自由度给了积分上下限里的 $\xi$
      - <font color='chartreuse'>本题的精华在于说明了这一过程只需上限函数 + 积分第一中值定理即可完成</font>
- **Abel判别法**：
  - 若 $\dis\int^{+\infty}_af(x)dx$ 收敛，$g(x)$ 单调有界
  - 则 $\dis\int^{+\infty}_a f(x)g(x)dx$ 收敛
  - **证明（柯西收敛原理）**：
    - 由 $g$ 单调性，其满足积分第二中值定理条件，从而 $$\dis\int^{A'}_A f(x)g(x)dx = g(a)\dis\int^{\xi}_A f(x)dx + g(b)\dis\int^{A'}_\xi f(x)dx$$
    - 由 $g$ 单调有界，得其收敛。从而 $g(a),g(b)$ 有界
    - 再将 $f$ 反常积分的收敛性和柯西收敛原理结合，即得这两项都趋于0。故由柯西收敛原理即得结论
- **Dirichlet判别法**：
  - 若 $F(A) = \dis\int^A_a f(x)dx$ 在 $[a,+\infty)$ 有界，$g(x)$ 单调趋于 $0$
    - <font color='chartreuse'>不可以说反常积分有界，因为如果反常积分不收敛，那么我们就不能认为其是一个确定的量。所以必须说上限函数在无穷区间上有界</font>
  - 则 $\dis\int^{+\infty}_a f(x)g(x)dx$ 收敛
  - **证明**：同上用积分第二中值定理即可
  - **推论**：若将条件弱化为 $g(x)$ 单调有界，则设 $g(x)$ 极限为 $A$，那么由Dirichlet判别法就有 $\dis\int f(x)\Big[ g(x)-A \Big]dx$ 收敛
    - 也就是说，只凭一个单调和两个有界即可推出反常积分收敛
- **理解**：只考虑极限处即可，如果积分有界，函数单调有界，则可以用积分第二中值定理化为单函数的积分

### 平方可积

- **平方可积**：若 $\dis\int^{+\infty}_a f^2(x)dx$ 收敛，则称 $f$ 在 $(a,+\infty)$ 上平方可积
  - **平方可积性与条件收敛性无关**
    - **反例**：
      - $f(x)= \dfrac{\sin x}{\sqrt{x}}$ 条件收敛，但不平方可积
        - **证明**：平方消正负
      - $f(x) = \dfrac{1}{x}$ 平方可积，但不条件收敛
        - **证明**：平方是减小
  - **平方可积性与绝对收敛性**
    - **无界函数情况**：平方可积 $\red\Rt$ 绝对收敛
    - **无穷区间情况**：绝对收敛和平方可积无关
      - **反例**：
        - $f(x) = \frac{1}{x}$ 平方可积，但不绝对收敛
          - **证明**：平方是减小
        - 区间极小的无界函数：绝对收敛，但平方不可积
          - **证明**：平方是增大
    - **绝对收敛且连续则平方可积**

### 一致连续性

- 若 $\dis\int^{+\infty}_a f(x)dx$ 收敛，且 $f$ 在 $[a,+\infty)$ 一致连续，则 $\lim\limits_{x\to +\infty}f(x) = 0$
  <font color='chartreuse'>跟前面无穷远处极限为0则导数极限为0是一样的</font>
  - **反证法+柯西收敛原理**
    - 反设一致连续，但极限不为 $0$。则存在 $\e_0 > 0$ 使得 $\forall X>0$，都存在 $x_0>X$ 使得 $|f(x_0)| \geq \e_0$
    - 取足够小的邻域 $O(x_0,\d)$，则由积分上界不等式有 $\dis|\int^{x_0+\d}_{x_0-\d} f(x)dx| > 2\d\e_0$
    - 取 $\e < 2\d\e_0$ 即可由柯西收敛定理发现矛盾
  - **正证法**：一致连续性得极限处的连续性，原函数收敛得其导数（函数）为0（导数定义）
- 若 $\dis\int^{+\infty}_a f(x)dx$ 收敛，且 $\lim\limits_{x\to +\infty}f(x) = A$，则 $A = 0$
  - **证明（定义法）**：
    - 反设 $A$ 不为 $0$，则由极限保号性得存在某个 $[G,+\infty)$ 使得 $f(x) > \dfrac{A}{2}$
    - 再由柯西收敛原理或比较判别法即得矛盾
- 设 $f$ 在 $[a,+\infty)$ 可导，且 $\dis\int^{+\infty}_a f(x)dx$ 和 $\dis\int^{+\infty}_a f'(x)dx$ 都收敛，则 $\lim\limits_{x\to +\infty}f(x) = 0$
  - **证明**：
    - 由导数反常积分收敛，可得 $f$ 收敛
    - 再由 $f$ 反常积分收敛，由上题即得结论

- **超周期函数**：$\dis\int^{+\infty}_0 x\sin x^4\sin xdx$ 收敛

  - **理解**：$\sin x^4$ 的周期过小，$x\to\infty$ 时会把其它两个函数都忽略，变成 $asinx^4$ 形式的函数
- **伪 · $p$ 级数比较**：

  - <font color='chartreuse'>明确一点，反常积分要收敛必定在区间上处处收敛（无奇点）</font>
  - **证明（比较判别法）**：
    - 换元 $y = \dfrac{1}{x}$，则 $\lim\limits_{y\to +\infty}g(y) = +\infty$，且 $\dis\int^{+\infty}_1 g(y)dy$ 收敛
    - 反设 $\lim\limits_{y\to +\infty}\dfrac{g(y)}{y} = A > 0$，则由比较判别法，$g$ 的反常积分发散，矛盾
  - **证明（积分上界不等式）**：
    - 易得 $f$ 只能单减，故 $\dis\frac{x}{2}f(x) = \int^x_{\dfrac{x}{2}} f(x)dt \leq \int^x_{\dfrac{x}{2}} f(t)dt$
    - 再由反常积分收敛性得 $右式 < \e$，从而结论成立
    - <font color='chartreuse'>这个方法更好一些，本质是把函数通过积分中值定理放缩为积分，然后由单调性比较积分大小</font>

  - <font color='chartreuse'>这里就发现，p级数比较不是万能的。实际上，对任意 $p\to 1$，都存在收敛速度介于 $\dfrac{1}{x}$ 和 $\dfrac{1}{x^p}$ 之间的函数</font>
  - **证明（比较判别法）**：先证 $\lim\limits_{x\to +\infty} xf(x) = 0$，再证 $\ln x < \dfrac{1}{x^p}\pad (\forall p>1,x\to +\infty)$ 即可
    - $xf(x)\to 0$：
      - 若 $f(x)>0$，则不会跳跃。再由积分收敛，其必须收敛于0，否则 $f(x)\sim \frac{k}{x}$ 积分不收敛
      - 若 $f(x)<0$，则 $xf(x)$ 不收敛于0，则$ |f(x)|\sim\frac{k}{x}$ 积分不收敛。若跳跃，跳跃幅度也不会太大
