# 不定积分

- **原函数**：若区间 $I$ 上有 $F'(x) = f(x)$，则称 $F$ 是 $f$ 在 $I$ 上的原函数
  - **不唯一性**：存在参量 $C$
- **不定积分**：$f$ 的全体原函数称为 $f$ 的不定积分 $\int f(x)dx$
  - <font color='chartreuse'>一般来说，全体原函数的表达式之间只相差一个常数 $C$
  - 但是也有可能出现原函数无法用表达式写出的情况</font>
  - **被积函数**：$f$
  - **积分变量**：$x$
- **性质**：
  - **线性**：由导数线性易得

## 初等积分方法

- **第一类换元积分法（拼凑法）**：<font color='chartreuse'>通过删去被积函数的部分因子，将其化为简单形式</font>
  - 若存在 $g$ 使得 $f(x) = \wt f\Big[ g(x) \Big]g'(x)$，其中 $\wt f$ 的积分易求
  - 那么设 $u = g(x)$，则积分结果为 $\wt F\Big[ g(x) \Big] + C$
  - **实例**：
    - $\int f(2x) dx = \dfrac{1}{2}\int f(u)du = \dfrac{1}{2}F(2x)$
- **第二类换元积分法（展开法）**：<font color='chartreuse'>通过将 $x$ 展开，来将被积函数化为简单形式</font>
  - 若存在 $\p$ 使得 $f(x) = \wt f\Big[ \p(t) \Big]\p'(t)$，其中 $f(\p(t))\p'(t)$ 的积分易求
  - 那么设 $x=\p(t)$，则积分结果为 $\wt F\Big[ \p^{-1}(t) \Big] + C$
  - **实例**：
    - **三角换元**：$\int \sqrt{a^2-x^2}dx$ 中，设 $x = a\sin t$ 即可化简被积函数
- **分部积分公式**：$$\int uv' dx = uv - \int vu'dx$$<font color='chartreuse'>前项是只积不导的结果。后项是积一个、导一个的结果</font>
  - **证明**：逆用导数的乘法公式即可

### 习题

- **垃圾题**：设 $p(x)= \sum\limits_{i=0}^{n}a_ix^i$，若系数满足 $\sum\limits_{i=1}^n \dfrac{a_i}{(i-1)!} = 0$，则 $\int{p(\frac{1}{x})e^xdx}$ 是初等函数
  - **证明**：
    - <font color='chartreuse'>不难看出应当循环使用积分来配凑出题设系数等式</font>
    - 不断分部积分，可将右式化为 $\dis q_k(x)+ \sum\frac{(-1)^k}{(i-1)(i-2)...(i-k)}\int \frac{e^x}{x^k}dx$ 的形式
    - 最后由系数条件，可以把余项（即无初等函数积分的项）给消掉

## 有理函数的不定积分

- **因式分解定理（实根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x) = (x-\a)^kq_1(x)\pad (\a\in\R)$
  - 则存在实数 $\l$ 与多项式 $p_1(x)$ 使得 $$\frac{p(x)}{q(x)} = \frac{\l}{(x-\a)^k} + \frac{p_1(x)}{(x-\a)^{k-1}q_1(x)}$$
  - **证明**：
    - <font color='chartreuse'>我们希望转化为 $p(x) = p_1(x)(x-\a) + \l q_1(x)$ 的形式</font>
    - 由带余除法（零点分解公式）易得 $p(x) = p_1(x)(x-\a) + p(\a)$
    - <font color='chartreuse'>这启发我们，是否可以设 $\l = \cfrac{p(\a)}{q_1(\a)}$ 呢？我们来验证一下</font>
      - 由带余除法得 $p(x) = \cfrac{p(\a)}{q_1(\a)}q_1(x) + r(x)$
      - 代入 $x=\a$ 易得 $r(\a) = 0$，即 $(x-\a)\mid r(x)$，从而得到结论
- **因式分解定理（复根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x)$ 有 $l$ 重共轭复根 $\a = \b+i\g$，余式为 $q^*(x)$
  - 则存在实数 $\mu,\nu$ 和多项式 $p^*$ 使得 $$\frac{p(x)}{q(x)} = \frac{\mu x+\nu}{\Big[ (x-\a)(x-\ol\a) \Big]^l} + \frac{p^*(x)}{\Big[ (x-\a)(x-\ol\a) \Big]^{l-1}q^*(x)}$$
  - **证明**：
  - **实例**：
    - $x^4+1$ 的四个共轭复根为 $e^{\dfrac{\pi}{4}i},e^{\dfrac{3\pi}{4}i},e^{\dfrac{5\pi}{4}i},e^{\dfrac{7\pi}{4}i}$
      - 它可因式分解为 $(x^2+\sqrt{2}x+1)(x^2-\sqrt{2}x-1)$
- **推论**：
  - 有理函数的不定积分都是初等函数
    - **证明**：上面两个因式分解定理的结果都是初等可积的
  - 三角函数的不定积分都是初等函数
    - **证明**：通过万能诱导公式，都可把三角函数化为 $\tan \dfrac{x}{2}$ 的形式，此时 $$\int R(\sin x,\cos x)dx = \int R(\frac{2t}{1+t^2},\frac{1-t^2}{1+t^2})\frac{2}{1+t^2}dt$$ 是有理函数积分，从而初等可积

## 求不定积分

- 正文里说的很明白了，有理函数的不定积分本质上只有一个解决办法：裂项。任何换元法，本质上都是待定系数法来裂项。

### 基础积分

- **分部积分更本质**：$\dis\int \sqrt{x^2\pm a^2}dx = \frac{1}{2}\Big[ x\sqrt{x^2\pm a^2} \pm a^2\ln|x+\sqrt{x^2+a^2}| \Big]$
  - **解（换元法）**：先换元 $x = a\tan t$，变形后换元 $u = \sin t$
  - **解（分部法）**：直接分部积分，然后解方程
  - <font color='chartreuse'>对于这道题来说，分部积分更好，更容易理解和记忆
  - 因为最后的结果中，前项就是分部积分的前项，后项就是分部积分的后项</font>

### 多项式分式

- **通分换元法**：
  - $\dis\int \frac{\sqrt{x^4+1}}{x} = \frac{x\sqrt{x^4+1}}{x^2}$
  - $\dis\int \frac{x^{14}dx}{(x^5+1)^4}$
- **添项裂项法**：
  - $\dis\int \frac{dx}{x(x+1)...(x+n)} = \frac{1}{n!}\sum^n_{k=0}(-1)^kC^k_n\ln (x+k)$
    - **证明**：
      - 待定系数法，设系数为 $A_k$，则 $A_0(x+1)...(x+n)+\cdots = 1$
      - 取 $x=-k$ 即得 $A_k = \cfrac{1}{n!}(-1)^kC^k_n$
- **裂项配凑法**：
  - $\dis\int\frac{x}{x^2-x+1}dx$
    - **解**：
      - 裂项得 $\dis\frac{1}{2}\int\frac{2x-\frac{1}{2}}{x^2-x+1}dx + \int \frac{1}{(x-\frac{1}{2})^2+\frac{3}{4}}dx$
      - <font color='chartreuse'>此时前者换元 $u = x^2-x$ 即可，后者换元 $u = x-\frac{1}{2}$ 即可</font>
- **高阶裂项**：
  - **例1**：$\dis\int\frac{1}{x^3+1}$
    - **解**：
      - 通过共轭复根易得可因式分解为 $(x+1)(x^2-x+1)$
      - 待定系数法得 $\dis\frac{1}{3}\int\frac{dx}{x+1} - \frac{1}{3}\int \frac{x-2}{x^2-x+1}dx$
  - **例2**：$\dis\int\frac{dx}{x^4+1}$
    - **解**：
      - $M = \dis\int \frac{x^2+1}{x^4+1}dx = \int\frac{1+\frac{1}{x^2}}{x^2+\frac{1}{x^2}}dx = \int\frac{d(x-\frac{1}{x})}{(x-\frac{1}{x})^2+2}$
        - <font color='chartreuse'>本质也是裂项配凑，只不过加了通分这一步</font>
      - $N = \dis\int \frac{x^2-1}{x^4+1}dx = \int\frac{1-\frac{1}{x^2}}{x^2+\frac{1}{x^2}}dx = \int\frac{d(x+\frac{1}{x})}{(x+\frac{1}{x})^2-2}$
        - <font color='chartreuse'>同上</font>
      - 裂项易得 $\dis\int\frac{dx}{x^4+1} = \frac{M-N}{2}$
        - <font color='chartreuse'>这种做法比待定系数法简单很多</font>
  - **例3**：$\dis\int\frac{dx}{x^6+1}$
    - **解**：
      - <font color='chartreuse'>这种复杂度，基本不愿意用待定系数法了</font>
      - 通过共轭复根易得 $x^6+1 = (x^2+1)(x^4-x^2+1)$
        - <font color='chartreuse'>偶数次项可以用平方差公式，这是一个大优势</font>
      - 裂项易得 $\dis\int\frac{dx}{x^6+1} = \frac{1}{2}\int\frac{x^4+1}{x^6+1}dx - \frac{1}{2}\int\frac{x^4-1}{x^6+1}dx$
      - 再对前后项分别处理即可
        - <font color='chartreuse'>前项：化为 $\cfrac{(x^4-x^2+1) + x^2}{x^6+1}$，裂项约分即可
        - 后项：平方差公式，约分即可</font>

### 根式（裂项和换元结合）

- **直接换元法**：$\dis\int R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx$，换元 $t = \sqrt[n]{\cfrac{ax+b}{cx+d}}$ 即可
- **配方换元法**：$\dis\int R(x,\sqrt{ax^2+bx+c})dx$，配方后换元 $u = (x+\dfrac{a}{2})$ 即可
- **三角换元法**：$\dis\int R(x,\sqrt{a+x},\sqrt{b-x})dx$，换元 $\begin{cases} a+x = (b-a)\sin^2t \\ b-x = (b-a)\cos^2 t\end{cases}$，再用三角函数方法即可
- **分步换元法**：$\dis\int R(x,\sqrt{a+x},\sqrt{b+x})dx$
  - **解**：
    - 设 $u = \sqrt{a+x}$，则转化为 $R(u^2-a,u,\sqrt{u^2+(b-a)})$
    - **三角换元法**
      - 再设 $u = \sqrt{b-a}\tan^2t$，则转化为 $R(\tan^2 t-a,\tan t,\dfrac{\sqrt{b-a}}{\cos t})$
      - 最后由万能公式，即转化为有理函数的不定积分
    - **齐次换元法**：
      - 再设 $\sqrt{u^2+(b-a)} = t+u$，易得 $u = \cfrac{b-a-t^2}{2t}$，则转化为有理函数
- **换元裂项法**：把根式变成多项式，或变成复合三角函数式
  - **添项裂项**：$$\frac{x^2}{\sqrt{1+x-x^2}} = \frac{x^2}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}} \\\ \\ = \sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}} + \frac{x-\frac{1}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}} + \frac{\frac{3}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}}$$，这三项的解法都已知
  - **转化为齐次根式分式**：$\Large\frac{1}{\sqrt[3]{(x-2)(x+1)^2}} = \frac{1}{x+1}\sqrt[3]{\frac{x+1}{x-2}}$。令 $t = \sqrt[3]{\lfrac{x+1}{x-2}}$ 即可
- **奇葩**：
  - $\dis\int \frac{dx}{\sqrt[4]{1+x^4}}$
    - **解**：
      - 换元 $x = \sqrt[\large 4]{\cfrac{1}{t^4-1}}$
  - $\dis\int \frac{dx}{\sqrt[n]{1+x^n}}$
    - **解**：
      - 同上换元，可转化为 $\dis (1-n)\int\frac{t^{n-2}}{t^n-1}dt$
  - $\dis\int \frac{dx}{\sqrt[n]{1-x^n}}$
    - **解**：
      - 换元 $x = \sqrt[\large n]{\cfrac{1}{t^n+1}}$

### 指数对数

- **部分分部法**：
  - $\dis\int e^x\Big( \frac{1-x}{1+x^2} \Big)^2dx$
    - <font color='chartreuse'>看到这类题，首先想到拆括号分解成多项。然后再想到指数相关积分肯定要分部积分（出题人不会难为你的，肯定是存在一个形式非常好的函数）</font>
    - **解**：拆括号后，化作 $\dis\int \frac{e^x}{1+x^2} dx$ 和 $\dis\int \frac{e^x}{1+x^2}d\Big( \frac{1}{1+x^2} \Big)$ 两项，发现它们可以相消
  - $\dis\int e^{\sin x} \frac{x\cos^3x - \sin x}{\cos^2x} dx$
    - <font color='chartreuse'>看到后面这个大分式，首先尝试将大分式直接化为分部积分，但积不出，于是先线性拆项再尝试分部积分</font>
    - **解**：化作 $\dis\int xd(e^{\sin x})$ 和 $\dis\int e^{\sin x}d(\frac{1}{\cos x})$<font color='chartreuse'>（一定要牢记三角函数的微分和积分公式）</font>，发现它们可以相消
- **配对积分法**：
  - $\dis\int e^{ax}\sin bx$ 与 $\dis\int e^{ax}\cos bx$
    - **解**：分别对两个分部积分，得到二元一次方程组，解出答案
  - $\dis\int \sin (\ln x)dx$ 与 $\dis\int \cos (\ln x)dx$
    - **解**：同上
- **复数法**：
  - $\dis\int e^{ax}\sin bx$ 与 $\dis\int e^{ax}\cos bx$
    - **解**：已知 $e^{(a+bi)x} = e^{ax}(\cos bx + i\sin bx)$，故只需求 $\dis\int e^{(a+bi)x}dx$，然后比较实虚部即可
- **指数归一化**：$\dis\int \frac{6^x}{9^x-4^x}dx$
  - **解**：化为 $(\dfrac{3}{2})^x$ 的函数，然后换元 $u = (\dfrac{3}{2})^x$ 积分即可

## 抽象不定积分

- **导数法表示抽象积分**：设 $p_n(x)$ 是 $n$ 次多项式，则 $$\dis\int \frac{p_n(x)}{(x-a)^{n+1}}dx = \sum\limits^n_{k=0} \frac{p^{(k)}(a)}{k!}\int \frac{dx}{(x-a)^{n-k+1}}$$

- **隐函数的不定积分**：设 $(\dfrac{y}{x})^2(x-y)=1$，求 $y(x)$ 的原函数
  - **解**：
    - 设 $\cfrac{y}{x} = t$，则 $\begin{cases} x = \dfrac{1}{t^2(1-t)} \\\\ y=\dfrac{1}{t(1-t)} \end{cases}$
    - 此时可以写出原函数的参数形式 $Y(t) = \dis\int y(t)x'(t)dt = \ln t + \ln (1-t) + C$<font color='chartreuse'>（至于显函数形式，需要求 $x(t)$ 的反函数，比较难）</font>
