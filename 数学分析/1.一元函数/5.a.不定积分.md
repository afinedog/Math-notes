# 不定积分

- **原函数**：若区间 $I$ 上有 $F'(x) = f(x)$，则称 $F$ 是 $f$ 在 $I$ 上的原函数
  - **不唯一性**：存在参量 $C$
- **不定积分**：$f$ 的全体原函数称为 $f$ 的不定积分 $\int f(x)dx$
  - **被积函数**：$f$
  - **积分变量**：$x$
- **性质**：
  - **线性**：由导数线性易得

## 初等积分方法

- **第一类换元积分法（拼凑法）**：<font color='chartreuse'>通过删去被积函数的部分因子，将其化为简单形式</font>
  - 若存在 $g$ 使得 $f(x) = \wt f\Big[ g(x) \Big]g'(x)$，其中 $\wt f$ 的积分易求
  - 那么设 $u = g(x)$，则积分结果为 $\wt F\Big[ g(x) \Big] + C$
  - **实例**：
    - $\int f(2x) dx = \dfrac{1}{2}\int f(u)du = \dfrac{1}{2}F(2x)$
- **第二类换元积分法（展开法）**：<font color='chartreuse'>通过将 $x$ 展开，来将被积函数化为简单形式</font>
  - 若存在 $\p$ 使得 $f(x) = \wt f\Big[ \p(t) \Big]\p'(t)$，其中 $f(\p(t))\p'(t)$ 的积分易求
  - 那么设 $x=\p(t)$，则积分结果为 $\wt F\Big[ \p^{-1}(t) \Big] + C$
  - **实例**：
    - **三角换元**：$\int \sqrt{a^2-x^2}dx$ 中，设 $x = a\sin t$ 即可化简被积函数
- **分部积分公式**：$$\int uv' dx = uv - \int vu'dx$$<font color='chartreuse'>前项是只积不导的结果。后项是积一个、导一个的结果</font>
  - **证明**：逆用导数的乘法公式即可
  - **实例**：
    - **指数-三角循环**：$\int e^x\sin x dx$

### 习题

- **垃圾题**：设 $p(x)= \sum\limits_{i=0}^{n}a_ix^i$，若系数满足 $\sum\limits_{i=1}^n \dfrac{a_i}{(i-1)!} = 0$，则 $\int{p(\frac{1}{x})e^xdx}$ 是初等函数
  - **证明**：
    - 首先看出系数条件就是用来积分的，而且是单项式积分。所以首先把p(x)线性分割
    - 分部积分得 $\int e^xf(x)dx = e^x\int f(x)dx - \int e^x(\int f(x)dx)dx$，可以化为 $q_k(x)+\int \frac{e^x}{x}dx$
    - 最后由系数条件，把最后一个无初等函数积分的项给消掉

## 有理函数的不定积分

- **因式分解定理（实根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x) = (x-\a)^kq_1(x)\pad (\a\in\R)$
  - 则存在实数 $\l$ 与多项式 $p_1(x)$ 使得 $$\frac{p(x)}{q(x)} = \frac{\l}{(x-\a)^k} + \frac{p_1(x)}{(x-\a)^{k-1}q_1(x)}$$
  - **证明**：
    - <font color='chartreuse'>目的是分解出可以积分的项，并且降次</font>
    - $\Large\frac{p(x)}{q(x)} = \frac{\frac{p(\alpha)}{q_1(\alpha)}}{(x-\alpha)^k} + \frac{p_1(x)}{(x-\alpha)^{k-1}q_1(x)}$
      - 由加法构造的自由性，设 $p(x) = \frac{p(\alpha)}{q_1(\alpha)}q_1(x)+(x-\alpha)p_1(x)$，然后同时除一个$q(x)$
      - 本质就是利用加法构造，分离出一个 $() + (x-\alpha)·()$
      - 接下来具体的式子利用分式隔离性和根与因式的关系得到。而自由度由没有明确定义的$p_1(x)$撑着
- **因式分解定理（复根）**：
  - 设真有理函数为 $\dfrac{p(x)}{q(x)}$，$q(x)$ 有 $l$ 重共轭复根 $\a = \b+i\g$，余式为 $q^*(x)$
  - 则存在实数 $\mu,\nu$ 和多项式 $p^*$ 使得 $$\frac{p(x)}{q(x)} = \frac{\mu x+\nu}{\Big[ (x+\a)(x-\a) \Big]^l} + \frac{p^*(x)}{\Big[ (x+\a)(x-\a) \Big]^{l-1}q^*(x)}$$
  - **证明**：
- **推论**：
  - 有理函数的不定积分都是初等函数
    - **证明**：上面两个因式分解定理的结果都是初等可积的
  - 三角函数的不定积分都是初等函数
    - **证明**：通过万能诱导公式，都可把三角函数化为 $\tan \dfrac{x}{2}$ 的形式，此时 $$\int R(\sin x,\cos x)dx = \int R(\frac{2t}{1+t^2},\frac{1-t^2}{1+t^2})\frac{2}{1+t^2}dt$$ 是有理函数积分，从而初等可积

## 求不定积分

- 正文里说的很明白了，有理函数的不定积分本质上只有一个解决办法：裂项。任何换元法，本质上都是待定系数法来裂项。

### 多项式分式

- **通分换元法**：
  - $\dis\int \frac{\sqrt{x^4+1}}{x} = \frac{x\sqrt{x^4+1}}{x^2}$
  - $\dis\int \frac{x^{14}dx}{(x^5+1)^4}$
    - 实际上用循环分部积分也能做出来
- **添项裂项法**：
  - $\dis\int\frac{1}{x^3+1} = \frac{1}{3}\int\frac{(x^2-x+1)-(x-2)(x+1)}{(x+1)(x^2-x+1)}$
    - <font color='chartreuse'>奇数次幂可直接分解，偶数次需要用平方差公式</font>

- **常用的奇葩微分**：
  - $M = \dis\int \frac{x^2+1}{x^4+1}dx = \int\frac{1+\frac{1}{x^2}}{x^2+\frac{1}{x^2}}dx = \int\frac{d(x-\frac{1}{x})}{(x-\frac{1}{x})^2+2}$
    - <font color='chartreuse'>分式</font>
  - $N = \dis\int \frac{x^2-1}{x^4+1}dx = \int\frac{1-\frac{1}{x^2}}{x^2+\frac{1}{x^2}}dx = \int\frac{d(x+\frac{1}{x})}{(x+\frac{1}{x})^2-2}$
    - <font color='chartreuse'>分式</font>
  - 则 $\dis\int\frac{dx}{x^4+1} = \frac{M-N}{2}$
### 根式（裂项和换元结合）

- **直接换元法**：$\dis\int R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx$，换元 $t = \sqrt[n]{\cfrac{ax+b}{cx+d}}$ 即可
- **配方换元法**：$\dis\int R(x,\sqrt{ax^2+bx+c})dx$，换元 $\sqrt{ax^2+bx+c} = \sqrt{a}x+t$ 即可
- **换元裂项法**：把根式变成多项式，或变成复合三角函数式
  - 添项裂项：$\Large\frac{x^2}{\sqrt{1+x-x^2}} = \sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}} + \frac{x-\frac{1}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}} + \frac{\frac{3}{2}}{\sqrt{-(x-\frac{1}{2})^2+\frac{5}{4}}}$，然后三角换元即可
  - t最好是齐次根式分式：$\Large\frac{1}{\sqrt[3]{(x-2)(x+1)^2}} = \frac{1}{x+1}\sqrt[3]{\frac{x+1}{x-2}}$。令$t = \sqrt[3]{\frac{x+1}{x-2}}$即可

### 三角函数

- **思路**：
  - 若含有平方项，则考虑倍角诱导公式
  - 若只有一次项，则考虑万能公式或升幂公式
- **万能公式法**：
  - - $\dis\int\frac{dx}{a+b\sin x}$ 类型
    - **解**：用 $t = \tan\cfrac{x}{2}$ 代换
    - **实例**：$$\int\frac{dx}{1+\sin x+\cos x} = \int\frac{\dfrac{1}{\cos^2\frac{x}{2}}}{2\tan\frac{x}{2}+2}dx = \int\frac{d(\tan\frac{x}{2})}{2\tan\frac{x}{2}+2}dx$$
- **直接换元法**：$$\int\frac{dx}{1+\sin x+\cos x} = \int\frac{d(x+\frac{\pi}{4})}{1+\sin (x+\frac{\pi}{4})}$$
- **观察换元法**：
  - $\dis\int\frac{dx}{a+b\sin^2x}$ 类型 $\LR \dis\int\frac{dx}{a\sin^2 x + b\cos^2 x}$ 类型
    - **解**：用 $t = \cfrac{1}{\tan x}$ 代换
    - **实例**
  - $\dis\int\frac{dx}{a\sin x + b\cos x + c}$ 类型
    - **解**：倍角公式化为 $\dfrac{x}{2}$ 的形式，然后发现容易换元为 $\tan\dfrac{x}{2}$（但不是万能公式）
    - **实例**：
- **待定系数法**：$$\int \frac{a\sin x+b\cos x}{c\sin x + d\cos x} = \int \frac{k_1(c\sin x+d\cos x)+k_2(c\cos x-d\sin x)}{c\sin x+d\cos x} \\\ \\ = \int k_1dx + \int \frac{d(c\sin x+d\cos x)}{c\sin x+d\cos x}$$
- **和差化积法**：
  - $\dis\int \cos(ax)\cos(bx)dx$
  - $\dis\int\frac{dx}{\sin(x+a)\cos(x+b)} = \frac{1}{\cos(a-b)}\int\frac{\cos[(x+a)-(x+b)]}{\sin(x+a)\cos(x+b)}dx$，拆解后裂项即可
- **换元法 + 诱导公式**：$$\dis\int\frac{1}{\sin^3x+\cos^3x}dx = \frac{1}{\sqrt{2}}\int\frac{1}{\cos(\frac{\pi}{4}-x)[\frac{1}{2}+\sin^2(\frac{\pi}{4}-x)]}$$

### 指数对数

- **部分分部法**：
  - $\dis\int e^x\Big( \frac{1-x}{1+x^2} \Big)^2dx$
    - <font color='chartreuse'>看到这类题，首先想到拆括号分解成多项。然后再想到指数相关积分肯定要分部积分（出题人不会难为你的，肯定是存在一个形式非常好的函数）</font>
    - **解**：拆括号后，化作 $\dis\int \frac{e^x}{1+x^2} dx$ 和 $\dis\int \frac{e^x}{1+x^2}d\Big( \frac{1}{1+x^2} \Big)$ 两项，发现它们可以相消
  - $\dis\int e^{\sin x} \frac{x\cos^3x - \sin x}{1+x^2} dx$
    - <font color='chartreuse'>看到后面这个大分式，首先尝试将大分式直接化为分部积分，但积不出，于是先线性拆项再尝试分部积分</font>
    - **解**：化作 $\dis\int xd(e^{\sin x})$ 和 $\dis\int e^{\sin x}d(\frac{1}{\cos x})$<font color='chartreuse'>（一定要牢记三角函数的微分和积分公式）</font>，分部积分发现可以相消
- **配对积分法**：
  - $\dis\int e^{ax}\sin bx$ 与 $\dis\int e^{ax}\cos bx$
    - **解**：分别对两个分部积分，得到二元一次方程组，解出答案
  - $\dis\int \sin (\ln x)dx$ 与 $\dis\int \cos (\ln x)dx$

## 抽象不定积分

- **导数法表示抽象积分**：设 $p_n(x)$ 是 $n$ 次多项式，则 $$\dis\int \frac{p_n(x)}{(x-a)^{n+1}}dx = \sum\limits^n_{k=0} \frac{p^{(k)}(a)}{k!}\int \frac{dx}{(x-a)^{n-k+1}}$$
- 设 $R(u,v,w)$ 是三个变量的有理函数，求 $\dis\int R(x,\sqrt{a+x},\sqrt{b+x})dx$
  - **解**：只需通过换元，将三个自变量化为有理函数即可
- **隐函数的不定积分**：设 $(\dfrac{y}{x})^2(x-y)=1$，求 $y(x)$ 的原函数
  - **解**：
    - 设 $\cfrac{y}{x} = t$，则 $\begin{cases} x = \dfrac{1}{t^2(1-t)} \\\\ y=\dfrac{1}{t(1-t)} \end{cases}$
    - 此时可以写出原函数的参数形式 $Y(t) = \dis\int y(t)dt = \ln t + \ln (1-t) + C$<font color='chartreuse'>（至于显函数形式，需要求 $x(t)$ 的反函数，比较难）</font>
