# 级数

## 数项级数

- **级数**：可列个实数的和 $\sum\limits^\infty_{n=1}x_n$
  - **实例**：
    - $p$ 级数
    - 调和级数
- **部分和数列**：$S_n = \sum\limits^n_{k=1} x_k$
  - **实例**：
    - 等比级数的部分和数列为 $S_n = x_1\dfrac{1-q^n}{1-q}$
- **余和数列**：$r_n = \sum\limits^\infty_{k=n+1}x_k$
- **级数的收敛性**：若 $\lim\limits_{n\to\infty}S_n$ 存在，则称级数收敛
- **收敛级数的性质**：
  - **线性收敛性**：两个收敛级数的线性组合依然收敛
  - **加法结合律**：两个收敛级数满足 $\sum\limits^\infty_{n=1} x_n +\sum\limits^\infty_{n=1} y_n = \sum\limits^\infty_{n=1} (x_n+y_n)$
    - **证明**：易得收敛级数的子列均收敛
- **反例**：发散级数不满足结合律，比如 $\sum\limits^\infty_{n=1}(-1)^n$ 和 $\sum\limits^\infty_{n=1} (-1)^{n+1}$
  - **理解**：结合律本质是改变其终点

### 上下极限

- **极限点**：子列的极限
- **确界极限点定理**：极限点集合 $E$ 的确界均可取到
  - **证明**：
    - 若 $E$ 是有限集，则确界就是最大值
    - 若 $E$ 是无限集，则
      - 由定义，对每个极限点 $A_k$，都存在一个子列 $x_{n_k}\to A_k$
      - 在该子列中取一点 $x^{(k)}$，由 $E$ 无限性，可以一直取下去
      - 由于 $\{x^{(n)}\}$ 单增有上界，故收敛于 $E$ 的上确界 $H$
      - 由于 $\{x^{(n)}\}$ 也是子列，故 $H$ 也是极限点，即 $H\in E$（**证毕**）
    - <font color='chartreuse'>这是高阶分析课程的常用方法：无穷个横向列组成的二维点列中，按照某种规律取纵向列</font>
- **上极限**：极限点集合的上确界
- **极限点收敛定理**：有界数列 $x_n$ 收敛 $\LR x_n$ 的上下极限相等
  - **证明**：讨论子列即可
- **上下极限的 $\e-\d$ 定义**：
  - 若
    - $\forall\e >0，\exist N$，对 $\forall n>N$ 都有 $x_n < H+\e$，
    - 且存在无穷多项满足 $x_n > H-\e$
  - 则 $H$ 是 $x_n$ 的上极限
    - <font color='chartreuse'>第一条保证最值性，第二条保证收敛性（可取性）</font>
- **上下极限的性质**
  - **三角不等式**：$\overline{x_n+y_n} \leq \overline{x_n} + \overline{y_n}$
    - **证明**：利用极限点的 $\e-\d$ 定义即可
    - **理解**：不相等的情况就是极限最大的子列彼此错开
      - 比如 $A_{1n}>A_{2n}$，$-B_{1n}和B_{2n}$，则 $a_n+b_n \to A_{2n}+B_{2n}$，但它们不是两个数列的上极限之和
  - **收敛等式**：若 $\lim\limits_{n\to\infty} x_n$ 存在，则 $\overline{x_n+y_n} = x_n+\overline{y_n}$
    - **证明**：$\overline{y_n}$ 添项即可
    - **理解**：收敛数列只有一个极限点，所以两个极限最大的子列必定一一对应
- **上下极限的确界定义**
  - 设 $x_n$ 有界，$b_n = \sup\{x_{n+1},x_{n+2},...\}，a_n = \inf\{x_{n+1},x_{n+2},...\}$
    - <font color='chartreuse'>$n$ 越大，$b_n$ 可选择的值越少，从而 $b_n$ 越小，即 $b_n$ 单调递减。同理 $a_n$ 单调递增</font>
  - 则由单调有界收敛定理，上下极限可定义为 $H^* = \lim\limits_{n\to\infty} b_n，h^* = \lim\limits_{n\to\infty} a_n$
  - **证明**：只需证 $H^*$ 是 $x_n$ 的最大极限点
    - **最大性**：由 $a_n$ 和 $b_n$ 定义可得
    - **可取性**：由 $x_n$ 收敛，得 $b_{n_k}-\e_0 < x_{n_{k+1}} \leq b_{n_k}$，夹逼得 $x_n$ 可以取到 $H^*$

### 习题

- 若 $\lim\limits_{n\to\infty} x_n = x\in (-\infty,0)$，则 $$\ol{\lim\limits_{n\to\infty}}(x_ny_n) = \lim\limits_{n\to\infty} x_n \cdot \lim\limits_{\ol{n\to\infty}} y_n$$
  - **证明**：定义+放缩易得，$\e-\d$ 语言就不写了，尝试更简单的方法
    - 首先易得，左式一定等于 $x$ 乘上某个 $y_n$ 的极限点。设极限点集合为 $E_y$
    - 然后显然有公式 $\sup (xE_y) = -|x|\cdot \inf E_y$
    - 所以就得到了结论

### 正项级数 + 收敛判别法

- **正项级数收敛**：此时部分和数列 $S_n$ 单增，故 $S_n$ 有上界 $\red\Rt x_n$ 收敛
- **比较判别法**
- **Cauchy判别法**：设**幂极限公比** $\bar r(C) = \overline{\lim\limits_{n\to\infty}}\sqrt[n]{x_n}$
  - 若 $r<1$，则收敛
  - 若 $r>1$，则发散
  - 若 $r=1$，则判别法失效
  - **证明**：将 $x_n$ 和等比级数比较即可
  - **反例**：应用于指数函数和幂函数，$\frac{1}{n}$ 失效
- **d'Alembert判别法**：
  - 若**除法上极限公比** $\bar{r}(d) = \overline{\lim\limits_{n\to\infty}} \cfrac{x_{n+1}}{x_n} < 1$，则级数收敛
  - 若**除法下极限公比** $\underline{r}(d) = \underline{\lim\limits_{n\to\infty}} \cfrac{x_{n+1}}{x_n} > 1$，则级数发散
  - 其余情况判别法失效
  - <font color='chartreuse'>依然是和等比级数的公比比较，不过需要区分上下极限。所以弱于Cauchy判别法。同时在大多数情况下它也被Rabbe判别法完爆，除了某些极限难求但上下极限容易求，且恰好收敛的数列</font>
  - **引理**：$\underline{r}(d) \leq \underline{r}(C) \leq \overline{r}(C) \leq \overline{r}(d)$
    - **证1**：上极限定义 $\frac{x_{n+1}}{x_n} \leq \bar{r}(d)+\e$
    - **证2**：与等比数列比较
  - **证明**：
  - **反例**：对于 $x_n = \dfrac{1}{2^n} + \dfrac{1}{3^n}$ 失效
- **Raabe判别法**：设 $r = \lim\limits_{n\to \infty} n(\cfrac{x_n}{x_{n+1}}-1)$
  - 若 $r>1$，则收敛。若 $r<1$，则发散
  - <font color='chartreuse'>比较（级数的减少率）和 $\frac{1}{n}$ 的阶数关系</font>
  - **证明**：易得 $\forall s>t>1$，都存在 $(0,\d)$ 成立 $1+sx > (1+x)^t$
    - 则当 $r>1$ 时，取 $1<t<s<r$ 就有 $\dfrac{x_n}{x_{n+1}} > 1+\dfrac{s}{n} > \dfrac{(n+1)^t}{n^t}$
    - 即 $\{n^tx_n\}$ 单调递减有上界，从而收敛
    - 再由 $p$ 级数比较即得 $x_n$ 收敛
  - **反例**：对于 $p$ 级数和 $x_n = \dfrac{1}{n\ln^q n}$ 时失效
- **Bertrand判别法**：$r = \lim\limits_{n\to \infty} \ln n\Big[ n(\cfrac{x_n}{x_{n+1}}-1)-1 \Big]$ 
  - <font color='chartreuse'>比较（级数的减少率）和 $\cfrac{1 + \frac{1}{\ln n }}{n} > \cfrac{1}{n}$ 的关系
  - 接下来更高级的判别法，也无非是将（级数减少率）与更低阶的无穷小量比较而已</font>
- **积分判别法**：
  - 设 $a_n$ 单增趋于正无穷
  - 若 $u_n = \dis\int^{a_{n+1}}_{a_n} f(x)dx$，则反常积分 $\dis\int^{+\infty}_af(x)dx$ 和正项级数 $\sum\limits^\infty_{n=1} u_n$ 同时收敛或发散
  - **证明**：由积分区间不等式+夹逼法则即可
  - **推论**：
    - 若 $f(x)$ 单减，则取 $a_n = n$，接着可以取 $u_n = f(n)$
    - 若 $f(x)$ 单增，则无穷远处区间无限小，和数列一样了
  - **应用**：
    - 正用：判断反常积分收敛<font color='chartreuse'>（其实使用区间可加性可以达到同样的效果，区别只是多了级数收敛判别法而已）</font>
    - 逆用：判断数列收敛。根据 $u_n$ 构造函数

### 习题

- **周期积分简化为级数**：$\dis\int^\infty_0 \frac{dx}{1+x^2\sin^2x}$ 发散
  - **证明**：
    - **三角函数取端点**：设 $u_n = \dis\int^{(n+1)\pi}_{n\pi} \frac{dx}{1+x^2\sin^2x}$
    - 换元 $x=n\pi+t$，则 $u_n = \dis\int^\pi_{0} \frac{dt}{1+(n\pi + t)^2\sin^2t}$
    - 取 $t$ 足够小，则极限下分母 $<n$ ，$p$ 级数比较即得发散
  - <font color='chartreuse'>如果用反常积分的方法，则需要在无穷远处进行比较。但由于 $\sin x$ 的周期性，不存在一个可以良好比较的被积函数。
  - 但是取级数的话，就可以通过周期性，保持原函数不变的同时，将无穷区间简化为有界区间。从而将周期震荡函数变为非周期函数，这就是本题的意义。以后遇到周期函数都可以这么简化</font>
- $\dis\int^\infty_0 \frac{dx}{1+x^4\sin^2x}$ 收敛
  - **证明**：
    - 取端点 + 换元同上，不过这次放缩时需要讨论区间
    - $t\to 0$ 时，分为两种情况
      - $t\leq\dfrac{1}{n^{3/2}}$ 时，被积函数不收敛，但积分区间极小，故 $\to 0$
      - $t>\dfrac{1}{n^{3/2}}$ 时，$p$ 级数比较得被积函数收敛
    - $t$ 有界（$t \in[B,\pi]$）时，被积函数等价于 $\dfrac{1}{n^4}$ 收敛
- **重要比较**：$\sum\limits^\infty_{n=1}\dfrac{1}{n\ln^q n}$ 在 $q > 1$ 时收敛，在 $q\leq 1$ 时发散
  - **解**：
    - 化为函数，易得单调递减。解出反常积分即可得结论

### 任意项级数 + 收敛判别法

- **交错级数**：设 $u_n$ 是正项级数，则形如 $x_n = (-1)^{n+1}u_n$ 的级数都称为交错级数
- **Leibniz级数**：若交错级数的 $u_n$ 单减趋于 $0$，则称为莱布尼茨级数
- **Leibniz判别法**：莱布尼茨级数收敛
  - **证明**：加法结合律
  - **实例**：易得 $\sin(\sqrt{n^2+1}\pi) = (-1)^n\sin(\sqrt{n^2+1}-n)\pi$，右式是莱布尼茨级数，从而收敛
- **Abel变换**：设 $a_n,b_n$ 是两数列，$B_n$ 是 $b_n$ 的部分和数列，则 $$\sum^p_{k=1} a_k(B_k-B_{k-1}) = a_pB_p - \sum^{p-1}_{k=1} (a_{k+1}-a_k)B_k$$
  - **证明**：直接变形即可。本质就是把和差因子给颠倒了
- **Abel引理（简单放缩）**：若 $a_k$ 单调，$\forall |B_k|\leq M$，则 $$|\sum^p_{k=1} a_kb_k| \leq M(|a_1|+2|a_p|)$$
  - **证明**：
    - 左式由abel变换 + 三角不等式 + 上界放缩得 $\leq |a_pM| + |\sum\limits^{p-1}_{k=1} (a_{k+1}-a_k)M|$
    - 再由三角不等式即得 $\leq$ 右式
- **Abel判别法**：若 $a_n$ 单调有界，$B_n$ 收敛，则 $\sum\limits^\infty_{n=1}a_nb_n$ 收敛
  - **证明**：
    - 由 $B_n$ 收敛得存在上下界。由 $a_n$ 单调有界得其收敛。故由Abel引理 + $a_n$ 收敛性即得 $\sum\limits^\infty_{n=1} a_nb_n$ 的余和 $r_n\to 0$，从而收敛
  - <font color='chartreuse'>实际上我们发现根本用不着 $B_n$ 收敛，只需要其有界即可
  - 同时，$a_n$ 单调有界等价于 $a_n-\sup a_n$ 单调趋于0
  - 也就是说，Abel判别法是Dirichlet判别法的特例</font>
- **Dirichlet判别法**：$a_n$ 单调趋于0，$\{B_n\}$ 有界，则 $\sum\limits^\infty_{n=1}a_nb_n$ 收敛
  - **证明**：由Abel引理 + $a_n$ 收敛性，同上即得结论
- **绝对收敛**：定义同反常积分
- **条件收敛**：定义同反常积分
- **正项衍生列判别法**：设 $x_n^+ = \dfrac{|x_n|+x_n}{2}$，其正项不变，负项变为0
  - $x_n$ 绝对收敛 $\LR x_n^+$ 收敛
  - $x_n$ 条件收敛 $\LR x_n$ 收敛，$x_n+$ 发散
  - **证明**：利用收敛级数的线性即可
- **更序级数**：
  - **借贷性**：由于级数有无限项，所以可以把任意个后面的项借到前面来。根据借来的项的多少，可能会改变一些结果
  - 若 $x_n$ 绝对收敛，则其更序级数均收敛
    - **证明**：
      - 若为正项级数，则易得 $\forall n$ 都有 $S_n' \leq \lim\limits_{k\to\infty} S_k$
        - 两边都对 $n$ 取极限，由极限保序性得 $\lim\limits_{n\to\infty}S_n' \leq \lim\limits_{n\to\infty} S_n$
        - 反过来同理可得 $\lim\limits_{n\to\infty}S_n \leq \lim\limits_{n\to\infty} S_n'$，从而相等
        - <font color='chartreuse'>实变函数中也使用了这样的方法，即有限项不等式两侧取无穷，以及在发散列不等式的两侧取确界</font>
      - 若为任意项级数，分为正负衍生列即可套用上面的结论
  - **求更序级数**：
    - 首先写出更序的部分和数列 $S_n'$
    - 然后将其写为原部分和数列的线性组合 $f(S_n)$
    - 利用已知 $S_n$ 的值来求 $S_n'$ 即可
  - **实例**：
    - 调和级数的更序级数：把它们化成调和级数的差即可 $$\sum\limits^n_{k=1}(\frac{1}{4k-3} + \frac{1}{4k-1} - \frac{1}{2k}) = \sum\limits^{4n}_{k=1} \frac{1}{k} - \frac{1}{2}\sum\limits^{n}_{k=1} \frac{1}{k} - \frac{1}{2}\sum\limits^{2n}_{k=1} \frac{1}{k}$$
    - （考虑 $4$ 的完系，配凑调和级数即可）
- **Riemann级数定理（更序级数全集性）**：若 $x_n$ 条件收敛，则 $\forall a\in [-\infty,+\infty]$，都存在更序级数收敛到 $a$
  - **证明**：排列两个衍生列即可
    - 易得存在 $n_1$ 使得 $\sum\limits^{n_1}_{k=1}x_k^+ > a$
    - 易得存在 $m_1$ 使得 $\sum\limits^{n_1}_{k=1}x_k^+ - \sum\limits^{m_1}_{k=1} x_k^- < a$
    - 再取 $n_2>n_1，m_2>m_1$ 使得级数和摆动在 $a$ 的两侧
    - 不断做下去，最终得到一个更序级数
  - **理解**：如果是条件收敛，虽然依然是无穷小，但累加起来无界。此时衍生列发散，不同的借贷方式可以改变结果

        - 那么由上面的结论，即得 $\sum\limits^\infty_{n=1} x_n = \sum\limits^\infty_{n=1} f(\dfrac{1}{n})$ 收敛

### 级数的乘法

- **排列方式**：满足 $\sum\limits^\infty_{n=1}c_n = (\sum\limits^\infty_{n=1} a_n)(\sum\limits^\infty_{n=1}b_n)$ 的 $c_n$ 不唯一。每个这样的 $c_n$ 都表示了 $\{a_n\}$ 和 $\{b_n\}$ 的一种排列方式
  - **更序性**：$c_n = a_n'b_n$，其中 $\{a_n'\}$ 是 $a_n$ 的更序级数
  - **收敛性**：级数 $a_n,b_n$ 绝对收敛时，它们的所有乘积排列方式均收敛
    - **证明**：
      - 由乘积排列的更序性 + 更序级数绝对收敛的传递性，$a_n'$ 也绝对收敛
        - 从而存在 $N_1$ 使得 $\forall n>N_1$ 都有 $|a_n'| < 1$
      - 再由 $b_n$ 绝对收敛性 + 柯西收敛原理，存在 $N_2>N_1$ 使得 $\forall m,n>N_2$ 都有 $|a_m'b_m + \cdots + a_n'b_n| < |b_m + \cdots + b_n| < \e$（**证毕**）
- **三角型排列**，$\sum\limits^\infty_{i=1}\sum\limits^i_{j=1}a_ib_j$
  - <font color='chartreuse'>它只能取到一半的 $a_ib_j$，即 $(1\leq i\leq j\leq n)$ 而不是 $(1\leq i,j\leq n)$</font>
- **Cauchy乘积（对角线排列）**：$\sum\limits^\infty_{n=1}\Big[ \sum\limits_{i+j=n+1} a_ib_j \Big]$
  - <font color='chartreuse'>它可以取到所有的 $a_ib_j$</font>
  - **二项式性**：$$(\sum \frac{x^n}{n!})(\sum \frac{y^n}{n!}) = \sum \frac{(x+y)^n}{n!}$$
    - **证明**：
      - 易得左式级数绝对收敛，故柯西乘积收敛，从而只需对柯西乘积证明该结论即可
      - 它就是 $e^x\cdot e^y = e^{x+y}$ 的幂级数展开形式，按照柯西乘积排列即得结论
  - **等比级数平方**：$$\Big[\sum\limits^\infty_{n=0} q^n\Big]^2 = \sum\limits^\infty_{n=0} (n+1)q^n$$
    - **证明**：
      - 易得等比级数收敛，故只需对柯西乘积证明即可
      - 易得柯西乘积为 $\sum\limits^\infty_{n=0} \sum\limits^n_{k=0} q^n = 右式$
  - **最小性**：若Cauchy乘积收敛，则绝对收敛
    - <font color='chartreuse'>因为Cauchy乘积走的是斜线，如果 $\sum a_ib_j$ 是任意项级数的话，对角线就是正负衍生列，是相消程度最小的绝对值</font>
  - （类似前面的 $\lim\limits_{n\to\infty}\frac{a_1b_n+...+a_nb_1}{n}$ 的）
- **正方形排列**：$\sum\limits^\infty_{n=1}\Big[ \sum\limits^n_{i=1} a_ib_n + \sum\limits^n_{j=1} a_nb_j - a_nb_n\Big]$  
  - **收敛性**：若 $a_n,b_n$ 收敛，则正方形排列也收敛。且有 $\sum d_n = (\sum a_n)(\sum b_n)$
    - **证明**：用 $S_n$ 即可
    - **理解**：因为它遍历所有 $a_ib_j$，同时并不完全遍历每一行 $a_i\sum\limits^\infty_{j=1}b_j$ 和每一列 $b_j\sum\limits^\infty_{i=1} a_i$

#### 反例

- $a_n,b_n$ 收敛，但它们的柯西乘积不收敛
  - **反例**：$a_n = b_n = \dfrac{(-1)^{n+1}}{\sqrt{n}}$

### 无穷乘积

- **无穷乘积**：可列个数的乘积
- **部分积数列**：$P_n$
  - **有界**：若 $p_n\to 1$，则 $P_n$ 有界
  - **收敛**：$P_n$ 收敛于非零的数 $P$
  - **发散**：$P_n$ 发散，或 $P_n$ 收敛于 $0$
- **Wallis公式**：设 $p_n = 1-\frac{1}{(2n)^2}$，则部分积数列为 $P_n = \cfrac{\Big[ (2n-1)!! \Big]^2}{\Big[ (2n)!! \Big]^2} \cdot (2n+1)$
  - **证明**：
    - 设 $I_n = \dis\int^{\frac{\pi}{2}}_0 \sin^nxdx$，则由之前结论可得 $\cfrac{\pi}{2}P_n = \cfrac{I_{2n}}{I_{2n+1}}$
    - 再由 $I_n$ 收敛于 $1$，所以 $\lim\limits_{n\to\infty}P_n = \prod\limits^\infty_{n=1}\Big[ 1-\cfrac{1}{(2n)^2} \Big] = \cfrac{2}{\pi}$
  - 最终得到 $$\frac{\pi}{2} = \Big[ \frac{2}{1}\cdot\frac{2}{3} \Big]\cdot\Big[ \frac{4}{3}\cdot\frac{4}{5} \Big]\cdots \Big[ \frac{2n}{2n-1}\cdot\frac{2n}{2n+1} \Big]$$
- **Viete公式**：
  - 设 $p_n = \cos\frac{x}{2^n}$
  - 对 $\sin x$ 不断应用倍角公式得 $\dis\sin x = 2^n\cos\frac{x}{2} \cdot \cos\frac{x}{2^2}\cdots\cos\frac{x}{2^n}\cdot \sin\frac{x}{2^n}$
  - 变形取极限即得 $\prod \cos\frac{x}{2^n} \to \dfrac{\sin x}{x}$
  - 最后令 $x = \dfrac{\pi}{2}$ 即可得到 $$ \frac{2}{\pi} =  \cos\frac{\pi}{4}\cdot\cos\frac{\pi}{8}\cdots \cos\frac{\pi}{2^n}$$
- **$\pi$ 的级数表示法**：
  - 取 $\arcsin(x)$ 的泰勒展开式，令 $x=1$

#### 无穷级数与无穷乘积

- **对数等价性**：$\prod p_n$ 收敛 $\LR \sum \ln p_n$ 收敛
  - **证明**：即 $P_n$ 收敛 $\LR \ln P_n$ 收敛。由数列极限的知识知其成立
- **常用对数代换**：
  - 若 $a_n$ 不变号，则 $\prod (1+a_n)$ 收敛 $\LR \sum a_n$ 收敛
    - **证明**：泰勒展开 + 比较判别法易得
  - 若 $a_n$ 变号，但 $\sum a_n$ 收敛，则 $\prod (1+a_n)$ 收敛 $\LR \sum a^2_n$ 收敛
    - **证明**：泰勒展开 + 比较判别法易得
  - **理解**：
    - 由泰勒公式得 $\ln(1+a_n) = a_n - \frac{1}{2}a_n^2 + o(a_n^2)$
      - 由 $a_n$ 是无穷小量，故误差也是无穷小量，从而泰勒公式收敛
    - 和反常积分不同，级数绝对收敛 $\supsetneqq$ 级数平方收敛
    - $ln(1+x)$ 并不等于 $x$，它还有泰勒余项。所以还需要考虑平方收敛条件
  - **反例**：
    - 在 $a_n \sim \dfrac{1}{n^p}\pad (p<\dfrac{1}{2})$ 时，还会有 $a_n$ 和 $a^2_n$ 都发散，但无穷乘积 $\prod(1+a_n)$ 收敛的情况 $$\begin{cases} a_{2n-1} & = & -\frac{1}{\sqrt{n}} \\ a_{2n} & = & \frac{1}{\sqrt{n}} + \frac{1}{n}(1+\frac{1}{\sqrt{n}}) \end{cases}$$
    - 因为泰勒余项是 $a^2_n$，所以 $\frac{1}{2}$ 处的平方项正好消去了多余的项

#### 习题

- **绝对值形式**：$\prod(1+|p_n|)$ 收敛 $\LR$ $\sum \ln p_n$ 绝对收敛
  - **证明**：由上面对数代换直得
- **Stirling公式**：$n! \sim \sqrt{2\pi}n^{n+\frac{1}{2}}e^{-n}$
  - **证明**：
    - 首先设 $b_n = \cfrac{n!}{\sqrt{2\pi}n^{n+\frac{1}{2}}e^{-n}}$，$a_n$ 是 $b_n$ 的增长率

      - <font color='red'>由于 $(2n)!! = (n!)2^n$，故 $\dfrac{(n!)^2}{(2n)!!} = \dfrac{(n!)}{(2n-1)!!}\cdot\dfrac{(n!)}{(2n)!!} = \dfrac{(n!)}{(2n-1)!!}\cdot\dfrac{1}{2^n} = \dfrac{(2n)!}{(2n-1)!!}\cdot\dfrac{1}{4^n}$，</font>

## 函数项级数

- **函数项级数**：一列函数的和 $\sum\limits^\infty_{n=1} f_n(x)$
  - **序列意义**：函数项级数可看作每项都包括未知量 $x$ 的级数，
  - **函数意义**：函数项级数也可看作是多元函数 $f:\R\times\N\to \R，(x,n)\mapsto f_n(x)$ 关于 $\N$ 的离散定积分
  - <font color='chartreuse'>所以说本章其实已经涉及到了二重极限的理念</font>

### 两种收敛性

- **函数项级数的图像**：
- **点态收敛**：
  - **定义**：
    - 若 $\forall \e>0，\exist N(\e,x_0) > 0$，对 $\forall n>N(\e,x_0)$ 都有 $|S_n(x)-S(x)| < \e$
    - 则称函数项级数在 $x_0$ 处收敛
    - <font color='chartreuse'>函数项级数在每个非极限点上收敛（当趋于某点的阶数确定时，也称该点为非极限点）</font>
  - **收敛点**：若 $x=x_0$ 时收敛，则称 $x_0$ 为收敛点
  - **收敛域**：收敛点组成的集合
    - **部分和函数 $S_n(x)$**：保留 $x$ 未知性的基础上，对 $n$ 求和
    - **极限函数（也称为和函数）**$S(x)$：取单个极限的二元函数。它只能定义在收敛域上
      - <font color='chartreuse'>定积分具有消元性，求和也是如此。对 $n$ 无穷加和（取极限）后，$n$ 就不再作为一个参量出现在 $S(x)$ 中，也就是 $S_n(x)$ 退化成了一元函数</font>
- **一致收敛**：
  - **定义**：
    - 若 $\forall \e>0，\exist N(\e) > 0$，对 $\forall n>N(\e)$ 和 $\forall x\in D$，都有 $|S_n(x) - S(x)| < \e$
    - 则称函数项级数在 $D$ 内一致收敛
    - <font color='chartreuse'>函数项级数在任意点都收敛（即使是极限阶数未知的点）
    - 函数意义：二重极限的值与两个自变量 $x$ 与 $y$ 彼此的阶数无关
    - 收敛性虽然还是关于 $n$ 的，但此时 $x$ 也可以取极限，所以实际是讨论二重极限是否存在</font>
  <!-- - **一致收敛必定收敛于点态收敛的极限函数** -->
- **Heine定理**：
  - 设 $S_n(x)$ 在 $D$ 上点态收敛
  - 则 $S_n$ 一致收敛 $\LR$ 对任意 $D$ 中点列 $\{x_n\}$ 都成立 $\lim\limits_{n\to\infty} \Big[ S_n(x_n)-S(x_n) \Big] = 0$
  - **证明**：和函数极限、一致连续的Heine定理相同
- **奇点**：若 $x\to x_0$ 的阶数和 $n\to+\infty$ 的阶数不同时 $S(x)$ 也不同，则称 $x_0$ 为函数项级数的奇点
- **内闭一致收敛**：若函数项级数在开区间 $D$ 内不一致收敛，但在任何闭区间 $I\subset D$ 上都一致收敛，则称在 $D$ 上内闭一致收敛
  - **实例**：$S_n(x) = x^n，x\in [0,1)$
    - **理解（Heine定理）**：
      - 此时对非极限点 $x_0$ 都有 $S(x_0)=0$，故 $S(x) = \lim\limits_{n\to\infty} S_n(x) = 0$
        - 从而 $|S_n(x) - S(x)| = x^n <\e$，点态收敛
      - 但如果取点列 $x_n = 1-\dfrac{1}{n}$，则在其极限点处 $|S_n(\xi)-S(\xi)| = \lim\limits_{n\to\infty}(1-\dfrac{1}{n})^n = \dfrac{1}{e} > \e$，故不一致收敛
      - 但是如果取任意的闭区间，则相当于把 $x$ 变成了非极限点，所以内闭一致收敛

### “极限点”勘误

- **点态性质**：对非极限点成立的性质
  - **实例**：$p$ 级数比较法对 $\R$ 点态成立，但不一致成立，因为 $p_n\to 1+$ 时导数定义失效
- **一致性质**：对极限点也成立的性质
  - **实例**：一致有界、等度连续[（详见泛函分析）](../../泛函分析/1.度量空间.md)、一致连续、一致收敛
  - <font color='chartreuse'>其实我在这里所称的“极限点”是很容易招人误解的。正如“无穷小量不是一个具体存在的数，而是对阶数的一个衡量标准”一样，极限点也并不是真正存在的点。比如上面取 $p_n = 1+\dfrac{1}{n}$ 时，它的极限是 $1$，故我们此时只能取 $p=1$，而不是一个无限接近于 $1$ 的点。
  - 严谨的说法有两种：
    - 第一个是一致收敛定义的描述，即不依赖于（集合中该元素自身属性）的性质，比如一致收敛不依赖于 $x_0$ 的值
    - 第二个就是Heine定理的描述，对于任意点列的极限都成立该性质
  - 提出这个刁钻的“一致性”，是因为点态连续、点态收敛等性质已经无法满足我们的需求了
    - 因为“一致成立”还意味着“两个量是独立关系”，比如一致连续对应 $x$ 与 $\d$ 在 $f(x)$ 连续时可以相互独立，一致收敛对应 $x$ 与 $n$ 在 $S_n(x)$ 收敛时可以相互独立
    - 后面在概率论中可以学到，独立往往意味着非常良好的性质，比如交换律、结合律、分配律等
    - 在这里提出的一致收敛性就是为了得到“交换律”。即交换求极限、交换求积分、交换求导数等方便我们求级数的性质。这是点态性质给不了的</font>

### 判别一致收敛

- 已知部分和函数 $S_n(x)$ 时
  - **观察讨论法**：
    - **实例**：$S_n(x) = \dfrac{x}{1+n^2x^2}$，则
      - $x$ 为无穷小量时，受分子影响 $S_n(x) \to 0$
      - $x$ 为无穷大量时，分母是分子的高阶无穷大，则 $S_n(x) \to 0$
      - $x$ 为有界量时，受分母影响 $S_n(x) \to 0$
      - 综上，$S(x)$ 始终都是 $0$，从而在 $\R$ 上一致收敛
  - **级数收敛法**：把 $x$ 当作未定常数，直接使用数项级数的收敛判别法<font color='chartreuse'>（此时需要讨论 $x$ 相对于 $n$ 的阶数）</font>
    - **实例**：$|S_n(x) - S(x)| = \cfrac{1}{\frac{1}{x} + n^2x} \leq \cfrac{1}{2n}$
  - **求极值法**：
    - **引理**：
      - 若 $S_n(x)$ 点态收敛，且 $d(S_n,S) = \sup|S_n(x) - S(x)|$
      - 则 $S_n(x)$ 一致收敛 $\LR \lim\limits_{n\to\infty}d(S_n,S) = 0$
    - **证明**：$\e-\d$ 语言即可
    - **应用**：
      - 把 $n$ 当成未定常数，将 $S_n(x)$ 对 $x$ 求偏导，得到关于 $n$ 的极值点 $x(n)$，
      - 查看极值点处的 $\lim\limits_{n\to\infty} d$ 是否为0即可
      - <font color='chartreuse'>因为极值点是最有可能不收敛的点。若极值点都收敛，则其它点肯定不会超出 $S(x)$ 太多</font>
  - **Heine定理法**：实际上就是求极值法的反证用法
- 已知级数的通项公式 $f_n(x)$ 时
  - **级数法**
    - **Cauchy收敛原理**：
      - $\sum\limits^\infty_{n=1} f_n(x)$ 在 $D$ 上一致收敛 $\LR \forall \e>0，\exist N(\e)>0$ 使得 $\forall m,n>N(\e)$ 有 $|u_m(x)+\cdots + u_n(x)| < \e$
      - **证明**：和数项级数的方法相同
    - **Weierstrass判别法**：
      - 设数项级数 $a_n$ 绝对收敛
      - 若 $\forall x\in D，\forall n$ 都有 $|f_n| \leq a_n$，则 $\sum\limits^\infty_{n=1} f_n(x)$ 在 $D$ 上一致收敛
      - **证明**：由柯西收敛原理直得
      - **应用（比较判别法）**：对于形式比较简单的 $f_n(x)$，直接比较判别
      - **应用（求极值法）**：对于形式不太简单的 $f_n(x)$，可以先求极大值 $f_n(x(n)) = u_n$<font color='chartreuse'>（起到一个简化的作用）</font>，然后再针对 $u_n$ 应用数项级数的收敛判别法
        - <font color='chartreuse'>若结果是一致收敛，就是应用了比较判别法
        - 若结果是不一致收敛，就是应用了Heine定理</font>
    - **Heine定理法**：
      - **反证用法**：不必多说
      - **正证用法**：证明内闭一致收敛时有奇效
  - **判别法**：
    - **Abel判别法**：
      - 若 $a_n(x)对于\forall x\in D$ 单调且一致有界，$\sum b_n(x)$ 一致收敛
      - 则 $\sum\limits^\infty_{n=1} a_n(x)b_n(x)$ 一致收敛
      - **证明**：和数项级数方法相同
    - **Dirichlet判别法**：
      - 若 $a_n(x)$ 在 $D$ 上单调且一致收敛到 $0$，$\sum b_n(x)$ 在 $D$ 上一致有界
      - 则 $\sum\limits^\infty_{n=1} a_n(x)b_n(x)$ 一致收敛
      - **证明**：和数项级数方法相同
    - **应用**：固定 $x$ 后讨论 $n$，再由一致性得均成立（均收敛）

### 一致收敛的性质

- **连续性定理**：
  - 若闭区间 $I$ 中 $S_n(x)$ 的每一项连续，且 $S_n(x)$ 一致收敛
  - 则 $S(x)$ 在 $I$ 上连续
  - **证明**：
    - $S(x) \stackrel{收敛}{\rightarrow} S_n(x) \stackrel{连续}{\rightarrow} S_n(x+\delta) \stackrel{收敛}{\rightarrow} S(x+\delta) \stackrel{一致性}{\rightarrow} 推广到整个区间$
    - 用 $\e-\d$ 语言将上面的思路说明一下即可
  - **推论**：若条件改为一致连续，则结论也可改为一致连续
  - **极限可交换性**：$\lim\limits_{x\to x_0}\lim\limits_{n\to\infty} S_n(x) = \lim\limits_{n\to\infty}\lim\limits_{x\to x_0} S_n(x)$
    - 先对 $n$ 求极限，后对 $x$ 求极限 $\LR$ 先对 $x$ 求极限，再对 $n$ 求极限
    - <font color='chartreuse'>累次极限存在只需要一致收敛（阶数无关）即可，点态收敛和连续性是极限存在的基本条件</font>
  - **应用（逐项求和）**：$\lim\limits_{x\to x_0}\sum\limits^\infty_{n=1} u_n(x) = \sum\limits^\infty_{n=1}\lim\limits_{x\to x_0} u_n(x)$
    - 先极限 $x$，后加和 $n\LR$ 先加和 $n$，后极限 $x$
    - <font color='chartreuse'>因为反正收敛了，意味着无限项的$\e$加和仍然不会发散，所以自由度就高了</font>
- **积分可交换性**：
  - 若闭区间 $I$ 中 $S_n(x)$ 每一项都连续，且 $S_n(x)$ 一致收敛
  - 则 $S(x)$ 可积，且 $\lim\limits_{n\to\infty}\Big[ \int S_n(x)dx \Big] = \int \Big[ \lim\limits_{n\to\infty} S_n(x) \Big]dx$
  - **证明**：
    - 左右两式作差，由积分三角不等式 + 积分上界不等式即得结论
  - **应用（逐项积分）**：
    - 先积分 $x$ 后加和 $n \LR$ 先加和 $n$ 后积分 $x$
    - 实际上交换性只需要满足收敛性即可，强调一致性是为了将交换性推广到整个区间
    - 左边是先x轴积分后n轴积分，右边是先n轴积分后x轴积分
- **可导性定理**：
  - 设闭区间 $I$ 上 $S_n(x)$ 点态收敛于 $S(x)$<font color='chartreuse'>（目的是给出 $S(x)$ 的定义）</font> 
  - 若
    - $S_n(x)$ 存在连续的导函数<font color='chartreuse'>（要求导函数连续是为了配合一致收敛得出 $f$ 可积性）</font>
    - $S_n'(x)$ 一致收敛于 $f(x)$<font color='chartreuse'>（目的是应用积分可交换性，以及得到 $f$ 连续性）</font>
  - 则 $S(x)$ 可导，且 $S'(x) = f(x)$
  - **证明**：
    - 由 $S'_n(x)$ 连续性 + 一致收敛性，其极限函数 $f(x)$ 连续，从而可积
    - 取上限函数，应用积分可交换性 + N-L公式即得 $\dis\int^x_a f(x)dx = S(x)-S(a)$
      - 由积分可导性即得 $S(x)$ 可导性
      - 两端求导得 $S'(x) = f(x)$
  - **导数可交换性**：$S'(x) = \dfrac{d}{dx}\Big[ \lim\limits_{n\to\infty} S_n(x) \Big] =  \lim\limits_{n\to\infty} S_n'(x) = f(x)$
  - **应用（逐项求导）**
- **反例（一致收敛只是充分条件）**：
  - $S_n(x) = \cfrac{nx}{1+n^2x^2}$ 在 $[0,1]$ 上不一致收敛，但满足逐项求极限和逐项可积性
  - **理解**：在 $x$ 部分与 $n$ 部分互为等价量时，$S_n(x)$ 在极限点处的值不固定，但内闭一致收敛
- **反例（极限与求导不可交换）**：
  - $S_n(x) = \cfrac{x}{1+n^2x^2}$ 在 $[-1,1]$ 上不一致收敛，且不满足交换求导性
- **Dini定理（类似Cantor定理）**：设 $I$ 是闭区间，若 $S_n(x)$ 在其上满足以下条件，则一致收敛
    - $S_n$ 点态收敛
    - $S_n(x)$ 和 $S(x)$ 均连续
      - <font color='chartreuse'>要求连续性是为了在两个函数之间传递 $\e$
      - 因为Cantor定理只需要处理一个函数，而Dini定理需要处理两个函数，故多了这个条件</font>
    - $S_n(x)$ 关于 $n$ 单调
      - <font color='chartreuse'>此时由单调有界收敛定理，$S(x)$ 就是 $S_n(x)$ 的确界，从而 $|S_n(x)-S(x)|$ 关于 $n$ 单调递减
      - 需要注意的是，收敛的正项级数不一定单调，即为了得出上一句的结论，单调性是必不可少的条件
      - 要求单调性是为了
      - 因为Cantor定理处理的是一元函数，而Dini定理的 $S_n(x)$ 实际上是二元函数，故多了这个条件</font>
  - **证明**：
    - **连续性**：首先用实数基本定理压缩成一个点，再用连续性得到无穷小量 $\e$
      - $S_n(x)$ 连续只能是有限项连续，即 $S_N(x)$ 连续
    - **单调性**：
      <!-- - 取点列 $\{x_k\}$ 最终得到的点并不是定点，而是极限点，因此不满足点态收敛性 -->
      - 当我们取了很多定点时，它们每个都具有不同的 $N_k(x_k,\e)$。为了一致性，需要把它们统一起来
      - 通常的想法就是取最大的 $N_k$。遗憾的是，收敛的正项数列不一定是单调的。
      - 所以必须要有单调性条件，才能得到 $|S_{\max N_k}(x_k)-S(x_k)| \leq \forall |S_{N_k}(x_k)-S(x_k)| < \e$
    - **凝聚定理证明**：
      - 反设 $S_n(x)$ 不一致收敛，则由Heine定理，存在 $\e_0$ 和一个点列 $x_n$ 满足 $\forall N>0，\exist n_0>N$ 使得 $|S_{n_0}(x_{n_0})-S(x_{n_0})| > \e_0$
        - <font color='chartreuse'>由单调性，该结论可改为 $\exist N>0，\forall n>N$ 都有 $|S_{n}(x_n)-S(x_n)| > \e_0$</font>
      - 由凝聚定理，$x_n$ 存在子列 $x_{n_k}\to x_0$
        - 由点态收敛性，$\exist N(x_0,\e)|$ 使得 $|S_N(x_0)-S(x_0)| < \e$
        - 再由 $S_n(x)，S(x)$ 的连续性 + 子列 $x_{n_k}$ 的收敛性即得 $\lim\limits_{k\to\infty}|S_N(x_{n_k})-S(x_{n_k})| < \e$。这与前面单调性的结论矛盾
    - **闭区间套证明**：反设不一致收敛，用二分法不断取不一致收敛的闭区间，同时不断增加 $n$ 的值
      - 由闭区间套定理，它最终收缩于一个点，但其与点态收敛性+单调性矛盾，故不成立
    - **有限覆盖定理证明**：
      - 由点态收敛性，可对一列点取邻域 $O(x_k,\d_k(x,\e))$ 使得其中所有点都满足 $|S_{N_k}(x)-S(x)| < \e$
        - <font color='chartreuse'>其中 $N_k$ 是单增且依赖于 $x,\e$ 的数列</font>
      - 由于闭区间 $I$ 长度有限，故取足够多点的情况下，这些邻域可以形成闭区间 $I$ 的开覆盖。由有限覆盖定理，其中存在一个有限的子覆盖
      - 由有限性，可选取 $N = \max\{N_k\}$，显然它只依赖于 $\e$
      - 由单调性，$|S_N(x)-S(x)| \leq \forall |S_{N_k}(x)-S(x)| < \e$，从而得到一致收敛性
  - **反例**：
    - 缺少单调条件：$f_n(x) = \begin{cases} (\sin\dfrac{\pi}{x})^2 & x\in (\dfrac{1}{n+1},\dfrac{1}{n}) \\ 0 & 其它情况 \end{cases}$ 在 $[0,1]$ 上不一致收敛
    - 缺少连续条件：$S_n(x) = \dfrac{1-n^2x^2}{1+n^2x^2}$ 在 $[0,1]$ 上不一致收敛
      - <font color='chartreuse'>实际上 $S(x)$ 存在就蕴含了点态收敛性，因为 $S(x)$ 的定义就是点态极限……所以随便想一个不一致收敛情况，基本都是因为 $S_n(x)$ 和 $S(x)$ 不连续</font>
  - **条件充分性**：不满足Dini定理条件，但依然一致收敛
