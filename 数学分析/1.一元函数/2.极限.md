# 极限

- 所有的极限证明题，几乎都要用到实数系基本定理，因为整个数分都构建在这些定理上
- 这些定理的作用一个是引出无穷小量 $\e$（连续函数很需要这个引子），一个找到某种规律，从而确定“无界”的确切点（因为不能直接任取一个无界点）（这种规律其实也定义了无穷大的阶数。没有阶数描述的无穷大不能被直接定义/取到）
- 这些定理之间并不是完全可以等价，但也不是毫不相关。在证明中应用它们时，具体要看“概念上的需要”，比如题目的条件是数列还是区间还是确界为主，甚至混合着来

## 数列极限

- **数列极限**：设 $\{x_n\}$ 是一个给定数列，$a$ 是一个实常数
  - 若 $\forall \e，\exist N>0$, 对于 $\forall n>N$，有 $|x_n-a| < \e$，则称**数列收敛于a，或a是数列的极限**
  - **定义式**：xx定义中的等式和不等式以后简称为xx的定义式
  - **极限的两种表示方法**：$\lim\limits_{n\to \infty}x_n = a$ 和 $x_n\to a \quad (n\to \infty)$
  <!-- - **核心**：$\e$ 的有阶任意性 -->
- **邻域**：以 $a$ 为中心，$\e$ 为半径的一个开区间（开集），称为点 $a$ 的 $\e$ 邻域
- **常见极限**：<font color='chartreuse'>用放缩法都能证明，下面只是介绍一些常见放缩思路</font>
  - 分式极限：
  - 指数极限
    - $|q^n| < \e$
    - $|\sqrt[n]{a} - 1| < \e$
      - **证明**：利用二项式定理拆成多项式即可（实际上是应用了伯努利不等式）
        - 设 $y=|\sqrt[n]{a} - 1|$，则 $a = 1+ny +... > 1+ny$，即 $y < \dfrac{a-1}{n} \to 0$，夹逼法即得结论
    - $|\sqrt[n]{n}-1| < \e$
      - **证明1**：利用二项式定理：$n = 1+ny + \dfrac{n(n-1)}{2}y^2 + ... > \dfrac{n(n-1)}{2}y^2$
      - **证明2**：利用均值不等式，$\sqrt[n]{n} = \sqrt[n]{\sqrt{n}\sqrt{n}\cdot 1\cdot 1\cdots} \leq \dfrac{\sqrt{n}+\sqrt{n}+1+1+\cdots}{n} = \dfrac{2}{\sqrt{n}} + 1-\dfrac{2}{n}$
      - **推论**：这样可以一直得到 $|\sqrt[n]{n^p} - 1| < \e$（$p$ 是有限数）
  - 根式极限
    - 平方差配凑
    - $\lim\limits_{n\to\infty} (\sqrt{n^2+n+1} - \dfrac{n^2+5n+1}{n+3})$
  - 奥特曼极限：
    - $\sqrt[n]{a_1^n + a_2^n + ... + a_p^n} = \max\{a_i\}$
      - **夹逼证明**：
        - 左式 $\leq \sqrt[n]{n\max a_i^n} = $ 右式
        - 左式 $\geq \sqrt[n]{\max a_i^n} = $ 右式
      - 取对数证明（未完？）
- **无穷消有穷（均值极限）**：若 $\lim\limits_{n\to\infty}a_n = a$，则 $\lim\limits_{n\to\infty}\dfrac{a_1+a_2+...+a_n}{n} = a$
  - **证明**：任取有限值 $N$，分开前 $N$ 项和后 $n-N$ 项后，应用三角不等式实现分离
  - <font color='chartreuse'>三角不等式可以在很多时候进行添项、分离项，然后分隔成两个绝对值式子，可以配凑出想要的结果。比如应用和差化积公式可证 $|\sin x_1-\sin x_2| < |x_1-x_2|$</font>

### 极限的性质

- **唯一性**：收敛数列的极限必定唯一
  - **证明**：
    - 反设不唯一，则 $\forall \e>0，\exist N_1,N_2$，使得 $\forall n>N_1$ 有 $|x_n-a| < \e$，且 $\forall n>N_2$ 有 $|x_n-b| < \e$
    - 则 $\forall n>\max\{N_1,N_2\}$，都有 $|a-b| \leq |x_n-a|+|x_n-b| < \e$
    - 再由 $\e$ 的任意性即得 $a=b$
- **有界性**：收敛数列必定是有界数列（既有上界又有下界）
  - **证明**：
    - 设 $x_n\to a$，则有 $\forall n>N，|x_n-a| < \e$
    - 前 $N$ 项是有限个项，其中可取最大值，且最大值必定是有界量<font color='chartreuse'>（归纳原理的直接推论）</font>
    - 后 $n-N$ 个项是无限项，但都位于 $(a-\e，a+\e)$ 中，故也有界
- **保序性**：设两个收敛数列满足 $x_n\to a<y_n\to b$，则 $\exists N，\forall n>N$ 有 $x_n<y_n$
  - **证明**：
    - 取 $N = \max\{N_1,N_2\}$，则 $|x_n-a| < \e$ 且 $|y_n-b| < \e$
    - 则此时 $x_n \leq a+\e < b-\e \leq y_n$
  - **理解**：利用 $\e$ 极小性
  - **推论**：$<$ 可改为 $\leq$
- **保号性**：若 $\lim\limits_{n\to \infty}x_n = a > 0$，则 $\forall r\in (0,a)$，$\exists N，\forall n>N$ 有 $x_n>r$
  - **证明**：
    - 由极限定义中 $\e$ 的任意性，取 $\e > a-r$ 即得题设结论
- **夹逼性**：设 $\forall n$ 都有 $x_n\leq y_n \leq z_n$，若 $\forall n>N_0$ 有 $x_n\to a，z_n\to a$，则 $y_n\to a$
  - **证明**：由极限定义易得
  - **应用**：一般用于无限求和，两端推导
- **极限的四则运算**
  - 加法：定义 + 三角不等式直得
  - 乘法：添项 + 乘法分配律 + 三角不等式
  - 除法：类似乘法的证明，配凑即可
    - **证明**：$|\dfrac{x_n}{y_n}-\dfrac{a}{b}| = |\dfrac{bx_n-ab - ay_n+ab}{y_nb}| \leq \dfrac{|\max\{|b-a|,|b+a|\}|}{|y_nb|}\e$，再由 $\e$ 的任意性即得结论
  - 根号：$\lim\limits_{n\to \infty}\sqrt{x_n} = \sqrt{\lim\limits_{n\to \infty}x_n}$
    - **证明**：设 $x_n\to a$，则只需证明 $\sqrt{x_n}\to \sqrt{a}$
      - $|x_n-a| = |\sqrt{x_n}-\sqrt{a}|\cdot |\sqrt{x_n}+\sqrt{a}| < \e$。再由 $\sqrt{x_n}+\sqrt{a}$ 是有界量即得结论
  - <font color='chartreuse'>本质都是函数极限的Heine定理罢了</font>

### 习题

- **放缩法**：<font color='chartreuse'>放缩是数学分析的精髓，学会了放缩就是学会了数分</font>
  - **分步放缩**：每一步尽可能地少放缩，最大限度地保留原函数的自由度（比如前面证明无上界时，用未知数 $k$ 当系数，可在完全保留自由度的基础上，实现合并同类型的目的）
  - **部分放缩**：只放缩函数的其中一部分
  - **分段放缩**：若函数在某些区域上单调（$\{\dfrac{1}{n}+5^{-n}\}$），可以对定义域中的不同部分采取不同的放缩
- **变量是哪个**：$\lim\limits_{n\to \infty}x_{n+k} = a$，极限定义中依然是取 $n>N$，因为 $n$ 才是数列的自变量
- 几个跟级数有关的有意思的证明：
  - **等比级数收敛**：设 $a_n>0$，且 $\lim\limits_{n\to\infty}\dfrac{a_{n+1}}{a_n} = q \in (0,1)$，则 $a_n \to 0$
    - **证明**：柯西收敛原理 + 等比数列求和公式即可
  - **等比数列开根**：若 $\lim\limits_{n\to\infty}\dfrac{a_{n+1}}{a_n} = q$，则 $\lim\limits_{n\to\infty}\sqrt[n]{a_n} = q$
    - **证明**：取对数后化为等差数列，柯西收敛原理即可
  - 若 $S_n$ 极限存在，则 $\lim\limits_{n\to \infty} \dfrac{a_1+2a_2+...+na_n}{n} = 0$
    - **作差证法**：$$\frac{a_1+2a_2+...+na_n}{n} - \frac{a_1+2a_2+...+(n+1)a_{n+1}}{n+1} \\\ \\ = \frac{a_1+2a_2+...+na_n-(n+1)a_{n+1}}{n(n+1)}$$
      - 再放缩为 $\dfrac{n(a_1+...+a_n)}{n(n+1)} - \dfrac{a_{n+1}}{n} = 0$ （√）
    - **级数证法（换元证法）**：
      - 原式可转化为部分和 $\dfrac{nS_n - S_{n-1} - ... - S_1}{n}$
      - 设 $\lim\limits_{n\to\infty} S_n = A$，$b_k = A-S_k$
      - 则由连续函数的极限性质，原式可化为 $\lim\limits_{n\to\infty} \dfrac{b_1+...+b_n}{n} = \lim\limits_{n\to\infty} b_n = 0$
  - 若 $S_n$ 极限存在，则 $\lim\limits_{n\to \infty} \sqrt[n]{n!·a_1a_2...a_n} = 0$
    - **证明**：利用上式结论 + 均值不等式或取对数即可
    - <font color='chartreuse'>无穷乘积也是放在无穷级数后面讲的。加法比乘法简单，但结论可类似推广到乘法上</font>
  - **双级数**：设 $a_n\to a，b_n\to b$，则 $\lim\limits_{n\to \infty} \cfrac{a_1b_n+a_2b_{n-1} + ... + a_nb_1}{n} = ab$
    - **证明**：添项分离
      - 易得 $(a_i-a)(b_j-b) = a_ib_j - a_ib - b_ia + ab$
      - 右式是左式取换元 $c_nb_n = ab$
        - <font color='chartreuse'>题设左式的分子 $\sum\limits^n_{i=1}a_ib_j$ 是柯西乘积
        - 再易得 $\cfrac{1}{n}\sum\limits^n_{i=1} a_ib \to ab，\cfrac{1}{n}\sum\limits^n_{i=1}ab_i \to ab，\cfrac{1}{n}\sum\limits^n_{i=1}ab = ab$</font>
        - 故题设等式等价于 $\lim\limits_{n\to\infty}\cfrac{\sum\limits^n_{i=1}(a_i-a)(b_i-b)}{n} = 0$，其分子也是柯西乘积
        - 由均值极限易得该式成立<font color='chartreuse'>（分子前 $N$ 项是有界量，值 $\to 0$。分子后 $N$ 项 $\to 0$，综上即得结论）</font>

### 无穷大量

- **无穷大量**：前面的数列极限值都必须有界，但其实可以扩充它的定义。
- **广义极限**：设正无穷大量的极限为 $+\infty$，负无穷大量的极限为 $-\infty$
- **stolz($\dfrac{\infty}{\infty}$ 型)**：若 $y_n$ 是严增正无穷大量，则 $\lim\limits_{n\to\infty}\cfrac{x_n-x_{n-1}}{y_n-y_{n-1}} = a \red\Rt \lim\limits_{n\to\infty}\cfrac{x_n}{y_n} = a$
  - **证明**：
    - 当 $a=0$ 时，有 $|x_n-x_{n-1}| < \e|y_n-y_{n-1}|$
      - 由 $y_n$ 单增性，右式绝对值可改为括号
      - 由三角不等式，$|x_n-x_{N_1}| \leq |x_n-x_{n-1}| + \cdots + |x_{N_1+1}-x_{N_1}|$
      - 右式放缩得 $|x_n-x_{N_1}| < \e(y_n-y_{N_1})$
      - 即 $|\dfrac{x_n}{y_n}-\dfrac{x_{N_1}}{y_n}| < \e(1-\dfrac{y_1}{y_n}) < \e$
    - 当 $a\neq 0$ 且有限时，作差即可化为上式情况
    - 当 $a=+\infty$ 时，取倒数即可化为上式情况
  - **理解**：
    - 保证 $y_n$ 单增是为了合并成 $\e(y_n-y_N)$
    - 保证 $y_n$ 无穷大是为了无穷消有穷
- **stolz($\dfrac{0}{0}$ 型)**：若 $y_n$ 单减趋于 $0$ 且 $x_n\to 0$，则 $\lim\limits_{n\to\infty}\cfrac{x_n-x_{n-1}}{y_n-y_{n-1}} = a \red\Rt \lim\limits_{n\to\infty}\cfrac{x_n}{y_n} = a$
  - **证明**：只需证明 $a=0$ 的情况即可
  - **证明（$x_n>0$）**：
    - 此时 $\dfrac{1}{x_n}\to +\infty，\dfrac{1}{y_n}$ 单增趋于正无穷
    - 由stolz得 $$\frac{\frac{1}{x_n}-\frac{1}{x_{n-1}}}{\frac{1}{y_n}-\frac{1}{y_{n-1}}} \to 0 \red\Rt \frac{y_n}{x_n}\to 0$$
    - 再由于 $$\frac{\frac{1}{x_n}-\frac{1}{x_{n-1}}}{\frac{1}{y_n}-\frac{1}{y_{n-1}}} = \dfrac{x_n-x_{n-1}}{y_n-y_{n-1}}\cdot\dfrac{y_n}{x_n}\dfrac{y_{n-1}}{x_{n-1}}$$
      - 若 $\dfrac{y_n}{x_n}\to+\infty$，则 $\dfrac{x_n}{y_n}\to 0$，此即为所需结果，得证
      - 若 $\dfrac{y_n}{x_n}$ 有界，再由题设条件 $\dfrac{x_n-x_{n-1}}{y_n-y_{n-1}}\to 0$，则此时 $\dfrac{y_n}{x_n}\to 0$，矛盾
  - **证明（其它情况）**：取绝对值 $|x_n|$，由上面的推导可得 $\dfrac{|x_n|}{y_n}\to 0$，显然它蕴含 $\dfrac{x_n}{y_n}\to 0$
  - **理解**：题目条件中等式的意思是，到了某个很大的 $N$ 时，$x_n$ 的收敛速度已经远远慢于 $y_n$，说明早就收敛完成了，所以取同一个 $n$ 时，$x_n$ 当然也远远小于 $y_n$ 了
- **stolz函数极限**：设 $T>0$ 为常数
  - 若
    - **区间单增**：$\forall x\geq a，g(x+T) > g(x)$
    - **内闭有界**：$f,g$ 在 $[a,+\infty)$ 内闭有界
    - **无穷大量**：$\lim\limits_{x\to\infty}g(x) = +\infty$
  - 则 $$\lim\limits_{x\to\infty}  \frac{f(x+T)-f(x)}{g(x+T)-g(x)} = l \red\Rt \lim\limits_{x\to\infty} \frac{f(x)}{g(x)} = l$$Cauchy

#### 习题

- **陷阱**：$\lim\limits_{n\to \infty} \dfrac{a_1+a_2+...+a_n}{n} = +\infty$
  - **证明**：大体上和普通极限相同，但要注意一点：因为 $n-N<n$，所以需要把前面的 $N$ 项的和放缩到 $\dfrac{G}{2}$，这样后面 $n-N$ 可以多分到 $\dfrac{3G}{2}$
- **双重加和**：设 $0<\lambda<1，\lim\limits_{n\to \infty} a_n = a$，则 $\lim\limits_{n\to \infty}(a_n + \lambda a_{n-1}+..+\lambda^n a_0) = \dfrac{a}{1-\lambda}$

    - 显然其可利用stolz定理，计算即得结论
- **双重加和**：
  - 设
    - $A_n = \mathop{\sum}\limits_{k=1}^n a_k$，当 $n\to\infty$ 时有极限
    - $\{p_n\}$ 为单调递增的正数数列，发散到正无穷
  - 求 $\lim\limits_{n\to \infty} \cfrac{p_1a_1+ ... + p_na_n}{p_n}$
  - **解（换元法变为单重加和）**：

    - 再逆用stolz定理，即可得到 $A_n$

    - <font color='red'>这种题的核心问题在于，$p_n$ 和 $a_n$ 的趋于极小时的项数对应两个不同的 $P,A$，那它们之间的项怎么办呢？
    - 如果 $P < A$，那么 $P$ 右边的项有 $p_n\to 0$，$A$ 左边的项有 $|a_n-a|\to 0$，可以覆盖整个 $1\sim n$

      - 所以我们可以取 $n > f(\e)$，就转化为 $P < A$ 的情况</font>
    

### 收敛准则

- **单调有界数列收敛定理**
- **闭区间套定理**：
  - 若
    - 一列闭区间满足 $[a_{n+1}, b_{n+1}] \subset [a_n,b_n]$
    - $\lim\limits_{n\to \infty}(b_n-a_n) = 0$
  - 则
    - 存在唯一的 $\xi$ 位于所有闭区间中
    - $\lim\limits_{n\to \infty}a_n = \lim\limits_{n\to \infty}b_n = \xi$
  - **证明**：
    - **$\xi$ 存在性**：由闭区间的两端点单调有界，得两极限存在。取 $\xi = \lim\limits_{n\to\infty}a_n$ 即可
    - **$\xi$ 唯一性**：反证两极限相等即可
  - **推论**：如果推广到开区间上，则区间不能是包含关系，必须是两端点的严格大小关系（防止 $\xi$ 最终收敛到区间端点上导致取不到）
  - **应用**：实数集是不可列集
    - **证明（三分法构造闭区间套）**：
      - 反设存在排列 $\R = \{x_1,x_2,...\}$
      - 任取不包含 $x_1$ 的闭区间 $[a_1,b_1]$，则将其三等分，必然有一个不包含 $x_2$，设为 $[a_2,b_2]$
        - <font color='chartreuse'>一般取闭区间套都是二分法，为什么这里是三分呢？
        - 因为防止 $x_2$ 刚好处于 $[a_1,b_1]$ 的中点处，此时两个二分子区间都包含 $x_2$，就找不到 $[a_2,b_2]$ 了</font>
      - 不断做下去，得到闭区间套，收敛到 $\xi$，此时发现 $\xi$ 不可能等于任何实数，矛盾
- **凝聚定理**：有界数列必有收敛子列
  - **子列**：$\{x_n\}$ 是一个数列，定义 $n_1 < n_2 < ... < n_k$，则 $\{x_{n_k}\}$ 形成一个数列，称为其子列
    - **收敛性质**：任意子列收敛 $\LR$ 数列收敛
      - **必要性**：数列本身也是其子列，故其收敛
      - **充分性**：定义易得
  - **证明**：
    - 设数列的上下界为 $B,A$，则取二分查找闭区间套 $$[A_n,B_n] = \begin{cases} [A_{n-1},\dfrac{A_{n-1}+B_{n-1}}{2}] & 若该区间内有无限多项 \\\\ [\dfrac{A_{n-1}+B_{n-1}}{2},B_{n-1}] & 若该区间内有无限多项 \end{cases}$$
    - 再由闭区间套定理，它最终收敛到一个数 $\xi$。在每个区间套中取点，则 $\xi$ 即为该点列的极限
  - **无界数列**：$\exist \{x_{n_k}\}$，使得 $\lim\limits_{k\to \infty}x_{n_k} = \infty$<font color='chartreuse'>（子列无界的数列称为无界数列）</font>
- **Cauchy收敛原理**：$\{x_n\}$ 收敛 $\LR \{x_n\}$ 是基本数列
  - **基本数列**：若 $\forall \e>0，\exist N，\forall m,n>N$ 都有 $|x_m-x_n|<\e$，则 $x_n$ 称为基本数列
  - **证明**：
    - **必要性**：定义 + 三角不等式添项即可
    - **充分性**：
      - 易得基本列有界

#### 单调有界收敛的衍生定理

##### 压缩映射（更一般的形式见泛函分析）

- **压缩映射方法**：
  - 设 $f$ 可微，$x_{n+1} = f(x_n)$
  - 若 $\exist r\in\R$ 使得 $|f'(x)| \leq r < 1\pad (\forall x\in\R)$，则 $x_n$ 收敛
    - <font color='chartreuse'>中间必须有一个小于1的数，而不能是 $|f'(x)|< 1$，否则可能是 $|f'(x)| \to 1-$</font>
  - **证明**：
    - 导数增长性 + 三角不等式得 $|x_{n+p}-x_n| \leq \sum\limits^{n+p}_{k=n+1} r^{k-1}|x_1-x_0| \leq \\ |x_1-x_0|\dfrac{r^n}{1-r}$
    - 由柯西收敛原理即得 $x_n$ 收敛

##### 不动点定理（更一般的形式见泛函分析）

- **不动点方法**：
  - 设 $f$ 是区间 $I$ 上连续增函数，$x_{n+1} = f(x_n)$ 
  - 若 $f$ 在 $I$ 上有不动点 $x^*$，且 $\Big[ x_1-f(x_1) \Big](x_1-x^*) \geq 0$
    - <font color='chartreuse'>若 $f$ 可导，则条件弱化为 $0 < f'(x) < 1$
    <!-- - ：$x_1$ 和 $x^*$ 之间 $\begin{cases} f'(x) \geq 1 & x_1 > x^* \\ f'(x) \leq 1 & x_1 < x^* \end{cases}$ -->
    - 最理想图像为左边高于 $y=x$，右边低于 $y=x$ 的函数</font>
  - 则 $x_n$ 收敛，且 $\lim\limits_{n\to\infty} x_n = f(\lim\limits_{n\to\infty} x_n)$
  - **证明**：
    - 若题设式子成立等号，则 $x_1$ 即为不动点，从而 $x_n$ 恒等，当然收敛。故只需考虑不等情况
    - 接下来我们证明 $\forall x_1\in I$，都有 $x_n\to x^*$
    - 设 $g(x) = f(x) - x$，则 $x^*$ 是 $g$ 的零点
      - 由题设不等式，讨论易得 $g$ 单增
      - 再由于 $x_{n+1}-x_n = g(x_n)$，故 $g(x_n)$ 的正负性等价于 $x_n$ 的单调性
    - 由题设不等式，$g(x_1)(x_1-x^*) < 0$，故不妨设 $\begin{cases} x_1 > x^* \\ g(x_1) < 0 \end{cases}$
      - 此时 $x_1$ 减小，从而 $g$ 也减小
      - <font color='chartreuse'>也就是说，后面 $g$ 始终小于 $0$，即 $x_n$ 始终缩小，从而单减
      - 再由于 $x_n$ 有下界 $x^*$，故其收敛到 $x^*$</font>
    <!-- - 若 $\begin{cases} x_1 > x^* \\ g(x_1) > 0 \end{cases}$，则 $x_1$ 增加，从而 $g(x_1)$ 也增加，则其最终会不断发散 -->
  - **理解**：其实画个函数图像就很容易发现了
- **递推函数单减时**：
  - 设 $f$ 在 $I$ 上严格单减，$x_{n+1} = f(x_n)$
  - 若 $f$ 有不动点 $x^*$
  - 则奇偶子列 $b_n = x_{2n}，c_n = x_{2n-1}$ 分别在 $x^*$ 两侧，且 $F(x) = f\circ f(x)$ 是它们的递推函数，严格单增
  - **应用**：
    - $x_1 = 1，x_2 = \dfrac{1}{2}，x_{n+1} = \cfrac{1}{1+x_n}$
    - **解**：
      - 显然 $f$ 单减，故 $F = f\circ f$ 单增，易得 $x_{2n}$ 和 $x_{2n-1}$ 在 $x^* = \dfrac{\sqrt{5}-1}{2}$ 均满足不动点定理，故存在极限
      - <font color='chartreuse'>此时 $x_{2n}$ 在 $x^*$ 左端单增收敛，$x_{2n-1}$ 在 $x^*$ 右端单减收敛。通过直接计算就容易发现这一点</font>
    - $x_1 = 2，x_{n+1}  = 2+\cfrac{1}{x_n}$
    - **解**：
      - 显然 $f$ 单减，故 $F = \cfrac{5}{2}-\cfrac{1}{2(2x+1)}$ 单增，不动点为 $x^* = 1+\sqrt{2}$
      - 易得 $x_{2n}$ 在 $(1-\sqrt{2},1+\sqrt{2})$ 区域单增，其余情况单减，$x_2 = \dfrac{5}{2} > 1+\sqrt{2}$
        - 故由单调有界收敛原理，其单减收敛到 $x^*$
      - 同理 $x_{2n-1}$ 单增收敛到 $x^*$
      - 综上即得结论

### 实数系的完备性

- **完备性**：基本数列必定收敛（任何一个满足收敛性条件的数列，都有唯一一个极限实数与它匹配）
- **连续性**：非空有上界的集合必有上确界（引出 $\e$）
- **六大定理的等价性**：用Cauchy收敛原理逆推确界存在定理
  - **闭区间套定理**
    - **证明**：
      - 取 $m>n$，则 $a_m - a_n < b_n - a_n\to 0$，从而 $a_n$ 收敛，同理 $b_n$ 收敛
      - 设 $\xi_1 = \lim\limits_{n\to\infty} a_n，\xi_2 = \lim\limits_{n\to\infty} b_n$，反设 $\xi_1 \neq \xi_2$，证明同上
  - **确界存在定理**
    - **证明**：构造同时包含上界和集合的闭区间，同时取二分缩小，形成闭区间套。由闭区间套定理，其最终收敛于一点。再用定义验证该点是上确界
  - **单调有界数列收敛定理**：利用上界定义，构造新的数列
    - **闭区间套法**：
      - <font color='chartreuse'>构造单调递减的上界数列</font>
      - 首先对于 $x_1$，取一个上界 $M_1$，再对 $x_2$，取一个上界 $M_2 < M_1$
        - 可以证明 $M_{k+1}$ 可以无限取到，否则 $M_k$ 就根据定义成为上确界
      - 这样，定义每个 $|M_k - x_k| < \e_k\to 0$
        - 则对于 $\forall \e>0 ，\exist k > 0，\forall n>k$ 有 $|M_n-x_n|<\e$
        - 又由上界定义有 $M_n>x_n$，即 $[x_n,M_n]$ 构成闭区间套。由闭区间套定理，它最终收敛于 $\xi$。可以验证 $\xi$ 就是上确界，同时也是 $x_n$ 的极限
    - **反证法**：
      - 反设不收敛，则由单增性，$x_{n+p}-x_n > \e_0 > 0$，累加起来，得到一个无界子列，与有界矛盾
