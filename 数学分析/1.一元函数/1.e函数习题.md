## 反函数

    - $x_1$ 是 $f(x) + x = a$ 的根

  - **解1（图像法）**：
    - **引理（反函数图像对称性）**：反函数和原函数的图像关于 $l_1:y=x$ 对称

      - **证（几何法）**：

## 奇偶函数

- 奇函数的反函数也是奇函数
  - **证明**：定义即可
- **函数空间直和分解**：任意函数都可写为奇偶函数之和
  - **证明**：类似后面的正负分离
    - 奇函数部分 $g(x) = \cfrac{f(x)-f(-x)}{2}$，偶函数部分 $h(x) = \cfrac{f(x)+f(-x)}{2}$

## 对称函数

- **对称的奇偶性**：（奇奇为偶，奇偶为奇，偶偶为偶）
  - 若 $f,g$ 均轴对称，则 $fg$ 也轴对称
  - 若 $f,g$ 均中心对称，则 $fg$ 轴对称
  - 若 $f$ 关于 $x=x_0$ 轴对称，$g$ 关于 $(x_0,0)$ 中心对称，则 $fg$ 中心对称

## 周期函数

- 设周期函数 $f$ 连续，若其没有最小正周期，则其为常值函数
  - **证明**：
    - 由于没有最小正周期，故存在 $T_n\searrow 0$ 使得每项都是 $f$ 的周期
    - 易得 $\lim\limits_{n\to\infty} [\dfrac{x}{T_n}]T_n = x$，由连续性即得 $f(x) = \lim\limits_{n\to\infty} f\Big( 0 + [\dfrac{x}{T_n}]T_n \Big)$
    - 再由于是整数个周期，故右式 $ = f(0)$，从而是常函数
- 设一元实函数 $f(x)$ 在 $\R$ 上有界
  - 若 $\exist h$ 使得 $f(x+h) = \dfrac{f(x+2h)+f(x)}{2} \pad (\forall x\in \R)$
  - 则 $h$ 是 $f$ 的周期
  - **解（其实就是累加法）**：
    - 取 $F(x) = f(x+h)-f(x)$，则只需证 $F(x) \equiv 0$ 即可
    - 易得题设式可变形为 $F(x+h) = F(x)$
      - 累加即得 $f(x+nh)-f(x) = \sum\limits^{n-1}_{k=1}F(x+kh) = nF(x)$
    - 反设 $\exists F(x_0)\neq 0$，此时取 $n\to\infty$，即得右式为无穷大，从而左式也为无穷大，但其与 $f$ 有界矛盾
  - <font color='chartreuse'>每段都相差同一个差值，则在无穷远处该差值会累加到无穷，与有界矛盾</font>
- **均值积分性**：设 $f$ 是周期为 $T$ 的连续函数，则 $$ \lim\limits_{x\to +\infty} \frac{\int^x_0 f(t)dt}{x} = \frac{\int^T_0 f(t)dt}{T}$$
  - **证明**：
    - 由连续函数可积性，可设 $[0,T]$ 内 $f$ 积分为 $A$
    - 再由周期函数的积分周期性，设 $x=nT+y \quad (y\in [0,T])$，则原式变为数列极限 $\lim\limits_{n\to\infty} \dfrac{nA + B}{nT+y} = \dfrac{A}{T}$，容易证明（**证毕**）

## 不等式

- **伯努利不等式**：设 $x>-1$，则 $(1+x)^\a \geq 1+\a x\pad (\a \geq 1)$，其它情况相反
  - **证明**：求导即可

## 求和式

- 求 $\sum\limits^n_{i=1} i^3$ 的通项公式
  - **解**：
    - 易得通项公式必定为 $n$ 的多项式，再由阶的估计可得其不超过 $4$ 次
    - 设 $S_n = a_0 + ... + a_4n^4$，则 $S_n-S_{n-1} = n^3$
    - 比较上面方程的各项系数，得到线性方程组 $\begin{cases} a_1-a_2+a_3-a_4 = 0 \\ 2a_2-3a_3+4a_4 = 0 \\ 3a_3-6a_4 = 0 \\ 4a_4 = 1 \end{cases}$
    - 解得 $S_n = \dfrac{1}{4}n^4 + \dfrac{1}{2}n^3 + \dfrac{1}{4}n = \Big[ \dfrac{n(n+1)}{2} \Big]^2$
- **弱化的L插值多项式**：任意 $y_1,...,y_m$，都存在 $m$ 次多项式 $f(n)$ 使得 $\forall f(i) = y_i$
  - **秩比较法**：代入 $f(i) = y_i$，得到非齐次线性方程组，易得 $y_i$ 取任意值都有解
  - **数列法**：
    - 以 $m=4$ 为例，数列 $\a = (1,2,3,y)$ 可写成 $(1,2,3,4) + (0,0,0,y-4)$
