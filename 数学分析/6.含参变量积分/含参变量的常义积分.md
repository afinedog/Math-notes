# 含参积分

- 函数项级数的积分形式

## 积分函数

- $I(y) = \int^b_a f(x,y)dx$ 是一个关于y的函数，其中y被包裹在积分中
  - **第二类椭圆积分**：利用参数方程求椭圆积分
- **连续性定理**：被积函数 $f(x,y)$ 在闭矩形上连续，则积分函数 $I(y)$ 在y轴上连续
  - **证明**
    - 首先得到一致连续
    - 然后由二维推一维即可
  - 积分和极限运算交换：$\lim\limits_{y\to y_0}\int f(x,y)dx = \int\lim\limits_{y\to y_0}f(x,y)dx$
  - **积分次序交换定理（交换参量）**：被积函数在闭矩形上连续，则 $\int dy\int f(x,y)dx = \int dx\int f(x,y)dy$（二重积分作为媒介，连接两个累次积分）
  - **积分号下求导定理（参量求导）**：$f(x,y)、f_y(x,y)$ 都在闭矩形上连续，则 $I(y) = \int f(x,y)dx$ 在y轴方向上可导，且 $\frac{dI(y)}{dy} = \int f_y(x,y)dx$
    - **证明**：
      - 对y轴使 用微分中值定理：$\frac{I(y+\triangle y) - I(y)}{\triangle y} = \int f_y(x,y+\theta\triangle y)dx$
      - 然后交换极限，$\frac{dI(y)}{dy} = \lim\int f_y(x,y+\theta\triangle y) = \int\lim f_y(x,y+\theta\triangle y) = \int f_y(x,y)dx$
    - **理解**：本质还是偏微分中值定理，然后利用连续性消去中值即可
    - **应用**：求导运算和积分运算交换（仅限参数）
- **含参变量的上限函数导数**：$f(x,y)，f_y(x,y)$ 在闭矩形上连续，另外有 $a(y)，b(y)$ 在y轴方向上可导，且在闭矩形内，则 $F(y) = \int^{b(y)}_{a(y)}f(x,y)dx$ 在y轴方向上可导
  - 导数为 $F'(y) = \int^{b(y)}_{a(y)} f_y(x,y)dx + [f(b(y),y)b'(y) - f(a(y),y)a'(y)]$
  - **本质**：
    - 构造复合函数，利用链式法则即可
  - **证明**：
    - 化为三变量复合函数：$\begin{cases} u = a(y) \\ v = b(y) \\ F(y) = \int^u_v f(x,y)dx = I(y,u,v)\end{cases}$
    - 对y求导：$\frac{\partial I}{\partial y} = \int f_ydx$（积分号下求导）
    - 对u、v求导：上限函数导数，$\begin{cases} \frac{\partial I}{\partial u} = f(u,y) \\ \frac{\partial I}{\partial v} = -f(v,y)\end{cases}$
    - 最后链式法则总和起来即可
  - **理解**：一个是N-L公式，一个是积分号下求导

## 含参反常积分

### 一致收敛性

- **收敛**：在矩形 $[a,+\infty)×[c,d]$ 上对某个 $y_0\in [c,d]$ 收敛
  - 收敛点、收敛域（$I(y)$ 的定义域）
- **一致收敛**：矩形上反常积分存在，且逼近于一个 $I(y)$，则关于y在 $[c,d]$ 上一致收敛
- Cauchy收敛原理、Weierstrass判别法
- Abel判别法：积分在y上一致收敛 + 函数关于x单调，y上一致有界
- Dirichlet判别法：积分在y上一致有界 + 函数关于x单调，y上一致趋于0
- **Dini定理**：$f(x,y)$ 在无穷开矩形上连续且定号（含参反常积分单调），且含参反常积分在 $[c,d]$ 上连续，则含参反常积分在 $[c,d]$ 上一致收敛
  - **证明**：反证，首先构造不收敛函数数列 $\int^{+\infty}_{n}f(x,y_n)dx > \e_0$
    - 然后因为单调性，$n$ 变化为 $A$，因为连续性，$y_n$ 变为 $y_0$，由点态收敛（反常积分存在即收敛），得到矛盾
  - **比较**：函数项级数中，$N\to n$ 结果增大。含参积分中，$A\leftarrow n$ 结果增大

### 性质

- **数列引理**：一致收敛性 $\int^{+\infty}_{a}f(x,y)dx \LR \sum\limits^\infty_{n=1}u_n(y)$
  - **证明**：数列的柯西收敛原理
- 连续性定理（反常版本）：矩形上函数连续，积分一致收敛，则积分对于y连续
  - **证明**：利用函数项级数 $u_n$ 的性质
- 积分换序定理（反常版本）：
  - **证明**：利用函数项级数 $u_n$ 的性质
  - 若y上也改为无穷区间，则 （？）
- 内部求导定理（反常版本）：矩形上函数和偏导数都连续，积分收敛，偏导数积分一致收敛，则积分对于y可导
  - **证明**：类似函数项级数证明
    - 偏导数积分一致收敛，偏导数连续，则偏导数积分连续
    - 则偏导数积分换序，N-L公式得到 $I(y)-I(c)$
    - 求导即可

## Euler积分

### Beta函数（第一类Euler积分）

- $B(p,q) = \int^1_0 x^{p-1}(1-x)^{q-1}dx$
  - 定义域为第一象限（两个奇点，拆为两部分积分）
  - **连续性**：一致收敛性
  - **对称性**：pq可互换（积分区间对称性）
  - **递推关系**：
    - $B(p,q) = \frac{q-1}{p+q-1}B(p,q-1),p>0,q>1$
    - **证明**：分部积分
    - $B(p,q) = \frac{(p-1)(q-1)}{(p+q-1)(p+q-2)}B(p,q-1),p,q>1$
  - **基底**：
    - $B(1,1) = 0$
  - 三角函数表示：$x = cos^2\varphi$
  - 有理分数表示：$\begin{cases}x=\frac{1}{1+t} \\ t=\frac{1}{u}\end{cases}$

### Gamma函数（第二类Euler积分）

- $\Gamma(s) = \int^{+\infty}_0x^{s-1}e^{-x}dx$
  - 定义域为 $(0,+\infty)$（两个奇点，拆为两部分积分）
  - **可导性**：函数和导数一致收敛性
  - **递推关系**：$\Gamma(s+1) = s\Gamma(s)，(s>0)$
    - **证明**：分部积分
  - **基底**
    - $\Gamma(n+1) = n!$
    - $\Gamma(\frac{1}{2}) = \sqrt{\pi}$
  - 幂表示：$x=t^2$
  - 线性表示：$x=\alpha t$
  - **定义域延拓**：$\Gamma(s) = \frac{\Gamma(s+1)}{s}$，右侧有意义，从而可以在左侧补充定义，延拓到整个 $R$ 

### B、G函数转化

- $B(p,q) = \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)}，(p,q>0)$
  - **证明**：
    - 取 $\Gamma(p)\Gamma(q)$ 的二次幂表示
    - 化为二重积分：$4\iint s^{2p-1}e^{-s^2} t^{2q-1}e^{-t^2}dsdt$
    - 极坐标变换：$(2\int cos^{2p-1}\theta sin^{2p-1}\theta d\theta)(2\int r^{2(p+q)-1} e^{-r^2}dr)$
      - 化一公式消去exp上的三角函数
  - **本质**：累次积分的常用方法，积分合并后换序

### 应用

- $I = \int^{+\infty}_0 x^{2n}e^{-x^2}$
  - G函数的二次幂表示，递推公式计算即可
- $I = 2\int^\frac{\pi}{2}_0 sin^{2p-1}xcos^{2q-1}xdx$
  - B函数转为G函数形式，再递推计算
- $\int^\frac{\pi}{2}_0 sin^\alpha x（\alpha > -1）$
  - B函数转化为G函数

### G函数公式

- **Legendre公式**：$\Large\Gamma(s)\Gamma(s+\frac{1}{2}) = \frac{\sqrt{\pi}}{2^{2s-1}}\Gamma(2s)，s>0$
  - **证明**：
    - $B(s,s)$，合并配方，换元为 $\frac{1}{2}-x = \frac{1}{2}\sqrt{t}$，得到 $\frac{1}{2^{2s-1}}B(\frac{1}{2},s)$
  - **理解**：两个s可以配凑合并成平方，换元同构。$\frac{1}{2} (\sqrt{·})$、$\frac{1}{2^{2s-1}}$（幂内公因$\frac{1}{2}$）都是换元的痕迹
- **三角函数分解级数式**：$\frac{\pi}{sin\ \pi x} = \frac{1}{x} + \sum\limits^\infty_{n=1}(-1)^n(\frac{1}{x+n} + \frac{1}{x-n})$
  - **证明**：
    - 已知 $sin\ x$ 的无穷乘积式，取对数，逐项求导得到cot。换元得到tan
    - 利用 $\frac{1}{sinx} = \frac{1}{2}(tan\frac{x}{2}+cot\frac{x}{2})$，即可得到结果
- **余元公式**：$\Large\Gamma(s)\Gamma(1-s) = \frac{\pi}{sin\ \pi s}，0<s<1$
  - **证明**：
    - 引理：连续可积的序列若内闭一致收敛，则其在开区间上可交换极限
    - B函数转化为 $B(s,1-s)$
      - $\frac{t^{s-1}}{1+t}$，分数幂级数展开，积分加和得到 $\sum\limits^\infty_{n=0} \frac{(-1)^n}{s+n}$
      - $(0,1)$ 和 $(1,+\infty)(换元)$ 的积分结果都可以求出
    - 最后依赖上面的算式完成转化
- **Stirling公式**：$\Large\Gamma(s+1) = \sqrt{2\pi s}(\frac{s}{e})^se^{\frac{\theta}{12s}}$
