## Euler积分

### Beta函数（第一类Euler积分）

- **贝塔函数**：$B(p,q) = \dis\int^1_0 x^{p-1}(1-x)^{q-1}dx$
  - **定义域**：第一象限
    - <font color='chartreuse'>存在两个奇点 $0,1$，故一般拆成两部分积分</font>
  - **连续性**：一致收敛性
  - **对称性**：pq可互换（积分区间对称性）
  - **递推关系**：
    - $B(p,q) = \dfrac{q-1}{p+q-1}B(p,q-1)\quad (p>0,q>1)$
    - $B(p,q) = \dfrac{(p-1)(q-1)}{(p+q-1)(p+q-2)}B(p,q-1)\quad (p,q>1)$
      - **证明**：分部积分即可
  - **基底**：
    - $B(1,1) = 0$
  - **表示方式**：
    - **三角函数表示法**：$x = \cos^2\varphi$
    - **有理分数表示法**：$\begin{cases}x=\frac{1}{1+t} \\ t=\frac{1}{u}\end{cases}$

### Gamma函数（第二类Euler积分）

- **伽马函数**：$\G(s) = \dis\int^{+\infty}_0x^{s-1}e^{-x}dx$
  - **定义域**： $(0,+\infty)$
    - <font color='chartreuse'>存在两个奇点 $0,+\infty$，故一般拆成两部分积分</font>
  - **内闭一致收敛性**：不一致收敛，但内闭一致收敛
  - **可导性**：无穷阶可微，$\G^{(n)}(s) = \dis\int^{+\infty}_0 x^{s-1}e^{-x}(\ln x)^n$
  - **递推关系**：$\G(s+1) = s\G(s)，(s>0)$
    - **证明**：分部积分
  - **基底**
    - $\G(n+1) = n!$
      - **证明**：直接计算
    - $\G(\frac{1}{2}) = \sqrt{\pi}$
      - **证明**：换元 $x=t^2$，利用泊松积分
  - **表示方式**：
    - **幂表示**：$x=t^2$
    - **线性表示**：$x=\a t$
  - **定义域延拓**：$\G(s) = \frac{\G(s+1)}{s}$，右侧有意义，从而可以在左侧补充定义，延拓到整个 $R$ 

### 欧拉积分转化公式

- **转化公式**：$B(p,q) = \cfrac{\G(p)\G(q)}{\G(p+q)}，(p,q>0)$
  - **证明**：
    - 取 $\G(p)\G(q)$ 的二次幂表示
    - 化为二重积分：$4\iint s^{2p-1}e^{-s^2} t^{2q-1}e^{-t^2}dsdt$
    - 极坐标变换：$(2\int cos^{2p-1}\theta sin^{2p-1}\theta d\theta)(2\int r^{2(p+q)-1} e^{-r^2}dr)$
      - 化一公式消去exp上的三角函数
  - **本质**：累次积分的常用方法，积分合并后换序

### 计算例

- $I = \dis\int^{+\infty}_0 x^{2n}e^{-x^2}dx$
  - **解**：
    - 此为G函数的二次幂表示，递推公式计算即可
- $I = \dis 2\int^\frac{\pi}{2}_0 \sin^{2p-1}x \cos^{2q-1}xdx$
  - **解**：
    - B函数转为G函数形式，再递推计算
- $\dis\int^\frac{\pi}{2}_0 \sin^\a xdx$，此时 $\a > -1$
  - **解**：B函数转化为G函数

### 伽马函数公式

- **Legendre公式**：$$\G(s)\G(s+\frac{1}{2}) = \frac{\sqrt{\pi}}{2^{2s-1}}\G(2s)，s>0$$
  - **证明**：
    - $B(s,s)$，合并配方，换元为 $\frac{1}{2}-x = \frac{1}{2}\sqrt{t}$，得到 $\frac{1}{2^{2s-1}}B(\frac{1}{2},s)$
  - **理解**：两个s可以配凑合并成平方，换元同构。$\frac{1}{2} (\sqrt{·})$、$\frac{1}{2^{2s-1}}$（幂内公因$\frac{1}{2}$）都是换元的痕迹
- **三角函数分解级数式**：$$\frac{\pi}{\sin \pi x} = \frac{1}{x} + \sum\limits^\infty_{n=1}(-1)^n(\frac{1}{x+n} + \frac{1}{x-n})$$
  - **证明**：
    - 已知 $sin\ x$ 的无穷乘积式，取对数，逐项求导得到cot。换元得到tan
    - 利用 $\frac{1}{sinx} = \frac{1}{2}(tan\frac{x}{2}+cot\frac{x}{2})$，即可得到结果
- **余元公式**：$\G(s)\G(1-s) = \cfrac{\pi}{\sin\ \pi s}，(0<s<1)$
  - **证明**：
    - 引理：连续可积的序列若内闭一致收敛，则其在开区间上可交换极限
    - B函数转化为 $B(s,1-s)$
      - $\frac{t^{s-1}}{1+t}$，分数幂级数展开，积分加和得到 $\sum\limits^\infty_{n=0} \frac{(-1)^n}{s+n}$
      - $(0,1)$ 和 $(1,+\infty)(换元)$ 的积分结果都可以求出
    - 最后依赖上面的算式完成转化
- **Stirling公式**：$\Large\G(s+1) = \sqrt{2\pi s}(\frac{s}{e})^se^{\frac{\theta}{12s}}$
