# Euler积分

- 这一章没有什么新东西，主要是背诵一些二级结论。但是这些结论很重要

## Beta函数（第一类Euler积分）

- **贝塔函数**：$B(p,q) = \dis\int^1_0 x^{p-1}(1-x)^{q-1}dx$，其中 $p>0,q>0$
  - **一致收敛性**
    - <font color='chartreuse'>它不是反常积分，显然一致收敛</font>
  - **连续性**
    - <font color='chartreuse'>含参积分的连续性定理即可</font>
  - **对称性**：$p,q$ 互换后结果不变
    - <font color='chartreuse'>换元即可</font>
  - **递推关系**：
    - **一元**：$$B(p,q) = \frac{q-1}{p+q-1}B(p,q-1)\quad (p>0,q>1)$$
      - **证明**：
        - 首先进行分部积分，然后裂项分解即可
    - **二元**：$$ B(p,q) = \frac{(p-1)(q-1)}{(p+q-1)(p+q-2)}B(p-1,q-1)\quad (p,q>1)$$
  - **基底**：
    - $B(1,1) = 1$
      - <font color='chartreuse'>直接计算易得</font>
      - $B(2,1) = B(1,2) = \cfrac{1}{2}$
      - $B(2,2) = \cfrac{1}{6}$
    - $B(\dfrac{1}{2},\dfrac{1}{2}) = \pi$
      - <font color='chartreuse'>转化为三角函数表示法，计算易得</font>
  - **表示方式**：
    - **三角函数表示法**：换元 $x = \cos^2 t$，则 $$B(p,q) = 2\dis\int^{\dfrac{\pi}{2}}_0 \cos^{2p-1}t \sin^{2q-1}tdt$$
    - **无穷表示法**：换元 $x=\cfrac{1}{1+t}$，则可表示为 $$ B(p,q) = \int^{+\infty}_0 \frac{t^{q-1}}{(1+t)^{p+q}}dt = \int^{+\infty}_0 \frac{t^{p-1}}{(1+t)^{p+q}}dt$$
    - **有理分数表示法**：换元 $x=\cfrac{1}{1+t}$，分离成 $(0,1)$ 和 $(1,+\infty)$，后者再换元，即可重组为 $$ B(p,q) = \int^1_0 \frac{t^{p-1} + t^{q-1}}{(1+t)^{p+q}}dt$$

## Gamma函数（第二类Euler积分）

- **伽马函数**：$\G(s) = \dis\int^{+\infty}_0x^{s-1}e^{-x}dx$，其中 $s\in (0,+\infty)$
  - **内闭一致收敛性**：不一致收敛，但内闭一致收敛
    - <font color='chartreuse'>存在两个奇点 $0,+\infty$，故一般拆成两部分积分</font>
  - **光滑性**：$$\G^{(n)}(s) = \dis\int^{+\infty}_0 x^{s-1}e^{-x}(\ln x)^ndx $$
    - <font color='chartreuse'>由内闭一致收敛性，内部求导即可</font>
  - **递推关系**：$\G(s+1) = s\G(s)$
    - **证明**：分部积分
  - **基底**
    - $\G(n+1) = n!$
      - <font color='chartreuse'>直接计算即可</font>
    - $\G(\dfrac{1}{2}) = \sqrt{\pi}$
      - <font color='chartreuse'>换元 $x=t^2$，利用泊松积分</font>
    - $\G(n+\dfrac{1}{2}) = \dfrac{(2n-1)!!}{2^{n-1}}\sqrt{\pi}$
  - **其它表示**：
    - **二次表示**：换元 $x=t^2$，则 $$ \G(s) = 2\int^{+\infty}_0 t^{2s-1}e^{-t^2}dt$$
  - **定义域延拓**：
    - 对于 $\G(s) = \cfrac{\G(s+1)}{s}$，易得右式在 $(-1,0)$ 有定义，故 $\G(s)$ 可延拓到 $(-1,0)$ 上
    - 同理，可由 $(-1,0)$ 上的值，延拓到 $(-2,-1)$ 上
    - 不断做下去，可延拓到 $\R \j \{0,-1,-2,...\}$  上

### 计算例


- $I = \dis\int^{+\infty}_0 x^{2n}e^{-x^2}dx$
  - **解**：
    - 由伽马函数的二次表示，$I = \dfrac{1}{2}\G(n+\dfrac{1}{2})$
    - 反复利用递推公式即得 $I = \cfrac{(2n-1)!!}{2^n}\sqrt{\pi}$
- $I = \dis\int^{+\infty}_0 e^{-x^n}dx$
  - **解**：
    - 换元 $t = x^n$，则转化为 $\dfrac{1}{n}\G(\dfrac{1}{n})$
- $I = \dis\int^1_0 (-\ln x)^ndx$
  - **解**：
    - 换元 $x = e^t$，则转化为 $\G(n+1) = n!$

## 欧拉积分转化公式

- **转化公式**：$B(p,q) = \cfrac{\G(p)\G(q)}{\G(p+q)}，(p,q>0)$
  - **证明**：
    - 取 $\G(p)\G(q)$ 的二次表示，化为二重积分得 $$4\iint s^{2p-1}e^{-s^2} t^{2q-1}e^{-t^2}dsdt$$
    - 取极坐标变换 + 累次积分得 $$\dkh{2\int^{\dfrac{\pi}{2}}_0 \cos^{2p-1}\t \sin^{2p-1}\t d\t}\dkh{2\int^{+\infty}_0 r^{2(p+q)-1} e^{-r^2}dr}$$
    - 即 $\G(p)\G(q) = B(p,q)\G(p+q)$
    - <font color='red'>B函数的递推公式显然比G函数复杂，故以后遇到B函数，一般转化为G函数</font>

## 伽马函数计算公式

- **Legendre公式**：$$\G(s)\G(s+\frac{1}{2}) = \frac{\sqrt{\pi}}{2^{2s-1}}\G(2s)$$
  - **证明**：
    - 将 $B(s,s)$ 合并后配方得 $$B(s,s) = \int^1_0 \fkh{\frac{1}{4}-\dkh{\frac{1}{2}-x}^2 }^{s-1}dx$$
    - 易得被积函数关于 $x=\dfrac{1}{2}$ 对称，再换元 $\frac{1}{2}-x = \frac{1}{2}\sqrt{t}$ 即得 $$B(s,s) = \frac{1}{2^{2s-1}}B\dkh{\frac{1}{2},s}$$
    - 最后由转化公式即得结论
- **三角函数分解级数式**：$$\frac{\pi}{\sin \pi x} = \frac{1}{x} + \sum\limits^\infty_{n=1}(-1)^n(\frac{1}{x+n} + \frac{1}{x-n})$$
  - **证明**：
    - 已知 $\sin x = x\prod\limits^\infty_{n=1} (1-\dfrac{x^2}{n^2\pi^2})$ ，取对数后逐项求导得到 $\cot x$ 的展开式
    - 换元 $\dfrac{\pi}{2}-x$ 得到 $\tan x$ 的展开式
    - 最后由 $\cfrac{1}{\sin x} = \cfrac{1}{2}(\tan\frac{x}{2}+\cot\frac{x}{2})$，即可得到结果
- **余元公式（反射公式）**：$$\G(s)\G(1-s) = \cfrac{\pi}{\sin \pi s}，s\in (0,1)$$
  - **证明**：
    - 引理：连续可积的序列若内闭一致收敛，则其在开区间上可交换极限
    - 由转化公式，它等于 $B(s,1-s)$
      - 写成无穷形式得 $$B(s,1-s) = \int^{+\infty}_0\frac{t^{s-1}}{1+t}dt$$
      - 在 $(0,1)$ 上，将被积函数幂级数展开后积分极限交换，结果为 $\sum\limits^\infty_{n=0} \frac{(-1)^n}{s+n}$
      - 在 $(1,+\infty)$ 上换元并计算，结果为 $\sum\limits^\infty_{n=0} \frac{(-1)^n}{s-n}$
    - 最后由上面的引理即得结论
- **Stirling公式**：$$\G(s+1) = \sqrt{2\pi s}\dkh{\frac{s}{e}}^se^{\dfrac{\t}{12s}}$$
  - <font color='chartreuse'>它是整数Stirling公式的推广，证明复杂</font>



## 习题

- $I = \dis\int^1_0 \ln \G(x)dx$
  - **解**：
    - 由积分区间对称性，易得 $\dis\int^1_0 \ln \G(x)dx = \int^1_0 \ln \G(1-x)dx$
    - 故由余元公式，$2I = \dis\int^1_0 \ln\dkh{\frac{\pi}{\sin \pi x}}dx$
    - 再由前面结论即得 $I = \ln\sqrt{2\pi}$

    - 再取极坐标换元，则 $$I = $$
      - 第一项转化为 $\dfrac{2}{\a\b}B(\dfrac{1}{\a},\dfrac{1}{\b})$
      - 第二项取 $r^2 = t$，则转化为 $\dis\frac{1}{2}\int^{+\infty}_0 \frac{t^{\frac{1}{\a}+\frac{1}{\b}-1}}{1+t}dt$
