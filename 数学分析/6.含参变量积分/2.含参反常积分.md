# 含参反常积分

- **含参反常积分**：积分区域改为 $[a,+\infty)\times [c,d]$

## 一致收敛性

- **点态收敛**：在无穷矩形 $[a,+\infty)×[c,d]$ 上对某个 $y_0\in [c,d]$ 收敛
- **一致收敛**：若 $[c,d]$ 上任意点反常积分都存在，且极限均为 $I(y)$，则反常积分关于 $y$ 在 $[c,d]$ 上一致收敛到 $I(y)$
- **收敛判别法**：
  - **Cauchy收敛原理**
    - <font color='chartreuse'>重要例子：$I(a) = \dis\int_{\R} e^{-(x-a)^2}dx$ 在无穷远处不收敛，因为余项为 $$\lim\limits_{a\to +\infty}\dis\int^{+\infty}_A e^{(x-a)^2}dx = \int^{+\infty}_{-\infty} e^{u^2}du = \sqrt\pi$$</font>
  - **Weierstrass判别法**
  - **Abel判别法**：
    - 若 $\dis\int f(x,y)dx$ 在 $y$ 上一致收敛，$g(x,y)$ 关于 $x$ 单调，关于 $y$ 一致有界
    - 则 $\dis\int f(x,y)g(x,y)dx$ 在 $y$ 上一致收敛
  - **Dirichlet判别法**：
    - 若 $\dis\int f(x,y)dx$ 在 $y$ 上一致有界，$g(x,y)$ 关于 $x$ 单调，关于 $y$ 一致趋于 $0$
    - 则 $\dis\int f(x,y)g(x,y)dx$ 在 $y$ 上一致收敛

## 性质

- **柯西积分判别法**：$\dis\int^{+\infty}_{a}f(x,y)dx$ 一致收敛 $\LR \sum\limits^\infty_{n=1}u_n(y)$ 一致收敛
  - **证明**：数列的柯西收敛原理
- **连续性定理（反常版本）**：若开矩形上函数连续，积分一致收敛，则 $F$ 连续
  - **证明1**：利用函数项级数 $u_n$ 的性质
  - **证明2**：定义式 + 区间分段 + 三角不等式
- **积分换序定理（反常版本）**：若开矩形上函数连续，积分一致收敛，则 $F$ 可积，且积分可换序
  - **证明1**：利用函数项级数 $u_n$ 的性质
  - **证明2**：易得极限函数存在。作差 + 积分换序即可
  - **推论**：若 $y$ 上也改为无穷区间，则条件变为：
    - 两个单独积分 $F(y),G(x)$ 都内闭一致收敛
    - 两个二重积分中至少有一个绝对收敛
    - **证明**：作差 + 区间分段 + 积分三角不等式放缩即可
      - <font color='chartreuse'>由于有两个无穷区间，故会有一个 $[M,+\infty)\times [N,+\infty)$ 的积分被分离出来，这时候需要应用二重积分的绝对收敛性</font>
- **极限-积分换序定理**：
  - 若 $\lim\limits_{x\to+\infty} f(x,t) = \p(t)$ 关于 $t$ 一致成立，且 $F(x) = \dis\int^{+\infty}_0 f(x,t)dt$ 一致收敛，则积分和极限可换序
  - **证明**：
    - 首先柯西收敛原理 + 添项 $f$ 易得 $\p(t)$ 的含参积分收敛
    - 然后作差 + 区间分段即可
- **内部求导定理（反常版本）**：若开矩形上函数和偏导数都连续，积分点态收敛，偏导数积分一致收敛，则 $F$ 可导
  - **证明1**：类似函数项级数证明
    - 偏导数积分一致收敛，偏导数连续，则偏导数积分连续
    - 则偏导数积分换序，N-L公式得到 $I(y)-I(c)$
    - 求导即可
  - **证明2**：区间分段 + L中值定理 + 三角不等式
- **Dini定理**：
  - 若
    - $f(x,y)$ 在无穷矩形上连续且定号<font color='chartreuse'>（含参反常积分单调）</font>
    - 含参反常积分在 $[c,d]$ 上连续<font color='chartreuse'>（含参反常积分点态收敛，逐项连续）</font>
  - 则含参反常积分在 $[c,d]$ 上一致收敛<font color='chartreuse'>（推论易得一致连续）</font>
  - **证明**：反证即可
    - 反设不一致收敛，则数列 $\dis\int^{+\infty}_{n}f(x,y_n)dx > \e_0$
    - 再因为单调性，$n$ 可变化为 $A$。再因为连续性，$y_n$ 可变为 $y_0$，最后由点态收敛性（反常积分存在等价于收敛），得到矛盾
  - **比较**：函数项级数中，$N\to n$ 结果增大。含参积分中，$A\leftarrow n$ 结果增大

### 一致连续性

- 设 $\dis\int^{+\infty}_{-\infty} |f(x)|dx$ 收敛，则 $g(t) = \dis\int^{+\infty}_0 f(x)\sin(tx)dx$ 一致收敛且一致连续
  - **证明**：
    - 一致收敛由比较判别法易得
    - 一致连续由定义易得
  - <font color='chartreuse'>想要得到一致连续性并没有那么容易，最好还是用定义</font>

## 习题

### 求含参反常积分

- **收敛因子法**：
  - **Dirichlet反常积分**：$I = \dis\int^{+\infty}_0 \frac{\sin x}{x}dx$
    - **解（收敛因子+求导）**：引入收敛因子 $e^{-\a x}$
      - **积分号求导法**：积分号下求导计算得 $I(\a) = -\arctan\a + C$
      - **确定参数**：由于 $|I(\a)| \leq |\dis\int^{+\infty}_0 e^{-\a x}dx| = \frac{1}{\a}$，故 $\lim\limits_{\a\to\infty}|I(\a)| = 0$，即 $C = \dfrac{\pi}{2}$
      - **连续性**：易得 $f(x,\a) = \begin{cases} \dfrac{e^{-\a x}\sin x}{x}, & x\neq 0 \\ 1, & x=0 \end{cases}$ 连续，再易得其一致收敛，从而 $I(\a)$ 在 $[0,+\infty)$ 上连续
        - <font color='chartreuse'>连续性的作用是保证 $C$ 是固定的常数，不依赖于 $\a$</font>
      - **消去收敛因子**：由连续性得 $I = I(0) = \dfrac{\pi}{2}$
    - **解（构造+积分）**：
      - 对 $I = \dis\int^{+\infty}_0 e^{-px}\frac{\sin bx-\sin ax}{x}dx$ 用积分法求得 $I = \arctan\dfrac{b}{p} - \arctan\dfrac{a}{p}$
      - 令 $b=0$，易得 $G(p) = \dis\int^{+\infty}_0 e^{-px}\frac{\sin ax}{x}dx$ 一致收敛
      - 令 $p\to 0$，则 $G(0) = \lim\limits_{p\to 0}\arctan\dfrac{a}{p} = \dfrac{\pi}{2}\sgn(a)$
- **求导法**：
  - $I(x) = \dis\int^{+\infty}_0 e^{-t^2}\cos 2xt\text{dt}$
    - **解**：容易判断可积分号下求导
      - $I'(x)$ 分部积分得 $\dfrac{I'(x)}{I(x)} = -2x$，即对数积分。求得 $I(x) = Ce^{-x^2}$
      - 再由 $I(0) = \dfrac{\sqrt{\pi}}{2}$，故 $C = \dfrac{\sqrt{\pi}}{2}$
  - $g(\a,\b) = \dis\int^{+\infty}_0 \frac{\arctan\a x\cdot\arctan\b x}{x^2}dx$
    - **解**：易得 $g$ 关于 $\a,\b$ 都是奇函数，故只需讨论 $\a,\b>0$ 即可
      - 易得 $g(\a,\b)$ 一致收敛，$g_\a$ 在 $(\a>0)$ 上内闭一致收敛，且仅有一个可去间断点 $x=0$，故可求导
      - 易得 $g_{\a\b} = \dfrac{\pi}{2(\a+\b)}$
      - 先对 $\a$ 积分，由连续性求出 $C(\a)$，再对 $\b$ 积分，由连续性求出 $C(\b)$，最后得 $g = \cfrac{\pi}{2}\ln\cfrac{(\a+\b)^{\a+\b}}{\a^\a\b^\b}$
- **积分法**：
  - $I(\a) = \dis\int^{+\infty}_0 \frac{\arctan(\a x)}{x(1+x^2)}$
    - **解**：
      - 转化为 $\dis\int^{+\infty}_0 dx\int^\a_0 \frac{1}{(1+y^2x^2)(1+x^2)}dy$
      - 易得关于 $y\in [0,\a]$ 一致收敛
      - 积分换序
- **求导 + 微分方程**：
  - $\p(r) = \dis\int^{+\infty}_0 e^{-x^2}\cos rxdx$
    - **解**：
      - 易得其在 $r\in\R$ 上一致收敛
      - 易得其求导后一致收敛
      - 分部积分计算易得 $\p'(r) = -\dfrac{r}{2}\p(r)$，解得 $\p(r) = \dfrac{\sqrt{\pi}}{2}e^{-\frac{r^2}{4}}$
  - $F(y) = \dis\int^{+\infty}_0 \frac{\sin\sqrt{xy}}{x(1+x)}dx$
    - **解**：
      - 由于是奇函数，故只需讨论 $y>0$ 的情况
      - 求二阶导，易得均连续且内闭一致收敛，并由狄利克雷积分得 $F(y)-F''(y) = \pi$
      - 解得 $F(y) = \pi(1-e^{-y})$


### 转化为含参积分

- $I = \dis\int^{+\infty}_0 \frac{e^{-x^2}-e^{-2x^2}}{x}dx$
  - **解**：令 $t = x^2$
- $J = \dis\int^{+\infty}_0 \frac{e^{-ax^2}-\cos bx}{x^2}dx$
  - **解**：分部积分（积 $\dfrac{1}{x^2}$）后，可转化为泊松积分和狄利克雷积分

### 性质题

- **p含参积分比较法**：设 $\p(\l) = \dis\int^{+\infty}_0 x^\l f(x)dx$，若其在 $\l = a,b$ 时均收敛，则在 $[a,b]$ 上一致收敛
  - **证明**：
    - 瑕点只可能是 $0,+\infty$，下面分情况讨论
    - 先看 $[1,+\infty)$
      - $\p(\l) = \dis\int^{+\infty}_0 \frac{x^b f(x)}{x^{b-\l}}dx$，分子一致收敛，分母单调趋于0，故一致收敛
    - 再看 $[0,1]$
      - $\p(\l) = \dis\int^{+\infty}_0 \Big[ x^a f(x) \Big]x^{\l-a}dx$，左项一致收敛，右项单调趋于0，故一致收敛
- **端点收敛性**：
  - 设 $\p(t) = \dis\int^{+\infty}_c f(x,t)dx$ 在 $(a,b)$ 上一致收敛
  - 若 $f$ 在 $[c,+\infty)\times [a,b)$ 上连续
  - 则 $\p$ 在 $x=a$ 处也一致收敛
  - <font color='chartreuse'>若被积函数在端点连续，则一致收敛性可延拓到端点上</font>
  - **证明**：三个三角不等式即可
  - **推论**：不一致收敛性也可延拓到端点上


### 常用结论

- $\dis\int^{+\infty}_1 \frac{\sin x}{x^\a}dx$ 和 $\dis\int^{+\infty}_1 \frac{\cos x}{x^\a}dx$ 在 $(0,+\infty)$ 上内闭一致收敛 在 $(0,+\infty)$ 上内闭一致收敛
- $\dis\int^{1}_0 \frac{\sin x}{x^\a}dx$ 在 $(0,2)$ 上内闭一致收敛，$\dis\int^{+\infty}_1 \frac{\cos x}{x^\a}dx$ 在 $(0,1)$ 上内闭一致收敛


### 不连续的被积函数

- 设 $f$ 在任意有限区间可积，$\dis\int^{+\infty}_0 f(x)dx$ 收敛，则 $$\lim_{y\to 0+}\int^{+\infty}_0 e^{-xy}f(x)dx = \int^{+\infty}_0 f(x)dx$$
  - **证明（放缩法）**：
    - Abel判别法易得左式在 $y\in[0,1]$ 一致收敛，
    - 由 $f$ 可积性得有界性，设 $|f|\leq M \pad (x\in [0,A])$
    - 作差 + 区间分段 + 三角不等式
- 设 $f$ 在 $[0,1]$ 可积，在 $x=0$ 处右连续，则 $$\lim\limits_{y\to +\infty} \int^1_0 ye^{-x^2y^2}f(x)dx = \frac{\sqrt\pi}{2}f(0)$$
