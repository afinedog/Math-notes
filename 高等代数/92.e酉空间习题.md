
# 酉空间习题

## 正规算子

### 复数域

- **交换定义法**：正规算子是与伴随可交换的算子
- **谱定义法**：正规方阵是酉相似于对角阵的方阵
- **特征定义法**：设 $V$ 是 $n$ 维酉空间，则
  - $\p$ 是正规算子 $\LR \p$ 和 $\p^*$ 的特征向量相同，特征值共轭
  - **证明**：<font color='chartreuse'>应用（正规算子的伴随模长不变性）即可</font>
    - **必要性**：
      - 易得 $(\l I-\p)^* = \ol\l I-\p^*$
      - 设 $\a$ 是 $\p$ 的关于 $\l$ 的特征向量，则由正规算子的性质，$\|(\l I-\p)\a\| = \|(\l I-\p)^*\a\| = 0$，从而 $\a$ 也是 $\p^*$ 的关于 $\ol\l$ 的特征向量
    - **充分性**：逆推即可
  - **推论**：最小多项式的系数也是共轭关系
    - **证明**：由共轭的性质易得 $f(x) = 0 \LR \ol f(\ol x) = 0$。代入矩阵即得结论
- **自伴分解定义法**：设 $V$ 是 $n$ 维酉空间，则
  - $\p$ 是正规算子 $\LR \p = \p_1+i\p_2$，其中 $\p_1,\p_2$ 都是自伴算子且彼此可交换
  - **证明**：
    - **充分性**：
      - 此时 $\p^* = \p_1 + i\p_2$，故 $\p\p^* = (\p_1+i\p_2)^2 = \p^*\p$。由交换定义即得结论
    - **必要性**：
      - 取 $\p_1 = \cfrac{\p+\p^*}{2}，\p_2 = \cfrac{\p-\p^*}{2i}$，易得自伴且可交换

- **多项式定义法**：设 $V$ 是 $n$ 维酉空间，则
  - $\p$ 是正规算子 $\LR$ 存在复多项式 $f$ 使得 $\p^* = f(\p)$
  - **充分性**：由交换定义易得结论
  - **必要性**：
    - **引理**：谱分解具有多项式传递性
      - 由谱分解，$\p = \l_1E_1 + \cdots + \l_k E_k$
      - 易得正交投影满足自伴性，故 $\p^* = \ol\l_1E_1 + \cdots + \ol\l_k E_k$
      - 再由正交投影的标准正交性，$\p^m = \l_1^m E_1 + \cdots + \l_k^m E_k$
      - 从而 $f(\p) = \sum\limits^k_{i=1} f(\l_i) E_i$
    - 再取多项式的Kronecker基即可
      - 取 $f_j(x) = \prod\limits_{i\neq j} \cfrac{x-\l_i}{\l_j-\l_i}$，则其满足 $f_j(\p) = E_j$
      - 取 $f(x) = \sum\limits^k_{j=1} \ol \l_j f_j(x)$，则其满足 $f(\p) = \sum\limits^k_{j=1} \ol \l_j E_j = \p^*$
- **极分解定义法**：设 $V$ 是 $n$ 维酉空间，则
  - $\p$ 是正规算子 $\LR \p = w\psi$，其中 $w$ 是酉算子，$\psi$ 是半正定自伴算子，且彼此可交换
  - **充分性**：
    - 此时 $\p^* = \psi w^{-1}$。由可交换性得 $\p\p^* = w \psi^2 w^{-1} = \psi^2 = \p^*\p$，从而正规
  - **必要性**：
    - 设 $\l_i$ 为特征值，$r_i = |\l_i|，s_i = \cfrac{\l_i}{|\l_i|}$
      - （当 $\l_i = 0$ 时，$r_i = 0，s_i = \pm 1$）
      - <font color='chartreuse'>$r_i$ 是模长，$s_i$ 是方向向量</font>
    - 取 $w = \sum\limits^k_{i=1} s_iE_i，\psi = \sum\limits^k_{i=1} r_iE_i$ 即可
      - <font color='chartreuse'>酉算子负责旋转，正交投影负责投影</font>

#### 性质

- **伴随交换传递性**：设 $\p$ 是正规算子，$\p\psi = \psi\p$，则 $\p^*\psi = \psi\p^*$
