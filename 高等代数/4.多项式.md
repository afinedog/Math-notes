### 同余

- **同余**：设 $f_1(x),f_2(x),g(x)\in F[x]$，若 $g(x)\mid \big(f_1(x)-f_2(x)\big)$，则 $f_1(x),f_2(x)$ 模 $g(x)$ 同余
  - **加法传递性**：同余式的加法和也同余
  - **乘法传递性**：同余式的乘法积也同余
  - **证明**：定义易得
- **互素的同余定义**
  - **推论**：互素的乘积传递性、素因式的单元性
- **中国剩余定理**：设 $g_1(x),...,g_s(x)$ 是 $F$ 上两两互素的多项式，$f_1(x),...,f_s(x)$ 是 $F$ 上任意多项式，则存在 $f(x)$ 使得 $f(x)\equiv f_i(x) \pmod g_i(x)$ 均成立
  - **证明**：
    - 设余式向量函数 $\sigma:F[x]\to F[x]^s，\p\mapsto (r_1(x),...,r_s(x))$ ，其中 $r_i(x)$ 是 $\p$ 在 $g_i(x)$ 下的余式
    - 由同余的加法乘法传递性，只需找到 $h_i(x)$ 使得  $\sigma h_i(x) = \vec e_i$，那么 $f(x) = \sum\limits^s_{i=1} f_ih_i$ 就是所需的函数
      - 取 $\g_i(x) = \prod\limits_{1\leq j\leq s，j\neq i} g_i(x)$，易得其余 $g_j\pad (j\neq i)$ 互素，故存在 $u_i\g_i + v_ig_i = 1$，取 $h_i = u_i\g_i$ 即可

- **唯一分解定理**：（$F$ 上每个非零多项式）均可分解为 $F$ 上（有限个不可约多项式的乘积），同时在排列意义下唯一
  - **归纳证明**：
    - **存在性**：对次数归纳易得
    - **唯一性**：对因式个数归纳易得
- **余数定理**：若 $f(x)$ 以 $c$ 为根，则 $(x-c)\mid f(x)$
  - **证明**：带余除法易得
  - **推论（根数定理）**：$f(x)$ 不同根的个数不超过 $\deg f(x)$
    - **证明**：唯一分解定理 + 余数定理直得

      - 若 $A$ 不可逆，则使用**扰动法**。设 $A_\l = \l I_n + A$，则 $f(\l) = |A_\l|$ 至多只有 $n$ 个不同根，从而题设等式对无穷多个 $\l$ 成立
        - 再由于题设等式两边均为多项式，故只能是两式相等
        - 最后取 $\l = 0$ 即可

    - 归纳法证明
    - 反证法证明：$f(a) = 0，g(x) = (x-a)(x-\bar{a})$，则 $f(a) = q(a)g(a)+r(x) = 0$，发现 $r(x) = 0$，所以 $\exist f(x)$ 可以被二次多项式 $g(x)$ 整除

### 多项式函数

- **单位根**
  - **n次单位根**：多项式 $w^n-1$ 的根
  - **n次本原单位根**：$|w| = 1$ 的n次单位根
    - **棣莫弗公式**：详见复变
    - **本原性**：$w = \cos\dfrac{2m\pi}{n} + i\sin\dfrac{2m\pi}{n}$ 是n次本原单位根 $\LR$ m和n互素
    - 也就是说，只要是缩系，就可以表出所有的单位根（×）
      - 这又不是整数……而且它们的表出性是体现在棣莫弗公式中的，而棣莫弗公式是把乘方变成了加法（倍数），缩系乘以一个倍数当然可以表出所有的缩系
- **分圆多项式**：$\varPhi_m(x) = \prod\limits_{1\leq k<m，(k,m)=1} (x-w_k)$
  - **根的本原性**：它的根都是复平面上单位圆的 $m$ 等分点
  - **因子性**：$x^n-1$ 可被分解为 $(x-1)\Big[ \prod\limits_{d\mid n} (x^{d-1}+...+1) \Big]\varPhi_n$
  - **整系数性**：它的系数都是整数
    - **归纳证明**：
      - $m=1$ 时，易得 $\varPhi_1(x) = x-1$
      - $m>1$ 时，可唯一分解为 $m=p_1...p_s$
        - 若 $s=1$，则素数 $p$ 的完系就是缩系，单位根全部为本原单位根
          - 也就是熟知的 $x^n -1 = (x-1)(x^{p-1}+...+x+1)$
        - 若 $s\geq 2$，则 $\Phi_m$ 就是 $x^n-1$ 除去所有本原 $d$ 次单位根的根式 $x-w_d$ 后（其中 $d\mid m$）剩下的部分
          - 即 $\varPhi_m(x) = \cfrac{x^m-1}{\prod\limits_{1\leq d<m,d\mid m}\varPhi_d(x)}$
        - 由归纳假设，分母中的多项式是整系数的本原多项式，而易得 $x^m-1$ 是本原多项式，因此 $\varPhi_m(x)$ 也是本原多项式

### 有理系数多项式

- **本原多项式**：各项系数均为互素整数的多项式
  - **本原性**：所有非零有理系数多项式 $f$ 均可写成本原多项式 $g$ 的有理数倍 $\dfrac{a}{b}g$
- **高斯引理**：本原多项式 $f,g$ 的乘积 $h$ 也是本原多项式
  - **证明**：反设存在公因子 $d$，其中 $p$ 是 $d$ 的一个素因子
    - 由本原性，$p$ 不整除 $f$ 和 $g$ 的任意系数
    - 将 $f,g$ 被 $p$ 除后的余式设为 $f_1,g_1$，则 $h = pf_0g_0 + f_1g_1$
      - 此时 $k+s$ 次系数 $h_{k+s} = kp + r_k,\ol r_s$ 与 $p$ 互素，矛盾
  - **推论（除法传递性）**：设 $f,g\in \Z[x]$。若 $g$ 是本原多项式，则 $f$ 除以 $g$ 的商 $q$ 是整系数多项式。若 $f$ 是本原多项式，则 $q$ 是本原多项式
    - **证明**：
- **有理化整定理**：若 $f$ 是有理系数多项式的乘积，则其为也整系数多项式的乘积
  - **证明**：易得
- **有理根定理**：若 $x = \dfrac{r}{s}$ 是 $f(x) = a_nx^n + ... + a_0$ 的根，则 $r\mid a_0，s\mid a_n$  
  - **证明**：取 $f(\dfrac{r}{s}) = 0$ 后两边同乘 $s^n$ 即可
  - **理解**：最高次项的分母次数最大，要防止其被孤立。常数项的分子次数最小，要防止其被孤立（类似数论中的证明调和级数不为整数）
    - 分子必须有 $x-r$ 因式，分母必须消除 $s$ 因子
- **Eisenstein判别法**：
  - 设 $f(x) = a_nx^n + ... + a_0$ 是整系数多项式
  - 若存在素数 $p$ 满足

    - $p\nmid a_n$
    - $p^2\nmid a_0$

### 习题 

- 求出所有（两边长度之比是有理数）且（内角为有理数）的三角形
  - **解**：
    - 设 $A = \dfrac{u}{v}$，$n = 360v$，则 $360\mid nA$
      - 设 $w = \cos A + i\sin A$，则 $w^n = 1$
      - 从而共轭根式 $g(x) = x^2-2x\cos A + 1$ 整除 $x^n-1$
    - 若两边长度之比均为有理数，由余弦定理易得 $\cos A$ 也为有理数，设为 $\dfrac{s}{2t}$，则 $tg\mid (x^n-1)$ 是本原多项式
      - 由本原多项式的除法传递性，只能是 $t=1$，即 $2\cos A = s$。即 $\cos A$ 只能是 $\pm 1,\pm \dfrac{1}{2},0$，即 $A = 60\degree,90\degree,120\degree$
- 整系数多项式满足 $(b-a)\mid \Big[ f(b)-f(a) \Big]$
  - **证明**：设 $f$ 被 $x-a$ 除的商为 $q(x)$，余数为 $r$
    - 则由余式定理，$r = f(a)$。再代入 $x=b$ 即得 $f(b)-f(a) = q(b)(b-a)$
- **换元法应用艾氏判别法**：
  - 我们希望能将首项系数变小，后面的其它系数均变大，这可以用换元的方法来实现
    - **换元可约定理**：多项式可约 $\LR$ 多项式换元后可约
      - **证明**：若 $f(x) = f(y+c) = g(y) = g_1(y)g_2(y)$，则 $f(x) = g_1(x-c)g_2(x-c) = h_1(x)h_2(x)$，当然可约
  - 设 $x = y+c$，若 $f(y+c)$ 在整数环上不可约，则 $f(x)$ 在有理数域上不可约
- **同余化简法应用艾氏判别法**：
  - $x^p+px+1$，其中 $p$ 是奇素数
  - **解**：设 $x = y+1$，则 $f(x) = (y-1)^p+p(y-1)+1$
    - 发现 $f(x) \equiv y^p + (-1)^p + 1 \equiv y^p \pmod p$
    - 此时素数 $p$ 满足艾氏判别法，故不可约
- 设 $a,b,c\in \Q$，若 $\dfrac{a}{b} + \dfrac{b}{c} + \dfrac{c}{a}，\dfrac{b}{a} + \dfrac{c}{b} + \dfrac{a}{c}$ 都是整数，则 $|a| = |b| = |c|$
  - **证明**：设 $f(x) = (x-\dfrac{a}{b})(x-\dfrac{b}{c})(x-\dfrac{c}{a})$，则由韦达定理易得各项系数都是整数（**核心**）
    - 从而 $f(x)$ 是整系数多项式，由有理根定理，其根只能为 $|x_i| = 1$，从而 $|\dfrac{a}{b}| = |\dfrac{b}{c}| = |\dfrac{c}{a}| = 1$，**证毕**

      - 这个其实用棣莫弗公式就可以证明，但也可以采用缩系（费马定理）的方法：
      - 设$x^n-1 = f(x)g(x)h(x)$，若$a^p$不是$f(x)$的根，但它同时也是$x^n-1$的根，所以设它为$g(x)$的根，所以$a$是$g(x^p)$的根，$f(x)和g(x^p)$有公因式
      - 再由$f(x)$不可约，得到$f(x)\mid g(x^p)$，设补式为$u(x)$
      - 但在p元域上$g(x^p) = (g(x))^p$（费马定理），从而$f(x)和g(x)$有公因式$v(x)$，则$x^n-1$有重因式，但通过求导发现它不可能有重因式，矛盾
    - $a$其实就是m次本原单位根$w$，因此都是$f(x)$的根，则$\varPhi_m(x)\mid f(x)$
    - 由$f(x)$不可约，得到$f(x) = \varPhi_m(x)$，从而分圆多项式不可约

- 少$x_i^{n-1}$一行的Vandermonde行列式
  - 首先扩充为V，然后比较系数即可，得到$\triangle_n = \sigma_1\prod (x_j-x_i)$
  - 或者直接通过结果发现可以把第$n-i+1$行的$(-1)^i\sigma_i$倍加入第n行，直接因子分离
- $(x_{ij})_{n×n}$的行列式是$n^2$自变量的n次齐次多项式，它在有理数域上不可约（行列式互异性）
  - **证明**：
    - 以每个字母为主元，可以分为$f = ax_{ij} + b$
    - 反设$f= f_1f_2$
      - 由行列式的互异性，$x_{11}$只能在一个多项式中出现，$f = af_2x_{11} + bf_2$
      - 同理$x_{1j}、x_{i1}$只能在$f_1$中出现
      - 所以$x_{ij}$最后都含于$f_1$，$f_2$退化，发现$f$不可约
- 裴蜀方程的多项式空间解法：
  - $u(x)f(x) + v(x)g(x) = (f(x),g(x))$，已知f、g、公因式d
  - 以幂为基，构造n维列向量，得到列向量方程（线性方程组）（矩阵乘法），使用矩阵求逆或Cramer法则即可
- **一般裴蜀等式的表出性、因子次数关系**：$u(x)f(x)+v(x)g(x) = h(x)$（$\forall deg\ h(x) < deg\ f+deg\ g$） $\Leftrightarrow$ $(f,g) = 1$
  - 充分性：令$h(x)=1$即可得到相应的u和v
  - 必要性：直接把裴蜀等式两边乘h即可，次数关系也满足
- **结式**：$f(x)$是n次多项式，$g(x)$是m次多项式，则$R(f,g)$是它们的结式
  - $(f(x),g(x)) = 1 \Leftrightarrow R(f,g)\neq 0$
  - **证明**：
    - **（多项式基）基的线性无关性**：由裴蜀等式的表出性，在多项式空间中，$m+n$维空间的任意向量$h(x)$都可以被多项式基$x^if(x)和x^jg(x)$所表出，所以这些幂基组成的矩阵的行列式（结式）不为0
    - **直接使用非零式**：$u(x)f + v(x)g = 0$，若不存在公因子，则由互素纯性得到$g(x)\mid u(x)$，但因为次数关系，得到矛盾，所以$u(x)=0$，仅存在零解，线性无关，$R(f,g)\neq 0$
- **二元高次方程组**
  - 求方程组的解，本质就是求公共解（公因子），而选定数域为$F[y]$，则只要使$R_x(f,g) = 0$，即有公共解。这通过选取y的值来实现
  - 列出结式方程，解得y的值，从而化为一元方程组，然后可以得到解
- **对称多项式的迭代（Newton公式引理）**：
  - 三个元：$s_k-\sigma_1 s_{k-1} + \sigma_2 s_{k-2} - \sigma_3s_{k-3} = 0$
  - n个元：$s_k+\sum\limits^n (-1)^i\sigma_is_{k-i} = 0$
  - **证明**：构造$f(w) = (w-x_1)(w-x_2)...(w-x_n)$，然后由$x_i^{k-3}f(x_i) = 0$，相加得到对称式，它就是递推公式
- **Newton公式**：在$k\leq n$时，有$s_k + \sum (-1)^{i}\sigma_{k-i} + (-1)^kk\sigma_k = 0$
  - **证明**：首先$f(x)$求微商，用分式$\sum\frac{f(x)}{x-x_i}$表示，然后$x^{k+1}f'(x)$裂项配凑s，得到$x^{k+1}f'(x) = f(x)(\sum s_ix^{k-i}) + \sum x^{k+1}\frac{f(x)}{x-x_i}$
  - 然后再不配凑再计算一次，比较两端系数，就得到Newton公式
