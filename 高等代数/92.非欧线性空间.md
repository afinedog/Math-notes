# 非欧线性空间

- 由泛函分析知，严格凸的线性空间 $\LR$ 内积空间

## 酉空间（复数域上的共轭内积空间）

- **内积**：$\overline{X}^TY$
  - **共轭对称性**：交换位置后需要整体取共轭
  - **共轭双线性**：提取第一个系数时取共轭，第二个则不取共轭
  - **正定性**
- **Cauchy-Schwarz不等式**：$|(\alpha,\beta)| \leq (\alpha,\alpha)(\beta,\beta)$
- **三角不等式**：$|\alpha+\beta| = |\alpha| + |\beta|$
- **无夹角性**：
- 正交
  - 正交关系具有对称性（0的共轭为其本身）
  - 线性无关性
- **度量矩阵**：Hermite方阵
  - **共轭对称性**：$\overline{S}^T = S$<font color='chartreuse'>（学完泛函后可知，实方阵的对称阵、酉方阵的共轭对称阵，都是矩阵算子的共轭算子，故可用 $S^*$ 表示）</font>
  - **共轭相合**：$B = \overline{P}^TAP$
- **标准正交基**
  - **正交化方法**
  - **酉方阵（正交方阵）**：$\overline{P}^T = P$

### 酉相似

- **酉变换（正交变换）**：内积不变
  - 标准正交基下为酉方阵（正交阵）
  - 酉变换
- **Hermite变换（对称变换）**：矩阵是Hermite方阵（共轭对称矩阵）
  - 标准正交基下为Hermite方阵
- **伴随变换**
- **规范变换**：与伴随变换可交换的变换
- **规范方阵**：$A\overline{A}^T = \overline{A}^TA$
  - 酉方阵、Hermite方阵、反Hermite方阵
  - **酉相似封闭性**
    - **理解**
      - 规范方阵是对称矩阵和酉矩阵的超集
      - 酉相似是（相合）对称变换和酉变换的超集
  - **不变子空间直和扩张性**：上三角规范方阵 $\Rightarrow$ 对角矩阵
    - **证明**：老一套，讨论迹证明矩阵为0，以及规范方阵的对角传递性（对角阵乘积性）
- **部分对角化（复Schur分解定理）**：任意复方阵酉相似于上三角形矩阵
  - **证明**：数学归纳法
    - 还是特征向量、退化那一套
- **规范标准型**：规范方阵酉相似于对角矩阵
  - **酉方阵标准型**：$diag(\lambda_i)$
    - 其中$|\lambda_i| = 1 \Leftrightarrow \lambda_i = cos\alpha_i + isin\alpha_i$（复数三角表示）
    - **证明**：由共轭转置是逆，模为1的旋转向量恰好满足
      - 上旋 $\alpha$ 和下旋 $\alpha$，综合起来等于没有旋转
  - **Hermite阵标准型**：$diag(\lambda,...,\lambda_p,..., -\lambda_{r-p},O_{n-r})$
    - $\lambda$ 是实数
    - **证明**：由共轭对称性，只能是实数
  - **反Hermite方阵标准型**：$diag(\lambda_1,...\lambda_s,O_{n-s})$
    - $\lambda$ 为纯虚数
    - **证明**：由反共轭对称性，只能是纯虚数
  - **关系**：H和反H阵，相差一个因子 $i$（$A = iB$）

## 其它内积空间

### Lorentz空间（以D为度量矩阵的内积空间）

- **事件的向量**：$X = (x,y,z,ct)$
  - 前三个分量是事件发生位置的坐标
  - c是光速，t是事件发生时间
- **距离**：$|PQ| = (X-Y)^TD(X-Y)$，其中 $D = \diag(1,1,1,-1)$
- **洛伦兹空间**：上述四维空间是内积空间
  - **证明**：
- **洛伦兹变换**：保持洛伦兹内积的变换

### 辛空间（零定度量矩阵内积空间）

- **实数域上的零定二次型**：反对称方阵（行列式归零性）
  - **证明**：
    - 二次型中给的是转置证法
    - 也可以用“分”析法
- **辛内积**：反对称方阵 $A$ 作为度量矩阵的内积 $(X,Y) = X^TAY$
  - 无对称性
  - 无正定性
  - 模零性
- **辛空间**：类似于欧氏空间，定义了辛内积的空间

### 双线性函数（内积函数）

- **双线性函数**：满足双线性的向量函数
  - 单元函数 $f_\alpha$：固定一个向量（但是仍然满足双线性）
  - 双元函数 $f$：两个向量均不固定
- **坐标表示**：设基为 $M = (\vec{m}_i)$，其下坐标为 $X,Y$，则 $f(\alpha,\beta) = X^TAY$
  - **代数构造性证明**：基分解后，左式为 $\sum\limits^n_{i=1}\sum\limits^n_{j=1} f(x_i\vec{m}_i,y_j\vec{m}_j)$，右式为 $\sum\limits^n_{i=1}\sum\limits^n_{j=1} x_iy_ja_{ij}$
    - 则只需要定义 $a_{ij} = \cfrac{f(x_i\vec{m}_i,y_j\vec{m}_j)}{x_iy_j}$
    - 由函数线性，$a_{ij} = f(\vec{m}_i,\vec{m}_j)$，此时它只依赖于 $f$ 和 $M$
  - **本质**：内积是添加新性质的双线性函数，而新性质体现在度量矩阵上
  - **双线性函数在基M下的矩阵**：$A = (f(\alpha_i,\alpha_j)_{ij})_{n×n}$
      - A可以是任意矩阵（无正定性、对称性）
      - **非度量性**：$f(\alpha,\alpha) = \frac{1}{2}(A+A^T)$，是对称矩阵
        - 由A的无正定性，不能用 $|\alpha|^2 = f(\alpha,\alpha)$ 定义向量的长度
      - **相合性**：在两组基下，$f$ 的矩阵满足相合关系
- **对偶空间**：V上的双线性函数整体：$L(V,V,F)$
  - 它是线性空间，但不是内积空间（双线性函数有加法和数乘，但没有乘法）
  - 数组空间同构

### 正交性

- **$\alpha$ 关于 $\beta$ 左正交**：$f(\alpha,\beta) = 0 \Leftrightarrow \alpha_{\perp_L}\beta$
  - **无对称性**：左正交和右正交应分别讨论
    - 例子：$A = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}$，则 $f(e_1,e_2) = 0$，但 $f(e_2,e_1) = 1$
- **$V$ 在 $f$ 下的左根基**：$V^{\perp_{L}} = \{\beta\in V \mid \beta \perp_L V\}$
  - **求解左根基**：定义即可
- **维数定理**：
  - 左、右根基的维数（解空间的维数）：$dimV^{\perp_L} = dimV^{\perp_R} = dimV-rankA$
  - 左、右根基的关系：
    - 左根基：$AY=0$ 的解
    - 右根基：$Y^TA = 0（A^TY = 0）$ 的解
- **子空间 $W$ 的左根基**：设 $W$ 的基矩阵为 $B$
  - **维数定理**：
    - $dimW^{\perp_L} = dimW^{\perp_R} = dimV - dimW$
    - $dimW^{\perp_L} = dimV - rank\ AB$
    - $dimW^{\perp_R} = dimV - rank\ B^TA$
    - **证明**：
      - $B$ 是子空间的基，从而非退化（但是不是方阵，不可逆）
      - $A$ 是双线性函数的变换阵，非退化
      - 非退化矩阵相乘，秩不变，从而相等
      - 再由根基的补性，得维数互补
  - **对称性**：$(dimW^{\perp_L})^{\perp_R} = dimW =  (dimW^{\perp_R})^{\perp_L}$
- **双线性函数的秩**：$rankA$
- **非退化映射**：$rankf = dimV$
  - 等价条件：
    - 左根基 或 右根基为 $\bold{0}$
      - **证明**：维数定理即可
    - $\forall \alpha$，设 $\begin{cases} f_\alpha:\beta\mapsto f(\alpha,\beta) \\ \varphi_\alpha: \beta\mapsto f(\beta,\alpha) \end{cases}$，则 $\alpha\mapsto f_\alpha/\varphi_\alpha$ 是线性空间到对偶空间的同构
      - **证明**：
      - **理解**：核为0，从而是双射，从而同构
    - $\forall \varphi，\exist! \alpha_1,\alpha_2，\forall \beta\in V$，$\varphi(\beta) = (\alpha_1,\beta) = (\beta,\alpha_2)$
      - **证明**：
      - **理解**：伴随变换唯一性
- **非退化子空间**：$f|_W$ 非退化
  - 等价条件：
    - *可逆性*：$A_1$ 可逆
    - *不交性*：$W\cap W^{\perp_L} = W\cap W^{\perp_R} = \bold{0}$
    - **证明**：不难，重要的是理解
  - **直和性**：若 $W$ 是 $V$ 关于 $f$ 的非退化子空间，则 $V = W\oplus W^{\perp_L} = W\oplus W^{\perp_R}$
    - **证明**：维数定理 + 不交性
- **正交的补性**：
  - **左右正交的扩充性**：$(S^{\perp_{L}})^{\perp_{R}} \supseteq S$
  - **包含关系的相反性**：$W \subseteq U \Rightarrow (W^{\perp_{L}}) \supseteq U^{\perp_{L}}$

### 对称正交（辛空间）

- 对称性
  - **对称双线性函数**：$f(\alpha,\beta) = f(\beta,\alpha)$，A对称
  - **反对称双线性函数**
- **传递性**：$f$ 定义的正交关系对称 $\Leftrightarrow$ $f$ 对称或反对称
- **对称双线性标准型**：若 $f$ 对称，则有正交基 $M$，在其下为对角阵 $A = diag(a_1,...,a_r,0,...,0)$
  - $r = rankf$
  - **证明**：二次型标准型直得
  - **推论**：实数域上，A是 $\pm I$ 准对角，复数域上，A是 $I$ 准对角
- **反对称双线性标准型**：$D = diag(A_1,...,A_s,0,...,0)$
  - A是左90度旋转阵
  - **证明**：数学归纳法
    - $n=1$ 和 $f=0$ 都易得
    - $f\neq 0$ 时
      - **求逆算法**：设 $\alpha_2 = \frac{\beta}{f(\alpha_1,\beta)}$，则 $f(\alpha_1,\alpha_2) = 1$
      - 取两个满足 $f(\alpha,\alpha) = 0$ 的向量 $\alpha_1,\beta$，则
      - 取 $W$ 为它们生成的子空间，则限制映射的矩阵为 $D_1 = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$
      - 由正交补直和性，退化二阶
      - 由归纳假设，得 $D_2$，合并得总标准型 $D = D_1D_2$
  - **本质**：反对称方阵标准型，辛空间的度量矩阵标准型

### 共轭双线性（酉空间）

- **Hermite共轭双线性函数**：$f(\alpha,\beta) = \overline{f(\beta,\alpha)}$，A是Hermite方阵
- **Hermite型**：单变量函数 $H(\alpha) = f(\alpha,\alpha)$
- **反Hermite**：反共轭对称性

## 更多的例子

- **快速构造酉方阵**：Hermite方阵 $H$ 
  - $I\pm iH$ 可逆
  - $U = (I+iH)(I-iH)^{-1}$ 是酉方阵
  - **证明**：
    - 由H阵标准型得，$I\pm iH = P(I\pm iD)P^{-1}$，非退化
    - 由H阵标准型得，$U = P(I+iD)(I-iD)^{-1}P^{-1}$，计算得特征值模均为1，从而是酉方阵
  - **理解**：标准型的应用
  - **本质**：快速构造酉方阵的方法
- **迹相等定理**：酉相似的方阵，所有元的模平方和相等 $\sum\limits_{1\leq i,j\leq n} |a_{ij}|^2 = \sum\limits_{1\leq i,j\leq n} |b_{ij}|^2$
  - **证明**：由于 $A\overline{A}^T$ 的迹就是其所有元的模平方和
    - 再由迹与相似的关系即可得到结果
  - **本质**：酉相似矩阵的迹相等
- **Schur不等式**：复数域上，$tr(\overline{A}^TA) = \sum\limits_{1\leq i,j\leq n} |a_{ij}|^2 \geq \sum\limits^n_{i=1} |\lambda_i|^2$
  - **证明**：酉相似于上三角矩阵，其中对角元是特征值
    - 则不等号易得
  - **推论**：
    - 只有规范方阵（可酉相似于对角矩阵）才成立等号
    - 实数域中，$\|A\|_F^2 = tr(A^TA) \geq \sum\limits^n_{i=1} \lambda_i^2$
      - 只有规范方阵（可正交相似于对角矩阵）才成立等号
- **O.Tausky——樊畿fan'ji**：两个H方阵 $H_1,H_2$，若 $H_1$ 正定，且 $H_1H_2$ 也是H方阵，则 $H_1H_2$ 正定 $\Leftrightarrow$ $H_2$ 正定
  - **证明**：
    - *过渡矩阵累加*：设 $H_1$ 的对角化过渡矩阵是 $P$，$H_2$ 的酉相似过渡矩阵是 $U$，$Q = P_1U$
    - $H_1^{-1} = \overline{P}^TP，H_2 = \overline{P}^TDP$，则 $H_1H_2 = P^{-1}DP$
    - 由于都以D为标准型，故 $H_1H_2$ 和 $H_2$ 正定性相同
  - **理解**：标准型在正定上的体现
  - **本质**：
- **H方阵平方根存在性**：半正定Hermite方阵 $H$ 存在唯一的平方根，其也是半正定Hermite方阵
  - **证明**：
    - 存在性：过渡矩阵不变，标准型取根号即可（$P^T\sqrt{D}P$）
    - 唯一性：反证，特征标准型唯一性已知
      - 过渡矩阵唯一性：根据不同特征值写成分块形式，分别比较即可
- **酉相抵标准型**：任意复方阵 $A_{m\times m}$，存在酉方阵 $U_{m\times m},Q_{n\times n}$，使得 $UAQ = diag(\mu_1,...,\mu_r,O)$
    - $r = rankA$
  - **奇异值**：对角元 $\mu$ 为 $A$ 非零特征值的算数平方根
  - **左右奇异向量**：U和Q的列向量。其满足 $\begin{cases} A^Tu_i = \mu_i u_i \\ Aq_i = \mu_i q_i \end{cases}$
  - **构造性证明**：$\overline{A}^TA$ 是半正定实对称矩阵，设其到实对称标准型的正交过渡矩阵为 $Q$，对角元单位化矩阵为 $D$
    - 则设 $B = AQD$，那么 $\overline{B}^TB = diag(I_r,O_{n-r})$，$B$ 标准正交
    - 设 $B$ 到 $I$ 的过渡矩阵为 $U$，则 $UB = diag(e_1,...,e_r,0,...,0)$
    - 从而 $UAQ = UBD^{-1}$ 就是标准型，且都是酉过渡方阵
  - **本质**：它不是任意复矩阵均可对角化的意思，因为 $U$ 和 $Q$ 不是互逆关系
    - 它更类似于初等矩阵对角化（秩相抵标准型），但对P和Q做了更严格的约束（必须是酉方阵）
  - **推论**：
    - **奇异值分解**：$A = U_1diag(\mu_1,...,\mu_r,O)\overline{U_2}^T$
    - 实对称矩阵的奇异值是特征值的绝对值
- **极分解**：任意复方阵 $A_{n\times n}$ 均可以分解为半正定H方阵 $H$ 和酉方阵 $U$ 的乘积
  - $A = H_1U$ 或 $A = UH_2$
  - **唯一性**：$H$ 由A唯一确定
  - **构造性证明**：奇异值分解为 $A = U_1DU_2$
    - 添项得 $A = (U_1D\overline{U_1}^T)(U_1U_2) = (U_1U_2)(\overline{U_2}^TDU_2)$
    - 设 $U = U_1U_2，\begin{cases} H_1 = U_1D\overline{U_1}^T \\ H_2 = \overline{U_2}^TDU_2 \end{cases}$
    - **唯一性**：$A\overline{A}^T = H^2$，由平方根唯一性得到 $H$ 唯一性
  - **本质**：和数值线性代数一样，所谓的分解本质都是标准型 + 添项 = 新过渡矩阵
- **Witt扩张定理**：$V_m$ 中两组向量 $S=\{\alpha_i\}_m，T=\{\beta_i\}_m$ 满足 $(\alpha_i,\alpha_j) = (\beta_i,\beta_j)$，则存在正交变换 $\mathscr{A}:\alpha_i\mapsto \beta_i$
  - **证明**：选取标准正交基 $M$，其下坐标分别为 $X_i,Y_i$，从而 $X_i^TX_i = Y_i^TY_i$，即坐标阵 $A^TA = B^TB$
    - 由半正定实对称性，正交相似于对角矩阵，过渡矩阵 $P$
    - 再通过单位化矩阵 $D$ 变为 $A_1^TA_1 = B_1^TB_1 = diag(I_r,O_{m-r})$
    - $A_1、B_1$ 扩充为标准正交基矩阵 $Q_1,Q_2$
    - 设 $P_2 = Q_2Q_1^{-1}$，可证其为 $\mathscr{A}$ 的坐标阵
  - **理解**：标准化后扩张，取过渡矩阵即可，此过程由正交封闭性保正交
  - **本质**：保内积的两个向量组，可扩充为正交变换的像与原像
