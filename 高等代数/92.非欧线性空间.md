# 非欧线性空间

- 由泛函分析知，严格凸的线性空间 $\LR$ 内积空间

## 酉空间（复数域上的内积空间）

- **内积**：设标准正交基下坐标为 $X,Y$，则 $(\a,\b) = \overline{X}^TY$
  - **共轭对称性**：$(\a,\b) = \ol{(\b,\a)}$
  - **共轭双线性**：提取第一个系数时取共轭，第二个则不取共轭
  - **自反正定性**
- **Cauchy-Schwarz不等式**：$|(\a,\b)| \leq (\a,\a)(\b,\b)$
- **三角不等式**：$|\a+\b| = |\a| + |\b|$
- **几何意义**：
  - **无夹角性**：酉空间中不能用内积定义向量的夹角
  - **正交**：酉空间中依然可以定义正交，但不再具有垂直意义
    - **对称性**
    - **传递性**
    - <font color='chartreuse'>无自反性（正交关系不是等价关系）</font>
- **度量矩阵**：Hermite方阵
  - **共轭对称性**：$\overline{S}^T = S$<font color='chartreuse'>（学完泛函后可知，实方阵的对称阵、酉方阵的共轭对称阵，都是矩阵算子的共轭算子，故可用 $S^*$ 表示）</font>
  - **共轭相合**：$B = \overline{P}^TAP$
- **标准正交基**
  - **正交化方法**
  - **酉方阵（共轭正交方阵）**：$\overline{P}^TP = I$

### 酉相似

- **酉变换（共轭正交变换）**：保持内积不变的变换
  - **变换阵**：酉方阵（共轭正交阵）
- **Hermite变换（对称变换）**：
  - **变换阵**：Hermite方阵（共轭对称阵）
- **伴随变换**
- **正规变换（规范变换）**：与伴随变换可交换的变换
- **正规方阵（规范方阵）**：$A\overline{A}^T = \overline{A}^TA$
  - 酉方阵、Hermite方阵、反Hermite方阵
  - **酉相似封闭性**
    - **理解**
      - 规范方阵是对称矩阵和酉矩阵的超集
      - 酉相似是（相合）对称变换和酉变换的超集
  - **不变子空间直和扩张性**：上三角规范方阵 $\Rightarrow$ 对角矩阵
    - **证明**：老一套，讨论迹证明矩阵为0，以及规范方阵的对角传递性（对角阵乘积性）
- **部分对角化（复Schur分解定理）**：任意复方阵酉相似于上三角形矩阵
  - **证明（矩阵归纳法）**：
    - 还是特征向量、退化那一套
- **规范标准型**：规范方阵酉相似于对角矩阵
  - **酉方阵标准型**：$\diag(\l_i)$，其中 $|\l_i| = 1$，即 $\l_i = \cos\a_i + i\sin\a_i$（复数三角表示）
    - **证明**：由共轭转置是逆，模为1的旋转向量恰好满足
      - 上旋 $\a$ 和下旋 $\a$，综合起来等于没有旋转
  - **Hermite阵标准型**：$\diag(\l_1,...,\l_p, -\l_{p+1},..., -\l_{r},O_{n-r})$，其中 $\l$ 是实数
    - **证明**：由共轭对称性，只能是实数
  - **反Hermite方阵标准型**：$\diag(\l_1,...\l_s,O_{n-s})$，其中 $\l$ 为纯虚数
    - **证明**：由反共轭对称性，只能是纯虚数
  - **关系**：H和反H阵只相差一个因子 $i$，即 $A = iB$

### 习题

- **酉上三角化定理**：任意复矩阵都酉相似于上三角矩阵
  - **证明**：
    - 由之前结论，任意复矩阵相似于上三角矩阵 $R_1 = P^{-1}AP$
    - 由Schmidt正交化，得任意复矩阵可分解为酉矩阵与上三角矩阵的乘积 $P = UR_2$
    - 综上 $A = (UR_2)R_1(UR_2)^{-1} = U^TRU$（**证毕**）
  - <font color='chartreuse'>其实它完全可被奇异值分解平替</font>
- **实矩阵的复特征值**：设 $A$ 是 $n$ 阶实方阵，$a+bi$ 是特征值，$u+vi$ 是对应特征向量，则 $u,v$ 线性无关
  - **证明**：由 $A(u+vi) = (a+bi)(u+vi)$ 易得结论
  - **推论**：若 $A$ 是正规矩阵，则 $u,v$ 相互正交、长度相同
    - **证明**：由正规算子的（伴随算子的特征值共轭性）易得结论

## 其它内积空间

### Lorentz空间（以D为度量矩阵的内积空间）

- **事件的向量**：$X = (x,y,z,ct)$
  - **物理意义**：
    - 前三个分量是事件发生位置的坐标
    - $c$ 是光速，$t$ 是事件发生时间
- **距离**：$|PQ| = (X-Y)^TD(X-Y)$，其中 $D = \diag(1,1,1,-1)$
- **洛伦兹空间**：上述四维空间是内积空间
  - **证明**：
- **洛伦兹变换**：保持洛伦兹内积的变换

### 辛空间（零定度量矩阵内积空间）

- **实数域上的零定二次型**：反对称方阵的二次型
  - **证明**：
    - 显然对任意向量都有 $X^TAX = 0$
- **辛内积**：反对称方阵 $A$ 作为度量矩阵的内积 $(X,Y) = X^TAY$
  - 无对称性
  - 无正定性
  - 自反归零性
- **辛空间**：定义了辛内积的空间

## 双线性函数（内积函数）

- **双线性函数**：满足双线性的向量函数 $f:(V,V)\to\R$
  - 单元函数 $f_\a$：固定一个向量（但是仍然满足双线性）
  - 双元函数 $f$：两个向量均不固定
- **坐标表示**：
  - 设 $f$ 是双线性函数，$M = (\vec{m}_i)$ 是基，向量 $\a,\b$ 在其下坐标为 $X,Y$
  - 则存在方阵 $A$ 满足 $f(\a,\b) = X^TAY$
  - <font color='chartreuse'>方阵 $A$ 依赖于 $f$ 和 $M$</font>
  - **证明（代数构造）**：
    - 基分解后，左式为 $\sum\limits^n_{i=1}\sum\limits^n_{j=1} f(x_i\vec{m}_i,y_j\vec{m}_j)$，右式为 $\sum\limits^n_{i=1}\sum\limits^n_{j=1} x_iy_ja_{ij}$
    - 则只需要定义 $a_{ij} = \cfrac{f(x_i\vec{m}_i,y_j\vec{m}_j)}{x_iy_j}$
    - 由函数线性，$a_{ij} = f(\vec{m}_i,\vec{m}_j)$，此时它只依赖于 $f$ 和 $M$
  - **本质**：内积不过是添加了新性质的双线性函数，这些新性质体现为度量矩阵的性质
- **双线性函数在基M下的矩阵**：$A = \Big( f(\a_i,\a_j)_{ij} \Big)_{n×n}$
    - **方阵性**：$A$ 必须是方阵<font color='chartreuse'>（但无对称性、正定性等要求）</font>
    - **非度量性**：$f(\a,\a) = \frac{1}{2}(A+A^T)$，是对称矩阵
      - <font color='chartreuse'>由A无正定性，不能用 $|\a|^2 = f(\a,\a)$ 定义向量的长度</font>
    - **相合性**：
      - <font color='chartreuse'>在不同的基下，$f$ 的矩阵是仿合同关系</font>
- **对偶空间**：$V$ 上的全体双线性函数 $L(V,V,F)$
  - **线性空间**
    - **证明**：易得
  - **非内积空间**：


### 正交性

- **$\a$ 关于 $\b$ 左正交**：$f(\a,\b) = 0 \LR \a\perp_L\b$
  - **无对称性**：左正交和右正交不同，需分别讨论
    - **实例**：$A = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}$，则 $f(e_1,e_2) = 0$，但 $f(e_2,e_1) = 1$
- **$V$ 在 $f$ 下的左根基**：$V^{\perp_{L}} = \{\b\in V \mid \b \perp_L V\}$
  - <font color='chartreuse'>即 $f(\b,V) = 0$，也可称为 $V$ 关于 $f$ 的左正交补
  - 它只依赖于 $f$，不依赖于基</font>
- **根基求解定理**：
  - 设
    - 双线性函数 $f$ 在基 $M$ 下的矩阵为 $A$
    - 子空间 $W$ 的基矩阵为 $B$
  - 则
    - $W$ 的左根基在 $M$ 下的坐标同构是 $Y^TAB=0$ 的解空间
    - $W$ 的右根基在 $M$ 下的坐标同构是 $B^TAY = 0$ 的解空间
  - **证明**：
    - 定义易得
- **维数定理**：
  - 设
    - 双线性函数 $f$ 在基 $M$ 下的矩阵为 $A$
    - 子空间 $W$ 的基矩阵为 $B$
  - 则
    - $\dim W^{\perp_L} = \dim V - \rank AB$
    - $\dim W^{\perp_R} = \dim V - \rank B^TA$
    - **证明**：
      - 根基求解定理 + 像核维数公式易得结论
- **运算性质**：
  - **左右正交的扩充性**：$(S^{\perp_{L}})^{\perp_{R}} \supseteq S$
  - **对偶律**：
    - **包含对偶**：$W \subseteq U \red\Rt (W^{\perp_{L}}) \supseteq U^{\perp_{L}}$
    - **并对偶**：$(W\cup U)^{\perp_L} = W^{\perp_L} \cap U^{\perp_L}$
    - **交对偶**
    - **差对偶**

### 退化性

- **双线性函数的秩**：$\rank A$
  - <font color='chartreuse'>$f$ 在任意基下的矩阵秩相同</font>
- **非退化双线性函数**：若 $\rank f = \dim V$，则称 $f$ 非退化
  - **根基定义法**：左根基 或 右根基为 $\bold{0}$
      - **证明**：维数定理即可
  - **共轭定义法**：$\forall \a$，设 $\begin{cases} f_\a:\b\mapsto f(\a,\b) \\ \p_\a: \b\mapsto f(\b,\a) \end{cases}$，则 $\a\mapsto f_\a 或 \p_\a$ 都是线性空间到对偶空间的同构
    - **证明**：由同构是双射，故 $\ker f_\a = 0$，从而 $\rank f_\a = \rank A = \dim V$
  - **欧氏内积定义法**：
    - 任意线性函数 $\p$，存在唯一的 $\a_1,\a_2$，使得 $\forall \b\in V$ 都有 $\p(\b) = (\a_1,\b) = (\b,\a_2)$
      - **证明**：
      - **理解**：伴随变换唯一性
- **非退化子空间**：若 $f|_W$ 非退化，则称 $W$ 是 $V$ 关于 $f$ 的非退化子空间
  - **可逆定义法**：$f|_W$ 的矩阵 $A_1$ 可逆
  - **相交定义法**：$W\cap W^{\perp_L} = W\cap W^{\perp_R} = \bold{0}$
    - <font color='chartreuse'>定义易得</font>
- **非退化维数定理**：
  - 设 $f$ 在 $W$ 上非退化
  - 则 $\dim W^{\perp_L} = \dim W^{\perp_R} = \dim V-\rank A$
  - **证明**：
    - 由根基求解定理 + 像核维数公式易得结论
- **非退化对称定理**：
  - 设 $f$ 在 $W$ 上非退化
  - 则 $(\dim W^{\perp_L})^{\perp_R} = \dim W =  (\dim W^{\perp_R})^{\perp_L}$
  - **证明**：
- **非退化直和定理**：
  - 设 $f$ 在 $W$ 上非退化，则 $V = W\oplus W^{\perp_L} = W\oplus W^{\perp_R}$
  - **证明**：维数定理 + 直和定义法即可

### 对称正交

- **正交对称性**：若左正交等价于右正交，则称 $f$ 具有正交对称性
  - **实例**：
    - **对称双线性函数**：$f(\a,\b) = f(\b,\a)$，此时 $A$ 对称
    - **反对称双线性函数**：$f(\a,\b) = -f(\b,\a)$，此时 $A$ 反对称
- **传递定理**：$f$ 定义的正交关系对称 $\LR f$ 对称或反对称
  - **证明**：定义易得
- **对称双线性标准型**：
  - 设 $f$ 对称，则存在正交基 $M$ 使得其下为对角阵 $A = \diag(a_1,...,a_r,0,...,0)$，其中 $r = \rank f$
  - **证明**：由二次型标准型直得结论
  - **推论**：
    - 若 $F = \R$，则 $a_i = \pm 1$
    - 若 $F = \C$，则 $a_i = 1$
    - **证明**：由二次型规范型易得结论
- **反对称双线性标准型**：
  - 设 $f$ 对称，则存在基 $M$ 使得其下为对角阵 $D = \diag(A_1,...,A_s,0,...,0)$，其中 $A$ 是左90度旋转阵
  - <font color='chartreuse'>这里的基和上面不同，不再是正交基</font>
  - **证明（矩阵归纳法）**：
    - $n=1$ 和 $f=0$ 都易得
    - $f\neq 0$ 时
      - **求逆算法**：设 $\a_2 = \frac{\b}{f(\a_1,\b)}$，则 $f(\a_1,\a_2) = 1$
      - 取两个满足 $f(\a,\a) = 0$ 的向量 $\a_1,\b$，则
      - 取 $W$ 为它们生成的子空间，则限制映射的矩阵为 $D_1 = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}$
      - 由正交补直和性，退化二阶
      - 由归纳假设，得 $D_2$，合并得总标准型 $D = D_1D_2$
  - **本质**：反对称方阵标准型，辛空间的度量矩阵标准型

## 共轭双线性函数

- **共轭双线性函数**：满足共轭双线性的函数 $f:(V,V)\to \C$
- **Hermite共轭双线性函数**：$f(\a,\b) = \overline{f(\b,\a)}$，此时 $A$ 是Hermite方阵
- **Hermite型**：单变量函数 $H(\a) = f(\a,\a)$
- **反Hermite**：反共轭对称性

## 更多的例子

- **快速构造酉方阵**：设 $H$ 是Hermite方阵，则
  - $I\pm iH$ 可逆
  - $U = (I+iH)(I-iH)^{-1}$ 是酉方阵
  - **证明**：
    - 由H阵标准型得，$I\pm iH = P(I\pm iD)P^{-1}$，非退化
    - 由H阵标准型得，$U = P(I+iD)(I-iD)^{-1}P^{-1}$，计算得特征值模均为1，从而是酉方阵
  - **理解**：标准型的应用
  - **本质**：快速构造酉方阵的方法
- **Schur不等式**：复数域上，$\tr(\overline{A}^TA) = \sum\limits_{1\leq i,j\leq n} |a_{ij}|^2 \geq \sum\limits^n_{i=1} |\l_i|^2$
  - **证明**：
    - 由复数域上，共轭对称阵均酉相似于上三角矩阵（对角元是特征值）可直得结论
  - **推论**：
    - 只有规范方阵（可酉相似于对角矩阵）才成立等号
    - 实数域中，$\|A\|_F^2 = tr(A^TA) \geq \sum\limits^n_{i=1} \l_i^2$
      - 只有规范方阵（可正交相似于对角矩阵）才成立等号
- **谱分解**：
  - **变换形式**：
    - 设 $V$ 是 $n$ 维内积空间
      - 当 $V$ 是欧氏空间，则 $\p$ 是自伴算子
      - 当 $V$ 是酉空间，则 $\p$ 是复正规算子
    - 设 $\l_i$ 是特征值，$V_i$ 是特征子空间，$E_i$ 是 $V\to V_i$ 的正交投影
    - 则 $\p = \l_1 E_1 + \cdots + \l_k E_k$
  - **矩阵形式**：
    - 实对称矩阵 $A$ 正交相似于对角矩阵 $D$，且 $D = \l_1 E_{11} + \cdots + \l_k E_{kk}$
    - 复规范方阵 $A$ 酉相似于对角矩阵 $D$，且 $D = \l_1 E_{11} + \cdots + \l_k E_{kk}$
- **奇异值分解**：
  - **映射形式**：
    - 设 $U,V$ 是 $n,m$ 维酉空间，$\p:U\to V$ 是线性映射
    - 则存在 $U,V$ 的标准正交基，使得 $\p$ 在其下的矩阵为 $\diag(\mu_1,...,\mu_r,O)_m$
  - **矩阵形式**：
    - 任意复矩阵 $A_{m\times n}$，都存在酉方阵 $U_{m\times m},Q_{n\times n}$，使得 $UAQ = \diag(\mu_1,...,\mu_r,O)_m$。其中 $r = \rank A$
  - **奇异值**：
    - **映射意义**：对某个 $v\in V，u\in U$ 满足 $\p(v) = \mu_i u，\p^*(u) = \mu_i v$ 的值
    - **矩阵意义**：对角元 $\mu_i$
  - **左右奇异向量**：
    - **映射意义**：$\p$ 对应的是右奇异向量（向量右乘），$\p^*$ 对应的是左奇异向量（向量左乘）
    - **矩阵意义**：$U$ 和 $Q$ 的列向量。其满足 $\begin{cases} A^Tu_i = \mu_i u_i \\ Aq_i = \mu_i q_i \end{cases}$
  - **矩阵证明**：
    - 已知 $\overline{A}^TA$ 是半正定实对称矩阵，设其到实对称标准型的正交过渡矩阵为 $Q$，对角元单位化矩阵为 $D$
    - 则设 $B = AQD$，那么 $\overline{B}^TB = \diag(I_r,O_{n-r})$，$B$ 是标准正交基矩阵
    - 设 $B$ 到 $I$ 的过渡矩阵为 $U$，则 $UB = \diag(e_1,...,e_r,0,...,0)$
    - 从而 $UAQ = UBD^{-1}$ 就是标准型，且都是酉过渡方阵
  - **本质**：它不是任意复矩阵均可对角化的意思，因为 $U$ 和 $Q$ 不是互逆关系
    - 它更类似于初等矩阵对角化（秩相抵标准型），但对P和Q做了更严格的约束（必须是酉方阵）
- **极分解**：
  - **变换形式**：
    - 设 $V$ 是 $n$ 维酉空间，$\p$ 是 $V$ 上的线性变换
    - 则 $\p = w\psi$，其中 $w$ 是酉算子，$\psi$ 是半正定自伴算子
      - 其中 $\psi$ 唯一。当 $\p$ 可逆时，$w$ 也唯一
  - **矩阵形式**：任意复方阵 $A_{n\times n}$ 均可分解为半正定共轭对称阵 $H$ 和酉方阵 $U$ 的乘积
    - <font color='chartreuse'>即 $A = H_1U = UH_2$</font>
    - 其中 $H$ 唯一
  - **矩阵证明**：
    - **存在性**：
      - 设奇异值分解为 $A = U_1DU_2$
      - 添项得 $A = (U_1D\overline{U_1}^T)(U_1U_2) = (U_1U_2)(\overline{U_2}^TDU_2)$
        - <font color='chartreuse'>由酉相似的封闭性，第二个等号是成立的（实际上就是正规子群的共轭不动性）</font>
      - 设 $U = U_1U_2，\begin{cases} H_1 = U_1D\overline{U_1}^T \\ H_2 = \overline{U_2}^TDU_2 \end{cases}$ 即可
    - **唯一性**：此时 $A\overline{A}^T = H^2$，由平方根唯一性得到 $H$ 唯一性
  - **本质**：和数值线性代数一样，所谓的分解本质都是标准型 + 添项 = 新过渡矩阵

## 习题

### 前面结论的复述

- **（迹的可交换性）**：酉相似的方阵，所有元的模平方和相等 $\sum\limits_{1\leq i,j\leq n} |a_{ij}|^2 = \sum\limits_{1\leq i,j\leq n} |b_{ij}|^2$
  - **证明**：
    - 已知 $A\overline{A}^T$ 的迹就是其所有元的模平方和。再由迹是相似不变量即可得到结论
  - **本质**：酉相似矩阵的迹相等
- **Witt扩张定理（内积同构的存在性定理）**：
  - 若 $V_m$ 中两组向量 $S=\{\a_i\}_m，T=\{\b_i\}_m$ 满足 $(\a_i,\a_j) = (\b_i,\b_j)$
  - 则存在正交变换 $\mathscr{A}:\a_i\mapsto \b_i$
  - **证明**：选取标准正交基 $M$，其下坐标分别为 $X_i,Y_i$，从而 $X_i^TX_i = Y_i^TY_i$，即坐标阵 $A^TA = B^TB$
    - 由半正定实对称性，正交相似于对角矩阵，过渡矩阵 $P$
    - 再通过单位化矩阵 $D$ 变为 $A_1^TA_1 = B_1^TB_1 = \diag(I_r,O_{m-r})$
    - $A_1、B_1$ 扩充为标准正交基矩阵 $Q_1,Q_2$
    - 设 $P_2 = Q_2Q_1^{-1}$，可证其为 $\mathscr{A}$ 的坐标阵
  - **理解**：标准化后扩张，取过渡矩阵即可，此过程由正交封闭性保正交
  - **本质**：保内积的两个向量组，可扩充为正交变换的像与原像
- **O.Tausky——樊畿fan'ji（正定性的乘积传递）**：
  - 设 $H_1,H_2$ 是两个共轭对称阵
  - 若 $H_1$ 正定，且 $H_1H_2$ 也是共轭对称阵，则 $H_1H_2$ 正定 $\LR H_2$ 正定
  - **证明**：
    - *过渡矩阵累加*：设 $H_1$ 的对角化过渡矩阵是 $P$，$H_2$ 的酉相似过渡矩阵是 $U$，$Q = P_1U$
    - $H_1^{-1} = \overline{P}^TP，H_2 = \overline{P}^TDP$，则 $H_1H_2 = P^{-1}DP$
    - 由于都以D为标准型，故 $H_1H_2$ 和 $H_2$ 正定性相同
  - **理解**：标准型在正定上的体现
- **共轭对称阵平方根存在性**：半正定Hermite方阵 $H$ 存在唯一的平方根，其也是半正定Hermite方阵
  - **证明**：
    - 存在性：过渡矩阵不变，标准型取根号即可（$P^T\sqrt{D}P$）
    - 唯一性：反证，特征标准型唯一性已知
      - 过渡矩阵唯一性：根据不同特征值写成分块形式，分别比较即可
