# 映射习题（补）

- 高代有两种等价的证明方法：映射法和矩阵法
- 映射法直接与几何意义相联系，是最适合过渡到泛函分析的方法
- 矩阵法可无缝衔接数学归纳法，是最容易出结果的证明方法

### 谢启鸿

- **线性映射性质的矩阵法**：设 $\p$ 是 $n$ 维空间 $V$ 上的线性变换
  - 若 $\forall \a\in V，\exist m\in\N^+$ 使得 $\p^m(\a) = 0$，则 $1_V-\p$ 是自同构
  - **证明**：
    - 自同构等价于可逆性
    - 由于基有限，故 $\p$ 实际上是幂零变换
    - 设 $A$ 是某坐标阵，则等价于：若 $A$ 是幂零矩阵，则 $I-A$ 可逆
    - **法1（二级结论）**：根据前面习题中幂零矩阵的坐标阵性质，易得结论（实际上就是Jordan标准型秒了）
    - **法2（泰勒求逆公式）**：$(I-A)(I+A+A^2+...+A^{m-1}) = I-A^m = I$，故可逆
    - **法3**：反设 $\a\in\ker(1_V-\p)$，则 $\p(\a) = \a$，两边作用 $m$ 次 $\p$ 得 $\p^m(\a) = \a = 0$，即 $\ker(1_V-\p) = \bs 0$
- **像核对偶性**：设 $A,B\in F^{m\times n}$，若 $\ker A = \ker B$，则存在可逆矩阵 $P$ 使得 $B = PA$
  - **证明**：由相抵标准型（或线性扩张定理），$P$ 等价于任意初等行变换。由阶梯标准型存在性，只需证明 $\rank A = \rank B$ 即可。而由维数公式，其显然成立
  - **构造性证明**：
    - 转化为已知 $\ms A,\ms B:U\to V$，证明存在线性变换 $\sigma:U\to U$ 使得 $\ms B = \sigma\ms A$
    - 设 $\rank A = r$，则 $\dim\ker A = n-r$
    - **原象空间扩张**：设 $e_{r+1},...,e_{n}$ 是 $\ker\ms A$ 的一组基，将其扩张为 $U$ 的基 $(e_i)$
    - **像空间扩张**：
      - 将 $\ms A(e_1,...,e_r)$ 扩张为 $(\a_1,...,\a_r,f_{r+1},...,f_n)$
      - 将 $\ms B(e_1,...,e_r)$ 扩张为 $(\b_1,...,\b_r,g_{r+1},...,g_n)$
    - **构造线性变换**：
      - 设 $\sigma(\a_i) = \b_i，\sigma(f_i) = g_i$，易得满足条件
  - <font color='chartreuse'>知道了核相等就能知道像的维数相等</font>
  - **实例（线性函数核的唯一性）**：
    - 设 $V$ 是 $F$ 上 $n$ 维线性空间，$f,g$ 是两个线性函数
    - 若 $\ker f = \ker g$，则存在常数 $c\neq 0$ 使得 $g = cf$
    - **证明**：反设 $\begin{cases} f(\a) = a & f(\b) = b \\ g(\a) = c_1a & g(\b) = c_2b \end{cases}$
      - 则 $g(\dfrac{1}{c_1a}\a - \dfrac{1}{c_2b}\b) = 0$，但 $f(\dfrac{1}{c_1a}\a - \dfrac{1}{c_2b}\b) \neq 0$，矛盾
- **特殊的幂零变换**：
  - 设 $V$ 是 $F$ 上 $n$ 维线性空间
  - 若两个线性变换满足 $\p^2 = 0，\psi^2 = 0，\p\psi+\psi\p = 1_V$，则
  - **直和性**：$V = \ker\p \oplus \ker \psi$
    - **证明**：
      - $\p\psi(\a) + \psi\p(\a) = \a$，而 $\p\psi(\a) \in \ker\p，\psi\p(\a) \in \ker\psi$，故 $V$ 是两者的和
      - 元素分析法易得 $\ker\p\cap \ker\psi = \bs 0$，故是直和
  - **维数关系**：$n$ 是偶数
    - **证明**：
      - 设 $\dim\ker\psi = k$，一组基为 $e_1,...,e_k$
      - 设 $\p(e_i) = e_{i+k}$，由幂零性得它们都属于 $\ker\p$
        - 非零式 + 两边作用 $\psi$ 易得线性无关性，即 $\dim\ker\p \geq \dim\ker\psi$
        - 同理可得 $\dim\ker\psi \geq \dim\ker\p$
        - 综上即得它是 $\ker\p$ 的一组基（**证毕**）
  - **坐标阵通式**：设 $n=2k$，则存在一组基使得 $\p,\psi$ 的坐标阵分别为 $\diag(A,...,A)_k$ 和 $\diag(B,...,B)_k$，即每个都有 $k$ 个矩阵元
    - **证明**：
      - 将上面求得的基向量按照 $(e_1,e_{k+1}，e_2,e_{k+2},...,e_{k},e_{2k})$ 的格式排列
      - <font color='chartreuse'>（A和B都是二维直和不变子空间的坐标阵）</font>
- **杂化定理推论**：若 $\p_1,...,\p_n$ 是互不相同的线性变换，则存在 $\a\in V$ 使得 $\p_i(\a)$ 两两不同
  - **证明**：
    - 设 $V_{ij} = \{v\in V \mid \p_i(v) = \p_j(v)\}$，则 $V_{ij} = \ker(\p_i-\p_j)$
    - 由于两两不同性，其为真子空间
    - 再易得 $n$ 维线性空间的真子空间最多有 $2^n-1$ 个，故由之前的杂化定理，存在 $\a\notin\forall V_{ij}$
- **投影变换**：设 $\p:V\to V$，若 $\Im\p = U，\p|_{U} = 1_U$，则
  - 存在 $W$ 使得 $\p|_{W} = 0_W$，且 $V = U\oplus W$
    - **证明**：
      - 显然 $U = \Im\p，W = \ker\p$，容易证明它们不交，故为直和
  - **对偶投影**：存在 $\psi$ 使得 $\psi|_{U} = 0_U$，且 $\rank\p + \rank\psi = \dim V$
    - **证明**：
      - 设 $\psi(u_i) = 0，\psi(w_i) = w_i$，由线性扩张定理得存在唯一性。容易证明秩关系

### 无穷维线性空间的特殊结论

- 设无穷维空间 $V$ 中，若线性变换 $fg，gf$ 都可逆，则 $f,g$ 都可逆
  - **证明**：
    - 设 $(fg)^{-1} = \p，(gf)^{-1} = \psi$
      - 则 $(\p f)g = 1_V$，即 $g$ 有左逆，是单射
      - 同时 $g(f\psi) = 1_V$，即 $g$ 有右逆，是满射
    - 综上即得 $g$ 是双射，从而可逆。$f$ 同理
- 无穷维空间中，存在线性变换使得 $\p\psi-\psi\p = 1_V$
  - **证明**：设 $V$ 是多项式空间，其中 $\p(f) = f'，\psi(f) = xf(x)$
  - **推论**：有穷维空间中，该变换不存在
    - **证明**：即 $AB-BA = I$，由迹的线性+可交换性，两边取迹得 $0=\dim V$，矛盾
