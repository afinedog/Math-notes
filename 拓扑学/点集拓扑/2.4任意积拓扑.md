# 任意维度的积拓扑

- **符号约定**：
  - $\R^\o$ 表示可数个实数空间的笛卡尔积

## 任意笛卡尔积

- **元组**：
  - **笛卡尔积定义法**：
    - 设 $J$ 是一个集合，若用其中元素充当笛卡尔积的索引，则称具有索引的笛卡尔积 $\bs x = (x_\a)_{\a\in J}$ 为一个 $J$ 元组
  - **映射定义法**：
    - 设 $J$ 是一个集合，则映射 $\bs{x}:J\to X，\a \mapsto x_\a$ 称为 $X$ 的元素的 $J$ 元组
    - <font color='chartreuse'>虽然 $J$ 元组在定义中是映射，但容易证明它和以 $J$ 作为指标族下的笛卡尔积一一对应</font>
    - 若取 $X = \mathop{\bigcup}\limits_{\a\in J} A_\a$，则全体 $A_\a$ 的笛卡尔积可表示为全体 $J$ 元组的集合
- **指标集合**：集合 $J$
- **指标映射**：$\bs{x}(\a) = \bs x_\a = x_\a$


## 任意积空间

### 两种积诱导拓扑

- **任意积空间**：$\prod\limits_{\a\in J} X_\a$
  - **因子空间**：$X_\a$
  - <font color='red'>由于是积运算诱导的拓扑，故开集只能为开矩形。区别在于粗细程度</font>
- **箱拓扑**：
  - 设 $\hkh{\Big( X_\a,\tau_\a \Big)}_{\a\in J}$ 是拓扑空间的集族
  - 则 $\mc B = \hkh{\prod\limits_{\a\in J} U_{\a} \Bigm| U_\a\in \tau_\a}$ 可在积空间 $\prod\limits_{\a\in J} X_\a$ 上生成一个拓扑，称为箱拓扑
    - <font color='chartreuse'>全体开集的积组成的拓扑基，形状为任意开矩形</font>
  - **开集**：$\prod\limits_{\a\in J} U_\a$
- **积拓扑**：
  - 设 $\hkh{\Big( X_\a,\tau_\a \Big)}_{\a\in J}$ 是拓扑空间的集族
  - 则取 $\mc S = \hkh{\pi_{\a}^{-1}(U_{\a}) \mid \a\in J，U_\a\in\tau_\a}$ 为子基，其可在积空间 $\prod\limits_{\a\in J} X_\a$ 上生成一个拓扑，称为积拓扑
    - <font color='chartreuse'>全体开集的逆投影组成的子基，形状为长条的有限交
    - 无限维积空间被有限交后，长条依然不能变成点，而是变成低维的无限维长条</font>
  <!-- - **子基的另一种写法**：设 $\mathcal{S}_\b = \{\pi_\b^{-1}(U_\b)\mid U_\b \in \tau_\b\}$，则子基为 $\mathcal{S} = \mathop{\bigcup}\limits_{\beta\in J} \mathcal{S}_\beta$ -->
  - **指标投影映射** $\pi_\beta:\prod\limits_{\a\in J} X_\a\to X_\beta$
    - <font color='chartreuse'>将积元素投影到具体维度</font>
    - **连续性**
  - **投影性**：$S\in \mc S \LR \forall \a\in J，\pi_\a(S) = U_\a$
    - <font color='chartreuse'>定义易得</font>
  - **有限支性**：设 $U$ 是积拓扑上的开集，则至多存在有限个 $\a$ 满足 $U_\a\neq X_\a$
    - <font color='chartreuse'>定义易得</font>
  - **拓扑基**：
    - **写法1**：$B = \pi_{\beta_1}^{-1}(U_{\beta_1}) \cap ... \cap \pi_{\beta_n}^{-1}(U_{\beta_n})$
      - <font color='chartreuse'>（因子空间开集的逆投影）的（全体有限交）</font>
    - **写法2**：$B = \prod\limits_{\a\in J} U_\a，U_\a = \begin{cases} X_\a，& \a\neq\beta_1,...,\beta_n（长条的无界面） \\\\ U_\a，& 其它情况\qquad （长条的有界面）\end{cases}$
- **（定理19.1）箱拓扑和积拓扑的关系**
  - 若指标集 $J$ 是有限集合，则箱拓扑等价于积拓扑
    - **证明**：
      - 长条可以通过有限交化为点
  - 若指标集 $J$ 是无限集合，则箱拓扑细于积拓扑
    - **证明**：
      - 元素分析法得显然
  
### 性质的传递
  
- **（定理19.2）基的任意积性**：因子空间的基的积 $\hkh{\prod\limits_{\a\in J} B_\a }$ 是箱拓扑的积
  - **证明**：  箱拓扑定义即可
- **（定理19.3）子空间的任意积性**：若因子空间的子空间具有箱/积拓扑，则其任意积也具有积空间中的箱/积拓扑
  - **证明**：将前面的有限情况证明推广即可
- **（定理19.4）H空间的任意积性**：若因子空间是H空间，则其任意积空间取箱拓扑或积拓扑时都是H空间
  - **证明**：将豪斯多夫空间的有限情况证明推广即可
- **（定理19.5）闭包的任意积性**：无论取箱拓扑或积拓扑，都有 $\prod\limits_{\a\in J} \ol{A_\a} = \overline{\prod\limits_{\a\in J} A_\a}$
  - **证明**：
    - 只需考虑积拓扑情况
    - **左 $\subset$ 右**：
      - 任取 $\bs x\in \prod \ol{A_\a}$，设 $U$ 是其邻域
      - 由映射保包含性得 $\forall \a，\pi_\a(\bs x) \in (U_\a\cap A_\a)$
      - 即 $\bs x$ 的任意邻域都和 $\prod A_\a$ 有重叠，即 $\bs x$ 是其聚点，从而 $\bs x\in \ol{\prod A_\a}$
    - **右 $\subset$ 左**：
      - 任取 $\bs x\in \ol{\prod A_\a}$，设 $V_\a$ 是 $x_\a$ 的邻域
      - 则由逆投影的开映射性得 $\pi^{-1}(V_\a)$ 是 $\prod A_\a$ 中的开集，包含一点 $y$ 使得 $y_\beta\in V_\beta\cap A_\beta$
      - 从而 $x_\a$ 的任意邻域都和 $A_\a$ 有重叠，即 $x_\a\in \ol{A_\a}$。再由 $\a$ 的任意性即得结论
  - **理解**：
- **（定理19.6）任意积连续定理**：
  - 设
    - $J$ 是指标集，$\hkh{f_\a: A\to X_\a}_{\a\in J}$ 是一族映射
    - $f:A\to \prod\limits_{\a\in J}X_\a，f(a) = \Big( f_\a(a) \Big)_{\a\in J}$
    - <font color='chartreuse'>任意维度的向量值函数</font>
  - 若 $\prod\limits_{\a\in J}X_\a$ 上存在积拓扑，则 $f$ 连续 $\LR \forall f_\a$ 均连续
  - **证明**：
    - **必要性**：指标投影映射的连续性 + 复合传递连续性即可
    - **充分性**：
      - 由开集的有限交传递性，$f$ 连续性等价于 $f^{-1}(S)$ 均为开集
      - 由穿脱原理，$f^{-1}\Big( \pi^{-1}_\beta(U_\beta) \Big) = f^{-1}_\beta(U_\beta)$，左边是子基的逆像，右边为开集（**得证**）
  - **理解**：积拓扑的意义是将有限引入了无限，从而同胚性质得以传递到任意维度上
  - **反例（箱拓扑不满足任意积连续定理）**：
    - 设恒等扩充映射 $f:\R\to\R^\o，f(t) = (t,...,t,...)$，其在积拓扑上连续，但在箱拓扑上不连续
    - **证明**：
      - 取箱拓扑的开集<font color='chartreuse'>（它不满足有限支性，不是积拓扑的开集）</font> $$B = \prod^\infty_{n=1} (-\frac{1}{n},\frac{1}{n}) = (-1,1)\times (-\frac{1}{2},\frac{1}{2})\times \cdots$$ 
      - 易得 $f^{-1}(B) = \{0\in\R\}$，不是开集
        - <font color='chartreuse'>若原象为开集，则只能是开区间 $(-\delta,\delta)$，则其属于 $\forall (-\frac{1}{n},\frac{1}{n})$，是不可能的。</font>
