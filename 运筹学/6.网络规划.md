# 网络计划技术

## 基本概念

- **节点和箭线**：带箭头的射线
- **工作**：将项目分解成消耗成本的子项目
- **B是A的紧前工作**：A必须在B后开始
- **事件**：某项工作开始或结束
- **路线**：从开始事件到最终事件
- **网络图**
  - **箭线图**：双节点图
  - **节点图**：单节点图

### 箭线图

- 节点数值：开始或结束时刻
- 箭头数值：工作名称 + 消耗的成本

#### 绘制步骤

1. 画出开始节点 $0$
2. 找出没有紧前工作的工作，画出箭线和后节点
3. 有紧前工作的工作
   1. 单个：直接在紧前工作后画节点
   2. 多个：虚拟工作
4. 画出最后节点，将没有后续箭线的节点通过虚拟工作与之连接

### 节点图

- 节点数值：工作名称
- 箭头数值：紧前关系
- 没有时间参数

## 时间参数

### 工作持续时间

- **单时估计法**：以工作班作为时间单位
  - 工作的量 $Q$
  - 投入设备量 $R$
  - 设备工作效率 $S$
  - 工作班数 $n$
  - 工作持续时间 $D = \cfrac{Q}{R\cdot S\cdot n}$
- **三时估计法**：
  - 乐观时间 $a$
  - 最可能时间 $m$
  - 悲观时间 $b$
  - 工作持续时间 $D = \cfrac{a+4m+b}{6}$
  - 方差 $\d^2 = (\cfrac{b-a}{6})^2$

### 节点时间

- **节点时间**：节点出现的最早和最晚时间
  - **最早时间 $ET$** 
  - **最晚时间 $LT$**
- **时间公式**：
  - $ET_j = \max\limits_{(i,j)\in A} \{ET_i + D_{ij}\}$
  - $LT_j = \min\limits_{(i,j)\in A} \{LT_j - D_{ij}\}$
- **工作时间**：
  - **最早开始时间**：
    - $ES(i,j) = ET_i$
  - **最早完成时间**：按最早开始时间，能达到的完工时间
    - $EF(i,j) = ES + D_{ij}$
  - **最晚开始时间**：
    - $LF = LT_j$
  - **最晚完成时间**：
    - $LS = LF - D_{ij}$
