# 整数线性规划

- **分类**：0-1问题、混合整数线性规划、纯整数线性规划
- **格点**：ILP的解是离散整数点

## Gomory割平面法

- **松弛问题**：ILP松弛为等价LP问题
- **分数对偶割平面算法**：
  1. 单纯形法解松弛问题，得非整数最优解 $x_0$
  2. 添加线性约束（割平面方程），割去 $x_0$ 的非整数邻域
  3. 用对偶单纯形法求新问题
     - 表中含有分数，且始终保持对偶可行性
- **割平面方程**：$- \sum\limits_{j\in \bar{S}} f_{lj}x_j + s = -f_l$（$l$ 行的解不为整数，即 $(\ol b_l)\neq 0$）
  - 若 $\bar{b}_l$ 不是整数，设其约束方程 $x_{B_l} + \sum\limits_{j\in \bar{S}}\bar{a}_{lj}x_{j} = \bar{b}_l$
  - 由x是整数，得 $x_{B_l} + \sum\limits_{j\in \bar{S}} \big[ \bar{a}_{lj} \big]x_{j} \leqslant \big[ \bar{b}_l \big]$
  - 等式减去不等式，得 $- \sum\limits_{j\in \bar{S}} (a_{lj})x_j \geqslant (b)_l$
  - 取反后添加松弛变量 $s$，即得割平面方程
- **对偶可行性**：$s= -f_l$ 对应新松弛问题的基本不可行解和对偶可行解
  - **证明**：
    - 新单纯形表中基本解中 $s$ 不满足非负性，不可行
    - 由于添加割平面方程后不改变 $\zeta$，从而对偶可行

## 分枝定界法

- **分枝**：求出松弛问题最优解 $x^0$
  - 若分量 $x^0_i$ 不是整数，则该分量上解位于 $x_i\leqslant [x^0_i]$ 或 $x_i\geqslant [x^0_i]+1$
  - 依次进行下去，便得到一个二叉树（**分枝树**）
- **定界**：单纯形法解松弛问题，解即为ILP的下界
- **剪枝（死点）**：由于松弛问题解的下界性，若某个中间松弛问题 $(P_k)$ 的最优值 $z_k < z_m$，则 $(P_k)$ 的分枝均小于 $(P_m)$ 的分枝，从而可以舍去
- 步骤
  1. 初始化
     - **活点集合** live = 原问题O
     - 上界 $U = +\infty$
     - 当前最优解 $z_0 = \varnothing$
  2. 选择分枝点 $k$，从活点集合中删去。若无活点则到7
  3. 解松弛问题 $(P_k)$，若无解则回到2
  4. 若 $z_k \geq U$，则将k剪枝，回到2
  5. 若 $(P_k)$ 的解是整数，则更新 $U$ 和 $z_0$，回到2
  6. 若 $(P_k)$ 的解不是整数，则某个分量生成后代点，将其加入活点，回到2
  7. 若 $z_0 = \varnothing，U=+\infty$，则无解。否则有最优解
