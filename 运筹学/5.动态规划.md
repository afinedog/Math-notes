# 动态规划

## 多阶段决策问题

### 几个经典问题

- **最短路问题**：
  - **第一类**：各个路线的边数相等
  - **第二类**：各个路线的边数可以不等
  - **实例**：
    - 管道设计问题：铺设 $A\to E$ 的管道，其中三个中间节点分别从 $\{B_i\}，\{C_i\}，\{D_i\}$ 中选取<font color='chartreuse'>（其是一个三阶段最短决策问题）</font>
    - 最大流问题？
- **资源分配问题**：
  - 资源总数 $x$
  - 生产方式 $A,B$
  - 投入-回报函数 $g(y) + h(x-y)$
    - <font color='chartreuse'>其中 $y$ 是投入 $A$ 的资源，从而 $x-y$ 是投入 $B$ 的资源，$g,h$ 分别是方式 $A,B$ 的回报</font>
  - 回收率：$a,b$
    - <font color='chartreuse'>资源被投入后，还可以收回一部分</font>
  - （多阶段的资源分配问题是动态规划）
- **生产-库存问题**：
  - 生产周期有 $n$ 个阶段
  - 最初库存 $X_1$
  - 第 $i$ 阶段的需求量 $d_i$
  - 固定成本 $C$，单位成本 $L$
  - 单位库存费 $h$，库存容量 $M$
  - 阶段生产能力 $B$

### 一般模型

- **一般的多阶段决策问题**：存在多个阶段的系统
  - 状态 $\{x_k\}^n_{k=1}$
  - 决策集族 $\{Q_k(x_k)\}^n_{k=1}$，决策 $q_k\in Q_k(x_k)$
  - 状态转移方程 $x_{k+1} = T_k(x_k,q_k)$
  - 效益 $R_k(x_k,q_k)$
  - 总收益 $\sum\limits^n_{k=1} R_k(x_k,q_k)$
- **分类**
  - 根据阶段数：
    - 有限阶段决策问题
    - 无限阶段决策问题
  - 根据变量取值
    - 连续多阶段决策问题
    - 离散多阶段决策问题
  - 根据阶段个数是否已知
    - 定期多阶段决策
    - 不定期多阶段决策
  - 根据参数取值情况
    - 确定多阶段决策
    - 不确定多阶段决策

### 反递推法

- **动态规划方法（反递推法）**：
  - $f_1(x_n) = \min\limits_{q_k\in Q_k(x_k)} \set{R_n(x_n,q_n)}$
  - $f_{n-k+1}(x_k) = \min\limits_{q_k\in Q_k(x_k)} \set{R_k(x_k,q_k) + f_{n-k}(T_k(x_k,q_k))}$
- **实例**：
  - 最短路问题中，假设 $A\to D_1$ 已取最短路的 $A\to D_1$ 部分，那么从 $E$ 反推 $D_1\to E$ 时，必定取的是 $D_1\to E$ 的最短路<font color='chartreuse'>（因为整体最短已经确定了，前面最短也已经确定了，后面没理由不取最短）</font>

### 最优化原理

- **动态规划最优化原理**：某动态规划系统中，最优策略对第一个决策后的状态作为初始状态时，必须构成最优策略<font color='chartreuse'>（与初始状态和策略无关）</font>
  - **递推公式**：
    - $f_1(x) = \min\limits_{0\leq y\leq x} \set{g(y)+h(x-y)}$
    - $f_k(x) = \min\limits_{0\leq y\leq x} \set{g(y)+h(x-y) + f_{k-1}(ay+b(x-y))}$
  - **前向最优化原理**：若 $A\to C$ 的最优路线是 $I$，则 $B\to C$ 的最优路线一定是 $I$ 在其上的限制
    - **证明**：反证法易得
  - **后向最优化原理**：若 $A\to C$ 的最优路线是 $I$，则 $A\to B$ 的最优路线一定是 $I$ 在其上的限制
    - **证明**：反证法易得
- **算法**：
  - 找出问题种类、阶段数量
  - 确定变量，找出状态变量、决策变量
  - 找出状态转移方程
  - 写出递推关系式
  - 求解递推关系式

### 最优线路问题

- **最优线路问题**：给定有限个点 $\{p_i\}^N_{i=1}$，距离为 $c_{ij}$
  - 设无弧长度为 $+\infty$，自身距离为 $0$
  - 指定某点 $p_N$ 为终点，求 $p_i\to p_N$ 的最短路线
- **最优化原理**：$\begin{cases} f(i) = \min\limits_{j} \{c_{ij} + f(j)\} \\ f(N) = 0 \end{cases}$

#### 解决方法

- **函数空间迭代法**：
  - 设初始函数为 $\begin{cases} f_1(i) = c_{iN} \\ f_1(N) = 0 \end{cases}$
    - <font color='chartreuse'>从倒数第二点到最终点 $N$ 的最优距离</font>
  - 反推易得递推关系为 $\begin{cases} f_k(i) = \min\limits_j \{c_{ij} + f_{k-1}(j)\} \\ f_k(N) = 0 \end{cases}$
    - <font color='chartreuse'>从倒数第 $k+1$ 点到最终点 $N$ 的最优距离</font>
  - **单调下降收敛性**：该递推函数单不增收敛于解 $f(i)$
    - **证明（单不增性）**：由最小性，$f_k(i) \leq c_{ii} + f_{k-1}(i) = f_{k-1}(i)$
    - **证明（收敛性）**：
      - 由于 $f(i)$ 只有有限个，故 $\forall \e>0，\exists k_0>0，\forall k\geq k_0$ 有 $|f_k(i) - f(i)| \leq \e$ 一致成立
    - **证明（解性）**：取 $k\geq k_0$，则有 $f(i) < \e + f_{k+1}(i)$
      - 再由 $f_{k+1}(i)$ 的定义，即得 $f(i) < \min\limits_j \{c_{ij} + f(j)\} + \e$
      - 同理可得 $f(i) > \min\limits_j \{c_{ij} + f(j)\} - \e$
    - 综上即得 $f(i) = \min\limits_j \{c_{ij} + f(j)\}$，也就是最优化原理的解
    - <font color='chartreuse'>本质和最短路问题相同，只是需要多证明一次收敛性</font>
- **策略空间迭代法**：
  - **策略**：设初始点为 $p_i$，若选定下一步到达点 $s(i)$，则称为选择了策略 $\{s(i)\}$
  - **初始策略**：$s_0(i)$
  - **回路**：$i_1\to i_2 = s_0(i_1) \to \cdots \to i_1 = s_0(i_p)$
  - **策略方程组**：$\begin{cases} f_0(i) = c_{is_0}(i) + f_0\Big(s_0(i)\Big) \\ f_0(N) = 0 \end{cases}$
    - 若无策略回路，则存在唯一解 $f_0(i)$。此时可求 $\min\limits_j \{c_{ij} + f_0(j)\}$
    - 设此时最小值的 $j = i_1$，设 $s_1(i) = i_1$，即得新策略，再作策略方程组，逐步迭代

### 实例

- **广义性**：函数空间迭代法是策略空间迭代法的特殊情况，即取 $s_0(i)$ 均为 $N$ 时就是函数空间迭代法
